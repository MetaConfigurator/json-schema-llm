The schema describes the specification of a rendering rule for how the requested image should be rendered. It is an object type. The main properties of the schema are: 'outputPixelType' is a string that defines the output image's pixel type. It is optional and defaults to 'UNKNOWN'. Allowed values are 'C128', 'C64', 'F32', 'F64', 'S16', 'S32', 'S8', 'U1', 'U16', 'U2', 'U32', 'U4', 'U8', 'UNKNOWN'. 'rasterFunction' is a string that identifies the processing or rendering to be performed. It is optional. 'rasterFunctionArguments' is an object that holds the arguments for the referenced raster function. It is optional. 'variableName' is a string that serves as the variable name for the raster function. It is optional. No additional properties are allowed at the root level.
