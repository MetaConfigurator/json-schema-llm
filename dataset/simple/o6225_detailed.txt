The schema describes a configuration for handling statements. It is an object type. The main properties of the schema are: 'statements' is an array of objects that must contain at least one item. Each item in the array is a statement object which has the following properties: 'expression' is a string that describes a JSON path wrapped in {{this.is.a.path}}, and it is required with a minimum length of 1. 'resultPath' is a string that describes the JSON path to place the result of the evaluated expression, and it is also required with a minimum length of 1. No additional properties are allowed at the root level.
