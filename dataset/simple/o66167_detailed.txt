The schema describes a Simple Interpolated Publisher. It is an object type. The main properties of the schema are: 'publisher_type' is a string that is required and must be one of the enum values: 'interpolated-pub'. 'glob' is a boolean that is optional with a default value of false. 'relative_paths' is a boolean that is optional with a default value of false. 'publish' is an object that is required. No additional properties are allowed at the root level.
