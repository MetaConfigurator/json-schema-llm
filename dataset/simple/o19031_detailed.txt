The schema describes localized language(s) and characterset(s) used within the resource being described. It is an object type. The main properties of the schema are: 'characterSet' is a string representing the full name of the character encoding standard used by the resource. It is required. 'country' is a string designating the specific country of the locale language (ISO 3166-1). It is optional. 'language' is a string representing the language(s) used within the resource. It should be a valid ISO 639-3 code. It is required. No additional properties are allowed at the root level.
