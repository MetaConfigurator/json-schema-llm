The schema describes a DirectAwardProject. It is an object type. The main properties of the schema are: 'active' is a boolean that indicates whether the project is active; 'created_at' is a string formatted as date-time representing when the project was created; 'downloaded_at' is a string formatted as date-time indicating when the project was downloaded; 'external_id' is a string used for an external identifier; 'id' is an integer that serves as the unique identifier for the project and is required; 'locked_at' is a string formatted as date-time indicating when the project was locked; 'name' is a string with a maximum length of 255 characters and is required; 'ready_to_assess_at' is a string formatted as date-time indicating when the project is ready for assessment; 'still_assessing_at' is a string formatted as date-time indicating when the project is still under assessment. No additional properties are allowed at the root level.
