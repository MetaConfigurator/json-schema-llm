The schema describes an authentication ADFS proxy profile. It is an object type. The main properties of the schema are: 'adfstruststatus' is a string, which is readonly and can have one of the following values: 'INIT', 'FAILED', 'ESTABLISHED', 'ESTABLISHED/CONFIGURED', 'ESTABLISHED_RENEW_SUCCESS', 'ESTABLISHED_RENEW_FAILED', 'RENEWED/CONFIGURED'. 'certkeyname' is a string, optional. 'name' is a string, optional. 'password' is a string, optional. 'serverurl' is a string, optional. 'username' is a string, optional. No additional properties are allowed at the root level.
