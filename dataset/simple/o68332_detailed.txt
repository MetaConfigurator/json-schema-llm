The schema describes parameters accepted by the Sharethrough adapter. It is an object type. The main properties of the schema are: 'pkey' is a string placement key to use, required. 'iframe' is a boolean whether or not to stay in iframe, optional, with a default value of false. 'iframeSize' is an array of two integers representing iframe dimensions, optional, with a default value of [0, 0], having minimum and maximum items set to 2. 'bidfloor' is a number representing the floor price, or minimum amount, a publisher will accept for an impression, given in CPM in USD, optional. No additional properties are allowed at the root level.
