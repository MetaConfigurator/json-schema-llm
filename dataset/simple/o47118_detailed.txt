The schema describes a mutation text. It is an object type. The main properties of the schema are: 'mutation' is a string providing the mutation description, 'id' is a string representing the unique identifier, 'geneMutations' is an array of strings representing gene mutation entries, with no additional items allowed, 'type' is a string specifying the type of mutation, and 'start' is a string indicating the starting position of the mutation. No additional properties are allowed at the root level.
