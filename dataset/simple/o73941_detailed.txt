The schema describes settings for Azure Webjob configuration. It is an object type. The main properties of the schema are: 'stopping_wait_time' is an integer that specifies the graceful shutdown period for an Azure Webjob in seconds. It is optional. 'is_singleton' is a boolean that specifies if the Webjob should be treated as a singleton, ensuring it does not scale on multiple Azure Website instances like the web application itself. It is optional. No additional properties are allowed at the root level.
