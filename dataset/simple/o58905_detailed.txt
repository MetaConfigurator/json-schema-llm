The schema describes an account structure. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'account' is an object that contains properties with the following structure: 'contact_name' is a string that is required, with a minimum length of 3 and a maximum length of 255. 'contact_email' is a string that is required, formatted as an email. 'contact_phone_number' is a string that is required, with a minimum length of 7 and a maximum length of 20. 'host' is a string that is optional, with a minimum length of 3 and a maximum length of 50. 'name' is a string that is required, with a minimum length of 4 and a maximum length of 255. No additional properties are allowed at the root level.
