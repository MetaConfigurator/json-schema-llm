The schema describes a message request for triggering operations. It is an object type. The main properties of the schema are: 'requestedMessage' is a string which is required and must be one of the following values: 'BootNotification', 'DiagnosticsStatusNotification', 'FirmwareStatusNotification', 'Heartbeat', 'MeterValues', 'StatusNotification'. 'connectorId' is an integer and is optional. No additional properties are allowed at the root level.
