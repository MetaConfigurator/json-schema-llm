The schema describes a media item. It is an object type. The main properties of the schema are: 'id' is a string optional. 'title' is a string optional with a minimum length of 3 and a maximum length of 50. 'rating' is a number optional with a minimum value of 1 and a maximum value of 5. 'director' is a string optional. 'actors' is an array optional with string items. 'category' is a string optional with allowed values: 'thriller', 'comedy'. 'createdAt' is a string optional in the date-time format. No additional properties are allowed at the root level.
