The schema describes controlling LEDs. It is an object type. The main properties of the schema are: 'leds' is an array containing objects that represent individual LEDs. It is a required field and must have at least one item. Each item must contain the properties 'id', which is a string describing the identification of the LED, and 'status', which is a string representing the status of the LED and must be one of the following values: 'on', 'off'. No additional properties are allowed at the root level.
