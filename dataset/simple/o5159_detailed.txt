The schema describes the arguments for the RGB procedure. It is an object type. It defines several reusable components under 'definitions': 'ColorComponent' is an integer, representing a color component with a minimum value of 0 and a maximum value of 255. The main properties of the schema are: 'red' is a ColorComponent representing the red parameter, required; 'green' is a ColorComponent representing the green parameter, required; 'blue' is a ColorComponent representing the blue parameter, required. No additional properties are allowed at the root level.
