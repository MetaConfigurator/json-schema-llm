The schema describes a request for shipping details. It is an object type. The main properties of the schema are: 'carrier' is a string representing the acronym of the carrier you want to use. It is required and can be one of the following values: 'dhl', 'ups', 'dpd', 'hermes', 'gls'. 'pickup_date' is a string that must match the pattern '^[0-9]{4}/[0-9]{2}/[0-9]{2}$'. It is required. No additional properties are allowed at the root level.
