The schema describes a structure for tagging data. It is an object type. The main properties of the schema are: 'tags' is an object that categorizes data or tracks references in external systems. It is optional and defines a pattern for keys, which must match '^[a-zA-Z_]{1}[\w\.:-]+$', and for values, which must match '^[\w\/\.:-]+$'. The object does not allow additional properties for the key-value pairs. No additional properties are allowed at the root level.
