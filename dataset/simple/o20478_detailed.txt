The schema describes a refresh token request. It is an object type. The main properties of the schema are: 'grant_type' is a string that is required and must be one of the following values: 'refresh_token'. 'redirect_uri' is a string that is optional. 'refresh_token' is a string that is required and must have a minimum length of 1. 'scope' is an array that is optional and may contain any combination of the following values: 'openid', 'profile', 'email', 'address', 'phone'. No additional properties are allowed at the root level.
