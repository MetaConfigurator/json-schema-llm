The schema describes a system configuration. It is an object type. The main properties of the schema are: 'architecture' is a string representing the architecture of the system the agent is running on. It has a maximum length of 1024 and is optional. 'hostname' is a string representing the hostname of the system the agent is running on. It has a maximum length of 1024 and is optional. 'platform' is a string representing the name of the system platform the agent is running on. It has a maximum length of 1024 and is optional. No additional properties are allowed at the root level.
