The schema describes an industry code from a standard code list (e.g. NAICS 2007 or NACE 2). It is an object type. The main properties of the schema are: 'name' is a string optional. 'code' is a string required. 'code_scheme_id' is a string required, with a description indicating the identifier representing industry code scheme, including values like eu_nace_2, uk_sic_2003, and others. 'start_date' is a string optional, with a format of date. 'end_date' is a string optional, with a format of date. No additional properties are allowed at the root level.
