The schema describes parameters accepted by the Consumable adapter. It is an object type. The main properties of the schema are: 'siteId' is a number, The site ID from Consumable, required, with a pattern of '^[0-9]+$'. 'networkId' is a number, The network ID from Consumable, required, with a pattern of '^[0-9]+$'. 'unitId' is a number, The unit ID from Consumable, required, with a pattern of '^[0-9]+$'. 'unitName' is a string, The unit name from Consumable (expected to be a valid CSS class name), optional, with a pattern of '^-?[_a-zA-Z]+[_a-zA-Z0-9-]*$'. No additional properties are allowed at the root level.
