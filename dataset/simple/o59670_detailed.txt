The schema describes a product. It is an object type. The main properties of the schema are: 'id' is an integer representing the product identifier, required, with a default value of 1. 'name' is a string representing the product name, required, with a default value of 'A green door'. 'price' is a number representing the product price, required, with a default value of 12.5. 'tags' is an array of strings representing product tags, required, with a minimum of 1 item, and items must be unique, with a default value of 'home'. No additional properties are allowed at the root level.
