The schema describes information about cities. It is an object type. The main properties of the schema are: 'id' is a string unique identifier for this city, required. 'order' is a number externally defined ordering of this city in a list, required. 'zoom' is a number initial zoom level to use when showing the city on a map, required. 'city' is a string name of the city used for user viewable labels, required. 'state' is a string state abbreviation for the city, required. 'lat' is a number latitude of the city center, required. 'lon' is a number longitude of the city center, required. No additional properties are allowed at the root level.
