The schema describes the response for an update operation. It is an object type. The main properties of the schema are: 'matched' is an integer representing the number of documents matched by the update, and it is required with a minimum value of 0; 'modified' is an integer indicating the number of documents modified by the update, and it is required with a minimum value of 0; 'upsertedId' is a string representing the ID of the upserted document, and it is optional. No additional properties are allowed at the root level.
