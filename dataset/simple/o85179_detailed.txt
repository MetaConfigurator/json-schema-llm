The schema describes a MWF button component's schema definition. It is an object type. The main properties of the schema are: 'config' is an object that defines the button's configuration, which is required. It includes a required property 'element-type' of type string with allowed values 'button' and 'anchor'. 'content' is an object that defines the content for the button, which is also required. It includes two required properties: 'text' of type string and 'href' of type string. No additional properties are allowed at the root level.
