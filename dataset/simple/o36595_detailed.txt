The schema describes a configuration for bookings. It is an object type. The main properties of the schema are: 'name' is a string and is required. 'validation' is a boolean and is required, with the description: If true, bookings must be validated by moderators. Otherwise, they are automatically validated. 'school_id' is a string and is required. 'color' is a string and is required. 'extendcolor' is a boolean and is optional. 'slotprofile' is a string and is optional. No additional properties are allowed at the root level.
