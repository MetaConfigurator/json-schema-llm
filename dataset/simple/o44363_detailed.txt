The schema describes a phenomenon. It is an object type. The main properties of the schema are: 'name' is a string and required, describing the name of the phenomenon. 'unit' is a string and required, detailing the unit of measurement for the phenomenon. 'created' is a string that must follow the 'date-time' format and is optional. 'modified' is a string that must also follow the 'date-time' format and is optional. No additional properties are allowed at the root level.
