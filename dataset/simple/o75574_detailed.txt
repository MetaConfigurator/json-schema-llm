The schema describes a webhook configuration. It is an object type. The main properties of the schema are: 'id' is a string that represents the webhook id that can be used for requesting info about a webhook or deleting it, required. 'url' is a string that specifies the URL of the webhook, required. 'event_types' is an array that lists the subscribed event types, required. 'deactivated' is a boolean that indicates if the call of the webhook url fails ten times, optional. No additional properties are allowed at the root level.
