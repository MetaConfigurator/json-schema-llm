The schema describes a structured object representing device attributes. It is an object type. The main properties of the schema are: 'attrInfos' is an array of objects representing attributes, which is required and must have at least one item. Each item must include 'key', a string that cannot be empty, and 'valueTemplate', also a string that cannot be empty. 'deviceId' is a string representing the device identifier, which is required and cannot be empty. No additional properties are allowed at the root level.
