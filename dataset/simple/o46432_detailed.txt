The schema describes Other Items provided annually. It is an object type. The main properties of the schema are: 'nonResidentLandlord' is a boolean indicating an explicit declaration that the user is a Non-Resident Landlord. It is optional, with a default of false. 'rarJointLet' is a boolean indicating a notification that the Rent A Room income (RAR) is shared with another individual. It is optional. No additional properties are allowed at the root level.
