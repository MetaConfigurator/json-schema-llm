The schema describes configuration settings for a service using Spark and Zeppelin. It is an object type. The main properties of the schema are: 'service' is an object that contains configuration options for the service, including: 'name' which is a string that defaults to 'zeppelin', and 'zeppelin_java_opts' which is a string that defaults to '-Dspark.mesos.coarse=true -Dspark.mesos.executor.home=/opt/spark/dist' and sets ZEPPELIN_JAVA_OPTS with arbitrary Spark configuration properties. 'spark' is an object that contains settings for Spark, including: 'cores_max' which is a string that defaults to an empty value and sets spark.cores.max, 'executor_docker_image' which is a string describing the Docker image for Spark executors, 'executor_memory' which is a string that sets spark.executor.memory, and 'uri' which is a string that provides a URI for the Spark distribution to use with the Zeppelin Spark context. No additional properties are allowed at the root level.
