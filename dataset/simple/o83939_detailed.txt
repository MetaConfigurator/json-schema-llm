The schema describes a clinical concept display group. It is an object type. The main properties of the schema are: 'displayGroupOnSameAxis' is a boolean indicating whether the concepts in this group should be displayed on the same axis, or each on their individual axis in the same card. It is required. 'graphType' is a string that specifies the way the graph should be plotted for this display group. If NONE, then graph will not be displayed. Allowed values are 'SCATTER', 'LINE', 'DIAGNOSTIC', 'MICROBIO', 'STEP', 'NONE'. It is required. 'groupName' is a string representing the display name for this clinical concept group. It is required. 'parentGroupName' is a string describing the name of the parent group that this group should be displayed with. It is optional. No additional properties are allowed at the root level.
