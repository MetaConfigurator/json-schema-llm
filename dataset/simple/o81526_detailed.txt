The schema describes a user profile. It is an object type. The main properties of the schema are: 'email' is a string that must match the pattern '^[\w|-|.]+@[\w]+\.[A-Za-z]{2,4}$' and is required. 'favoriteTopic' is a string that is optional. 'firstName' is a string that is required. 'lastName' is a string that is required. 'tags' is an array of strings that is optional. Additionally, if 'favoriteTopic' is provided, the 'tags' property must also be present. No additional properties are allowed at the root level.
