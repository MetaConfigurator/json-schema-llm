The schema describes a MWF content toggle component's schema definition. It is an object type. The main properties of the schema are: 'config' is an object that defines the configuration settings. It is required. It has properties: 'expanded' which is a boolean, and 'id' which is a string. Both 'id' and 'expanded' are required. 'content' is an object that defines the content text. It is required. It has properties: 'less-text' which is a string, 'more-text' which is a string, and 'text' which is a string. All three of 'text', 'more-text', and 'less-text' are required. No additional properties are allowed at the root level.
