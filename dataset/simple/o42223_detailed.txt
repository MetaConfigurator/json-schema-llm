The schema describes a JSON Transformer. It is an object type. The main properties of the schema are: 'sourceLocale' is a string with a default value of 'en-US'. 'targetLocale' is a string with a default value of 'en-US'. 'transformMap' is an array that must contain at least one item. Each item in the array is an object requiring 'sourcePtr' and 'targetPtr' as required properties, along with optional 'customFormatter' and 'defaultValue' properties. The 'defaultValue' can be an array, object, boolean, integer, number, or string. No additional properties are allowed at the root level.
