The schema describes an acquisition source. It is an object type. The main properties of the schema are: 'acquisition_source' is an object representing the source of acquisition (former 541). It contains the following properties: 'date' is a string describing the date of acquisition. 'email' is a string describing the email of the submitter. 'method' is an enum representing the method of acquisition, allowing values 'OAI', 'batchupload', and 'submission'. 'source' is a string describing the source of acquisition (e.g. ID of the user). 'submission_number' is an integer describing the internal submission number. No additional properties are allowed at the root level.
