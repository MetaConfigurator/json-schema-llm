The schema describes options for configuring a webpack transpiler. It is an object type. The main properties of the schema are: 'webpack' is an object configuration for '@stryker-mutator/webpack-transpiler', which is optional. It has no additional properties and defaults to an empty object. Its properties are: 'configFile' is a string describing the location of your webpack config file, 'context' is a string describing the webpack context, and 'silent' is a boolean specifying whether to remove the 'ProgressPlugin' from your webpack config file, defaulting to true. No additional properties are allowed at the root level.
