The schema describes a network configuration. It is an object type. The main properties of the schema are: 'MACAddress' is a string that follows the pattern '^([0-9a-fA-F][0-9a-fA-F]:){5}([0-9a-fA-F][0-9a-fA-F])$' and is readonly. 'NetworkSpeed' is an integer with a default value of 0, with a minimum value of 0 and a maximum value of 0. 'StaticIPv4Address' is a string formatted as an ipv4 with a default value of '192.168.0.69'. 'StaticIPv4Gateway' is a string formatted as an ipv4 with a default value of '192.168.0.201'. 'StaticIPv4SubNetMask' is a string formatted as an ipv4 with a default value of '255.255.255.0'. 'UseDHCP' is a boolean with a default value of false. The schema requires 'StaticIPv4Address', 'StaticIPv4SubNetMask', 'StaticIPv4Gateway', and 'UseDHCP' to be present. No additional properties are allowed at the root level.
