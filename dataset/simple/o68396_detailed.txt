The schema describes a structure for MML commands to create a generator. It is an object type. It defines several reusable components under 'definitions': 'column_def' is an object, which requires a 'stattype' property. It has an optional 'guess' property. The 'stattype' property is a string that must be one of the following values: 'CATEGORICAL', 'NUMERICAL', 'IGNORE'. The main properties of the schema are: 'metamodel' is a string and is required. 'columns' is an object and is required, with additional properties that reference the 'column_def' definition. No additional properties are allowed at the root level.
