The schema describes a StyleSymbolReference. It is an object type. The main properties of the schema are: 'name' is a string that identifies a symbol in the style by name. 'styleName' is a string that represents a registered web style name, such as `EsriThematicTreesStyle`. 'styleUrl' is a string that provides a URL to a style definition. 'type' is a string that indicates the type of the symbol, with an allowed value of 'styleSymbolReference'. The schema requires either the properties 'type', 'name', and 'styleUrl' or 'type', 'name', and 'styleName' to be present, which are mutually exclusive configurations. No additional properties are allowed at the root level.
