The schema describes a Vulnerability. It is an object type. The main properties of the schema are: 'Name' is a string with a minimum length of 1 and a maximum length of 512, and it is required. 'Severity' is a string and it is required, with allowed values being 'negligible', 'Negligible', 'low', 'Low', 'medium', 'Medium', 'high', and 'High'. No additional properties are allowed at the root level.
