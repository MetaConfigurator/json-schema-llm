The schema describes RDF JSON definition, see https://github.com/iand/rdf-json. It is an object type. The main properties of the schema are: 'additionalProperties' is an object, describing a subject (root object) with a nested structure where each property can have a value defined as an array of objects. Each object in the array is defined as having the following properties: 'datatype' is a string with a format of uri; 'lang' is a string with a description referring to a BCP 47 document; 'type' is a string with an enum of allowed values including 'uri', 'bnode', and 'literal'; 'value' is a string. No additional properties are allowed at the root level.
