The schema describes Tabris.js Font Resources. It is an object type. It defines several reusable components under 'definitions': 'fontResources' is an object, which contains properties such as '$schema' (a string), '$scaleFactor' (an enum with allowed values 'nearest', 'lower', 'higher'), and '$fallbackLanguage' (a string with a specific pattern). It also includes pattern properties that match strings and reference 'fontResource'. 'fontResource' is a oneOf type that can refer to either 'fontDefinition' or 'fontSelector'. 'fontDefinition' is a oneOf type that can be an object with properties 'family' (an array of fontFamily), 'size' (a number with a minimum value of 1), 'style' (an enum with allowed values 'italic', 'normal'), and 'weight' (an enum with allowed values 'black', 'bold', 'medium', 'thin', 'light', 'normal'). Alternatively, it can be a string matching a specific pattern. 'fontSelector' is a oneOf type with several object configurations including a 'Platform Selector' requiring 'ios' and 'android', a 'Resource Reference' requiring a 'ref' property, an 'Inherit Marker' requiring 'inherit' set to true, a 'Scale Selector' with pattern properties, and a 'Locale Selector' with pattern properties but not requiring 'ios'.
