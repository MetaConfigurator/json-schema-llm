The schema describes filtering points based on the value of a specified bitfield attribute. It is an object type. The main properties of the schema are: 'field' is a string, the name of the field that is used for the filter. It is required. 'requiredClearBits' is an array, listing all bit numbers that must be cleared (=0) for the point to be kept; bit 0 is the least significant bit (LSB). It contains items of type number and is optional. 'requiredSetBits' is an array, listing all bit numbers that must be set (=1) for the point to be kept; bit 0 is the least significant bit (LSB). It contains items of type number and is optional. 'type' is a string with an enum value of 'pointCloudBitfieldFilter', and it is required. The schema requires that at least one of 'requiredClearBits' or 'requiredSetBits' is present due to the 'anyOf' condition. No additional properties are allowed at the root level.
