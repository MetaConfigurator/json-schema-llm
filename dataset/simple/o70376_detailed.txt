The schema describes a data structure for managing LISA records. It is an object type. It defines several reusable components under 'definitions': 'LISAManagerReferenceNumberType' is a string, matching the pattern '^Z\d{6}$'. The main properties of the schema are: 'LISAManagerReferenceNumber' is a string that is required and references 'LISAManagerReferenceNumberType'. 'LISAProviderName' is a string that is required. '_links' is an object. No additional properties are allowed at the root level.
