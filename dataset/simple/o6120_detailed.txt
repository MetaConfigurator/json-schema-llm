The schema describes URL parameter. It is an object type. The main properties of the schema are: 'name' is a string that represents the parameter name and is required. 'type' is a string that represents the parameter type and is optional, with allowed values: 'any', 'int', 'string', 'boolean'. 'pattern' is an item that can be either a string or null and is optional. No additional properties are allowed at the root level.
