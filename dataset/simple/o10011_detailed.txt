The schema describes a client creation request. It is an object type. The main properties of the schema are: 'grantType' is a string that is required. 'redirectUris' is an array of strings that is optional. 'responseType' is a string that is optional. 'scopes' is an array of strings that is optional. The schema has a oneOf constraint, requiring either: (1) 'grantType' to be 'authorization_code' with 'responseType' defaulting to 'code', or (2) 'grantType' to be 'client_credentials' with 'responseType' defaulting to 'token'. No additional properties are allowed at the root level.
