The schema describes a list of products due on a delivery round. It is an object type. The main properties of the schema are: 'doc_type' is a string matching the pattern '^packingList$', required. 'version' is a string formatted as semver, required. 'productID' is a string generated with a 32-character hash length, required. 'expectedQty' is an integer, required. 'packedQty' is an integer, optional. 'deliveredQty' is an integer, optional. 'adjustmentReason' is a string, optional. No additional properties are allowed at the root level.
