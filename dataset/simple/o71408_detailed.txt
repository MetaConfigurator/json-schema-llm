The schema describes a Links object. It is an object type. The main properties of the schema are: 'route' is a string describing the API URL Route GET call for the current route. It is an optional property. 'view' is a string describing a link to the GET operation for the optimization problem. It is an optional property. 'optimization_problem_id' is a string describing the API URL optimization_problem_id GET call for the current route. It is an optional property. No additional properties are allowed at the root level.
