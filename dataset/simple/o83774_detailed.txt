The schema describes a user profile. It is an object type. The main properties of the schema are: 'current' is a boolean with a default value of false, 'dob' is a string, 'image' is a string, 'likes' is a number, and 'location' is a string. 'name' is an object containing 'first', 'last', and 'middle', each of which is a string. The 'name' and 'location' properties are required. No additional properties are allowed at the root level.
