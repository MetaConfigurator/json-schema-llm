The schema describes an object holding application metadata and updates. It is an object type. The main properties of the schema are: 'info' is an object that contains application details, it is required and must have the properties 'name' which is a string, and 'version' which is a string. 'updates' is an array of objects, it is required, and must have at least 0 items, each item must include the properties 'property' which is a string and 'type' which is a string. Each update must be unique. No additional properties are allowed at the root level.
