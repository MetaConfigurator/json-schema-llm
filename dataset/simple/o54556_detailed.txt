The schema describes the data contained in a Table settings object. It is an object type. The main properties of the schema are: 'title' is a string optional status. 'description' is a string optional status. 'maxValues' is an integer optional status with a minimum value of 1. 'showTableGrid' is a boolean optional status. 'newLinePosition' is a string optional status with allowed values 'top' and 'bottom'. 'allowFilter' is a boolean optional status. 'allowSort' is a boolean optional status. 'pageSize' is an integer optional status with a minimum value of -1. No additional properties are allowed at the root level.
