The schema describes a structure for sentiment analysis data. It is an object type. The main properties of the schema are: '@id' is a string optional. 'nif:beginIndex' is an integer optional. 'nif:endIndex' is an integer optional. 'nif:anchorOf' is a string optional, describing a piece of context that contains the sentiment. 'marl:hasPolarity' is a string optional, with allowed values 'marl:Positive', 'marl:Negative', 'marl:Neutral'. 'marl:polarityValue' is a number optional. 'prov:wasGeneratedBy' is a string required, describing the ID of the analysis that generated this sentiment; the full object should be included in the 'analysis' property of the root object. No additional properties are allowed at the root level.
