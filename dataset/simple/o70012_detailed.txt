The schema describes a configuration for filters on elements and archives. It is an object type. It defines several reusable components under 'definitions': 'patterns' is an array, containing strings with a minimum of 1 item and unique values. 'filter' is an object, which requires at least one property from 'include' and 'exclude', both referencing the 'patterns' definition. The main properties of the schema are: 'elements' is a filter, required, referencing the 'filter' definition. 'archives' is a filter, required, referencing the 'filter' definition. No additional properties are allowed at the root level.
