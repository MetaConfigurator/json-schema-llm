The schema describes an object representing a content item. It is an object type. The main properties of the schema are: 'title' is a string representing the title of the content item. 'content' is a string describing the content body. 'status' is an integer indicating the current status of the content. 'publication_date' is a string timestamp representing the start date of the booking. 'expiration_date' is a string timestamp representing the start date of the booking. 'is_headline' is a boolean indicating whether the item is a headline. 'thread_id' is an integer representing the identifier of the thread. No additional properties are allowed at the root level.
