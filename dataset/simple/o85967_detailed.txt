The schema describes configuration for a piece of audio content, over a stream. It is an object type. The main properties of the schema are: 'audio_codec' is a string that describes the codec used to encode the audio stream (e.g. mpeg), optional. 'bitrate' is a number that describes the bitrate of the audio in kilobytes per second, optional. 'filesize' is a number that describes the size of the audio file in bytes, optional. 'stream_type' is a string that describes the type of audio (e.g. mp3), optional. 'url' is a string that describes the file location of the stream and must be in URI format, required. No additional properties are allowed at the root level.
