The schema describes a configuration object. It is an object type. The main properties of the schema are: 'id' is a string and is required. 'vendor' is a string and is required. 'descriptor_version' is a string and is required. 'number_of_endpoints' is an integer and is required. 'root_requirement' is a string and is required. 'test_access' is a string and is required. 'connection' is an array of strings and is required, with each item being a string. 'connectivity_type' is a string and is required. No additional properties are allowed at the root level.
