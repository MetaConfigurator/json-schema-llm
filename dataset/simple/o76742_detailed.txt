The schema describes a user. It is an object type. The main properties of the schema are: 'users' is an object that contains the properties 'content' as a string, 'id' as a string, and 'links' as an object with required property 'user-groups' which is an array. The properties 'username', 'email', 'name', and 'surname' are required at the 'users' level. No additional properties are allowed at the root level.
