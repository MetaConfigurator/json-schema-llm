The schema describes common schema for user application processing stage volumes - used for MI data based save and return completion-rate calculation. It is an object type. The main properties of the schema are: '_id' is a string unique url-friendly base64 encoded, utf8 encoded concatenation identifier. This property is required. '_timestamp' is a string ISO8601 combined date and time in UTC e.g. '2014-12-01T00:00:00+00:00'. This property is required. 'count' is an integer with a minimum value of 0. This property is required. 'period' is a string describing the period covered by each data point from the _timestamp, with allowed values: 'hour', 'day', 'week', 'month', 'quarter'. This property is required. 'stage' is a string describing the application specific process stage. This property is required. No additional properties are allowed at the root level.
