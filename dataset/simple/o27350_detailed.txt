The schema describes build metadata plugin manifest. It is an object type. The main properties of the schema are: 'provider' is a string that is optional and must be one of the values: 'ec2'. 'plugins' is an object that is required and must contain a property 'build_metadata'. 'build_metadata' is an object that is required and must contain a property 'path', which is a string. No additional properties are allowed at the root level.
