The schema describes minimum required output for a server configuration object. It is an object type. The main properties of the schema are: 'host' is a string representing the host name, IPv4 or IPv6 address of the server and is required. 'port' is an integer indicating the port on the server to connect to, with a minimum value of 0 and a maximum value of 65535, and is also required. 'ssl' is a boolean indicating whether or not SSL should be used for the connection and is optional. 'tls' is a boolean indicating whether or not TLS should be used for the connection and is optional. No additional properties are allowed at the root level.
