The schema describes a user's profile. It is an object type. The main properties of the schema are: 'email' is a string that must match the pattern '^[\w|-|.]+@[\w]+\.[A-Za-z]{2,4}$' and is required. 'firstName' is a string and is required. 'lastName' is a string with a maximum length of 20 and is required. 'postedSlides' is a boolean and is required. 'rating' is a number with a maximum value of 5 and is required. No additional properties are allowed at the root level.
