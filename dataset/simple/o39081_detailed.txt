The schema describes a configuration for the go-tag-apis tool. It is an object type. The main properties of the schema are: '$schema' is a string, optional; '$tool' is a string, must be set to 'go-tag-apis'; 'dist' is a string, required, specifying the output dir path (relative to the pwd); 'src' is an array of strings, required, describing input go files (Glob), which must have at least one item and expand with env first; 'structs' is an array of strings, required, listing Go struct names to parse. No additional properties are allowed at the root level.
