The schema describes a spatial reference configuration. It is an object type. The main properties of the schema are: 'latestVcsWkid' is a number (optional) representing the current wkid value of the vertical coordinate system. 'latestWkid' is a number (optional) that identifies the current wkid value associated with the same spatial reference. 'vcsWkid' is a number (optional) for the wkid value of the vertical coordinate system. 'wkid' is a number, which specifies the well-known ID (WKID) of the coordinate system with a default value of 102100. 'wkt' is a string that represents the well-known text (WKT) of the coordinate system. The schema has compositional constraints where at least one of 'wkt' or 'wkid' must be provided, as specified by the oneOf condition. No additional properties are allowed at the root level.
