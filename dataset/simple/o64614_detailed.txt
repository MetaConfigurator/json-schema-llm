The schema describes telemetry data for DOTS. It is an object type. The main properties of the schema are: 'ietf-dots-telemetry:telemetry' is an object containing telemetry data. It has an optional property 'pre-mitigation', which is an array of objects. Each object in 'pre-mitigation' has a property 'target', which is an object containing 'target-prefix', an array of strings. No additional properties are allowed at the root level.
