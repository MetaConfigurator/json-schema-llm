The schema describes the configuration for a test suite. It is an object type. The main properties of the schema are: 'basePath' is a string that specifies the base path of the test suite. It is required. 'casesToRun' is an array of strings that consists of IDs of the cases to be run. If the field is not present, all cases are run. This property is optional. 'manifestPath' is a string that indicates the path of the manifest.jsonld file relative to the validation folder inside the test suite. It is required. No additional properties are allowed at the root level.
