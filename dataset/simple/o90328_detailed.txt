The schema describes the transportation network used for routing in the scene. It is an object type. The main properties of the schema are: 'name' is a string that represents the name of the transportation network. It is required. The name must be unique within a scene. 'path' is a string that indicates the path to the geodatabase which contains the network. It is required and must match the pattern '^file:.+\.geodatabase?$'. 'title' is a string that provides a title or alias of the network layer. This can be used in the client user interface. It is required. No additional properties are allowed at the root level.
