The schema describes an array of objects. It is an array type. The main properties of the schema are: [items] is an object that contains two properties: 'key' is a string that must match the pattern '^[0-9a-zA-Z_-]{1,255}$' and is optional; 'value' is a string with a minimum length of 1 and a maximum length of 255 and is also optional. The array has a maximum length of 100 items. No additional properties are allowed at the root level.
