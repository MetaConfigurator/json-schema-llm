The schema describes appending to a stream. It is an object type. The main properties of the schema are: 'messageId' is a string matching the pattern '^[0-9A-Fa-f]{8}-[0-9A-Fa-f]{4}-[0-9A-Fa-f]{4}-[0-9A-Fa-f]{4}-[0-9A-Fa-f]{12}$', which is required. 'type' is a string, which is required. 'jsonData' is an object, which is required. 'jsonMetadata' is an object, which is optional. No additional properties are allowed at the root level.
