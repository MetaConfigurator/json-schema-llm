The schema describes options available for the legend for visual variables. It is an object type. The main properties of the schema are: 'customValues' is an array of numbers allowing multiple values, which is optional. 'showLegend' is a boolean indicating whether to show the color/size/opacity ramp in the legend, which is optional. 'title' is a string that represents the title of the legend, which is also optional. No additional properties are allowed at the root level.
