The schema describes an object representing a notification. It is an object type. The main properties of the schema are: 'level' is a string that indicates the severity of the notification and is required. Allowed values are 'info', 'warning', 'error', 'critical'. 'state' is a string that represents the current state of the notification and is required. Allowed values are 'new', 'acknowledged', 'resolved'. 'subject' is a string indicating the subject of the notification and is required with a minimum length of 1 and a maximum length of 255. 'message' is a string that provides the notification content and is optional. No additional properties are allowed at the root level.
