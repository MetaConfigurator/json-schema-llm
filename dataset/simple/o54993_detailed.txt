The schema describes a data structure for storing information about items. It is an object type. The main properties of the schema are: 'DATA' is an array containing at least one item, where each item is an object that requires the fields 'ItemId', 'SpecimenId', 'ItemTypeId', and 'LastUpdateTimeStamp'. Each item has additional optional fields: 'ItemBarcode' (string), 'Amount' (number), 'TrialUnitSpecimenId' (integer), 'ItemSourceId' (integer), 'ContainerTypeId' (integer), 'ScaleId' (integer), 'StorageId' (integer), 'UnitId' (integer), 'ItemStateId' (integer), 'LastMeasuredDate' (string), 'LastMeasuredUserId' (string), 'ItemOperation' (string), and 'ItemNote' (string). Additional properties are allowed for the items. No additional properties are allowed at the root level.
