The schema describes integration configuration in Contentful. It is an object type. It defines several reusable components under 'definitions': 'webhook' is an object, configuration for setting up a webhook. It requires the fields 'endpointUrl' and 'name'. 'endpointUrl' is a string and must be a valid URL format, while 'name' is a string. No additional properties are allowed for the webhook. The main properties of the schema are: 'webhooks' is an array a list of webhooks managed by this integration. It contains items that must conform to the 'webhook' definition. No additional properties are allowed at the root level.
