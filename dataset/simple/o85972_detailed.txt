The schema describes a keyword used in describing a piece of content. It is an object type. The main properties of the schema are: 'keyword' is a string that is required and describes the keyword used to describe a piece of content. 'score' is a number that is required and represents an arbitrary weighting to give the keyword. 'tag' is a string that is optional and describes the Part of Speech tag for this keyword. 'frequency' is an integer that is optional and provides a count of the frequency of the keyword as it appears in the content it describes. No additional properties are allowed at the root level.
