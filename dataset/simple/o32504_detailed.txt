The schema describes the structure of an LTI app. It is an object type. The main properties of the schema are: 'id' is a string that represents the uuid of the lti app. 'title' is a string that represents the title of the lti app and is required with a minimum length of 1. 'url' is a string that represents the url of the lti app and is also required with a minimum length of 1. 'appKey' is a string or null that represents the key of the lti app. 'secret' is a string or null that represents the secret of the lti app. 'description' is a string or null that represents the description of the lti app. No additional properties are allowed at the root level.
