The schema describes a structure for requests. It is an object type. The main properties of the schema are: 'requests' is an array of objects, optional, containing a sequence of request objects. Each request object may have: 'seq' which is an integer, representing the sequence of the request; 'op' which is a string, describing one of the operations allowed: 'find', 'update', 'insert', 'delete', 'save'; and 'request' which is an object representing one of the request types. No additional properties are allowed at the root level.
