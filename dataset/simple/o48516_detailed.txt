The schema describes a server configuration. It is an object type. The main properties of the schema are: 'Cores' is a string optional status. 'DVID datastore' is a string optional status. 'Maximum Cores' is a string optional status. 'Server uptime' is a string optional status. 'Storage backend' is a string optional status. 'Storage driver' is a string optional status. 'version' is a string optional status. No additional properties are allowed at the root level.
