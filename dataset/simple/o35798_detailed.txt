The schema describes the strict format for writing a list of team members. It is an array type. The main properties of the schema are: Each item in the array is an object that must contain the following properties: 'firstName' is a string and is required. 'lastName' is a string and is required. 'identity' is a string and is required. 'role' is a string and is required. The array must contain at least one item and no additional items are allowed. No additional properties are allowed at the root level.
