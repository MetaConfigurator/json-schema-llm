The schema describes an object structure for event reporting. It is an object type. The main properties of the schema are: 'state' is a string indicating the state of the event, required and must be one of the following values: 'new', 'acknowledged', 'resolved'. 'comment' is a string that has a maximum length of 32767 characters, optional. 'data' is an object with no defined properties, optional. 'eventTags' is an object where properties must match the regex pattern '^[0-9a-zA-Z_-]{1,255}$' and each property must be a string with a minimum length of 1 and a maximum length of 255 characters. No additional properties are allowed at the root level.
