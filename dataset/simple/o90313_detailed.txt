The schema describes the origin of the style from which the symbol was originally referenced. It is an object type. The main properties of the schema are: 'name' is a string that identifies a symbol in the style by name and is required when using 'styleName' or 'styleUrl'. 'styleName' is a string that represents a registered web style name, such as 'EsriThematicShapesStyle', and is required when using 'name'. 'styleUrl' is either an absolute URL (type: string, format: uri) or a relative path starting with './' (type: string, pattern: '^\./.+$') and is required when using 'name'. It must be either 'styleName' or 'styleUrl' that is specified, but not both. No additional properties are allowed at the root level.
