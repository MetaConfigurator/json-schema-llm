The schema describes how to describe an external source. It is an object type. The main properties of the schema are: 'path' is a string that represents the full path to the source (may include port number), and it is required. 'referenceId' is a string that indicates which property should be used to name the feature, and it is required. 'refreshIntervalMSec' is an integer that specifies the interval in milliseconds to refresh the source, and it is required. 'type' is a string that specifies the subscriber to use, and it is required, with allowed values being 'REST'. No additional properties are allowed at the root level.
