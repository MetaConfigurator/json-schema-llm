The schema describes later defined parameters provided for a CBSD. It is an object type. The main properties of the schema are: 'groupingParam' is an array container for information about a CBSD membership group. It requires items that are objects with required fields 'groupType' and 'groupId'. Each 'groupType' is an enum restricted to the value 'INTERFERENCE_COORDINATION'. Each 'groupId' is a string with a maximum length of 128. 'airInterface' is an object that serves as the air interface parameter for a CBSD. It requires the field 'radioTechnology', which is an enum allowing the values 'E_UTRA' and 'CAMBIUM_NETWORKS'. No additional properties are allowed at the root level.
