The schema describes a prebootstrapped plugin manifest. It is an object type. The main properties of the schema are: 'volume' is an object that contains the property 'backing', which is a string and must be one of the following values: 'raw', 'ebs', 's3', 'vdi', 'vmdk'. This property is required. 'plugins' is an object that contains an optional property 'prebootstrapped', which is also an object that has optional properties 'snapshot' and 'image', both of which are strings. The root schema defines that 'volume' is required. No additional properties are allowed at the root level.
