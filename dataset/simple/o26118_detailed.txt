The schema describes a squash training routine containing many activities. It is an object type. The main properties of the schema are: 'id' is a string unique identifier for the routine. 'name' is a string name of the routine. 'activities' is an array list of activities in the routine, which is required to have a minimum of 1 item. Each item in 'activities' is an object with the properties 'id' and 'time'. 'id' is a string representing the id of the activity, and 'time' is a number that represents the time in seconds that the activity should run, with a minimum value of 1. Both 'id' and 'time' are required fields for each activity object. No additional properties are allowed at the root level.
