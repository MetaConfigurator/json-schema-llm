The schema describes a return object from the inventory controller. It is an object type. The main properties of the schema are: 'direction' is a string that describes the sort direction and can be either 'ASCENDING' or 'DESCENDING'. 'items' is an array of objects representing inventory objects. 'limit' is an integer that describes the maximum number of return results. 'total' is an integer that describes the number of items present in the database. No additional properties are allowed at the root level.
