The schema describes a recipient profile. It is an object type. The main properties of the schema are: 'id' is a string reserved for future use as DID. 'name' is a string, name of recipient, http://schema.org/name. 'publicKey' is a string in URI format, describing the recipient's public key in Blockcerts, typically represented with a '<scheme>:' prefix. No additional properties are allowed at the root level.
