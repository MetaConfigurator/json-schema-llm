The schema describes a system global authentication TACACS policy binding. It is an object type. The main properties of the schema are: 'builtin' is an object that defines the modification type with enum values 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'feature' is a string that defines the feature type with enum values including 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', 'LoadBalancing', and many others. 'globalbindtype' is a string that defines the global bind type with enum values 'SYSTEM_GLOBAL', 'VPN_GLOBAL', 'RNAT_GLOBAL'. 'gotopriorityexpression' is a string representing the go-to priority expression. 'nextfactor' is a string representing the next factor. 'policyname' is a string representing the policy name. 'priority' is an integer representing the priority level. No additional properties are allowed at the root level.
