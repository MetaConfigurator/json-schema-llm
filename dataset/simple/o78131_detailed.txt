The schema describes a hash of keys that should be returned, to paths to the data from each item that should be returned. It is an object type. The main properties of the schema are: 'additionalProperties' is an array that describes a path to the data that should be returned for this key, for each item. Each entry in the array represents a key that should be traversed in the results. Each item in the array is a string representing the next key to traverse. No additional properties are allowed at the root level.
