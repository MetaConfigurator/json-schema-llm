The schema describes an array of objects. It is an array type. The main properties of the schema are: 'shared_secret' is a string and is required. 'header' is a string and is required; it must match the regex pattern '^[a-zA-Z0-9]([a-zA-Z0-9-]+[a-zA-Z0-9])?$'. 'algorithm' is a string and is required; it must be one of the following values: 'hmac-md5', 'hmac-sha1', 'hmac-sha256', 'hmac-sha384', 'hmac-sha512'. No additional properties are allowed at the root level.
