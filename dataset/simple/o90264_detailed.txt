The schema describes filters for points based on the value of the return number/return count. It is an object type. The main properties of the schema are: 'field' is a string that specifies the name of the field used for the filter, required. 'includedReturns' is an array that retains all points with at least one specified return status. It allows values: 'last', 'firstOfMany', 'lastOfMany', 'single', required, with string items. 'type' is a string that indicates the filter type, specifically 'pointCloudReturnFilter', required. No additional properties are allowed at the root level.
