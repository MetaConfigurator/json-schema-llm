The schema describes a configuration for network settings. It is an object type. The main properties of the schema are: 'port' is an integer that specifies the network port, required. It must be between 0 and 65535. 'priority' is an integer representing the priority level, required, and must also be between 0 and 65535. 'target' is a string that specifies the hostname, required, and must conform to the hostname format. 'weight' is an integer that indicates the weight of the setting, required, and must be between 0 and 65535. No additional properties are allowed at the root level.
