The schema describes an API request format. It is an object type. The main properties of the schema are: 'description' is a string API call description. It is optional. 'secure' is a boolean The call must be secure and user must be logged in. It is required. 'uri' is a string The json call URI. It is required. 'version' is a string API call version. It must match the pattern ^(\d+\.)?(\d+\.)?(\*|\d+)$ and is required. No additional properties are allowed at the root level.
