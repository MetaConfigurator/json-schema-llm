The schema describes a data structure for file processing details. It is an object type. The main properties of the schema are: 'files' is an array containing objects, each with a required 'name' string property and an optional 'appliedFixers' array of unique strings. 'time' is an object with a required 'total' number property. 'memory' is a number property that is optional. No additional properties are allowed at the root level.
