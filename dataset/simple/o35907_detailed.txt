The schema describes an array of proxy configurations. It is an array type. The main properties of the schema are: 'proxyBaseUrl' is a string representing the base URL for the proxy and is required. 'headers' is an array of objects, each containing 'name' and 'value' as required string fields. 'proxyRequests' is an array of objects, each containing 'path' and 'key' as required string fields. No additional properties are allowed at the root level.
