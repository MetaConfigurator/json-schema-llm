The schema describes a Dataset ContactPoint as a vCard object. It is an object type. The main properties of the schema are: '@type' is a string that specifies the IRI for the JSON-LD data type and must be one of the values: 'vcard:Contact'. 'fn' is a string that represents a full formatted name, which is required and must have a minimum length of 1. 'hasEmail' is a string that represents the email address for the contact, which is required and can conform to either of two specified patterns. One pattern is a standard mailto format, while the other includes a redacted format indicated by a specific pattern. No additional properties are allowed at the root level.
