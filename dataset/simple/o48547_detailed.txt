The schema describes the configuration for a key derivation function. It is an object type. The main properties of the schema are: 'kdf' is a string that is required and must be equal to 'pbkdf2'. 'kdfparams' is an object that is required and defines several properties: 'c' is an integer that is required and must be greater than or equal to 1. 'dklen' is an integer that is required and must be greater than or equal to 1. 'prf' is a string that is required and must be equal to 'hmac-sha256'. 'salt' is a string that is required and must match the pattern '^(([0-9]|[a-f]){2}){0,32}$'. No additional properties are allowed at the root level.
