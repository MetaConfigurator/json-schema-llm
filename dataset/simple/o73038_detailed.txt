The schema describes the structure for detailing a prorogation. It is an object type. The main properties of the schema are: 'situacao' is an object representing the current situation with properties 'id' (a string) and 'descricao' (a string); 'novaDataFimVigencia' is a string or null that indicates the new end date of validity; 'dataInicioVigencia' is a string or null that denotes the start date of validity; 'dataFimVigencia' is a string or null that specifies the end date of validity; 'dataCadastroSolicitacao' is a string or null representing the request registration date; 'dataUltimaAlteracao' is a string or null indicating the last modification date; 'justificativa' is a string providing justification. No additional properties are allowed at the root level.
