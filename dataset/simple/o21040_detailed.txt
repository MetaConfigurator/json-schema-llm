The schema describes the details of an agreement in the SupplierFramework. It is an object type. The main properties of the schema are: 'frameworkAgreementVersion' is a string that must have a minimum length of 1 and is required. 'signerName' is a string that must have a minimum length of 1 and is required. 'signerRole' is a string that must have a minimum length of 1 and is required. 'uploaderUserId' is an integer with a minimum value of 1 and is required. No additional properties are allowed at the root level.
