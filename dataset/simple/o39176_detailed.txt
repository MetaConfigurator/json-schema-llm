The schema describes a GeoJSON FeatureCollection. It is an object type. The main properties of the schema are: 'type' is a string that is required and must be one of the enum values: 'FeatureCollection'. 'properties' is an object that is required and does not allow additional properties. It contains: 'sensor' is a string that describes the name of an existing sensor resource; 'name' is a string; 'description' is a string. No additional properties are allowed at the root level.
