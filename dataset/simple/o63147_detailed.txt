The schema describes a single annotation value within a BEL Annotation. It is an object type. The main properties of the schema are: 'annotation_value' is an object which is required. It has no additional properties and requires the fields 'type', 'identifier', and 'name'. The 'type', 'identifier', 'name', and 'match_text' fields are all strings, while 'match_text' is optional. No additional properties are allowed at the root level.
