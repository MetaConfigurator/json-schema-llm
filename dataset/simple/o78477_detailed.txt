The schema describes configuration options for Jest runner in Stryker. It is an object type. It defines several reusable components under 'definitions': 'jestProjectType' is a string, representing the type of project you are working on with allowed values: 'create-react-app', 'create-react-app-ts', 'react', 'react-ts', and 'custom'. The main properties of the schema are: 'jest' is an object that configures @stryker-mutator/jest-runner, optional and defaults to an empty object. It includes properties: 'projectType', which is a reference to 'jestProjectType', with a default value of 'custom'; 'configFile', which is a string describing the path to the Jest config file; 'config', which is an object for a custom Jest configuration; and 'enableFindRelatedTests', which is a boolean indicating whether to run tests related to the mutated file, defaulting to true. No additional properties are allowed at the root level.
