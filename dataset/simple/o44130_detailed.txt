The schema describes a numerical value with a unit of measure. It is an object type. The main properties of the schema are: 'value' is a number that represents the numeric value of the element and is required. 'unit' is a string that represents the unit of measure of the element, with allowed values drawn from the Common synonyms (non-UCUM) column of [subset of] UCUM, SI and English units, and is also required. No additional properties are allowed at the root level.
