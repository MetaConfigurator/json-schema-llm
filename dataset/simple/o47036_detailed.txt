The schema describes the IoTOSHI Sensor Event. It is an object type. The main properties of the schema are: 'cpu' is a number representing CPU utilization (%) and is required with a default value of 0.0; it must be between 0.0 and 100.0. 'memory' is a number representing Memory utilization (%) and is required with a default value of 0.0; it must be between 0.0 and 100.0. No additional properties are allowed at the root level.
