The schema describes a configuration for a search operation. It is an object type. The main properties of the schema are: 'search' is an object that is required and includes the property 'resource', which is a string required to be formatted as a topic. Additionally, 'i18n' is an optional object that may contain the property 'locale', which is a string with a default value of 'default' and must also be formatted as a topic. No additional properties are allowed at the root level.
