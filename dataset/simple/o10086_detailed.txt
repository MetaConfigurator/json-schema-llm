The schema describes a configuration object. It is an object type. The main properties of the schema are: 'component' is a string matching the pattern '^[a-z_]$' and is optional. 'group' is a string matching the pattern '^[a-z_]$' and is optional. 'name' is a string matching the pattern '^[a-zA-Z0-9_\-\./]+$' and is optional. No additional properties are allowed at the root level.
