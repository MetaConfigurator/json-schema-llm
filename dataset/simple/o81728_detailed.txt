The schema describes permissions for system users. It is an object type. The main properties of the schema are: 'Registry - read' is a boolean that indicates if registry data can be read, with a default value of true. 'Registry - write' is a boolean that indicates if registry data can be modified, with a default value of true. 'System Admin' is a boolean that indicates if the user has system administrator privileges, with a default value of true. 'System User' is a boolean that indicates if the user is a system user, with a default value of true. No additional properties are allowed at the root level.
