The schema describes owner information. It is an object type. The main properties of the schema are: 'owner' is an object containing information about the owner of this content. It is required and has the following properties: 'id' is a string that serves as the unique identifier for this owner, often used to store the organization id in the Arc ecosystem. 'name' is a string representing the name of this owner. 'sponsored' is a boolean necessary for including/excluding advertorial and native content in search results and story card features. No additional properties are allowed at the root level.
