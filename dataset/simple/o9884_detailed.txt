The schema describes a configuration object. It is an object type. The main properties of the schema are: 'version' is a string or null that may have a minimum length of 1 and a maximum length of 255 characters. 'slug' is a string that must have a minimum length of 4 and a maximum length of 63 characters, and must match the pattern '^[0-9a-z-]*$'. No additional properties are allowed at the root level.
