The schema describes a classification from a code list. It is an object type. The main properties of the schema are: 'code_scheme_id' is a string that describes the scheme of the classification. It is optional. 'code' is a string that describes the classification. It is optional. 'name' is a string that describes a title or description of the classification, especially if the value is a code. It is optional. The schema requires either 'name' to be present or both 'code' and 'code_scheme_id' to be present for a valid configuration. No additional properties are allowed at the root level.
