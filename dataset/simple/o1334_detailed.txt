The schema describes the configuration for an Alyle UI ng-add schematic. It is an object type. The main properties of the schema are: 'gestures' is a boolean indicating whether gesture support should be set up, and it has a default value of true. 'project' is a string representing the name of the project, with a default that sources from 'projectName'. 'themes' is an array of strings that lists themes to be added to the project, defaulting to ['minima-light'], and it must contain unique items. No additional properties are allowed at the root level.
