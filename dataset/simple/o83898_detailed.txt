The schema describes the configuration for creating a service worker. It is an object type. The main properties of the schema are: 'configuration' is a string that describes the configuration to apply service worker to and has a default value of 'production'. 'project' is a string that describes the name of the project and is required. 'target' is a string that describes the target to apply service worker to and has a default value of 'build'. No additional properties are allowed at the root level.
