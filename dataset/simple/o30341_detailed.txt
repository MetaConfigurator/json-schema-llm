The schema describes a binding for a content rewriting policy to a virtual server. It is an object type. The main properties of the schema are: 'domain' is a string which is optional. 'gotopriorityexpression' is a string which is readonly. 'hits' is an integer which is readonly. 'labelname' is a string which is readonly. 'labeltype' is a string which is readonly and has an enum of allowed values: 'reqvserver', 'resvserver', 'policylabel'. 'pihits' is an integer which is readonly. 'pipolicyhits' is an integer which is readonly. 'policyname' is a string which is optional. 'priority' is an integer which is readonly. No additional properties are allowed at the root level.
