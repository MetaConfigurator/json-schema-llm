The schema describes a collection of objects. It is an array type. The main properties of the schema are: Each object has an 'id' which is a string and is required. Each object has a 'title' which is a string and is required. Each object has a 'description' which is a string and is required. Each object has a 'journal' which is a string and is required. Each object has a 'question' which is a number and is required. Each object may contain an optional 'races' which is an array of strings.
