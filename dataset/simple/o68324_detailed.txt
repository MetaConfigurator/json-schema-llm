The schema describes a structure validating parameters accepted by the Openx adapter. It is an object type. The main properties of the schema are: 'customFloor' is a number representing the minimum CPM price in USD, optional and must be greater than or equal to 0. 'customParams' is an object representing user-defined targeting key-value pairs, optional. 'delDomain' is a string for the delivery domain for the customer, required, must match the pattern '\.[a-zA-Z]{2,3}$'. 'unit' is a string for the ad unit id, required, must match the pattern '^[0-9]+$'. No additional properties are allowed at the root level.
