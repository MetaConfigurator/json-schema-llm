The schema describes details of a candidate's ability to apply for a given job. It is an object type. The main properties of the schema are: 'allowed' is a boolean indicating true if the candidate is allowed to apply for this job, otherwise false, and is required; 'notAllowedMessage' is a string detailing why the candidate is not allowed to apply, for example, 'You've already applied', and is required. No additional properties are allowed at the root level.
