The schema describes an address. It is an object type. The main properties of the schema are: 'country_name' is a string and is required. 'extended_address' is a string and is optional. 'locality' is a string and is required. 'post_office_box' is a string and is optional. 'postal_code' is a string and is optional. 'region' is a string and is required. 'street_address' is a string and is optional. No additional properties are allowed at the root level.
