The schema describes OpenQuantum job status. It is an object type. The main properties of the schema are: 'job_id' is a string representing the Backend job_id and is required. 'status' is a string indicating the Status of the job and is required, with allowed values being 'DONE', 'QUEUED', 'CANCELLED', 'RUNNING', 'ERROR'. 'status_msg' is a string providing the Status message and is required. No additional properties are allowed at the root level.
