The schema describes a sandbox structure. It is an object type. The main properties of the schema are: 'guid' is a string that represents the unique identifier of the sandbox. It is optional. 'name' is a string that also represents the unique identifier of the sandbox. It is optional. 'checksums' is an array of strings describing uploaded checksums. Each string in the array must be exactly 32 characters long, consist of lowercase hexadecimal characters, and is optional. 'create_time' is a string that describes the timestamp at which the sandbox was created, formatted as an ISO 8601 string or the string 'infinity'. It is optional. 'is_complete' is a boolean indicating whether all URLs in this sandbox were successfully uploaded. It is optional. No additional properties are allowed at the root level.
