The schema describes a range of numeric values. It is an object type. The main properties of the schema are: 'minimum' is a number that describes the minimum value. 'maximum' is a number that describes the maximum value. 'exclusive_minimum' is a boolean that describes whether the minimum value is not included in the range, i.e. x > minimum (not x >= minimum). 'exclusive_maximum' is a boolean that describes whether the maximum value is not included in the range, i.e. x < maximum (not x <= maximum). No additional properties are allowed at the root level.
