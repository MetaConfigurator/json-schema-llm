The schema describes the presentation of an image for a step. It is an object type. The root schema resolves to the referenced definition for 'image', which is an object, consisting of data about an image. The 'image' definition has a required field 'filename', which is a string that specifies the filename of the image in the local filesystem. The 'image' definition also contains an optional field 'alt', which is a string for alternative text for the image. No additional properties are allowed at the root level.
