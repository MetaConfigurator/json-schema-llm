The schema describes an object structure for email, relative reference, and template context. It is an object type. The main properties of the schema are: 'email' is a string that must follow the email format and has a maximum length of 1024 characters. 'relativeTo' is a oneOf type allowing either a string, a number, or an object with a required property '$date' of type string and no additional properties allowed. 'templateContext' is a string with a maximum length of 32767 characters. No additional properties are allowed at the root level.
