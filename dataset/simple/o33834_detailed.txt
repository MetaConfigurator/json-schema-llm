The schema describes nodes in a Guice injector graph. It is an object type. The main properties of the schema are: 'id' is a string required property. 'type' is a string, required property with allowed values: 'implementation', 'interface'. 'members' is an array of objects, where each object contains a 'name' property of type string. 'className' is a string optional property. 'annotationName' is a string optional property. 'instanceName' is a string optional property. 'sourceName' is a string optional property. No additional properties are allowed at the root level.
