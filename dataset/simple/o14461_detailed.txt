The schema describes an action target. It is an object type. The main properties of the schema are: 'name' is a string that serves as a unique identifier for the action target within the action target template and organization. It is required. 'actionTargetTemplateId' is an integer that specifies the action target template to instantiate. It is required. 'organizationId' is an integer that refers to the organization. It is optional. 'public' is a boolean that indicates if the action target is available to everyone. It is optional. 'description' is a string providing a human-readable description of the action target. It is optional. 'configuration' is an object that represents a valid JSON object. It is required. No additional properties are allowed at the root level.
