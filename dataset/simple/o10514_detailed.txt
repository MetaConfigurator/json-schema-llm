The schema describes an address structure. It is an object type. The main properties of the schema are: 'Address' is an object which is required. It has the following required properties: 'Recipient' is a string which denotes the recipient's name. 'House' is a string representing the house number. 'Street' is a string indicating the street name. 'Town' is a string for the town name. 'PostCode' is a string for the postal code. Additionally, 'Country' is a string which can only be one of the following values: 'IN', 'DE', 'ES', 'UK', 'US'. 'County' is a string for the county name. No additional properties are allowed at the root level.
