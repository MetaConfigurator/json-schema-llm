The schema describes an instance of a camera gaze ring geometry and material configuration. It is an object type. The main properties of the schema are: 'color' is a oneOf type, representing the hexadecimal color code of the ring, which can be either a string or a number. 'innerRadius' is a number, representing the inner radius of the ring, with a minimum value of 0 and a default of 0.02. 'opacity' is a number, representing the opacity of the ring, with a minimum value of 0 and a maximum value of 1, and an example of 0.5. 'outerRadius' is a number, representing the outer radius of the ring, with a minimum value of 0 and a default of 0.04. No additional properties are allowed at the root level.
