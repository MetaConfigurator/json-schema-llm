The schema describes a transcription of a piece of data in a language, optionally in multiple orthographies. It is an object type. The main properties of the schema are: Each key must be an abbreviation for an orthography, matching the pattern ^[(a-z)|(A-Z)|(0-9)]+$, and the value for that key is a string representing the transcription for the specified orthography. No additional properties are allowed at the root level.
