The schema describes a MWF flipper component's schema definition. It is an object type. The main properties of the schema are: 'config' is an object that specifies the configuration settings for the flipper. It has required fields 'direction', which is a string with allowed values of 'left' and 'right', and 'hide-from-screen-readers', which is a boolean. 'content' is an object that defines the content of the flipper. It has required fields 'label' and 'title', both of which are strings. No additional properties are allowed at the root level.
