The schema describes shampoo response specification. It is an object type. The main properties of the schema are: 'message' is a string that provides a human readable status message. 'request_id' is an integer representing the request_id of the request this is a response to, with a minimum value of 0. 'response_data' is an object containing the response data from the called method. 'status' is an integer representing the status code, using the same codes as HTTP, with a minimum value of 100. 'type' is a string that must match the pattern 'response'. All properties are required. No additional properties are allowed at the root level.
