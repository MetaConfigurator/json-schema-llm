The schema describes response data for SAS communications. It is an object type. The main properties of the schema are: 'responseCode' is an integer indicating the type of the response. It is required and can be one of the following values: 0, 100, 101, 102, 103, 104, 105, 200, 201, 202, 300, 400, 401, 500, 501, 502. 'responseMessage' is a string providing a detailed human-readable string describing the response. It is optional. 'responseData' is an object providing additional response-dependent structured data containing more information about an error. It is optional and allows additional properties. No additional properties are allowed at the root level.
