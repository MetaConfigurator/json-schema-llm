The schema describes the structure of a Net configuration. It is an object type. The main properties of the schema are: 'vars' is an object that contains a dictionary where its keys are names of variables and its values are multi-dimensional arrays. It is required. 'params' is an array that holds the names of 'vars', which are parameters. The items in this array are of type string. It is required. 'blocks' is an object that contains a dictionary where its keys are output vars and the values are objects with 'inputs' as a string array and 'function' as a string. It is required. No additional properties are allowed at the root level.
