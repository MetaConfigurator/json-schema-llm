The schema describes Git LFS HTTPS Lock Creation API Response. It is an object type. The main properties of the schema are: 'lock' is an object representing the lock details. It is a required field. It contains the following required properties: 'id' is a string, 'path' is a string, and 'locked_at' is a string. 'message' is an optional string, 'request_id' is an optional string, and 'documentation_url' is an optional string. No additional properties are allowed at the root level.
