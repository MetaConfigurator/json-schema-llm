The schema describes the data contained in a BarChart InPlace update object. It is an object type. The main properties of the schema are: 'inplace' is an array of objects, each containing required properties. Each object in the array has a required 'position' property of type object, which has required integer fields 'x' (minimum 0) and 'y' (minimum 0). It also has a required 'data' property of type number. No additional properties are allowed at the root level.
