The schema describes granting permissions for a user. It is an object type. The main properties of the schema are: 'username' is a string that represents the name of the user to grant the accessLevel to. It is required. 'accessLevel' is a string that specifies the type of permission to grant access to. It is required and its allowed values are 'NONE', 'READ', 'WRITE', and 'ADMIN'. No additional properties are allowed at the root level.
