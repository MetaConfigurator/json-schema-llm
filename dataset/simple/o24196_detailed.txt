The schema describes payload for receiving a chunk of raw records. It is an object type. The main properties of the schema are: 'last' is a boolean indicates if the current chunk is the last one associated with the same JobExecution (required). 'total' is an integer total number of chunks associated with the same JobExecution (required). 'records' is an array list of raw records (required). Each item in the records array is a string. No additional properties are allowed at the root level.
