The schema describes a project configuration. It is an object type. The main properties of the schema are: 'name' is a string that describes the name of the project. 'short_name' is a string that describes the short name of the project. 'description' is a string that provides a short description of the project. 'category_id' is an integer that represents the category ID of the project. 'webhook' is a string that indicates a webhook URL for the project. No additional properties are allowed at the root level.
