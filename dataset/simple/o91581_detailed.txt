The schema describes schemas for detecting valid ID formats, specifically strings of format URI or arrays containing one item of a string, format URI. It is an object type. It defines several reusable components under 'definitions': 'arraySingleStringUri' is an array or a reference to 'stringUri', allowing either a single string of URI format or an array containing one string of URI format. 'idValueFound' is an object type with a required property 'id' that references 'arraySingleStringUri'. 'stringUri' is a string type that must conform to the URI format. The main properties of the schema are: 'arraySingleStringUri' is a reference to the definition 'arraySingleStringUri', which is required; 'idValueFound' is a reference to the definition 'idValueFound', which is required; 'stringUri' is a reference to the definition 'stringUri', which is required. No additional properties are allowed at the root level.
