The schema describes flight information. It is an object type. The main properties of the schema are: 'price' is an integer, representing the flight price; 'airlineName' is a string, representing the airline name; 'departureDate' is a string, representing the flight departure date; 'planeType' is a string, representing the plane type; 'origination' is a string, representing the three letter airport code for the airport of origination; 'flightCode' is a string, representing the flight code; 'emptySeats' is an integer, representing the number of available seats; 'destination' is a string, representing the three letter airport code for the airport of destination. No additional properties are allowed at the root level.
