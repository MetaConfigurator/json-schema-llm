The schema describes a data structure for managing paginated objects. It is an object type. The main properties of the schema are: 'meta' is an object that contains pagination information. It includes required properties: 'limit' which is an integer, 'next' which is a string, 'offset' which is an integer, 'previous' which can be either an integer or null, and 'total_count' which is an integer. 'objects' is an array that holds the actual data items. No additional properties are allowed at the root level.
