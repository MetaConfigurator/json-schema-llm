The schema describes an extended property in the form of a name-value pair. It is an object type. The main properties of the schema are: 'description' is a string that provides the description of the property. 'id' is a string that serves as a unique identifier for the property. 'name' is a string that represents the name of the property (e.g., Ethernet Speed). 'value' is a string that indicates the value of the property (e.g., 1 Gbit/s). No additional properties are allowed at the root level.
