The schema describes a transparency stop object. It is an object type. The main properties of the schema are: 'label' is a string used to label the stop in the legend. It is optional. 'transparency' is an integer representing a numeric transparency value for a stop ranging from 0-100, where 0 is opaque and 100 is 100% transparent. It is required and must be at least 0 and at most 100. 'value' is a number that describes the pixel intensity value that the color should be associated with. It is required. No additional properties are allowed at the root level.
