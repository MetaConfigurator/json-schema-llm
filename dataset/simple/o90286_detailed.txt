The schema describes a range domain specifying a range of valid values for a field. It is an object type. The main properties of the schema are: 'name' is a string representing the domain name and is optional. 'range' is an array that must contain exactly two unique numbers, where the first element is the minValue and the second element is the maxValue, and it is required. 'type' is a string that must be equal to 'range' and is required. No additional properties are allowed at the root level.
