The schema describes a configuration for output handling. It is an object type. The main properties of the schema are: 'append' is a boolean indicating whether to append to the output or whether to overwrite it. It is optional and has a default value of false. 'minSeverity' is a string that specifies the minimum severity level, with allowed values being 'NON_BREAKING', 'POTENTIALLY_BREAKING', and 'BREAKING'. It is optional and has a default value of 'POTENTIALLY_BREAKING'. 'output' is a string that indicates the output destination. It is optional and has a default value of 'out'. 'template' is a string for a template configuration. It is optional. No additional properties are allowed at the root level.
