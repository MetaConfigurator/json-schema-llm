The schema describes a ParaDrop chute monitor-mode interface. It is an object type. The main properties of the schema are: 'type' is a string that describes the network type, must be 'wifi' for monitor mode and is required. 'intfName' is a string that describes the name of the network interface inside the chute, must match the pattern '[a-z][a-z0-9]*' and is required. 'mode' is a string that describes the wireless interface mode, must be 'monitor' and is required. No additional properties are allowed at the root level.
