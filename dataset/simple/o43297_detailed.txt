The schema describes a Git LFS HTTPS Batch API request. It is an object type. The main properties of the schema are: 'transfers' is an array of strings, optional. 'operation' is a string, required. 'objects' is an array of objects, required. Each object in 'objects' must have the properties: 'oid', which is a string, and 'size', which is a number with a minimum value of 0. The property 'authenticated' is a boolean, optional. Each object does not allow additional properties. No additional properties are allowed at the root level.
