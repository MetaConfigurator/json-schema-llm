The schema describes a filter for HTML injection variables. It is an object type. The main properties of the schema are: 'builtin' is an object with a readonly status and can only take one of the enum values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'feature' is a string with a readonly status that must be one of the enum values including 'WL', 'WebLogging', 'SP', 'SurgeProtection', and many others. 'type' is a string with a readonly status, constrained to the enum values: 'System internal', 'User defined'. 'value' is an optional string. 'variable' is an optional string. No additional properties are allowed at the root level.
