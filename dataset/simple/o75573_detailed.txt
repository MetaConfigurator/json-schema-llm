The schema describes webhooks that allow you to subscribe to certain events. It is an object type. The main properties of the schema are: 'id' is a string that represents the unique identifier for a webhook and will be generated by shipcloud, required. 'url' is a string, required. 'event_types' is an array of strings that must contain one or more of the following allowed values: '*', 'shipment.*', 'shipment.tracking.*', 'shipment.tracking.label_created', 'shipment.tracking.picked_up', 'shipment.tracking.transit', 'shipment.tracking.out_for_delivery', 'shipment.tracking.delivered', 'shipment.tracking.awaits_pickup_by_receiver', 'shipment.tracking.canceled', 'shipment.tracking.delayed', 'shipment.tracking.exception', 'shipment.tracking.not_delivered', 'shipment.tracking.destroyed', 'shipment.tracking.notification', 'shipment.tracking.unknown', required. No additional properties are allowed at the root level.
