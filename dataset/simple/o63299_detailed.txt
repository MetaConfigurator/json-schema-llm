The schema describes coordinates of genes and genome. It is an object type. The main properties of the schema are: 'patternProperties' defines a set of properties where each key matches the regex '^[a-zA-Z0-9*_-]+$' and each value is an object. Each object contains the following properties: 'seqid' is a string that describes the sequence on which the coordinates are valid, and could be a viral segment, bacterial contig, etc. 'type' is a string that describes the type of the feature, which could be mRNA, CDS, or similar. 'start' is a number that describes the gene start position, one-based, following GFF format. 'end' is a number that describes the gene end position, one-based closed, following GFF format. 'strand' is a string that denotes the positive or negative strand and can be either '-' or '+'. No additional properties are allowed at the root level.
