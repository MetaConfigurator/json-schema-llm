The schema describes a tag as a global keyword that could be set for some entries. It is an object type. The main properties of the schema are: 'active' is a boolean optional property. 'color' is a string optional property that describes the color that marks this tag. 'comment' is a string optional property that describes a comment related to the tag. 'created' is a string optional property with a datetime format. 'group' is a string optional property that provides a short text description of the tag. 'guid' is a string required property with a uuid format. 'internal' is a boolean optional property. 'text' is a string required property that offers a short text description of the tag. No additional properties are allowed at the root level.
