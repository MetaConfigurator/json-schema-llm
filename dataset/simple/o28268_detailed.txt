The schema describes a time representation. It is an object type. The main properties of the schema are: 'hour' is an integer representing the hour, which must be greater than or equal to 0. 'minute' is an integer representing the minute, which must be between 0 and 59 inclusive. 'second' is an integer representing the second, which must be between 0 and 59 inclusive. No additional properties are allowed at the root level.
