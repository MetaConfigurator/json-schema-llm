The schema describes input data for builders. It is an object type. The main properties of the schema are: 'currentDirectory' is a string that is required. 'id' is a number that is required. 'info' is an object that is required. 'options' is an object that is optional. 'target' is an object that is optional but must have the properties 'project' (a required string) and 'target' (a required string). 'workspaceRoot' is a string that is required. No additional properties are allowed at the root level.
