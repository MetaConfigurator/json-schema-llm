The schema describes configuration for a JGroups channel. It is an object type. The main properties of the schema are: 'name' is a string optional. 'stack' is an array containing objects, which are required. Each object in the 'stack' array must have a 'protocol' property, which is a string and contains a description stating that it is the name of the protocol with the same resolution rules as in the standard JGroups configuration. No additional properties are allowed at the root level.
