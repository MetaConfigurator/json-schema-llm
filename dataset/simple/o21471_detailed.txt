The schema describes the queens award for enterprise applications by stage. It is an object type. The main properties of the schema are: '_id' is a string unique url-friendly base64 encoded, utf8 encoded concatenation identifier, required. '_timestamp' is a string ISO8601 combined date and time in UTC e.g. '2014-12-01T00:00:00+00:00', required. 'award' is a string queens award applied for, required, allowed values: 'international-trade', 'innovation', 'sustainable-development', 'qaep'. 'count' is an integer The number of applications for an award at a specific stage of completeness, required, minimum value of 0. 'period' is a string The period covered by each data point from the _timestamp, required, allowed values: 'hour', 'day', 'week', 'month', 'quarter'. 'stage' is a string the applications stage of completion, required, allowed values: '0-percent', '1-24-percent', '25-49-percent', '50-74-percent', '75-99-percent', '100-percent'. No additional properties are allowed at the root level.
