The schema describes an image commands plugin manifest. It is an object type. The main properties of the schema are: 'plugins' is an object that contains a required property 'image_commands', which is also an object. 'image_commands' has a required property 'commands', which is an array that must have at least one item. Each item in the 'commands' array is itself an array containing strings, and must also contain at least one string. No additional properties are allowed at the root level.
