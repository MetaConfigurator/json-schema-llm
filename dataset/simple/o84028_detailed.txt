The schema describes an array of objects representing game maps with scores. It is an array type. The main properties of the schema are: 'map' is a string representing the map name, optional. 'team1Score' is a number representing the score for Team 1, optional. 'team2Score' is a number representing the score for Team 2, optional. 'current' is a boolean indicating whether the current map being played is this one, optional. 'show' is a boolean indicating whether to show this map on the upper maps text, optional. No additional properties are allowed at the root level.
