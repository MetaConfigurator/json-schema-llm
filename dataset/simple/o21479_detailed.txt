The schema describes a Google Analytics Trending Collector Query. It is an object type. The main properties of the schema are: 'dimensions' is an array of strings, optional. 'filters' is an array of strings, optional. 'id' is a string that must match the pattern '^[a-z0-9:]+$', optional. 'metric' is a string that must match the pattern '^[A-Za-z:]+$', required. No additional properties are allowed at the root level.
