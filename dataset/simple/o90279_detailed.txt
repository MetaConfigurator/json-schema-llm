The schema describes configuration for text in popup elements. It is an object type. The main properties of the schema are: 'text' is a string that indicates the text to be displayed within the popup element; it is optional and if not provided, the popup element will display whatever is specified in the [popupInfo.description](popupInfo.md) property. 'type' is a string that must have the value 'text'; it is required. No additional properties are allowed at the root level.
