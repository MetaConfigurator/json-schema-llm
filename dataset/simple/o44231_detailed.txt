The schema describes an animation track. It is an object type. The main properties of the schema are: 'default' is a string that specifies the default easing to apply; it is optional and defaults to 'linear'. Allowed values are: 'linear', 'easeInSine', 'easeInOutSine', 'easeOutSine', 'easeInQuad', 'easeInOutQuad', 'easeOutQuad', 'easeInCubic', 'easeInOutCubic', 'easeOutCubic', 'easeOutBounce', 'easeInBounce'. 'start' is a number that indicates the time (in ms) to reach the first position of the track from the current position of the camera; it is optional and defaults to 0. No additional properties are allowed at the root level.
