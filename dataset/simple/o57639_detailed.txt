The schema describes O3D3xx spatial bilateral filter configuration. It is an object type. The main properties of the schema are: 'MaskSize' is an integer that specifies the size of the mask, required, with a minimum value of 0, a maximum value of 1, and a default value of 0. 'SigmaPixel' is a number that specifies the pixel sigma value, optional, with a minimum value of 0 and a default value of 2. 'SigmaDistance' is a number that specifies the distance sigma value, optional, with a minimum value of 0 and a default value of 0.1. No additional properties are allowed at the root level.
