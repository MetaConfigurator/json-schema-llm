The schema describes configuration profile properties for managed defaults. It is an object type. The main properties of the schema are: 'DefaultsData' is an object that represents the dictionary structure that will be set as the preferences content under the mobile user. 'DefaultsDomainName' is a string that indicates the app bundle ID reverse domain for default settings. 'PayloadIdentifier' is a string with a default value of 'com.apple.defaults', representing a unique identifier for the payload, dot-delimited. 'PayloadType' is a string with a default value of 'com.apple.defaults.managed', representing the type of the payload, a reverse dns string. 'PayloadUUID' is a string with a default value of '' and must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$', representing a unique identifier for the payload. 'PayloadVersion' is a number with a default value of 1, indicating the version of the whole configuration profile. No additional properties are allowed at the root level.
