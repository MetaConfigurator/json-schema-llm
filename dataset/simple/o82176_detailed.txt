The schema describes a contact representation. It is an object type. It defines several reusable components under 'definitions': 'PostalAddress' is an object, which includes properties 'street', 'locality', 'region', 'postalCode', 'postOfficeBox', and 'country', all of which are strings. The main properties of the schema are: 'PostalAddress' is a reference to the 'PostalAddress' definition, and it is required. No additional properties are allowed at the root level.
