The schema describes an attribute card within the game. It is an object type. The main properties of the schema are: '_id' is an integer that represents the unique identifier for the attribute and is required. 'name' is a string that represents the attribute's name and is required. 'abbreviation' is a string that represents the attribute's 3 letter abbreviation, must have a length of exactly 3 characters, and is required. 'type' is a string that indicates the type of attribute with allowed values of 'Goalkeeper', 'Defence', 'Attack', 'Generic' and is required. 'description' is a string that represents the description of the attribute and is required. No additional properties are allowed at the root level.
