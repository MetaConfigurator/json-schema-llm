The schema describes metadata extracted from audio files. It is an object type. The main properties of the schema are: 'title' is a string optional. 'author' is a string optional. 'xmpDM:duration' is a string optional. 'xmpDM:audioSampleRate' is a string optional. 'xmpDM:audioSampleType' is a string optional. 'xmpDM:audioChannelType' is a string optional. 'xmpDM:audioCompressor' is a string optional. 'version' is a string optional. No additional properties are allowed at the root level.
