The schema describes a list of objects. It is an array type. The main properties of the schema are: Each item in the array is an object with the following properties: 'id' is a number and is required. 'name' is a string and is required. 'icon' is a string and is required. 'attributes' is an object and is required, containing the properties: 'Toughness' which is a number, 'Power' which is a number, and 'CritDamage' which is a number, all of which are optional. No additional properties are allowed at the root level.
