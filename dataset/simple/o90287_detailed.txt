The schema describes a Range object. It is an object type. The main properties of the schema are: 'interactionMode' is a string indicating the mode in which the active range should be presented to the user. It is required and can only be one of the following values: 'picker', 'slider'. 'numberOfStops' is a number used to generate the activeRangeValues if activeRangeValues are not specified. It is optional. 'stopInterval' is a number indicating the interval in which stops should be generated. It is optional. The schema requires either 'interactionMode' and 'numberOfStops' or 'interactionMode' and 'stopInterval' to be present, satisfying one of the conditions defined in the 'oneOf' constraint. No additional properties are allowed at the root level.
