The schema describes AWS EBS storage options. It is an object type. The main properties of the schema are: 'device_name' is a string that describes the name of the device on the node (sde, sdg, etc), required. 'volume_type' is a string that describes the storage type on EBS (gp2, etc), required. 'volume_size' is a number that describes the size of the storage, in gigabytes, required. 'delete_on_termination' is a boolean that describes whether or not to delete the volume after the node is terminated, required. 'snapshot_id' is a string or null that describes the id of a snapshot to recover from, required. 'encrypted' is a boolean that describes whether or not the volume is encrypted within AWS, required. No additional properties are allowed at the root level.
