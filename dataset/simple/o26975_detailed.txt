The schema describes a keyframe segment. It is an object type. The main properties of the schema are: 's' is a number representing the start value of the keyframe segment, which is optional. 't' is a number representing the start time of the keyframe segment, which is optional. 'i' is an object representing the Bezier curve interpolation in value, which is optional and contains properties: 'x' is a number representing the bezier x axis, which is optional, and 'y' is a number representing the bezier y axis, which is optional. No additional properties are allowed at the root level.
