The schema describes the asset.json file for mesh settings objects. It is an object type. The main properties of the schema are: 'asset_type' is a string matching the pattern 'mesh_settings', and it is required. 'mesh' is a string describing a path to a mesh asset, and it is required. 'material' is a string describing a path to a material asset, and it is required. No additional properties are allowed at the root level.
