The schema describes a video optimization pacing action. It is an object type. The main properties of the schema are: 'builtin' is an object that is readonly. It has an enum with allowed values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'comment' is an optional string. 'feature' is a string that is readonly. It has an enum with allowed values including 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', and many others up to 'Bot'. 'hits' is an integer that is readonly. 'name' is an optional string. 'newname' is an optional string. 'rate' is an optional integer. 'referencecount' is an integer that is readonly. 'undefhits' is an integer that is readonly. No additional properties are allowed at the root level.
