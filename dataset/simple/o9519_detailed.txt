The schema describes an applicant applying to a program at an organization. It is an object type. The main properties of the schema are: 'href' is a string Hypertext reference to this resource. 'programs' is an array of objects representing programs, each containing required fields: 'id' which is an integer and serves as the unique identifier of this program, 'name' which is a string describing the human-readable name of this program, and 'organization_name' which can be a string or null representing the human-readable name of the organization whose program this is. No additional properties are allowed at the root level.
