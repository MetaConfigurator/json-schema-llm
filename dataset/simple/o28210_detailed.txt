The schema describes an assignment made by the Seller of a SVLAN ID to a UNI or ENNI endpoint. It is an object type. The main properties of the schema are: 'objectType' is a string that describes the type of the object, svlanIdAssignment. It is required and must be one of the enum values: 'svlanIdAssignment'. 'sellerEnniId' is a string that describes the id assigned by the Seller for the ENNI for this SVLAN ID assignment. It has a maximum length of 45 and is optional. 'sellerUniId' is a string that describes the id assigned by the Seller for the UNI for this SVLAN ID assignment. It has a maximum length of 45 and is optional. 'svlanId' is an integer that describes the SVLAN ID assigned to this UNI or ENNI. It is required and must be greater than or equal to 0. No additional properties are allowed at the root level.
