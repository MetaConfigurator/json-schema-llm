The schema describes the licensing as asserted in a package metadata. It is an object type. The main properties of the schema are: 'license' is a string that is defined using oneOf, allowing either a license as asserted (a text, a name, or anything) or a null value. 'notice' is a string that is defined using oneOf, allowing either a license notice for this package or a null value. 'text' is a string that is defined using oneOf, allowing either license text as asserted or a null value. 'url' is a string that is defined using oneOf, allowing either a URL to a web page for this license or a null value. No additional properties are allowed at the root level.
