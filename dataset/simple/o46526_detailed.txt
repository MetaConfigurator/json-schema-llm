The schema describes Project objects. It is an object type. The main properties of the schema are: 'name' is a string that describes the Name of the project. 'author' is a string that describes the Author of the project. 'description' is a string that describes the Description of the project. 'createDate' is an array that describes Date the project was created. 'latestVersion' is an integer that describes the latest version of the file. No additional properties are allowed at the root level.
