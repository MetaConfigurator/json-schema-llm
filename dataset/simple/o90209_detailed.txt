The schema describes a font used for text symbols. It is an object type. The main properties of the schema are: 'decoration' is a string that represents the text decoration, required to be one of the values: 'line-through', 'none', or 'underline', with a default value of 'none'. 'family' is a string that represents the font family, which is optional. 'size' is a number that represents the font size in points, which is also optional and ignored when the font is used on TextSymbol3DLayer. 'style' is a string that represents the text style, required to be one of the values: 'italic', 'normal', or 'oblique', with a default value of 'normal'. 'weight' is a string that represents the text weight, required to be one of the values: 'bold', 'bolder', 'lighter', or 'normal', with a default value of 'normal'. No additional properties are allowed at the root level.
