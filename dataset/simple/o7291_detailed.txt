The schema describes the mapping of optional features in an object type. The main properties of the schema are: 'dead_features' is an array that maps declared optional features that are dead with the time points in which they are dead in case of evolution, empty list otherwise. It is required. 'false_optionals' is an array that maps declared optional features that are false optionals with the time points in which they must be selected in case of evolution, empty list otherwise. It is required. No additional properties are allowed at the root level.
