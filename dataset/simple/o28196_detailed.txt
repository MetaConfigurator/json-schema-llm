The schema describes relationships between the current document and an external resource. It is an object type. The main properties of the schema are: 'href' is a string. Specifies the location of the linked document. It is required and has a maximum length of 255. 'method' is a string. Specifies how to send the data (e.g. get, post). It is optional and has an enum of allowed values: 'get', 'post'. 'objectType' is a string. The type of the object, link. It is required and has a maximum length of 25. 'rel' is a string. Specifies the relationship between the current document and the linked document. It is required and has a maximum length of 25. 'title' is a string. Provides additional/advisory information (expand on the meaning of the link). It is optional and has a maximum length of 25. 'type' is a string. Specifies the media type of the linked document. It is optional and has a maximum length of 25. No additional properties are allowed at the root level.
