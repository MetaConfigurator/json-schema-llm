The schema describes a configuration for a button. It is an object type. The main properties of the schema are: 'type' is a string required field that must be one of the allowed values: 'virtualButton'. 'outputIds' is an array of arrays where each inner array contains strings as items. 'meta' is an object that is optional. 'config' is an object that may not contain additional properties. No additional properties are allowed at the root level.
