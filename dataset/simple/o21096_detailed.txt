The schema describes a Lot. It is an object type. The main properties of the schema are: 'data' is an object containing 'unitPlural' and 'unitSingular' as optional string properties. 'id' is an integer and required. 'name' is a string and required. 'one_service_limit' is a boolean and optional. 'slug' is a string and required. No additional properties are allowed at the root level.
