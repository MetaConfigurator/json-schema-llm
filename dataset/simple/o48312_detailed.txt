The schema describes properties associated with a virtual machine. It is an object type. The main properties of the schema are: 'additionalProperties' is an object that contains properties related to the virtual machine's cost and resources. It is required and must contain the fields 'cost' and 'resources'. The 'cost' property is an integer. The 'resources' property is an object that can have additional properties where each property represents a resource (as defined in ABS standard library) and its provided amount, which is an integer. No additional properties are allowed at the root level.
