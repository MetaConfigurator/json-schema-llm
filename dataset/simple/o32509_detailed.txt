The schema describes a message structure. It is an object type. The main properties of the schema are: 'id' is a string that describes the uuid of the message. It is optional. 'title' is a string that describes the title of the message. It is required. 'content' is a string that describes the content of the message. It is required. 'workspace' is an object that describes the workspace of the message. It is optional. It must contain a required property 'uuid', which is a string that describes the uuid of the workspace. No additional properties are allowed at the root level.
