The schema describes a volume type creation request. It is an object type. The main properties of the schema are: 'volume_type' is an object that represents a partial representation of a volume type used in the creation process. It is required. It contains properties: 'name' which is a string that indicates the name of the volume type, and 'extra_specs' which is an object that includes a property 'capabilities' that is also a string. No additional properties are allowed at the root level.
