The schema describes CDN provider settings. It is an object type. The main properties of the schema are: 'cdn' is a string that represents the name or URL template of the JavaScript and/or stylesheet CDN provider and is optional with a default value of 'jsdelivr' and nullable. 'fontcdn' is a string that represents the name or URL template of the webfont CDN provider and is optional with a default value of 'google' and nullable. 'iconcdn' is a string that represents the name or URL of the fontawesome icon font CDN provider and is optional with a default value of 'fontawesome' and nullable. No additional properties are allowed at the root level.
