The schema describes a user profile. It is an object type. The main properties of the schema are: 'about' is a string that is required. 'company' is a string that is required. 'email' is a string that is required. 'firstName' is a string that is required. 'lastName' is a string that is required. 'tags' is an array of strings that is required. It does not allow additional properties at the root level.
