The schema describes a configuration object for logging messages. It is an object type. The main properties of the schema are: 'level' is a string representing the severity level of the message, required, and must be one of the following values: 'info', 'warning', 'error', 'critical'. 'subjectTemplate' is a string that specifies the subject of the message, required, and must have a minimum length of 1. 'messageTemplate' is a string that defines the content of the message, required, and must have a minimum length of 1. No additional properties are allowed at the root level.
