The schema describes Jupyter Interactive Widget View JSON schema. It is an object type. The main properties of the schema are: 'version_major' is a number representing the format version (major), required, with a minimum value of 1 and a maximum value of 1. 'version_minor' is a number representing the format version (minor), optional. 'model_id' is a string representing the unique identifier of the widget model to be displayed, required. No additional properties are allowed at the root level.
