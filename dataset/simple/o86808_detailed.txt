The schema describes owner information. It is an object type. The main properties of the schema are: 'id' is a string that represents the machine-readable unique identifier of the organization whose database this content is stored in. It is optional. 'name' is a string that denotes the human-readable name of the original producer of the content. It is optional. 'sponsored' is a boolean indicating the necessity for including/excluding advertorial and native content in search results. It is optional. No additional properties are allowed at the root level.
