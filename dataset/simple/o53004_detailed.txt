The schema describes an object containing a new RPC Job. It is an object type. The main properties of the schema are: 'query' is an object a MongoDB query. For more information on the syntax: https://docs.mongodb.com/manual/reference/method/db.collection.find/. It is required. 'instruction' is an object an instruction representing a Mercury RPC method call. It is required and has required field 'method' which is a string the RPC method we are targeting. Available methods can be enumerated using /active/computers/{mercury_id}. It also has optional properties 'args', which is an array positional arguments being passed to the method, and 'kwargs', which is an object keyword arguments being passed to the method. No additional properties are allowed at the root level.
