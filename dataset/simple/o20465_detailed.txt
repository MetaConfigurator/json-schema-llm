The schema describes an update client request. It is an object type. The main properties of the schema are: 'client_name' is a string which is required. 'client_uri' is a string which is required. 'logo_uri' is a string which is required. 'policy_uri' is a string which is required. 'client_id' is a string which is optional. 'tos_uri' is a string which is optional. 'contacts' is an array which is optional. 'redirect_uris' is an array which is required. No additional properties are allowed at the root level.
