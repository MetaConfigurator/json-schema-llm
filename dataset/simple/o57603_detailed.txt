The schema describes a cuboid configuration for O3D3xx clipping. It is an object type. The main properties of the schema are: 'Xmin' is a number representing the minimum X coordinate, required with a default value of 0. 'Xmax' is a number representing the maximum X coordinate, required with a default value of 0. 'Ymin' is a number representing the minimum Y coordinate, required with a default value of 0. 'Ymax' is a number representing the maximum Y coordinate, required with a default value of 0. 'Zmin' is a number representing the minimum Z coordinate, required with a default value of 0. 'Zmax' is a number representing the maximum Z coordinate, required with a default value of 0. No additional properties are allowed at the root level.
