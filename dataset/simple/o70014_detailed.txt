The schema describes a configuration object. It is an object type. The main properties of the schema are: 'enabled' is a boolean indicating whether the feature is enabled, defaulting to false. 'versionIncreaseAllows' is an object that defines the allowed changes for version increments. It includes the properties: 'major', 'minor', and 'patch', each being a string with specific enum values: 'none', 'equivalent', 'nonBreaking', 'potentiallyBreaking', 'breaking'. The defaults are 'breaking', 'nonBreaking', and 'none', respectively. 'passThroughDifferences' is an array of strings for differences that should be passed through. No additional properties are allowed at the root level.
