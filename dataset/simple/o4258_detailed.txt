The schema describes an object structure for game content metadata. It is an object type. The main properties of the schema are: 'name' is a string that is required and must have a minimum length of 1. 'description' is a string that is required and must have a minimum length of 1. 'gameBaseVariantId' is a string that is required and must have a minimum length of 1. 'iconUrl' is a string that is required and must have a minimum length of 1. 'id' is a string that is required and must have a minimum length of 1. 'contentId' is a string that is required and must have a minimum length of 1. No additional properties are allowed at the root level.
