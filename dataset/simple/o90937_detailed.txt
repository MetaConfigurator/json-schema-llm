The schema describes a configuration involving options. It is an object type. It defines several reusable components under 'definitions': 'option' is an object, containing a required boolean property 'enabled' with a description of 'some text'. The main properties of the schema are: 'knownProp' is an object resolving to the referenced definition 'option', with a title of 'title' and a description of 'text'. No additional properties are allowed at the root level.
