The schema describes an API path configuration. It is an object type. The main properties of the schema are: 'ParentId' is a string that must match the pattern '^[a-z0-9]+$' and is required. 'PathPart' is a string that must match the pattern '^(\{[a-z]+\})|([a-z]+)$' and is required. 'RestApiId' is a string that must match the pattern '^[a-z0-9]{10}$' and is required. No additional properties are allowed at the root level.
