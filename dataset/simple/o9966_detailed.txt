The schema describes an object structure for managing response configurations. It is an object type. The main properties of the schema are: 'name' is a string with a minimum length of 1 and a maximum length of 255. 'responseCode' is an integer with a minimum value of 100 and a maximum value of 599. 'verificationType' is a string with an enum that can be one of the following values: 'facebook', 'fitbit', 'none', 'twilio', 'alexa'. 'verificationCode' is a string with a maximum length of 32767. 'waitForReply' is a boolean. 'basicAuthUsername' is a string with a maximum length of 255. 'basicAuthPassword' is a string with a maximum length of 255. No additional properties are allowed at the root level.
