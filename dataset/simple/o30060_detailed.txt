The schema describes a binding for a syslog policy. It is an object type. The main properties of the schema are: 'builtin' is an object that defines system attributes with four allowed values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'feature' is a string that specifies the feature with numerous allowed values including 'WL', 'WebLogging', 'SP', and many others. 'globalbindtype' is a string with three allowed values: 'SYSTEM_GLOBAL', 'VPN_GLOBAL', 'RNAT_GLOBAL'. 'numpol' is an integer, marked as readonly. 'policyname' is a string, and 'priority' is an integer. No additional properties are allowed at the root level.
