The schema describes a message sent via Short Message Service. It is an object type. It defines several reusable components under 'definitions': 'body' is a string, The text of your sms. 'from' is a string, The number you're trying to send an sms from. 'to' is a string, The number you're trying to send an sms to. The main properties of the schema are: 'body' is a string that is required. 'from' is a string that is optional. 'to' is a string that is required. No additional properties are allowed at the root level.
