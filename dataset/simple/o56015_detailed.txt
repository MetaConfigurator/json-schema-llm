The schema describes Fibaro Home Center Section JSON Schema. It is an object type. The main properties of the schema are: 'id' is an integer that specifies the identifier for the section, it is optional and must be greater than or equal to 0. 'name' is a string that specifies the name for the section, it is required and has a maximum length of 20 characters. 'created' is an integer that specifies the section creation timestamp, it is optional and must be greater than or equal to 0. 'modified' is an integer that specifies the section modification timestamp, it is optional and must be greater than or equal to 0. 'sortOrder' is an integer that specifies the sort order when displayed in GUI, it is optional and must be greater than or equal to 0. No additional properties are allowed at the root level.
