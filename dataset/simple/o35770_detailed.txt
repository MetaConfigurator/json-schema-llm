The schema describes a representation of a virtual machine. It is an object type. The main properties of the schema are: 'id' is a string that describes the id of the VM and is required with a default value of an empty string. 'ip-of-VM' is a string that describes the ip of the VM where the resource is to be deployed and is required with a default value of '127.0.0.1'. 'port-of-VM' is a string that describes the port of the VM where the resource is to be deployed and is required with a default value of '4243'. No additional properties are allowed at the root level.
