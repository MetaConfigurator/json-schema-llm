The schema describes the configuration for a batch deep learning job. It is an object type. The main properties of the schema are: 'config_file' is a string that represents the 'configFile' argument to the CNTK command line. It is required. 'id' is a string that represents the name of the job, which will be used as the name of the submitted job. It is required. 'input_directory' is a string that represents the 'RootDir' argument to the CNTK command line. It is required. No additional properties are allowed at the root level.
