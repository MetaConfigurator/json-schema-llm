The schema describes a workspace site configuration. It is an object type. The main properties of the schema are: 'displayroamingaccounts' is a string that accepts values from the enum: 'ON', 'OFF'. 'html5receiver' is a string that accepts values from the enum: 'ALWAYS', 'FALLBACK', 'OFF'. 'sitename' is a string. 'state' is a string that accepts values from the enum: 'UP', 'INITIALIZING', 'DOWN-HostUnknown', 'DOWN-ReqTimeout', 'DOWN-Wrong Store', 'DOWN-SF Error', 'DOWN-SSL Error', 'DOWN-Conn Reset', 'DOWN' and is readonly. 'storefronturl' is a string. 'storename' is a string. 'workspacecontrol' is a string that accepts values from the enum: 'ON', 'OFF'. 'xframeoptions' is a string that accepts values from the enum: 'ALLOW', 'DENY'. No additional properties are allowed at the root level.
