The schema describes configuration for application firewall JSON content types. It is an object type. The main properties of the schema are: 'builtin' is an object that is readonly and has an enum with values 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'feature' is a string that is readonly and has an enum with values 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', 'LoadBalancing', 'CS', 'ContentSwitching', 'CR', 'CacheRedirection', 'SC', 'SureConnect', 'CMP', 'CMPcntl', 'CompressionControl', 'PQ', 'PriorityQueuing', 'HDOSP', 'HttpDoSProtection', 'SSLVPN', 'AAA', 'GSLB', 'GlobalServerLoadBalancing', 'SSL', 'SSLOffload', 'SSLOffloading', 'CF', 'ContentFiltering', 'IC', 'IntegratedCaching', 'OSPF', 'OSPFRouting', 'RIP', 'RIPRouting', 'BGP', 'BGPRouting', 'REWRITE', 'IPv6PT', 'IPv6protocoltranslation', 'AppFw', 'ApplicationFirewall', 'RESPONDER', 'HTMLInjection', 'push', 'NSPush', 'NetScalerPush', 'AppFlow', 'CloudBridge', 'ISIS', 'ISISRouting', 'CH', 'CallHome', 'AppQoE', 'ContentAccelerator', 'SYSTEM', 'RISE', 'FEO', 'LSN', 'LargeScaleNAT', 'RDPProxy', 'Rep', 'Reputation', 'URLFiltering', 'VideoOptimization', 'ForwardProxy', 'SSLInterception', 'AdaptiveTCP', 'CQA', 'CI', 'ContentInspection', 'Bot'. 'isregex' is a string that has an enum with values 'REGEX', 'NOTREGEX'. 'jsoncontenttypevalue' is a string that is optional. No additional properties are allowed at the root level.
