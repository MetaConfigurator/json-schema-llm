The schema describes a redirect step. It is an object type. The main properties of the schema are: 'name' is a string and required. 'kind' is an enum with the allowed value 'redirect' and is required. 'label' is a string and required. 'title' is a string and required. 'url' is a string with a minimum length of 1 and is required. 'method' is one of: null or an enum with the allowed values 'GET' and 'POST', and is required. 'iface' is an object and is optional. No additional properties are allowed at the root level.
