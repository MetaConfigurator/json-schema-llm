The schema describes a data submission for producer share information. It is an object type. The main properties of the schema are: '@type' is a string representing the IRI for the JSON-LD data type, required with a single allowed value of 'ag:ProducerShare'. 'producerSharePercent' is a string describing the producer's share of a crop and benefits, required, with a minimum length of 0 and a maximum length of 8. No additional properties are allowed at the root level.
