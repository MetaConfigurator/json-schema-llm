The schema describes a USSD Language step. It is an object type. The main properties of the schema are: 'name' is a string and is required. 'kind' is a string with an enum of allowed values: 'ussdLanguage' and is required. 'label' is a string and is required. 'title' is a string and is required. 'language' is a string or null and is required. 'iface' is an object and is optional. No additional properties are allowed at the root level.
