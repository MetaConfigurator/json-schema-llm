The schema describes all input types of builders that perform operations on one or multiple sub-builders. It is an object type. The main properties of the schema are: 'builders' is an array required, containing at least one item where each item is an object with a required 'builder' property of type string that matches the pattern '.*:.*' and an optional 'options' property of type object. 'targets' is an array required, containing at least one item where each item is an object with a required 'target' property of type string that matches the pattern '.*:.*' and an optional 'overrides' property of type object. No additional properties are allowed at the root level.
