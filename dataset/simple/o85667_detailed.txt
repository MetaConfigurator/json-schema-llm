The schema describes content restrictions of an ANS object. It is an object type. The main properties of the schema are: 'content_code' is a string that describes the content restriction code/level/flag associated with the ANS object. 'embargo' is an object that describes the embargo configuration to enforce publishing restrictions. It is optional and must not contain additional properties. The 'embargo' object has a required property 'active' which is a boolean that indicates if the embargo is active or not. If 'active' is false, the embargo should be ignored. The 'embargo' also defines optional properties: 'end_time', which is a string in date-time format indicating when the embargo ends if applicable, and 'description', which is a string providing an optional description for the embargo. No additional properties are allowed at the root level.
