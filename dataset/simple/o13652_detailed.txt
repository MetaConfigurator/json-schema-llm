The schema describes a wishlist item. It is an object type. The main properties of the schema are: 'product' is a string that must match the pattern '^.+', and is required. 'amount' is an integer that has a minimum value of 1, is required, and has a default value of 1. 'note' is a string that is optional. 'createdAt' is a string that must be in date-time format and is optional. 'category' is a string that is optional. 'link' is a string that is optional. No additional properties are allowed at the root level.
