The schema describes an application firewall profile cookie consistency binding. It is an object type. The main properties of the schema are: 'alertonly' is a string that accepts the values 'ON' or 'OFF'. 'comment' is a string that is optional. 'cookieconsistency' is a string that is optional. 'isautodeployed' is a string that accepts the values 'AUTODEPLOYED' or 'NOTAUTODEPLOYED'. 'isregex' is a string that accepts the values 'REGEX' or 'NOTREGEX'. 'name' is a string that is optional. 'state' is a string that accepts the values 'ENABLED' or 'DISABLED'. No additional properties are allowed at the root level.
