The schema describes a geographical surrounding defined by a point and a radius. It is an object type. The main properties of the schema are: 'center' is an object representing the geographic center, which is required. It contains the required fields 'type' and 'coordinates'. 'coordinates' is an array of numbers that must contain exactly 2 items, representing a '[lng, lat]' ordered coordinate sequence. 'radius' is a number representing the radius in kilometers, which is also required. No additional properties are allowed at the root level.
