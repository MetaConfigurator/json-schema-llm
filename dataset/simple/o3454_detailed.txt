The schema describes IOx application descriptor - version 1.0. It is an object type. It defines several reusable components under 'definitions': 'descriptorschema_1_0.AppInfo' is an object, which requires 'name' and 'version' fields. It allows 'author-name', 'author-link', and 'description' as optional fields and does not allow additional properties. The main properties of the schema are: 'info' is a required property that resolves to the referenced definition 'descriptorschema_1_0.AppInfo'. 'app' is an optional property, which requires 'apptype' and can be one of the following configurations: an object with the required 'apptype' field of type string, that must be equal to 'vm'. 'manifest-version' is a required property, which can be either a string or a number. No additional properties are allowed at the root level.
