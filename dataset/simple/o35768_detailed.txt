The schema describes representing a Docker Image. It is an object type. The main properties of the schema are: 'id' is a string that represents the id of the Docker Image and is required with a default value of ''. 'name' is a string that represents the name of the Docker Image and is required with a default value of ''. 'author' is a string that represents the author of the Docker Image and is required with a default value of ''. 'version' is an integer that represents the Version of the Docker Image and is optional with a default value of ''. 'script' is a string that represents the Customization script for Docker Image and is optional with a default value of ''. No additional properties are allowed at the root level.
