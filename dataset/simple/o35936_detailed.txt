The schema describes a configuration for scenarios. It is an array type. The main properties of the schema are: 'name' is a string that describes the name of this scenario. It is required. 'weighting' is a number that describes the weighting for this scenario. It is required. 'loop' is a boolean that indicates whether the user running this scenario should loop or stop after one run. It is required. 'throttleFrequencyInMinutes' is a number that describes how frequently the scenario should be run if looping, with a minimum value of 0. It is optional. No additional properties are allowed at the root level.
