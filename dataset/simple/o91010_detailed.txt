The schema describes a Legacy Brightlight. It is an object type. It defines several reusable components under 'definitions': 'core.brightlight' is an object, which contains properties: 'name' is a string describing the friendly device name (e.g. Kitchen Light); 'power' is an integer describing the current sensed or set value for Brightness; 'state' is a boolean describing true if on, false if off. The main properties of the schema are: 'core.brightlight' is a reference to the 'core.brightlight' definition. The root schema is required to include 'core.brightlight'. No additional properties are allowed at the root level.
