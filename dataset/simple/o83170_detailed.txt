The schema describes Angular Guard Options. It is an object type. The main properties of the schema are: 'appRoot' is a string optional; 'flat' is a boolean optional, with a default value of true; 'module' is a string optional, allows specification of the declaring module, and has an alias of 'm' and a subtype of 'filepath'; 'name' is a string required; 'path' is a string optional, with a default value of 'app'; 'sourceDir' is a string optional, with a default value of 'src'; 'spec' is a boolean optional, with a default value of true. No additional properties are allowed at the root level.
