The schema describes an unattended upgrades plugin manifest. It is an object type. The main properties of the schema are: 'plugins' is an object that contains the following properties: 'unattended_upgrades' is an object that contains the properties: 'update_interval' is an integer that is required, 'download_interval' is an integer that is required, and 'upgrade_interval' is an integer that is required. The property 'unattended_upgrades' is required within 'plugins', and 'plugins' itself is required at the root level. No additional properties are allowed at the root level.
