The schema describes the specification for materials. It is an object type. The main properties of the schema are: 'asset_type' is a string that must match the pattern 'material'. It is required. 'shader' is a string that represents the path to a shader asset. It is required. 'parameters' is an object that contains the parameters to override from the shader. It is required. The 'parameters' object allows any property name, where each value can be a string, a number, or an array of numbers with a minimum of 2 and a maximum of 4 items. No additional properties are allowed at the root level.
