The schema describes a configuration object. It is an object type. The main properties of the schema are: 'grid' is an object that contains a required property 'resource', which is a string formatted as 'topic' with an 'axRole' of 'inlet'. 'tabs' is an object that contains an optional property 'parameter', which is a string formatted as 'topic' with a default value of 'events'. No additional properties are allowed at the root level.
