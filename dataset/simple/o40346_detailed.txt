The schema describes a certificate message. It is an object type. The main properties of the schema are: 'type' is a string that must have a value of 'certificate' and is required. 'certificate' is a string that is required. 'chain' is an array that is optional, must contain at least one item, and each item must be a string. 'refresh' is a string that is optional. No additional properties are allowed at the root level.
