The schema describes a chapter object. It is an object type. The main properties of the schema are: 'id' is a string that represents the uuid of the chapter, 'slug' is a string that represents the slug of the chapter, 'title' is a string that represents the title of the chapter, 'text' is a string that represents the text of the chapter, 'parentSlug' is a string that represents the slug of the chapter's parent chapter, 'previousSlug' is a string that represents the slug of the chapter's previous sibling, and 'nextSlug' is a string that represents the slug of the chapter's next sibling. No additional properties are allowed at the root level.
