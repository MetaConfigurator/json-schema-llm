The schema describes a VPN connection configuration. It is an object type. The main properties of the schema are: 'channelnumber' is an integer, readonly. 'destip' is a string, readonly. 'destport' is an integer, readonly. 'domain' is a string, readonly. 'nodeid' is an integer, optional. 'peid' is an integer, readonly. 'srcip' is a string, readonly. 'srcport' is an integer, readonly. 'username' is a string, optional. No additional properties are allowed at the root level.
