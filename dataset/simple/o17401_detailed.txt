The schema describes the Activity Dump File. It is an object type. The main properties of the schema are: 'url' is a string that represents the retrieval URL at which the peer can retrieve the activity dump file. It is required. 'checksum' is a string that denotes the SHA-1 checksum of the contents of the activity dump file referred to by url. It is required. 'size' is a number indicating the size of the activity dump file in bytes. It is required. 'version' is a string that specifies the version of the SAS-SAS protocol used for generating this file. It is required. 'recordType' is a string that describes the type of records contained in the activity dump file. It is required. No additional properties are allowed at the root level.
