The schema describes an array of objects containing tracking information. It is an array type. The main properties of the schema are: 'trackedPackages' is an array of objects, where each object must contain the properties 'id' and 'name', both as strings with a minimum length of 1. 'type' is a string with a minimum length of 1 and is required. 'url' is a string formatted as a URI and is also required. At least one item is required in the array. No additional properties are allowed at the root level.
