The schema describes required parameters for the SmartRTB server adapter. It is an object type. The main properties of the schema are: 'pub_id' is a string assigned publisher ID with a minimum length of 4 characters and is required. 'med_id' is a string property ID not zone ID not provided and is optional. 'zone_id' is a string specific zone ID for this placement, belonging to app/site with a minimum length of 20 characters and is optional. 'force_bid' is a boolean that indicates whether to force bids with a test creative and is optional. No additional properties are allowed at the root level.
