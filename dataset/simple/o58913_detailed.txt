The schema describes an integration configuration. It is an object type. The main properties of the schema are: 'integration' is an object that contains additional properties, where 'is_active' is a boolean (optional) and 'name' is a string that must be one of the following values: 'hipchat', 'hub', 'mailgun', 'nexmo', 'postmark', 'slack', 'twilio' (required). No additional properties are allowed at the root level.
