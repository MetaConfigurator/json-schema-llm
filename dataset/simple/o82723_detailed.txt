The schema describes a swipeable mixin. It is an object type. The main properties of the schema are: 'swipe' is a string that is optional. It has a minimum length of 0 and a maximum length of 255. It describes a swipe (aka banner/label/overlay) used in a visual treatment on the node, with examples such as 'Exclusive', 'NSFW', 'Breaking Bad Mojo'. The property 'swipe' also has a 'pbj' object with a type of string and a rule of 'single'. No additional properties are allowed at the root level.
