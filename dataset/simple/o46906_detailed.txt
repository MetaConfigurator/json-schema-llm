The schema describes the capability of a switch level. It is an object type. The main properties of the schema are: 'valueType' is a number that specifies the level of the switch, with a minimum value of 0 and a maximum value of 100; it is optional. 'writable' is a boolean that indicates whether the property can be modified; it is set to false. It defines several reusable components under 'definitions': 'inputData' is an object that contains several properties: 'level' is a number, required, with a minimum value of 0 and a maximum value of 100; 'transitiontime' is a number, with a minimum value of 0 and a maximum value of 6553. No additional properties are allowed at the root level.
