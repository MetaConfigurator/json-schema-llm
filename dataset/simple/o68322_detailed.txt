The schema describes parameters accepted by the NanoInteractive adapter. It is an object type. The main properties of the schema are: 'pid' is a string Placement idd, required. 'nq' is an array of strings search queries, optional. 'category' is a string IAB Category, optional. 'subId' is a string any segment value provided by publisher, optional. 'ref' is a string referer, optional. No additional properties are allowed at the root level.
