The schema describes valid @context values. It is an object type. It defines several reusable components under 'definitions': 'contextValue' is an enumeration, it specifies that the value must be 'http://www.w3.org/ns/anno.jsonld'. 'contextArray' is an object, it ensures that any items in the array must not be of type string and must not reference 'contextValue'. The main properties of the schema are: '@context' is a property that can be either a string referencing 'contextValue' or an array referencing 'contextArray'. The '@context' property is optional and requires only one of the two specified forms. No additional properties are allowed at the root level.
