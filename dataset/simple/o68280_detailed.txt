The schema describes parameters accepted by the Adform adapter. It is an object type. The main properties of the schema are: 'mid' is an integer or string that identifies the placement selling the impression, and it is required. 'priceType' is a string that indicates the expected price type (net or gross) of bids, and it can take one of the following values: 'gross', 'net'. 'mkv' is a string for comma-separated key-value pairs, with a specific pattern restriction, and forbidden symbols include '&'. 'mkw' is a string for comma-separated keywords, also with forbidden symbols including '&', and it has a specific pattern restriction. No additional properties are allowed at the root level.
