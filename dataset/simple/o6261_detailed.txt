The schema describes a configuration object. It is an object type. The main properties of the schema are: 'key' is a string that must match the pattern '^[0-9a-zA-Z_-]{1,230}$', and it is required. 'type' is a string with an enum value of 'mqttTopic', and it is required. 'outputIds' is an array where each item is an array of strings. 'meta' is an object. 'config' is an object that allows no additional properties. No additional properties are allowed at the root level.
