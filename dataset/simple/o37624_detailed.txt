The schema describes a contact information structure. It is an object type. The main properties of the schema are: 'address' is an object that contains required properties 'streetAddress' and 'city', both of which are strings. 'phoneNumber' is an array of objects, where each object contains required properties 'location' (a string) and 'code' (an integer). The schema requires both 'address' and 'phoneNumber' properties at the root level. No additional properties are allowed at the root level.
