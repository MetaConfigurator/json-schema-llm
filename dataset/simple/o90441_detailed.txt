The schema describes a cookie object. It is an object type. The main properties of the schema are: 'comment' is a string optional description of the cookie. 'domain' is a string optional domain for which the cookie is valid. 'expires' is a string or null that follows the date-time format, optional, indicating when the cookie expires. 'httpOnly' is a boolean optional flag indicating if the cookie is HTTP only. 'name' is a string required for the name of the cookie. 'path' is a string optional path for which the cookie is valid. 'secure' is a boolean optional flag indicating if the cookie should only be transmitted over secure protocols. 'value' is a string required for the cookie's value. No additional properties are allowed at the root level.
