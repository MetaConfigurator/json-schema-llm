The schema describes a list of moves played by each child. It is an array type. The main properties of the schema are: Each item in the array is an object representing a move, which has the following properties: 'childno' is an integer that serves as the unique child identifier and is required, with a minimum value of 0. 'split' is a boolean indicating whether the split is made in this move or not and is required. 'pause' is a boolean indicating whether the child comes to stand still or not and is required. 'relativeangle' is a number representing the angle relative to the current angle and is required, with a minimum value of -360.0 and a maximum value of 360.0. 'ejectmass' is a boolean indicating whether the child ejects mass or not and is required. The array must contain at least 1 item, and all items must be unique. No additional properties are allowed at the root level.
