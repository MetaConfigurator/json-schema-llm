The schema describes a structure for a question and answers format. It is an object type. The main properties of the schema are: '_id' is a string optional identifier for the question, 'icon' is a string optional icon representation, 'question' is a string required text of the question, 'end' is a string required indication of the question's end, and 'answers' is an array required containing at least two items. Each item in 'answers' is an object that defines 'value' as a string representing the answer and 'votes' as an array of strings within the object, which cannot contain additional properties. No additional properties are allowed at the root level.
