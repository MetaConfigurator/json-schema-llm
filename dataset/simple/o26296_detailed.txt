The schema describes a user. It is an object type. The main properties of the schema are: 'email' is a string with format 'email', required. 'first' is a string, required. 'last' is a string, required. 'password' is a string with a minimum length of 8, optional. 'roles' is an array, required. 'username' is a string with a minimum length of 3, required. 'verification' is a string, optional. 'verified' is a boolean, required. No additional properties are allowed at the root level.
