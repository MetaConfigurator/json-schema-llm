The schema describes a USSD Say step. It is an object type. The main properties of the schema are: 'name' is a string and required, 'kind' is a string with an enum value of 'ussdSay' and required, 'label' is a string and required, 'title' is a string and required, 'text' is a string and required, 'language' is a string or null and optional, 'iface' is an object and optional. No additional properties are allowed at the root level.
