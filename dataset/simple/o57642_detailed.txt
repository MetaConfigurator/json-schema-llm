The schema describes O3D3xx temporal exponential filter configuration. It is an object type. The main properties of the schema are: 'MinSmoothDiff' is a number with a minimum of 0, default value of 0.3, and marked as private. 'MinSDAlpha' is a number with a minimum of 0, maximum of 1, default value of 0.2, and marked as private. 'MaxSmoothDiff' is a number with a minimum of 0, default value of 1, and marked as private. 'MaxSDAlpha' is a number with a minimum of 0, maximum of 1, default value of 0.95, and marked as private. No additional properties are allowed at the root level.
