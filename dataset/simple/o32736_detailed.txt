The schema describes template meta data (aka front matter). It is an object type. The main properties of the schema are: 'target' is a string that can be an absolute or relative path that will be resolved relative to the directory confgen is called in (i.e. not relative to the template file). 'chmod' is a number representing file permissions to apply to the target file (chmod mode), with a minimum value of 0 and a maximum value of 777. Permissions must be given in decimal notation, which will be converted to octal representation internally. 'reload' is a string for a command to execute after writing the configuration file. 'description' is a string for an optional descriptive text for template authors. No additional properties are allowed at the root level.
