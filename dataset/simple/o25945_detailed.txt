The schema describes a top level object representing MyObject. It is an object type. The main properties of the schema are: 'opacity' is a number representing the opacity, a required field with a minimum value of 0 and a maximum value of 100. 'position' is an array of numbers representing the position, a required field that must contain exactly 3 items. 'rotation' is an object representing the rotation, a required field with a property 'yaw' which is a number with a minimum value of -90 and a maximum value of 90. No additional properties are allowed at the root level.
