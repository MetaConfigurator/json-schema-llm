The schema describes a business address. It is an object type. The main properties of the schema are: 'businessName' is a string optional; 'contactName' is a string optional; 'line1' is a string required; 'city' is a string required; 'state' is a string required; 'zip' is a string required; 'country' is a string required. The 'businessName' property, if present, requires the 'contactName' property. No additional properties are allowed at the root level.
