The schema describes relationships between shaders and Mindbender IDs. It is an object type. The main properties of the schema are: '_schema' is a string schema identifier for payload, and is required. 'shader' is an array of strings, describing names of directories with each item being a Mindbender ID and optional face indexes, e.g. 'f9520572-ac1d-11e6-b39e-3085a99791c9.f[5002:5185]', and is required. No additional properties are allowed at the root level.
