The schema describes an Address following the convention of http://microformats.org/wiki/hcard. It is an object type. The main properties of the schema are: 'post-office-box' is a string optional, 'extended-address' is a string optional, 'street-address' is a string optional, 'locality' is a string required, 'region' is a string required, 'postal-code' is a string optional, 'country-name' is a string required. Compositional constraints state that 'post-office-box' and 'extended-address' depend on 'street-address'; 'street-address' depends on 'region'; 'locality' depends on 'region'; and 'region' depends on 'country-name'. No additional properties are allowed at the root level.
