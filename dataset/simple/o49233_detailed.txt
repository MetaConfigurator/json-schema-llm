The schema describes a structure for holding an array of results. It is an object type. The main properties of the schema are: 'results' is an array containing objects, where each object has properties 'type', 'label', 'size', and 'wells'. All properties are required. 'type' is a string, 'label' is a string, 'size' is an integer, and 'wells' is an object that allows any additional properties. No additional properties are allowed at the root level.
