The schema describes the structure for booking information. It is an object type. The main properties of the schema are: 'booking_reason' is a string optional field. 'slots' is an array representing the list of slots for the bookings, required field, containing a maximum of 1 item. Each item in the array is an object with the following properties: 'start_date' is an integer representing the Unix timestamp for the start date of the booking, required field. 'end_date' is an integer representing the Unix timestamp for the end date of the booking, required field. 'iana' is a string representing the IANA identifier of the booking owner, required field. No additional properties are allowed at the root level.
