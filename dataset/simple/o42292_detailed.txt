The schema describes the structure of an address object. It is an object type. The main properties of the schema are: 'address' is an object that represents the address details. It is required and does not allow additional properties. The 'address' object contains the following properties: '@id' is an integer which represents the identifier of the address. '@city' is a string which represents the city name. 'street' is a string which represents the street name. No additional properties are allowed at the root level.
