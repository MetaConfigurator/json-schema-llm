The schema describes Theme objects. It is an object type. The main properties of the schema are: 'name' is a string representing the name of the theme. 'author' is a string representing the author of the theme. 'description' is a string representing the description of the theme. 'body' is an array. 'main' is an array. 'diffHeader' is an array. 'diffBody' is an array. No additional properties are allowed at the root level.
