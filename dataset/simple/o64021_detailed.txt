The schema describes an array of search objects to be performed by the search layer. It is an array type. The main properties of the schema are: Each item in the array is an object that contains the following properties: 'field' is a string that represents the field to search and is required. 'value' is a string that represents the value to search and is required. 'search_type' is a string that represents the type of search to perform, with allowed values 'WILDCARD', 'MATCH', and 'VERSION', and is required. No additional properties are allowed at the root level.
