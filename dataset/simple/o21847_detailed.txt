The schema describes a data structure containing a dictionary, a list of IDs, and a last tag ID. It is an object type. The main properties of the schema are: 'dictionary' is an object that contains string properties identified by numeric strings as keys. No additional properties are allowed. 'ids' is an array of strings, required to contain unique items. 'lastTagId' is an integer that must be greater than or equal to -1. All three properties are required. No additional properties are allowed at the root level.
