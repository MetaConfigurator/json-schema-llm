The schema describes an object indicating the field display order for the related records and whether they should be sorted in ascending or descending order. It is an object type. The main properties of the schema are: 'field' is a string describing the attribute value of the field selected that will drive the sorting of related records. 'order' is a string describing the ascending or descending sort order of the returned related records. It is required and must be one of the following values: 'asc', 'desc'. No additional properties are allowed at the root level.
