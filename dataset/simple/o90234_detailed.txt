The schema describes a level of detail (LOD) for tiles in a mapping context. It is an object type. The main properties of the schema are: 'level' is an integer ID for each level, optional. 'levelValue' is a string to be used when constructing a URL to access a tile from this LOD, optional. 'resolution' is a number representing the resolution in map units of each pixel in a tile for each level, optional. 'scale' is a number indicating the scale for each level, optional. No additional properties are allowed at the root level.
