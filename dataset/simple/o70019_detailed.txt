The schema describes configuration options for output handling. It is an object type. The main properties of the schema are: 'append' is a boolean indicating whether to append to the output or overwrite it, with a default value of false; 'indent' is a boolean indicating whether to produce indented output for easier human readability, with a default value of false; 'minSeverity' is a string representing the minimum severity level required, with a default value of 'POTENTIALLY_BREAKING' and allowed values being 'NON_BREAKING', 'POTENTIALLY_BREAKING', 'BREAKING'; 'output' is a string specifying the output destination, with a default value of 'out'. No additional properties are allowed at the root level.
