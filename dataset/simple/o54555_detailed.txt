The schema describes the data contained in a Table. It is an object type. The main properties of the schema are: 'headers' is an array containing strings, required. 'datasets' is an array, required, containing objects. Each object in 'datasets' has a property 'row', which is an array of objects. Each object in 'row' has properties 'color', 'background', and 'value', where 'color' and 'background' are strings matching the pattern '^#[0-9a-fA-F]{6}$', and 'value' is a string that is required. No additional properties are allowed at the root level.
