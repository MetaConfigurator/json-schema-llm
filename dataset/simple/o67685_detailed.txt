The schema describes sidebar configurations. It is an object type. The main properties of the schema are: 'left' is an object defining the left sidebar configurations. It contains the property 'sticky', which is a boolean indicating whether the sidebar sticks to the top when the page scrolls, with a default value of false. 'right' is an object defining the right sidebar configurations. It contains the property 'sticky', which is a boolean indicating whether the sidebar sticks to the top when the page scrolls, with a default value of false. No additional properties are allowed at the root level.
