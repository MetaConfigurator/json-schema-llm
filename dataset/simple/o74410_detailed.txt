The schema describes a collection of test cases. It is an array type. The main properties of the schema are: Each item in the array is an object containing: 'description' is a string that is required and describes the test case. 'schema' is an unspecified type and is required. 'tests' is an array that is required, containing at least one object. Each test object has 'description' as a string that is required, 'data' as an unspecified type that is required, and 'valid' as a boolean that is required. No additional properties are allowed at the root level.
