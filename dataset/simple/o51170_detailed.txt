The schema describes a registry for qooxdoo libraries in a repository/package. It is an object type. The main properties of the schema are: '$schema' is a string that describes the json schema of the version of qooxdoo.json. It is required and must be one of the following values: 'https://qooxdoo.org/schema/qooxdoo-1-0-0.json', 'https://raw.githubusercontent.com/qooxdoo/qooxdoo-compiler/master/source/resource/qx/tool/schema/qooxdoo-1-0-0.json'. 'libraries' is an array that contains objects with at least a 'path' property, which specifies the path to the library. It is required. Each object in the array can have an optional 'path' property of type string and an optional 'main' property of type boolean. No additional properties are allowed at the root level.
