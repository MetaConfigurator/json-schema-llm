The schema describes Angular Playground's Sandbox. It is an object type. The main properties of the schema are: 'flat' is a boolean flag to indicate if a dir is created. It is optional and has a default value of false. 'name' is a string that represents the name of the component to add the sandbox to. It is optional and has a default value sourced from argv at index 0. 'path' is a string describing the path to create the sandbox. It is optional and has a format of 'path'. No additional properties are allowed at the root level.
