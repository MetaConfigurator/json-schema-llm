The schema describes the scene media events configuration. It is an object type. The main properties of the schema are: 'onEnded' is an array or a string that represents the onEnded event of the media. The action uid or an array of action uids. Works only on video media. 'onPause' is an array or a string that represents the onPause event of the media. The action uid or an array of action uids. Works only on video media. 'onPlay' is an array or a string that represents the onPlay event of the media. The action uid or an array of action uids. Works only on video media. 'onSeeked' is an array or a string that represents the onSeeked event of the media. The action uid or an array of action uids. Works only on video media. No additional properties are allowed at the root level.
