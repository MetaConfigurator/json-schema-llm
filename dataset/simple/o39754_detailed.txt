The schema describes a JSON schema for Azure Functions extensions.json files. It is an object type. The main properties of the schema are: 'extensions' is an array that represents the list of installed extensions. Each item in the array is an object with optional properties: 'name' is a string that describes the short extension name with a minimum length of 0; 'typeName' is a string that describes the type assembly qualified name; 'hintPath' is a string that describes an optional file path hint to the assembly containing the type. No additional properties are allowed at the root level.
