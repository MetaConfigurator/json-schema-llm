The schema describes an object representation of a content piece. It is an object type. The main properties of the schema are: 'title' is a string and is required. 'content' is a string and is optional. 'status' is an integer and is optional. 'publication_date' is a string representing the start date of the booking and is optional. 'expiration_date' is a string representing the start date of the booking and is optional. 'is_headline' is a boolean and is optional. 'thread_id' is an integer and is required. No additional properties are allowed at the root level.
