The schema describes the value returned by GET /{db}/{docid}. It is an object type. The main properties of the schema are: '_id' is a string that is required. '_rev' is a string that is required. '_deleted' is a boolean that is optional. '_attachments' is an object that is optional. '_conflicts' is an array that is optional. '_deleted_conflicts' is an array that is optional. '_local_seq' is a number that is optional. '_revs_info' is an array that is optional. '_revisions' is an object that is optional. No additional properties are allowed at the root level.
