The schema describes player performance data in a game. It is an object type. The main properties of the schema are: 'PrevSpartanRank' is an integer required field representing the previous Spartan rank. 'SpartanRank' is an integer required field representing the current Spartan rank. 'PrevTotalXP' is an integer required field representing the previous total experience points. 'TotalXP' is an integer required field representing the current total experience points. 'SpartanRankMatchXPScalar' is a number required field representing a scalar value for Spartan rank match experience points. 'PlayerTimePerformanceXPAward' is an integer required field representing the experience points awarded for time performance. 'PerformanceXP' is either an integer or null required field for performance experience points. 'PlayerRankXPAward' is an integer required field representing the experience points awarded for player rank. 'BoostAmount' is an integer required field representing the amount of boost. 'MatchSpeedWinAmount' is an integer required field representing the speed win amount for matches. 'ObjectivesCompletedAmount' is an integer required field representing the total number of completed objectives. No additional properties are allowed at the root level.
