The schema describes a quantity to be used for specifying information rates. It is an object type. The main properties of the schema are: 'amount' is a number that represents a quantity of something, typically size, value, extent, or money and is required. 'objectType' is a string that represents the type of the object, which must be 'informationRateQuantity' and is required. 'units' is a string that indicates the unit of measure for the quantity, allowing values 'Mbps' or 'Gbps', and is required. No additional properties are allowed at the root level.
