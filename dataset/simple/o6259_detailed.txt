The schema describes a structure for device configuration. It is an object type. The main properties of the schema are: 'key' is a string that is required and represents the Device Tag Pair. 'type' is a string that is required, with a value restricted to 'deviceTag'. 'outputIds' is an array of arrays, where each nested array contains strings. 'meta' is an object. 'config' is an object with no additional properties allowed. No additional properties are allowed at the root level.
