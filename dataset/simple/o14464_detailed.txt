The schema describes an action type. It is an object type. The main properties of the schema are: 'name' is a string that identifies the action type and must be unique in the organization. 'description' is a string that provides a human description of the action type. 'public' is a boolean that indicates whether the action type is available to everybody. 'type' is a string that represents a valid URL to define the action type. 'schema' is an object that represents a valid JSON Schema. No additional properties are allowed at the root level.
