The schema describes an individual log message. It is an object type. The main properties of the schema are: 'args' is an object optional. 'created' is an object optional. 'exc_info' is an object optional. 'exc_text' is an object optional. 'filename' is an object optional. 'funcName' is an object optional. 'levelname' is an object optional. 'levelno' is an object optional. 'lineno' is an object optional. 'message' is a string required. 'module' is an object optional. 'msecs' is an object optional. 'name' is an object optional. 'pathname' is an object optional. 'process' is an object optional. 'processName' is an object optional. 'relativeCreated' is an object optional. 'thread' is an object optional. 'threadName' is an object optional. No additional properties are allowed at the root level.
