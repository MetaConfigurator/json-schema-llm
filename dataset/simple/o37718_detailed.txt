The schema describes a schedule of deliveries to multiple facilities. It is an object type. The main properties of the schema are: 'doc_type' is a string matching the pattern '^dailyDelivery$', and it is required. 'version' is a string formatted as semver, and it is required. 'deliveryRoundID' is a string with specific chances for a hash of length 32, and it is optional. 'driverID' is a string, and it is optional. 'facilityRounds' is an array, and it is optional. 'packingLists' is an array, and it is optional. No additional properties are allowed at the root level.
