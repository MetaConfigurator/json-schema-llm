The schema describes a Piwik Core Collector Query. It is an object type. The main properties of the schema are: 'api_method' is a string optional. 'api_method_arguments' is an object optional that contains 'idGoal', which is a string optional. 'frequency' is a string optional. 'period' is a string optional. 'site_id' is a string required. No additional properties are allowed at the root level.
