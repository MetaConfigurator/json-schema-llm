The schema describes the specification for the asset type for loading sound files. It is an object type. The main properties of the schema are: 'asset_type' is a string that must match the pattern 'sound' and is required. 'sound_file' is a string that describes the path of the file relative to the current asset directory and is optional. 'file_type' is a string that describes the type of file to load and must match the pattern 'OGG' and is optional. No additional properties are allowed at the root level.
