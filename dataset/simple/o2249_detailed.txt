The schema describes a login request. It is an object type. The main properties of the schema are: 'authMode' is a string required to determine the authentication mode. 'email' is a string optional field intended for user email. 'name' is a string optional field for the user's first name. 'surname' is a string optional field for the user's last name. 'password' is a string optional field used for user authentication. No additional properties are allowed at the root level.
