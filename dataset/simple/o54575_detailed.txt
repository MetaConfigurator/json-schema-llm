The schema describes a product. It is an object type. The main properties of the schema are: 'brand' is a string required field. 'category' is a string required field. 'description' is a string required field. 'id' is a string required field. 'name' is a string required field. 'price' is a string optional field. 'schema' is a string required field with a format of url. No additional properties are allowed at the root level.
