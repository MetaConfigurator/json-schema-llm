The schema describes a configuration for breakpoints within a context. It is an object type. The main properties of the schema are: 'context' is a string representing the context element selector, required. 'breakpoints' is an array containing objects, each structured with required properties: 'name' (a string for the breakpoint name) and 'size' (an object with required properties 'width' and 'height', both integers representing viewport dimensions). No additional properties are allowed at the root level.
