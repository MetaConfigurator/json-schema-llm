The schema describes a tcheckbox atom with optional label. It is an object type. The main properties of the schema are: 'tag' is an enum, representing allowed values 'div', 'li', 'span' and is required. 'label' is a string and is optional. 'inputId' is a string and is required. 'inputName' is a string and is required. 'inputValue' is a string and is required. 'checked' is a boolean and is optional. '_state' is an enum, representing allowed values '' and 'disabled' and is optional. No additional properties are allowed at the root level.
