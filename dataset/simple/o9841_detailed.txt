The schema describes an array of Device states. It is an array type. The main properties of the schema are: 'items' is an object, representing a single Device state with a required 'data' field. 'time' is an optional field that can be a string, a number, or an object containing a required field '$date' of type string. 'meta' is an optional field without defined properties. 'data' is a required object where its properties match the pattern '^[0-9a-zA-Z_-]{1,255}$' and values can be of type number, string, or boolean. 'flowVersion' is an optional string field with a minimum length of 1 and a maximum length of 255. No additional properties are allowed at the root level.
