The schema describes an authorization response. It is an object type. The main properties of the schema are: 'idTagInfo' is an object that contains information about the ID tag. It is required. The properties of 'idTagInfo' are: 'status' is a string that indicates the status of the ID tag and must be one of the following values: 'Accepted', 'Blocked', 'Expired', 'Invalid', or 'ConcurrentTx'. 'expiryDate' is a string formatted as a date-time that indicates when the tag expires. 'parentIdTag' is a string with a maximum length of 20 characters that represents the parent ID tag. No additional properties are allowed at the root level.
