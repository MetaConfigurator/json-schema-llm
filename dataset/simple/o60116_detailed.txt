The schema describes the value returned by GET /{db}/_all_docs. It is an object type. The main properties of the schema are: 'offset' is a number required field. 'rows' is an array required field, containing items that are objects with properties: 'id' as a string, 'key' as a string, and 'value' as an object with a property 'rev' as a string. 'total_rows' is a number required field. 'update_seq' is a number optional field. No additional properties are allowed at the root level.
