The schema describes a set of manifest items for image save and load. It is an array type. The main properties of the schema are: Each item is an object that includes 'Config', 'Layers', 'Parent', and 'RepoTags' properties. 'Config' is a string, required. 'Layers' is an array of strings, with a minimum of one item, required. 'Parent' is a string, optional. 'RepoTags' is an array of strings, with a minimum of one item, optional. No additional properties are allowed at the root level.
