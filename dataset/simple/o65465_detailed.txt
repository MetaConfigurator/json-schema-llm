The schema describes a share parcel. It is an object type. The main properties of the schema are: 'data_type' is a string that must be one of the following values: 'share_parcel'. It is required. 'properties' is an object that is required and contains the following properties: 'number_of_shares' is an integer that indicates the number of shares, if known. 'percentage_of_shares' is a number that indicates the percentage of shares, if known, with a maximum value of 100 and a minimum value of 0. 'shareholders' is an array of objects that must have at least one item. Each item can be a natural person or a company and must contain a required field 'name'. It may also include 'jurisdiction', 'company_number', 'identifier', 'type', 'address', and 'address_country', where 'type' must be one of the following values: 'Company', 'Person'. No additional properties are allowed at the root level.
