The schema describes a generic related package. It is an object type. The main properties of the schema are: 'name' is a string Name of the package. It is required. 'payload_type' is a string or null The type of payload for this package. It must be one of: 'source', 'binary', 'doc' or null. 'type' is a string Descriptive name of the type of package: RubyGem, Python Wheel, Java Jar, Debian package, etc. It is required. 'version' is a string or null Version of the package. It can be a string or null. No additional properties are allowed at the root level.
