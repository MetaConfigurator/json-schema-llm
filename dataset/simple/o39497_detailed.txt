The schema describes a contact detail. It is an object type. The main properties of the schema are: 'content' is a string required, 'date_added' is a string required, it has a date-time format, 'id' is a string required, it has a uuid format, 'type' is a string required, with allowed values: 'email_address', 'phone_number', 'address'. No additional properties are allowed at the root level.
