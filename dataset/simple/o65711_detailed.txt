The schema describes an OpenTok Broadcast. It is an object type. The main properties of the schema are: 'id' is a string that must match the pattern '^[a-z0-9]{8}-[a-z0-9]{4}-[a-z0-9]{4}-[a-z0-9]{4}-[a-z0-9]{12}$' and is required; 'sessionId' is a string and is required; 'partnerId' is an integer and is required; 'createdAt' is an integer and is required; 'updatedAt' is an integer and is required; 'broadcastUrls' is a nullable type that can be either null or an object with a property 'hls' which is a string. No additional properties are allowed at the root level.
