The schema describes a specification for Rocket Document assets. It is an object type. The main properties of the schema are: 'asset_type' is a string that must match the pattern 'rocket_document' and is required. 'rml_file' is a string that provides the path to the .rml file, relative to the current asset directory, and is also required. 'required_fonts' is an array of strings that lists the fonts to load for the document and is optional. No additional properties are allowed at the root level.
