The schema describes a block mixin for canvas elements. It is an object type. The main properties of the schema are: 'etag' is a string that matches the pattern '^[\w\.:-]+$'. 'css_class' is a string that matches the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML. 'updated_date' is a string in date-time format representing an update that occurred on the node this block is attached to. No additional properties are allowed at the root level.
