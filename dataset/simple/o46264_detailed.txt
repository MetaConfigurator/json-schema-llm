The schema describes transaction information. It is an object type. The main properties of the schema are: 'flag' is an integer optional property. 'investment' is a boolean required property. 'splitChild' is a boolean required property. 'splitParent' is a boolean required property. 'transfer' is a boolean required property. 'transferTo' is a boolean required property. 'void' is a boolean required property. No additional properties are allowed at the root level.
