The schema describes an index of machine-readable issuer JSON URLs. It is an object type. The main properties of the schema are: 'formulary_urls' is an array that contains string items formatted as URIs. It is required. 'plan_urls' is an array that contains string items formatted as URIs, with a minimum of 1 item. It is required. 'provider_urls' is an array that contains string items formatted as URIs, with a minimum of 1 item. It is required. No additional properties are allowed at the root level.
