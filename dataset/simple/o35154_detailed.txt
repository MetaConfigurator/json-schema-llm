The schema describes configuration properties for a data source. It is an object type. The main properties of the schema are: 'scheme' is a string representing the title 'INSPIRE', which is optional. 'source' is a string that must be one of the following values: 'INSPIRE', 'submitter', 'conference', 'curator', 'publisher', 'magpie', and it is optional. 'term' is a string representing the title 'INSPIRE term', which must be one of the following values: 'Accelerators', 'Astrophysics', 'Computing', 'Data Analysis and Statistics', 'Experiment-HEP', 'Experiment-Nucl', 'General Physics', 'Gravitation and Cosmology', 'Instrumentation', 'Lattice', 'Math and Math Physics', 'Other', 'Phenomenology-HEP', 'Theory-HEP', 'Theory-Nucl', and it is optional. No additional properties are allowed at the root level.
