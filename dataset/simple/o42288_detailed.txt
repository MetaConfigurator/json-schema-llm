The schema describes a data structure for binary data mappings. It is an object type. The main properties of the schema are: 'my-data' is an object containing three properties: 'data', 'my-read-only-bytes', and 'write-only-bytes', all of which are optional. Each of these properties can either be a string or an object. The object version includes an 'Include' property, which is an object that contains a 'href' property of type string. No additional properties are allowed at the root level.
