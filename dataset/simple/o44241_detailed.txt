The schema describes available events to bind actions to for the controller gamepad. It is an object type. The main properties of the schema are: 'onDown' is a string or an array of strings describing the action or array of actions uid to trigger on button down event. 'onHold' is a string or an array of strings describing the action or array of actions uid to trigger on button hold event. 'onUp' is a string or an array of strings describing the action or array of actions uid to trigger on button up event. No additional properties are allowed at the root level.
