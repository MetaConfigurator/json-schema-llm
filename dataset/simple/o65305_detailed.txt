The schema describes a PayPal account configuration. It is an object type. The main properties of the schema are: 'currency' is a string that represents the currency you accept payments in, with a default value of 'USD'. 'enabled' is a boolean that indicates if PayPal IPN processing is enabled, with a default value of false. 'environment' is a string that specifies the PayPal environment type, either sandbox or production, with a default value of 'production'. 'hosted_button_id' is a string that represents the hosted button ID of the configured button, with a default value of 'hosted_button_id'. 'receiver_id' is a string that represents the ID of your PayPal account, with a default value of 'receiver_id'. No additional properties are allowed at the root level.
