The schema describes a list of available Open Badges. It is an object type. The main properties of the schema are: 'badgelist' is an array containing badge elements. It is required and contains items that are objects. Each badge object is required to have a 'loc' property. The 'loc' property is a string representing the location URI of a badge assertion, formatted as a URI. The badge object may also have an optional 'priority' property, which is a number ranging from 0.0 to 1.0. No additional properties are allowed at the root level.
