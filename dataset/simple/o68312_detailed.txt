The schema describes parameters accepted by Improve Digital adapter. It is an object type. The main properties of the schema are: 'placementId' is an integer that specifies an ID which identifies this placement of the impression, optional, with a minimum value of 1. 'publisherId' is an integer that specifies an ID which identifies the publisher, required when using a placementKey, with a minimum value of 1. 'placementKey' is a string that specifies a unique name which identifies this placement of the impression, optional, and it must be used with publisherId. 'keyValues' is an object that contains one or more key-value pairings for key-value targeting, optional. 'size' is an object that specifies the placement size, which includes 'w' (integer) and 'h' (integer) as required properties. The schema has oneOf constraints which require either 'placementId' or both 'publisherId' and 'placementKey' to be present. No additional properties are allowed at the root level.
