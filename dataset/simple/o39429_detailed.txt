The schema describes creating a version request. It is an object type. The main properties of the schema are: 'commits' is an array that contains strings, describing a list of commit ID's that will be used to find the parent commit. It is optional. 'message' is a string that describes the commit message. It is optional. 'commitId' is a string that represents the ID of the new commit. It is optional. No additional properties are allowed at the root level.
