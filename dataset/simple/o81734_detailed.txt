The schema describes a person or entity with related affiliations. It is an object type. The main properties of the schema are: 'club' is an objectLink representing the club's city and name; 'employer' is an objectLink representing the employer's name and type; 'firstName' is a string indicating the person's first name; 'organization' is an objectLink referencing an organization registry; 'school' is an objectLink referencing a school registry. No additional properties are allowed at the root level.
