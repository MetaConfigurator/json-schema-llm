The schema describes a collection of messages. It is an object type. The main properties of the schema are: 'messages' is an array that contains objects representing individual messages. Each message object is required to have the properties: 'messageId' (an integer), 'receivedDate' (an integer), 'senderEmail' (a string), 'recipients' (a string), 'subject' (a string), and 'hasAttachments' (a boolean). No additional properties are allowed at the root level.
