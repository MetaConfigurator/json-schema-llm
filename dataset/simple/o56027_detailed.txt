The schema describes a data structure for managing projects with associated persons and sponsors. It is an object type. The main properties of the schema are: 'id' is an integer or null that must be greater than or equal to 0; 'typeCode' is a required integer; 'sourceSystem' is a required string with a maximum length of 20; 'sourceIdentifier' is a required string with a maximum length of 50; 'sourceStatus' is a string or null with a maximum length of 75; 'persons' is an array or null containing objects, where each object requires 'personId' (string, max length 40), 'sourcePersonType' (string, max length 20), and 'roleCode' (string, max length 50); 'sponsors' is an array or null containing objects, where each object requires 'sourceSystem' (string, max length 20) and 'sourceIdentifier' (string, max length 50); 'startDate' is a string or null; 'endDate' is a string or null; 'title' is a string or null with a maximum length of 2000; 'sponsorCode' is a string or null with a maximum length of 6; 'sponsorName' is a string or null with a maximum length of 200. No additional properties are allowed at the root level.
