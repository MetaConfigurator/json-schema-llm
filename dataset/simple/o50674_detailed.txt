The schema describes a counter type. It is an object type. The main properties of the schema are: 'type' is a string required property that must have a value of 'counter'. 'min' is an integer or null optional property with a default value of null. 'max' is an integer or null optional property with a default value of null. 'step' is an integer optional property with a default value of 1. 'default' is an integer optional property with a default value of 0. No additional properties are allowed at the root level.
