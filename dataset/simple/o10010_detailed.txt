The schema describes an OAuth bearer token. It is an object type. The main properties of the schema are: 'accessToken' is a string with a maximum length of 100, optional. 'clientId' is a string that serves as a MongoDB object ID for the user, with a minimum and maximum length of 12, and must match the pattern '[0-9a-fA-F]+', optional. 'expirationTime' is a number, optional. 'refreshToken' is a string with a maximum length of 100, optional. 'scope' is a string, optional. No additional properties are allowed at the root level.
