The schema describes additional options for templates. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: '$schema' is a string optional. 'id' is a string required. 'addname' is a string optional. 'creatematch' is a string optional, described as 'If the result of the file appears mcgen line with a predetermined pattern that are created with the file. Otherwise, no.'. 'ext' is a string required. 'description' is a string optional. 'vars' is an array optional, with items that are objects containing properties: 'name' is a string optional, 'description' is a string optional, and 'value' is optional and can be of any type. No additional properties are allowed at the root level.
