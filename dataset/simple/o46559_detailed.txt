The schema describes the properties of a Loop Animation. It is an object type. The main properties of the schema are: 'style' is a string that specifies the Loop Animation style, which is required. It must be one of the following values: 'vibrate', 'blink', 'wiggle', 'path', 'sprite'. 'repeat' is an integer that represents how many times the animation should repeat, also required, with a default value of 1. No additional properties are allowed at the root level.
