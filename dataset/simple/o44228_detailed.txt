The schema describes an action property. It is an object type. The main properties of the schema are: 'name' is a string that describes the name of the property and is optional. 'operation' is a string that describes the kind of operation you want to apply to this property and is optional. It can be one of the following: 'set', 'add', 'substract', 'multiply', 'divide', 'toggle'. 'value' is a value that can be of any valid JSON type and is optional. No additional properties are allowed at the root level.
