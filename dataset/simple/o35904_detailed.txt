The schema describes a collection of travelling capabilities. It is an object type. It defines several reusable components under 'definitions': 'jumpGateCapability' is an object, this capability allows the usage of jump gates between systems. 'jumpDriveCapability' is an object, this capability allows the usage of jump drives between systems of New Eden. It requires a field 'distanceLimit', which is a number with a minimum value of 0. The main properties of the schema are: 'jumpGate' is an object that references the 'jumpGateCapability' definition. 'jumpDrive' is an object that references the 'jumpDriveCapability' definition. No additional properties are allowed at the root level.
