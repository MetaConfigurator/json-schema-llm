The schema describes a step configuration for a 'say' action. It is an object type. The main properties of the schema are: 'name' is a string and required. 'kind' is a string with an enum value of 'say' and required. 'label' is a string and required. 'title' is a string and required. 'phrase' is a string and required. 'voice' is a string and optional. 'language' is a string and optional. 'loop' is an integer and optional. 'iface' is an object and optional. No additional properties are allowed at the root level.
