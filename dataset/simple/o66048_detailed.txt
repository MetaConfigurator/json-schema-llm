The schema describes an Access Token. It is an object type. The main properties of the schema are: 'scope' is a string that describes scopes expressed in the form of resource URL endpoints. The value is expressed as a list of space-delimited, case-sensitive strings. 'access_token' is a string that describes the access token issued by PayPal. The access token will expire (see expires_in), after which you’ll have to request a new access token. 'token_type' is a string that describes the type of the token issued as described in OAuth2.0 RFC6749, Section 7.1. The value is case insensitive. 'expires_in' is a number that describes the lifetime in seconds of the access token. No additional properties are allowed at the root level.
