The schema describes the configuration for class extraction from WAR files. It is an object type. The main properties of the schema are: 'disabled' is a boolean that indicates if true, the extraction of classes from WAR files is disabled altogether, with a default value of false. 'scan' is an array of objects, each containing: 'archive', which is a string describing the regex of the name of the archive to match; and 'prefixes', which is an array of strings describing the list of path prefixes to compose the 'JAR view' from. 'scan' has a default value of an array containing an object where 'archive' is '.*' and 'prefixes' is an array containing '/WEB-INF/classes/'. No additional properties are allowed at the root level.
