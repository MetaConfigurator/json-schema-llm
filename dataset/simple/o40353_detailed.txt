The schema describes a JSON Web Key (JWK) structure. It is an object type. The main properties of the schema are: 'kty' is a string and must be one of the allowed values: 'RSA'. 'e' is a string and must be one of the allowed values: 'AQAB'. 'n' is a string that must match the pattern '^[-_=0-9A-Za-z]+$'. All properties are required. No additional properties are allowed at the root level.
