The schema describes a MWF rating component's schema definition. It is an object type. The main properties of the schema are: 'config' is an object that contains several properties. It is required and defines the following: 'communityRating' is a number that has a minimum value of 0 and a maximum value of 5. 'disabled' is a boolean that is optional. 'readOnly' is a boolean that is optional. 'userRating' is a number that has a minimum value of 0 and a maximum value of 5. No additional properties are allowed at the root level.
