The schema describes the usage of available hours by consulate. It is an object type. The main properties of the schema are: 'consulate' is a string the consular level for the number of events. It is required. 'percent' is a number the percentage utilisation (between 0 and 100). It is required and must be between 0 and 100. 'period' is a string representing the period described by each data point. It is required and can have one of the following values: 'second', 'minute', 'hour', 'day', 'week', 'month', 'year'. No additional properties are allowed at the root level.
