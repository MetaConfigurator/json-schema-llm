The schema describes a team. It is an object type. The main properties of the schema are: 'homeCity' is a string that describes the name of the city to which this team belongs. 'id' is a string that describes a three-letter code that identifies the team id, with a minimum and maximum length of 3. 'name' is a string that describes the name of the team. 'stadium' is a string that describes the name of the stadium. No additional properties are allowed at the root level.
