The schema describes an object representing a webhook configuration. It is an object type. The main properties of the schema are: 'id' is a string that must match the pattern '^[A-Fa-f\d]{24}$'. 'webhookId' is a string that must match the pattern '^[A-Fa-f\d]{24}$'. 'applicationId' is a string that must match the pattern '^[A-Fa-f\d]{24}$'. 'creationDate' is a string in date-time format. 'lastUpdated' is a string in date-time format. 'name' is a string with a minimum length of 1 and a maximum length of 255. 'token' is a string. No additional properties are allowed at the root level.
