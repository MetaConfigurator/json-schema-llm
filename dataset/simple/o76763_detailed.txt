The schema describes a user profile. It is an object type. The main properties of the schema are: 'description' is a string or null required field. 'followers' is an integer required field. 'following' is an integer required field. 'isVerified' is a boolean required field. 'name' is a string required field. 'userId' is a string required field. 'username' is a string required field, it describes the last segment of what Instagram calls a "vanity url". 'posts' is an integer required field. No additional properties are allowed at the root level.
