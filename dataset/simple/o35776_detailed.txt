The schema describes a Virtual Machine where the service is going to be deployed. It is an object type. The main properties of the schema are: '$schema-location' is a string that is required and must be one of the allowed values: 'Machine-Schema.json'. 'id' is a string that is required. 'arch' is a string that is optional and has a default value of ''. 'cpu-cores' is a string that is optional and has a default value of ''. 'memory' is a string that is optional and has a default value of ''. No additional properties are allowed at the root level.
