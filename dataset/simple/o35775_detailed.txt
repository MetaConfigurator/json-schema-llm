The schema describes the interface of a Juju Charm. It is an object type. The main properties of the schema are: '$schema-location' is a string that must be one of the enum values: 'Interface-Schema.json'. 'id' is a string that is required to associate an interface object with a charm object. 'name' is a string with a default value of an empty string and is also required. No additional properties are allowed at the root level.
