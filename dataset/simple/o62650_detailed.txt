The schema describes ASN.1 Types Library General Information. It is an object type. It defines several reusable components under 'definitions': 'nonEmptyString' is a string, which must have a minimum length of 1. The main properties of the schema are: 'name' is a non-empty string and is required. 'description' is a string and is optional. 'vendor' is a string and is optional. 'version' is a string and is optional. 'license' is a string and is optional. No additional properties are allowed at the root level.
