The schema describes an array of sort configurations for search results. It is an array type. The main properties of the schema are: Each item in the array is an object that contains the following properties: 'field' is a string that is required and represents the field to sort by. 'sort_type' is an optional string that can be either 'ASC' or 'DESC', indicating the type of sort. 'sort_flag' is an optional string that can only be 'VERSION', representing additional sort criteria. No additional properties are allowed at the root level.
