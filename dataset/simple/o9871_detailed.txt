The schema describes the configuration for SSL credentials. It is an object type. The main properties of the schema are: 'sslKey' is a string with a minimum length of 50 and a maximum length of 32767, optional. 'sslCert' is a string with a minimum length of 50 and a maximum length of 32767, optional. 'sslBundle' is a string with a minimum length of 50 and a maximum length of 32767, optional. 'domainName' is a string with a minimum length of 3 and a maximum length of 1024, required. 'version' is either a string with a minimum length of 1 and a maximum length of 255 or null, optional. No additional properties are allowed at the root level.
