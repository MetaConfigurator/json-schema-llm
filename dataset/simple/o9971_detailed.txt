The schema describes a data structure for a grid format configuration. It is an object type. The main properties of the schema are: 'format' is a string that is required and must have the value 'grid-1.0.0'. 'layers' is an array that is required. Each item in the array can either be null or an object that must have an integer 'spriteId' property, along with optional integer 'targetX' and 'targetY' properties. No additional properties are allowed at the root level.
