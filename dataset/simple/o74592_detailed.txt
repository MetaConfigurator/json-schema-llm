The schema describes dashboards. It is an object type. The main properties of the schema are: 'name' is a string required field. 'layout' is an object optional field which contains properties that match the pattern '^\w+$'. Each property under 'layout' is an object that must have the required fields 'col', 'row', and 'size_x', all of which are numbers. No additional properties are allowed at the root level.
