The schema describes an object representing an interactive tool event. It is an object type. The main properties of the schema are: 'active' is a boolean indicating if the mouse button is down and is required. 'char' is a property that can be either null or a string of maximum length 20, and it is optional. 'element' is a string representing the id attribute, constrained by the pattern '^[a-f][a-f0-9]{31}$', and it is optional. 'time' is an integer representing milliseconds per Date.getTime and is required. 'tool' is a string that must be one of the following values: 'Pen', 'Hand', 'Eraser', and it is required. 'x' is a number indicating the x-coordinate in SVG coordinates and is optional. 'y' is a number indicating the y-coordinate in SVG coordinates and is optional. No additional properties are allowed at the root level.
