The schema describes a confidence interval effect for a performance table entry of absolute data. It is an object type. The main properties of the schema are: 'type' is a string required property that must be 'exact'. 'value' is a number required property. 'input' is an object required property. It contains required fields: 'value', 'lowerBound', and 'upperBound', with no additional properties allowed. 'value' within 'input' is a number. 'lowerBound' must be either a number or the string 'NE'. 'upperBound' must be either a number or the string 'NE'. 'scale' is an optional string property that can be either 'percentage' or 'decimal'. No additional properties are allowed at the root level.
