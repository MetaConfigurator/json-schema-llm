The schema describes a collection of products. It is an array type. The main properties of the schema are: Each item is an object with the following properties: 'brand' is a string, required; 'description' is a string, required; 'id' is a string that must match the pattern '^[\d]+$', required; 'image' is a string, optional; 'name' is a string, required. No additional properties are allowed at the root level.
