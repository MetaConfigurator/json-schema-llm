The schema describes IHEC Data Hub Analysis. It is an object type. The main properties of the schema are: 'analysis_group' is a string that indicates the group that ran the bioformatics analysis to produce this dataset tracks and is required. 'alignment_software' is a string that indicates the name of the software used for mapping and is required. 'alignment_software_version' is a string that indicates the version of the software used for mapping and is required. 'analysis_software' is a string that indicates the name of the software used for determining signal (read density) and is required. 'analysis_software_version' is a string that indicates the version of the software used for determining signal (read density) and is required. No additional properties are allowed at the root level.
