The schema describes a filter for points based on the value of a specified attribute. It is an object type. The main properties of the schema are: 'field' is a string that describes the name of the field that is used for the filter and is required. 'mode' is a string that defines if values should be included or excluded and is required; allowed values are 'exclude' and 'include'. 'type' is a string that indicates the filter type and is required; the only allowed value is 'pointCloudValueFilter'. 'values' is an array that provides a list of values and is required; the items in the array are numbers. No additional properties are allowed at the root level.
