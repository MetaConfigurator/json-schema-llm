The schema describes a request to update account settings for Twitter users. It is an object type. The main properties of the schema are: 'end_sleep_time' is a boolean indicating the hour that sleep time should end if it is enabled, with expected values in ISO8601 format; 'lang' is a string specifying the language for Twitter to render for the user, represented by a two-letter ISO 639-1 code; 'sleep_time_enabled' is a boolean that, when set to true, will enable sleep time for the user; 'start_sleep_time' is an integer representing the hour that sleep time should begin if it is enabled, also in ISO8601 format; 'time_zone' is a string that must be one of the Rails TimeZone names; 'trend_location_woeid' is a string for the Yahoo! Where On Earth ID as the user’s default trend location, with global information available using the WOEID 1. No additional properties are allowed at the root level.
