The schema describes contact information. It is an object type. The main properties of the schema are: 'address1' is a string (optional), 'city' is a string (optional), 'contact_name' is a string (required), 'email' is a string (required), 'id' is an integer (required), 'personal_data_removed' is a boolean (optional), 'phone_number' is a string (optional), 'postcode' is a string (optional), 'supplier_id' is an integer (optional). No additional properties are allowed at the root level.
