The schema describes the MCDA beta distribution for the performance table entry of absolute data. It is an object type. The main properties of the schema are: 'type' is a string that is required and must be one of the following values: 'dbeta'. 'parameters' is an object that is required and must not have additional properties. It requires 'alpha' and 'beta', which are both numbers. 'input' is an object that is optional and requires 'events' and 'sampleSize', both of which are numbers. No additional properties are allowed at the root level.
