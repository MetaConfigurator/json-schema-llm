The schema describes options for Bazel Builder. It is an object type. The main properties of the schema are: 'bazelCommand' is a string representing common commands supported by Bazel, required. Valid values are 'run', 'build', 'test'. 'leaveBazelFilesOnDisk' is a boolean that determines if Bazel files should be left on disk after running the command, optional, with a default value of false. 'targetLabel' is a string indicating the target to be executed under Bazel, required. 'watch' is a boolean indicating if the filesystem should be watched using ibazel, optional, with a default value of false. No additional properties are allowed at the root level.
