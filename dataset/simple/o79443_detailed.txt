The schema describes an object structure. It is an object type. It defines several reusable components under 'definitions': 'httpUri' is a string, validated as a URI format and must match the pattern '^https?://'. The main properties of the schema are: 'name' is a string and required. 'uri' is a string that references the 'httpUri' definition and is required. 'index' is an integer, optional, with a minimum value of 0. No additional properties are allowed at the root level.
