The schema describes a single namespace value within a BEL Namespace. It is an object type. The main properties of the schema are: 'namespace_value' is an object which is required. It does not allow additional properties and requires 'type', 'identifier', 'name', 'title', and 'species' properties. The 'type' property is a string, 'identifier' is a string, 'name' is a string, 'title' can be a null or string, 'species' can be a null or string, and 'match_text' is a string. No additional properties are allowed at the root level.
