The schema describes the value returned by GET /{db}/_changes. It is an object type. The main properties of the schema are: 'last_seq' is a number and required. 'results' is an array of objects, each containing required properties 'changes', 'id', and 'seq'. The 'changes' property is an array of objects, each having a required property 'rev' of type string. No additional properties are allowed at the root level.
