The schema describes a partial update payload for a note. It is an object type. The main properties of the schema are: 'body' is a string that describes the note's body text. It is optional. 'tags' is an array that contains the names of the tags the note is associated with. It is optional, and each item in the array must be a string. The array must have unique items. The schema requires a minimum of 1 property. No additional properties are allowed at the root level.
