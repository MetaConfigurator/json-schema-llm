The schema describes the structure for browsing streams. It is an object type. The main properties of the schema are: 'p' is a string representing the pattern to search for. It is required. 't' is a string representing the pattern type. Valid values are 's' (startsWith), 'e' (endsWith), or null (match any stream). It is optional. 'm' is an integer representing the maximum number of results to return. It must be between 1 and 100. It is required. 'c' is a string representing a continuation token used for paging. It is optional. No additional properties are allowed at the root level.
