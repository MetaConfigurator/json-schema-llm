The schema describes a competency object. It is an object type. The main properties of the schema are: 'id' is a string representing the competency uuid, optional. 'name' is a string representing the competency name, required, with a minimum length of 1. 'description' is either a string or null representing the competency description, optional. 'parent' is an object or null representing the competency parent, optional, containing an 'id' property which is a string. 'scale' is an object representing the associated scale, optional, containing an 'id' property which is a string. No additional properties are allowed at the root level.
