The schema describes an instance of a hotspot geometry offset. It is an object type. The main properties of the schema are: 'x' is a number representing the offset on x in word units. It is optional with a default value of 0. 'y' is a number representing the offset on y in word units. It is optional with a default value of 0. 'z' is a number representing the offset on z in word units. It is optional with a default value of 0. No additional properties are allowed at the root level.
