The schema describes the configuration for getting feedback related to emoting in a Delivery Pipeline. It is an object type. The main properties of the schema are: 'app-region' is a string that describes the bluemix region and is required. 'app-organization' is a string that describes the bluemix org and is required. 'app-space' is a string that describes the bluemix space and is required. No additional properties are allowed at the root level.
