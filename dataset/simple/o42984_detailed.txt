The schema describes a specification for the asset.json file for rocket fonts. It is an object type. The main properties of the schema are: 'asset_type' is a string matching the pattern 'rocket_font', required. 'font_file' is a string describing the path to a ttf or otf font file, required. 'rcss_name' is a string describing the name of the font family to use in RCSS, extracted from the file by default, optional. 'font_style' is a string matching the pattern 'normal|italic', describing the overridden font style, required if 'rcss_name' is supplied, optional. 'font_weight' is a string matching the pattern 'normal|bold', describing the overridden font weight, required if 'rcss_name' is supplied, optional. No additional properties are allowed at the root level.
