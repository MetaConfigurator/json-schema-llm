The schema describes Health for an instance. It is an object type. The main properties of the schema are: 'srvcId' is a string Service ID, normally same as Module ID and is required. 'instId' is a string Instance ID, unique on node and is required. 'healthMessage' is a string Health message and is required. 'healthStatus' is a boolean Whether healthy and is required. No additional properties are allowed at the root level.
