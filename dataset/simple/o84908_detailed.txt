The schema describes a Datasift Webhook Data. It is an object type. The main properties of the schema are: 'id' is a string which is optional. 'hash' is a string which is optional. 'hash_type' is a string which is optional. 'count' is a number which is optional. 'delivered_at' is a string formatted as date-time which is optional. 'interactions' is an array of objects which is optional, where each item is of type 'org.apache.streams.datasift.Datasift'. No additional properties are allowed at the root level.
