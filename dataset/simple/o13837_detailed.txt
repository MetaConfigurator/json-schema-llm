The schema describes an address structure. It is an object type. The main properties of the schema are: 'city' is a string the city of origin, required. 'country' is a string the country of origin, optional. 'county' is a string the state/county the user is in, optional. 'line1' is a string the first line of the address, required. 'line2' is a string the second line of the address, optional. 'postcode' is a string user's zipcode/postcode (or similar), required. No additional properties are allowed at the root level.
