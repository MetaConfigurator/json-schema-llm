The schema describes version specific attributes of a stack. It is an object type. The main properties of the schema are: 'createTimestamp' is a string , 'projectIteration' is a integer , 'projectStep' is a integer , 'snapshotRootPath' is a string , 'stackResolutionX' is a number , 'stackResolutionY' is a number , 'stackResolutionZ' is a number , 'versionNotes' is a string . No additional properties are allowed at the root level.
