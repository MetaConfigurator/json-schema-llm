The schema describes EPUB extension metadata. It is an object type. The main properties of the schema are: 'rendition' is an object that contains properties detailing the presentation of the resource. It is optional. The properties of 'rendition' are: 'layout' is a string that hints how the layout of the resource should be presented, with allowed values 'fixed' and 'reflowable'. 'orientation' is a string that suggests the orientation for the device when displaying the linked resource, with allowed values 'auto', 'landscape', and 'portrait'. 'overflow' is a string that suggests the method for handling overflow while displaying the linked resource, with allowed values 'auto', 'paginated', 'scrolled', and 'scrolled-continuous'. 'spread' is a string that indicates the condition to be met for the linked resource to be rendered within a synthetic spread, with allowed values 'auto', 'both', 'none', and 'landscape'. No additional properties are allowed at the root level.
