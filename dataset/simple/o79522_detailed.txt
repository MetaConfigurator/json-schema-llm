The schema describes a package of information about an application, e.g. a list of the well known fields and user defined items that should be returned. It is an object type. The main properties of the schema are: 'id' is an integer representing the id of this view type and is required. 'label' is a string providing a label for the view type and is required. 'defaultFlag' is a boolean indicating true if this view is the default view, with at most one view allowed to have this set, and is required. No additional properties are allowed at the root level.
