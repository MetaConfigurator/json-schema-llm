The schema describes a Google Analytics Content Collector Query. It is an object type. The main properties of the schema are: 'dimensions' is an array of strings, optional. 'filters' is an array of strings, optional. 'id' is a string that must match the pattern '^[a-z0-9:]+$', optional. 'maxResults' is an integer, optional. 'metrics' is either a string or an array of strings, and it is required. 'sort' is an array of strings, optional. No additional properties are allowed at the root level.
