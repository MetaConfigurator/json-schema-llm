The schema describes a search layer field. It is an object type. The main properties of the schema are: 'exactMatch' is a boolean defining whether or not the field is an exact match. 'name' is a string defining the field name. 'type' is a string defining the field type, with allowed values: 'esriFieldTypeBlob', 'esriFieldTypeDate', 'esriFieldTypeDouble', 'esriFieldTypeGUID', 'esriFieldTypeGeometry', 'esriFieldTypeGlobalID', 'esriFieldTypeInteger', 'esriFieldTypeLong', 'esriFieldTypeOID', 'esriFieldTypeRaster', 'esriFieldTypeSingle', 'esriFieldTypeSmallInteger', 'esriFieldTypeString', 'esriFieldTypeXML'. No additional properties are allowed at the root level.
