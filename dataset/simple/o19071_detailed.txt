The schema describes the structure of an event log. It is an object type. The main properties of the schema are: 'user_id' is a string and is required. 'events' is an array of objects, and is required, where each object has properties: 'type' which is an integer, 'show_id' which is a string, and 'time' which is a number. 'is_test' is an integer with allowed values 0 and 1, and is required. 'cid' is a string and is optional. No additional properties are allowed at the root level.
