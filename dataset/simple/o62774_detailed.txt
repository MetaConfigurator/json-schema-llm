The schema describes a list atom. It is an object type. The main properties of the schema are: 'modifier' is a string that describes a pattern modifier. It is required and can be one of the following values: '', 'mix'. 'listItems' is an array that contains objects. Each object must have 'icon', which is a string that can be one of 'arrow', 'check', 'nitro', and 'text', which is a string. The array must have between 1 and 3 items. Both 'modifier' and 'listItems' are required fields. No additional properties are allowed at the root level.
