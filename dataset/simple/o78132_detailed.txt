The schema describes a search result set. It is an object type. The main properties of the schema are: 'total' is an integer representing the total number of results returned by the search. It is optional. 'start' is an integer representing the starting index of this set of results within the total set of results. It is optional. 'rows' is an array of objects representing the results. Each object in the array contains: 'url' is a string representing the URL to the object on the server (e.g. https://api.chef.io/organizations/foo/nodes/bar). It is optional and must be in URI format. 'data' is a string representing the desired data where each key corresponds to a key from the POST. It is optional and allows additional properties, which represent the value of the attribute on the object (or null if the attribute was not on the object). No additional properties are allowed at the root level.
