The schema describes definitions that are used by several descriptors. It is an object type. It defines several reusable components under 'definitions': 'memory_units' is an enumeration, representing allowed memory unit values: 'B', 'kB', 'KiB', 'MB', 'MiB', 'GB', 'GiB', 'TB', 'TiB', 'PT', 'PiT'. 'bandwidth_units' is an enumeration, representing allowed bandwidth unit values: 'bps', 'kbps', 'Mbps', 'Gbps', 'Tbps'. 'images_formats' is an enumeration, representing allowed image format values: 'raw', 'vhd', 'vmdk', 'vdi', 'iso', 'qcow2', 'docker', 'ova', 'ovf', 'bare'. 'interfaces' is an enumeration, representing allowed interface values: 'interface'. The main properties of the schema are: 'memory_units' is a reference to the 'memory_units' definition, required. 'bandwidth_units' is a reference to the 'bandwidth_units' definition, required. 'images_formats' is a reference to the 'images_formats' definition, required. 'interfaces' is a reference to the 'interfaces' definition, required. No additional properties are allowed at the root level.
