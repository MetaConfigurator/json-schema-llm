The schema describes AWS security credential configuration. It is an object type. The main properties of the schema are: 'accessKey' is a string or null representing the AWS access key for the API, required. 'accessSecret' is a string or null representing the AWS access secret for the API, required. 'credentialsFile' is a string or null representing the AWS credentials file for the API, required. 'credentialsProfile' is a string or null representing the profile to use within the credentials file, required. 'securityToken' is a string or null representing the Session Token to use when access credentials are from temporary STS grants, optional. No additional properties are allowed at the root level.
