The schema describes a plugin configuration. It is an object type. The main properties of the schema are: 'actions' is an array A list of actions associated to this plugin. It is optional and items must be strings. 'data' is an object The eventual data associated to the plugin, specific to each plugin. It is optional. 'events' is an object Events to react to, specific to each plugin. It is optional. 'options' is an object The options of this instance, specific and different for each plugin. It is optional. No additional properties are allowed at the root level.
