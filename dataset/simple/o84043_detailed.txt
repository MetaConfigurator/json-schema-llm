The schema describes the configuration for a web publishing tool. It is an object type. The main properties of the schema are: 'scripts' is an object that represents a hash of script names (keys) and shell commands (values) to execute. It is required and does not allow additional properties. The keys must match the regex pattern '^([a-z0-9-])+$' and the values are strings. 'vars' is an object that represents a hash of variable names (keys) and values (values). It is optional and does not allow additional properties. The keys must match the regex pattern '^[a-zA-Z_][a-zA-Z_0-9]*$' and the values are strings. 'web-dir' is a string that specifies the relative directory containing files to be published as a website. It is required. No additional properties are allowed at the root level.
