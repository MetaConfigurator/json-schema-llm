The schema describes all the Fcc related Information of SAS. It is an object type. The main properties of the schema are: 'certificationConditions' is a string human-readable string or reference annotating the certification, required. 'certificationDate' is a string date of certification, in format YYYY-MM-DD, required. 'certificationExpiration' is a string date of certification expiration, in format YYYY-MM-DD, required. 'certificationId' is a string any FCC-issued certification ID, required. 'frn' is a string the FRN of the certified entity, if applicable. For a device, shall not be included, required. 'sasPhase' is a string if this is a SAS information object, defines the Phase (1 or 2) of certification, required, with allowed values '1' and '2'. No additional properties are allowed at the root level.
