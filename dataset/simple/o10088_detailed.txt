The schema describes a resource registration format. It is an object type. The main properties of the schema are: 'schemas' is an array of objects that represents multiple schema registrations. Each object in the array has the following properties: 'component' is a string that must match the pattern '^[a-z_]$', and is optional; 'group' is a string that must match the pattern '^[a-z_]$', and is optional; 'path' is a string that must match the pattern '^[a-zA-Z0-9_\-\./]+$', and is optional; 'schema' is an object, and is optional. No additional properties are allowed at the root level.
