The schema describes settings for an Espruino project run from VS Code. It is an object type. The main properties of the schema are: 'port' is a string which specifies the COM port which the device will connect via, required; 'baudrate' is a number which indicates the baudrate for the connection to use when connecting to the port, required; 'autoConnect' is a boolean which determines whether the specified port should be connected to when VSCode activates, required; 'resetBeforeSend' is a boolean which indicates whether to reset the device before sending the new file, required. No additional properties are allowed at the root level.
