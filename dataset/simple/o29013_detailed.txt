The schema describes a message format. It is an object type. The main properties of the schema are: 'message.sub-id' is a string matching the pattern '[0-9a-fA-F]{8}-([0-9a-fA-F]{4}-){3}[0-9a-fA-F]{12}', required. 'message.topic' is a string matching the pattern '^/[a-z\-]+$', required. 'message.since' is a number, required. 'message.snapshot' is a boolean, required. 'message.filter' is a string with a maximum length of 137, matching the pattern '^[\w\-\.#@\*\[\]\?]+$', required. No additional properties are allowed at the root level.
