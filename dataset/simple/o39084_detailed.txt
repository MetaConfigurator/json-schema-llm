The schema describes replace-in-file options. It is an object type. The main properties of the schema are: '$schema' is a string optional. '$tool' is a string required; it must be set to 'replace-in-file'. 'allowEmptyPaths' is a boolean optional; it defaults to false. 'encoding' is a string optional; it defaults to 'utf-8'. 'files' is a string or an array of strings required; it specifies the file(s) (glob) to process. 'from' is a string or an array of strings or RegExp required; it specifies RegExp(s) to find from files. 'to' is a string or an array of strings required; it specifies multiple replacements with same/different strings. No additional properties are allowed at the root level.
