The schema describes a configuration for a reject step. It is an object type. The main properties of the schema are: 'name' is a string and is required. 'kind' is an enum with the allowed value 'reject' and is required. 'label' is a string and is required. 'title' is a string and is required. 'reason' is an enum with the allowed values 'busy' and 'rejected' and is optional. 'iface' is an object and is optional. No additional properties are allowed at the root level.
