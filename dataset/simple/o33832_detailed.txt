The schema describes edges in a Guice injector graph. It is an object type. The main properties of the schema are: 'fromId' is a string required field representing the starting point of the edge. 'toId' is a string required field representing the endpoint of the edge. 'type' is a string required field with an enum of allowed values: 'binding', 'dependency'. 'bindingType' is a string optional field representing the type of binding. No additional properties are allowed at the root level.
