The schema describes a specification for custom jars that can be uploaded by users. It is an object type. The main properties of the schema are: 'name' is a string that represents the unique name for a custom jar and is required. 'description' is a string that provides the description for a custom jar and is optional. 'version' is a string that indicates the custom jar version and is required. 'author' is a string that specifies the custom jar author and is optional. 'license' is a string that denotes the software license type for a custom jar and is optional. 'options' is an object that represents the JSON schema for some optional parameters and is also optional. No additional properties are allowed at the root level.
