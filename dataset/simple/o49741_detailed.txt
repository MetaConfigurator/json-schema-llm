The schema describes the followers' IDs response from Twitter's API. It is an object type. The main properties of the schema are: 'ids' is an array containing integers representing follower IDs. It is required. 'next_cursor' is an optional integer representing the next cursor for pagination. 'next_cursor_str' is an optional string representing the next cursor as a string. 'previous_cursor' is an optional integer representing the previous cursor for pagination. 'previous_cursor_str' is an optional string representing the previous cursor as a string. No additional properties are allowed at the root level.
