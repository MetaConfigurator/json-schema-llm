The schema describes a structured data model for testing JAXB annotations. It is an object type. It defines several reusable components under 'definitions': 'Middle1' is an object, containing properties 'middle1Attr', 'middle2Attr', 'bottomAttr', and 'value', all of which are strings. No additional properties are allowed at this level. The main properties of the schema are: 'top' is an object that contains a required property 'middle1' which references the 'Middle1' definition. No additional properties are allowed at the root level.
