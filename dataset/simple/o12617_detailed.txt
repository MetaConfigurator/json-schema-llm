The schema describes an object representing a resource. It is an object type. The main properties of the schema are: 'created' is a string formatted as an ISO 8601 timestamp, required. 'description' is a string, required. 'id' is an integer, required. 'modified' is a string formatted as an ISO 8601 timestamp, required. 'name' is a string, required. 'resource_uri' is a string, required. No additional properties are allowed at the root level.
