The schema describes a grading job message to be run. It is an object type. The main properties of the schema are: 'jobId' is a string or integer, representing the unique ID for this job; it is required. 'image' is a string, representing the image that this job will be executed in; it is required. 'entrypoint' is a string, representing the script that will be run to start this job; it is required. 'timeout' is an integer, representing the number of seconds after which the grading job will timeout; it is optional. 's3Bucket' is a string, representing the AWS S3 bucket containing this job's files; it is required. 's3RootKey' is a string, representing the root key for the job's files; it is required. No additional properties are allowed at the root level.
