The schema describes an acquisition source. It is an object type. The main properties of the schema are: 'acquisition_source' is an object that describes the source of acquisition (former 541). It has optional properties: 'date' is a string representing the date of acquisition, 'email' is a string for the submitter's email, 'method' is an enum with allowed values: 'OAI', 'batchupload', 'submission', 'source' is a string for the source of acquisition (e.g. ID of the user), and 'submission_number' is an integer for the internal submission number. No additional properties are allowed at the root level.
