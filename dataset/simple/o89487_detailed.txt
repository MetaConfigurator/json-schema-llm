The schema describes an object with creation information and an array of entries. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'created' is a string that is required. 'entries' is an array of objects, each of which is required and contains the following properties: 'permanent' is a boolean that is required. 'rule' is a string that is optional. 'source' is a string that is optional. 'target' is a string that is required. No additional properties are allowed at the root level.
