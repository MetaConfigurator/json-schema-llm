The schema describes a single athlete's performance in a single track event. It is an object type. The main properties of the schema are: 'event' is a string representing the standard event type short name, required, with allowed values being '60', '60H', '100', '110H', '200', '200H', '400', '400H', '800', '1500', 'mile', '3000', '3000SC', '5000', '10000'. 'performance' is a number or string representing the time taken to complete the event, required, with a minimum value of 0 and a minimum length of 1. 'laps' is an array representing an ordered list of lap times or 'null' if missing, optional, with a minimum of 1 item and a maximum of 25 items, where each item can be null, a number, or a string. No additional properties are allowed at the root level.
