The schema describes a structure for the output of the evaluate method in the Scenario Reasoner. It is an object type. The main properties of the schema are: 'result' is an object that describes the result of the method evaluate. It is required and may contain additional properties where each property is an object with a required 'value'. 'id' is a value identifying this response, equal to the id of the request or null. It is required and optional in nature. 'error' is a null type, indicating an absence of errors. It is required. 'version' is a string that specifies the version of the response. It is required. No additional properties are allowed at the root level.
