The schema describes options for the WctRunner. It is an object type. The main properties of the schema are: 'wct' is an object that configures the @stryker-mutator/wct-runner. It is optional. It contains properties: 'configFile' is a string that specifies the location of the 'wct.conf.json' file. 'verbose' is a boolean that enables or disables verbose WCT logging, defaulting to false. 'npm' is a boolean that allows the use of WCT with the `--npm` flag, also defaulting to false. No additional properties are allowed at the root level.
