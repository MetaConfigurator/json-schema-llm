The schema describes parameters accepted by the Admixer adapter. It is an object type. The main properties of the schema are: 'zone' is a string Zone ID. It is required and must match the pattern '^([a-fA-F\d\-]{36})$'. 'customFloor' is a number The minimum CPM price in USD. It has a minimum value of 0. 'customParams' is an object User-defined targeting key-value pairs. No additional properties are allowed at the root level.
