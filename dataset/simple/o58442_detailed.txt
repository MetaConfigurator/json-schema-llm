The schema describes App Shell target options for Build Facade. It is an object type. The main properties of the schema are: 'browserTarget' is a string that specifies the target to build. It is required and follows the pattern '^[^:\s]+:[^:\s]+(:[^\s]+)?$'. 'serverTarget' is a string that specifies the server target to use for rendering the app shell. It is required and follows the pattern '^[^:\s]+:[^:\s]+(:[^\s]+)?$'. 'appModuleBundle' is a string that specifies the script that exports the Server AppModule to render. 'route' is a string that specifies the route to render. It is optional and has a default value of '/'. 'inputIndexPath' is a string that specifies the input path for the index.html file. 'outputIndexPath' is a string that specifies the output path of the index.html file. No additional properties are allowed at the root level.
