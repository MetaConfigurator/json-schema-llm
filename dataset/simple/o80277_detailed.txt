The schema describes CSS contributions to package.json. It is an object type. The main properties of the schema are: 'contributes' is an object that contains properties related to CSS contributions. It is optional and may include: 'css.customData' which is an array. It contains relative file paths pointing to JSON files following the custom data format. The items in the array are of type string and represent relative paths to CSS custom data files. No additional properties are allowed at the root level.
