The schema describes the common options that shared by all tasks. It is an object type. It defines several reusable components under 'definitions': 'Timeout' is an integer, this property can be used to specify a maximum amount of time a task may be run, in milliseconds. By default, this value is 24 hours. A value of 0 or -1 means infinite timeout. 'SchedulerOverrides' is an object, which has an optional property 'timeout' that references the 'Timeout' definition. The main properties of the schema are: '_taskTimeout' is a Timeout that specifies the maximum amount of time a task may be run, in milliseconds. 'schedulerOverrides' is a SchedulerOverrides that may contain a timeout property. No additional properties are allowed at the root level.
