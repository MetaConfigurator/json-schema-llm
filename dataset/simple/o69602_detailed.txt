The schema describes options for a MultipleTrueFalse question. It is an object type. The main properties of the schema are: 'text' is a string that provides text to precede the set of statements being given. It is optional. 'trueStatements' is an array that contains a list of true statements for the question. Each item is a string that represents an HTML string. This property is required. 'falseStatements' is an array that contains a list of false statements for the question. Each item is a string that represents an HTML string. This property is required. No additional properties are allowed at the root level.
