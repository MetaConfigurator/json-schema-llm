The schema describes service bindings. It is an object type. The main properties of the schema are: 'ipaddress' is a string and is readonly. 'port' is an integer and is readonly. 'servicename' is a string and is optional. 'svrstate' is a string and is readonly with enumerated values: 'UP', 'DOWN', 'UNKNOWN', 'BUSY', 'OUT OF SERVICE', 'GOING OUT OF SERVICE', 'DOWN WHEN GOING OUT OF SERVICE', 'NS_EMPTY_STR', 'Unknown', 'DISABLED'. 'vservername' is a string and is readonly. No additional properties are allowed at the root level.
