The schema describes a configuration for a Winnow Option. It is an object type. The main properties of the schema are: 'name' is a string optional field. 'type' is a string required field. 'description' is a string optional field. 'images' is an array of objects, where each object contains: 'asset' which is a string required field, and 'type' which is a string optional field. Additional properties are not allowed for items in this array. 'scopes' is an array of strings, where each string must match one of the patterns: 'customer', 'maker', 'operator', 'admin', or 'designer'. No additional properties are allowed at the root level.
