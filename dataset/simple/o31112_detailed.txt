The schema describes a video optimization detection policy binding. It is an object type. The main properties of the schema are: 'activepolicy' is an integer, readonly. 'boundto' is a string, optional. 'gotopriorityexpression' is a string, readonly. 'labelname' is a string, readonly. 'labeltype' is a string, readonly, with allowed values 'vserver' and 'policylabel'. 'name' is a string, optional. 'priority' is an integer, readonly. No additional properties are allowed at the root level.
