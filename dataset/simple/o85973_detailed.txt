The schema describes a named entity. It is an object type. The main properties of the schema are: '_id' is a string that serves as a unique identifier for the concept of the named entity, required. 'name' is a string that represents the actual string of text that was identified as a named entity, required. 'type' is a string that describes what the named entity is, such as 'organization', 'person', or 'location', required. 'score' is a number that represents an optional relevancy for this named entity. No additional properties are allowed at the root level.
