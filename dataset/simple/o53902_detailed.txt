The schema describes a 7/CMX applied configuration request. It is an object type. The main properties of the schema are: 'configId' is a string that identifies the applied configuration and is required. 'reasonPhrase' is a string that provides a human-readable explanation of the cause of an error, and is optional. 'statusCode' is a number that indicates the status code based on HTTP status codes, with a default value of 200. No additional properties are allowed at the root level.
