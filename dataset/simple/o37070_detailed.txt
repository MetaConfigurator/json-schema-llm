The schema describes configuration for instruments. It is an object type. The main properties of the schema are: 'instruments' is an object that must include properties matching any string as keys. Each matching property must be an object with 'destination' and 'mode' as required string properties. 'modes' is an object that must include properties matching the pattern '^mode', where each corresponding value must be an array. Each item in this array must be an object with 'begin' and 'end' as required string properties. No additional properties are allowed at the root level.
