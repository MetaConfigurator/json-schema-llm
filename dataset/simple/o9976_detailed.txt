The schema describes a menu configuration. It is an object type. The main properties of the schema are: 'name' is a string that has a default value sourced from the first command-line argument. 'path' is a string that describes the path to create the component, and it is not visible. 'project' is a string that describes the name of the project and has a default value sourced from the project name. 'proxy' is a boolean that indicates if the component needs to proxy its calls to an API. No additional properties are allowed at the root level.
