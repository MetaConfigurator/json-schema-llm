The schema describes validation parameters accepted by the AdOcean adapter. It is an object type. The main properties of the schema are: 'emiter' is a string that represents the AdOcean emiter, it is required and must match the pattern '.+'. 'masterId' is a string that represents the master's id, it is required and must match the pattern '^[\w.]+$'. 'slaveId' is a string that represents the slave's id, it is required and must match the pattern '^adocean[\w.]+$'. No additional properties are allowed at the root level.
