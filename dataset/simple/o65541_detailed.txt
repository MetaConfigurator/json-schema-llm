The schema describes an array of key-value pairs that represent Apple HealthKit metadata information. It is an array type. The main properties of the schema are: 'items' is an object that contains properties defining each key-value pair. 'key' is a string that describes a predefined or custom metadata key. It is required. 'value' is a oneOf type that can be a string, number, or boolean. It is also required. No additional properties are allowed at the root level.
