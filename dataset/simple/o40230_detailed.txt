The schema describes a user account configuration. It is an object type. The main properties of the schema are: 'email' is a string that must match the pattern '^[_a-z0-9-]+(\.[_a-z0-9-]+)*@[_a-z0-9-]+(\.[_a-z0-9-]+)*(\.[a-z]{2,3})$', and is required. 'macAddress' is a string that must match the pattern '([0-9a-fA-F][0-9a-fA-F](:|-)){4}([0-9a-fA-F][0-9a-fA-F])', and is required. 'password' is a string with a minimum length of 6 and a maximum length of 12, and is required. No additional properties are allowed at the root level.
