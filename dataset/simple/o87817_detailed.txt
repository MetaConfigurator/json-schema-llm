The schema describes an Address following the convention of http://microformats.org/wiki/hcard. It is an object type. The main properties of the schema are: 'post-office-box' is a string optional address line that represents a post office box. 'extended-address' is a string optional field for further address details. 'street-address' is a string optional field for the main street address. 'locality' is a string optional field for the city or locality. 'region' is a string optional field for the state or region. 'postal-code' is a string optional field for the postal or zip code. 'country-name' is a string optional field for the country name. Compositional constraints include dependencies where if 'post-office-box' is provided, 'street-address' must also be provided, and if 'extended-address' is provided, 'street-address' must also be present. No additional properties are allowed at the root level.
