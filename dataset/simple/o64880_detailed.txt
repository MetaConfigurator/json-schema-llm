The schema describes generating a new, generic enum definition for the given or default project. It is an object type. The main properties of the schema are: 'lintFix' is a boolean that applies lint fixes after generating the enum when true. It is optional and has a default value of false. 'name' is a string that represents the name of the enum. It is required. 'path' is a string that specifies the path at which to create the enum definition, relative to the current workspace. It is optional and has a format of 'path'. 'project' is a string that denotes the name of the project in which to create the enum. It is optional and defaults to the configured default project for the workspace. No additional properties are allowed at the root level.
