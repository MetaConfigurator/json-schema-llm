The schema describes a SQL configuration. It is an object type. The main properties of the schema are: 'tableName' is a string required field that specifies the name of the SQL table. 'databaseType' is a string required field that must be one of the following values: 'MySQL', 'Postgres', 'SQLite', 'Oracle', or 'MSSQL'. 'createTable' is a boolean optional field that indicates whether to create the table. 'dropTable' is a boolean optional field that indicates whether to drop the table. 'encloseWithBackquotes' is a boolean optional field that specifies whether to enclose identifiers with backquotes. 'statementType' is a string optional field that must be one of the following values: 'insert', 'insertignore', or 'update'. 'insertBatchSize' is an integer optional field that specifies the number of records to insert per batch. 'addPrimaryKey' is a boolean optional field that indicates whether to add a primary key. No additional properties are allowed at the root level.
