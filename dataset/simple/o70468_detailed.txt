The schema describes parameters for generating an address. It is an object type. The main properties of the schema are: 'options' is an object that specifies options to control how the address and secret are generated. It is optional and does not have any default value. It has the following properties: 'entropy' is an array which must contain 16 integers, each between 0 and 255, specifying the entropy to use to generate the seed. 'algorithm' is a string that must be one of the following values: 'ecdsa-secp256k1' or 'ed25519', with 'ecdsa-secp256k1' as the default algorithm. 'test' is a boolean indicating whether the address is for a test network, where 'true' indicates it is for testing and the address starts with 'T', while 'false' indicates it is for mainnet with the address starting with 'X'. 'includeClassicAddress' is a boolean that specifies whether the classic address should also be included in the returned payload. No additional properties are allowed at the root level.
