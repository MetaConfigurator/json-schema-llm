The schema describes settings for print representations. It is an object type. The main properties of the schema are: 'TempDirectory' is a string catalog for storing temporary files during HTML/PDF generation, and is required. 'WkHtmlToPdfPath' is a string location of the wkhtmltopdf utility - http://wkhtmltopdf.org/, and is required. 'WkHtmlToPdfArguments' is a string format of the arguments for the wkhtmltopdf utility - http://wkhtmltopdf.org/, and is required. No additional properties are allowed at the root level.
