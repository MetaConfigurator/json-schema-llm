The schema describes combining signed transactions. It is an object type. The main properties of the schema are: 'signedTransactions' is an array that contains signed transactions (from the output of [sign](#sign)) to combine. It is required and must have a minimum length of 1. Each item in the array is a string that must match the pattern '^[A-F0-9]+$', representing a single signed transaction as an uppercase hexadecimal string (from the output of [sign](#sign)). No additional properties are allowed at the root level.
