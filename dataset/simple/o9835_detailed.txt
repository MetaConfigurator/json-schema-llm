The schema describes a data structure for processing specific fields. It is an object type. The main properties of the schema are: 'nameColumn' is a string optional description of a column name. 'descriptionColumn' is a string optional description of a column description. 'csv' is a string required field representing the CSV data. 'makeUniqueKeySecret' is a boolean optional flag indicating whether to make the unique key secret. 'email' is a string optional field formatted as an email with a maximum length of 1024 characters. No additional properties are allowed at the root level.
