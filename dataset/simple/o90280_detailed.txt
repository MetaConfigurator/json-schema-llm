The schema describes an Arcade expression added to the pop-up. It is an object type. The main properties of the schema are: 'expression' is a string that represents the Arcade expression. It is optional. 'name' is a string that represents the unique identifier for the expression. It is optional. 'returnType' is a string that indicates the return type of the Arcade expression. It can be either 'number' or 'string'. It is optional and defaults to 'string'. 'title' is a string that represents the title of the expression. It is optional. No additional properties are allowed at the root level.
