The schema describes a representation for big numbers. It is an object type. It defines several reusable components under 'definitions': 'MyBigDecimal' is a string, formatted as a big-decimal. 'MyBigInteger' is a string, formatted as a big-integer. The main properties of the schema are: 'bigInt' is a string referencing the 'MyBigInteger' definition. 'bigDec' is a string referencing the 'MyBigDecimal' definition. 'anotherBigInt' is a string formatted as a big-integer. 'anotherBigDec' is a string formatted as a big-decimal. No additional properties are allowed at the root level.
