The schema describes a mailing address. It is an object type. The main properties of the schema are: 'streetAddress' is a string or null, describing the primary street address. 'streetAddress2' is a string or null, describing an optional secondary street address. 'city' is a string or null, describing the city of the address. 'county' is a string or null, describing the county of the address. 'stateProvince' is a string or null, describing the state or province of the address. 'postalCode' is a string or null, describing the postal code of the address. No additional properties are allowed at the root level.
