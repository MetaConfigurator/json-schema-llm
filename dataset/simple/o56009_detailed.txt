The schema describes Fibaro Home Center Login JSON Schema. It is an object type. The main properties of the schema are: 'status' is a boolean indicating the User Authorized Flag and is required. 'userID' is a number representing the Identifier for the user, with a minimum value of 0 and is optional. 'username' is a string for the Name of the user, with a maximum length of 20 characters, and is optional. 'type' is a string defining the User type, with allowed values of 'user' and 'superuser', and is optional. No additional properties are allowed at the root level.
