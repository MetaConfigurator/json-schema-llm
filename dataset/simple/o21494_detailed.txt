The schema describes Appointments by service type and consulate. It is an object type. The main properties of the schema are: 'consulate' is a string that describes the consular level for the number of events and is required. 'count' is an integer that describes the count of the number of events, with a minimum value of 0, and is required. 'period' is a string that describes the period described by each data point, with allowed values being 'second', 'minute', 'hour', 'day', 'week', 'month', 'year', and is required. 'serviceType' is a string that describes the name of the service-type along which to segment the count and is required. No additional properties are allowed at the root level.
