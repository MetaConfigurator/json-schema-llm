The schema describes an object structure. It is an object type. The main properties of the schema are: 'comment' is a string optional description. 'mimeType' is a string required description. 'params' is an array of objects with each object containing properties: 'comment' is a string optional description, 'contentType' is a string optional description, 'fileName' is a string optional description, 'name' is a string required description, and 'value' is a string optional description. The 'name' property in each object of the 'params' array is required. No additional properties are allowed at the root level.
