The schema describes info on user created collections. It is an object type. The main properties of the schema are: '_$id' is a string that describes the name of a user created collection's schema. It is optional. 'fields' is an object that contains additional properties. Each additional property is an object that has required fields 'app_title' and 'app_type', both of which are strings. The 'app_values' field, which is also an object, can contain additional properties that are either strings or numbers. No additional properties are allowed at the root level.
