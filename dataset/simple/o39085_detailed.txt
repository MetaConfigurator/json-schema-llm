The schema describes the configuration for svg-patterns. It is an object type. The main properties of the schema are: '$schema' is a string an optional property that may define the schema version. '$tool' is a string a required property that must be set to 'svg-patterns'. 'dist' is a string a required property that specifies the output file path (relative to the pwd), supporting .ts, .js, or .json file extensions. 'url' is a string a required property that specifies the target of the raw Patterns-Gallery HTML page, defaulting to 'https://gitcdn.xyz/repo/progers/Patterns-Gallery/master/index.html'. No additional properties are allowed at the root level.
