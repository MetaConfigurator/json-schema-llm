The schema describes order billing and shipping information. It is an object type. The main properties of the schema are: 'orders' is an array containing objects that represent individual orders. Each order object is required to have 'order', 'billTo', and 'billAddress' fields. The 'shipTo' field, if present, requires a corresponding 'shipAddress'. The 'loyaltyId' field, if present, requires a corresponding 'loyaltyBonus'. No additional properties are allowed at the root level.
