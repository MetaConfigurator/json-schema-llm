The schema describes a structure for cryptographic settings. It is an object type. The main properties of the schema are: 'algorithm' is an integer representing an algorithm setting, required; it must be between 1 and 2. 'fingerprint' is a string representing a fingerprint, required; it must match the pattern '^[0-9A-Fa-f]{40}$'. 'type' is an integer representing a type, required; it must be exactly 1. No additional properties are allowed at the root level.
