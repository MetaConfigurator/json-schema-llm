The schema describes a list of people for a Timezone GNOME Shell Extension. It is an array type. The main properties of the schema are: Each item in the array is an object representing a person, which has the following properties: 'name' is a string representing the person's name. It is optional. 'avatar' is a string representing the URL for the person's avatar. It is optional. 'city' is a string representing the person's location (city, country...). It is optional. 'tz' is a string representing the person's timezone, and it is required. The timezone must be chosen from the TZ List found at http://bit.ly/1BKGEkq. 'gravatar' is a string representing the Gravatar ID (person's email). It is optional. 'github' is a string representing the GitHub ID. It is optional. No additional properties are allowed at the root level.
