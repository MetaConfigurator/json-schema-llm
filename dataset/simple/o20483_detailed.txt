The schema describes a token introspection request. It is an object type. The main properties of the schema are: 'token' is a string that represents the token to inspect, required with a minimum length of 1. 'token_type_hint' is a string that provides an optional hint about the type of the submitted token; it must be one of the following values: 'refresh_token' or 'access_token'. No additional properties are allowed at the root level.
