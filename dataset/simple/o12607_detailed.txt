The schema describes ParaDrop host configuration - dhcp object. It is an object type. The main properties of the schema are: 'leasetime' is a string describing the duration of client leases, e.g. 2h, which is optional. 'limit' is an integer describing the size of address range beginning at start value, which is optional and must be a minimum of 1. 'start' is an integer describing the starting offset for address assignment, which is optional and must be a minimum of 0. No additional properties are allowed at the root level.
