The schema describes a Shampoo request specification. It is an object type. The main properties of the schema are: 'method' is a string that specifies the method on the endpoint that has to be called. It is required. 'request_data' is an object that represents data to be passed to the method. It is required. 'request_id' is an integer that acts as an identifier for the request, with a minimum value of 0. It is required. 'type' is a string that must match the pattern 'request'. It is required. No additional properties are allowed at the root level.
