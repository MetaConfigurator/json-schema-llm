The schema describes a Point of Interest (POI) location. It is an object type. The main properties of the schema are: 'wgs84' is an object that represents the World Geodetic System, which is optional. It consists of properties: 'latitude' is a number describing degrees north, and 'longitude' is a number describing degrees east. 'height' is a number describing meters up from sea level and is optional. No additional properties are allowed at the root level.
