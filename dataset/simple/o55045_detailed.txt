The schema describes a message structure. It is an object type. The main properties of the schema are: 'message.topic' is a string that must match the pattern '^/[a-z\-]+$' and is required. 'message.since' is an integer and is required. 'message.snapshot' is a boolean and is required. 'message.filter' is a string with a maximum length of 137 characters that must match the pattern '^[\w\-\.#@\*\[\]\?]+$' and is required. No additional properties are allowed at the root level.
