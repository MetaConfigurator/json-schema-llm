The schema describes a configuration for managing page event monitoring and filtering. It is an object type. The main properties of the schema are: 'pageInfo' is an object that inspects page models received through wrapper events. It requires the field 'resource', which is a string that denotes the event topic through which to monitor pages, with a format of 'topic' and an axRole of 'inlet'. 'filter' is an object that allows filtering by topic or widget/activity instance. It also requires the field 'resource', which is a string that represents the topic through which to publish filter information, with a format of 'topic' and an axRole of 'outlet'. No additional properties are allowed at the root level.
