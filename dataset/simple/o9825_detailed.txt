The schema describes a structure for an event record. It is an object type. The main properties of the schema are: 'time' is a value that can be a string, a number, or an object with a required '$date' string property. The 'name' is a string with a minimum length of 1 and a maximum length of 255, which is required. The 'payload' can be of any type but is optional. No additional properties are allowed at the root level.
