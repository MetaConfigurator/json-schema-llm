The schema describes customer microservice toolchain. It is an object type. The main properties of the schema are: 'deploy-region' is a string Deploy to this Bluemix Region, required. 'deploy-org' is a string Deploy to this Bluemix Organization, required. 'deploy-space' is a string Deploy to this space, required. 'new-relic-lic-key' is a string New Relic License Key, optional. 'eureka-url' is a string Eureka Registry URL, optional. No additional properties are allowed at the root level.
