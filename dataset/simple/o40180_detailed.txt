The schema describes a course. It is an object type. The main properties of the schema are: 'credits' is a number describing course credit hours. It is optional. 'department' is a string describing the department that offers this course. It has a maximum length of 25 characters and is optional. 'prerequisites' is a string describing courses that must have been completed before this course. It is optional. 'schedule' is a string describing the days of the week and hours that this course will be offered. It is optional. 'startDate' is a string formatted as a date describing the date this course commences. It is required. 'syllabus' is a string describing a summary of topics to be covered in this course. It is required. 'title' is a string describing the title of this course. It is required. No additional properties are allowed at the root level.
