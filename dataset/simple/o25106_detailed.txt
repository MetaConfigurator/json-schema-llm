The schema describes a resource configuration for scalable dimensions. It is an object type. The main properties of the schema are: 'ResourceId' is a string and is required. 'ScalableDimension' is a string and is required. 'ServiceNamespace' is a string and is required. 'PolicyType' is a string and is optional. 'StepScalingPolicyConfiguration' is an object and is optional. No additional properties are allowed at the root level.
