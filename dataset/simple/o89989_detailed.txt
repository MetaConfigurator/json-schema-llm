The schema describes a package repository. It is an object type. The main properties of the schema are: 'mirror_urls' is an array that may contain a list of URLs for mirrors of this repository, or it may be null. 'nickname' is a string representing the nickname used for well known 'named' public repos such as: Maven Central, Pypi, RubyGems, npmjs.org or their mirrors, or it may be null. 'public' is a boolean flag set to true if this is a public repository, or it may be null. 'type' is a string representing the type of package repository for this repository, which must be one of: Bower, CPAN, Debian, Rubygems, Godoc, IVY, Maven, NPM, Packagist, Nuget, Pypi, YUM, or it may be null. 'url' is a string that represents the URL to this repository, or it may be null. No additional properties are allowed at the root level.
