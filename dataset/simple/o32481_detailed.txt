The schema describes an organization. It is an object type. The main properties of the schema are: 'id' is a string that represents the organization id and is required. 'name' is a string or null that represents the organization name and must have a minimum length of 1. 'code' is a string that represents the organization code and is required, with a minimum length of 1. 'email' is a string or null that represents the organization email. No additional properties are allowed at the root level.
