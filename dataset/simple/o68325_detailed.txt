The schema describes parameters accepted by the Orbidder adapter. It is an object type. The main properties of the schema are: 'accountId' is a string which describes the marketer's accountId and is required. 'placementId' is a string which describes the placementId of the ad unit and is required. 'bidfloor' is a number which describes the minimum CPM price in EUR, has a minimum value of 0, and is optional. No additional properties are allowed at the root level.
