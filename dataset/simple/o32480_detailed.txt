The schema describes a message structure. It is an object type. The main properties of the schema are: 'id' is a string that describes the message id. 'content' is a string that describes the message content. 'meta' is an object that contains further properties: 'creator' is an optional property with no specified type. 'created' is a string that describes the subject creation date. 'updated' is a string that describes the subject update date. No additional properties are allowed at the root level.
