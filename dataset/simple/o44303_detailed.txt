The schema describes the configuration of a thumbnail associated to a scene. It is an object type. The main properties of the schema are: 'height' is a number describing the height of the thumbnail. It is optional. 'uid' is a string describing the UID of the thumbnail. It is optional. 'url' is a string describing the URL of the thumbnail: the file can be local or on a distant server, but must be a valid image format. It is optional. 'width' is a number describing the width of the thumbnail. It is optional. No additional properties are allowed at the root level.
