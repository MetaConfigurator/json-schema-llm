The schema describes a configuration object. It is an object type. The main properties of the schema are: 'model' is an object that represents a model configuration. 'targetFile' is a string that specifies the target file name. 'template' is a string that indicates the template name. 'templateDict' is an object that contains properties: 'kind' which is a string denoting the type of template, and 'template' which is a string for the template content. 'templatePath' is an array of strings that lists paths to templates. No additional properties are allowed at the root level.
