The schema describes a representation of a dog. It is an object type. It defines several reusable components under 'definitions': 'Person' is an object, a human being. It has an optional field 'name' which is a string. 'Breed' is a string, most popular breeds, and it has an enum with allowed values: 'LABRADOR', 'BULLDOG', 'YORKSHIRE'. The main properties of the schema are: 'name' is a string How to call the dog. It is optional. 'age' is an integer Number of years ( > 0 ) with a minimum value of 0. It is optional. 'owner' is a referenced definition from 'Person', The person taking care of the dog. It is optional. 'breed' is a referenced definition from 'Breed', The breed if pure. It is optional. No additional properties are allowed at the root level.
