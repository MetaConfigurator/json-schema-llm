The schema describes an array of objects. It is an array type. The main properties of the schema are: 'id' is a string and is required. 'title' is a string and is required. 'file' is a string and is required. 'preview' is a string and is optional. 'tags' is an array of strings and is required. Array items in 'tags' must be strings. No additional properties are allowed at the root level.
