The schema describes get crawl result response. It is an object type. The main properties of the schema are: 'id' is a string which is required and must have a minimum length of 1. 'links' is an object which is required and must contain a property 'self' that is also an object. The 'self' object is required to have a property 'href', which is a string formatted as a URI. 'results' is an object which is required and allows additional properties. No additional properties are allowed at the root level.
