The schema describes Kubernetes DNS configurations. It is an object type. The main properties of the schema are: 'cluster_ip' is a string This should be the same as the IP set in the deployment.clusters[x].dns in the main configuration file. It is required. 'dns_domain' is a string This should be the same as the domain set in deployment.clusters[x].domain in the main configuration file. It is required. 'namespace' is a string Kubernetes is expecting DNS to be in kube-system. It is optional. No additional properties are allowed at the root level.
