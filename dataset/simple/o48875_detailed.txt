The schema describes a person object. It is an object type. It defines several reusable components under 'definitions': 'name' is a string, representing a name that must have at least 1 character in length. The main properties of the schema are: 'firstname' is a string that must conform to the 'name' definition and is required. 'lastname' is a string that must conform to the 'name' definition and is required. 'title' is an optional string with allowed values: 'mr', 'mrs', 'ms', 'dr', 'prof'. No additional properties are allowed at the root level.
