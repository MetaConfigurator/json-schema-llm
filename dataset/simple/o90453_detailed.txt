The schema describes the configuration parameters for a network operation. It is an object type. The main properties of the schema are: 'blocked' is a number that is optional and must be greater than or equal to -1. 'comment' is a string that is optional. 'connect' is a number that is optional and must be greater than or equal to -1. 'dns' is a number that is optional and must be greater than or equal to -1. 'receive' is a number that is required and must be greater than or equal to -1. 'send' is a number that is required and must be greater than or equal to -1. 'ssl' is a number that is optional and must be greater than or equal to -1. 'wait' is a number that is required and must be greater than or equal to -1. No additional properties are allowed at the root level.
