The schema describes the result of the getClassInfo procedure. It is an object type. The root schema directly defines properties. The main properties of the schema are: 'class_info' is an object that holds the result. It is required. It contains five properties: 'firmid' is a string parameter that provides the firm's ID. 'name' is a string parameter that provides the name. 'code' is a string parameter that provides the code. 'npars' is a number parameter that specifies the number of parameters. 'nsecs' is a number parameter that specifies the number of sections. No additional properties are allowed at the root level.
