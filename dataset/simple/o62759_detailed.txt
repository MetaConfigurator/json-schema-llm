The schema describes an icon atom. It is an object type. The main properties of the schema are: 'icon' is a string that represents the short name of the icon. It is a required field with allowed values: 'arrow-right', 'chevron-up', 'close', 'exit-search', 'expert-tool-user', 'flag-ch', 'flag-de', 'flag-fr', 'heading', 'icon-info', 'icon-info-circle', 'icon-remove-tag', 'lightbox-close', 'lightbox-open', 'nitro', 'nitro-full'. No additional properties are allowed at the root level.
