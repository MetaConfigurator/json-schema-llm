The schema describes account creation permissions. It is an object type. The main properties of the schema are: 'canCreateAccounts' is a boolean indicating true if this account manager can create accounts, in which case TAS SSO will display a sign up button. It is a required property. 'createAccountTermsAndConditionsUrl' is a string formatted as a URI, describing a link to the terms and conditions that the user must agree to before creating an account. It is optional and only present if 'canCreateAccounts' is true. No additional properties are allowed at the root level.
