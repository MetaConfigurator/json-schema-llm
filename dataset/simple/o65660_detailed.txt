The schema describes a range of numerical values with a unit of measure. It is an object type. The main properties of the schema are: 'low_value' is a number that represents the lower boundary of the range, required; 'high_value' is a number that represents the upper boundary of the range, required; 'unit' is a string that represents the unit of measure of the element, required, with allowed values drawn from the Common synonyms (non-UCUM) column of [subset of] UCUM, SI and English units as detailed in the referenced document. No additional properties are allowed at the root level.
