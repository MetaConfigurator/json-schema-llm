The schema describes the dimensions and weight of a package for shipping. It is an object type. The main properties of the schema are: 'carrier' is a string that specifies the acronym of the carrier you want to use. It is required with allowed values: 'dhl', 'ups', 'dpd', 'hermes', 'gls'. 'width' is a number that defines the width of the package. It is required. 'height' is a number that defines the height of the package. It is required. 'length' is a number that defines the length of the package. It is required. 'weight' is a number that defines the weight of the package. It is required. No additional properties are allowed at the root level.
