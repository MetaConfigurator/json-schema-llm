The schema describes a configuration for a DynamoDB stream. It is an object type. The main properties of the schema are: 'TableName' is a string which is required. 'StreamViewType' is a string with an enum of allowed values: 'NEW_IMAGE', 'OLD_IMAGE', 'NEW_AND_OLD_IMAGES', 'KEYS_ONLY', and is required. No additional properties are allowed at the root level.
