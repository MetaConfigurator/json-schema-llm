The schema describes the value returned by GET /{db}. It is an object type. The main properties of the schema are: 'committed_update_seq' is a number required, representing the committed update sequence. 'compact_running' is a boolean required, indicating whether compaction is currently running. 'db_name' is a string required, representing the name of the database. 'disk_format_version' is a number required, specifying the version of the disk format. 'data_size' is a number required, representing the size of the data. 'disk_size' is a number required, indicating the total size on disk. 'doc_count' is a number required, representing the count of documents. 'doc_del_count' is a number required, indicating the count of deleted documents. 'instance_start_time' is a string required, representing the start time of the instance. 'purge_seq' is a number required, indicating the purge sequence. 'update_seq' is a number required, representing the update sequence. No additional properties are allowed at the root level.
