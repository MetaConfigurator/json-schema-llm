The schema describes an array of user objects. It is an array type. The main properties of the schema are: 'id' is a string that must match the pattern '^[A-Fa-f\d]{24}$'. 'email' is a string with an email format that has a maximum length of 1024 characters. 'role' is a string that must be one of the following values: 'admin', 'edit', or 'view'. 'inviteDate' is a string with a date-time format. 'ttl' is a number. 'hasExpired' is a boolean. No additional properties are allowed at the root level.
