The schema describes an array of objects. It is an array type. The main properties of the schema are: 'name' is a string required field representing the name of the object. Each item in the array must also contain 'checksums', which is an object that includes: 'darwin', 'linux', and 'windows', all of which are required fields of string type with a pattern that matches exactly 64 hexadecimal characters (uppercase or lowercase). No additional properties are allowed at the root level.
