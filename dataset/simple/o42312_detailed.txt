The schema describes an array of employee records. It is an array type. The main properties of the schema are: Each item in the array is an object representing an employee with the following properties: 'id' is an integer and is required. 'employee_id' is a string and is optional. 'first_name' is a string and is required. 'middle_name' is a string and is optional. 'last_name' is a string and is required. No additional properties are allowed at the root level.
