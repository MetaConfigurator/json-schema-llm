The schema describes a VirtualBox manifest. It is an object type. The main properties of the schema are: 'bootstrapper' is an object that may contain the property 'guest_additions', which is an optional string. 'system' is an object that may contain the property 'bootloader', which is an optional string with allowed values: 'grub', 'extlinux'. 'volume' is an object that is required and must contain the property 'backing', which is a string with allowed values: 'raw', 'vdi', 'vmdk'. Additionally, 'volume' may contain the property 'partitions' which is an object that may have the property 'type', which is a string with allowed values: 'none', 'msdos', 'gpt'. No additional properties are allowed at the root level.
