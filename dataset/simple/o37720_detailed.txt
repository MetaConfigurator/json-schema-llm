The schema describes a vaccine delivery driver. It is an object type. The main properties of the schema are: 'doc_type' is a string that must match the pattern '^driver$' and is required. 'email' is a string that must conform to the email format and is required. 'forename' is a string with a minimum length of 1 and is required. 'phone' is a string and is required. 'surname' is a string with a minimum length of 1 and is required. 'version' is a string that must conform to the semver format and is required. No additional properties are allowed at the root level.
