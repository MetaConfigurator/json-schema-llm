The schema describes a file's metadata. It is an object type. The main properties of the schema are: 'fileSize' is a number representing the size of the file. 'contentType' is a string with a maximum length of 1024 characters denoting the MIME type of the file. 'fileDimensions' is an object containing dimensions of the file; it includes 'width' as a number for the file's width and 'height' as a number for the file's height. No additional properties are allowed at the root level.
