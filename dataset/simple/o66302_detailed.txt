The schema describes a configuration object. It is an object type. The main properties of the schema are: 'array' is a string and is required. 'contains' is a string and is required; it must be one of the following values: '$any', '$all', '$none'. 'values' is an array and is required; it must contain at least one item, consists of strings, and all items must be unique. No additional properties are allowed at the root level.
