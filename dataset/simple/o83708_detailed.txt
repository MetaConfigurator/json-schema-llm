The schema describes IP address/port for primary/backup server pair. It is an array type. The main properties of the schema are: Each item is an object that includes 'address', which is a string and required, and 'port', which is an integer, required, with a minimum value of 0 and a maximum value of 65535. The array must have exactly 2 items, and all items must be unique. No additional properties are allowed at the root level.
