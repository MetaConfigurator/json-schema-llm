The schema describes a collection of keys. It is an object type. The main properties of the schema are: 'keys' is an array that contains at least one item. Each item is an object that must have the properties 'verifying-key' and 'id', both of which are required. The 'verifying-key' property is an object that requires 'data' and 'type', both of which are strings. No additional properties are allowed at the root level.
