The schema describes information about a VPN store. It is an object type. The main properties of the schema are: 'storeapisupport' is a string and can be either 'YES' or 'NO', it is readonly. 'storelist' is a string, it is readonly. 'storeserverissf' is a string and can be either 'YES' or 'NO', it is readonly. 'storeserverstatus' is a string and can be either 'UP' or 'DOWN', it is readonly. 'storestatus' is a string and can be either 'UP' or 'DOWN', it is readonly. 'url' is a string and is optional. No additional properties are allowed at the root level.
