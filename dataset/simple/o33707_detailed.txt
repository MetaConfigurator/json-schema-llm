The schema describes a document structure. It is an object type. The main properties of the schema are: 'schemaType' is a string that describes the schema type, it is required, searchable, and has a locked status. '_documentID' is a string that represents the internal document GUID, it is required, searchable, and has a locked status. '_corpusID' is a string that represents the internal Corpus GUID, it is required, searchable, and has a locked status. No additional properties are allowed at the root level.
