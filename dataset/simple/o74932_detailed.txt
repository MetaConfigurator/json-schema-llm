The schema describes a shipment tracking object. It is an object type. The main properties of the schema are: 'id' is a string that represents the shipment id that can be used for requesting info about a shipment or tracking it and it is required. 'carrier_tracking_no' is a string that denotes the original tracking number that can be used on the carrier's website and it is required. 'tracking_url' is a string that provides a URL you can send your customers so they can track this shipment and it is required. 'label_url' is a string that indicates the URL where you can download the label in pdf format and it is required. 'price' is a number that signifies the price that we're going to charge you (excluding VAT) and it is required. No additional properties are allowed at the root level.
