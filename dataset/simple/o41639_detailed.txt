The schema describes a collection of pet definitions. It is an object type. It defines several reusable components under 'definitions': 'pets' is an object, which requires the fields 'dog' and 'cat', both of which are strings. 'morePets' is an object, which requires the field 'horse' (string) in addition to the 'pets' definition. The main properties of the schema are: 'pets' is a object that resolves to the 'pets' definition. 'morePets' is a object that resolves to the 'morePets' definition. No additional properties are allowed at the root level.
