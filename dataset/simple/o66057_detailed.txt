The schema describes a PayPal link json schema. It is an object type. The main properties of the schema are: 'href' is a string URL of the related HATEOAS link you can use for subsequent calls. It is required. 'rel' is a string that describes how this link relates to the previous call. It is required. 'method' is a string that specifies the HTTP method required for the related call. It is required. No additional properties are allowed at the root level.
