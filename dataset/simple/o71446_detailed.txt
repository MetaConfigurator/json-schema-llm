The schema describes geometry texture properties. It is an object type. The main properties of the schema are: 'angle' is an integer that is optional. 'height' is an integer that is optional. 'image' is a string that is optional. 'leftToRightOriented' is a boolean that is optional. 'name' is a string that is optional. 'width' is a number that is optional. No additional properties are allowed at the root level.
