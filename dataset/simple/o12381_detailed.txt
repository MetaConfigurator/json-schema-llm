The schema describes a generic link between two (or more) entities. It is an object type. The main properties of the schema are: 'end_date' is a string or null that represents the date on which the relationship ended. It is optional and follows the format of a date. 'id' is an integer that serves as the ID of the link. It is hidden and required. 'start_date' is a string or null that represents the date on which the relationship began. It is optional and follows the format of a date. No additional properties are allowed at the root level.
