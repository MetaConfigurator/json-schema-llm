The schema describes a table data source. It is an object type. The main properties of the schema are: 'dataSourceName' is a string specifying where the data source is derived, optional. 'gdbVersion' is a string indicating the version of the geodatabase, optional. 'type' is a string indicating the type for the data source, required, with an enum value of 'table'. 'workspaceId' is a string used to identify the data source's workspace, optional. No additional properties are allowed at the root level.
