The schema describes the structure for a product return response. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'items' is an object that contains the following properties: 'seq' is an integer, optional; 'codigo' is either a string or null, optional; 'erros' is an array of strings, optional; 'sucesso' is a boolean, optional. No additional properties are allowed at the root level.
