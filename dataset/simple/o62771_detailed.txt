The schema describes a heading atom. It is an object type. The main properties of the schema are: 'titleTag' is a string that indicates the heading level, which is optional and can take the values 'h1', 'h2', or 'h3'. 'title' is a string that is required and contains the main text of the heading. 'subTitle' is a string that is optional and can contain additional text related to the heading. No additional properties are allowed at the root level.
