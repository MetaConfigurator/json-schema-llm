The schema describes information about tanks. It is an object type. The main properties of the schema are: 'id' is a string unique identifier for this tank and is required. 'stationid' is a string unique identifier for the station where this tank is located and is required. 'fuel' is an enum with allowed values 'regular', 'premium', and 'diesel', and is required. 'capacity' is a number representing the fuel capacity of the tank and is required. No additional properties are allowed at the root level.
