The schema describes the preview object associated with the media. It is an object type. The main properties of the schema are: 'tile' is a number that represents the size of a tile, which corresponds to the width and height of a tile. It is optional and has an example value of 512. 'url' is a string that specifies the URL for the preview of the media, with a {face} parameter to indicate the face when parsing. It is optional and has an example value of 'multi/preview/{face}.jpg'. No additional properties are allowed at the root level.
