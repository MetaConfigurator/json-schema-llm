The schema describes a generic type that describes the entry of a selection list. It is an object type. The main properties of the schema are: 'active' is a boolean that indicates whether this entry is still active (optional). 'description' is a string that provides some more words to explain (optional). 'gid' is a string with a format of uuid that serves as a global identifier (required). 'name' is a string that acts as a describing name (required, unique, and marked as visKey). Additional properties are not allowed at the root level.
