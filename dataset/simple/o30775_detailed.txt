The schema describes the details of a migration process. It is an object type. The main properties of the schema are: 'migrationendtime' is a string that is readonly. 'migrationrollbackstarttime' is a string that is readonly. 'migrationstarttime' is a string that is readonly. 'migrationstatus' is a string that is readonly and has an enum of allowed values: 'Migration is not yet started', 'Migration is in progress and Failover is not yet done', 'Migration is in progress and Failover is completed', 'Rollback is initiated', 'Migration is completed'. No additional properties are allowed at the root level.
