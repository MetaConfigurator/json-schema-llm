The schema describes a Clear Charging Profile Request. It is an object type. The main properties of the schema are: 'id' is an integer, optional. 'connectorId' is an integer, optional. 'chargingProfilePurpose' is a string, optional, with allowed values: 'ChargePointMaxProfile', 'TxDefaultProfile', 'TxProfile'. 'stackLevel' is an integer, optional. No additional properties are allowed at the root level.
