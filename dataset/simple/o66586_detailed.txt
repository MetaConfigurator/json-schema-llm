The schema describes a microservices deployment stage. It is an object type. The main properties of the schema are: 'dev-region' is a string that describes the bluemix region and is required. 'dev-organization' is a string that describes the bluemix org and is required. 'dev-space' is a string that describes the bluemix space and is required. 'catalog-app-name' is a string that describes the catalog app name and is required. No additional properties are allowed at the root level.
