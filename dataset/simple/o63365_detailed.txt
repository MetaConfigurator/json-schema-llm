The schema describes a bounding box as defined by TopoJSON. It is an array type. It defines several reusable components under 'definitions': 'dimension' is an array, which should have an entry per dimension in the geometries. It contains items of type number. The main properties of the schema are: The schema is an array that must contain exactly 2 items, each of which references the 'dimension' definition. No additional properties are allowed at the root level.
