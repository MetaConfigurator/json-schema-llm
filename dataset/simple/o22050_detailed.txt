The schema describes Ng Deploy target options for Firebase. It is an object type. The main properties of the schema are: 'buildTarget' is a string that specifies the target to build. It has a pattern constraint: it must match '^[^:\s]+:[^:\s]+(:[^\s]+)?$'. 'preview' is a boolean that, when set to true, indicates to not deploy the application, just set up the Firebase Function in the project output directory for testing with 'firebase serve'. No additional properties are allowed at the root level.
