The schema describes a configuration for elements and attributes. It is an object type. The main properties of the schema are: '_attributes' is an object representing optional attributes with a required field 'patterns_system'; 'patterns_system' must be one of the enum values 'zypper' or 'tasksel'. '_elements' is an array that can take one of two forms: it can contain objects each with required fields 'name', 'version', and 'release', or it can contain objects each with a required field 'name'. No additional properties are allowed at the root level.
