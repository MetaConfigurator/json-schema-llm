The schema describes a Blog Post. It is an object type. The main properties of the schema are: 'Body' is a string that is required. 'Day' is a string that is required. 'IsPublished' is a boolean that is required. 'Month' is a string that is required. 'PublishedDateTime' is a string with a format of date-time that is required. 'Subtitle' is a string that is optional. 'Tags' is an array of strings, where each item must be one of the following values: 'csharp', 'nodejs', 'typescript', and it is required. 'Title' is a string that is required. 'Year' is a string that is required. No additional properties are allowed at the root level.
