The schema describes the structure for a help resource. It is an object type. It defines several reusable components under 'definitions': 'hyperLink' is an object, representing a hyperlink which must include a 'title' and a 'url'. The main properties of the schema are: 'title' is a string and is required. 'description' is a string and is required. 'thumbnail' is a string and is required. 'order' is an integer and is required. 'navigateLink' is a string and is optional. 'helpTopics' is an array of hyperlinks, where each item must adhere to the 'hyperLink' definition. No additional properties are allowed at the root level.
