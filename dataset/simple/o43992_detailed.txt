The schema describes a Start Transaction Request. It is an object type. The main properties of the schema are: 'connectorId' is an integer required field. 'idTag' is a string required field with a maximum length of 20 characters. 'meterStart' is an integer required field. 'reservationId' is an integer optional field. 'timestamp' is a string required field formatted as date-time. No additional properties are allowed at the root level.
