The schema describes GKE specific node configuration. It is an object type. The main properties of the schema are: 'diskSize' is a number that represents the amount of disk that should be allocated to the node, in GB. It is required. 'machineType' is a string that indicates the machine type that should be used for the node (n1-standard-1, etc.). It is required. 'scopes' is an array of strings that specifies the scopes given to the node. It is required. No additional properties are allowed at the root level.
