The schema describes an artist's information. It is an object type. The main properties of the schema are: 'name' is a string that describes the name of the artist. 'popularity' is an integer that describes the popularity of the artist, required to be between 0 and 100. 'type' is a string that describes the object type as 'artist'. 'uri' is a string that describes the Spotify URI for the artist. No additional properties are allowed at the root level.
