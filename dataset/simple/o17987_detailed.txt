The schema describes the command for the PlanetWars game. It is an object type. The main properties of the schema are: 'moves' is an array describing what you want to do in the current turn. Each item must be an object with required fields 'origin', 'destination', and 'ship_count'. The field 'ship_count' must be an integer with a minimum value of 0. No additional properties are allowed at the root level.
