The schema describes the configuration for an authentication HTTP service. It is an object type. The main properties of the schema are: 'SuccessUrl' is a string that specifies the path for redirection upon successful authentication through an external provider. It is optional with a default value of null. 'FailureUrl' is a string that specifies the path for redirection in case of an authentication error through an external provider. It is also optional with a default value of null. No additional properties are allowed at the root level.
