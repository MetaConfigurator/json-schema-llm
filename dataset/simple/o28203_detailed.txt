The schema describes a request for Order status information. It is an object type. The main properties of the schema are: 'buyerId' is a string The ID of the Service Provider (Buyer) organization placing the order. It is required and has a maximum length of 25 characters. 'buyerPurchaseOrderNumber' is a string A number that uniquely identifies an order within the buyer's enterprise. It is required and has a maximum length of 16 characters. 'objectType' is a string The type of the object, orderStatusRequest. It is required and must be one of the allowed values: 'orderStatusRequest'. 'orderId' is a string Unique identifier for the order for which status information is being requested (if known, otherwise the buyerPurchaseOrderNumber is required). It is required and has a maximum length of 45 characters. 'sellerId' is a string The ID of the Partner/Access Provider (Seller) organization receiving the order. It is required and has a maximum length of 25 characters. No additional properties are allowed at the root level.
