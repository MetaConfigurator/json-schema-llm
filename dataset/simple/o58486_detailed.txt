The schema describes a pause step. It is an object type. The main properties of the schema are: 'name' is a string and required; 'kind' is a string with the enum value 'pause' and required; 'label' is a string and required; 'title' is a string and required; 'length' is an integer with a minimum value of 1 and optional; 'iface' is an object and optional. No additional properties are allowed at the root level.
