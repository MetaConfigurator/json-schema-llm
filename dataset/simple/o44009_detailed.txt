The schema describes a response for getting configuration. It is an object type. The main properties of the schema are: 'configurationKey' is an array of objects, each containing the properties 'key', 'readonly', and 'value'. The 'key' is a required string with a maximum length of 50 characters, 'readonly' is a required boolean, and 'value' is an optional string with a maximum length of 500 characters. 'unknownKey' is an optional array of strings, each with a maximum length of 50 characters. No additional properties are allowed at the root level.
