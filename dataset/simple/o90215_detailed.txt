The schema describes the characteristics of the vertical coordinate system used by the web scene. It is an object type. The main properties of the schema are: 'heightModel' is a string that describes the surface type or height model of the vertical coordinate system, required, with allowed values 'ellipsoidal' and 'gravity_related_height', and a default value of 'ellipsoidal'. 'heightUnit' is a string that describes the unit of the vertical coordinate system, required, with allowed values '150-kilometers', '50-kilometers', 'benoit-1895-b-chain', 'clarke-foot', 'clarke-link', 'clarke-yard', 'foot', 'gold-coast-foot', 'indian-1937-yard', 'indian-yard', 'meter', 'sears-1922-truncated-chain', 'sears-chain', 'sears-foot', 'sears-yard', 'us-foot'. 'vertCRS' is a string that describes the optional datum realization of the vertical coordinate system. No additional properties are allowed at the root level.
