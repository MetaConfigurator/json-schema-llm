The schema describes a reference to an ordered UNI. It is an object type. The main properties of the schema are: 'buyerPurchaseOrderNumber' is a string for the purchase order number for the order that contains the referenced UNI. It is optional and has a maximum length of 16 characters. 'objectType' is a string that describes the type of the object, specifically 'orderedUniReference'. It is required and must be one of the allowed values: 'orderedUniReference'. 'orderId' is a string that serves as a unique identifier for an order. It is optional and has a maximum length of 45 characters. 'orderItemReference' is a string for the order item reference for the UNI on the referenced order, or on the current order if a buyerPurchaseOrderNumber is not provided. It is required and has a maximum length of 4 characters. No additional properties are allowed at the root level.
