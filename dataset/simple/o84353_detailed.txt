The schema describes the entity sent when creating a new user. It is an object type. The main properties of the schema are: 'email' is a string representing the email address of the user, it is required and must be unique. 'password' is a string for identifying the user, it is required and must have a minimum length of 6. 'username' is a string used when locally logging in, it is required, must have a minimum length of 3 and a maximum length of 30, and it must be unique. No additional properties are allowed at the root level.
