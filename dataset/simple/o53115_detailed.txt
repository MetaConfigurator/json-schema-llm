The schema describes resource types. It is an object type. The main properties of the schema are: 'id' is an integer representing the resource type ID, required, with a minimum value of 0. 'abbrev' is a string representing the resource type abbreviation, required, with a maximum length of 10. 'description' is a string representing the resource type description, required, with a maximum length of 50. No additional properties are allowed at the root level.
