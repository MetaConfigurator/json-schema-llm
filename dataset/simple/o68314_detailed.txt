The schema describes parameters accepted by the Kidoz adapter. It is an object type. It defines several reusable components under 'definitions': 'non-empty-string' is a string type, which must have a minimum length of 1. The main properties of the schema are: 'access_token' is a string that references the 'non-empty-string' definition, it is required. 'publisher_id' is a string that also references the 'non-empty-string' definition, it is required. No additional properties are allowed at the root level.
