The schema describes a configuration for chart display properties. It is an object type. The main properties of the schema are: 'fields' is an array of strings, representing the names of fields to display in the chart. It is required and must consist of unique items; 'normalizeField' is an optional string that represents a field used for normalizing values in the chart; 'tooltipField' is an optional string that indicates the field used for tooltips in the chart. No additional properties are allowed at the root level.
