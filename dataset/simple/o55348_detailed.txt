The schema describes the configuration for a network service. It is an object type. The main properties of the schema are: 'port' is an integer that is required and must be between 0 and 65535. 'priority' is an integer that is required and must be between 0 and 65535. 'target' is a string that is required and must conform to the hostname format. 'weight' is an integer that is required and must be between 0 and 65535. No additional properties are allowed at the root level.
