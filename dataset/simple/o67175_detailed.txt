The schema describes a structure for job result metrics. It is an object type. The main properties of the schema are: 'duration' is an integer representing the duration of the job, required. 'logs' is an array of strings containing log messages, optional. 'memory' is an integer indicating the memory usage, required. 'schema' is a string that must be one of the values: 'http://comrade.forma-pro.com/schemas/JobResultMetrics.json', optional. 'startTime' is an integer indicating the start time, required. 'stopTime' is an integer indicating the stop time, required. No additional properties are allowed at the root level.
