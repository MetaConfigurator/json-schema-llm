The schema describes a device configuration. It is an object type. The main properties of the schema are: 'attribute' is a string matching the pattern '^[0-9a-zA-Z_-]{1,255}$', which is optional. 'deviceTags' is an array of objects, where each object contains a 'key' that is a string matching the pattern '^[0-9a-zA-Z_-]{1,255}$' and a 'value' that is a string with a minimum length of 1 and a maximum length of 255; 'deviceTags' is optional. 'deviceIds' is an array of strings, where each string matches the pattern '^[A-Fa-f\d]{24}$'; 'deviceIds' is optional. No additional properties are allowed at the root level.
