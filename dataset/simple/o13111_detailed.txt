The schema describes OpenQuantum backend status. It is an object type. The main properties of the schema are: 'backend_name' is a string specifying the Backend name and is required. 'backend_version' is a string describing the Backend version in the form X.X.X, requires a pattern match of [0-9]+.[0-9]+.[0-9]+$, and is required. 'operational' is a boolean indicating if the Backend is operational and accepting jobs (true/false) and is required. 'pending_jobs' is an integer that must be at least 0, representing the number of pending jobs on the backend and is required. 'status_msg' is a string describing the Status message and is required. No additional properties are allowed at the root level.
