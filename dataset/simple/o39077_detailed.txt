The schema describes retrieval of attributes of arukas. It is an object type. The main properties of the schema are: '$schema' is a string optional, '$tool' is a string required, must be set to 'arukas', 'attr' is a string required, attribute name to fetch, only supports 'host', 'secret' is a string optional, 'token' is a string optional. No additional properties are allowed at the root level.
