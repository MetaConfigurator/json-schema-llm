The schema describes the configuration for linkerd-viz. It is an object type. The main properties of the schema are: 'linkerd-viz' is an object that contains properties for configuring linkerd-viz. It is required. The properties within 'linkerd-viz' are: 'cpus' is a number that specifies CPU shares to allocate to linkerd-viz, with a minimum of 1.0 and a default value of 1.0. 'mem' is a number that specifies memory (MB) to allocate to linkerd-viz, with a minimum of 512.0 and a default value of 512.0. 'public-port' is an integer that specifies the public port for accessing the linkerd-viz dashboard, with a default value of 3000. 'resource-role' is a string that specifies the accepted resource role for deployment, with a default value of 'slave_public'. 'stats-port' is an integer that specifies the stats port that gives the dashboard access to the backend stats server, with a default value of 9090. No additional properties are allowed at the root level.
