The schema describes a configuration for object paths selection. It is an object type. The main properties of the schema are: 'donate' is an array of strings containing URLs for donations, it is required and must have exactly 2 unique items, which can only be one of 'https://pledgie.com/campaigns/28037' or 'https://gratipay.com/aminassian'. 'licence' is a string representing the licence code, and it is required. 'path' is an array of strings representing the path result, and it is also required. No additional properties are allowed at the root level.
