The schema describes configuration for node licenses. It is an object type. The main properties of the schema are: 'limit_exceeded' is a boolean indicating whether the number of nodes granted under the license has been exceeded, optional; 'node_license' is an integer representing the number of nodes granted under the license, optional; 'node_count' is an integer denoting the number of nodes currently used, optional; 'upgrade_url' is a string formatted as a URI for more information on how to upgrade, optional. No additional properties are allowed at the root level.
