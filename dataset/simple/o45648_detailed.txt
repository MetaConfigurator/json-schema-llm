The schema describes pagination links. It is an object type. The main properties of the schema are: 'first' is a string Pagination link to first page. It is required. 'last' is a string Pagination link to last page. It is required. 'next' is a string Pagination link to next page. It is optional. 'prev' is a string Pagination link to previous page. It is optional. 'self' is a string Pagination link to this page. It is required. No additional properties are allowed at the root level.
