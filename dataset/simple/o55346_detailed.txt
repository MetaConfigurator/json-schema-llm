The schema describes a DNS zone configuration. It is an object type. The main properties of the schema are: 'expire' is an integer representing the expiration time, required. It must be between 0 and 214748364. 'minimum' is an integer representing the minimum time, required. It must be between 0 and 214748364. 'mname' is a string representing the master name, required. It must conform to the hostname format. 'refresh' is an integer representing the refresh time, required. It must be between 0 and 214748364. 'retry' is an integer representing the retry time, required. It must be between 0 and 214748364. 'rname' is a string representing the responsible name, required. It must conform to the hostname format. 'serial' is an integer representing the serial number, required. It must be between 0 and 214748364. No additional properties are allowed at the root level.
