The schema describes a collection of contacts with a name and freeform contact methods. It is an object type. The main properties of the schema are: 'name' is a string that is required. The schema allows for a minimum of 1 property and a maximum of 5 properties in total. Additional properties must be strings that match the pattern '^([0-9]{3}) [0-9]{3}-[0-9]{4}$'. No additional properties are allowed at the root level.
