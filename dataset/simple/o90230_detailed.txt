The schema describes information on how the scene is lit. It is an object type. The main properties of the schema are: 'datetime' is a number representing the current date and time of the simulated sun, which is optional. 'directShadows' is a boolean indicating whether to show shadows cast by the sun, with a default value of false. 'displayUTCOffset' is a number representing the UTC offset in decimal hours, which is optional. No additional properties are allowed at the root level.
