The schema describes parameter selection. It is an array type. The main properties of the schema are: 'items' is an object containing two properties. 'value' is a type that can be a string, an array, a number, or an object which requires 'expression_type' to be present. If 'value' is an object, it has properties: 'output' (a string), 'expression_type' (a string with a default value of 'stage-output-selector' and an enum value of 'stage-output-selector'), 'stages' (a string), 'flatten' (a boolean), and 'unwrap' (a boolean). 'key' is a string representing a key or attribute. No additional properties are allowed at the root level.
