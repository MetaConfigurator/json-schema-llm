The schema describes pagination configuration. It is an object type. The main properties of the schema are: 'requestElements' is a number and required, representing the number of elements requested per page. 'keyVisible' is a number and optional, possibly indicating a specific visibility key for pagination. 'requestPage' is a number and required, indicating the requested page number. 'totalPages' is a number and optional, representing the total number of pages available. 'totalElements' is a number and required, denoting the total number of elements across all pages. 'isFirstPage' is a boolean and optional, indicating whether the current page is the first page. 'isLastPage' is a boolean and optional, indicating whether the current page is the last page. 'elementsInPage' is a boolean and optional, possibly indicating if elements are present on the current page. 'keysOfPages' is an array of numbers and optional, defaulting to an empty array, representing the keys for different pages. No additional properties are allowed at the root level.
