The schema describes a POST request to create a new project. It is an object type. The main properties of the schema are: 'description' is a string required field describing the project. 'members' is an array of strings that is optional, allowing a minimum of 0 items with unique entries. 'name' is a string required field for the project name. 'owner' is a string optional field representing the project owner's name. No additional properties are allowed at the root level.
