The schema describes a PaperHive annotation schema for POSTing. It is an object type. The main properties of the schema are: '$schema' is a string, required, with the allowed value 'http://paperhive.org/api/v0/schemas/annotation_post.json#'. 'publication_id' is a string, required. 'target' is an object, required, that must satisfy one of the following conditions: it must have a property 'type' with the allowed value 'FragmentSelector', a 'page' property which is an integer with a minimum value of 1, and 'x', 'y', 'w', and 'h' properties which are numbers. 'body' is an object, required, that must have a property 'type' with the allowed value 'markdown', and a 'value' property which is a string. The 'language' property is optional and is a string. No additional properties are allowed at the root level.
