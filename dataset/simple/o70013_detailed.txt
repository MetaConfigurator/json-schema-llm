The schema describes an array of objects. It is an array type. The main properties of the schema are: 'items' is an object type that must contain the properties 'code' and 'justification' as required fields. 'regex' is a boolean type with a default value of 'false'. 'code' is a string type that is required. 'old' is a string type that is optional. 'new' is a string type that is optional. 'justification' is a string type that is required. Additional properties of the objects may be any string type. No additional properties are allowed at the root level.
