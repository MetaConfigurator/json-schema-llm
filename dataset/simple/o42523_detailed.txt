The schema describes tagging functionality. It is an object type. The main properties of the schema are: 'tags' is an object that categorizes data or tracks references in external systems. It is optional and allows for a map where keys must match the pattern '^[a-zA-Z_]{1}[\w\.:-]+$' and values must be strings matching the pattern '^[\w\/\.:-]+$'. Additionally, no additional properties are allowed for keys in the 'tags' object. No additional properties are allowed at the root level.
