The schema describes an edge in a graph. It is an object type. The main properties of the schema are: 'from_ref' is a string matching the pattern '^[\w\/\.:-]+$', and it is required. 'to_ref' is a string matching the pattern '^[\w\/\.:-]+$', and it is required. 'multiplicity' is a string with a default value of 'multi' and it can be one of the following values: 'unknown', 'multi', 'simple', 'many2one', 'one2many', 'one2one'. 'created_at' is a string matching the pattern '^[1-9]{1}[0-9]{12,15}$'. No additional properties are allowed at the root level.
