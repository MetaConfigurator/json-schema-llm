The schema describes an array of device objects. It is an array type. The main properties of the schema are: [type] is a string, required; [properties] is an object, required, with the following required properties: [Device_Name] is a string; [Device_Code_Name] is a string; [Device_Maker] is a string; [Device_Pointing_Method] is a string; [Device_Brand_Name] is a string. [standard] is a boolean, required. The schema enforces a minimum of 1 item in the array and all items must be unique. No additional properties are allowed at the root level.
