The schema describes an array of test periods. It is an array type. The main properties of the schema are: 'name' is a string that describes the name of this test period. It is optional. 'periodType' is a string that describes the type of period. It is required and can take one of the following values: 'RampUp', 'RampDown', 'Constant'. 'users' is an integer that describes how many users this period affects. It is optional and must be a minimum of 0. 'durationMinutes' is a number that describes how long this period should run for in minutes. It is required, must be greater than 0, and cannot be equal to 0. No additional properties are allowed at the root level.
