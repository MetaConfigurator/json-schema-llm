The schema describes the data contained in a DonutChart. It is an object type. The main properties of the schema are: 'datasets' is an array of objects representing the datasets for the DonutChart. It is required. Each object in the array must contain a 'value' which is a number and is required. The 'name' is an optional string, while 'color' is an optional string that must match the pattern '^#[0-9a-fA-F]{6}$'. No additional properties are allowed at the root level.
