The schema describes the general application settings. It is an object type. The main properties of the schema are: 'AppName' is a string that represents the application name. It is required and has a default value of 'InfinniPlatform'. 'AppInstance' is a string that represents the application instance identifier. It is optional. 'StaticFilesMapping' is an object that represents the relative request paths that map to static resources. It is optional and does not define any properties. No additional properties are allowed at the root level.
