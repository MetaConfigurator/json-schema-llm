The schema describes a coupon created webhook. It is an object type. The main properties of the schema are: 'code' is a string that represents the code to be shared, which is required. 'dateCreated' is an integer that indicates the date this code was created, which is also required. 'programId' is a string or null that denotes the Id of the program this code was created for, which is optional. No additional properties are allowed at the root level.
