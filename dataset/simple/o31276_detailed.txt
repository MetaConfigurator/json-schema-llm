The schema describes a structured object for communication messages. It is an object type. The main properties of the schema are: 'protocol' is a string that is required. Its value must be 'choria:secure:reply:1'. 'message' is a string that is required. It describes YAML encoded choria:reply:1. 'hash' is a string that is required. It describes a SHA256 hash of the message. No additional properties are allowed at the root level.
