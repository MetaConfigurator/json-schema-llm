The schema describes the impact on operations if this construct were to be realized. It is an object type. The main properties of the schema are: 'level' is an integer representing the estimated severity of the impact, which is optional and must be between 0 and 10. 'intended_effects' is an array of strings listing the intended effects, which is optional and can include values such as 'Military Advantage', 'Economic Advantage', 'Political Advantage', 'Intellectual Property Theft', 'Identify Theft', 'Brand Damage', 'Degredation of Service', 'Denial and Deception', 'Destruction', 'Disruption', 'Exposure', 'Extortion', 'Fraud', 'Harassment', and 'Watch the World Burn'. 'description' is a string providing a prose description of the impact, which is optional. 'credibility' is an integer representing the credibility of this statement on the Admirality scale, which is optional and must be between 0 and 6. No additional properties are allowed at the root level.
