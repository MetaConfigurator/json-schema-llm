The schema describes a configuration object for storing key-value pairs. It is an object type. The main properties of the schema are: 'keyName' is a string that must have a minimum length of 1 and is required. 'valuePath' is a string that must have a minimum length of 1 and is required. 'defaultValue' is a string that must have a minimum length of 1 and is optional. No additional properties are allowed at the root level.
