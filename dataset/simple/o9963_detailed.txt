The schema describes a response format. It is an object type. The main properties of the schema are: 'type' is a string optional; 'message' is a string optional; 'validationErrors' is an array optional, containing items that are objects with properties: 'type' is a string optional; 'name' is a string optional; 'id' is either a string that matches the pattern '^[A-Fa-f\d]{24}$' or a string; 'message' is a string optional. No additional properties are allowed at the root level.
