The schema describes an IconSymbol3DLayer resource. It is an object type. The main properties of the schema are: 'dataURI' is a string that represents an image encoded as a base64 string, starting with `data:image/`, and must match the pattern '^data:image/(.|\n|\r)+$'. 'href' is a string that serves as a URL to the returned image, which can be either an absolute URL matching the pattern '^https?://.+$' or a relative path starting with './' that matches the pattern '^\./.+$'. 'primitive' is a string that specifies the type of symbol used, with allowed values being 'circle', 'cross', 'kite', 'square', 'triangle', and 'x'. The schema requires one of the following configurations: the presence of 'primitive', 'href', or 'dataURI'. No additional properties are allowed at the root level.
