The schema describes player statistics in a game. It is an object type. The main properties of the schema are: 'TotalTimePlayed' is a string, required; 'TotalMatchesStarted' is an integer, required; 'TotalMatchesCompleted' is an integer, required; 'TotalMatchesWon' is an integer, required; 'TotalMatchesLost' is an integer, required; 'TotalLeaderPowersCast' is an integer, required. Each property in the nested object must adhere to the specified type. No additional properties are allowed at the root level.
