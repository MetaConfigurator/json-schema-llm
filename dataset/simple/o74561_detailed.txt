The schema describes the clique memberships of each terrain in a Wangscape configuration. It is an object type. The main properties of the schema are: '[0-9a-zA-Z_-]+' is an array that lists all the cliques which contain a certain terrain. It is optional and it must match the pattern '^[0-9a-zA-Z_-]+$'. Each item in this array is a clique that is also an array, containing one or more terrain IDs. Each terrain ID is a string that specifies a terrain type and must match the pattern '^[0-9a-zA-Z_-]+$'. No additional properties are allowed at the root level.
