The schema describes an extension that allows issuers to add signature lines to the visual representation of the badge. It is an object type. It defines several reusable components under 'definitions': 'ImageUri' is a string, allowing either a data URI in the format 'data:image/png;base64,' or an IRI (typically HTTP) for representing the signature image. The main properties of the schema are: 'image' is a string and required, referencing the 'ImageUri' definition; 'jobTitle' is a string and optional, describing the job title of the signer; 'name' is a string and optional, describing the full name of the signer. No additional properties are allowed at the root level.
