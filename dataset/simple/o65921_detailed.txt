The schema describes a collection of user objects. It is an array type. The main properties of the schema are: 'name' is a string with a minimum length of 1 and is required. 'password' is a string and is required. 'gid' is an integer or null with a minimum value of 0 and is required. 'users' is an array of strings with a minimum length of 1 for each item and is required. No additional properties are allowed at the root level.
