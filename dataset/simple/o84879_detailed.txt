The schema describes player match data. It is an object type. The main properties of the schema are: 'leftPlayerName' is a string which is the value for Left Player Name, required, with a default of an empty string. 'leftPlayerCharacters' is a string which is the value for Left Player Characters, required, with a default of an empty string. 'leftPlayerScore' is a number which is the value for Left Player Score, required, with a default of 0. 'rightPlayerName' is a string which is the value for Right Player Name, required, with a default of an empty string. 'rightPlayerCharacters' is a string which is the value for Right Player Characters, required, with a default of an empty string. 'rightPlayerScore' is a number which is the value for Right Player Score, required, with a default of 0. No additional properties are allowed at the root level.
