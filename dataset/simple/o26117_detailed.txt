The schema describes a list of squash training activities. It is an array type. The main properties of the schema are: Each item in the array is an object with the following properties: 'id' is a string and is required; it serves as a unique identifier for the activity. 'name' is a string and is required; it represents the name of the activity. 'shortDescription' is a string and is required; it provides a brief description of the activity. 'description' is a string and is required; it contains a full description of the activity. No additional properties are allowed at the root level.
