The schema describes container image build configuration. It is an object type. It defines several reusable components under 'definitions': 'platform' is a string, representing a platform identifier. 'platforms' is a oneOf schema that can be either a platform (a string) or an array of platforms (an array of strings). The main properties of the schema are: 'platforms' is an object with optional properties: 'only' which references the 'platforms' definition, and 'not' which also references the 'platforms' definition. No additional properties are allowed at the root level.
