The schema describes a user profile. It is an object type. The main properties of the schema are: 'email' is a string and is required. 'firstName' is a string and is required. 'lastName' is a string and is required. 'tags' is an array of strings and is optional, where each item in the array must be a string. 'favoriteTopic' is a string and is optional. The presence of 'favoriteTopic' necessitates that 'tags' must also be provided. No additional properties are allowed at the root level.
