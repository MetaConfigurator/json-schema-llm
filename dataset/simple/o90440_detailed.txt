The schema describes an object structure. It is an object type. The main properties of the schema are: 'comment' is a string optional field providing additional comments. 'compression' is an integer optional field representing the compression level. 'encoding' is a string optional field indicating the encoding type. 'mimeType' is a string required field specifying the MIME type. 'size' is an integer required field indicating the size of the object. No additional properties are allowed at the root level.
