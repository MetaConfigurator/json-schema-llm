The schema describes shifting a symbol along the vertical world axis by a given length. It is an object type. The main properties of the schema are: 'maxWorldLength' is a number that describes the maximum vertical symbol lift in world units. It acts as an upper bound to avoid lift becoming too big. 'minWorldLength' is a number that describes the minimum vertical symbol lift in world units. It acts as a lower bound to avoid lift becoming too small and has a default value of 0. 'screenLength' is a number that describes the maximal screen length of lift in points. The property 'screenLength' is required. No additional properties are allowed at the root level.
