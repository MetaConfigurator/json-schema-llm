The schema describes filters for a search. It is an array type. The main properties of the schema are: 'name' is a string that represents the name of the passenger and is required. 'from' is a string that indicates the lower limit of the filter and is optional. 'to' is a string that signifies the upper limit of the filter and is optional. 'companies' is an array of strings, representing the operators from which packages are to be obtained, and is optional. Each item in the 'companies' array is a string that denotes the name of the operator. No additional properties are allowed at the root level.
