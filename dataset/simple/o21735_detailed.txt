The schema describes a structured object for specifying donation URLs, a license code, and patterns. It is an object type. The main properties of the schema are: 'donate' is an array of strings specifying donation URLs, required, with a minimum of 2 and maximum of 2 items, each must be one of the following: 'https://pledgie.com/campaigns/28037', 'https://gratipay.com/aminassian', and contains unique items. 'licence' is a string that specifies the license code, required. 'patterns' is an array of objects, each item must have a 'pattern' string and a 'flag' string, where 'flag' must be one of the following: '+', '-'. No additional properties are allowed at the root level.
