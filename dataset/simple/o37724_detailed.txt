The schema describes an Image. It is an object type. The main properties of the schema are: 'doc_type' is a string optional. 'title' is a string optional. 'mimeType' is a string required, with allowed values 'image/jpeg' and 'image/png'. 'data' is a string required, representing Image Data, with media constraints indicating it must be base64 encoded and have allowed values 'image/jpeg' and 'image/png'. No additional properties are allowed at the root level.
