The schema describes a count of logs. It is an object type. The main properties of the schema are: 'email' is a string, required, formatted as an email. 'filter' is an array, required, with items constrained to the enum values: 'sent_at', 'delivered_at', 'clicked_at', 'opened_at', 'dropped_at', 'spam_complained_at', 'hard_bounced_at'. 'from' is a string, optional, formatted as date-time, with the title 'From filter'. 'to' is a string, optional, formatted as date-time, with the title 'To filter'. No additional properties are allowed at the root level.
