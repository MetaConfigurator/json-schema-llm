The schema describes configuration for the json2ts tool. It is an object type. The main properties of the schema are: '$schema' is a string that specifies the schema version. It is optional. '$tool' is a string that must be set to json2ts. It is required. 'dist' is a string that indicates the output file path (relative to the pwd), supporting .ts files. It is required. 'src' is an array of strings specifying input JSON files (Glob), relative to the pwd. It is required and must contain at least one item. No additional properties are allowed at the root level.
