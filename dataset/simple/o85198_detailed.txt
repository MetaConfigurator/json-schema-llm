The schema describes a MWF logo component's definition. It is an object type. The main properties of the schema are: 'content' is an object that represents the logo component's content. It is required and has the following properties: 'href' is a string that is optional; 'image' is an object that represents the logo image and is required to have 'src' and 'alt' as properties; 'label' is a string that is optional; 'logoType' is a string that is required. The 'content' object has a dependency where 'href' requires 'label' to be present if 'href' is used. No additional properties are allowed at the root level.
