The schema describes an object structure used for assertions. It is an object type. The main properties of the schema are: 'description' is a string that is optional. 'context' is an optional property with no defined type. 'assertions' is an array of objects, which is required and must contain at least one item. Each item must be an object that includes required properties: 'input' and 'output', both of which have no defined type. No additional properties are allowed at the root level.
