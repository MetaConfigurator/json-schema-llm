The schema describes a base structure for the IIJmio Coupon API. It is an object type. It defines several reusable components under 'definitions': 'couponUse' is a boolean, which indicates whether a coupon is in use (true) or not (false). 'hddServiceCode' is a string representing the HDD service code. 'hdoServiceCode' is a string representing the HDO service code. 'plan' is a string with an enum of allowed values: 'Family Share', 'Minimum Start', 'Light Start'. 'returnCode' is a string indicating the return code.
