The schema describes a list of formulas. It is an array type. It defines several reusable components under 'definitions': The main properties of the schema are: Each item is an object containing: 'input' is a string, required, representing the formula input. 'inputhash' is a string, required, representing the md5 hash of the input. 'type' is a string, optional, constrained to match the pattern '(LATEX|MATHML|MATH_TEMPLATE|MVAR_TEMPLATE|LATEXII|LATEXCE)'. No additional properties are allowed at the root level.
