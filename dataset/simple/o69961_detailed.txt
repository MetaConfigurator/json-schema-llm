The schema describes the structure of fetching engines request output. It is an array type. It defines several reusable components under 'definitions': 'result' is an object, Founded engine. It requires 'name' and 'found' fields. The 'name' field is a string, described as 'The engine name'. The 'found' field is a string, described as 'The way how engine founded', with allowed values being 'sourceExtension', 'engineConfig', and 'projectConfig'. The main properties of the schema are: The items in the array follow the referenced definition. No additional properties are allowed at the root level.
