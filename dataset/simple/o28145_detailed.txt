The schema describes a description of the confidence a Seller has in delivering services. It is an object type. The main properties of the schema are: 'color' is a string indicating the confidence, required; allowed values are 'GREEN', 'YELLOW', 'RED'. 'estimatedResponseDate' is a string that indicates when the Seller plans to provide a more complete response if the color is 'YELLOW', optional; it must be formatted as YYYY-MM-DD or YYYY-MM-DDThh:mmTZD. 'objectType' is a string indicating the type of the object, required; allowed value is 'serviceabilityConfidence'. 'reason' is a string that describes the reason for the color provided, optional; it has a maximum length of 100 characters. No additional properties are allowed at the root level.
