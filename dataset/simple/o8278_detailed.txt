The schema describes a Taxonomy. It is an object type. The main properties of the schema are: 'vocabularyId' is an integer the unique identifier for the vocabulary, required. 'vocabularyName' is a string the vocabulary name, required. 'vocabularyValues' is an array types allowed in this vocabulary, required, with a minimum of 1 item and unique items, where each item is an object containing 'valueId' (an integer the unique identifier for the vocabulary type, required) and 'valueName' (a string the value the vocabulary type, required). No additional properties are allowed at the root level.
