The schema describes an object structure for categorizing items. It is an object type. The main properties of the schema are: 'itemType' is a string that must be one of the following values: 'application', 'device', 'flow', 'dashboard', 'organization'. 'parentId' is a string that must match the pattern '^[A-Fa-f\d]{24}$'. 'items' is an array of objects, where each object contains the properties: 'id' which is a string that must match the pattern '^[A-Fa-f\d]{24}$', and 'name' which is a string with a minimum length of 1 and a maximum length of 255. No additional properties are allowed at the root level.
