The schema describes an object representing an organization's member details. It is an object type. The main properties of the schema are: 'orgName' is a string that must have a minimum length of 1 and a maximum length of 255. 'email' is a string formatted as an email and has a maximum length of 1024. 'role' is a string with an enum of allowed values: 'admin', 'edit', 'collaborate', 'view', and 'none'. 'inviteDate' is a string formatted as a date-time. 'ttl' is a number. No additional properties are allowed at the root level.
