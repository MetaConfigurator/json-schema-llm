The schema describes a newsletter subscription form. It is an object type. The main properties of the schema are: 'name' is a string with a minimum length of 8 and a maximum length of 80, and is required. 'email' is a string with a maximum length of 120, and is required. 'lists' is a string with an enum value that can be either 'Daily New' or 'Promotion', and is required. No additional properties are allowed at the root level.
