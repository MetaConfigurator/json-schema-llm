The schema describes an array of objects. It is an array type. The main properties of the schema are: Each item in the array is an object with the following properties: 'id' is a number and required. 'name' is a string and required. 'description' is a string and required. 'icon' is a string and required. 'ranks' is an array of objects, where each rank object must have a 'cost' (a number and required) and an 'effect' (a string and required). No additional properties are allowed at the root level.
