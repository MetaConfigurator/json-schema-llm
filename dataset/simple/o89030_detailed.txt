The schema describes an error response. It is an object type. The main properties of the schema are: 'code' is a string that is required and must be one of the following values: 'not_found', 'object_not_found', 'object_exists', 'syntax_error', 'validation_error', 'unknown_error'. 'message' is a string that is required. 'detail' is an object that can have any number of properties, and each property must be a string. No additional properties are allowed at the root level.
