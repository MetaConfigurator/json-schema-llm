The schema describes sports activity registration for the 12-14 age bracket. It is an object type. The main properties of the schema are: 'name' is a string and is required. 'sex' is a string defined with an enum of allowed values 'M' and 'F' and is required. 'age' is an integer that must be between 12 and 14 (inclusive) and is required. No additional properties are allowed at the root level.
