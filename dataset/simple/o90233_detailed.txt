The schema describes how location information will be retrieved from a CSV file referenced through the web, i.e., referenced by URL. It is an object type. The main properties of the schema are: 'latitudeFieldName' is a string defining the field name that holds the latitude (Y) coordinate. It is optional. 'locationType' is a string value indicating location type. It is required and must be one of the enumerated values: 'coordinates'. 'longitudeFieldName' is a string defining the field name that holds the longitude (X) coordinate. It is optional. No additional properties are allowed at the root level.
