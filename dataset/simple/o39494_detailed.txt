The schema describes a schema for an app client. It is an object type. The main properties of the schema are: 'app_id' is a string and represents a unique identifier for the app in UUID format. It is required. 'id' is a string and represents a unique identifier for the client in UUID format. It is required. 'label' is a string and denotes the label of the client with a maximum length of 64 characters. It is required. No additional properties are allowed at the root level.
