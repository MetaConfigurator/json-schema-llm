The schema describes parameter selection. It is an array type. The main properties of the schema are: Each item in the array is an object containing the properties 'key' and 'value'. 'key' is a string, required. 'value' is a type that can be either a string, an array, a number, or an object requiring the 'expression_type' property. If 'value' is an object, it has the properties 'output' (string), 'expression_type' (a string with a default value of 'stage-output-selector' and an enum value of 'stage-output-selector'), 'stages' (string), 'flatten' (boolean), and 'unwrap' (boolean). No additional properties are allowed at the root level.
