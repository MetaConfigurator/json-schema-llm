The schema describes a capability for color temperature. It is an object type. The main properties of the schema are: 'valueType' is a number representing the color temperature in Kelvin (K) and is optional. 'writable' is a boolean indicating that the property is not writable. The actions section defines a single action: 'setcolortemperature' with type 'st:setcolortemperature'. This action takes input data of type object with a required property 'colortemperature', which is a number in Kelvin (K). The action has a hrefs array with a single value 'colortemperatureActions'. No additional properties are allowed at the root level.
