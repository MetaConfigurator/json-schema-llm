The schema describes an email message structure. It is an object type. The main properties of the schema are: 'toEmail' is an array of strings representing the recipient's email addresses, required, with a minimum of 1 and a maximum of 10 items, each formatted as an email and having a maximum length of 1024 characters. 'subject' is an optional string with a maximum length of 255 characters. 'message' is an optional string with a maximum length of 32767 characters. 'theme' is an optional string that must be one of the following values: 'dark' or 'light'. 'time' is an optional number. 'timezone' is an optional string with a maximum length of 255 characters. No additional properties are allowed at the root level.
