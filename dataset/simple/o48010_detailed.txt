The schema describes information needed for files in records. It is an object type. The main properties of the schema are: 'key' is a string optional field. 'file_id' is a string optional field. 'bucket' is a string optional field. 'checksum' is a string optional field. 'size' is an integer optional field. 'version_id' is a string optional field. No additional properties are allowed at the root level.
