The schema describes common json schema for time-based event aggregation records (datum). It is an object type. The main properties of the schema are: '_id' is a string unique url-friendly base64 encoded, utf8 encoded concatenation identifier. It is required. '_timestamp' is a string ISO8601 combined date and time in UTC e.g. '2014-12-01T00:00:00+00:00'. It is required. 'period' is a string representing the period covered by each data point from the _timestamp. It is required and must be one of the following values: 'hour', 'day', 'week', 'month', 'quarter'. No additional properties are allowed at the root level.
