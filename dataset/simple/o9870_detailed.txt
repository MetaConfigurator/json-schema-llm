The schema describes the configuration for SSL settings. It is an object type. The main properties of the schema are: 'sslKey' is a string or null that must have a minimum length of 50 and a maximum length of 32767. 'sslCert' is a string or null that must have a minimum length of 50 and a maximum length of 32767. 'sslBundle' is a string or null that must have a minimum length of 50 and a maximum length of 32767. 'version' is a string or null that must have a minimum length of 1 and a maximum length of 255. 'domainName' is a string that must have a minimum length of 3 and a maximum length of 1024. No additional properties are allowed at the root level.
