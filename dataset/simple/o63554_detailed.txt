The schema describes a plugin manifest for a Jigsaw plugin. It is an object type. The main properties of the schema are: 'name' is a string A unique name for the plugin, required. 'dependencies' is an array A list of plugins that must be loaded before this plugin can be loaded, optional, default is an empty array. Each item in the array is a string representing the name of a required plugin. 'module_name' is a string The name of the module that will be created from importing this plugin, optional. 'path' is a string The path of the file that contains the main plugin class, optional, default is '__init__.py'. 'main_class' is a string The name of the main plugin class, optional, default is 'Plugin'. No additional properties are allowed at the root level.
