The schema describes atomic-reactor plugins configuration. It is an object type. It defines several reusable components under 'definitions': 'base_plugins_phase' is an array, containing items that are objects with a required field 'name' of type string and an optional field 'args' of type object. 'general_plugins_phase' is an allOf composition that combines 'base_plugins_phase' and includes additional properties: 'is_allowed_to_fail' of type boolean and 'required' of type boolean. The main properties of the schema are: 'prebuild_plugins' is a general_plugins_phase, which is a required field. 'buildstep_plugins' is a base_plugins_phase, which is a required field. 'postbuild_plugins' is a general_plugins_phase, which is a required field. 'prepublish_plugins' is a general_plugins_phase, which is a required field. 'exit_plugins' is a general_plugins_phase, which is a required field. No additional properties are allowed at the root level.
