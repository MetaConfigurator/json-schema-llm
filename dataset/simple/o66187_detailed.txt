The schema describes a configuration for a string interpolated script. It is an object type. The main properties of the schema are: 'interpreter' is a string that specifies the interpreter to use; it is optional and has a default value of 'sh'. 'process_type' is a string that specifies the type of process; it is required and can only take the value 'interpolated-script-cmd'. 'script' is a string that contains the script; it is required. No additional properties are allowed at the root level.
