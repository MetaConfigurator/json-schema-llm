The schema describes an instance of a hotspot transform rotation. It is an object type. The main properties of the schema are: 'x' is a number that represents rotation on x in the local hotspot system; it is optional. 'y' is a number that represents rotation on y in the local hotspot system; it is optional. 'z' is a number that represents rotation on z in the local hotspot system; it is optional. No additional properties are allowed at the root level.
