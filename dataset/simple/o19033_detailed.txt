The schema describes number and type of units. It is an object type. The main properties of the schema are: 'type' is a string that describes the type of measurement. It is required and must be one of the following values: 'distance', 'length', 'angle', 'measure', 'scale'. 'unitOfMeasure' is a string that describes the type of units to associate with the value. It is required and must have a minimum length of 1. 'value' is a number that describes the number of units in the measurement. It is required. No additional properties are allowed at the root level.
