The schema describes a URI structure. It is an object type. The main properties of the schema are: 'fragment' is a string optional field. 'host' is a string that must be a valid hostname and is an optional field. 'path' is a string optional field. 'port' is an integer that must be 0 or greater and is an optional field. 'query' is a string optional field. 'scheme' is a string optional field. 'userinfo' is a string optional field. No additional properties are allowed at the root level.
