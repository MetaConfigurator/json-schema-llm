The schema describes a data object containing search result information. It is an object type. The main properties of the schema are: 'count' is a string representing the number of resources in the data object and is required. 'limit' is a string that indicates the search limit used in the request and is required. 'offset' is a string that shows the search offset used in the request and is required. 'total' is a string for the total number of search results without limit being applied and is required. No additional properties are allowed at the root level.
