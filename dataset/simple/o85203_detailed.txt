The schema describes a MWF paragraph component's definition. It is an object type. The main properties of the schema are: 'config' is an object that contains a required property 'type', which is an integer and must be one of the following values: 1, 2, 3, or 4. 'content' is an object that may contain a property 'text', which is a string. Both 'config' and 'content' are required. No additional properties are allowed at the root level.
