The schema describes a JSON Schema identifier. It is an object type. The main properties of the schema are: 'namespace' is a string that serves to disambiguate schemas with conflicting names. It is required. 'name' is a string representing the name of the schema. It is required. 'version' is a string that indicates the version of the schema, e.g. 1.0. It is required. 'url' is a string that provides a URL to retrieve the schema. It is optional and formatted as a URI. No additional properties are allowed at the root level.
