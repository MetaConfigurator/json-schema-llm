The schema describes the data contained in a Table Stream update object. It is an object type. The main properties of the schema are: 'stream' is an array that contains items, each of which is an object with a 'row' property. The 'row' property is an array of objects that require a 'value' field. Each object in the 'row' can have optional properties 'color' and 'background', both of which must match the pattern '^#[0-9a-fA-F]{6}$' if present. No additional properties are allowed at the root level.
