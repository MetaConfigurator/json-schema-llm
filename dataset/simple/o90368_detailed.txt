The schema describes properties pertaining to a Bastion. It is an object type. The main properties of the schema are: 'name' is a string, representing the name of the Bastion configuration, and it is required. 'kind' is a string, representing the type of bastion config, with an allowed value of 'bastion', and it is required. 'listenPort' is a number, indicating what port number to open for the ssh tunnel, with a default value of 22, and it must be between 0 and 65535. No additional properties are allowed at the root level.
