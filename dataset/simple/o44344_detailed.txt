The schema describes an announcement structure. It is an object type. The main properties of the schema are: 'created' is a string representing the creation date-time, optional; 'modified' is a string representing the modification date-time, optional; 'id' is a string, required; 'category' is a string, required; 'priority' is a string, required; 'content' is an object containing at least one property, required, with all properties being strings and no additional properties allowed. No additional properties are allowed at the root level.
