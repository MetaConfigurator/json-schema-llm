The schema describes info files for news items. It is an object type. The main properties of the schema are: 'uuid' is a string that serves as a unique identifier (UUID v4), with a pattern constraint of '^[0-9a-fA-F]{8}-[0-9a-fA-F]{4}-[0-9a-fA-F]{4}-[0-9a-fA-F]{4}-[0-9a-fA-F]{12}$', and it is required. 'title' is a string that represents the title of the news item, and it is required. 'author' is a string that denotes the author of the news item, and it is optional. 'visible_to_students' is a boolean that indicates whether the news item should be shown to students, and it is required. No additional properties are allowed at the root level.
