The schema describes a wiki configuration. It is an object type. The main properties of the schema are: 'id' is a string representing the uuid of the wiki. It is required. 'mode' is a string or null indicating the wiki's mode, with allowed values '0', '1', or '2'. It is optional. 'display' is an object containing properties: 'sectionNumbers' is a boolean representing the option to display/hide the wiki's section number and 'contents' is a boolean representing the option to display/hide the wiki's contents. Both properties are optional. No additional properties are allowed at the root level.
