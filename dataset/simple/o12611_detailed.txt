The schema describes ParaDrop host configuration - Wi-Fi interface section. It is an object type. The main properties of the schema are: 'device' is a string representing the physical device used by this interface, must match a device id in the wifi section; 'encryption' is a string indicating the type of wireless network security to use, e.g. none, psk2, wpa2 (Enterprise); 'mode' is a string operating mode for the interface, allowed values are 'airshark', 'ap', 'managed', 'monitor'; 'network' is a string specifying the network name the interface should be attached to, typically lan for ap mode interfaces; 'ssid' is a string for ESSID for ap and managed mode interfaces. No additional properties are allowed at the root level.
