The schema describes the MCDA survival distribution for the performance table entry of absolute data. It is an object type. The main properties of the schema are: 'type' is a string required, with an enum of allowed values: 'dsurv'. 'parameters' is an object required, with additional properties not allowed. It requires the fields 'alpha', 'beta', and 'summaryMeasure'. 'alpha' is a number. 'beta' is a number. 'summaryMeasure' is a string required, with an enum of allowed values: 'mean', 'median', 'survivalAtTime'. 'time' is a number optional. No additional properties are allowed at the root level.
