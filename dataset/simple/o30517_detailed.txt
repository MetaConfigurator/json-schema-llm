The schema describes IPsec parameters. It is an object type. The main properties of the schema are: 'encalgo' is an object specifying the encryption algorithm, with allowed values being 'AES', '3DES', 'AES192', 'AES256'; 'hashalgo' is an object specifying the hash algorithm, with allowed values being 'HMAC_SHA1', 'HMAC_SHA256', 'HMAC_SHA384', 'HMAC_SHA512', 'HMAC_MD5'; 'ikeretryinterval' is an integer specifying the interval for IKE retries; 'ikeversion' is a string specifying the IKE version, with allowed values being 'V1', 'V2'; 'lifetime' is an integer specifying the lifetime duration; 'livenesscheckinterval' is an integer specifying the interval for liveness checks; 'perfectforwardsecrecy' is a string specifying PFS setting, with allowed values being 'ENABLE', 'DISABLE'; 'replaywindowsize' is an integer specifying the replay window size; 'responderonly' is a string specifying responder-only mode, with allowed values being 'YES', 'NO'; 'retransmissiontime' is an integer specifying the retransmission time. No additional properties are allowed at the root level.
