The schema describes a list of timed Launch movement actions. It is an array type. The main properties of the schema are: [items] is an object that represents an action, which is required and contains three properties: 'at' is an integer indicating the time in milliseconds when the action is executed, with a minimum value of 0; 'pos' is an integer indicating the position to move to, with a minimum value of 0 and a maximum value of 99; 'spd' is an integer indicating the speed to move at, with a minimum value of 0 and a maximum value of 99. No additional properties are allowed at the root level.
