The schema describes the configuration of a Weave network fabric. It is an object type. The main properties of the schema are: 'network' is a string that specifies the Network CIDR for pods in the fabric. It is required. 'nodeConnectionLimit' is a number that indicates the maximum number of connections made per node, with a minimum value of 1 and a default value of 30. It is required. No additional properties are allowed at the root level.
