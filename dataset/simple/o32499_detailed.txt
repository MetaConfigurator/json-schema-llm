The schema describes a scale with labels. It is an object type. The main properties of the schema are: 'id' is a string that describes the scale uuid and is required. 'name' is a string that describes the scale name, has a minimum length of 1, and is required. 'labels' is an array of objects, where each object contains 'id' as a string (the level uuid) and 'value' as a string (the level name). No additional properties are allowed at the root level.
