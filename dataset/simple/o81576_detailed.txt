The schema describes a structure for a speaker. It is an object type. The main properties of the schema are: 'speaker' is an object that contains speaker details. It is required and includes the following properties: 'firstName' is a string representing the speaker's first name. It is required. 'lastName' is a string representing the speaker's last name. It is required. 'email' is a string representing the speaker's email address. It is required. 'postedSlides' is a boolean indicating whether the speaker has posted slides. It is required. 'rating' is a number representing the speaker's rating. It is required. 'tags' is an array of strings representing tags associated with the speaker. It is required and does not allow additional items. No additional properties are allowed at the root level.
