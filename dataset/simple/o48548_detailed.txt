The schema describes parameters for a key derivation function (KDF). It is an object type. The main properties of the schema are: 'kdf' is a string that is required and can only be 'scrypt'. 'kdfparams' is an object that is required and must contain the following properties: 'dklen' is an integer with a minimum value of 1; 'n' is an integer with a minimum value of 1; 'p' is an integer with a minimum value of 1; 'r' is an integer with a minimum value of 1; 'salt' is a string that must match the pattern '^(([0-9]|[a-f]){2}){0,32}$'. No additional properties are allowed at the root level.
