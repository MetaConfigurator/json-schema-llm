The schema describes the configuration for Nebular Theme ng-add schematics. It is an object type. The main properties of the schema are: 'animations' is a boolean that indicates whether Angular browser animations should be set up or not. It is optional and defaults to true. 'customization' is a boolean that determines whether to use prebuilt css styles or import customizable scss. It is optional and defaults to true. 'layout' is a boolean that specifies whether the root component has to be wrapped in nb-layout or not. It is optional and defaults to true. 'project' is a string that represents the name of the project. It is optional and derives its default value from 'projectName'. 'theme' is a string that defines the visual theme to be installed. It is optional and defaults to 'default'. The allowed values are 'default', 'dark', 'cosmic', and 'corporate'. No additional properties are allowed at the root level.
