The schema describes a home page structure. It is an object type. The main properties of the schema are: 'title' is a string and is required. 'description' is a string and is required. 'image' is a string and is optional. 'order' is an integer and is required. 'content' is an object and is optional, containing a property 'helpTopics', which can either be an array of objects or null. Each object in the 'helpTopics' array must have a required 'title' of type string and a required 'url' of type string. Additionally, there are no additional properties allowed at the root level.
