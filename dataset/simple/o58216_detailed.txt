The schema describes free-form, usually human-readable, text. It is an object type. The main properties of the schema are: 'format' is a string that indicates how the content should be parsed. It is optional and can take the values 'text', 'html', or 'asciidoc'. 'href' is a string that references a related document containing human readable text. It is optional. 'content' is a string that represents the text contents. It is optional. No additional properties are allowed at the root level.
