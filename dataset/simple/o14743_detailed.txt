The schema describes the configuration for connecting to Twitch chat channels and a streen instance. It is an object type. The main properties of the schema are: 'channels' is an array which contains strings and describes which Twitch chat channels to join and monitor. It is required and has a default value of an empty array. 'streen' is an object which contains properties: 'url' is a string that describes the full URL (including http:// or https://) of the streen instance to connect to; it is required. 'secretKey' is a string that describes a pre-shared key used to authenticate with streen; it is also required. No additional properties are allowed at the root level.
