The schema describes a data structure for user information. It is an object type. The main properties of the schema are: 'entityID' is a string representing the entity identifier, 'nameID' is a string representing the name identifier, 'tas.personal.givenName' is a string representing the given name, 'tas.personal.familyName' is a string representing the family name, 'tas.personal.email' is a string representing the email address, 'tas.personal.image' is a string with a URI format representing the user's image, and 'tas.roles' is an array of strings representing user roles. No additional properties are allowed at the root level.
