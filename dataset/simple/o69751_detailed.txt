The schema describes a configuration object for hyperty policies. It is an object type. The root schema resolves to the referenced definition 'HypertyInterceptorConfiguration', which is an object, and does not define any properties. The main properties of the schema are: 'configuration' is a reference to 'HypertyInterceptorConfiguration', and 'policies' is a reference to 'HypertyPolicyList'. No additional properties are allowed at the root level.
