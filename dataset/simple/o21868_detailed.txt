The schema describes an index of available Open Badge lists. It is an object type. The main properties of the schema are: 'badgeindex' is an array container for a set of badge lists. It is required and must have at least one item, with unique items only. Each item is an object type that contains the data needed to describe a badge list. The object must have a required field 'loc', which is a string formatted as a URI, indicating the location URI of the badge list. No additional properties are allowed at the root level.
