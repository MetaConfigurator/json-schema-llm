The schema describes a user profile. It is an object type. The main properties of the schema are: 'email' is a string that is required. 'firstName' is a string that is required. 'lastName' is a string that is required. 'postedSlides' is a value that can be either a boolean or a string. If it is a string, it must be one of the following values: 'yes', 'Yes', 'no', 'No'. 'postedSlides' is required. 'rating' is a number that is required. No additional properties are allowed at the root level.
