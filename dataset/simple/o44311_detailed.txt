The schema describes an instance of the view options configuration. It is an object type. The main properties of the schema are: 'repeatX' is a boolean indicating whether the view repeats the media on the X axis (ViewFlat only), optional with a default value of 'false'. 'repeatY' is a boolean indicating whether the view repeats the media on the Y axis (ViewFlat only), optional with a default value of 'false'. No additional properties are allowed at the root level.
