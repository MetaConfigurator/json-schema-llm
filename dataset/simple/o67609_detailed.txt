The schema describes configuring a custom proxy in Postman for specific URL matches. It is an object type. The main properties of the schema are: 'match' is a string that defines the URL match for which the proxy config is applied and has a default value of 'http+https://*/*'. 'host' is a string that specifies the proxy server host and is optional. 'port' is an integer representing the proxy server port, with a minimum value of 0 and a default value of 8080. 'tunnel' is a boolean providing tunneling details for the proxy config and has a default value of false. 'disabled' is a boolean that, when set to true, ignores this proxy configuration entity, with a default value of false. No additional properties are allowed at the root level.
