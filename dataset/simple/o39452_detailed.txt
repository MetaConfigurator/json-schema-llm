The schema describes an animal profile. It is an object type. It defines several reusable components under 'definitions': 'Name' is a string, 'Age' is an integer with a minimum value of 0, 'Breed' is a string with allowed values: 'LABRADOR', 'BULLDOG', 'YORKSHIRE', and 'Date' is a string formatted as a date. The main properties of the schema are: 'name' is a string referencing the 'Name' definition, 'breed' is a string referencing the 'Breed' definition, 'age' is an integer referencing the 'Age' definition, and 'birthDate' is a string referencing the 'Date' definition. No additional properties are allowed at the root level.
