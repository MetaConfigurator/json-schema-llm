The schema describes a candidate's preferences and statuses. It is an object type. It defines several reusable components under 'definitions': 'notifications' is a boolean, whether this candidate wants to receive email alerts about new jobs. 'internalFlag' is a boolean, whether this candidate is an internal. The main properties of the schema are: 'notifications' is a boolean referring to the definition 'notifications', required. 'internalFlag' is a boolean referring to the definition 'internalFlag', required. No additional properties are allowed at the root level.
