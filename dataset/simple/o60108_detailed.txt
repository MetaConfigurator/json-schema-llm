The schema describes the value returned by GET /_active_tasks. It is an array type. The main properties of the schema are: Each item in the array is an object with the following properties: 'changes_done' is a number and is required. 'database' is a string and is required. 'pid' is a string and is required. 'progress' is a number and is required. 'started_on' is a number and is required. 'status' is a string and is required. 'task' is a string and is required. 'total_changes' is a number and is required. 'type' is a string and is required. 'updated_on' is a number and is required. No additional properties are allowed at the root level.
