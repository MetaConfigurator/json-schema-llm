The schema describes generates a new, generic end-to-end test definition for the given or default project. It is an object type. The main properties of the schema are: 'relatedAppName' is a string the name of the app being tested. It is required. 'rootSelector' is a string the HTML selector for the root component of the test app. It is optional and has a default value of 'app-root'. No additional properties are allowed at the root level.
