The schema describes an array of images. It is an array type. The main properties of the schema are: 'link' is a string that represents the URL of the image, which is required. 'alt' is a string with a maximum length of 255 characters, providing alt text for the image, which is optional. 'title' is a string with a maximum length of 255 characters, used to caption the image, which is also optional. The array must contain a minimum of 1 item, and all items must be unique. No additional properties are allowed at the root level.
