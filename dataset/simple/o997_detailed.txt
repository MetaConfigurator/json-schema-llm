The schema describes a mapping between object names and classes. It is an array type. The main properties of the schema are: 'items' is an object representing the correspondence of one object name to one class. It includes required fields: 'N', which is a string representing the object name, and 'Cl', which is a string representing the class name. The items in the array must be unique, enforced by the '_uniqueItems' constraint. No additional properties are allowed at the root level.
