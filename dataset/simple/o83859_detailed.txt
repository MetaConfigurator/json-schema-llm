The schema describes a map of available project targets. It is an object type. It defines several reusable components under 'definitions': 'options' is an object, a description of target options. 'target' is an object, a description of target options with required fields 'builder' and 'options'. The main properties of the schema are: 'additionalProperties' is an object referencing the 'target' definition, allowing for any valid target configuration. No additional properties are allowed at the root level.
