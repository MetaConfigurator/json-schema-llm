The schema describes OAuth authorization codes. It is an object type. The main properties of the schema are: 'authorizationCode' is a string with a maximum length of 100 characters. It is optional. 'clientId' is a string with a minimum length of 12 characters, a maximum length of 12 characters, and a pattern of '[0-9a-fA-F]+'. It is optional. 'expirationTime' is a number and is optional. 'scope' is a string and is optional. No additional properties are allowed at the root level.
