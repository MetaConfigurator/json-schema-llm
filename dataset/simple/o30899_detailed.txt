The schema describes SNMP options. It is an object type. The main properties of the schema are: 'partitionnameintrap' is a string that is optional and must be one of the following values: 'ENABLED', 'DISABLED'. 'snmpset' is a string that is optional and must be one of the following values: 'ENABLED', 'DISABLED'. 'snmptraplogging' is a string that is optional and must be one of the following values: 'ENABLED', 'DISABLED'. 'snmptraplogginglevel' is a string that is optional and must be one of the following values: 'EMERGENCY', 'ALERT', 'CRITICAL', 'ERROR', 'WARNING', 'NOTICE', 'INFORMATIONAL', 'DEBUG'. No additional properties are allowed at the root level.
