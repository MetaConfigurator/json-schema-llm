The schema describes a publisher configuration. It is an object type. The main properties of the schema are: 'tryExact' is a boolean that has a default value of true and is optional. 'publisher_type' is a string that is required, with an allowed value of 'fromparjq-pub'. 'glob' is a boolean that has a default value of false and is optional. 'relative_paths' is a boolean that has a default value of false and is optional. 'script' is a string that has a default value of '.' and is optional. No additional properties are allowed at the root level.
