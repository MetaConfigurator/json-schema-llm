The schema describes the performance table entry of absolute data in MCDA events with respect to sample size effect. It is an object type. The main properties of the schema are: 'type' is a string and is required, with an enum value of 'exact'; 'value' is a number and is required; 'input' is an object and is required, containing two properties: 'events' is a number and is required, and 'sampleSize' is a number and is required. No additional properties are allowed at the root level.
