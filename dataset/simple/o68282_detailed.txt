The schema describes validation for parameters handled by the Adhese adapter. It is an object type. The main properties of the schema are: 'account' is a string your Adhese account name. If unknown, please contact your sales rep. It is required. 'location' is a string the location you want to refer to for a specific section or page, as defined in your Adhese inventory. It is required. 'format' is a string the format you accept for this unit, as defined in your Adhese inventory. It is required. 'targets' is an object target params, as defined in your Adhese setup. It is optional. No additional properties are allowed at the root level.
