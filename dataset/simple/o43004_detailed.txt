The schema describes an error response format. It is an object type. The main properties of the schema are: 'error' is an object that contains details about the error. It is a required property. The 'error' object has the following required properties: 'code' is an integer that specifies the error code, 'message' is a string that provides a description of the error, and 'details' is an array of strings that gives further information about the error. No additional properties are allowed at the root level.
