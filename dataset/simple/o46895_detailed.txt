The schema describes Hypercharge get params for GET /v2/scheduler. It is an object type. The main properties of the schema are: 'active' is a boolean indicating if schedulers are active (true if the scheduler is on, false if it is turned off but still visible), 'end_date_from' is a date filtering schedulers where end_date is greater than or equal to this date, 'end_date_to' is a date filtering schedulers where end_date is less than or equal to this date, 'page' is a page value to handle pagination, 'per_page' specifies the number of items per page, 'start_date_from' is a date filtering schedulers where start_date is greater than or equal to this date, and 'start_date_to' is a date filtering schedulers where start_date is less than or equal to this date. No additional properties are allowed at the root level.
