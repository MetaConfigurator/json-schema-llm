The schema describes a valid site. It is an object type. The main properties of the schema are: 'city' is a string optional; 'closetime' is a string optional; 'days' is a string optional; 'latitude' is a number optional; 'longitude' is a number optional; 'name' is a string optional; 'opentime' is a string optional; 'sitecoordinator' is an integer optional; 'sitestatus' is a string optional; 'slug' is a string optional; 'state' is a string optional; 'street' is a string optional; 'zip' is a string optional. No additional properties are allowed at the root level.
