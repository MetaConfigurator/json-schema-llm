The schema describes the configuration settings for a train system. It is an object type. The main properties of the schema are: 'autoStartCooldown' is a boolean indicating whether or not to automatically start the cooldown when a passenger is added to the train. It is optional and has a default value of false. 'resetAfterThreshold' is a boolean indicating whether or not to reset the number of passengers on the train when the threshold is reached. It is optional and has a default value of false. 'disableThresholdEditing' is a boolean that, if true, prevents the user from editing the threshold. It is optional and has a default value of false. No additional properties are allowed at the root level.
