The schema describes artifacts to be fetched from Koji builds. It is an array type. The main properties of the schema are: 'nvr' is a string that describes the Name-Version-Release of a Koji build and is required. 'archives' is an optional array containing items that are objects. Each item represents a matcher for archives in a Koji build, where 'filename' is a string that matches by archive filename and supports glob matching, and 'group_id' is a string that matches by archive group_id. No additional properties are allowed at the root level.
