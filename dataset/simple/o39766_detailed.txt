The schema describes a specification for validating metadata related to an area of interest (AOI). It is an object type. The main properties of the schema are: 'metadata_type' is a string and is required, with a default value of 'n/a'. 'name_of_aoi' is a string and is required, with a default value of 'n/a'. 'planned_map_type' is a string and is required, with a default value of 'n/a'. 'planned_satellite_data' is a string and is required, with a default value of 'n/a'. No additional properties are allowed at the root level.
