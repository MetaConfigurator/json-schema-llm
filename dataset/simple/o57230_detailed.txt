The schema describes user authentication data. It is an object type. The main properties of the schema are: 'date_created' is a string representing the timestamp of the user account creation time, required. 'hash' is a string representing the hash of the user password, required. 'models' is an array containing strings, referencing models owned by the user, optional. 'role' is a string representing the user role of the user, required. 'username' is a string representing the username of the user, required. No additional properties are allowed at the root level.
