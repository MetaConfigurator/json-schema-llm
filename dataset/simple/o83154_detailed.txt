The schema describes the metadata format for all federal open data. It is an object type. The main properties of the schema are: 'display_name' is a string that represents the display name of the entry. 'description' is a string that provides a description of the entry. 'id' is a string that represents a UUID for the entry. 'image_display_url' is a string that contains the URL of the image. 'title' is a string that specifies the title of the entry. 'name' is a string that represents the name of the entry. No additional properties are allowed at the root level.
