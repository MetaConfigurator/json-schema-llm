The schema describes a SignUpForm from Tiamat. It is an object type. The main properties of the schema are: 'confirmPassword' is a constant, being equal to the value of 'password'. 'email' is a string with a minimum length of 4 characters and a maximum length of 30 characters. It must be a valid email format and the field 'emailExists' is set to false. 'password' is a string with a maximum length of 50 characters and a minimum length of 4 characters. All three properties are required. No additional properties are allowed at the root level.
