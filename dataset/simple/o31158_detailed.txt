The schema describes a VPN global clientless access policy binding. It is an object type. The main properties of the schema are: 'builtin' is an object with an enum of allowed values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'; 'feature' is a string with an enum of allowed values including 'WL', 'WebLogging', and many others; 'globalbindtype' is a string with an enum of allowed values: 'SYSTEM_GLOBAL', 'VPN_GLOBAL', 'RNAT_GLOBAL'; 'gotopriorityexpression' is a string; 'groupextraction' is a boolean; 'policyname' is a string; 'priority' is an integer; 'secondary' is a boolean; 'type' is a string with an enum of allowed values: 'REQ_OVERRIDE', 'REQ_DEFAULT', 'RES_OVERRIDE', 'RES_DEFAULT'. No additional properties are allowed at the root level.
