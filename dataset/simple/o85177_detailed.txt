The schema describes a MWF badge component's schema definition. It is an object type. The main properties of the schema are: 'config' is an object that defines the configuration settings for the badge. It is required and contains the following properties: 'size' is a string that must be one of the following values: 'small', 'large'. 'type' is a string that must be one of the following values: 'highlight', 'lowlight'. No additional properties are allowed at the root level.
