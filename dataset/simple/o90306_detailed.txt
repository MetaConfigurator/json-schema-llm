The schema describes a sizeStop object that outlines the size of the symbol at various values of the expression. It is an object type. The main properties of the schema are: 'label' is a string that describes the value if a label is needed on the legend for a stop. It is optional. 'size' is a number that specifies the marker size to use for the specified value. It is required. 'value' is a number that represents the value to be mapped to a size. It is required. No additional properties are allowed at the root level.
