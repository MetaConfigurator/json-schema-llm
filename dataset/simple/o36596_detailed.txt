The schema describes a configuration for an export operation. It is an object type. The main properties of the schema are: 'startdate' is a string that represents the start date for the export, using the pattern yyyy-MM-dd and is required. 'enddate' is a string that represents the end date for the export, using the pattern yyyy-MM-dd and is required. 'format' is a string that specifies the export format, which must be one of the following values: 'PDF', 'ICAL' and is required. 'view' is a string that indicates the view rendering, which must be one of the following values: 'DAY', 'WEEK', 'LIST', 'NA' and is required. 'resourceIds' is an array of integers that represents an optional list of resource ids if the user wants to restrict the export. The array may have a minimum of 0 items and does not allow additional properties. No additional properties are allowed at the root level.
