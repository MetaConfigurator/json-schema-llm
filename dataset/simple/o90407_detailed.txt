The schema describes load balancer configuration for Kubernetes API servers. It is an object type. The main properties of the schema are: 'name' is a string, Name of the load balancer configuration. It is optional and has a max length of 32. 'type' is a string, Type of load balancer. It is required and must be one of the following values: 'cloud', 'flipbit'. No additional properties are allowed at the root level.
