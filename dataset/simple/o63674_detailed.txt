The schema describes a story record collection. It is an object type. It defines several reusable components under 'definitions': 'record' is an object, which contains properties for 'id', 'record_id', and 'databox_id'. The 'id' property is a string describing the unique identifier for a record and is optional. The 'record_id' property is an integer describing the identifier for a story contextualized by the databox and is required. The 'databox_id' property is an integer describing the databox identifier where the story is localized and is required. The main properties of the schema are: 'story_records' is an array describing a collection of story_record items, which is required and contains elements that must conform to the 'record' definition. No additional properties are allowed at the root level.
