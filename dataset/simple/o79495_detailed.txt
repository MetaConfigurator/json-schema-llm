The schema describes the properties of a tenant configuration. It is an object type. The main properties of the schema are: 'incarnation' is an integer representing a number that indicates the 'configuration version' of this tenant. It is required. 'securityGeneration' is an integer representing a number that indicates the 'security version' of this tenant. It is required. 'state' is a string that can take one of the following values: 'PREHALTING', 'PREHALTED', 'HALTING', 'HALTED', 'PRESTARTING', 'PRESTARTED', 'STARTING', 'STARTED', 'BROKEN'. It is optional. No additional properties are allowed at the root level.
