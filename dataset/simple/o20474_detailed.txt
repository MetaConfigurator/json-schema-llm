The schema describes hybrid flow response. It is an object type. The main properties of the schema are: 'access_token' is a string The OAuth 2.0 access token. Required for the UserInfo endpoint and other authorised protected resources. It is required. 'code' is a string The OAuth 2.0 authorisation code. Required to obtain the ID token and token from the token endpoint. It is required. 'expires_in' is a number Expiration time of the access token in seconds since the response was generated. It is optional. 'id_token' is a string The OpenID Connect ID token. It is optional. 'state' is a string The state value, if one was passed with the request. Clients must validate the value before proceeding. It is optional. 'token_type' is a string The OAuth 2.0 token type, set to bearer. It must be one of the following values: 'bearer'. It is optional. No additional properties are allowed at the root level.
