The schema describes a configuration object. It is an object type. The main properties of the schema are: 'includeFiles' is a boolean indicating whether to include files, optional, with a default value of false. 'includeDataTableRows' is a boolean indicating whether to include data table rows, optional, with a default value of false. 'includeDevices' is a boolean indicating whether to include devices, optional, with a default value of false. 'email' is a string formatted as an email with a maximum length of 1024 characters, optional. No additional properties are allowed at the root level.
