The schema describes a structured object with specific properties. It is an object type. The main properties of the schema are: 'lookup' is an object that contains properties related to the action with an optional sub-property 'action' of type string formatted as 'topic'. 'result' is an object that contains properties related to the result with an optional sub-property 'resource' of type string formatted as 'topic'. No additional properties are allowed at the root level.
