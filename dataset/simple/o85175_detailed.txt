The schema describes a MWF Action trigger component's schema definition. It is an object type. The main properties of the schema are: 'config' is an object that contains required fields 'context', 'glyph', 'element-type', and 'disabled'. 'context' is a string with allowed values 'text-with-glyph', 'text-only', and 'glyph-only'. 'element-type' is a string with allowed values 'anchor' and 'button'. 'disabled' is a boolean. 'glyph' is a string. 'content' is an object that contains required fields 'text' and 'href'. 'text' is a string and 'href' is a string. No additional properties are allowed at the root level.
