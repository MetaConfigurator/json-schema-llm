The schema describes a JSONAPI document. It is an object type. It defines several reusable components under 'definitions': 'resource' is an object, representing a resource. It has required field 'type' and optional field 'id', both of type string. The main properties of the schema are: 'data' is an anyOf type, representing the top-level resource or array of resources. It is a required property and can either be a single resource referenced by '#/definitions/resource' or an array of resources, each also referenced by '#/definitions/resource'. No additional properties are allowed at the root level.
