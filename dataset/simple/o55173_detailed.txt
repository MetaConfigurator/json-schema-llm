The schema describes base images definitions. It is an object type. It defines several reusable components under 'definitions': 'image' is an object, which includes properties 'purpose' and 'ref'. The 'purpose' property is a string with a default value of 'image' and can take the allowed values 'image', 'profile_image', or 'icon'. The 'ref' property can be a string with a maximum length of 2000 or null. The main properties of the schema are: 'image' is a reference to the definition 'image' and is required. No additional properties are allowed at the root level.
