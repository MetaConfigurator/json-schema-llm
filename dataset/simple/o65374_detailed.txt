The schema describes details about a single athlete. It is an object type. The main properties of the schema are: 'name' is a string representing the athlete's name, required, with a minimum length of 1. 'club' is a string representing the athlete's club for a given event/competition, optional, with a minimum length of 1. 'first_claim' is a string representing the athlete's first claim club, optional, with a minimum length of 1. 'vest_number' is a number or string representing the athlete's vest number for a given event/competition, optional, with a minimum value of 0 and a minimum length of 1. No additional properties are allowed at the root level.
