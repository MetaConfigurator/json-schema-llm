The schema describes a test schema for jsonstruct. It is an object type. The main properties of the schema are: 'name' is a string with a default value of 'bob'. It is required. 'type' is a string with a default value of 'person'. It is required. 'props' is an array of integers with a default item value of 3. It is required, and items must be integers. No additional properties are allowed at the root level.
