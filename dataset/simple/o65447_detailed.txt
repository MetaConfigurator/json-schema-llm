The schema describes a company which is the subject of a statement. It is an object type. The main properties of the schema are: 'name' is a string and is required. 'jurisdiction' is a string and is required. 'company_number' is a string and is optional. 'identifier' is a string, is optional, and has a description of 'An official identifier paired with a code for the issuer of the identifier, e.g sec/12345'. It must match the pattern '^[^/]+/[^/]+$'. No additional properties are allowed at the root level.
