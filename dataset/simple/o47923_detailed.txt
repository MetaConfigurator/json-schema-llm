The schema describes gate configuration. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'name' is a string which is required. 'address' is a string which is required. 'desc' is a string which is optional. 'inFlightThreshold' is an integer which is required, with a minimum value of 1. 'noSinkDropMessages' is a boolean which is required. No additional properties are allowed at the root level.
