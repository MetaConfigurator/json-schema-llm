The schema describes a plugin engine configuration. It is an object type. The main properties of the schema are: 'manifest' is a string that describes the manifest to read to know the configuration of the plugin. It is optional and has an example value of 'manifest.json'. 'prefix' is a string that describes the prefix of the url. It is optional and has an example value of './other-plugin-folder/'. 'uid' is a string that describes the UID of the engine. It is required and has an example value of 'com.awesome.plugin'. 'url' is a string that describes the url/root of the Plugin relative to the prefix. It is optional and has an example value of 'AwesomePlugin'. No additional properties are allowed at the root level.
