The schema describes a configuration for a sample deployment stage. It is an object type. The main properties of the schema are: 'api-key' is a string that contains IBM Cloud API keys, which grant access to organizations, spaces, and Kubernetes clusters. 'app-name' is a string for the app name. 'dev-organization' is a string that refers to the IBM Cloud organization. 'dev-region' is a string that refers to the IBM Cloud region. 'dev-space' is a string that refers to the IBM Cloud space. The required properties are 'dev-region', 'dev-organization', 'dev-space', and 'app-name'. No additional properties are allowed at the root level.
