The schema describes parameter selection. It is an array type. The main properties of the schema are: Each item in the array is an object with properties 'key' and 'value'. 'key' is a string and is optional. 'value' can be either a string or a number, or it can be an object that contains several properties. If 'value' is an object, it must have a required property 'expression_type', which is a string that must have the value 'stage-output-selector'. The object may also include optional properties: 'flatten' (a boolean), 'output' (a string), 'stages' (a string), and 'unwrap' (a boolean). No additional properties are allowed at the root level.
