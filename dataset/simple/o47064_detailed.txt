The schema describes the format for describing the statistics (distribution, etc.) of a data source. It is an object type. The main properties of the schema are: 'avg' is an integer representing the average frequency (in ms) for data updates, optional with a minimum value of 0. 'max' is an integer representing the maximum expected frequency (in ms) for data updates, optional with a minimum value of 0. 'min' is an integer representing the minimum expected frequency (in ms) for data updates, optional with a minimum value of 0. No additional properties are allowed at the root level.
