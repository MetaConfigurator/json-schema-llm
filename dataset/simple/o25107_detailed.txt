The schema describes domain validation configurations. It is an object type. The main properties of the schema are: 'DomainName' is a string required field. 'SubjectAlternativeNames' is an array of strings with a minimum of 1 item and unique items. 'IdempotencyToken' is a string optional field. 'DomainValidationOptions' is an array of objects with a minimum of 1 item and unique items. No additional properties are allowed at the root level.
