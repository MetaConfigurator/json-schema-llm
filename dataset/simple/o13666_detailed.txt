The schema describes research field categories. It is an array type. The main properties of the schema are: Each item in the array is an object containing properties: '_scheme' is a string that must be one of 'INSPIRE', 'arXiv', 'APS', 'PoS', or 'Elsevier'. '_term' is a string. 'scheme' is a string. 'source' is a string that must be one of 'INSPIRE', 'submitter', 'curator', or 'publisher'. 'term' is a string that must be one of 'Accelerators', 'Astrophysics', 'Computing', 'Experiment-HEP', 'Experiment-Nucl', 'General Physics', 'Gravitation and Cosmology', 'Instrumentation', 'Lattice', 'Math and Math Physics', 'Other', 'Phenomenology-HEP', 'Theory-HEP', or 'Theory-Nucl'. Each item in the array must be unique. No additional properties are allowed at the root level.
