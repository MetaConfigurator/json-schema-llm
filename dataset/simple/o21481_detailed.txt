The schema describes a Google Analytics Collector Query. It is an object type. The main properties of the schema are: 'dimensions' is an array that must contain string items. 'filters' is either an array of strings or a string. 'frequency' is a string. 'id' is a string that must match the pattern '^[a-z0-9:]+$'. 'maxResults' is an integer. 'metrics' is a required array that must contain string items. 'segment' is a string. 'sort' is an array that must contain string items. No additional properties are allowed at the root level.
