The schema describes a data structure for an algorithm configuration. It is an object type. The main properties of the schema are: 'algorithm' is an integer that is required, with a minimum value of 1 and a maximum value of 2. 'fingerprint' is a string that is required and must match the pattern '^[0-9A-Fa-f]{40}$'. 'type' is an integer that is required, with a minimum value of 1 and a maximum value of 1. No additional properties are allowed at the root level.
