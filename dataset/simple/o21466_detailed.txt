The schema describes a performance platform schema for GOV.UK customer-satisfaction single question plus comment records. It is an object type. The main properties of the schema are: '_id' is a string unique url-friendly base64 encoded, utf8 encoded concatenation identifier and is required. '_timestamp' is a string ISO8601 combined date and time in UTC e.g. '2014-12-01T00:00:00+00:00' and is required. 'comments' is an integer the number of respondents leaving a comment - optional dependent on whether information available. 'period' is a string the period covered by each data point from the _timestamp and is required, with allowed values: 'hour', 'day', 'week', 'month', 'quarter'. 'satisfaction_score' is a number the calculated user satisfaction score for the service and is required with a minimum value of 0. 'total_responses' is an integer the total number of respondents providing feedback and is required with a minimum value of 0. No additional properties are allowed at the root level.
