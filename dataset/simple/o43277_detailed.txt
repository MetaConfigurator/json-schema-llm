The schema describes a player’s experience and achievements in a game. It is an object type. It defines several reusable components under 'definitions': 'CompletedSpartanRank' is an object, representing a completed spartan rank. It requires 'Id' of type string and 'PacksAwarded' of type integer. The main properties of the schema are: 'PreviousTotalXP' is an integer, required; 'GameplayXP' is an integer, required; 'ChallengesXP' is an integer, required; 'UpdatedTotalXP' is an integer, required; 'CompletedSpartanRanks' is an array, required, containing items that reference the 'CompletedSpartanRank' definition. No additional properties are allowed at the root level.
