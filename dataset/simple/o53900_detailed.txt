The schema describes a configuration push request. It is an object type. The main properties of the schema are: 'config' is an arbitrary type configuration body defined by the request 'config_format', required. 'configId' is a string identifier of the current configuration, required. 'id' is a number ID of the message used to match client response to the request, required, and must be a multiple of 1.0. No additional properties are allowed at the root level.
