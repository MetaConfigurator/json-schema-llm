The schema describes user data. It is an object type. The main properties of the schema are: 'active' is a boolean that has a default value of 0. 'id' is a string that is optional. 'mail' is a string that is required. 'name' is a string that is required. 'role' is a string that has a default value of 'Anonymous' and allows the following values: 'Admin', 'Describer', 'Anonymous'. 'token' is a string that is required and is marked as hidden. No additional properties are allowed at the root level.
