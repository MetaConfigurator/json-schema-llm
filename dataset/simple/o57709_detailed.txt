The schema describes a collection of tree nodes. It is an array type. The main properties of the schema are: 'title' is a string required field that represents the node's title. 'key' is a string required field used as a unique identifier for the node. 'lazy' is a boolean required field indicating whether the node is loaded lazily. 'folder' is a boolean required field indicating whether the node is a folder. 'isArray' is a boolean optional field. 'icon' is a string optional field. 'tooltip' is a string optional field. No additional properties are allowed at the root level.
