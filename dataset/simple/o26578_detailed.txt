The schema describes the configuration for a StartStop application. It is an object type. The main properties of the schema are: 'status' is a string that indicates the status of the StartStop application. It is required and has a default value of 'initialized'. The allowed values are 'initialized', 'running', 'running_canceled', 'stopping', 'stopping_canceled', 'stopped', 'shutdown', and 'configerror'. 'betriebsmeldungen' is a boolean that indicates whether operation messages are enabled. It is required and has a default value of true. 'davconnection' is a boolean that indicates whether DAV connection is enabled. It is required and has a default value of false. No additional properties are allowed at the root level.
