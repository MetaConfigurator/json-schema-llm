The schema describes options for Material installation. It is an object type. The main properties of the schema are: 'skipPackageJson' is a boolean that specifies whether to add materials dependencies to package.json, with a default value of false. 'theme' is a string that indicates the theme to apply, allowed values are 'indigo-pink', 'deeppurple-amber', 'pink-bluegrey', 'purple-green', and 'custom', with a default value of 'indigo-pink'. No additional properties are allowed at the root level.
