The schema describes the Git LFS HTTPS Lock List API Response. It is an object type. The main properties of the schema are: 'locks' is an array that contains objects with properties 'id', 'path', 'locked_at', and 'owner'. Each object in the array is required to have 'id' as a string, 'path' as a string, 'locked_at' as a string, and 'owner' as an object that contains 'name' as a string. 'next_cursor' is an optional string. No additional properties are allowed at the root level.
