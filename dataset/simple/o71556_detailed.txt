The schema describes a Product. It is an object type. The main properties of the schema are: 'id' is a number representing the Product identifier and is required. 'name' is a string representing the Name of the product and is required. 'price' is a number that must be greater than or equal to 0 and is required. 'stock' is an object containing 'retail' and 'warehouse', both of which are numbers and optional. 'tags' is an array of strings representing product tags. No additional properties are allowed at the root level.
