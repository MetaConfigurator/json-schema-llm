The schema describes a resource configuration. It is an object type. The main properties of the schema are: 'name' is a string and is required. 'description' is a string and is optional. 'icon' is a string and is optional. 'periodic_booking' is a boolean and is required. 'is_available' is a boolean and is required. 'min_delay' is an integer with a minimum value of 0 and is optional, providing the booking advance time in seconds. 'max_delay' is an integer with a minimum value of 0 and is optional, specifying the maximum booking advance time in seconds. 'color' is a string and is optional. 'validation' is a boolean and is optional. No additional properties are allowed at the root level.
