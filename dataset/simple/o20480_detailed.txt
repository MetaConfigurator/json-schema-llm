The schema describes a token authorization code grant request. It is an object type. The main properties of the schema are: 'code' is a string that is required. It represents the code received with the authentication response. 'code_verifier' is a string that is optional. It describes the code verifier if Proof Key of Code Exchange (PKCE) is employed (for public OAuth clients). 'grant_type' is a string that is required. It must be set to 'authorization_code'. 'redirect_uri' is a string that is optional. It denotes the value of the redirect_uri parameter included in the original authentication request. No additional properties are allowed at the root level.
