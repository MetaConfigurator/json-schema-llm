The schema describes an event source. It is an object type. The main properties of the schema are: 'id' is an integer unique identifier given by the server, required. 'generatedIdentifier' is an optional string unique id generated by the server. 'name' is a string A name to identify the event source, required. 'organizationId' is an integer The organization, required. 'eventSourceTemplateId' is an integer The event source template instantiated, required. 'public' is a boolean Make the event source available to everybody, optional. 'deletable' is a boolean Define if the event source can be deleted, required. 'configuration' is an object A valid JSON object, required. No additional properties are allowed at the root level.
