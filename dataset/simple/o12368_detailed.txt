The schema describes an address object. It is an object type. The main properties of the schema are: 'country' is a string that uses the 'country-code' format; 'id' is a string that is hidden and has the title 'ID'; 'locality' is a string with the title 'City/Town'; 'postal_code' is a string with the title 'Postal code'; 'region' is a string with the title 'Region'; 'street_address' is a string with the title 'Street address'; 'text' is a string with the title 'Text'. No additional properties are allowed at the root level.
