The schema describes a user profile. It is an object type. The main properties of the schema are: 'id' is an integer and required, with a minimum value of 1. 'firstName' is a string and required, generating a fake name. 'lastName' is a string and required, generating a fake name. 'emailAddress' is a string and required, generating a fake email. 'userName' is a string and required, generating a fake username. 'createdAt' is a string and required, generating a date in the past. 'updatedAt' is a string and required, generating a date in the past. No additional properties are allowed at the root level.
