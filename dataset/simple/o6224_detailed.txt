The schema describes a configuration for JSON data transformation. It is an object type. The main properties of the schema are: 'source' is a string that specifies the JSON path for the source that contains the input, and it is required with a minimum length of 1. 'destination' is a string that specifies the JSON path for the destination to place the output, and it is required with a minimum length of 1. No additional properties are allowed at the root level.
