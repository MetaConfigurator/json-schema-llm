The schema describes import person configuration. It is an object type. The main properties of the schema are: 'mappings' is an array that contains objects with properties 'dataType', 'displayName', 'ldapName', and 'required'. The 'ldapName' and 'dataType' properties are required, while 'displayName' and 'required' are optional. The 'dataType' property must be one of the following values: 'string', 'numeric'. Each item in the 'mappings' array can have additional properties, but cannot have any additional properties at the object level. No additional properties are allowed at the root level.
