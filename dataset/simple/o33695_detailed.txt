The schema describes a representation of the master schema. It is an object type. The main properties of the schema are: '@context' is a string representing the JSON-LD context, which is required and has a format of 'url'. '@type' is a string representing the JSON-LD type, which is required. 'schemaType' is a string representing the schema type, which is required and has a searchable attribute set to true with a search mode of 'noop'. No additional properties are allowed at the root level.
