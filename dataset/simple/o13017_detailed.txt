The schema describes a test of a test. It is an object type. It defines several reusable components under 'definitions': 'email' is a string, test email with a format constraint of email. 'name' is a string, the name of the test. The main properties of the schema are: 'test' is an object that resolves to the referenced definition '#/definitions/test'. The root schema defines a link to create a new test. No additional properties are allowed at the root level.
