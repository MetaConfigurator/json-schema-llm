The schema describes a configuration object. It is an object type. The main properties of the schema are: 'search' is an object that contains a required property 'resource' of type string with a format of 'topic' and an axRole of 'inlet'. 'i18n' is an object that contains an optional property 'locale' of type string with a format of 'topic', an axRole of 'inlet', and a default value of 'default'. No additional properties are allowed at the root level.
