The schema describes a configuration for offering change policies. It is an object type. The main properties of the schema are: 'offeringChangePolicy' is a string that specifies the change policy and is required to be one of the following values: 'contains-all', 'exactly-match', 'no-restrictions'. 'offeringChangePolicyIgnoreTags' is an array of strings that can contain unique items and is optional. No additional properties are allowed at the root level.
