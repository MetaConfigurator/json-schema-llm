The schema describes a data structure for an interaction. It is an object type. The main properties of the schema are: 'author' is a string, optional; 'content' is an object, optional; 'contentUrl' is a string, optional; 'direction' is an enum, optional, with allowed values: 'in', 'out', 'inout'; 'player' is a string, optional; 'type' is an enum, required, with allowed values: 'chat', 'audio', 'video', 'av', 'screen', 'file', 'midi', 'activity_context', 'availability_context', 'location_context', 'heart_rate_context', 'user_steps_context', 'battery_context', 'sleep_context', 'light_context', 'humidity_context', 'power', 'user_activity_context', 'user_communication_context'. No additional properties are allowed at the root level.
