The schema describes the structure of a sandbox configuration. It is an object type. The main properties of the schema are: 'sandbox_id' is a string A unique identifier for the sandbox. It is optional. 'uri' is a string The URI of the sandbox. It is optional and must follow the URI format. 'checksums' is an object A hash where each key is one of the checksums that you are willing to upload. It is optional and has pattern properties matching a 32-character hexadecimal string, where each corresponding value must be an object containing 'url' (a string in URI format, specified if 'needs_upload' is true) and 'needs_upload' (a boolean, required). No additional properties are allowed at the root level.
