The schema describes a WMSLayer layer. It is an object type. The main properties of the schema are: 'legendUrl' is a string a string URL to a legend graphic for the layer. 'name' is a string a string containing a unique name for the layer. 'queryable' is a boolean specifying whether a layer is queryable or not. 'showPopup' is a boolean specifying whether to display popup or not. 'title' is a string a user-friendly string title for the layer that can be used in a table of contents. No additional properties are allowed at the root level.
