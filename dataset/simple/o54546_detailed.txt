The schema describes the data contained in a LineChart. It is an object type. The main properties of the schema are: 'labels' is an array of strings representing the labels for the chart, which is required. 'datasets' is an array of objects representing the datasets for the chart, which is required. Each dataset object contains: 'name', which is an optional string, 'color', which is an optional string that must match the pattern '^#[0-9a-fA-F]{6}$', and 'values', which is an array of numbers and is required. No additional properties are allowed at the root level.
