The schema describes a contribution. It is an object type. The main properties of the schema are: 'id' is a string representing the uuid of the contribution, required; 'title' is a string or null representing the contribution's title, optional; 'text' is a string representing the contribution's text, required; 'meta' is an object representing additional metadata, optional, containing properties: 'createdAt' which is a string representing the contribution's creation date, optional; and 'creator' which is an object or null representing the contribution's author, optional. No additional properties are allowed at the root level.
