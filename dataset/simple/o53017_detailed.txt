The schema describes a set of sprites for a pokemon. It is an object type. The main properties of the schema are: 'back_default' is a string representing a URL for the default back sprite, 'back_female' is a string representing a URL for the female back sprite, 'back_shiny' is a string representing a URL for the shiny back sprite, 'back_shiny_female' is a string representing a URL for the shiny female back sprite, 'front_default' is a string representing a URL for the default front sprite, 'front_female' is a string representing a URL for the female front sprite, 'front_shiny' is a string representing a URL for the shiny front sprite, and 'front_shiny_female' is a string representing a URL for the shiny female front sprite. No additional properties are allowed at the root level.
