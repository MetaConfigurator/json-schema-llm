The schema describes a data object for facts. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'sample_date' is a string that represents the date on which this fact was known to be true. It is a required property. 'source_url' is a string that specifies the place where this fact can be verified. It is a required property. No additional properties are allowed at the root level.
