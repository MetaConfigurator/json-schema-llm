The schema describes a temperature sensor's data. It is an object type. The main properties of the schema are: 'sensorId' is a string representing the unique identifier of the sensor. It is optional. 'temperature' is an object containing temperature readings. The 'temperature' object has properties: 'old' is a number representing the previous temperature reading. It is optional, and 'new' is a number representing the current temperature reading. It is optional as well. No additional properties are allowed at the root level.
