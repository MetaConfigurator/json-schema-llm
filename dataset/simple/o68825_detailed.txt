The schema describes an instance. It is an object type. The main properties of the schema are: 'data' is an object containing properties: 'compatibleInstances' is an array of strings, representing compatible plug-ins, and 'family' is a string describing the family of the instance. 'name' is a string, representing the name of the instance. Both 'name' and 'data' are required fields. No additional properties are allowed at the root level.
