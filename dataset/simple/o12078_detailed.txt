The schema describes the structure for a specific API request. It is an object type. The main properties of the schema are: 'body' is a string that is optional. 'headers' is an object that is required and allows additional properties. It contains the property 'X-Twilio-Signature', which is a string with a pattern that must match '^(?:[A-Za-z0-9+/]{4})*(?:[A-Za-z0-9+/]{2}==|[A-Za-z0-9+/]{3}=)$'. 'httpMethod' is a string that is required with a pattern that must match '^POST$'. 'path' is a string that is required with a pattern that must match '^/sms$'. No additional properties are allowed at the root level.
