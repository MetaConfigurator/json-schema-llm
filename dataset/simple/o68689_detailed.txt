The schema describes a configuration object. It is an object type. The main properties of the schema are: 'extends' is a string that specifies the path to extend a specific configuration file, which can be either relative or absolute. 'excludeFiles' is an array that disables style checking for specified paths declared with glob patterns, with a default value of ['node_modules/**']. 'additionalRules' is an array that includes file path matching patterns to load additional rules from. No additional properties are allowed at the root level.
