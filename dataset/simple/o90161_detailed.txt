The schema describes the file authoring information for a filter, including the filter type and its value settings. It is an object type. The main properties of the schema are: 'filterType' is a string that represents the filter type name. Name is a unique identifier and is required. 'filterValues' is an array that contains filter values, which are the attributes that can be stored for individual fields in a layer. It requires strings as items and is also required. No additional properties are allowed at the root level.
