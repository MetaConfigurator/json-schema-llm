The schema describes a statement. It is an object type. The main properties of the schema are: 'value' is a string A value from the enumeration for this statement, required. 'extended_value' is an object The extended value contains additional information, optional. It requires the 'value' property, which is a string The value in the extended vocabulary. 'description' is a string A textual description of this statement, optional. 'credibility' is an integer The credibility of this statement, using the Admirality scale, required, with a minimum of 0 and a maximum of 6. No additional properties are allowed at the root level.
