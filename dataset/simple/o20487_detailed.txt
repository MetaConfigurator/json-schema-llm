The schema describes a token refresh token grant request. It is an object type. The main properties of the schema are: 'grant_type' is a string that must be set to 'refresh_token' and is required. 'refresh_token' is a string that represents the refresh token issued to the client and is optional. 'scope' is a string that contains optional requested scope values for the new access token, treating omitted values as equal to the originally granted scope. No additional properties are allowed at the root level.
