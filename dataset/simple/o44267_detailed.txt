The schema describes the hotspot events object. It is an object type. The main properties of the schema are: 'onClick' is an anyOf type, which can either be a string or an array of strings that represents the action uids to trigger on click. 'onOut' is an anyOf type, which can either be a string or an array of strings that represents the action uids to trigger on out. 'onOver' is an anyOf type, which can either be a string or an array of strings that represents the action uids to trigger on over. No additional properties are allowed at the root level.
