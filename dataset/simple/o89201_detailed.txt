The schema describes diagnostic results. It is an array type. The main properties of the schema are: 'id' is a string A unique identifier for the diagnostic. 'sevirity' is an enum Describes how severe a diagnostic is. It can be one of the following values: 'Hidden', 'Info', 'Warning', 'Error'. 'line' is an integer The line number of location where the diagnostic appears. 'column' is an integer The column number of location where the diagnostic appears. 'path' is a string The path of the source document file. 'message-args' is an array containing strings Arguments to a format message string. No additional properties are allowed at the root level.
