The schema describes login information for OBS Websocket, used to communicate with and remotely control OBS Studio. It is an object type. It defines several reusable components under 'definitions': 'Status' is a string, an enumeration of allowed values: 'connected', 'connecting', 'disconnected', 'error', with a default value of 'disconnected'. The main properties of the schema are: 'ip' is a string with a default value of 'localhost', 'port' is a number with a default value of 4444, 'password' is a string with a default value of '', 'status' is a string that references the 'Status' definition. All four properties are required. No additional properties are allowed at the root level.
