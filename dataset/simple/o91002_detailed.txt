The schema describes a configuration for generating random text. It is an object type. The main properties of the schema are: 'startsWithLipsum' is a boolean required field indicating whether the text should start with 'lipsum'. 'minWords' is an integer required field with a minimum value of 1, specifying the minimum number of words in the generated text. 'maxWords' is an integer required field with a minimum value of 1, specifying the maximum number of words in the generated text. No additional properties are allowed at the root level.
