The schema describes a structure for MCDA range distribution in the performance table entry of absolute data. It is an object type. The main properties of the schema are: 'type' is a string required property with an enum value of 'range'. 'parameters' is an object required property that defines the bounds of the range, with required fields 'lowerBound' and 'upperBound', both of which are numbers. No additional properties are allowed at the root level.
