The schema describes a cursor response for an API. It is an object type. The main properties of the schema are: 'paging' is an object that contains pagination information, required. It has the required property 'uri' which is a string. The other properties include 'lastPage' which is an integer with a minimum value of 1, 'limit' which is an integer with a minimum value of 1, 'next' which is a string, 'page' which is an integer with a minimum value of 1, 'previous' which is a string, and 'total' which is an integer with a minimum value of 0. No additional properties are allowed at the root level.
