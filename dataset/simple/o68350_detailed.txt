The schema describes parameters accepted by the Yieldlab adapter. It is an object type. The main properties of the schema are: 'adslotId' is a string Yieldlab ID of the ad slot, required. 'supplyId' is a string Yieldlab ID of the supply, required. 'adSize' is a string size of the adslot in pixel, e.g. 200x50, required. 'extId' is a string external ID used for reporting, optional. 'targeting' is an object targeting information in key value pairs, optional. No additional properties are allowed at the root level.
