The schema describes a response structure. It is an object type. The main properties of the schema are: 'kind' is a string that defines that the message is a response and has a default value of 'resp'. 'reqId' is a string representing the ID of the request that originated this response, with a minimum length of 1 and a maximum length of 64, and has a default value of '0'. 'type' is a string that contains one of the APIs response type names. 'code' is an integer describing the response code according to the HTTP status codes standard. 'body' is an empty object. All properties are required. No additional properties are allowed at the root level.
