The schema describes an account structure. It is an object type. The main properties of the schema are: 'account' is an object that requires 'homePage' and 'name'. 'homePage' is a string formatted as an IRI. 'name' is a string. The properties 'mbox', 'mbox_sha1sum', and 'openid' are all null types and are optional. No additional properties are allowed at the root level.
