The schema describes the structure of a privacy statement collection. It is an object type. The main properties of the schema are: 'privacyStatements' is an array that contains objects, which are required to have 'id' and 'issuedDate' properties. Each 'id' is of string type, and each 'issuedDate' is a string formatted as a date. The array can have a minimum of 0 items. No additional properties are allowed at the root level.
