The schema describes a system session. It is an object type. The main properties of the schema are: 'all' is a boolean that is optional. 'currentconn' is a boolean that is readonly. 'expirytime' is an integer that is readonly. 'lastactivitytime' is a string that is readonly. 'logintime' is a string that is readonly. 'numofconnections' is an integer that is readonly. 'sid' is an integer that is optional. 'username' is a string that is readonly. No additional properties are allowed at the root level.
