The schema describes the format of Xsd2JsonSchema functional tests. It is an object type. It defines several reusable components under 'definitions': 'TestInstanceType' is an object, which includes the properties 'expectedToPass' which is a boolean and 'testData' which describes a JSON test instance. The main properties of the schema are: 'xsdPath' is a string which is optional. 'xsdFilename' is a string which is optional. 'testInstances' is an array which is optional, where each item is referenced from the 'TestInstanceType' definition. No additional properties are allowed at the root level.
