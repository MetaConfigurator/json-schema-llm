The schema describes configuration settings for ImboLauncher to start one or more Imbo servers. It is an object type. The main properties of the schema are: 'servers' is an array List of server definitions. It is required and must contain at least one item. Each item in the array is an object that contains the following required properties: 'version' is a string which specifies which Imbo version to use for this definition, 'host' is a string which specifies which host to use for this server, 'port' is a number which specifies which port to use for this server, and 'config' is a string which specifies which configuration file to use for this server. No additional properties are allowed at the root level.
