The schema describes a system restore point. It is an object type. The main properties of the schema are: 'backupfilename' is a string, readonly. 'createdby' is a string, readonly. 'creationtime' is a string, readonly. 'filename' is a string, optional. 'ipaddress' is a string, readonly. 'techsuprtname' is a string, readonly. 'version' is a string, readonly. No additional properties are allowed at the root level.
