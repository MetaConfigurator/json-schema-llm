The schema describes a vehicle or array of vehicles. It is an object type. It defines several reusable components under 'definitions': 'nc:VehicleType' is an object, a vehicle type. The main properties of the schema are: 'nc:Vehicle' is an array or a single vehicle type. It is optional and can be either a single instance of 'nc:VehicleType' or an array of 'nc:VehicleType'. No additional properties are allowed at the root level.
