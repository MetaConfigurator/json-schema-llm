The schema describes user contact information. It is an object type. The main properties of the schema are: 'emailAddress' is a string that must match the pattern '^[^@^\s]+@[^@^\.^\s]+(\.[^@^\.^\s]*)*.gov.uk' and is required. 'mobileNumber' is a string that must match the pattern '^\+44[\d]{10}$' and is required. 'password' is a string and is required. No additional properties are allowed at the root level.
