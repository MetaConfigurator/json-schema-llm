The schema describes a system utilization logical interface. It is an object type. The main properties of the schema are: 'cpu' is a number representing CPU Utilization %, with a minimum of 0, a maximum of 100, and a default value of 0.0. 'memory' is a number representing Memory utilization (%), with a minimum of 0.0, a maximum of 100, and a default value of 0.0. No additional properties are allowed at the root level.
