The schema describes the configuration for rate limiting. It is an object type. The main properties of the schema are: 'timeSincePath' is a string that represents a JSON path to store the time since the last not throttled payload. It is optional and must have a minimum length of 1. 'ratePerMinute' is an integer that defines the number of payloads to allow per minute. It is required and must have a minimum value of 1. No additional properties are allowed at the root level.
