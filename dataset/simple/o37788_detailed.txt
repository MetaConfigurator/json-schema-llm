The schema describes text resources for Tabris.js. It is an object type. It defines several reusable components under 'definitions': 'textResources' is an object, which contains properties '$schema' of type string, '$scaleFactor' with allowed values 'nearest', 'lower', 'higher', and '$fallbackLanguage' of type string that must match the pattern '^[a-zA-Z][a-zA-Z][a-zA-Z]?(-[\-a-zA-Z0-9]+)?$'. It allows any properties that match the pattern '^[a-z][a-zA-Z][a-zA-Z0-9]+$' that reference 'textResource'. 'textResource' is a oneOf type, which can either be 'textDefinition' or 'textSelector'. 'textDefinition' is a string type. 'textSelector' is a oneOf type with various configurations including required properties and each having no additional properties. The main properties of the schema are: The root schema resolves to the referenced definition 'textResources'. No additional properties are allowed at the root level.
