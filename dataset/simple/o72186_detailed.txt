The schema describes program analytics configuration. It is an object type. The main properties of the schema are: 'goals' is an object defining the program goals and how they relate to projected revenue. It is required and disallows additional properties. The 'goals' object has required fields 'event' and 'value', and also disallows additional properties. 'event' is a string that specifies the goal completion, with allowed values: 'rewardCreated' and 'rewardRedeemed'. 'value' is a number that records the following revenue in analytics. No additional properties are allowed at the root level.
