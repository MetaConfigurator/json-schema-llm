The schema describes validating map products. It is an object type. The main properties of the schema are: 'metadata_type' is a string with a default value of 'n/a'. 'name' is a string with a default value of 'n/a'. 'map_type' is a string with a default value of 'n/a'. 'production_site' is a string with a default value of 'n/a'. 'status' is a string with a default value of 'n/a'. 'availability' is a string with a default value of 'n/a'. 'product_link' is a string with a default value of 'n/a'. All properties are required. No additional properties are allowed at the root level.
