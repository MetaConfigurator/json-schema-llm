The schema describes a map configuration file. It is an array type. The main properties of the schema are: 'filename' is a string that represents the map file and is required. 'rounds' is an integer that indicates the minimum number of rounds that this map will be invoked in a row. It is optional with a default value of 0. 'weight' is an integer that signifies the weight of this map compared to others. It is also optional with a default value of 1. The schema requires at least one item in the array. No additional properties are allowed at the root level.
