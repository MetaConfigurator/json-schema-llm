The schema describes PayPal payout item json schema. It is an array type. The main properties of the schema are: 'recipient_type' is a string that must be one of the following values: 'EMAIL', 'PHONE', 'PAYPAL_ID'. 'amount' is an object that is required and must contain: 'currency', which is a string with a maximum length of 3, and 'value', which is a string. 'note' is an optional string. 'receiver' is a string that is required. 'sender_item_id' is an optional string. The array items must meet the conditions that 'amount' and 'receiver' are required. No additional properties are allowed at the root level.
