The schema describes a Docker Encapsulated Environment. It is an object type. The main properties of the schema are: 'env' is an object which is optional and has a default value of an empty object; 'environment_type' is a string which is required, has a default value of 'docker-encapsulated', and its only allowed value is 'docker-encapsulated'; 'envscript' is a string which is optional and has a default value of an empty string; 'image' is a string which is required; 'imagetag' is a string which is optional and has a default value of 'latest'; 'resources' is an array which is optional and has a default value of an empty array. No additional properties are allowed at the root level.
