The schema describes base labels definitions. It is an object type. It defines several reusable components under 'definitions': 'label' is an object, describing a label with properties. It has required fields: 'name' must be a string with a minimum length of 1 and maximum length of 191, matching the pattern '^([^,]|$)'. The 'slug' is an optional property that can be a string or null, with a maximum length of 191. Additional properties are not allowed in the 'label' object. The main properties of the schema are: 'label' is a reference to the 'label' definition and is required. No additional properties are allowed at the root level.
