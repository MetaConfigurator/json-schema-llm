The schema describes a tournament configuration. It is an object type. It defines several reusable components under 'definitions': 'text' is an anyOf type, allowing either a number or a string with a minimum length of 1. 'text_list' is an array type, requiring at least one unique entry of type 'text'. The main properties of the schema are: 'name' is a text type, required, and 'comments' is an optional text_list type. No additional properties are allowed at the root level.
