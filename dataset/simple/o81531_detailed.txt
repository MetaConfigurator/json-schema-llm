The schema describes a user profile. It is an object type. The main properties of the schema are: 'email' is a string that must match the pattern '^[\w|-|.]+@[\w]+\.[A-Za-z]{2,4}$' and is required. 'firstName' is a string and is required. 'lastName' is a string and is required. 'postedSlides' is a value that can either be a boolean or a string, where the string can be one of the following: 'yes', 'Yes', 'no', 'No' and is required. 'rating' is a number and is required. No additional properties are allowed at the root level.
