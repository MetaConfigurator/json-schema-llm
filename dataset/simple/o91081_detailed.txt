The schema describes time-location data (GPS traces). It is an array type. The main properties of the schema are: Each item is an object containing the following properties: 'timestamp' is a number representing the date time this data point was created and is required. 'longitude' is a number representing the longitude where this data point was created, with a minimum of -180 and a maximum of 180, and is required. 'latitude' is a number representing the latitude where this data point was created, with a minimum of -90 and a maximum of 90, and is required. 'altitude' is a number representing the altitude where this data point was created and is required. No additional properties are allowed at the root level.
