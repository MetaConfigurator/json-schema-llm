The schema describes a whitelist configuration. It is an object type. The main properties of the schema are: 'ignoreSevertiesAtOrBelow' is a string that is required and must be one of the following values: 'negligible', 'Negligible', 'low', 'Low', 'medium', 'Medium', 'high', 'High'. 'vulnerabilities' is an array of objects that is optional, where each object must contain the properties 'cveId' (a string with minLength 1 and maxLength 512) and 'rationale' (a string with minLength 1 and maxLength 512), both of which are required. Each object in this array cannot have additional properties, and the array must have unique items. No additional properties are allowed at the root level.
