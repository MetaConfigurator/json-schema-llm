The schema describes a User Reward Balance Changed Webhook. It is an object type. The main properties of the schema are: 'userId' is a string that represents the user's unique identifier. It is required. 'accountId' is a string that represents the user's unique account identifier. It is required. 'unit' is a string that corresponds to the reward balance. It is required. 'availableValue' is an integer that represents the reward balance available to be redeemed. It is required. 'resourceVersion' is an integer that reflects when the balance was generated; the greater the value, the more recent the balance. It is required. No additional properties are allowed at the root level.
