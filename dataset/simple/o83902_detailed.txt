The schema describes the configuration for a migration process. It is an object type. The main properties of the schema are: 'collection' is a string that describes the collection to load the migrations from and is required. 'from' is a string that describes the version installed previously and is required. 'package' is a string that describes the package to migrate and is required. 'to' is a string that describes the version to migrate to and is required. No additional properties are allowed at the root level.
