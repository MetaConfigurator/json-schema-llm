The schema describes a Thing data upload. It is an object type. The main properties of the schema are: 'thing' is an object that is required and must contain the following properties: 'id' is a string and is required; 'title' is a string and is required; 'description' is a string and is required; 'license' is a string and is required; 'license_url' is a string and is required. No additional properties are allowed at the root level.
