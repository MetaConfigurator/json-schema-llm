The schema describes a Google form driven interview. It is an object type. The main properties of the schema are: 'callbackId' is a string required property. 'createdOn' is a string required property that must follow the date-time format. 'doc_type' is a string required property that must match the pattern '^interview$'. 'humanId' is a string required property. 'reason' is a string required property with allowed values: 'burial', 'suspect'. No additional properties are allowed at the root level.
