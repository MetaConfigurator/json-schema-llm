The schema describes parameters accepted by the Avocet adapter. It is an object type. The main properties of the schema are: 'placement' is a string that represents an Avocet placement ID, and it is optional. 'placement_code' is a string that represents an Avocet placement external code, and it is also optional. The schema requires that either 'placement' or 'placement_code' must be provided. No additional properties are allowed at the root level.
