The schema describes a collection of wells. It is an object type. The main properties of the schema are: 'wells' is an array containing objects representing each well. Each well object has the following required properties: 'type' is a string representing the type of the well, 'index' is a string indicating the well's index, 'size' is an integer specifying the size of the well, and 'values' is an array of numbers representing associated values. No additional properties are allowed at the root level.
