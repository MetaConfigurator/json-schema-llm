The schema describes a structure for a display object. It is an object type. The main properties of the schema are: 'display' is an object containing properties. It is optional. The 'resource' property is a string that has a format of 'topic' and is described as 'The topic of the resource with content of a bower.json.' The 'i18nHtmlText' property is either a string or an object, described as 'The (internationalized) HTML link text.' It has a default value of 'Browse artifact repository.' No additional properties are allowed at the root level.
