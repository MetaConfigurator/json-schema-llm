The schema describes an error representation. It is an object type. The main properties of the schema are: 'objectType' is a string that is required and must be 'error'. 'context' is a string that is required and describes the context of execution where the error occurred. 'errorCode' is a string that is required and represents the error code. 'msg' is a string that is required and provides a somewhat user-friendly error message. No additional properties are allowed at the root level.
