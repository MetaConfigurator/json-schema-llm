The schema describes an address object. It is an object type. The main properties of the schema are: 'line1' is a string and required. 'city' is a string and required. 'state' is a string and required. 'zip' is a string that must match the pattern '^[0-9]{5}(-[0-9]{4})?$' and is required. 'country' is a string and required. No additional properties are allowed at the root level.
