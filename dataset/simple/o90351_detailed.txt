The schema describes auto-scaling configurations for a nodePool. It is an object type. The main properties of the schema are: 'minNodes' is a number indicating the minimum number of nodes the cluster auto-scaler should scale down to. It is required and must be at least 1. 'maxNodes' is a number indicating the maximum number of nodes the cluster auto-scaler should scale up to. It is required and must be at least 2. 'enabled' is a boolean indicating whether to enable or disable the cluster auto-scaler addon. It is required. No additional properties are allowed at the root level.
