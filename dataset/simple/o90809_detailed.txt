The schema describes a configuration for an OpenVPN admin setup. It is an object type. The main properties of the schema are: 'openvpn-admin' is an object with optional properties. It contains: 'cpus' is a number that describes CPU shares to allocate to each openvpn instance, with a default value of 0.01 and a minimum of 0.01. 'framework-name' is a string that describes the openvpn framework name, with a default value of 'openvpn-admin'. 'instances' is an integer that describes the number of openvpn instances to run, with a default value of 1 and a minimum of 1. 'mem' is a number that describes the memory (MB) to allocate to each openvpn task, with a default value of 128.0 and a minimum of 64.0. No additional properties are allowed at the root level.
