The schema describes a Location. It is an object type. The main properties of the schema are: 'address' is a string representing the address of the location. 'description' is a string providing a description of the location. 'name' is a string representing the name of the location. 'url' is a string, with the description 'Long url for embedding a google map', representing the URL for a Google map. No additional properties are allowed at the root level.
