The schema describes a MWF call-to-action component's definition. It is an object type. The main properties of the schema are: 'config' is an object that may contain 'secondary', which is an optional boolean property indicating if it is disabled. 'content' is an object that is required, containing 'href', which is a string, and 'textContent', which is a string and has a description stating: 'This is the text content for the call to action. Place your call to action content here!'. No additional properties are allowed at the root level.
