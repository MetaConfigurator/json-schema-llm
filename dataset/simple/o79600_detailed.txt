The schema describes details of the social networks enabled by this tenant. It is an array type. It defines several reusable components under 'definitions': 'socialNetwork' is an object, it describes a social network with three required fields: 'name', 'icon', and 'uri'. The 'name' is a string, describing the name of the network, e.g., LinkedIn. The 'icon' is a string with a format of uri, describing the uri of a thumbnail image representing the social network. The 'uri' is a string with a format of uri, describing the uri that a candidate can follow to connect to the social network. No additional properties are allowed at the root level.
