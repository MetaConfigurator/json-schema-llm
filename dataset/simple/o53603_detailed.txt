The schema describes an address. It is an object type. The main properties of the schema are: 'line1' is a string Address Line 1 and is required with a minimum length of 1. 'line2' is a string Address Line 2 and is optional with a minimum length of 1. 'city' is a string City of mailing address of food pantry and is required with a minimum length of 1. 'state' is a string Full name of state and is required with a minimum length of 1. 'zip' is a number Zip code of address and is required. No additional properties are allowed at the root level.
