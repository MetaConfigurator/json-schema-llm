The schema describes parameters for generating an address. It is an object type. The main properties of the schema are: 'options' is an object containing options to control how the address and secret are generated. It is required. It defines several properties: 'entropy' is an array of integers with a minimum value of 0 and a maximum value of 255, describing the entropy to use to generate the seed. 'algorithm' is a string, describing the digital signature algorithm to generate an address for. It has an enum with allowed values: 'ecdsa-secp256k1', 'ed25519'. No additional properties are allowed at the root level.
