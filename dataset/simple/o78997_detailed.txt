The schema describes a set of parameters for a functional MRI experiment. It is an object type. The main properties of the schema are: 'CogAtlasID' is a string representing the cognitive atlas ID, 'EchoTime' is a number representing the echo time, 'EffectiveEchoSpacing' is a number representing the effective echo spacing, 'PhaseEncodingDirection' is a string representing the phase encoding direction, 'RepetitionTime' is a number representing the repetition time, 'SliceEncodingDirection' is a string representing the slice encoding direction, 'SliceTiming' is an array of numbers representing the timing of the slices, and 'TaskName' is a string representing the name of the task. No additional properties are allowed at the root level.
