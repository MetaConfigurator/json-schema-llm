The schema describes parameters for the SetCell procedure. It is an object type. It defines several reusable components under 'definitions': 'MatrixIndex' is an integer, representing the index of a matrix element, with a minimum value of 0. The main properties of the schema are: 't_id' is a number, required. 'key' is a MatrixIndex reference, required. 'code' is a MatrixIndex reference, required. 'text' is a string, required. 'value' is either a number or null, optional. No additional properties are allowed at the root level.
