The schema describes identification number for taxation purposes. It is an object type. The main properties of the schema are: 'name' is a string which is required. 'ssn' is a string that must match the pattern '^[0-9]{3}-[0-9]{2}-[0-9]{4}$' and is optional. 'ein' is a string that must match the pattern '^[0-9]{2}-[0-9]{7}$' and is optional. 'itin' is a string that must match the pattern '^9[0-9]{2}-[0-9]{2}-[0-9]{4}$' and is optional. The schema requires that exactly one of the properties 'ssn', 'ein', or 'itin' be present. No additional properties are allowed at the root level.
