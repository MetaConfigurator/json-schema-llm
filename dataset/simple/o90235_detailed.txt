The schema describes map range information. It is an object type. The main properties of the schema are: 'activeRangeName' is a string Active range ID that slider/picker acts upon. It is required. 'currentRangeExtent' is an array Current range for the active range. It requires exactly 2 items, each of which must be a number. 'fullRangeExtent' is an array Full range extent for the active range to be presented in the UI. It also requires exactly 2 items, each of which must be a number. No additional properties are allowed at the root level.
