The schema describes an operation providing the base for Hypermedia as the Engine of Application State. It is an object type. The main properties of the schema are: 'rel' is a string relation type as defined by the server. It is a required property. 'href' is a string hyperlink to the resource. It is a required property. 'method' is a string indicating the method type, with a default of GET. It is an optional property. No additional properties are allowed at the root level.
