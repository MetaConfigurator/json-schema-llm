The schema describes a Pulse message sent whenever a new result-set is created. It is an object type. The main properties of the schema are: 'version' is a number that is required and must be 1. 'project' is a string that is optional and provides an identifier for the treeherder project, like 'try' or 'mozilla-central'. 'revision_hash' is a string that is required and provides an identifier for the result-set that was created. 'repository_url' is a string that is optional and provides the URL for the repository for the revision. No additional properties are allowed at the root level.
