The schema describes information about the modules in a Puppet code environment. It is an object type. The main properties of the schema are: 'modules' is an array of objects, each object represents the array of modules which exist in an environment. It is required and does not allow additional properties. Each module object has 'name' which is a string that represents the name of the puppet module and is required, and 'version' which can be a string or null and is also required. 'name' is a string that represents the name of the environment and is required. No additional properties are allowed at the root level.
