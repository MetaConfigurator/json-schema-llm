The schema describes the output structure for builders. It is an object type. The main properties of the schema are: 'error' is a string, optional; 'info' is an object, optional; 'success' is a boolean, required; 'target' is an object, optional. The 'target' object contains: 'configuration' which is a string, optional; 'project' which is a string, optional; 'target' which is a string, optional. No additional properties are allowed at the root level.
