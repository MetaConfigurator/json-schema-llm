The schema describes validation parameters accepted by the Pulsepoint adapter. It is an object type. The main properties of the schema are: 'cp' is an integer that identifies the publisher selling the impression and is required. 'ct' is an integer that identifies the ad slot being sold and is required. 'cf' is a string that describes the size of the ad slot being sold, following the pattern '^[0-9]+[xX][0-9]+$' (e.g., '300X250'), and is required. No additional properties are allowed at the root level.
