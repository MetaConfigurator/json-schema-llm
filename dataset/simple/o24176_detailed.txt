The schema describes a codex location. It is an object type. The main properties of the schema are: 'id' is a string and required; 'institution' is a string and optional; 'campus' is a string and optional; 'library' is a string and optional; 'parking' is an array of objects, allowing additional items and ensuring unique items; 'platform' is a string and optional; 'uri' is a string and optional. No additional properties are allowed at the root level.
