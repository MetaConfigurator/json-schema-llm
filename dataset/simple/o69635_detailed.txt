The schema describes messages to communicate with Rc2 compute engine. It is an object type. The main properties of the schema are: 'msg' is a string representing the command to perform, required. It has an enum of allowed values: 'open', 'close', 'saveEnv', 'execScript', 'execAndReturn', 'execFile', 'help', 'listVariables', 'getVariable', 'toggleVariableWatch', 'clearEnvironment', 'createEnvironment'. 'argument' is a string representing the main argument for the command, required. The schema has a minimum of 2 properties and contains conditional schemas under 'oneOf', requiring different fields depending on the value of 'msg'. No additional properties are allowed at the root level.
