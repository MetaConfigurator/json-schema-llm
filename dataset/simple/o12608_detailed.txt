The schema describes ParaDrop host configuration - firewall defaults object. It is an object type. The main properties of the schema are: 'conntrack' is an object optional. 'forward' is a string optional, with allowed values: 'ACCEPT', 'REJECT', 'DROP'. 'input' is a string optional, with allowed values: 'ACCEPT', 'REJECT', 'DROP'. 'masq' is an object optional. 'masq_src' is an array optional, describing a list of source addresses or subnets to which SNAT should be applied, with items being strings and unique items allowed. 'output' is a string optional, with allowed values: 'ACCEPT', 'REJECT', 'DROP'. No additional properties are allowed at the root level.
