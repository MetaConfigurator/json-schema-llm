The schema describes the validation for parameters of getPackageGroup. It is an object type. The main properties of the schema are: 'primary_group' is a string that marks the primary grouping used. It is required and can be one of the following values: 'hotel', 'hotel_regime', 'month_year', 'hote_room_kind', 'place', 'departure_date'. 'secondary_group' is a string that marks the secondary grouping used.
