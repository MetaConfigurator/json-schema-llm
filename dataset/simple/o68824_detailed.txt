The schema describes a Python exception. It is an object type. The main properties of the schema are: 'exc' is a string or null that represents the message from the extracted traceback (should be identical to `message`), which is optional. 'fname' is a string that indicates the name of the file from which the exception was thrown, which is optional. 'func' is a string that specifies the name of the function from which the exception was thrown, which is optional. 'line_number' is a number that denotes the line at which the exception occurred, which is optional. 'message' is a string that contains the message passed to the exception, and it is required. No additional properties are allowed at the root level.
