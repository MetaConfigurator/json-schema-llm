The schema describes a configuration for log management. It is an object type. The main properties of the schema are: 'LogGroupNamePrefix' is a string matching the pattern '.+', and is required. 'RetentionInDays' is an integer with allowed values of 1, 3, 5, 7, 14, 30, 60, 90, 120, 150, 180, 365, 400, 545, 731, and 1827, which are optional. No additional properties are allowed at the root level.
