The schema describes an Okapi Permission Object. It is an object type. The main properties of the schema are: 'permissionName' is a string, optional. 'displayName' is a string, optional. 'description' is a string, optional. 'subPermissions' is an array of strings, optional, where each item must be a string. 'visible' is a boolean, optional. No additional properties are allowed at the root level.
