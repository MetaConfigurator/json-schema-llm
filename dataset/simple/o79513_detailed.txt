The schema describes advertisement status for the supplied advertisement. It is an object type. The main properties of the schema are: 'advertisementId' is a string, optional. 'status' is a string, required, with allowed values 'open', 'expired'. 'postedBy' is a string, optional. 'datePosted' is a number, optional. 'expiredBy' is a string, optional. 'dateExpired' is a number, optional. 'title' is a string, optional. 'referenceCode' is a string, optional. No additional properties are allowed at the root level.
