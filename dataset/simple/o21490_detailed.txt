The schema describes Webtrends Keymetrics Collector Options. It is an object type. The main properties of the schema are: 'additionalFields' is an object, optional. 'chunk-size' is a number, optional. 'empty-data-set' is a boolean, optional. 'idMapping' is an array of strings, optional, with items being of type string. 'mappings' is an object, optional. 'row_type_name' is a string, optional. No additional properties are allowed at the root level.
