The schema describes the details of a rectification. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'situacao' is an object that contains properties 'id' of type string and 'descricao' of type string. 'dataRegistro' is an optional string. 'listaAlteracoes' is an array of objects, each with properties 'codigoCampo' (string), 'origem' (string), 'nomeCampo' (string), 'valorOriginal' (either string or null), and 'valorAlterado' (either string or null). No additional properties are allowed at the root level.
