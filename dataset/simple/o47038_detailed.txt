The schema describes the canonical interface for anything capable of reading Network I/O data. It is an object type. The main properties of the schema are: 'network' is an object that contains the following required properties: 'outbound' is a number representing the network out (KB/s), with a minimum value of 0.0 and a default of 0.0. 'inbound' is a number representing the network in (KB/s), with a minimum value of 0.0 and a default of 0.0. No additional properties are allowed at the root level.
