The schema describes a data structure for managing waiting lists. It is an object type. The main properties of the schema are: 'vorgaenger' is an array containing strings, required; 'warteart' is a string with an enum of allowed values 'beginn' and 'ende', optional with a default value of 'beginn'; 'rechner' is a string, optional; 'wartezeit' is a string, optional. No additional properties are allowed at the root level.
