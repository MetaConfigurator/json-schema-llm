The schema describes a configuration for advertising settings. It is an object type. The main properties of the schema are: 'ads_enabled' is a boolean that indicates whether ads are enabled, and it has a default value of true. 'dfp_ad_unit_path' is a string that must match the pattern '^[\w\/\.:-]+$'. 'dfp_cust_params' is an object that contains pattern properties where the keys must match the pattern '^[a-zA-Z_]{1}[\w\.:-]+$' and the values must be strings that also match the pattern '^[\w\/\.:-]+$'. This object disallows additional properties. No additional properties are allowed at the root level.
