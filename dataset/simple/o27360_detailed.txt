The schema describes EC2 manifest for instance store AMIs. It is an object type. The main properties of the schema are: 'credentials' is an object that contains optional properties: 'certificate' (a string), 'private-key' (a string), and 'user-id' (a string). 'image' is an object that contains a required property: 'bucket' (a string). 'volume' is an object that contains optional properties: 'partitions' (an object) which includes a property 'type' that has an enum with allowed value 'none'. The root schema requires the 'image' property. No additional properties are allowed at the root level.
