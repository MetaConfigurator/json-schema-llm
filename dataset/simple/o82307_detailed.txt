The schema describes time sampling data. It is an object type. The main properties of the schema are: 'ts_ymdh' is an integer representing a timestamp in YMDH format, optional, with a default value of 0, and must be between 0 and 4294967295. 'ts_ymd' is an integer representing a timestamp in YMD format, optional, with a default value of 0, and must be between 0 and 4294967295. 'ts_ym' is an integer representing a timestamp in YM format, optional, with a default value of 0, and must be between 0 and 16777215. No additional properties are allowed at the root level.
