The schema describes GEO Location. It is an object type. The main properties of the schema are: 'longitude' is a number representing the longitude coordinate; 'latitude' is a number representing the latitude coordinate; 'altitude' is a number representing the altitude above sea level; 'accuracy' is a number representing the horizontal accuracy of the location; 'altitude_accuracy' is a number representing the vertical accuracy; 'heading' is a number representing the direction of travel; 'speed' is a number representing the speed at which the object is moving. No additional properties are allowed at the root level.
