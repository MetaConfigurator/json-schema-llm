The schema describes how images should be retrieved or charts constructed. It is an object type. The main properties of the schema are: 'fields' is an array used with charts, containing the names of fields to display in the chart, with unique items; 'linkURL' is a string used with images, containing a URL to be launched in a browser when the image is clicked; 'normalizeField' is an optional string used with charts, containing the name of a field for normalization of data; 'sourceURL' is a string used with images, containing the URL to the image; 'tooltipField' is a string indicating the tooltip for a chart from another field, necessary when related records are not used. No additional properties are allowed at the root level.
