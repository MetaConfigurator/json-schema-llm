The schema describes a binding for an authentication policy label. It is an object type. The main properties of the schema are: 'activepolicy' is an integer, readonly. 'boundto' is a string, optional. 'gotopriorityexpression' is a string, readonly. 'name' is a string, optional. 'nextfactor' is a string, readonly. 'priority' is an integer, readonly. No additional properties are allowed at the root level.
