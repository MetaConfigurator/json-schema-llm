The schema describes a search configuration. It is an object type. The main properties of the schema are: 'search' is an object that is required and contains a required property 'resource' which is a string with a format of 'topic'. It also has an optional property 'parameter' which is a string with a default value of 'q'. 'i18n' is an optional object that may contain an optional property 'locale', which is a string with a default value of 'default'. No additional properties are allowed at the root level.
