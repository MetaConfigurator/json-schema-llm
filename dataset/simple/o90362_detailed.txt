The schema describes AWS Security Configuration. It is an object type. The main properties of the schema are: 'protocol' is a string that describes the protocol for this Security Rule. It is required and can take one of the following values: 'TCP', 'UDP', '-1', 'ICMP'. 'cidr_blocks' is an array that describes the CIDR to associate this rule with. It is required, has a default value of ['0.0.0.0/0'], and contains items of type string. 'from_port' is a number that describes the from port number. It is required. 'to_port' is a number that describes the to port number. It is required. No additional properties are allowed at the root level.
