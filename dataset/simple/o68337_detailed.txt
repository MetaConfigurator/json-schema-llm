The schema describes parameters accepted by the Sovrn adapter. It is an object type. The main properties of the schema are: 'tagid' is a string that identifies the Sovrn ad tag. It is optional. 'tagId' is a string that identifies the Sovrn ad tag (DEPRECATED, use "tagid" instead). It is optional. 'bidfloor' is a number representing the minimum acceptable bid, in CPM, using US Dollars. It is optional. The schema requires one of the following configurations: either 'tagid' must be present or 'tagId' must be present. No additional properties are allowed at the root level.
