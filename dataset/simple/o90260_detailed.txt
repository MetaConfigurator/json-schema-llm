The schema describes rendering points with fixed real world or screen space size. It is an object type. The main properties of the schema are: 'size' is a number representing the symbol size in real world units or display unit, and is required. 'type' is a string and must be one of the enumerated values: 'pointCloudFixedSizeAlgorithm', and is also required. 'useRealWorldSymbolSizes' is a boolean that indicates if the symbol size is in meters (true) or a display unit (pt) otherwise, and is optional. No additional properties are allowed at the root level.
