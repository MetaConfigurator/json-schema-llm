The schema describes a savings account annual summary. It is an object type. The main properties of the schema are: 'taxedUkInterest' is a number representing the total net taxed interest for the specified savings account. It is optional, with a value that must be between 0 and 99999999999.99 up to 2 decimal places, with a multiple of 0.01 and an example value of '1230.55'. 'untaxedUkInterest' is a number representing the total untaxed interest for the specified savings account. It is optional, with a value that must be between 0 and 99999999999.99 up to 2 decimal places, with a multiple of 0.01 and an example value of '500.23'. No additional properties are allowed at the root level.
