The schema describes an object structure for event tags. It is an object type. The main properties of the schema are: 'eventTags' is an array that contains objects. Each object has required properties 'key' and 'value'. The 'key' is a string that must match the pattern '^[0-9a-zA-Z_-]{1,255}$'. The 'value' is a string with a minimum length of 1 and a maximum length of 255. No additional properties are allowed at the root level.
