The schema describes an array of data table rows for insertion or updating. It is an array type. The main properties of the schema are: Each item is an object representing a data table row. Each object can have properties matching the pattern '^[0-9a-zA-Z_-]{1,255}$', which indicates that property names must be alphanumeric characters, underscores, or hyphens, and must be between 1 and 255 characters long.
