The schema describes a configuration object. It is an object type. The main properties of the schema are: 'client_token' is a string representing a token for the client and is required. 'policies' is an array of strings that must be provided and is required. 'metadata' is an object, which must be included and contains several properties: 'aws_region' is a string, 'username' is a string, 'is_admin' is a string, 'groups' is a string, and 'aws_iam_principal_arn' is a string. 'lease_duration' is an integer representing the duration of the lease and is required. 'renewable' is a boolean indicating if the lease is renewable and is required. No additional properties are allowed at the root level.
