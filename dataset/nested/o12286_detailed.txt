The schema describes formatting options for numbers. It is an object type. The main properties of the schema are: 'minimumIntegerDigits' is a number non-negative integer value indicating the minimum integer digits to be used. 'minimumFractionDigits' is a number non-negative integer value indicating the minimum and maximum fraction digits to be used. 'maximumFractionDigits' is a number non-negative integer value indicating the minimum and maximum fraction digits to be used. 'minimumSignificantDigits' is a number positive integer value indicating the minimum and maximum fraction digits to be shown. 'maximumSignificantDigits' is a number positive integer value indicating the minimum and maximum fraction digits to be shown. 'round' is a string with rounding method ceil, floor, round (default), or truncate. 'useGrouping' is a boolean (default is true) value indicating whether a grouping separator should be used. 'mustDivideBy100' is a boolean value indicating if the current value needs to be divided by 100 to be properly displayed by the formatter. 'emptyPlaceholder' is a string value used in case of null/undefined value. No additional properties are allowed at the root level.
