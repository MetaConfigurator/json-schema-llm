The schema describes a configuration object for fabric networking. It is an object type. The main properties of the schema are: 'fabric_uuid' is a string representing the UUID of the fabric object created. It is required and has a default value of an empty string with examples like '123456789'. 'fabric_fq_name' is an array of strings representing the fully qualified name of the fabric object created. It is required and can contain multiple items, each having a default value of an empty string and examples like 'default-global-system-config' and 'fabric01'. 'manage_underlay' is a boolean indicating whether to configure device underlay networking. It is optional. The schema enforces a minimum of 1 property and a maximum of 1 property at the root level. No additional properties are allowed at the root level.
