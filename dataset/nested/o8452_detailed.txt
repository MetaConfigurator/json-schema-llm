The schema describes configuration options for interface imports. It is an object type. The main properties of the schema are: 'import_configured' is a boolean not importing configured interfaces by default. Set this option to true if configured interfaces need to be imported. It is optional and has a default value of false. 'interface_filters' is an array of filter objects, each containing 'op' and 'expr' fields. It is optional. Each item in the array must be an object with a maximum of 2 properties and no additional properties allowed. The 'op' field must have an enum value of 'regex', and the 'expr' field is a string. No additional properties are allowed at the root level.
