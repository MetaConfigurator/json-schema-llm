The schema describes an invocation manifest for FreeSurfer: MBIRN Defacer for structural MRI (mri-deface v1.22). It is an object type. The main properties of the schema are: 'inputs' is an object which is required and contains a required property 'anatomical'. The 'anatomical' property is an object that includes 'type', which is a string with an enum of allowed values: 'nifti', 'mgh', 'dicom'. 'config' is an object which is required and contains a required property 'output_extension'. The 'output_extension' property is a string with a default value of '.nii.gz'. No additional properties are allowed at the root level.
