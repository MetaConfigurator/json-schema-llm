The schema describes a specification of an image transform. It is an object type. It defines several reusable components under 'definitions': 'anyTransformSpec' is an object, set of possible transform specification types that can be either interpolated, leaf, list, or reference transforms. 'interpolatedTransformSpec' is an object, specification for an interpolation between 2 transforms. It requires 'type', 'a', 'b', and 'lambda'. 'leafTransformSpec' is an object, specification for a transform that can be directly implemented. It requires 'className' and 'dataString'. 'listTransformSpec' is an object, ordered list of transform specifications. It requires 'type' and 'specList'. 'referenceTransformSpec' is an object, reference to another transform specification. It requires 'type' and 'refId'. 'transformId' is a string, uniquely identifying a transform specification. 'transformMetaData' is an object, any data about the transform that is not needed for rendering. It has an optional property 'labels', which is an array of unique strings.
