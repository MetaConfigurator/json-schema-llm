The schema describes pagination and filtering for data retrieval. It is an object type. The main properties of the schema are: 'page' is an integer that describes the page number. 'pageSize' is an integer that describes the page size. 'totalResults' is an integer that describes the number of results. 'sortBy' is an object that describes sorting, containing: 'direction', which is an integer describing the direction, and 'property', which is a string describing the property. 'filters' is an array that describes filters, containing unique items, where each item is an object with properties: 'property', which is a string describing the property, and 'value', which is a string describing the value. No additional properties are allowed at the root level.
