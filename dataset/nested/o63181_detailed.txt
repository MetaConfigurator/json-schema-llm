The schema describes validation of resolution rules. It is an object type. The main properties of the schema are: 'align' is an array that contains objects. Each object is required to have the fields 'includes', 'excludes', 'reason', 'author', 'date', and 'group', and cannot contain additional properties. The 'deny', 'exclude', 'reject', 'replace', and 'substitute' properties are all optional arrays. The schema requires the properties 'align', 'deny', 'exclude', 'reject', 'replace', and 'substitute' to be present. No additional properties are allowed at the root level.
