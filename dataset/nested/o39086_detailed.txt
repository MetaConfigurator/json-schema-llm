The schema describes a configuration for converting files to MPEG4 format. It is an object type. The main properties of the schema are: '$schema' is a string optional field. '$tool' is a string required field that must be set to 'to-mpeg4'. 'dist' is a string required field that specifies the output directory relative to the current working directory. 'history' is a string required field with a default value of '.history/' that specifies the history directory relative to the current working directory. 'outputOptions' is an array of strings optional field that contains Ffmpeg output options in key-value mode, such as '-vf scale=640:360'. 'src' is an array of strings required field that specifies input Go files, expanding with environment variables first; it must have at least one item and can include patterns like '~/movie/non-mp4/**/*.{mp4,mkv,avi,flv}'. No additional properties are allowed at the root level.
