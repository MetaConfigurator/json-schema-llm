The schema describes configuration settings for a screenshot tool. It is an object type. The main properties of the schema are: 'windowSize' is an object representing the size of the window with optional properties 'width' (integer, default 1024) and 'height' (integer, default 768). 'shotSize' is an object representing the size of the shot with optional properties 'width' (can be an integer or string, default 1024) and 'height' (can be an integer or string, default 768). 'shotOffset' is an object representing the offset of the shot with optional properties 'left' (integer, default 0), 'right' (integer, default 0), 'top' (integer, default 0), and 'bottom' (integer, default 0). 'phantomConfig' is an object containing the property 'ignore-ssl-errors' (string, default 'false'). 'cookies' is an array. 'customHeaders' is either null or an object (default null). 'defaultWhiteBackground' is a boolean (default false). 'customCSS' is a string (default ''). 'quality' is an integer (default 75). 'streamType' is a string (default 'png'). 'renderDelay' is an integer (default 0). 'timeout' is an integer (default 0). 'takeShotOnCallback' is a boolean (default false). 'errorIfStatusIsNot200' is a boolean (default false). 'errorIfJSException' is a boolean (default false). 'captureSelector' is either a boolean or a string (default false). The schema supports two configurations through 'oneOf': one requires 'siteType' as 'url' and 'url' (min length 1); the other requires 'siteType' as 'html' and 'html' (min length 1). No additional properties are allowed at the root level.
