The schema describes a configuration profile for a sidebar. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string that describes the payload, optional with a default value of 'Sidebar Preferences'. 'PayloadDisplayName' is a string that represents the name of the payload, optional with a default value of 'Sidebar Preferences'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, dot-delimited, optional with a default value of 'com.apple.MCX.sidebar'. 'PayloadOrganization' is a string that describes the issuing organization of the profile, as displayed to the user, optional. 'PayloadType' is a string that defines the type of the payload, a reverse dns string, optional with a default value of 'com.apple.MCX.sidebar'. 'PayloadUUID' is a string that is a unique identifier for the payload, formatted as 01234567-89AB-CDEF-0123-456789ABCDEF, optional with a default value of '', validated by a specific pattern. 'PayloadVersion' is a number representing the version of the whole configuration profile, optional with a default value of 1. 'SidebarDestination' is a string with an unknown purpose, optional. 'customBaseURLs' is an array that allows for base URLs to custom locations, optional. 'hideiCloudDrive' is a boolean that indicates whether to hide iCloud Drive, optional. 'shouldAddNetworkHome' is a boolean that indicates whether to add Network Home, optional. 'shouldAddUserHome' is a boolean that indicates whether to add User Home, optional. No additional properties are allowed at the root level.
