The schema describes the result of the getDepoEx procedure. It is an object type. The main properties of the schema are: 'depo_ex' is an object that contains the results with several required fields: 'sec_code', 'trdaccid', 'firmid', 'client_code', 'openbal', 'openlimit', 'currentbal', 'currentlimit', 'locked_sell', 'locked_buy', 'locked_buy_value', 'locked_sell_value', 'wa_position_price', 'limit_kind'.
