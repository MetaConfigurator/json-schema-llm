The schema describes Angular class options. It is an object type. The main properties of the schema are: 'appRoot' is a string optional field. 'export' is a boolean optional field that defaults to false and specifies if declaring module exports the directive. 'flat' is a boolean optional field that defaults to true and indicates if a dir is created. 'module' is a string optional field that allows specification of the declaring module. 'name' is a string required field. 'path' is a string optional field that defaults to 'app'. 'prefix' is a string optional field that defaults to 'app' and is the prefix to apply to generated selectors. 'selector' is a string optional field. 'skipImport' is a boolean optional field that defaults to false. 'sourceDir' is a string optional field that defaults to 'src'. 'spec' is a boolean optional field that defaults to true and specifies if a spec file is generated. No additional properties are allowed at the root level.
