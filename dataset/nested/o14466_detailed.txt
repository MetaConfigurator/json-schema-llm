The schema describes a collection of action types. It is an array type. The main properties of the schema are: [items] is an object that represents an action type. It includes the following required fields: 'id' is an integer that serves as a unique identifier given by the server, 'name' is a string that identifies the action type, 'public' is a boolean that indicates if the action type is available to everybody, 'type' is a string that must be a valid URL to define the action type, 'organizationId' is an integer that links to the organization, 'deletable' is a boolean that defines if the action type can be deleted, and 'schema' is an object that must conform to a valid JSON Schema. No additional properties are allowed at the root level.
