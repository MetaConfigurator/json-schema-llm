The schema describes a Poll object for normal or trivia polls. It is an object type. It defines several reusable components under 'definitions': 'poll_answer' is an object, a Poll answer object for both trivia and normal polls. It requires the fields 'id', 'uuid', and 'title'. Additional properties are not allowed. The 'id' is an integer, the 'uuid' is a string, and 'title' is a string. The field 'correct' is a boolean. The field 'un_enum' has an enum with allowed values: 'apo', 'apolo', 'apolinares'. 'image' is an object, describing an image with a required field '@type' (string with value 'image') and 'cloudinary_id' (string). It may also include 'caption' (string), 'height' (integer), 'width' (integer), and 'original_format' (string with allowed values: 'jpg', 'gif', 'png', 'jpeg'). The main properties of the schema are: '@type' is a string and required, with the enum value 'poll'. 'id' is an integer and required. 'uuid' is a string and required. '@poll_type' is a string and required, with allowed values: 'poll', 'trivia'. 'title' is a string and required. 'image' resolves to the referenced definition. 'answers' is an array of poll_answer objects and is required. No additional properties are allowed at the root level.
