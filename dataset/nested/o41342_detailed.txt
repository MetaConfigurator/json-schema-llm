The schema describes an invocation manifest for MRIQC. It is an object type. The main properties of the schema are: 'config' is an object containing configuration settings; it is required. It has four required properties: 'measurement' is a string with a default value of 'auto-detect' and allowed values: 'auto-detect', 'functional', 'T1', 'T2'; 'save_derivatives' is a boolean with a default value of false; 'save_outputs' is a boolean with a default value of false; 'verbose_reports' is a boolean with a default value of false. 'inputs' is an object containing input settings; it is required. It has one required property: 'nifti' is an object with a required property: 'type', which has an allowed value: 'nifti'. No additional properties are allowed at the root level.
