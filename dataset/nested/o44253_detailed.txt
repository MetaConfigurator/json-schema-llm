The schema describes an instance of a director animation. It is an object type. The main properties of the schema are: 'delay' is a number that specifies a timer to wait for start, with no default value and is optional. 'enabled' is a boolean indicating if the animation is enabled, with no default value and is optional. 'idleTime' is a number specifying a time to wait for start after user interaction, with no default value and is optional. 'loop' is a boolean indicating if the animation loops, with no default value and is optional. 'random' is a boolean specifying if the tracks are played randomly, with no default value and is optional. 'smooth' is a boolean indicating if the animation has a smooth start, with no default value and is optional. 'stoppable' is a boolean that determines if the user can stop the animation, with no default value and is optional. 'tracks' is an array of strings that lists the UIDs of tracks to execute, which is required. No additional properties are allowed at the root level.
