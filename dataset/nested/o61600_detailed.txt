The schema describes a CalDAV account configuration. It is an object type. The main properties of the schema are: 'CalDAVAccountDescription' is a string The display name of the account (e.g. "Company CalDAV Account"); it has a default value of 'My CalDAV Account'. 'CalDAVHostName' is a string The CalDAV server hostname or IP address; it is optional. 'CalDAVPassword' is a string The CalDAV password; it is optional. 'CalDAVPort' is a number The port on which to connect to the server; it has a default value of 443, with a minimum value of 0 and a maximum value of 65535. 'CalDAVPrincipalURL' is a string The base URL to the user's calendar; it is optional. 'CalDAVUseSSL' is a boolean Enable Secure Socket Layer communication with CalDAV server; it has a default value of true. 'CalDAVUsername' is a string The CalDAV username; it is optional. 'PayloadDescription' is a string Description of the payload; it has a default value of 'Configures a CalDAV account'. 'PayloadDisplayName' is a string Name of the payload; it has a default value of 'CalDAV'. 'PayloadIdentifier' is a string A unique identifier for the payload, dot-delimited. Usually root PayloadIdentifier+subidentifier; it has a default value of 'com.apple.caldav.account'. 'PayloadOrganization' is a string This value describes the issuing organization of the profile, as displayed to the user; it is optional. 'PayloadType' is a string The type of the payload, a reverse dns string; it has a default value of 'com.apple.caldav.account'. 'PayloadUUID' is a string Unique identifier for the payload (format 01234567-89AB-CDEF-0123-456789ABCDEF); it has a default value of '', and must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is a number The version of the whole configuration profile; it has a default value of 1. No additional properties are allowed at the root level.
