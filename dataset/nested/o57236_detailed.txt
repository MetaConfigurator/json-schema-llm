The schema describes a client profiles array. It is an array type. It defines several reusable components under 'definitions': The main properties of the schema are: 'name' is a string representing the name of the client profile. It is required. 'priority' is an integer representing the priority of the client profile. It is optional. 'hosts' is an array of objects representing the hosts. Each object in 'hosts' must have 'host' (a string), 'protocol' (a string), and 'port' (an integer), which are all required. No additional properties are allowed at the root level.
