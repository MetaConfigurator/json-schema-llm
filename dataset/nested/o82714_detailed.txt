The schema describes a YouTube video block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$'. It is optional and is used in rendering environments to append to the DOM elements' class attribute. 'updated_date' is a string in date-time format that indicates when the node this block is attached to was updated. It is optional. 'aside' is a boolean that indicates if the block represents content only indirectly related to the main content. It is optional. 'id' is a string that must match the pattern '^[\w-]+$'. It is required. 'autoplay' is a boolean that indicates if the video should autoplay. It is optional. 'start_at' is an integer with a default value of 0, having a minimum of 0 and a maximum of 65535. It is optional. 'poster_image_ref' is a string that must match the pattern '^[\w\/\.:-]+$'. It is optional and is a reference to an image asset to use as the poster. No additional properties are allowed at the root level.
