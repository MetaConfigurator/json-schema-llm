The schema describes the structure of a content item. It is an object type. It defines several reusable components under 'definitions': 'guid' is a string, with a pattern that ensures it follows the UUID format. 'guid_list' is an array, containing unique items that reference the 'guid' definition. The main properties of the schema are: 'bulk_publishing' is a boolean, optional, indicating whether bulk publishing is allowed. 'links' is an object, required, with properties that include multiple references to 'guid_list', each serving a specific purpose related to the content item. These properties include 'facet_groups', 'facet_values', 'finder', 'lead_organisations', 'linked_items', 'mainstream_browse_pages', 'meets_user_needs', 'ordered_related_items', 'ordered_related_items_overrides', 'organisations', 'original_primary_publishing_organisation', 'parent', 'policy_areas', 'primary_publishing_organisation', 'suggested_ordered_related_items', 'taxons', and 'topics'. 'previous_version' is a string, optional, representing the previous version of the content item. No additional properties are allowed at the root level.
