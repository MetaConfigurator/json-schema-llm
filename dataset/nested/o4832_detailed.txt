The schema describes USI submittable attributes schema. It is an object type. The main properties of the schema are: 'attributes' is an object describing attributes for describing a submittable. It is optional. It has no defined properties. It allows pattern properties which match any string, where each corresponding value must be an array containing at least one item. Each item in the array is an object that requires a 'value' field of type string with a minimum length of 1. It may optionally have a 'units' field of type string with a minimum length of 1. Additionally, it may include a 'terms' field that is an array of objects, each of which must have a 'url' field of type string formatted as a URI. No additional properties are allowed at the root level.
