The schema describes a logging configuration profile. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string that describes the payload. It is optional and has a default value of 'Unified Logging Configuration'. 'PayloadDisplayName' is a string that names the payload. It is optional and has a default value of 'Unified Logging Configuration'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, dot-delimited. It is optional and has a default value of 'com.apple.system.logging'. 'PayloadOrganization' is a string that describes the issuing organization of the profile, as displayed to the user. It is optional. 'PayloadType' is a string that indicates the type of the payload, a reverse dns string. It is optional and has a default value of 'com.apple.system.logging'. 'PayloadUUID' is a string that acts as a unique identifier for the payload, following a specific pattern format. It is optional and has a default value of an empty string. 'PayloadVersion' is a number that represents the version of the whole configuration profile. It is optional and has a default value of 1. 'Processes' is an object used to configure process logging. It is optional. 'Subsystems' is an object used to configure subsystem logging. It is optional. Within 'Subsystems', 'subsystemDomain' is an object with properties that require further description. 'System' is an object used to configure system logging. It is optional. No additional properties are allowed at the root level.
