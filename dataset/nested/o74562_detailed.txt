The schema describes lists of Wangscape output tiles with locations, grouped by corner terrain types. It is an object type. The main properties of the schema are: 'patternProperties' is an object that defines patterns for the keys. Each key must match the pattern '^[0-9a-zA-Z_-]+\.[0-9a-zA-Z_-]+\.[0-9a-zA-Z_-]+\.[0-9a-zA-Z_-]+$' and its value is an array of tile locations. Each item in the array is an object that specifies the location of a Wangscape output tile. The tile location schema is an object type that has the following required properties: 'filename' is a string that specifies the filename of the tileset containing the tile. 'x' is an integer that specifies the horizontal offset, in pixels, of the tile within the tileset, with a minimum value of 0. 'y' is an integer that specifies the vertical offset, in pixels, of the tile within the tileset, with a minimum value of 0. No additional properties are allowed at the root level.
