The schema describes GCP Notebook Scheduler Plugin Settings. It is an object type. The main properties of the schema are: 'acceleratorCount' is a string that describes the accelerator count used for the AI Platform training job. 'acceleratorType' is a string that describes the accelerator type used for the AI Platform training job. 'containerImage' is a string that describes the Docker container used in the AI Platform training job. 'gcsBucket' is a string that describes the top-level GCS Bucket path where Notebooks will be written to. 'jobRegion' is a string that describes the region where the AI Platform training job will execute the notebook. 'masterType' is a string that describes the machine type used for the AI Platform training job. 'oAuthClientId' is a string that describes the OAuth 2.0 Client ID for authenticating end-users for API requests. 'projectId' is a string that describes the Google Cloud project identifier. 'scaleTier' is a string that describes the scale tier used for the AI Platform training job. 'schedulerRegion' is a string that describes the region where the AppEngine app is deployed for the projectId. No additional properties are allowed at the root level.
