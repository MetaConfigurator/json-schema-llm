The schema describes a geometry object type. It is an object type. It defines several reusable components under 'definitions': 'GeometryCollection' is an object, it requires a 'geometries' property which is an array of geometries. 'Point' is an object, it requires a 'coordinates' property which is a position. 'MultiPoint' is an object, it requires a 'coordinates' property which is an array of position. 'LineString' is an object, it requires a 'coordinates' property which is a lineString. 'MultiLineString' is an object, it requires a 'coordinates' property which is an array of lineString. 'Polygon' is an object, it requires a 'coordinates' property which is a polygonCoordinates. 'MultiPolygon' is an object, it requires a 'coordinates' property which is an array of polygonCoordinates. The main properties of the schema are: 'type' is a string, it is required. 'crs' is an object, it is optional. The schema requires the object to comply with one of the following definitions: 'GeometryCollection', 'Point', 'MultiPoint', 'LineString', 'MultiLineString', 'Polygon', or 'MultiPolygon'. No additional properties are allowed at the root level.
