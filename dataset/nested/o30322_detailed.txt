The schema describes a content inspection action. It is an object type. The main properties of the schema are: 'builtin' is an object with a readonly status. It has an enum with allowed values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'feature' is a string with a readonly status. It has an enum with allowed values including 'WL', 'WebLogging', 'SP', 'SurgeProtection', and many others. 'hits' is an integer with a readonly status. 'icapprofilename' is an optional string. 'ifserverdown' is a string with an enum containing 'CONTINUE', 'DROP', and 'RESET'. 'name' is an optional string. 'referencecount' is an integer with a readonly status. 'reqtimeout' is an integer with a readonly status. 'reqtimeoutaction' is a string with an enum containing 'BYPASS', 'DROP', and 'RESET', and it is readonly. 'serverip' is an optional string. 'servername' is an optional string. 'serverport' is an optional integer. 'type' is a string with an enum containing 'ICAP', 'INLINEINSPECTION', 'MIRROR', and 'NOINSPECTION'. 'undefhits' is an integer with a readonly status. No additional properties are allowed at the root level.
