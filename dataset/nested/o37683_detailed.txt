The schema describes a user. It is an object type. The main properties of the schema are: 'id' is an integer the unique identifier. 'email' is a string an email address with a maximum length of 255 characters and must be in email format. 'name' is a string the user's name with a maximum length of 255 characters. 'password' is a string a hash of the user's password. 'role' is a string the role assigned to the user with allowed values 'anonymous', 'authenticated', 'admin'. 'status' is a string the status of the user with allowed values 'created', 'banned', 'authorized'. 'created' is a string Timestamp when created in date-time format. 'modified' is a string Timestamp when last modified in date-type format. No additional properties are allowed at the root level.
