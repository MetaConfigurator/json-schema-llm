The schema describes installation parameters provided for a CBSD. It is an object type. The main properties of the schema are: 'latitude' is a number representing Latitude in WGS84 coordinates, provided in decimal degrees. It is required with a minimum of -90 and a maximum of 90. 'longitude' is a number representing Longitude in WGS84 coordinates, provided in decimal degrees. It is required with a minimum of -180 and a maximum of 180. 'height' is a number representing the Height of CBSD antenna, provided in meters. It is required. 'heightType' is a string that must be one of the following: 'AMSL', 'AGL'. It is required. 'horizontalAccuracy' is a number representing a measurement in meters of the horizontal accuracy of the indicated location. It is optional with a minimum of 0. 'verticalAccuracy' is a number representing a measurement in meters of the vertical accuracy of the indicated location. It is optional with a minimum of 0. 'indoorDeployment' is a boolean indicating if the equipment is installed indoors, required if true, and false if outdoors. 'eirpCapability' is an integer representing the Maximum EIRP of the CBSD in units of dBm/10MHz. It is optional with a minimum of -127 and a maximum of 47. 'antennaAzimuth' is an integer representing the Heading of antenna azimuth relative to true north, provided in decimal degrees. It is optional with a minimum of 0 and a maximum of 359. 'antennaDowntilt' is an integer representing the Downtilt of the antenna (vertically down is positive) in decimal degrees. It is optional with a minimum of -90 and a maximum of 90. 'antennaGain' is an integer representing the Maximum gain in dB of the antenna along the boresight. It is required with a minimum of -127 and a maximum of 128. 'antennaBeamwidth' is an integer representing the Beamwidth to 3dB points of the antenna coverage pattern, provided in decimal degrees and measured at the horizon. It is optional with a minimum of 0 and a maximum of 360. 'antennaModel' is a string representing the model of the antenna if an external antenna is used. It is optional. No additional properties are allowed at the root level.
