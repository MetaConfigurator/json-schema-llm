The schema describes the metadata format for all federal open data. It is an object type. The main properties of the schema are: 'agencyCode' is a string that represents the Agency Code, which is optional. 'productNumber' is a string that represents the Product Number, which is required. 'agencyName' is a string that represents the Agency Name, which is required. 'engagementCode' is a string that represents the Engagement Code, which is optional. 'recommendationText' is a string that represents the Recommendation Text, which is optional. 'statusComments' is a string that represents the Status Comments, which is optional. 'status' is a string that represents the Status, which is required and can be either 'Open' or 'Closed'. No additional properties are allowed at the root level.
