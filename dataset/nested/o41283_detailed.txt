The schema describes an invocation manifest for FSL: fslreorient2std - Reorient Image to Standard Template. It is an object type. The main properties of the schema are: 'inputs' is an object that is required, containing a required property 'nifti', which is also an object with a required property 'type' that has an enum value 'nifti'. 'config' is an object that is required, containing an optional property 'output_name' that is a string. No additional properties are allowed at the root level.
