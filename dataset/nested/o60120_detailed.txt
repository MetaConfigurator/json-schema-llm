The schema describes a document returned by GET /{db}/_design/{ddoc}. It is an object type. The main properties of the schema are: '_id' is a string and is required. '_rev' is a string and is required. 'language' is a string and is optional. 'options' is an object and is optional. 'filters' is an object and is optional. 'lists' is an object and is optional. 'rewrites' is an array of objects and is optional. 'shows' is an object and is optional. 'updates' is an object and is optional. 'validate_doc_update' is a string and is optional. 'views' is an object and is optional. No additional properties are allowed at the root level.
