The schema describes a MWF In-page navigation component's schema definition. It is an object type. The main properties of the schema are: 'content' is an object that contains 'links' and 'dropdown-menu', both of which are required. 'links' is an array of objects, where each object must include required properties: 'href' (string), 'selected' (either 'f-active' or ''), and 'text' (string). 'dropdown-menu' is an object that contains required properties: 'config' (an object with 'context' as enum 'sticky-on-collapsed' and another boolean 'hidden'), and 'content' (an object with required properties 'href' (string) and 'menu-items' (an array of objects). Each menu-item object within 'menu-items' must include 'config' (an object with the required property 'type' as enum 'li') and 'content' (an object with required properties 'text' (string) and 'href' (string)). Additionally, 'config' in 'content' has a dependency on 'context' being 'vertical'. No additional properties are allowed at the root level.
