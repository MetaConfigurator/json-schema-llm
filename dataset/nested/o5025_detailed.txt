The schema describes a metric. It is an object type. The main properties of the schema are: '_id' is a string that is required with a minimum length of 1 and a maximum length of 36. 'kind' is a string that is required. 'metadata' is an object that is required and contains properties 'creationTimestamp' (a number), 'deletionTimestamp' (a number), and 'resourceVersion' (a number). 'data' is an object that is required. 'involvedObject' is an object that is required and contains properties 'kind' (a string), 'name' (a string), and 'uid' (a string). 'timestamp' is a string that is optional. No additional properties are allowed at the root level.
