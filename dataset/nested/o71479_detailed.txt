The schema describes an event configuration. It is an object type. The main properties of the schema are: 'calendarUrl' is a string optional field, 'city' is a string required field, 'classes' is a boolean required field, 'events' is an array of objects with 'title' as a string and 'url' as a string, 'groups' is an array of objects with 'calendarUrl' as a string, 'kind' as an enum with allowed values 'society' and 'school', 'title' as a string, and 'url' as a string, 'latLng' is an array of numbers required field, 'parties' is a boolean required field, 'socials' is a boolean required field, and 'styles' is an array of enums with allowed values 'Lindy Hop', 'Balboa', 'Charleston', 'Shag', and 'Blues' required field. No additional properties are allowed at the root level.
