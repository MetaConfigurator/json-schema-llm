The schema describes an order list for AMTweaker. It is an array type. The main properties of the schema are: [request] is a string requested item. Either an ore dictionary tag or an item in the form modid:item:damage. It is required. It must match one of the specified patterns: '^[^:]+$' or '^[^:]+:[^:]+:[0-9]+$'. [amount] is an integer amount of items necessary to fulfill the order. It is required and must be at least 1. [reward] is an integer MP reward for fulfilling the order. It is required and must be at least 1. [type] is a string order type. It is required and can be one of the following values: 'single', 'short', 'middle', or 'long'. [name] is a string display name for the order. It is required. [season] is a string order season. It can be one of the following values: 'none', 'spring', 'summer', 'autumn', or 'winter'. [biome] is a string order biome category. It can be one of the following values: 'none', 'plane', 'cold', 'arid', 'damp', or 'hell'. No additional properties are allowed at the root level.
