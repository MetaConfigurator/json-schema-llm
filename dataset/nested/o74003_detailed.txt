The schema describes a multi-type testing configuration. It is an object type. It defines several reusable components under 'definitions': 'action' is an object, containing optional properties 'default_title' (a string describing the tooltip for the main toolbar icon), 'default_popup' (an object reference to the 'icon' definition), and 'default_icon' (an object with optional properties '19' and '38', both referencing the 'icon' definition). 'icon' is an object with optional properties 'description' (a string) and 'suggested_key' (an object with pattern properties that require matching specific keys and a string pattern). No additional properties are allowed at the root level. The main properties of the schema are: 'model_version' is a number required with an enum value of 1, a maximum of 1, and a minimum of 1. 'default_locale' is a string optional with a default value of 'en'.
