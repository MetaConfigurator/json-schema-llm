The schema describes a Remote Start Transaction Request. It is an object type. The main properties of the schema are: 'connectorId' is an integer optional property. 'idTag' is a string required property with a maximum length of 20. 'chargingProfile' is an object required property that contains several properties. 'chargingProfileId' is an integer required property. 'transactionId' is an integer optional property. 'stackLevel' is an integer required property. 'chargingProfilePurpose' is a string required property with enum values: "ChargePointMaxProfile", "TxDefaultProfile", "TxProfile". 'chargingProfileKind' is a string required property with enum values: "Absolute", "Recurring", "Relative". 'recurrencyKind' is a string optional property with enum values: "Daily", "Weekly". 'validFrom' is a string optional property with date-time format. 'validTo' is a string optional property with date-time format. 'chargingSchedule' is an object required property that contains several properties. 'duration' is an integer optional property. 'startSchedule' is a string optional property with date-time format. 'chargingRateUnit' is a string required property with enum values: "A", "W". 'chargingSchedulePeriod' is an array required property with items that are objects. Each object in 'chargingSchedulePeriod' has 'startPeriod' as an integer required property and 'limit' as a number required property that must be a multiple of 0.1. 'numberPhases' is an integer optional property. 'minChargingRate' is a number optional property that must be a multiple of 0.1. No additional properties are allowed at the root level.
