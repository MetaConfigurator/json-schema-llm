The schema describes a list of tile contents and locations for a Wangscape project. It is an array type. The main properties of the schema are: items is an object that specifies the contents and location of a tile output by Wangscape. It has the following properties: 'corners' is an array that specifies the IDs of the terrain types at the corners of the tile, and it is required. The items of this array are strings that must match the pattern '^[0-9a-zA-Z_-]+$'. 'filename' is a string that specifies the filename of the tileset containing the tile, and it is required. 'x' is an integer that specifies the horizontal offset, in pixels, of the tile within the tileset, and it is required with a minimum value of 0. 'y' is an integer that specifies the vertical offset, in pixels, of the tile within the tileset, and it is required with a minimum value of 0. No additional properties are allowed at the root level.
