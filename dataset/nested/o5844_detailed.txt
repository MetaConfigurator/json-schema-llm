The schema describes a farmware manifest. It is an object type. The main properties of the schema are: 'package' is a string that represents the package name and is required. 'language' is a string that represents the package programming language and is optional. 'author' is a string that represents the package author and is required. 'description' is a string that represents the package description and is optional. 'package_version' is a string that represents the package version and is required. 'farmware_manifest_version' is a string that represents the farmware manifest version and is required. 'farmbot_os_version_requirement' is a string that represents the FarmBot OS version requirement and is required. 'farmware_tools_version_requirement' is a string that represents the Farmware Tools version requirement and is required. 'url' is a string that represents the URL to locate this package and is optional. 'zip' is a string that represents the URL to locate the release of the package and is required. 'executable' is a string that represents the name of the executable and is required. 'args' is a string that represents the arguments to the executable and is required. No additional properties are allowed at the root level.
