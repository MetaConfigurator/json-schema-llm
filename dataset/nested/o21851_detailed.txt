The schema describes the configuration for an API endpoint. It is an object type. The main properties of the schema are: 'AuthorizationType' is a string that is required and must be one of the following values: 'AWS_IAM', 'NONE'. 'HttpMethod' is a string that is required and must be one of the following values: 'DELETE', 'GET', 'HEAD', 'OPTIONS', 'PATCH', 'POST', 'PUT'. 'ResourceId' is a string that is required and must match the pattern '^[a-z0-9]+$'. 'RestApiId' is a string that is required and must match the pattern '^[a-z0-9]{10}$'. 'ApiKeyRequired' is a string that is optional and must be one of the following values: 'true', 'false'. 'RequestModels' is an object that is optional and does not allow additional properties. 'RequestParameters' is an object that is optional and does not allow additional properties. No additional properties are allowed at the root level.
