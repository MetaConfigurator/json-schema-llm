The schema describes an array of objects related to physics relations. It is an array type. The main properties of the schema are: 'physics_objects' is a string that represents available physics objects. 'angle_relation' is a string that matches the pattern '^[\d\.]+$', representing degrees. 'transverse_mass' is an object that contains properties: 'upper_range', a string matching the pattern '^[\d\.]+$', and 'lower_range', a string matching the pattern '^[\d\.]+$'. 'charge_relation' is a string with an enum of allowed values: 'same_sign', 'opposite_sign', representing the charge type. 'relation_type' is a string with an enum of allowed values: 'charge', 'angle', 'invariant_mass', 'transverse_mass', indicating the type of relation. 'invariant_mass' is an object containing properties: 'upper_range', a string matching the pattern '^[\d\.]+$', and 'lower_range', which is a number. No additional properties are allowed at the root level.
