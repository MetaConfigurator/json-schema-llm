The schema describes an album product. It is an object type. The main properties of the schema are: 'artist' is a string required field, 'currency' is a string or null value that must be exactly 3 characters long if it is a string, 'description' is a string required field, 'format' is a string required field, 'imageUrls' is an array of strings that must be valid URIs, 'name' is a string required field, 'nameYourPrice' is a boolean required field, 'offerMore' is a boolean required field, 'priceInCents' is an integer or null that must be at least 0 if it is an integer, 'soldOut' is a boolean required field, 'url' is a string that must be a valid URI required field. No additional properties are allowed at the root level.
