The schema describes a request for stopping a transaction. It is an object type. The main properties of the schema are: 'idTag' is a string with a maximum length of 20, optional. 'meterStop' is an integer, required. 'reason' is a string with an enum of allowed values: 'EmergencyStop', 'EVDisconnected', 'HardReset', 'Local', 'Other', 'PowerLoss', 'Reboot', 'Remote', 'SoftReset', 'UnlockCommand', 'DeAuthorized', required. 'timestamp' is a string formatted as date-time, required. 'transactionData' is an array of objects, where each object contains a required 'timestamp' (formatted as date-time) and a required 'sampledValue' (an array of objects). Each 'sampledValue' object has required property 'value' (a string). Other properties in the 'sampledValue' object are optional and include 'context' (enum values: 'Interruption.Begin', 'Interruption.End', 'Sample.Clock', 'Sample.Periodic', 'Transaction.Begin', 'Transaction.End', 'Trigger', 'Other'), 'format' (enum values: 'Raw', 'SignedData'), 'location' (enum values: 'Cable', 'EV', 'Inlet', 'Outlet', 'Body'), 'measurand' (enum values: 'Energy.Active.Export.Register', 'Energy.Active.Import.Register', 'Energy.Reactive.Export.Register', 'Energy.Reactive.Import.Register', 'Energy.Active.Export.Interval', 'Energy.Active.Import.Interval', 'Energy.Reactive.Export.Interval', 'Energy.Reactive.Import.Interval', 'Power.Active.Export', 'Power.Active.Import', 'Power.Offered', 'Power.Reactive.Export', 'Power.Reactive.Import', 'Power.Factor', 'Current.Import', 'Current.Export', 'Current.Offered', 'Voltage', 'Frequency', 'Temperature', 'SoC', 'RPM'), 'phase' (enum values: 'L1', 'L2', 'L3', 'N', 'L1-N', 'L2-N', 'L3-N', 'L1-L2', 'L2-L3', 'L3-L1'), and 'unit' (enum values: 'Wh', 'kWh', 'varh', 'kvarh', 'W', 'kW', 'VA', 'kVA', 'var', 'kvar', 'A', 'V', 'K', 'Celcius', 'Fahrenheit', 'Percent'). No additional properties are allowed at the root level.
