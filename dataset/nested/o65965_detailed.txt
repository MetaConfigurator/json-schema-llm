The schema describes a PayPal item JSON. It is an object type. The main properties of the schema are: 'quantity' is a string representing the number of a particular item, with a maximum length of 10 characters, and it is required. 'name' is a string for the item name, with a maximum length of 127 characters, and it is required. 'price' is a string indicating the item cost, with a maximum length of 10 characters, and it is required. 'currency' is a string representing a 3-letter currency code, with a maximum length of 3 characters, and it is required. 'sku' is a string for the stock keeping unit corresponding to the item, with a maximum length of 50 characters, and it is optional. 'description' is a string with a maximum length of 127 characters and is optional. 'tax' is a string and is optional. No additional properties are allowed at the root level.
