The schema describes options for a MultipleChoice question. It is an object type. The main properties of the schema are: 'text' is a string that describes the question HTML text that comes before the options. It is required. 'correctAnswers' is an array containing strings, representing a list of correct answers to the question. Each item is an HTML string. It is required. 'incorrectAnswers' is an array containing strings, representing a list of incorrect answers to the question. Each item is an HTML string. It is required. 'numberAnswers' is an integer that indicates the total number of answers in the list of possible answers. It must be at least 1. 'minCorrectAnswers' is an integer that specifies the minimum number of correct answers in the list. It must be at least 0. 'maxCorrectAnswers' is an integer that specifies the maximum number of correct answers in the list. It must be at least 0. No additional properties are allowed at the root level.
