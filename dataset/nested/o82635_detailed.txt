The schema describes an article block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string that must be in date-time format, representing an update that occurred on the node this block is attached to. It is optional. 'aside' is a boolean that, when true, indicates that this block represents a portion of a document whose content is only indirectly related to the main content. It is optional. 'node_ref' is a string that must match the pattern '^[\w\/\.:-]+$'. It is required. 'title' is a string that has a minimum length of 0 and a maximum length of 255, serving as an optional override for the title of the node. 'link_text' is a string that has a minimum length of 0 and a maximum length of 255. It is optional. 'show_image' is a boolean with a default value of true. It is optional. 'image_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and is an optional reference to an image asset to use as the poster. No additional properties are allowed at the root level.
