The schema describes adding publications. It is an object type. The main properties of the schema are: 'journal_title' is a string representing the title of the journal, optional. 'persistent_identifiers' is an array of strings representing identifiers, optional. 'journal_volume' is an integer representing the journal volume, optional. 'journal_year' is an integer representing the journal year, optional. 'journal_issue' is an integer representing the journal issue, optional. 'journal_page' is an integer representing the journal page, optional. No additional properties are allowed at the root level.
