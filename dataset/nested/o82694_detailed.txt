The schema describes a SoundCloud audio block. It is an object type. The main properties of the schema are: 'aside' is a boolean optional property indicating whether the block represents a portion of a document that is only indirectly related to the main content. 'auto_play' is a boolean optional property determining whether the audio should play automatically. 'css_class' is a string optional property that can hold a value conforming to the pattern '^[\w\s-]+$', which can be appended to the DOM elements' class attribute. 'etag' is a string optional property that must match the pattern '^[\w\.:-]+$'. 'hide_related' is a boolean optional property indicating whether related content should be hidden. 'poster_image_ref' is a string optional property containing a reference to an image asset for the poster, conforming to the pattern '^[\w\/\.:-]+$'. 'show_comments' is a boolean optional property specifying whether comments should be displayed. 'track_id' is a string required property that must match the pattern '^\d+$'. 'updated_date' is a string optional property in date-time format, representing the last update date of the node this block is attached to. 'visual' is a boolean optional property indicating whether to use the thumbnail as a poster image. No additional properties are allowed at the root level.
