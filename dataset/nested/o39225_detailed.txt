The schema describes a collection of products. It is an array type. The main properties of the schema are: Each item in the array is a Product object. 'id' is a number, representing the unique identifier for a product and is required. 'name' is a string and is required. 'price' is a number, must be greater than 0, and is required. 'tags' is an array of strings, must contain at least 1 item, and must have unique items. 'dimensions' is an object that requires 'length', 'width', and 'height', each of which is a number. 'warehouseLocation' is an optional field providing coordinates of the warehouse with the product. No additional properties are allowed at the root level.
