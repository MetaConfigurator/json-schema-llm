The schema describes a format used to define structures visible on town screen in VCMI. It is an object type. The main properties of the schema are: 'animation' is a string that represents the main animation file for this building and is required. 'area' is a string that indicates the area when the building is selected and must be an 8-bit image, but is optional. 'border' is a string that represents the golden border around the building displayed when selected, but is optional. 'builds' is a string that has no description, but is optional. 'hidden' is a boolean indicating if the upgrade will replace parent animation without altering its behavior, but is optional. 'x' is a number that specifies the position on screen and is required. 'y' is a number that specifies the position on screen and is required. 'z' is a number that specifies the position on screen, with higher values drawn on top of other buildings, but is optional. No additional properties are allowed at the root level.
