The schema describes a structure for URL shortening and lookup requests. It is an object type. The main properties of the schema are: 'shorten' is an object containing properties related to URL shortening. It includes an optional property 'action', which is a string with a description of an action to request lookup/insert of a URL (data.url), and an optional property 'resource', which is a string describing the insert result for a URL (url, key, shortUrl). 'lookup' is an object containing properties related to URL lookup. It includes an optional property 'action', which is a string describing an action to request lookup of a URL by key (data.key), and an optional property 'resource', which is a string describing the lookup result for a URL (url, key, shortUrl). No additional properties are allowed at the root level.
