The schema describes a collection of plates and their configurations. It is an object type. The main properties of the schema are: 'plates' is an array containing objects, each of which represents a plate with required fields: 'type', 'label', 'descriptor', 'rows', 'columns', 'size', 'wellsets', and 'wells'. The 'wellsets' property is an array of objects, where each object has required fields: 'label', 'size', and 'wells', with 'wells' being an array of strings. The 'wells' property is also an array of objects, requiring 'index' and 'values' properties, with 'values' being an array of numbers. No additional properties are allowed at the root level.
