The schema describes the structure of a financial data object. It is an object type. The main properties of the schema are: 'Status' is a string optional. 'Name' is a string required. 'Symbol' is a string required. 'LastPrice' is a number required. 'Change' is a number required. 'ChangePercent' is a number required. 'Timestamp' is a string required. 'MSDate' is a number required. 'MarketCap' is an object required. 'Volume' is an integer required. 'ChangeYTD' is a number required. 'ChangePercentYTD' is a number required. 'High' is an integer required. 'Low' is a number required. 'Open' is a number required. No additional properties are allowed at the root level.
