The schema describes a shader build-chain definition. It is an object type. It defines several reusable components under 'definitions': 'Shader' is an object, a shader to compile. It requires 'InputFilePath', 'EntryPointName', and 'ShaderType'. The properties include 'InputFilePath', a string; 'OutputFilePath', a string; 'EntryPointName', a string; 'ShaderType', which has an enum of values: 'vertex', 'pixel', 'geometry', 'hull', 'domain', and 'compute'; 'ShaderModel', which has an enum of values: '2.0', '3.0', '4.0', '4.1', and '5.0'; 'AdditionalIncludes', an array of strings; and 'Pre-ProcessorDefines', an array of strings. The main properties of the schema are: 'FXCPath' is a string, required; 'AdditionalIncludeDirectories' is an optional array of strings; 'ShadersToCompile' is an array of 'Shader' objects, required. No additional properties are allowed at the root level.
