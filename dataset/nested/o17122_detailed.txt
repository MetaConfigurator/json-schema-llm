The schema describes a Config Object for verbshaker. It is an object type. The main properties of the schema are: 'port' is an integer The Port the Server is listening to. 'proverbs' is a string The Server-Path to the Proverbs. 'show_docs' is a boolean Allow path /docs. 'default' is an object Default Values, with properties: 'backgroundcolor' is a string Color of the Background as (HEX). 'fontcolor' is a string Default Color of the Font (HEX). 'fontsize' is an integer Default Size of the Font (in vmin). 'fontfamily' is a string Default Family of the Font. 'hiddenNav' is a boolean Is the Navbar Hidden. 'proverbbutton' is a boolean Is the proverbbutton shown. 'autoupdate' is an integer AutoUpdate Value for Proverbs. 'shareButton' is a boolean Is the ShareButton shown. 'unhideNavButton' is a boolean Is the Button to show the Navbar shown. 'introjs' is a boolean Use intro.js to show new users around. 'changeable' is an object Defines what the user may edit, with properties: 'language' is a boolean Change the language. 'backgroundcolor' is a boolean Change the Color of the Background. 'fontcolor' is a boolean Change the Color of the Font. 'fontsize' is a boolean Change the Size of the Font. 'fontfamily' is a boolean Change the Family of the Font. 'hideNav' is a boolean Allow to Hide the Navbar. 'proverbbutton' is a boolean Toggle proverb-Button. 'autoupdate' is a boolean Change the Autoupdate Value. No additional properties are allowed at the root level.
