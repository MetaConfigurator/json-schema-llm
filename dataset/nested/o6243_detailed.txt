The schema describes a command configuration. It is an object type. The main properties of the schema are: 'nameTemplate' is a string, required, with a minimum length of 1. 'payloadTemplate' is a string, optional. 'sendToDeviceTags' is an array of objects, optional, with a minimum of 1 item, where each item must have properties 'key' and 'value', both of which are strings. 'sendToDeviceIds' is an array of strings, optional, with a minimum of 1 item, each string having a minimum length of 1. The schema requires 'nameTemplate' to be present. Additionally, at least one of 'sendToDeviceIds' or 'sendToDeviceTags' must be provided. No additional properties are allowed at the root level.
