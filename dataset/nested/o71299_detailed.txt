The schema describes a configuration for interface filters. It is an object type. The main properties of the schema are: 'interface_filters' is an array of filter objects, optional, with a default value of an empty array. Each item in the array is an object that has a maximum of 2 properties, with no additional properties allowed. The 'op' property is an enum that can be either 'regex' or 'startwith'. The 'expr' property is a string. No additional properties are allowed at the root level.
