The schema describes a configuration profile for identification. It is an object type. The main properties of the schema are: 'AuthMethod' is a string that allows the user to enter the password upon installation instead of being supplied in the payload. It has a default value of 'UserEnteredPassword' and can be either 'Password' or 'UserEnteredPassword'. 'EmailAddress' is a string that represents the address for the accounts. 'FullName' is a string that provides the full name of the designated accounts. 'Password' is a string where you can provide the password or choose to have the user provide it when he or she installs the profile. 'PayloadDescription' is a string that describes the payload with a default value of 'Identification Payload'. 'PayloadDisplayName' is a string that gives the name of the payload, also with a default value of 'Identification Payload'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, with a default value of 'com.apple.configurationprofile.identification'. 'PayloadOrganization' is a string describing the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string that indicates the type of the payload with a default value of 'com.apple.configurationprofile.identification'. 'PayloadUUID' is a string that serves as a unique identifier for the payload, following a specific pattern and with a default value of ''. 'PayloadVersion' is a number representing the version of the whole configuration profile, with a default value of 1. 'Prompt' is a string for short prompt text. 'PromptMessage' is a string for additional prompt description. 'UserName' is a string that provides the UNIX user name for the accounts. No additional properties are allowed at the root level.
