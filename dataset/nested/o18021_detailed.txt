The schema describes a single entry in a CV. It is an object type. The main properties of the schema are: '@context' is a string which must be the constant value 'http://schema.org'. '@type' is a string which must be the constant value 'Event'. 'cat' is a string that is optional. 'description' is a string that is optional. 'endDate' is a string which must follow ISO8601 format and is optional. 'endLabel' is a string that is optional. 'location' is a string that defines an object with properties '@type', 'coord', 'name', and 'url'. '@type' in location must be the constant value 'Place'. 'coord' is a string that must follow ISO6709 format and is optional. 'name' is a string that is optional. 'url' is a string that must follow URI format and is optional. 'name' is a string that is optional. 'startDate' is a string which must follow ISO8601 format and is optional. 'startLabel' is a string that is optional. 'uri' is a string that must follow URI format and is optional. No additional properties are allowed at the root level.
