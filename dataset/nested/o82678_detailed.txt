The schema describes a page break block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$', which provides a unique identifier. 'css_class' is a string that must match the pattern '^[\w\s-]+$', and can be appended to the DOM elements' class attribute in rendering environments that support HTML. 'updated_date' is a string formatted as date-time, representing an update that occurred on the node this block is attached to. 'aside' is a boolean that indicates whether the block represents a portion of a document whose content is only indirectly related to the document's main content, often presented as sidebars or call-out boxes. 'read_more_text' is a string with a minimum length of 0 and a maximum length of 255. No additional properties are allowed at the root level.
