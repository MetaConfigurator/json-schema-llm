The schema describes O3D3xx Dimensioning model configuration. It is an object type. The main properties of the schema are: 'Name' is a string which is required. 'PrivateType' is a string with a default value of 'Dimensioning'. 'Type' is a string which is required and has an enum value of 'Dimensioning'. 'FlowAngle' is a number which is required. 'UseFlowAngle' is a boolean which is required. 'BackgroundTaught' is a boolean marked as readonly. 'ROIs' is a string marked as readonly. 'ObjectType' is an integer that is required, with a minimum value of 0 and a maximum value of 1. 'TeachData' is a string marked as private and noexport. 'TeachROI' is a string marked as private and noexport. No additional properties are allowed at the root level.
