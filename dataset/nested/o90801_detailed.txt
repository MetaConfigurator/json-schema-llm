The schema describes the configuration for OpenLDAP Admin and DC/OS service. It is an object type. The main properties of the schema are: 'openldap-admin' is an object containing configuration properties for the OpenLDAP Admin service. It is required and includes: 'cpus' is a number representing CPU shares to allocate to each OpenLDAP node, with a default value of 0.1 and a minimum value of 0.1; 'mem' is a number representing memory to allocate to each OpenLDAP node, with a default value of 128.0 and a minimum value of 128.0. 'service' is an object containing configuration properties for the DC/OS service. It is optional and includes: 'name' is a string representing the name of the OpenLDAP Admin service instance, with a default value of 'openldap-admin'. No additional properties are allowed at the root level.
