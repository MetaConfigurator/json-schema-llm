The schema describes configuration settings. It is an object type. The main properties of the schema are: 'localFiles' is an object representing local file configurations, required; it includes 'configurationPath' and 'historyPath', both required strings. 'sync' is an object representing synchronization settings, required; it includes 'willSync', a required boolean with a default value of false; if 'willSync' is true, 'serverUrl' and 'key' become required strings with default values of empty. 'rawSources' is an array of objects representing data source configurations, required; each object includes 'name' (a required string), 'disabled' (a required boolean with a default of false), 'url' (a required string), and 'isMilestone' (a required boolean). Additionally, if 'isMilestone' is false, 'selector', 'getItem', and 'limit' are required properties. No additional properties are allowed at the root level.
