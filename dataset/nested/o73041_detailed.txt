The schema describes the structure of a request to include an order. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'codigoModelo' is a string that represents the model code. 'importadorExportador' is a string that identifies the importer or exporter. 'declarante' is a string that denotes the declarant. 'informacaoAdicional' is a string or null that provides additional information. 'listaNcm' is an array of objects that contains details about NCM (Mercosur Common Nomenclature) items. Each item has a 'ncm' property which is a string, 'listaCamposNcm', an array of objects with 'codigo' as a string and 'listaValor' as an array of strings, and 'listaAtributosNcm', an array of objects with the same structure. It also includes 'unidadeMedidaComercializada', which is a string or null representing the traded unit of measure. 'listaCamposFormulario' is an array of objects with 'codigo' as a string and 'listaValor' as an array of strings. 'unidadeMedidaComercializada' is a string or null representing the traded unit of measure. No additional properties are allowed at the root level.
