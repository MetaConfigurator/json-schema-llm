The schema describes an API Gateway integration configuration. It is an object type. The main properties of the schema are: 'Type' is a string that is required and must be one of the enum values: 'HTTP', 'AWS', 'MOCK'. 'HttpMethod' is a string that is required and must be one of the enum values: 'DELETE', 'GET', 'HEAD', 'OPTIONS', 'PATCH', 'POST', 'PUT'. 'ResourceId' is a string that is required and must match the pattern '^[a-z0-9]+$'. 'RestApiId' is a string that is required and must match the pattern '^[a-z0-9]{10}$'. 'CacheKeyParameters' is an optional array of strings. 'CacheNamespace' is an optional string. 'Credentials' is an optional string. 'IntegrationHttpMethod' is an optional string. 'RequestTemplates' is an optional object that does not allow additional properties but may contain properties matching the pattern '.+: string'. 'RequestParameters' is an optional object that does not allow additional properties but may contain properties matching the pattern '^integration\.request\.(querystring|path|header)\..+$' with string values that match the pattern '^(method\.request\.(querystring|header|path)\..+)|('.*')$'. 'Uri' is an optional string. No additional properties are allowed at the root level.
