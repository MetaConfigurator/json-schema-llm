The schema describes a structure containing search and location information. It is an object type. The main properties of the schema are: 'search' is an object that is required and contains a required property 'resource' which is a string formatted as 'topic' with an axRole of 'inlet'. 'locations' is an object that is required and contains a required property 'resource' which is a string formatted as 'topic' with an axRole of 'outlet', and an optional property 'searching' which is a string formatted as 'topic' with an axRole of 'outlet', axPattern of 'flag', and a description indicating that geocoding using the search term is still in progress. No additional properties are allowed at the root level.
