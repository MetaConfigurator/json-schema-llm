The schema describes a configuration for assets and images. It is an object type. The main properties of the schema are: 'assets' is an object containing 'container' and 'uris', both of which have additional properties restricted to strings and no other properties allowed. 'images' is an object with properties including 'icon-large', 'icon-medium', 'icon-small', each of which are strings with descriptions for their intended usage, and 'screenshots', which is an array of strings, each describing PNG screen URLs. No additional properties are allowed at the root level.
