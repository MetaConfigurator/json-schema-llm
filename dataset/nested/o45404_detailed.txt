The schema describes library options configuration. It is an object type. The main properties of the schema are: 'entryFile' is a string The path to create the library's public API file. It is optional and has a default value of 'public_api'. 'name' is a string The name of the library. It is optional and derives its default value from the first argument ('argv') provided in the command line. 'prefix' is a string The prefix to apply to generated selectors. It is optional and has a default value of 'lib'. 'skipPackageJson' is a boolean Do not add dependencies to package.json. It is optional and has a default value of false. 'skipTsConfig' is a boolean Do not update tsconfig.json for development experience. It is optional and has a default value of false. No additional properties are allowed at the root level.
