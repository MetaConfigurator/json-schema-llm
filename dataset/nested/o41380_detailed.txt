The schema describes an invocation manifest for AFNI: Brain Warp. It is an object type. The main properties of the schema are: 'inputs' is an object that contains required properties 'warp_target', 'native', and 'standard'. 'warp_target' is an object with a required property 'type' that must be equal to 'nifti'. 'standard' is an object with a required property 'type' that must be equal to 'nifti'. 'native' is an object with a required property 'type' that must be equal to 'nifti'. 'config' is an object that does not have any defined properties. No additional properties are allowed at the root level.
