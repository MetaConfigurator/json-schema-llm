The schema describes the invocation manifest for a Task TSV Converter. It is an object type. The main properties of the schema are: 'config' is an object containing properties specific to the configuration, which is required. 'inputs' is an object representing input data, which is required. Within 'inputs', 'file' is an object that contains a property, 'type', which is a string that can be one of the following values: 'text', 'tabular data'. Additionally, 'LogConfig' is an object with required properties: 'base' (a string), 'found' (a boolean), and 'value' (an unspecified type). No additional properties are allowed at the root level.
