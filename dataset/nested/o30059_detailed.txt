The schema describes a configuration for audit syslog actions. It is an object type. The main properties of the schema are: 'acl' is a string that can be one of 'ENABLED', 'DISABLED'. 'alg' is a string that can be one of 'ENABLED', 'DISABLED'. 'appflowexport' is a string that can be one of 'ENABLED', 'DISABLED'. 'builtin' is an object and is readonly. 'contentinspectionlog' is a string that can be one of 'ENABLED', 'DISABLED'. 'dateformat' is a string that can be one of 'MMDDYYYY', 'DDMMYYYY', 'YYYYMMDD'. 'dns' is a string that can be one of 'ENABLED', 'DISABLED'. 'domainresolvenow' is a boolean. 'domainresolveretry' is an integer. 'feature' is a string that can be one of multiple predefined features. 'ip' is a string and is readonly. 'lbvservername' is a string. 'logfacility' is a string that can be one of 'LOCAL0', 'LOCAL1', 'LOCAL2', 'LOCAL3', 'LOCAL4', 'LOCAL5', 'LOCAL6', 'LOCAL7'. 'loglevel' is an object. 'lsn' is a string that can be one of 'ENABLED', 'DISABLED'. 'maxlogdatasizetohold' is an integer. 'name' is a string. 'netprofile' is a string. 'serverdomainname' is a string. 'serverip' is a string. 'serverport' is an integer. 'sslinterception' is a string that can be one of 'ENABLED', 'DISABLED'. 'subscriberlog' is a string that can be one of 'ENABLED', 'DISABLED'. 'tcp' is a string that can be one of 'NONE', 'ALL'. 'tcpprofilename' is a string. 'timezone' is a string that can be one of 'GMT_TIME', 'LOCAL_TIME'. 'transport' is a string that can be one of 'TCP', 'UDP'. 'urlfiltering' is a string that can be one of 'ENABLED', 'DISABLED'. 'userdefinedauditlog' is a string that can be one of 'YES', 'NO'. No additional properties are allowed at the root level.
