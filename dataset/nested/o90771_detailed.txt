The schema describes configuration properties for a Minio service on DC/OS. It is an object type. The main properties of the schema are: 'networking' is an object with a description of Networking-related configuration properties for Minio on DC/OS. It is required. It contains properties: 'https-redirect' is a boolean with a default value of false, describing whether Marathon-lb should redirect HTTP traffic to HTTPS, and 'public-agent' is a string describing the IP address of the public agent configured for integration with Marathon-lb; it is required. 'service' is an object with a description of Minio DC/OS service properties. It is required. It contains properties: 'cpus' is a number with a default value of 1 and a minimum value of 0.5, describing CPU shares to allocate to each service instance, 'mem' is a number with a default value of 2048 and a minimum value of 2048, describing memory to allocate to each service instance, and 'name' is a string with a default value of 'minio', describing the name of this service instance; all three are required. 'storage' is an object with a description of Storage configuration for Minio's application and config on DC/OS. It is required. It contains properties: 'persistent-volume' is an object with a description of Persistent volumes configuration (local volumes), which are created on the machine with the specified hostname, and it contains the property 'host-volume', which is a string with a default value of 'export', describing the location of a volume on the host to be used for persisting Minio configuration and data; it is required. 'pinned-hostname' is a string, describing an optional DC/OS agent hostname to run this Minio instance on; it is optional. No additional properties are allowed at the root level.
