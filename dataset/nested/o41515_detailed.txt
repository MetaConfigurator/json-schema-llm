The schema describes an invocation manifest for a DICOM conversion tool. It is an object type. The main properties of the schema are: 'config' is an object that contains properties required for the conversion process, which includes 'convert_montage' (a boolean, required, defaults to true), 'convert_nifti' (a boolean, required, defaults to true), 'convert_png' (a boolean, required, defaults to false), 'convert_mux' (a boolean, required, defaults to false), and 'output_name' (a string, optional). 'inputs' is an object that contains properties related to the input data, which includes 'dicom' (an object, required, containing 'type' which is an enum with allowed value 'dicom'). The schema requires both 'config' and 'inputs'. No additional properties are allowed at the root level.
