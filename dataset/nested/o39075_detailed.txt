The schema describes configuration settings for a cloud environment. It is an object type. The main properties of the schema are: 'auth_type' is a string representing the name of the authentication plugin to be used, it is required and has a default value of 'password'. 'disable_vendor_agent' is an object that contains properties required to disable vendor agent, it is required and has no properties defined. 'floating_ip_source' is a string which indicates which service provides Floating IPs, it is required and has an enum of allowed values: 'neutron', 'nova', 'None', with a default value of 'neutron'. 'image_api_use_tasks' is a boolean indicating if the cloud requires the Image Task API, it is required and has a default value of false. 'image_format' is a string for the format of uploaded Images, it is required and has a default value of 'qcow2'. 'interface' is a string indicating which API Interface should connections hit, it is required and has an enum of allowed values: 'public', 'internal', 'admin', with a default value of 'public'. 'secgroup_source' is a string for which service provides security groups, it is required and has an enum of allowed values: 'neutron', 'nova', 'None', with a default value of 'neutron'. 'baremetal_api_version' is a string representing the Baremetal API Service Type, it is required and has a default value of '1'. 'compute_api_version' is a string representing the Compute API Version, it is required and has a default value of '2'. 'database_api_version' is a string representing the Database API Version, it is required and has a default value of '1.0'. 'dns_api_version' is a string representing the DNS API Version, it is required and has a default value of '2'. 'identity_api_version' is a string representing the Identity API Version, it is required and has a default value of '2'. 'image_api_version' is a string representing the Image API Version, it is required and has a default value of '1'. 'network_api_version' is a string representing the Network API Version, it is required and has a default value of '2'. 'object_store_api_version' is a string representing the Object Storage API Version, it is required and has a default value of '1'. 'volume_api_version' is a string representing the Volume API Version, it is required and has a default value of '2'. No additional properties are allowed at the root level.
