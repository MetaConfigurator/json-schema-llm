The schema describes Angular application options. It is an object type. The main properties of the schema are: 'appRoot' is a string optional; 'changeDetection' is a string optional, defaulting to 'Default', with allowed values: 'Default', 'OnPush'; 'export' is a boolean optional, defaulting to false; 'flat' is a boolean optional, defaulting to false; 'htmlTemplate' is a string optional; 'inlineStyle' is a boolean optional, defaulting to false; 'inlineTemplate' is a boolean optional, defaulting to false; 'module' is a string optional; 'name' is a string required; 'path' is a string optional, defaulting to 'app'; 'prefix' is a string optional, defaulting to 'app'; 'routing' is a boolean optional, defaulting to false; 'selector' is a string optional; 'skipImport' is a boolean optional, defaulting to false; 'sourceDir' is a string optional, defaulting to 'src'; 'spec' is a boolean optional, defaulting to true; 'styleext' is a string optional, defaulting to 'css'; 'viewEncapsulation' is a string optional, defaulting to 'None', with allowed values: 'Emulated', 'Native', 'None'. No additional properties are allowed at the root level.
