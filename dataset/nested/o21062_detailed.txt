The schema describes a detailed specification for user research participants responses in the context of Digital Outcomes and Specialists 4. It is an object type. The main properties of the schema are: 'availability' is a string that must have a minimum length of 1 and a maximum length of 100, and is required. 'essentialRequirements' is an array that must contain at least one item, where each item is an object with a required 'evidence' string property that must have a minimum length of 1 and match the pattern '^(?:\S+\s+){0,99}\S+$'. 'essentialRequirementsMet' is a boolean that must be true and is required. 'respondToEmailAddress' is a string that must have a minimum length of 1 and is formatted as an email, and is required. No additional properties are allowed at the root level.
