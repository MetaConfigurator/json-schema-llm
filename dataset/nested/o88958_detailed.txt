The schema describes Microsoft.ServerManagement Resource Types. It is an object type. The root schema resolves to the referenced definition. The referenced definition is an object that defines resource types for server management tools, including gateways and nodes. It contains properties for definitions of a gateway and a node, each with their own required and optional characteristics. The main properties of the schema are: 'resourceDefinitions.gateway' is a object referencing the definition for a gateway resource, which requires 'location', 'name', and 'properties'. 'resourceDefinitions.node' is a object referencing the definition for a node resource, which also requires 'name', 'location', and 'properties'. Compositional constraints state that both 'resourceDefinitions.gateway' and 'resourceDefinitions.node' are required fields. No additional properties are allowed at the root level.
