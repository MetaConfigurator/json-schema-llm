The schema describes a set of lobby servers. It is an array type. The main properties of the schema are: Each item in the array is an object representing a lobby server. "version" is a string representing the lobby server engine version, required. It must match the pattern '^\d+(\.\d+){1,3}$'. "host" is a string representing the lobby server host, required. It must conform to one of the following formats: host-name, ipv4, or ipv6. "port" is a number representing the TCP port on which the lobby server accepts connections, required. It must be between 0 and 65535. "https_port" is a number representing the HTTPS lobby server port, required. It must be between 0 and 65535. "message" is either a string or null, representing the message displayed when a user connects to an available lobby server, required. "error_message" is either a string or null, representing the message displayed when a user connects to an unavailable lobby server, required. No additional properties are allowed at the root level.
