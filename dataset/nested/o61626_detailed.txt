The schema describes a firewall configuration. It is an object type. The main properties of the schema are: 'Applications' is an array Optional. The list of applications, which contains items that are of type object representing a firewall application item. Each item includes 'Allowed', which is a boolean that specifies whether or not incoming connections are allowed, and 'BundleID', which is a string that identifies the application. 'BlockAllIncoming' is a boolean Optional. Corresponds to the 'Block all incoming connections' option. 'EnableFirewall' is a boolean Required. Whether the firewall should be enabled or not. 'EnableStealthMode' is a boolean Optional. Corresponds to 'Enable stealth mode.' 'PayloadDescription' is a string with a default value of 'Firewall Payload'. Description of the payload. 'PayloadDisplayName' is a string with a default value of 'Firewall Payload'. Name of the payload. 'PayloadIdentifier' is a string with a default value of 'com.apple.security.firewall'. A unique identifier for the payload, dot-delimited. Usually root PayloadIdentifier+subidentifier. 'PayloadOrganization' is a string. This value describes the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string with a default value of 'com.apple.security.firewall'. The type of the payload, a reverse dns string. 'PayloadUUID' is a string with a default value of ''. Unique identifier for the payload (format 01234567-89AB-CDEF-0123-456789ABCDEF) and it must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is a number with a default value of 1. The version of the whole configuration profile. No additional properties are allowed at the root level.
