The schema describes an array of objects. It is an array type. The main properties of the schema are: Each object in the array must have the following properties: 'id' is a string that is required. 'name' is a string that is required. 'description' is a string that is required. 'type' is a string that is required. 'stats' is an object that is required, containing the following properties: 'offense' is a number. 'defense' is a number. 'speed' is a number. 'overlay' is a string that is required. 'underlay' is a string that is required. 'skins' is an array that is required, containing objects which must have the following properties: 'id' is a number that is required. 'name' is a string that is required. 'icon' is a string that is required. 'default' is a boolean that is required. 'unlock_items' is an array of numbers that is required. No additional properties are allowed at the root level.
