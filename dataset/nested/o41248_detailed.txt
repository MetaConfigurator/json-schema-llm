The schema describes the invocation manifest for fMRIPREP: A Robust Preprocessing Pipeline for fMRI Data. It is an object type. The main properties of the schema are: 'inputs' is an object that is required and contains the property 'api_key', which is also an object with no defined properties. 'config' is an object that is required and contains the properties 'save_outputs' and 'freesurfer'. 'save_outputs' is a boolean with a default value of false. 'freesurfer' is a boolean with a default value of true. No additional properties are allowed at the root level.
