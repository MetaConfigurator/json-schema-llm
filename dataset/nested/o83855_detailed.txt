The schema describes a specification for a VecNet Experiment. It is an object type. The main properties of the schema are: 'name' is a string, user-defined name of the Experiment. 'description' is a string, user-defined description of the Experiment. 'base' is a string, Experiment Template (Experiment Base). Template or boilerplate for the scenario xml files. Includes placeholders that will be replaced with actual parameter values. 'basefile' is a string, Filename that will be used as Experiment Template. Can't be used together with base. 'sweeps' is an object, Sweeps with a property that itself is an object containing a parameterized arm structure. 'combinations' is an array or object defining combinations of sweeps with specific structural rules. It can either be an array of arrays representing all combinations of sweeps or an object that captures the cross-product of arrays as a list of scenarios. No additional properties are allowed at the root level.
