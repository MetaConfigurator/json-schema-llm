The schema describes getting weight details about given users. It is an object type. The main properties of the schema are: 'aggregation' is an object containing aggregation data about the weights, which is required and has required fields 'maxWeight', 'minWeight', 'avgWeight', and 'weightChange'. 'weights' is an array of objects, which is required and contains items that are objects also required to have the fields 'datetime' and 'weight'. No additional properties are allowed at the root level.
