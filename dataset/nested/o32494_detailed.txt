The schema describes configuration options for a blog. It is an object type. The main properties of the schema are: 'id' is a string The options' id and is required. 'authorizeComment' is a boolean Comments authorized and is optional. 'authorizeAnonymousComment' is a boolean Anonymous comments authorized and is optional. 'postPerPage' is an integer Post number per page and is optional. 'autoPublishPost' is a boolean Automatic publication of posts and is optional. 'commentModerationMode' is an integer Moderation mode option and is optional. 'displayPostViewCounter' is a boolean Display post views number and is optional. 'tagCloud' is an integer Tag cloud presentation mode and is optional. 'tagTopMode' is a boolean Display top tags only and is optional. 'maxtTag' is an integer Tag number to display and is optional. 'displayFullPosts' is a boolean Display full posts in view list and is optional. 'infos' is a string Information toolbar content and is optional. 'widgetOrder' is an object A collection of widget properties which includes 'nameTemplate' (a string describing widget template name), 'visibility' (a boolean describing visibility status), and 'id' (a string describing widget id). No additional properties are allowed at the root level.
