The schema describes an EC2 manifest. It is an object type. The main properties of the schema are: 'virtualization' is a string, optional, with allowed values 'pvm', 'hvm'. 'image' is an object, required. It has a property 'description' which is a string, optional. 'credentials' is an object, optional, with properties 'access-key' (a string, optional) and 'secret-key' (a string, optional). 'system' is an object, optional, with a property 'bootloader' which is a string, optional, with allowed values 'pvgrub', 'extlinux'. 'volume' is an object, optional, with a required property 'backing' which is a string, with allowed values 'ebs', 's3'. It also has an optional property 'partitions', which is an object containing a property 'type' which is a string, optional, with allowed values 'none', 'msdos'. No additional properties are allowed at the root level.
