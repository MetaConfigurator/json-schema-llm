The schema describes a form for user inputs that can be created using TheDistanceForms library for iOS. It is an object type. The main properties of the schema are: 'title' is a string that specifies the title for this form. It is required. 'questions' is an array that contains the questions that make up this form. It is required and must have at least one item. Each item in the array is an object representing a question entry in the form, which has the following properties: 'key' is a string that uniquely identifies the question in the form. It is required. 'question_type' is a string representing the type of the question, and it must be one of the following values: 'TextSingle', 'TextMultiline', 'Date', 'Time', 'DateTime', 'ChoiceDropdown', 'ChoiceSegments', 'Boolean', 'Button'. It is required. 'title' is a string that displays the title for this question, applicable only for the question types 'ChoiceSegments', 'Boolean', and 'Button', and is optional. 'subtitle' is a string that is optional. 'prompt' is a string that is optional. 'keyboard_type' is a string that may be one of the following values: 'Default', 'ASCIICapable', 'NumbersAndPunctuation', 'URL', 'NumberPad', 'PhonePad', 'NamePhonePad', 'EmailAddress', 'DecimalPad', 'Twitter', 'WebSearch', and is optional. 'capitalization' is a string that may be one of the following values: 'None', 'Words', 'Sentences', 'AllCharacters', and is optional. 'auto_correction' is a boolean that is optional. 'secure_text_entry' is a boolean that is optional. 'date_format_style' is a string that may be one of: 'Short', 'Medium', 'Long', 'Full', and is optional. 'time_formate_style' is a string that may be one of: 'Short', 'Medium', 'Long', 'Full', and is optional. 'minute_interval' is a number that may be one of the following values: 1, 2, 3, 4, 5, 10, 12, 15, 20, 30, and is optional. 'choices' is an array of strings that specifies the values the user can choose from in a 'ChoiceDropdown' or 'ChoiceSegments' question. It must have at least one item and cannot contain duplicates. 'default' is a boolean that specifies the default value for a Boolean type question and is optional. 'validation' is an object that defines validation for the question, which includes: 'type', a string that specifies the type of validation that will be performed, required to be one of: 'NotEmpty', 'Email', 'Number', 'Postcode', 'Regex'; 'value_type', a string that defines the type of value for validation, required to be one of: 'String', 'Date', 'Boolean', 'Number'; 'message', a string for the error message if validation fails, and is required; and 'regex', a string that indicates the regular expression for validation, which is optional. No additional properties are allowed at the root level.
