The schema describes a RenderTheFuck timeline diagram. It is an object type. The main properties of the schema are: 'lines' is an array of objects representing timeline line items, with each object requiring the properties 'title' (a string), 'from' (a number), and 'to' (a number). 'settings' is an object that requires the properties 'title' (a string with a minimum length of 1) and 'unit' (a string that must be 'year'). 'wtf' is a string that must be 'timeline'. All three main properties are required. No additional properties are allowed at the root level.
