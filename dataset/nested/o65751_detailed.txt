The schema describes a character set. It is an array type. The main properties of the schema are: [Character] is an object that describes a character. It has required fields: 'id', 'name', and 'character'. 'id' is a string that represents a Unicode identifier for the character. 'name' is a string that represents the character value itself. 'character' is a string that denotes the actual typed initial character. There is also an optional array property 'contextualforms', which consists of objects containing 'final', 'initial', 'isolated', and 'medial', each of which is a string. Additionally, there is an optional array property 'tags', which can contain unique items of type string with a minimum of 0 items. No additional properties are allowed at the root level.
