The schema describes the structure of an object containing images and assets. It is an object type. The main properties of the schema are: 'images' is an object representing image-related properties. It is required. It contains the following properties: 'screenshots' is an array of strings, where each string represents a PNG screen URL, preferably 1024 by 1024 pixels. 'icon-large' is a string representing a PNG icon URL, preferably 256 by 256 pixels. 'icon-small' is a string representing a PNG icon URL, preferably 48 by 48 pixels. 'icon-medium' is a string representing a PNG icon URL, preferably 128 by 128 pixels. 'assets' is an object representing asset-related properties. It is required. It contains the following properties: 'container' is an object that can include any properties defined with string values for its content, and it is disallowed from having additional properties itself. 'uris' is an object that can include any properties defined with string values for its content. No additional properties are allowed at the root level.
