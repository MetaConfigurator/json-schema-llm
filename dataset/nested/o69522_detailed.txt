The schema describes an object structure for representing an item. It is an object type. The main properties of the schema are: 'id' is a string required field. 'name' is a string optional field. 'price' is a number optional field. 'tags' is an array optional field. 'vector' is an array optional field. 'address' is an object optional field which has required fields: 'street', 'number', 'zip_code', and 'country'. It also includes properties: 'street' as a string, 'number' as a number, 'zip_code' as a string, 'state' as an object that has properties: 'name' as a string and 'code' as a string.
