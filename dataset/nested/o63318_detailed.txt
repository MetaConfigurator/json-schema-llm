The schema describes NgRx Action Options. It is an object type. The main properties of the schema are: 'api' is a boolean specifies if api success and failure actions should be generated. It is optional and defaults to false. 'creators' is a boolean specifies whether to use creator functions for handling actions and reducers. It is optional and defaults to true. 'flat' is a boolean flag to indicate if a dir is created. It is optional and defaults to true. 'group' is a boolean groups actions file within 'actions' folder. It is optional and defaults to false. 'name' is a string the name of the action. It is required to be sourced from 'argv' at index 0. 'path' is a string the path to create the component. It is optional and marked as not visible. 'project' is a string the name of the project. It is optional. 'skipTests' is a boolean when true, does not create test files. It is optional and defaults to false. No additional properties are allowed at the root level.
