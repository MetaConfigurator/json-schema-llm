The schema describes an Object with metadata behind a brid. It is an object type. The main properties of the schema are: 'brid' is a string that is required and describes the brid. 'domain' is a string that is required and describes the domain of the object. 'combined' is a string that is optional and describes the combined proverb. 'instanzen' is an array where each item is an object representing instance information for one primary system; it must contain properties: 'system' (string, required), 'id' (string, required), and 'description' (string, optional). 'anmerkung' is a string that is optional and describes additional information. No additional properties are allowed at the root level.
