The schema describes a representation of a FCC Bureau or Office. It is an object type. The main properties of the schema are: '_metadata' is an object that contains required properties 'request' and 'response'. 'request' is an object that contains required properties 'ip', 'params', 'time', and 'user'. 'ip' is a string, 'params' is null, 'user' is null, and 'time' is a string. 'response' is an object that contains the required property 'time', which is a string. 'bureau' is an object that contains required properties 'bureau_code', 'bureau_name', 'edocs_bureau_code', and 'id'. 'bureau_name' is a string, 'id' is an integer, 'edocs_bureau_code' is null, and 'bureau_code' is a string. No additional properties are allowed at the root level.
