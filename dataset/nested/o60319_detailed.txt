The schema describes testsuite metadata. It is an object type. The main properties of the schema are: 'scm_url_base' is a string Base URL to test source code. This property is optional. 'tests' is an object A collection of tests, each specified as a key-value pair where the key is the test identifier and the value is another object containing various test properties. This property is required. 'testsuite' is a string Testsuite full name. This property is required. 'testsuite_short' is a string Abbreviated testsuite name. This property is optional. 'timeout' is an integer Default timeout for testcases. This property is optional with a minimum value of 1. 'url' is a string Testsuite website URL. This property is optional. 'version' is a string Testsuite version. This property is optional. Compositional constraints: The root schema has required properties: 'testsuite' and 'tests'. No additional properties are allowed at the root level.
