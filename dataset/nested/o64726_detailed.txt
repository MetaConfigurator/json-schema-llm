The schema describes metadata returned by DataSources. It is an object type. It defines several reusable components under 'definitions': 'arrayOfString' is an array, containing items that are of type string. 'uri' is a string, formatted as a URI. The main properties of the schema are: '$schema' is a string that references '#/definitions/uri' and is required. 'allow' is a string that references '#/definitions/uri' and is required. 'annotations' is an array, containing items that are objects with additional properties allowed and is optional. 'description' is a string and is optional. 'encoding' is a string and is optional. 'format' is an array, referencing '#/definitions/arrayOfString' and is optional. 'language' is an array, referencing '#/definitions/arrayOfString' and is optional. 'license' is a string that references '#/definitions/uri' and is required. 'name' is a string and is required. 'size' is an integer with a minimum value of 0 and is optional. 'vendor' is a string that references '#/definitions/uri' and is required. 'version' is a string and is required. No additional properties are allowed at the root level.
