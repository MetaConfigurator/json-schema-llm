The schema describes a tunnel traffic policy binding. It is an object type. The main properties of the schema are: 'builtin' is an object with an enum of allowed values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'feature' is a string with an enum of allowed values: 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', 'LoadBalancing', 'CS', 'ContentSwitching', 'CR', 'CacheRedirection', 'SC', 'SureConnect', 'CMP', 'CMPcntl', 'CompressionControl', 'PQ', 'PriorityQueuing', 'HDOSP', 'HttpDoSProtection', 'SSLVPN', 'AAA', 'GSLB', 'GlobalServerLoadBalancing', 'SSL', 'SSLOffload', 'SSLOffloading', 'CF', 'ContentFiltering', 'IC', 'IntegratedCaching', 'OSPF', 'OSPFRouting', 'RIP', 'RIPRouting', 'BGP', 'BGPRouting', 'REWRITE', 'IPv6PT', 'IPv6protocoltranslation', 'AppFw', 'ApplicationFirewall', 'RESPONDER', 'HTMLInjection', 'push', 'NSPush', 'NetScalerPush', 'AppFlow', 'CloudBridge', 'ISIS', 'ISISRouting', 'CH', 'CallHome', 'AppQoE', 'ContentAccelerator', 'SYSTEM', 'RISE', 'FEO', 'LSN', 'LargeScaleNAT', 'RDPProxy', 'Rep', 'Reputation', 'URLFiltering', 'VideoOptimization', 'ForwardProxy', 'SSLInterception', 'AdaptiveTCP', 'CQA', 'CI', 'ContentInspection', 'Bot'. 'globalbindtype' is a string with an enum of allowed values: 'SYSTEM_GLOBAL', 'VPN_GLOBAL', 'RNAT_GLOBAL'. 'gotopriorityexpression' is a string. 'numpol' is an integer that is readonly. 'policyname' is a string. 'policytype' is a string that is readonly with an enum of allowed values: 'Classic Policy', 'Advanced Policy'. 'priority' is an integer. 'state' is a string with an enum of allowed values: 'ENABLED', 'DISABLED'. 'type' is a string with an enum of allowed values: 'REQ_OVERRIDE', 'REQ_DEFAULT', 'RES_OVERRIDE', 'RES_DEFAULT'. No additional properties are allowed at the root level.
