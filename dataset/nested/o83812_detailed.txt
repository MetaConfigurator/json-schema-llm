The schema describes a VCMI map header format. It is an object type. The main properties of the schema are: 'defeatIconIndex' is a number required field; 'defeatString' is a string required field; 'victoryIconIndex' is a number required field; 'victoryString' is a string required field; 'triggeredEvents' is an optional object which can have additional properties that are also objects. Each of these objects is required to have 'condition', 'message', and 'effect' fields. The 'effect' field is an object that must contain 'type' and 'messageToSend', both being strings. No additional properties are allowed at the root level.
