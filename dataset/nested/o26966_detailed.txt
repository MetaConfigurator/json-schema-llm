The schema describes a keyframe segment. It is an object type. The main properties of the schema are: 's' is a number that represents the start value of the keyframe segment. It is optional. 't' is a number that denotes the start time of the keyframe segment. It is optional. 'i' is an object representing the Bezier curve interpolation in value. It has properties: 'x', which is an array of numbers for the bezier x axis, and 'y', which is an array of numbers for the bezier y axis. Both properties are optional. 'o' is an object representing the Bezier curve interpolation out value. It has properties: 'x', which is an array of numbers for the bezier x axis, and 'y', which is an array of numbers for the bezier y axis. Both properties are optional. No additional properties are allowed at the root level.
