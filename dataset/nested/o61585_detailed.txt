The schema describes configuration settings for Time Machine. It is an object type. The main properties of the schema are: 'AutoBackup' is a boolean that sets Time Machine to back up automatically, with a default value of true. 'BackupAllVolumes' is a boolean that sets Time Machine to back up all volumes, defaulting to false. 'BackupDestURL' is a string that specifies the Time Machine backup destination, with a default value of 'afp://someserver.apple.com/Backups/'. 'BackupSizeMB' is a number that sets the per client backup limit in MB, with a default value of 5000. 'BackupSkipSys' is a boolean that sets Time Machine to skip system file backup, defaulting to true. 'BasePaths' is an array describing base paths to back up. 'MobileBackups' is a boolean setting for backing up mobile users, with a default value of true. 'PayloadDescription' is a string that describes the payload, defaulting to 'Time Machine Settings'. 'PayloadDisplayName' is a string representing the name of the payload, with a default value of 'Time Machine Settings'. 'PayloadIdentifier' is a string that provides a unique identifier for the payload, defaulting to 'com.apple.MCX.TimeMachine'. 'PayloadOrganization' is a string describing the issuing organization. 'PayloadType' is a string that specifies the type of the payload, with a default value of 'com.apple.MCX.TimeMachine'. 'PayloadUUID' is a string that serves as a unique identifier for the payload, defaulting to an empty string, and must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is a number indicating the version of the configuration profile, defaulting to 1. 'SkipPaths' is an array for paths to skip. No additional properties are allowed at the root level.
