The schema describes an array of objects representing rendering engines. It is an array type. The main properties of the schema are: 'inherits' is a string, optional, and 'properties' is an object, required. The 'properties' object includes: 'RenderingEngine_Name' is a string, optional; 'RenderingEngine_Version' is a string, required; 'RenderingEngine_Description' is a string, optional; 'RenderingEngine_Maker' is a string, optional; 'JavaScript' is a boolean, optional; 'Cookies' is a boolean, optional; 'Frames' is a boolean, optional; 'IFrames' is a boolean, optional; 'Tables' is a boolean, optional; 'JavaApplets' is a boolean, optional; 'VBScript' is a boolean, optional; 'ActiveXControls' is a boolean, optional; 'BackgroundSounds' is a boolean, optional; 'CssVersion' is an integer, optional. The array must contain at least 1 item and all items must be unique. No additional properties are allowed at the root level.
