The schema describes user configuration for transfers. It is an object type. The main properties of the schema are: 'przelewy' is an array containing objects that each represent a transfer configuration. Each object requires 'nazwa', 'tytuł', and 'odbiorca'. The 'aktywny' property is a boolean indicating if the transfer is active, with a default value of true. The 'sms' property is also a boolean indicating if additional SMS validation is enabled, with a default value of true. The 'kwota' property can be either a number or a string representing the default amount for the transfer. No additional properties are allowed at the root level.
