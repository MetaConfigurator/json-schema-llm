The schema describes a single survey action. It is an object type. The main properties of the schema are: 'engagementButtonLabel' is a string that serves as the text for the engagement button and is optional with a default value of an empty string. 'includeTelemetryUUID' is a boolean that indicates whether to include a unique user ID in the post-answer URL and telemetry, and is optional with a default value of false. 'learnMoreMessage' is a string that shows a message to the user to learn more, optional with a default value of an empty string. 'learnMoreUrl' is a string providing a URL to show to the user when they click Learn More, optional with a default value of an empty string. 'message' is a string that serves as the message to show to the user, marked as required with a default value of an empty string. 'postAnswerUrl' is a string that defines a URL to redirect to after rating Firefox or clicking the engagement button, optional with a default value of an empty string. 'surveyId' is a string that uniquely identifies this survey in telemetry and is required. 'thanksMessage' is a string that serves as a thanks message to show to the user after they've rated Firefox, marked as required with a default value of an empty string. No additional properties are allowed at the root level.
