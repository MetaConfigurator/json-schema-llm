The schema describes signal subscription configuration. It is an object type. The main properties of the schema are: 'name' is a string representing the name of the subscription, optional. 'signalClass' is a string representing the class of the signal, optional. 'signalSubclass' is a string representing the subclass of the signal, optional. 'level' is a string representing the signal level, optional, with allowed values: 'Very low', 'Low', 'Medium', 'High', 'Very high', 'Maximum'. 'conflate' is a boolean indicating whether to conflate signals, optional, with a default value of true. 'autoClose' is a number representing the auto-close time in seconds, optional, with a default value of 3. No additional properties are allowed at the root level.
