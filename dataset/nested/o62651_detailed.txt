The schema describes ASN.1 Types Library Module Metadata. It is an object type. It defines several reusable components under 'definitions': 'nonEmptyString' is a string type with a minimum length of 1. 'fileList' is an array type containing items that are non-empty strings. 'externalReference' is an object type with required properties 'submodule' and 'element', both of which are non-empty strings, and has no additional properties. 'referenceList' is an array type where items can be either non-empty strings or external references. The main properties of the schema are: 'name' is a non-empty string, which is required; 'description' is an optional string; 'submodules' is an array of objects where each object must contain a required non-empty string 'name'. Each object can have an optional string 'description' and must have an 'elements' array. Each element in 'elements' is an object that must include a required non-empty string 'name' and can optionally contain 'description', 'asn1Files', 'additionalFiles', 'conflicts', and 'requires'. No additional properties are allowed at the root level.
