The schema describes configuration settings for a Linkerd instance. It is an object type. The main properties of the schema are: 'linkerd' is an object which contains settings for the Linkerd instances. It is required. The properties of 'linkerd' include: 'admin-port' is an integer that provides an administrative UI for this instance. It is required and has a default value of 9990. 'cpus' is a number representing CPU shares allocated to each Linkerd instance. It is required, must be at least 0.25, and has a default value of 0.25. 'instances' is an integer indicating the number of instances to run. It is required, must be at least 1, and has a default value of 1. 'mem' is a number that specifies memory (MB) allocated to each Linkerd task. It is required, must be at least 256.0, and has a default value of 256.0. 'resource-roles' is a string specifying the accepted resource roles for the deployment. It is required, and has a default value of '"*", "slave_public"'. 'routing-port' is an integer representing the routing port. It is required and has a default value of 4140. No additional properties are allowed at the root level.
