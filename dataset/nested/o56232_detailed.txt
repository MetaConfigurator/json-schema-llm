The schema describes a JSON representation of /etc/fstab. It is an object type. It defines several reusable components under 'definitions': 'mntent' is an object, an fstab entry. It requires the fields 'device' and 'fstype'. The 'device' field is a string. The 'dump' field is an integer with a minimum value of 0. The 'fsck' field is an integer with a minimum value of 0. The 'fstype' field is a string. The 'options' field is an array of strings with a minimum of one item. The main properties of the schema are: 'swap' is an object that references the 'mntent' definition and is required. The root schema resolves to the referenced definition 'mntent' and has additional properties restricted at the root level.
