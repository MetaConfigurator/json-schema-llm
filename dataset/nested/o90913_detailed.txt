The schema describes a SHARE ChangeSet in JSONLD format. It is an object type. It defines several reusable components under 'definitions': 'primitives' is a union type, representing a string, number, boolean, or null. 'reference' is an object type, requiring the fields '@id' and '@type', prohibits additional properties, and contains properties '@id' (a string) and '@type' (a string). The main properties of the schema are: '@context' is an object, optional. '@graph' is an array that is required, used to express a graph, containing items that are objects requiring '@id' and '@type'. Each object in the array may have an optional property 'extra' of type object, and the '@id' field accepts either a string or an integer. The 'additionalProperties' for each item in the '@graph' array is constrained by oneOf, allowing a reference to 'reference', one of the 'primitives', or an array of references to 'reference'. No additional properties are allowed at the root level.
