The schema describes desktop wallpaper settings. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string that describes the payload. It has a default value of 'Desktop Wallpaper Settings'. 'PayloadDisplayName' is a string that provides the name of the payload. It has a default value of 'Desktop Wallpaper Settings'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, dot-delimited. It typically follows the format of root PayloadIdentifier and subidentifier, with a default value of 'com.apple.desktop'. 'PayloadOrganization' is a string that describes the issuing organization of the profile as displayed to the user. 'PayloadType' is a string that indicates the type of the payload, in reverse DNS format, with a default value of 'com.apple.desktop'. 'PayloadUUID' is a string that provides a unique identifier for the payload in the format 01234567-89AB-CDEF-0123-456789ABCDEF. It must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$' and has a default value of ''. 'PayloadVersion' is a number that indicates the version of the whole configuration profile with a default value of 1. 'locked' is a boolean indicating if the desktop picture is locked; it is optional and has a default value of false. 'override-picture-path' is a string that, if supplied, sets the path to the desktop picture; it is optional. No additional properties are allowed at the root level.
