The schema describes an invocation manifest for OpenSlide to PNG file converter. It is an object type. The main properties of the schema are: 'config' is an object that contains the configuration settings, which is required. Within 'config', 'gear-log-level' is a string that specifies the logging level, is required, has a default value of 'INFO', and may be one of the following: 'ERROR', 'WARNING', 'INFO', 'DEBUG'. 'inputs' is an object that holds the input data, which is required. Within 'inputs', 'image' is an object that currently has no properties defined and is required. No additional properties are allowed at the root level.
