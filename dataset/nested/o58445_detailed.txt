The schema describes Extract i18n target options for Build Facade. It is an object type. The main properties of the schema are: 'browserTarget' is a string target to extract from. It is required and must match the pattern '^[^:\s]+:[^:\s]+(:[^\s]+)?$'. 'format' is a string output format for the generated file. It is optional and has a default value of 'xlf'. Allowed values are 'xmb', 'xlf', 'xlif', 'xliff', 'xlf2', 'xliff2'. 'i18nFormat' is a string output format for the generated file. It is optional and has a default value of 'xlf'. Allowed values are 'xmb', 'xlf', 'xlif', 'xliff', 'xlf2', 'xliff2'. It is deprecated and suggests using the 'format' option instead. 'i18nLocale' is a string that specifies the source language of the application. It is optional and deprecated, suggesting to use 'i18n' project level sub-option 'sourceLocale' instead. 'progress' is a boolean that logs progress to the console. It is optional and has a default value of true. 'outputPath' is a string that specifies the path where output will be placed. It is optional. 'outFile' is a string that specifies the name of the file to output. It is optional. No additional properties are allowed at the root level.
