The schema describes a black market message structure. It is an object type. The main properties of the schema are: '$schemaRef' is a string required field. 'header' is an object required field with additional properties allowed, containing required fields: 'uploaderID', 'softwareName', and 'softwareVersion'. 'message' is an object required field with additional properties allowed, containing required fields: 'systemName', 'stationName', 'timestamp', 'name', 'sellPrice', and 'prohibited'. 'marketId' is a number optional field. 'name' is a string with a minimum length of 1. 'prohibited' is a boolean optional field. 'sellPrice' is an integer optional field. 'stationName' is a string with a minimum length of 1. 'systemName' is a string with a minimum length of 1. 'timestamp' is a string formatted as date-time. No additional properties are allowed at the root level.
