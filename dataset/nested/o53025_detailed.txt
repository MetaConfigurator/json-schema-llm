The schema describes 3D data upload. It is an object type. The main properties of the schema are: 'vertices' is an array of arrays containing numbers, required, with each inner array having a minimum and maximum of 3 items. 'facets' is an array of arrays containing integers, required, with each inner array also having a minimum and maximum of 3 items. 'camera' is an object, required, containing required properties 'x', 'y', and 'z', all of type number. No additional properties are allowed at the root level.
