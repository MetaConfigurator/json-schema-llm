The schema describes a gallery block used in a canvas. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string formatted as 'date-time', representing an update that occurred on the node this block is attached to. It is optional. 'aside' is a boolean indicating if the block represents a portion of a document only indirectly related to the main content. It is optional. 'node_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and is required. 'title' is a string with a minimum length of 0 and a maximum length of 255, serving as an optional override for the title of the node. 'launch_text' is a string with a minimum length of 0 and a maximum length of 255 and is optional. 'poster_image_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and is optional, providing a reference to an image asset to use as the poster. No additional properties are allowed at the root level.
