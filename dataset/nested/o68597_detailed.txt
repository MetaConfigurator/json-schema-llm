The schema describes an array of agents. It is an array type. The main properties of the schema are: 'apiVersion' is a string having an enum of allowed values: 'v1beta1', 'v1.0', 'v1'. 'kind' is a string having an enum of allowed values: 'Agent', 'agent', and is readOnly. It also includes readOnly properties 'createdOn' (string), 'modifiedOn' (string), and 'ref' (string) with minimum length 3 and maximum length 24. 'spec' is an object that is required, containing required properties 'credentials' and 'domains'. 'credentials' is an object that includes required properties 'username' (string, readOnly, minimum length of 3, maximum length of 30) and 'secret' (string, writeOnly). 'domains' is an array with a minimum of 1 item, each item being a string and readOnly, with unique items. No additional properties are allowed at the root level.
