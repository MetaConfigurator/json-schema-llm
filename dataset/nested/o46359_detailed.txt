The schema describes the structure for updating charitable givings. It is an object type. It defines several reusable components under 'definitions': 'amount' is a number, defining an amount value (to 2 decimal places), between 0 and 99999999999.99. 'charityNames' is an array that contains strings, each string must have a minimum length of 1 and a maximum length of 75, and must match the pattern '^[A-Za-z0-9 &'\(\)\*,\-\./@\u00a3]{1,75}$'. The main properties of the schema are: 'giftAidPayments' is an object representing Gift Aid Payments, containing properties: 'specifiedYear' (an amount), 'oneOffSpecifiedYear' (an amount), 'specifiedYearTreatedAsPreviousYear' (an amount), 'followingYearTreatedAsSpecifiedYear' (an amount), 'nonUKCharities' (an amount), and 'nonUKCharityNames' (charityNames); 'gifts' is an object representing Gifts, containing properties: 'sharesOrSecurities' (an amount), 'landAndBuildings' (an amount), 'investmentsNonUKCharities' (an amount), and 'investmentsNonUKCharityNames' (charityNames). No additional properties are allowed at the root level.
