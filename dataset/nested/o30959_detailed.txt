The schema describes configuration parameters for SSL settings. It is an object type. The main properties of the schema are: 'crlmemorysizemb' is an integer that specifies the CRL memory size in MB. 'cryptodevdisablelimit' is an integer that indicates the limit for disabling cryptographic devices. 'defaultprofile' is a string that specifies the default profile with allowed values: 'ENABLED', 'DISABLED'. 'denysslreneg' is a string that indicates SSL renegotiation denial with allowed values: 'NO', 'FRONTEND_CLIENT', 'FRONTEND_CLIENTSERVER', 'ALL', 'NONSECURE'. 'dropreqwithnohostheader' is a string that indicates whether to drop requests without a host header, allowing values: 'YES', 'NO'. 'encrypttriggerpktcount' is an integer defining the packet count for encryption triggering. 'heterogeneoussslhw' is a string indicating heterogeneous SSL hardware usage with allowed values: 'ENABLED', 'DISABLED'. 'hybridfipsmode' is a string that specifies hybrid FIPS mode with allowed values: 'ENABLED', 'DISABLED'. 'insertcertspace' is a string that indicates whether to insert certificate space, allowing values: 'YES', 'NO'. 'insertionencoding' is a string specifying the encoding for insertion with allowed values: 'Unicode', 'UTF-8'. 'montls1112disable' is a string that indicates whether TLS 1.1 and 1.2 are disabled; it is readonly and allows values: 'YES', 'NO'. 'ndcppcompliancecertcheck' is a string indicating whether NDC compliance certificate checks are enabled, allowing values: 'YES', 'NO'. 'ocspcachesize' is an integer defining the size of the OCSP cache. 'pushenctriggertimeout' is an integer that specifies the timeout for push encryption triggering. 'pushflag' is an integer that specifies the push flag. 'quantumsize' is a string indicating the size of quantum with allowed values: '4096', '8192', '16384'. 'sendclosenotify' is a string that indicates whether to send close notify messages, allowing values: 'YES', 'NO'. 'sigdigesttype' is an object specifying the signature digest type. 'snihttphostmatch' is a string that describes how to match SNI HTTP hosts with allowed values: 'NO', 'CERT', 'STRICT'. 'softwarecryptothreshold' is an integer defining the threshold for software cryptography. 'sslierrorcache' is a string indicating whether SSL error caching is enabled, with allowed values: 'ENABLED', 'DISABLED'. 'sslimaxerrorcachemem' is an integer that specifies the maximum memory for SSL error cache. 'ssltriggertimeout' is an integer that defines the timeout for SSL triggering. 'strictcachecks' is a string indicating whether strict cache checks are enabled, allowing values: 'YES', 'NO'. 'svctls1112disable' is a string that indicates whether TLS 1.1 and 1.2 are disabled; it is readonly and allows values: 'YES', 'NO'. 'undefactioncontrol' is a string for undefined action control. 'undefactiondata' is a string for undefined action data. No additional properties are allowed at the root level.
