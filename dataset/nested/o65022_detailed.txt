The schema describes a configuration object. It is an object type. The main properties of the schema are: 'title' is a string required field. 'icon' is a string optional field. 'background' is a string optional field. 'background-color' is a string optional field with a format of 'css-color'. 'icon-size' is an integer optional field. 'window' is an object optional field with properties: 'position' is an object with required properties 'x' (integer) and 'y' (integer), and 'size' is an object with required properties 'width' (integer) and 'height' (integer). 'format' is a string optional field that must be one of the following values: 'UDRW', 'UDRO', 'UDCO', 'UDZO', 'ULFO', 'UDBZ'. 'contents' is an array required field, where each item is an object with required properties: 'x' (integer), 'y' (integer), 'type' (string) which must be one of the values: 'link', 'file', 'position', and 'path' (string). No additional properties are allowed at the root level.
