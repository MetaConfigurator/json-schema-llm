The schema describes a Facebook post block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'; it is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and, in rendering environments that support HTML, the css_class can be appended to the DOM elements' class attribute; it is optional. 'updated_date' is a string in date-time format that represents an update that occurred on the node this block is attached to, not the block itself; it is optional. 'aside' is a boolean that, when true, indicates this block represents content indirectly related to the main content, often presented as sidebars or call-out boxes; it is optional. 'href' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$'; it is required. 'show_text' is a boolean; it is optional. No additional properties are allowed at the root level.
