The schema describes EPUB extension link properties. It is an object type. The main properties of the schema are: 'contains' is an array that identifies content contained in the linked resource, which cannot be strictly identified using a media type. It is optional and requires unique items consisting of strings that must be one of the following values: 'mathml', 'onix', 'remote-resources', 'js', 'svg', 'xmp'. 'layout' is a string that hints how the layout of the resource should be presented. It is optional and must be one of the following values: 'fixed', 'reflowable'. 'media-overlay' is a string that specifies the location of a media-overlay for the resource referenced in the Link Object. It is optional and formatted as a URI reference. 'overflow' is a string that suggests the method for handling overflow while displaying the linked resource. It is optional and must be one of the following values: 'auto', 'paginated', 'scrolled', 'scrolled-continuous'. 'spread' is a string that indicates the condition to be met for the linked resource to be rendered within a synthetic spread. It is optional and must be one of the following values: 'auto', 'both', 'none', 'landscape'. 'encrypted' is an object that indicates that a resource is encrypted/obfuscated and provides relevant information for decryption. It is optional and requires the 'algorithm' property to be a string formatted as a URI. In addition, it may contain 'compression' as a string, 'original-length' as an integer, 'profile' as a string formatted as a URI, and 'scheme' as a string formatted as a URI. No additional properties are allowed at the root level.
