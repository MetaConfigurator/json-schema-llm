The schema describes a configuration profile for a conference room display. It is an object type. The main properties of the schema are: 'Message' is a string optional. 'PayloadDescription' is a string with a default value of 'Conference Room Display Payload'. 'PayloadDisplayName' is a string with a default value of 'Conference Room Display Payload'. 'PayloadIdentifier' is a string with a default value of 'com.apple.SubmitDiagInfo'. 'PayloadOrganization' is a string optional. 'PayloadType' is a string with a default value of 'com.apple.conferenceroomdisplay'. 'PayloadUUID' is a string with a default value of '' and must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is a number with a default value of 1. No additional properties are allowed at the root level.
