The schema describes a metadata record from Zenodo (datacite3). It is an object type. The main properties of the schema are: 'authors' is an array of strings representing the authors of the work, required; must contain at least one item and must have unique items. 'affiliations' is an optional array of strings representing the authors' affiliations. 'title' is a string describing the title of the work, required. 'pubdate' is a string describing the publication date of the work, required. 'source' is a string describing the publisher of this metadata, optional. 'keywords' is an optional array of strings representing keywords, must have unique items. 'abstract' is a string describing a summary of the work, optional. 'references' is an optional array of strings representing references, must have unique items. 'properties' is an object containing: 'DOI' which is a string, required; 'PDF' which is a string, optional; 'OPEN' which is a number, optional; 'ELECTR' which is a string, optional. 'bibcode' is a string, optional. No additional properties are allowed at the root level.
