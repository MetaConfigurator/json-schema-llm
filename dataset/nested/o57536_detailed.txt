The schema describes options for an Exam test. It is an object type. The main properties of the schema are: 'qidGroups' is an array an array of QID difficulty groups, one per difficulty level of questions. 'availDate' is a string the date after which the test is available for students to view. 'nQuestions' is an integer the number of questions to select for each instance of this test. 'timeLimitMin' is an integer the number of minutes allowed for this test. 'text' is a string HTML text shown on the test overview page. No additional properties are allowed at the root level.
