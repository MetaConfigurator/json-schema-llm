The schema describes configuration for the TSQLLint tool. It is an object type. It defines several reusable components under 'definitions': 'rule' is a string, with an enum of allowed values: 'off', 'warning', 'error', and a default value of 'off'. The main properties of the schema are: 'rules' is an object, required, containing properties that reference the 'rule' definition with descriptions for many rules, with additional properties also referencing the 'rule' definition. 'plugins' is an optional object with additional properties that are strings. No additional properties are allowed at the root level.
