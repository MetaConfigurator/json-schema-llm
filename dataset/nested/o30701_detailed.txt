The schema describes a VLAN configuration. It is an object type. The main properties of the schema are: 'aliasname' is a string optional property. 'dynamicrouting' is a string optional property with an enum of allowed values: 'ENABLED', 'DISABLED'. 'id' is an integer optional property. 'ifaces' is a string readonly property. 'ifnum' is a string readonly property. 'ipv6dynamicrouting' is a string optional property with an enum of allowed values: 'ENABLED', 'DISABLED'. 'linklocalipv6addr' is a string readonly property. 'lsbitmap' is an integer readonly property. 'lstagbitmap' is an integer readonly property. 'mtu' is an integer optional property. 'partitionname' is a string readonly property. 'portbitmap' is an integer readonly property. 'rnat' is a boolean readonly property. 'sdxvlan' is a string readonly property with an enum of allowed values: 'YES', 'NO'. 'sharing' is a string optional property with an enum of allowed values: 'ENABLED', 'DISABLED'. 'tagbitmap' is an integer readonly property. 'tagged' is a boolean readonly property. 'tagifaces' is a string readonly property. 'vlantd' is an integer readonly property. 'vxlan' is an integer readonly property. No additional properties are allowed at the root level.
