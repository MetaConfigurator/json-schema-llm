The schema describes runtime capabilities and types. It is an object type. The root schema resolves to the referenced definition of RuntimeType. The effective root schema defines several properties: 'hypertyCapabilities' is an object that references the RuntimeHypertyCapabilities definition. 'protocolCapabilities' is an object that references the RuntimeProtocolCapabilities definition. 'runtimeType' is an object that references the RuntimeType definition. All three properties are required. The main properties of the schema are: 'hypertyCapabilities' is an object that contains properties describing capabilities related to hyperties, including 'camera', 'mic', 'ortc', 'sensor', and 'webrtc', each of type boolean and optional. 'protocolCapabilities' is an object that contains properties describing protocol capabilities, including 'coap', 'datachannel', 'http', 'https', 'ws', and 'wss', each of type boolean and optional. 'runtimeType' is an object that includes properties 'browser', 'gateway', 'server', and 'standalone', each of type null, and all are required. No additional properties are allowed at the root level.
