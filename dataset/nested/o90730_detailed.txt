The schema describes the configuration for a Linkerd instance. It is an object type. The main properties of the schema are: 'linkerd' is an object containing various configuration settings. It has the following required properties: 'admin-port' is an integer that specifies the Admin port, which provides an administrative UI for this instance. Its default value is 9990. 'cpus' is a number that indicates the CPU shares to allocate to each linkerd instance, with a minimum value of 0.25 and a default of 0.25. 'instances' is an integer representing the number of instances to run, with a minimum value of 1 and a default of 1. 'internal-port' is an integer that denotes the internal routing port, with a default value of 4141. 'mem' is a number indicating the memory (in MB) to allocate to each linkerd task, with a minimum value of 256.0 and a default of 256.0. 'resource-roles' is a string that specifies the accepted resource roles (e.g. slave_public), with a default value of '"*", "slave_public"'. 'routing-port' is an integer that indicates the routing port, with a default value of 4140. No additional properties are allowed at the root level.
