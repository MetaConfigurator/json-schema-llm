The schema describes inputs for processing. It is an array type. The main properties of the schema are: [files] is an object that contains properties for file inputs, which is required. The properties include: [data_inputfiles] is an array of strings representing data input files. [signal_inputfiles] is an array of strings representing signal input files. [signal_mcweight] is a number representing the signal MC weight. [bkg_mcinfo] is an array of objects which contain properties for background MC information, with each object having: [inputfiles] as an array of strings for background input files, [name] as a string for the name, and [weight] as a number for the weight. No additional properties are allowed at the root level.
