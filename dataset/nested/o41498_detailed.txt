The schema describes an invocation manifest for HDFT subsampled diffusion reconstruction. It is an object type. The main properties of the schema are: 'inputs' is an object containing required properties: 'dwi_file', 'bvecs_file', 'subsampling_vec', and 'bvals_file'. 'dwi_file' is an object with a required property 'type', which must be 'nifti'. 'bvecs_file' is an object with a required property 'type', which must be 'bvec'. 'subsampling_vec' is an object with a required property 'type', which must be 'tabular data'. 'bvals_file' is an object with a required property 'type', which must be 'bval'. 'config' is an object containing required properties: 'sh_filename', 'mean_diffusion_length', and 'spherical_harmonics_order'. 'sh_filename' is a string with a default value of 'sumas_sh_coeff.nii', describing the output filename of 4D spherical harmonic coefficient data. 'mean_diffusion_length' is a number with a default value of 1.2, describing the mean diffusion length for reconstruction of the GQI matrix. 'spherical_harmonics_order' is an integer with a default value of 8, describing the maximum order of spherical harmonics. No additional properties are allowed at the root level.
