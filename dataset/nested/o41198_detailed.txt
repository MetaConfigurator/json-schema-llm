The schema describes an invocation manifest for BXH-XCEDE-TOOLS: fMRI QA (v1.11.14). It is an object type. The main properties of the schema are: 'inputs' is an object that is required, with a required property 'fmri_input' which is also an object containing a property 'type' that is an enum with allowed values 'dicom' and 'nifti'. 'config' is an object that is required, with required properties 'save_zip', 'birn_phantom', and 'save_qa_stats', all of which are booleans with default values set to false. No additional properties are allowed at the root level.
