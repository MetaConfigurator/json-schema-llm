The schema describes information from the GLANCE database. It is an object type. The main properties of the schema are: 'status' is a string status. 'ref_code' is a string reference code. 'pub_short_title' is a string publication title. 'short_title' is a string short title. 'creation_date' is a string creation date. 'full_title' is a string full title. 'gitlab_group' is an object representing a Gitlab group, containing 'id' as a string and 'projects' as an array of objects each with 'url' as a string and 'id' as a string. 'phase_0' is an array of objects, each containing 'model_tested' as a string, 'main_physics_aim' as a string, 'dataset_used' as a string, 'id' as a string, and 'methods' as a string. 'id' is a string GLANCE ID. No additional properties are allowed at the root level.
