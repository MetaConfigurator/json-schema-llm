The schema describes a gallery block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string in date-time format representing the update of the node this block is attached to. It is optional. 'aside' is a boolean that indicates if the block represents content indirectly related to the main content. It is optional. 'node_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and is required. 'title' is a string with a minimum length of 0 and a maximum length of 255, serving as an optional override for the title of the node. 'launch_text' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'aspect_ratio' is a string with an enum of allowed values: 'unknown', 'auto', '1by1', '2by3', '3by2', '3by4', '4by3', '4by5', '5by4', '9by16', '16by9', 'original', 'custom'. It is optional. 'poster_image_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and references an image asset to use as the poster. It is optional. 'start_at_poster' is a boolean with a default value of true, indicating that the gallery should open at the selected poster image. It is optional. No additional properties are allowed at the root level.
