The schema describes configuration options for command execution. It is an object type. The main properties of the schema are: 'skipTerminateQuickPick' is a boolean that specifies whether to show Terminate QuickPick, with a default value of false. 'skipSwitchToOutput' is a boolean that specifies whether to switch to Output, with a default value of false. 'skipErrorMessage' is a boolean that specifies whether to popup an Error message, with a default value of false. 'commands' is an array that holds command objects. Each command object is required to have 'text' and 'command' properties. 'text' is a string that represents the displayed text of the status bar item. 'command' is a string that contains the command content. Additional properties in the command objects include: 'alignment' which is a string with the enum values 'left' and 'right', defaulting to 'left'; 'tooltip' which is a string for the tooltip of the status bar item; 'color' which is a string for the text color; 'language' which is a string for the language filter; 'priority' which is a number representing the placement priority; and 'commandType' which is a string with the enum values 'exec', 'script', 'palette', and 'file', defaulting to 'exec'. No additional properties are allowed at the root level.
