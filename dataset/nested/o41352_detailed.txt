The schema describes an invocation manifest for ROI to NIfTI. It is an object type. The main properties of the schema are: 'config' is an object that contains the configuration settings. It is required. It includes several properties: 'save_binary_masks' is a boolean which defaults to true; 'save_combined_output' is a boolean which defaults to false; 'combined_output_size' is a string with a default of 'int32' and allowed values are 'int8', 'int16', 'int32', 'int64'; 'save_slicer_color_table' is a boolean which defaults to false. 'inputs' is an object that contains input specifications. It is required. It includes properties: 'api-key' is an object; 'Input_File' is an object that includes: 'type' which is a string and allowed value is 'nifti'. No additional properties are allowed at the root level.
