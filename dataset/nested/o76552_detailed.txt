The schema describes the configuration for connecting to DC/OS and Mesos. It is an object type. The main properties of the schema are: 'dcos_acs_token' is a string that represents the token generated by authenticating to DC/OS with ACS. It is optional. 'dcos_url' is a string that denotes the public master IP of your DC/OS installation. It is optional and must be in the format of a URI. 'mesos_master_url' is a string that indicates the Mesos master URL and must be set in the format: "http://host:port". It is optional and must be in the format of a URI. 'pagination' is a boolean that specifies whether to paginate output. It is optional with a default value of true. 'reporting' is a boolean that determines whether to report usage events to Mesosphere. It is optional with a default value of true. 'ssl_verify' is a string that indicates whether to verify SSL certs for HTTPS or provides a path to certs. It is optional with a default value of 'false'. 'timeout' is an integer that represents the request timeout in seconds, with a minimum value of 1 and a default value of 5. No additional properties are allowed at the root level.
