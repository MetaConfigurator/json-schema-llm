The schema describes a ClinVar report. It is an object type. The main properties of the schema are: 'schema_version' is a string and is required. 'notes' is an object and is optional. 'metadata' is an object and is optional, containing 'genome_build', which is a string, and 'notes', which is a string. 'variants' is an array of objects and is optional, where each object has properties: 'chrom' (string), 'ref_allele' (string), 'alt_allele' (string), 'zygosity' (string), 'name' (string), 'pos' (integer), 'acc_url' (string), 'allele_freq' (string), and 'clinical_significance' (string). No additional properties are allowed at the root level.
