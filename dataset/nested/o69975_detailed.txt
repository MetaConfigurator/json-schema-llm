The schema describes patron types. It is an object type. The main properties of the schema are: '$schema' is a string for Schema to validate patron_types records against, required, with a minimum length of 9 and a default value of 'https://ils.rero.ch/schemas/patron_types/patron_type-v0.0.1.json'. 'pid' is a string for Patron type ID, required, with a minimum length of 1. 'name' is a string for the name of the patron type, required, with a minimum length of 2. 'description' is a string for the description of the patron type, optional. 'organisation' is an object for Organisation, containing a '$ref' property which is a string with a pattern matching '^https://ils.rero.ch/api/organisations/.*?$'. 'subscription_amount' is a number for the yearly amount due by patrons linked to this patron type, optional, with a default value of 0 and a minimum value of 0. No additional properties are allowed at the root level.
