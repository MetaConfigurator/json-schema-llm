The schema describes a structured event notification. It is an object type. The main properties of the schema are: 'meta' is an object containing metadata about the event. It is required and has properties including 'id' (a required string with a UUID pattern), 'type' (a required string with an enum of ["EiffelArtifactDeployedEvent"]), 'version' (a required string with an enum of ["1.0.0"] and a default value of "1.0.0"), 'time' (a required integer), and 'source' (a required object with properties 'domainId', 'host', 'name', 'serializer' [an object with required properties 'groupId', 'artifactId', and 'version'], and 'uri'; additional properties are not allowed). 'data' is an object containing relevant data. It is required and has properties 'uri' (an optional string) and 'customData' (an optional array of objects, each with required properties 'key' [a string] and 'value' [of any type]; additional properties are not allowed). 'links' is an array of objects, each containing required properties 'type' (a string) and 'target' (a string with a UUID pattern). No additional properties are allowed at the root level.
