The schema describes vCenter import. It is an object type. The main properties of the schema are: 'fabric_fq_name' is an array of strings representing the fully qualified name of the fabric. This property is optional. 'contrail_command' is an object containing related details. It has the following properties: 'username' is a string describing the Contrail Command Login Username, 'password' is a string for the Contrail Command Login Password, and 'host' is a string for the Contrail Command IP and/or port. All properties of 'contrail_command' are optional. 'vcenter' is an object encompassing vCenter details. It includes: 'username' which is a string for the Username of vCenter Server, 'password' which is a string for the Password of vCenter Server, 'datacenter' which is a string for the vCenter DataCenter name to Import ESXi Hosts, and 'host' which is a string for the vCenter Hostname/IP-Address including port. All properties of 'vcenter' are optional. No additional properties are allowed at the root level.
