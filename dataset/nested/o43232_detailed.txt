The schema describes a collection of statistics and timelapses related to medals and impulses. It is an object type. It defines several reusable components under 'definitions': 'StatCount' is an object, representing a statistical entry with required fields 'Id' (a string) and 'Count' (an integer). 'Timelapse' is an object, representing a timelapse entry with required fields 'Id' (a string) and 'Timelapse' (a string). The main properties of the schema are: 'MedalStatCounts' is an array containing items that must conform to the 'StatCount' definition. 'ImpulseStatCounts' is an array containing items that must conform to the 'StatCount' definition. 'MedalTimelapses' is an array containing items that must conform to the 'Timelapse' definition. 'ImpulseTimelapses' is an array containing items that must conform to the 'Timelapse' definition. All of the root-level properties are required. No additional properties are allowed at the root level.
