The schema describes the structure of a mitigation scope configuration. It is an object type. The main properties of the schema are: 'mitigation-scope' is an object that contains two properties: 'client-identifier' is an array of strings, which is optional; 'scope' is an array that contains objects, where each object has the following properties: 'mitigation-id' is an integer, 'target-ip' is an array of strings, 'target-prefix' is an array of strings, 'target-port-range' is an array of objects each having 'lower-port' and 'upper-port' as integers, 'target-protocol' is an array of integers, 'FQDN' is an array of strings, 'URI' is an array of strings, 'alias' is an array of strings, 'lifetime' is an integer, and 'attack-status' is an integer. All properties within 'scope' are optional. No additional properties are allowed at the root level.
