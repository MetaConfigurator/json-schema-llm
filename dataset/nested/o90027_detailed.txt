The schema describes an elevation layer used for rendering elevation in web scenes. It is an object type. The main properties of the schema are: 'id' is a string A unique identifying string for the layer, required. 'itemId' is a string Optional string containing the item ID of the service if it's registered on ArcGIS Online or your organization's portal. 'layerType' is a string String indicating the layer type, required, with allowed values: 'ArcGISTiledElevationServiceLayer'. 'listMode' is a string To show or hide the elevation layer in the layer list, with allowed values: 'show', 'hide'. 'title' is a string A user-friendly string title for the layer that can be used in a table of contents. If this is not included, a title is derived from the service, required. 'url' is a string The URL to the layer. If the layer is not from a web service but rather a feature collection, then the url property is omitted, required. 'visibility' is a boolean Boolean property determining whether the layer is initially visible in the web scene, with a default value of true. No additional properties are allowed at the root level.
