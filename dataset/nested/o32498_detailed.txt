The schema describes a competency ability. It is an object type. The main properties of the schema are: 'id' is a string that represents the uuid of the competency ability and is required. 'competency' is an object that contains an 'id', which is a string representing the uuid of the competency and is required. 'ability' is an object that includes 'id', a string representing the uuid of the ability which is required, and 'name', a string representing the name of the ability which is also required, as well as 'minResourceCount', an integer representing the number of resources to pass, and 'minEvaluatedResourceCount', an integer for the number of evaluated resources to pass. 'level' is an object with an 'id', which is a string representing the uuid of the level and is required. No additional properties are allowed at the root level.
