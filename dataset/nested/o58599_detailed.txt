The schema describes a dataset object. It is an object type. The main properties of the schema are: 'data' is an array listing identifiers (or URIs) of the references describing the data. It is optional. 'description' is a string providing a short description of the dataset. It is optional. 'environment' is an array listing identifiers (or URIs) of the environments associated to the dataset. It is optional. 'id' is an integer representing the dataset identifier. It is optional. 'name' is a string specifying the name of the dataset. It is required. 'networks' is an array listing identifiers (or URIs) of the networks in the dataset. It is required. 'owner' is a string representing a single related resource, which can be either a URI or a set of nested resource data. It is required. 'papers' is an array listing identifiers (or URIs) of the references to the papers associated with the dataset. It is optional. 'public' is a boolean indicating whether the dataset can be viewed by all users. It is required. 'resource_uri' is a string that contains Unicode string data. It is optional. No additional properties are allowed at the root level.
