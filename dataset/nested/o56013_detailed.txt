The schema describes Fibaro Home Center Room JSON Schema. It is an object type. The main properties of the schema are: 'id' is an integer identifier for the room, required field, with a minimum value of 0. 'name' is a string name for the room, required field, with a maximum length of 20. 'sectionID' is an integer identifier for the section the room belongs to, required field, with a minimum value of 0. 'icon' is a string icon name defined for the room, optional, with a maximum length of 20. 'defaultSensors' is an object representing room default sensors, optional, containing: 'temperature' as an integer default temperature sensor, optional, with a minimum value of 0; 'humidity' as an integer default humidity sensor, optional, with a minimum value of 0; 'light' as an integer default light sensor, optional, with a minimum value of 0. 'defaultThermostat' is an integer default thermostat, optional, with a minimum value of 0. 'created' is an integer room created timestamp, optional, with a minimum value of 0. 'modified' is an integer room modified timestamp, optional, with a minimum value of 0. 'sortOrder' is an integer sort order when displayed in GUI, optional, with a minimum value of 0. No additional properties are allowed at the root level.
