The schema describes a configuration profile for application access restrictions. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string that describes the payload. It has a default value of 'Configures restrictions (new)'. 'PayloadDisplayName' is a string that specifies the name of the payload. It has a default value of 'Restrictions (new)'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, formatted as a dot-delimited string. It has a default value of 'com.apple.applicationaccess.new'. 'PayloadOrganization' is a string describing the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string indicating the type of the payload, formatted as a reverse DNS string. It has a default value of 'com.apple.applicationaccess.new'. 'PayloadUUID' is a string that uniquely identifies the payload in a specific format (01234567-89AB-CDEF-0123-456789ABCDEF), and it has a pattern constraint for validation. It has a default value of an empty string. 'PayloadVersion' is a number indicating the version of the whole configuration profile. It has a default value of 1. 'familyControlsEnabled' is a boolean that is required to enable application access restrictions, and must be set to true. 'pathBlackList' is an array of strings that specifies optional paths to disallowed processes. 'pathWhiteList' is an array of strings for optional paths to allowed processes. 'whiteList' is an array of objects allowing whitelisted applications, each having a 'bundleID' string (optional) and a 'displayName' string that specifies either the display name or the absolute path to the application bundle. No additional properties are allowed at the root level.
