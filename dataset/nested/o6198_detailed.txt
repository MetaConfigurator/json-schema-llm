The schema describes a device configuration. It is an object type. The main properties of the schema are: 'name' is a string with a minimum length of 1 and a maximum length of 255. 'deviceName' is a string with a minimum length of 1 and a maximum length of 255. 'description' is a string with a maximum length of 32767. 'deviceDescription' is a string with a maximum length of 32767. 'tags' is an array of objects, each containing a required 'key' string that matches the pattern '^[0-9a-zA-Z_-]{1,255}$' and a required 'value' string with a minimum length of 1 and a maximum length of 255, with no additional properties allowed. 'attributes' is an array of objects, each containing a required 'name' string that matches the pattern '^[0-9a-zA-Z_-]{1,255}$' and a required 'dataType' string with allowed values 'string', 'number', 'gps', 'boolean', with no additional properties allowed. 'deviceClass' is a string with allowed values 'standalone', 'master', 'owned', 'floating', 'virtual'. 'masterId' is a string that matches the pattern '^[A-Fa-f\d]{24}$'. No additional properties are allowed at the root level.
