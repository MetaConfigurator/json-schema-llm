The schema describes configuration properties for a Grafana service. It is an object type. The main properties of the schema are: 'grafana' is an object representing grafana service configuration properties. It is required and includes the properties 'admin_password', 'cpus', and 'mem'. 'admin_password' is a string with a default value of 'admin'. 'cpus' is a number with a default value of 0.3 and a minimum value of 0.3. 'mem' is a number with a default value of 512.0 and a minimum value of 512.0. The 'networking' is an object representing Grafana networking configuration properties. It includes the property 'external_access', which is an object allowing the configuration of external access settings. The 'external_access' property has its own properties: 'enable', which is a boolean with a default value of true; 'external_access_port', which is a number with a default value of 13000; and 'virtual_host', which is a string with a default value of 'grafana.example.org'. The 'service' is an object representing DC/OS service configuration properties. It includes the property 'name', which is a string with a default value of 'grafana'. No additional properties are allowed at the root level.
