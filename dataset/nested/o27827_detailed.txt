The schema describes a collection of hardware parameters. It is an array type. It defines several reusable components under 'definitions': 'HardwareParameterLimits' is an object type, which requires the fields 'min', 'max', and 'value', all of which must be numbers with a minimum value of 1. The main properties of the schema are: 'offeringId' is a string and required; 'cpunumber' is a HardwareParameterLimits object and required; 'cpuspeed' is a HardwareParameterLimits object and required; 'memory' is a HardwareParameterLimits object and required. The schema enforces that items contain unique offeringIds. No additional properties are allowed at the root level.
