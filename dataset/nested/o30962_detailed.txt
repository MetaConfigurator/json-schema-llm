The schema describes a SSL policy. It is an object type. The main properties of the schema are: 'action' is a string optional field. 'builtin' is an object readonly field with an enum of allowed values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'comment' is a string optional field. 'description' is a string readonly field. 'feature' is a string readonly field with an enum of allowed values: 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', 'LoadBalancing', 'CS', 'ContentSwitching', 'CR', 'CacheRedirection', 'SC', 'SureConnect', 'CMP', 'CMPcntl', 'CompressionControl', 'PQ', 'PriorityQueuing', 'HDOSP', 'HttpDoSProtection', 'SSLVPN', 'AAA', 'GSLB', 'GlobalServerLoadBalancing', 'SSL', 'SSLOffload', 'SSLOffloading', 'CF', 'ContentFiltering', 'IC', 'IntegratedCaching', 'OSPF', 'OSPFRouting', 'RIP', 'RIPRouting', 'BGP', 'BGPRouting', 'REWRITE', 'IPv6PT', 'IPv6protocoltranslation', 'AppFw', 'ApplicationFirewall', 'RESPONDER', 'HTMLInjection', 'push', 'NSPush', 'NetScalerPush', 'AppFlow', 'CloudBridge', 'ISIS', 'ISISRouting', 'CH', 'CallHome', 'AppQoE', 'ContentAccelerator', 'SYSTEM', 'RISE', 'FEO', 'LSN', 'LargeScaleNAT', 'RDPProxy', 'Rep', 'Reputation', 'URLFiltering', 'VideoOptimization', 'ForwardProxy', 'SSLInterception', 'AdaptiveTCP', 'CQA', 'CI', 'ContentInspection', 'Bot'. 'hits' is an integer readonly field. 'name' is a string optional field. 'policytype' is a string readonly field with an enum of allowed values: 'Classic Policy', 'Advanced Policy'. 'reqaction' is a string optional field. 'rule' is a string optional field. 'undefaction' is a string optional field. 'undefhits' is an integer readonly field. No additional properties are allowed at the root level.
