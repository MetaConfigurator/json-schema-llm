The schema describes a configuration for a Flutter project. It is an object type. The main properties of the schema are: 'name' is a string representing the name of the project. 'uses-material-design' is a boolean indicating whether material design is used or not. 'assets' is an array of strings representing asset paths. 'services' is an array of strings representing service names. 'fonts' is an array containing objects that specify font configurations. Each font object contains 'family', a string representing the font family, and 'fonts', an array of objects, where each font object has 'asset', a string representing the font asset path, 'weight', an integer for font weight, and 'style', which must be either 'normal' or 'italic'. No additional properties are allowed at the root level.
