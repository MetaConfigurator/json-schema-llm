The schema describes a form field configuration. It is an object type. The main properties of the schema are: 'name' is a string that serves as a unique identifier for the field, required, and must match the pattern '^[a-zA-Z_]{1}[\w-]*$'. 'maps_to' is a string with a default value of 'cf' that specifies the schema field the answer will map to, must match the pattern '^[a-zA-Z_]{1}\w*$'. 'label' is a string with a minimum length of 0 and a maximum length of 255, serving as the main text for the question/field. 'placeholder' is a string with a minimum length of 0 and a maximum length of 255, providing a placeholder text. 'description' is a string with a minimum length of 0 and a maximum length of 65535, offering a short field explanation. 'is_required' is a boolean indicating if the field is mandatory. No additional properties are allowed at the root level.
