The schema describes an object configuration for a resource display. It is an object type. The main properties of the schema are: 'details' is an object that includes the required field 'resource', which is a string providing the name of the resource providing details. 'tabs' is an array that contains objects, each having optional properties: 'i18nHtmlTitle', which can be a string or an object; 'filter', which is an object with required fields 'fields' (a string, array, or object) and optional 'values' (a string, array, object, number, or boolean); and 'columns', which is a required array of objects, each containing an optional 'i18nHtmlTitle', an optional 'i18nHtmlFormat' with a default value of '[0]', an optional 'fields' (a string, array, or object), and an optional 'link' (a string). 'links' is an array of objects that require 'resource' (a string), and it includes optional 'field' (with a default of 'url') and 'i18nHtmlFormat' (with a default of '<a href="#">[0]</a>'). No additional properties are allowed at the root level.
