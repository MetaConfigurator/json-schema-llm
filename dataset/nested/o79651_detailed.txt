The schema describes a reference of exchanges published. It is an object type. The main properties of the schema are: '$schema' is a string that links to the schema for this reference, typically used to identify the reference document. 'description' is a string that provides a description of the set of exchanges in markdown. 'entries' is an array of objects where each object represents an exchange entry; each entry requires fields: 'type', 'exchange', 'name', 'title', 'description', 'routingKey', and 'schema'. 'exchangePrefix' is a string that specifies the prefix for all exchanges described in this document. 'title' is a string that serves as the title for the set of exchanges in markdown. 'version' is an enum that allows either the value 0 or the string '0.2.0'. All properties are required. No additional properties are allowed at the root level.
