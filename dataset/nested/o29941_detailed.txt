The schema describes configuration parameters for RADIUS authentication. It is an object type. The main properties of the schema are: 'accounting' is a string that specifies whether accounting is enabled or disabled, with allowed values 'ON' and 'OFF'. 'authentication' is a string indicating whether authentication is enabled or disabled, with allowed values 'ON' and 'OFF'. 'authservretry' is an integer that defines the number of retries for the authentication server. 'authtimeout' is an integer that sets the timeout value for authentication. 'builtin' is an object that describes built-in attributes and is read-only. 'callingstationid' is a string indicating whether the calling station ID is enabled or disabled, with allowed values 'ENABLED' and 'DISABLED'. 'defaultauthenticationgroup' is a string specifying the default authentication group. 'feature' is a string that lists various feature options as allowed values, including 'WL', 'WebLogging', 'SP', and many more. 'groupauthname' is a string that is read-only. 'ipaddress' is a string that provides the read-only IP address. 'ipattributetype', 'ipvendorid', 'pwdattributetype', 'pwdvendorid', 'radattributetype', 'radvendorid', 'serverport' are all integers defining respective attribute types and vendor IDs. 'passencoding' is a string that defines the password encoding method, with allowed values 'pap', 'chap', 'mschapv1', and 'mschapv2'. 'radgroupseparator', 'radgroupsprefix', 'radkey', and 'radnasid' are strings for various RADIUS configuration settings. 'radnasip' is a string that specifies whether the RADIUS NAS IP is enabled or disabled, with allowed values 'ENABLED' and 'DISABLED'. 'serverip' is a string specifying the server's IP address. 'tunnelendpointclientip' is a string indicating whether the tunnel endpoint client IP is enabled or disabled, with allowed values 'ENABLED' and 'DISABLED'. No additional properties are allowed at the root level.
