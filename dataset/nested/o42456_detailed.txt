The schema describes a dataset definition. It is an object type. The main properties of the schema are: 'datasetName' is a string, required, describing the name of dataset. 'models' is an array, required, with a minimum of 1 item, containing unique items. Each item is an object with the following properties: 'name' is a string, required, and 'fields' is an array, required, with a minimum of 1 item, containing unique items. Each item in 'fields' is an object with 'name' (a string, required) and 'type' (a string, required). No additional properties are allowed at the root level.
