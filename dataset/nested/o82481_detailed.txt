The schema describes an edge in a graph model. It is an object type. The main properties of the schema are: 'from_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and is required. 'to_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and is required. 'multiplicity' is a string with a default value of 'multi' and allows the following enum values: 'unknown', 'multi', 'simple', 'many2one', 'one2many', 'one2one'. 'created_at' is a string that must match the pattern '^[1-9]{1}[0-9]{12,15}$'. No additional properties are allowed at the root level.
