The schema describes a JSON schema, which is referred by ext.json. It is an object type. The main properties of the schema are: 'annotations' is an object arbitrary string key/value pairs. It allows any string as a key with a value of type string. 'apiversion' is a string version of the type. It is required. 'category' is a string three categories of entities. It is required and must be one of the following values: 'artifact', 'execution', 'container'. 'id' is a string unique identifier for this entity. It is required. 'kind' is a string type of this entity. It is required. 'name' is a string name of this entity. It is required. 'namespace' is a string namepace of the entity type. It is required. No additional properties are allowed at the root level.
