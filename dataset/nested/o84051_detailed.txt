The schema describes configuration settings for a system. It is an object type. The main properties of the schema are: 'metrics' is an object that configures the metrics exporter, containing an optional boolean field 'enabled'; 'persistence' is an object detailing the persistent volume size, containing an optional string field 'size' with specific slider values ranging from 1 to 100 Gi; 'postgresqlPassword' is a string representing the password for PostgreSQL, marked as optional; 'postgresqlUsername' is a string representing the admin user for PostgreSQL, marked as optional; 'replication' is an object that details replication settings, containing an optional boolean field 'enabled' and an optional integer field 'slaveReplicas', which is hidden based on the condition 'replication.enabled'; 'resources' is an object specifying required resources, containing an optional object 'requests' which includes an optional string field 'cpu' with slider values ranging from 10 to 2000 m and an optional string field 'memory' with slider values ranging from 10 to 2048 Mi; 'volumePermissions' is an object that manages volume permissions, containing an optional boolean field 'enabled'. No additional properties are allowed at the root level.
