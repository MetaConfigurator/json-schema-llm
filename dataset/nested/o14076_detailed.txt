The schema describes control network modes. It is an object type. The main properties of the schema are: [configuration] is an object that defines the network configuration. It is required and contains the following properties: [mode] is a string that determines the network mode, which must be either "HOME" or "HOTSPOT", with a default value of "HOME". [hostname] is a string representing the server's hostname, with a default value of "raspberrypi", and must have a minimum length of 4 characters. [ssid] is a string for the hotspot name, with a default value of "freeboard", and must also have a minimum length of 4 characters. [password] is a string for the hotspot password, with a default value of "raspberrypi", and must have a minimum length of 8 characters. [roam] is an object that defines hotspots to connect with, and is required. It includes the property [networks], which is an array of objects, each representing a wifi network to auto-connect, and must have the required fields [ssid] and [password], both of which are strings with a minimum length of 1 character. No additional properties are allowed at the root level.
