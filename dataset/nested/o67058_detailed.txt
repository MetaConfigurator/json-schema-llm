The schema describes a request-response structure. It is an object type. The main properties of the schema are: 'request' is an object that is required and defines a request structure. It requires 'method', 'url', and 'headers'. The 'method' property is a string that must be one of the following values: 'GET', 'POST', 'PUT', 'DELETE'. The 'url' property is a string that must conform to a URI format. The 'headers', 'params', and 'body' properties are objects but are optional. 'response' is an object that is required and defines a response structure. It requires 'headers'. The 'headers' property is an object, and 'body' is optional with no additional constraints. No additional properties are allowed at the root level.
