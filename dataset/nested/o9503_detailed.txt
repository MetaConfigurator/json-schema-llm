The schema describes a custom field used by admissions staff to store arbitrary data about an applicant. It is an object type. The main properties of the schema are: 'custom_fields' is an array of objects representing custom fields, where each object includes 'field_type', 'id', 'label', and 'options'. 'href' is a string hypertext reference to this resource. The 'custom_fields' array must not contain any additional properties. Each 'field_type' must be one of the following: 'boolean', 'number', 'date', 'string', 'select'. The 'id' is an integer representing a unique identifier for the custom field. The 'label' is a string that provides the human-readable name of the custom field. The 'options' property is either an array of strings when 'field_type' is 'select' or null otherwise. No additional properties are allowed at the root level.
