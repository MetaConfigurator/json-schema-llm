The schema describes an object representing a configuration. It is an object type. The main properties of the schema are: 'version' is an integer that is required. 'id' is a string that is required. 'ports' is an array of objects that is required, where each object must contain: 'pin' as an integer, 'label' as a string, and 'defaultDuration' as an integer. 'switches' is an array of objects that is required, where each object must contain: 'label' as a string, 'enabled' as a boolean, 'size' as an integer, and 'states' as an array of objects, where each object within must contain: 'label' as a string and three properties: 'portStates', 'otherSwitchEnables', and 'otherSwitchStates', which are all objects. 'sensors' is an array that is required but may contain any type of items. No additional properties are allowed at the root level.
