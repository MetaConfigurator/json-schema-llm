The schema describes a structure for retrieving data descriptors. It is an object type. It defines several reusable components under 'definitions': 'Datum' is an object type, which requires a 'realms' property. 'Realms' is an object type with additional properties representing other objects and requires at least one property. 'realm' is an object type that requires 'category', 'dimensions', 'metrics', and 'text' properties, with additional properties prohibited. 'dimensions' is an object type requiring at least one property with additional properties being objects. 'metrics' is an object type requiring at least one property with additional properties being objects. 'dimension' is an object type that requires 'info' and 'text' properties, with additional properties prohibited. 'metric' is an object type that requires 'info', 'std_err', and 'text' properties, with additional properties prohibited. The main properties of the schema are: 'totalCount' is an integer and required. 'data' is an array of items that must conform to the 'Datum' definition and is required. No additional properties are allowed at the root level.
