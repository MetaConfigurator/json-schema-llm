The schema describes printer configuration settings. It is an object type. The main properties of the schema are: 'AllowLocalPrinters' is a boolean indicating whether printers that connect directly to the user's computer are allowed. 'DefaultPrinter' is an object representing the default printer configuration, containing properties 'DeviceURI' (string) and 'DisplayName' (string). 'FooterFontName' is a string with a default value of 'Courier', describing the name of the font used for the footer. 'FooterFontSize' is a string with a default value of '7', describing the font size of the footer. 'PayloadDescription' is a string with a default value of 'Printer Settings', describing the payload. 'PayloadDisplayName' is a string with a default value of 'Printer Settings', naming the payload. 'PayloadIdentifier' is a string with a default value of 'com.apple.mcxprinting', acting as a unique identifier for the payload. 'PayloadOrganization' is a string describing the issuing organization of the profile. 'PayloadType' is a string with a default value of 'com.apple.mcxprinting', indicating the type of the payload. 'PayloadUUID' is a string with a default value of '', serving as a unique identifier for the payload, conforming to a specified pattern. 'PayloadVersion' is a number with a default value of 1, representing the version of the configuration profile. 'PrintFooter' is a boolean indicating if a page footer with user name and date will be printed. 'PrintMACAddress' is a boolean indicating whether the MAC address of the user's computer should be included in the footer. 'RequireAdminToAddPrinters' is a boolean specifying that only administrators may modify the printer list. 'RequireAdminToPrintLocally' is a boolean stating that an administrator password is needed to print to locally connected printers. 'ShowOnlyManagedPrinters' is a boolean indicating that only printers managed by profiles should be shown. 'UserPrinterList' is an object representing a list of printers to manage, including a 'QueueName' property, which is an object requiring properties such as 'DeviceURI', 'DisplayName', 'Location', 'Model', 'Option', and 'PPDURL', with 'PrinterLocked' defaulting to false, indicating whether the printer requires an administrator password. No additional properties are allowed at the root level.
