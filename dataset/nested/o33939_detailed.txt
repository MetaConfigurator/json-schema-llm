The schema describes an extension that allows an issuer to issue an Open Badge on the blockchain and provide proof of inclusion in a blockchain transaction. It is an object type. It defines several reusable components under 'definitions': 'JsonLdType' is a oneOf type, representing a type or an array of types defined in a JSON-LD context file, which can either be a string or an array of strings. The main properties of the schema are: 'type' is a JsonLdType, required; 'merkleRoot' is a string, required; 'targetHash' is a string, required; 'proof' is an array of objects, each containing 'right' and 'left' as strings, required; 'anchors' is an array of objects, each containing 'sourceId' and 'type' as strings, required. No additional properties are allowed at the root level.
