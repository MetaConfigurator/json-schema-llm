The schema describes load balancer parameters. It is an object type. The main properties of the schema are: 'allowboundsvcremoval' is a string that specifies whether bounds service removal is allowed and can be 'ENABLED' or 'DISABLED'. 'builtin' is an object that represents built-in parameters and is read-only. 'consolidatedlconn' is a string indicating whether consolidated load connection is enabled, with allowed values of 'YES' or 'NO'. 'cookiepassphrase' is a string representing the passphrase for cookies. 'dbsttl' is an integer defining the default database time-to-live. 'feature' is a string that denotes the load balancer feature in use, with numerous allowed values including 'WL', 'WebLogging', 'SP', and many others. 'httponlycookieflag' is a string that indicates whether HTTP-only cookie flag is enabled, with values of 'ENABLED' or 'DISABLED'. 'maxpipelinenat' is an integer representing the maximum NAT pipeline. 'monitorconnectionclose' is a string that specifies the behavior for connection closure, allowing 'RESET' or 'FIN'. 'monitorskipmaxclient' is a string indicating whether to skip the max client setting, with options 'ENABLED' or 'DISABLED'. 'preferdirectroute' is a string that determines if a direct route is preferred, with 'YES' or 'NO' as valid values. 'retainservicestate' is a string that indicates whether to retain service state, with values of 'ON' or 'OFF'. 'sessionsthreshold' is a read-only integer representing the session threshold. 'startuprrfactor' is an integer for the startup round-robin factor. 'useencryptedpersistencecookie' is a string that specifies if encrypted persistence cookies are used, with 'ENABLED' or 'DISABLED' as options. 'useportforhashlb' is a string indicating if the port should be used for hash load balancing, with 'YES' or 'NO'. 'usesecuredpersistencecookie' is a string that specifies whether secured persistence cookies are used, allowing 'ENABLED' or 'DISABLED'. 'vserverspecificmac' is a string that indicates if a virtual server specific MAC is enabled, with values of 'ENABLED' or 'DISABLED'. No additional properties are allowed at the root level.
