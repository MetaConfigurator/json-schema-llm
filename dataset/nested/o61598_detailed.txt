The schema describes a configuration profile for managing App Store settings. It is an object type. The main properties of the schema are: 'DisableSoftwareUpdateNotifications' is a boolean that indicates whether to disable software update notifications; 'PayloadDescription' is a string that describes the payload, with a default value of 'App Store'; 'PayloadDisplayName' is a string for the name of the payload, also with a default value of 'App Store'; 'PayloadIdentifier' is a string that serves as a unique identifier for the payload (default: 'com.apple.appstore'); 'PayloadOrganization' is a string describing the issuing organization of the profile; 'PayloadType' is a string indicating the type of the payload, with a default value of 'com.apple.appstore'; 'PayloadUUID' is a string acting as a unique identifier for the payload, formatted as '01234567-89AB-CDEF-0123-456789ABCDEF', with a pattern constraint; 'PayloadVersion' is a number indicating the version of the entire configuration profile, with a default value of 1; 'restrict-store-disable-app-adoption' is a boolean indicating whether to disable app adoption by users; 'restrict-store-mdm-install-softwareupdate-only' is a boolean that restricts app installations to MDM-installed apps and software updates; 'restrict-store-require-admin-to-install' is a boolean that restricts app installations to admin users; 'restrict-store-softwareupdate-only' is a boolean that restricts app installations to software updates only. No additional properties are allowed at the root level.
