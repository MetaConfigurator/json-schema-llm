The schema describes a Bezier curve configuration. It is an object type. The main properties of the schema are: 'c' is a boolean that indicates whether the curve is closed, and it is optional. 'i' is an array representing the In points of the Bezier curve, containing 2-dimensional arrays of numbers, and it is optional. Each item must have exactly 2 numbers. 'o' is an array representing the Out points of the Bezier curve, similarly structured as 'i', and it is optional. 'v' is an array representing the Vertices of the Bezier curve, also with the same structure as 'i' and 'o', and it is optional. No additional properties are allowed at the root level.
