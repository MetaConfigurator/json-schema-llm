The schema describes Fibaro Home Center Settings Info JSON Schema. It is an object type. The main properties of the schema are: 'serialNumber' is a string Home Center Serial Number, with a maximum length of 10 characters. 'mac' is a string Home Center MAC Address, following the pattern '^[A-Za-z0-9]{2}(:[A-Za-z0-9]{2}){5}$'. 'softVersion' is a string Home Center Software Version, with a maximum length of 10 characters. 'beta' is a boolean Home Center Beta Software Flag. 'zwaveVersion' is a string Home Center ZWave Version, with a maximum length of 10 characters. 'serverStatus' is an integer Home Center Server Timestamp. 'defaultLanguage' is a string Default Home Center Language, following the pattern '^[a-z]{2}$'. 'sunsetHour' is a string Sunset Hour, following the pattern '^[0-9]{2}:[0-9]{2}$'. 'sunriseHour' is a string Sunrise Hour, following the pattern '^[0-9]{2}:[0-9]{2}$'. 'hotelMode' is a boolean Hotel Mode Flag. 'updateStableAvailable' is a boolean Stable Image Update Availability Flag. 'temperatureUnit' is a string Default temperature units, with a maximum length of 1 character and a minimum length of 1 character. 'updateBetaAvailable' is a boolean Beta Image Update Availability Flag. 'batteryLowNotification' is a boolean Low Battery Notification Flag. 'newestStableVersion' is a string Newest Stable Version. 'newestBetaVersion' is a string Newest Beta Version. No additional properties are allowed at the root level.
