The schema describes an invocation manifest for MRIQC: No-reference image quality metrics for quality assessment of MRI. It is an object type. The main properties of the schema are: 'config' is an object containing the configuration settings for MRIQC and is required. It has properties: 'measurement' is a string with a default value of 'auto-detect' and must be one of the following: 'auto-detect', 'functional', 'T1', 'T2'; 'save_derivatives' is a boolean with a default value of false and is required; 'save_outputs' is a boolean with a default value of false and is required; 'verbose_reports' is a boolean with a default value of false and is required; 'include_rating_widget' is a boolean with a default value of false and is required. 'inputs' is an object that contains the input specifications for MRIQC and is required. It has properties: 'nifti' is an object containing the input type and is required, with the property 'type' being an enum that must be 'nifti'. No additional properties are allowed at the root level.
