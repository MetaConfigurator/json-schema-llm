The schema describes an eventsource configuration. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'maxInFlight' is an integer that specifies the message buffer, with a default value of 1000. 'name' is a string that is required. 'source' is an object that is required. 'streamKey' is a string that is optional. 'tags' is a string that is optional. 'targetGate' is a string that specifies the gate address, with a default value of 'akka.tcp://hub@localhost:12345/user/gatename' and is required. No additional properties are allowed at the root level.
