The schema describes an invocation manifest for MRIQC: NR-IQMs for Functional MRI (mriqc v0.9.0-0). It is an object type. The main properties of the schema are: 'inputs' is an object that is required and contains 'nifti', which is also an object that is required and has a property 'type' that is an enum with the allowed value 'nifti'. 'config' is an object that is required but has no properties defined. No additional properties are allowed at the root level.
