The schema describes uploaded file metadata entity. It is an object type. The main properties of the schema are: 'id' is a string that represents a UUID and is optional. 'sourcePath' is a string that denotes the path to the file location and is read-only. 'name' is a string that specifies the name of the file with extension and is required. 'jobExecutionId' is a string that links to JobExecution entity and is optional. 'uploadDefinitionId' is a string that links to UploadDefinition entity and is optional. 'loaded' is a boolean that indicates whether the file is loaded and is optional. 'createDate' is a string in date-time format that indicates when the file definition was created and is optional. 'uploadedDate' is a string in date-time format that indicates when the file was uploaded and is optional. No additional properties are allowed at the root level.
