The schema describes an image block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string in date-time format representing an update that occurred on the node this block is attached to. It is optional. 'aside' is a boolean indicating whether the block represents content only indirectly related to the main content. It is optional. 'node_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and is required. 'title' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'caption' is a string with a minimum length of 0 and a maximum length of 5000. It is optional. 'is_nsfw' is a boolean that is optional. 'url' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$'. It is optional. 'launch_text' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'aspect_ratio' is a string that must be one of the following values: 'unknown', 'auto', '1by1', '2by3', '3by2', '3by4', '4by3', '4by5', '5by4', '9by16', '16by9', 'original', 'custom'. It is optional. 'fallback_src_url' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$' and may be necessary for storing old URLs for imported image blocks. It is optional. No additional properties are allowed at the root level.
