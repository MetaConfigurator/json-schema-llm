The schema describes Azure credentials. It is an object type. The main properties of the schema are: 'credentials' is an object that contains credentials for both Azure Batch and Storage accounts. It is required. The 'credentials' property defines two required objects: 'batch' and 'storage'. The 'batch' object is also an object that has three required properties: 'account', which is a string describing the Batch Account Name; 'account_key', which is a string describing the Batch Account Key; and 'account_service_url', which is a string describing the Batch Account Service Url. The 'storage' object is an object that has three required properties: 'account', which is a string describing the Azure Storage Account Name; 'account_key', which is a string describing the Azure Storage Account Key; and 'endpoint', which is a string and must be equal to 'core.windows.net'. No additional properties are allowed at the root level.
