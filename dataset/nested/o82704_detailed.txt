The schema describes a Tumblr post block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$', 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML, 'updated_date' is a string formatted as date-time representing the update date of the node this block is attached to, 'aside' is a boolean that indicates if this block represents a portion of a document only indirectly related to the main content, 'href' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$', and is required, and 'canonical_url' is a string that must also match the same pattern as 'href'. No additional properties are allowed at the root level.
