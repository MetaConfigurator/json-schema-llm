The schema describes a VPN clientless access policy. It is an object type. The main properties of the schema are: 'builtin' is an object that must not contain any additional properties, with allowed values being 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', or 'PARTITION_ALL'. 'description' is a string that is readonly. 'feature' is a string that is readonly with allowed values including 'WL', 'WebLogging', 'SP', 'SurgeProtection', and many more. 'hits' is an integer that is readonly. 'isdefault' is a boolean that is readonly. 'name' is a string. 'profilename' is a string. 'rule' is a string. 'undefaction' is a string that is readonly. 'undefhits' is an integer that is readonly. No additional properties are allowed at the root level.
