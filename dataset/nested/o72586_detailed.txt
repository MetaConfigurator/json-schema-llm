The schema describes an Article. It is an object type. The main properties of the schema are: 'categories' is an array of strings that is optional. 'furtherReading' is an object that may contain properties where each is a string formatted as a URI. 'id' is a string that must match the pattern '^[a-z0-9-]+$' and is required. 'keywords' is an array of strings that is optional, with each string having a minimum length of 1. 'related' is an array of strings that is optional, with each string having a minimum length of 1. 'tags' is an array of strings that is optional, allowing values of 'inaccurate', 'incomplete', 'proprietary', and 'not-a-number'. 'title' is a string and is required. 'visible' is a boolean and is required. No additional properties are allowed at the root level.
