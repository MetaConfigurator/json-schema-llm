The schema describes a configuration of the sprite animation. It is an object type. The main properties of the schema are: 'end' is a number that describes the index of the ending frame. It is optional. 'frameRate' is a number that describes the number of frames per second to use for this animation, modifying the speed of the animation. It is optional and has a default value of 30. 'loop' is a boolean that describes whether the sprite animation will loop when set to true. It is optional and has a default value of true. 'name' is a string that describes the name of the animation. It is required. 'start' is a number that describes the index of the starting frame. It is optional. No additional properties are allowed at the root level.
