The schema describes a subscription. It is an object type. The main properties of the schema are: 'Name' is a string that represents the name of a subscription. It is required and must have a minimum length of 1 and a maximum length of 80. 'SubscriberId' is a string that represents the userid of the subscriber. It is required and must have a minimum length of 1 and a maximum length of 30. 'EmailAddress' is a string that represents the email address of the subscriber. It is required and must have a minimum length of 1 and a maximum length of 255. 'CollectionConceptId' is a string that represents the collection concept id of the granules subscribed. It is required and must have a minimum length of 1 and a maximum length of 255. 'Query' is a string that represents the search query for the granules that matches the subscription. It is required and must have a minimum length of 1 and a maximum length of 40000. No additional properties are allowed at the root level.
