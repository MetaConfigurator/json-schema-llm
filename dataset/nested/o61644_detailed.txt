The schema describes a configuration profile for Universal Access settings. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string describing the payload, optional with a default value of 'Universal Access Settings'. 'PayloadDisplayName' is a string representing the name of the payload, optional with a default value of 'Universal Access Settings'. 'PayloadIdentifier' is a string that acts as a unique identifier for the payload, optional with a default value of 'com.apple.universalaccess'. 'PayloadOrganization' is a string indicating the issuing organization of the profile, optional. 'PayloadType' is a string for the type of payload, optional with a default value of 'com.apple.universalaccess'. 'PayloadUUID' is a string that serves as a unique identifier for the payload, formatted as '01234567-89AB-CDEF-0123-456789ABCDEF', optional with a default value of ''. 'PayloadVersion' is a number representing the version of the whole configuration profile, optional with a default value of 1. 'closeViewHotkeysEnabled' is a boolean, optional. 'closeViewScrollWheelToggle' is a boolean, optional. 'flashScreen' is a boolean indicating whether the screen should flash when an alert sound occurs, optional. 'grayscale' is a boolean indicating whether to use grayscale, optional. 'mouseDriver' is a boolean allowing control of the mouse pointer using the keyboard number pad, optional. 'mouseDriverCursorSize' is a number indicating mouse cursor size, optional. 'slowKey' is a boolean adjusting the time between key press and activation, optional. 'stickyKey' is a boolean allowing modifier keys to be set without holding them down, optional. 'voiceOverOnOffKey' is a boolean providing spoken and brailled descriptions of items on the screen, optional. 'whiteOnBlack' is a boolean for inverting colors, optional. No additional properties are allowed at the root level.
