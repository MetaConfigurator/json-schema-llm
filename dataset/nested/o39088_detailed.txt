The schema describes generating a translation from a TypeScript interface. It is an object type. The main properties of the schema are: '$schema' is a string, optional; '$tool' is a string, required, must be set to 'ts-trans'; 'dist' is a string, required, the output root path (relative to the pwd); 'langs' is an array, required, containing at least one item, where each item is a string that matches the pattern '^[a-z]{2}(-[A-Z]{2})?$'; 'pipePrefix' is a string, required, with a default value of ''; 'src' is an array, required, containing at least one item, where each item is a string; 'tag' is a string, required, with a default value of 'tr'. No additional properties are allowed at the root level.
