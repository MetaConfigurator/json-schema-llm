The schema describes a ballot for ClearElection. It is an object type. The main properties of the schema are: 'clearelection-ballot' is a string with an enum of allowed values: 'v0.0', required. 'ballotId' is a string, required. 'uniquifier' is a string, required. 'contests' is an array of objects, required. Each object in 'contests' must have 'contestId' as a string and 'choices' as an array of objects, where each object must contain 'candidateId' as a string, specified as required. No additional properties are allowed at the root level.
