The schema describes a record for a facility round. It is an object type. 'deliveryPeriod' is a string representing the delivery time period and is optional. 'departedAt' is a string representing the date and time when the event departed, and it is required. 'doc_type' is a string with the pattern '^facilityRound$', and it is required. 'facility' is a string representing the facility name and is optional. 'note' is a string for any additional notes and is optional. 'orderID' is a string representing the order identifier and is required. 'pickedProducts' is an array that must contain specified items and is required. 'status' is a string representing the current status and is optional. 'survey' is an array for survey-related data and is optional. 'version' is a string with the format 'semver', and it is required. No additional properties are allowed at the root level.
