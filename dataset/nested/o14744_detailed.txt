The schema describes a configuration for daily and monthly cheer messages. It is an object type. It defines several reusable components under 'definitions': 'cheer' is an object type, representing a cheer message with required fields: 'channel', 'message', and 'userstate'. The 'userstate' property is also an object with required fields: 'bits' (a number) and 'display-name' (a string). The main properties of the schema are: 'daily' is an object representing the daily cheer message with required fields: 'channel', 'message', and 'userstate', and 'monthly' is an object representing the monthly cheer message with required fields: 'channel', 'message', and 'userstate'. No additional properties are allowed at the root level.
