The schema describes devices. It is an object type. The main properties of the schema are: 'name' is a string required field. 'privateIps' is an array of strings, each matching the pattern '^([1-9]{1,3}\.){3}[1-9]{1,3}$', optional. 'publicIps' is an array of strings, each matching the pattern '^([1-9]{1,3}\.){3}[1-9]{1,3}$', optional. 'os' is an object containing properties 'code' (a string) and 'name' (a string), with no additional properties allowed, optional. 'location' is an object containing properties 'countryCode' (a string or null), 'text' (a string), and 'countryName' (a string or null), with no additional properties allowed, optional. 'tags' is an array of strings formatted as 'mongoId', or it can be null, optional. No additional properties are allowed at the root level.
