The schema describes a base type for select ALPS elements. It is an object type. The main properties of the schema are: 'links' is an array of objects identifying a link between an ALPS element and some other (possibly external) resource. Each object is required to have 'rel' and 'href' properties, which are both strings. 'descriptors' is an array describing the data and transition definitions, with no specific properties required. 'doc' is an object that provides a description of the element, including properties 'format', which is an enum restricted to 'text', 'html', 'asciidoc'; 'href', a string; and 'content', a string. 'ext' is an object defining an author-specific extension, with a required 'id' property being a string and optional properties 'href', a string, and 'value', a string. No additional properties are allowed at the root level.
