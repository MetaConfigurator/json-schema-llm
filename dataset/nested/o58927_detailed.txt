The schema describes album info. It is an object type. It defines several reusable components under 'definitions': 'tag' is an object, it has a required field 'name' of type string and no additional properties are allowed. 'track' is an object, it has a required field 'name' of type string, an optional field 'duration' of type string, an optional field 'url' of type string with a format of uri, and no additional properties are allowed. The main properties of the schema are: 'artist' is a string and required. 'imageUrl' is a string with a format of uri and optional. 'raw' is an object and required. 'tags' is an array of items that reference the 'tag' definition and optional. 'title' is a string and required. 'tracks' is an array of items that reference the 'track' definition and required. 'url' is a string with a format of uri and required. No additional properties are allowed at the root level.
