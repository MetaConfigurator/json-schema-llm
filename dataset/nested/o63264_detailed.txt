The schema describes a Ghost Post. It is an object type. The main properties of the schema are: 'author' is a number representing the author of the post, 'created_at' is a string indicating when the post was created, 'created_by' is a number denoting who created the post, 'featured' is a boolean that specifies if the post is featured, 'html' is a string containing the HTML content of the post, 'id' is a number that uniquely identifies the post, 'image' is a string for the image URL, 'language' is a string indicating the language of the post, 'markdown' is a string for the Markdown content of the post, 'meta_description' is a string providing a meta description, 'meta_title' is a string for the meta title, 'page' is a boolean indicating if the post is a page, 'published_at' is a string showing when the post was published, 'published_by' is a number for who published the post, 'slug' is a string for the post's slug, 'status' is a string with an enum value of 'published', 'title' is a string for the post's title, 'updated_at' is a string indicating when the post was last updated, 'updated_by' is a number for who updated the post, 'url' is a string for the post's URL, 'uuid' is a string for the post's UUID. No additional properties are allowed at the root level.
