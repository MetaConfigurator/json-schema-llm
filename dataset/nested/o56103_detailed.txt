The schema describes a recipe. It is an object type. The main properties of the schema are: 'info' is an object that contains the required properties 'title' and 'description', and optional properties 'id', 'category', 'user', 'cook_time', 'date', and 'image'. 'ingredients' is an array of objects, each containing the required properties 'ingredient' and 'qty'. 'directions' is an array of objects, each containing the required properties 'id', 'direction', and 'step'. No additional properties are allowed at the root level.
