The schema describes the configuration for visual variables specific to authoringInfo. It is an object type. The main properties of the schema are: 'endTime' is a number or a string indicating a Unix stamp, which can be either 'startTime' or 'endTime' and must have different names if both are fields; 'field' is a string specifying the attribute field chosen in the Smart Mapping gallery, which must match either 'startTime' or 'endTime'; 'maxSliderValue' is a number indicating the maximum value displayed; 'minSliderValue' is a number indicating the minimum value displayed; 'startTime' is a number or a string indicating a Unix time stamp, which can be either 'startTime' or 'endTime' and must have different names if both are fields; 'style' is a string used for comparison rendering, with allowed values 'percent', 'ratio', and 'percentTotal'; 'theme' is a string for colorInfo visual variables with allowed values 'high-to-low', 'above-and-below', 'centered-on', and 'extremes'; 'type' is a string specifying the renderer's visual variable with allowed values 'colorInfo', 'sizeInfo', 'transparencyInfo', and 'rotationInfo'; 'units' is a string with allowed values 'seconds', 'minutes', 'hours', 'days', 'months', and 'years' applicable only to age renderers. No additional properties are allowed at the root level.
