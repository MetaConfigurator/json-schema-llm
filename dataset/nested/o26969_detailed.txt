The schema describes a keyframe segment configuration. It is an object type. The main properties of the schema are: 's' is an array that represents the start value of the keyframe segment. 't' is a number that indicates the start time of the keyframe segment. 'i' is an object describing the Bezier curve interpolation in value, which includes 'x', an array representing the bezier x axis, and 'y', an array representing the bezier y axis. 'o' is an object describing the Bezier curve interpolation out value, which includes 'x', an array representing the bezier x axis, and 'y', an array representing the bezier y axis. No additional properties are allowed at the root level.
