The schema describes AuthoringInfo Visual Variable. It is an object type. The main properties of the schema are: 'endTime' is a number or string A Unix stamp. Both `startTime` or `endTime` can be fields. If this is the case, their names must be different. 'field' is a string The attribute field the user chose in the Smart Mapping gallery. Must be the same as in either `startTime` or `endTime`. 'maxSliderValue' is a number A numeric value indicating the maximum value displayed. 'minSliderValue' is a number A numeric value indicating the minimum value displayed. 'startTime' is a number or string A Unix time stamp. Both `startTime` or `endTime` can be fields. If this is the case, their names must be different. 'style' is a string (This property is used for comparison rendering). It is used to map the ratio between two numbers. It is possible to express that relationship as percentages, simple ratios, or an overall percentage. It must be one of: 'percent', 'percentTotal', 'ratio'. 'theme' is a string Theme to be used only when working with visual variables of type `colorInfo`. Default is `high-to-low`. It must be one of: 'above-and-below', 'centered-on', 'extremes', 'high-to-low'. 'type' is a string A string value specifying the type of renderer's visual variable. It must be one of: 'colorInfo', 'rotationInfo', 'sizeInfo', 'transparencyInfo'. 'units' is a string (This property is used only with age renderers.) Units for startTime and endTime. It must be one of: 'days', 'hours', 'minutes', 'months', 'seconds', 'years'. No additional properties are allowed at the root level.
