The schema describes the options of a media object. It is an object type. The main properties of the schema are: 'autoPause' is a boolean that indicates whether the video needs to be paused automatically when the page is not visible anymore. It is optional and defaults to true. 'autoPlay' is a boolean that indicates whether the video needs to be played automatically at start. It is optional and defaults to false. 'autoResume' is a boolean that indicates whether the video needs to be resumed automatically when the page is visible again. It is optional and defaults to false. 'loop' is a boolean that indicates whether the video needs to be looped automatically at the end of playing. It is optional and defaults to false. 'startTime' is a number that sets the current position (in seconds) of the video playback. It is optional and defaults to 0. 'volume' is a number that represents the volume of the media, with a minimum value of 0 and a maximum value of 1. It is optional and defaults to 1. No additional properties are allowed at the root level.
