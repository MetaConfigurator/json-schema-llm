The schema describes a transform profile action binding. It is an object type. The main properties of the schema are: 'actioncomment' is a string that is readonly. 'actionname' is a string that is optional. 'cookiedomainfrom' is a string that is readonly. 'cookiedomaininto' is a string that is readonly. 'name' is a string that is optional. 'priority' is an integer that is readonly. 'profilename' is a string that is readonly. 'requrlfrom' is a string that is readonly. 'requrlinto' is a string that is readonly. 'resurlfrom' is a string that is readonly. 'resurlinto' is a string that is readonly. 'state' is a string that is readonly with allowed values: 'ENABLED', 'DISABLED'. No additional properties are allowed at the root level.
