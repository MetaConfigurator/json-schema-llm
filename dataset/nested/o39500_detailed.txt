The schema describes a currency. It is an object type. The main properties of the schema are: 'code' is a string with a maximum length of 4, required. 'id' is a string formatted as a UUID, required. 'label' is a string with a maximum length of 64, required. 'prefix' is a string with a maximum length of 1, optional. 'separator' is an object that requires 'thousands' and 'decimal' properties, both of which are strings with a maximum length of 1 and a minimum length of 1. No additional properties are allowed at the root level.
