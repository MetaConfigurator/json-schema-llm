The schema describes a configuration profile for a PKCS12 certificate. It is an object type. The main properties of the schema are: 'Password' is a string for the password protecting the PKCS12 file. 'PayloadCertificateFileName' is a string for the filename for the certificate. 'PayloadContent' is a string that is mandatory. It represents the base64 representation of the payload with a line length of 52. 'PayloadDescription' is a string with a default value of 'Adds a PKCS#12-formatted certificate' for the description of the payload. 'PayloadDisplayName' is a string with a default value of 'Certificate' for the name of the payload. 'PayloadIdentifier' is a string with a default value of 'com.apple.security.pkcs12', which is a unique identifier for the payload, dot-delimited. 'PayloadOrganization' is a string that describes the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string with a default value of 'com.apple.security.pkcs12', representing the type of the payload, a reverse DNS string. 'PayloadUUID' is a string with a default value of '', which is a unique identifier for the payload (format 01234567-89AB-CDEF-0123-456789ABCDEF) and must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is a number with a default value of 1, representing the version of the whole configuration profile. No additional properties are allowed at the root level.
