The schema describes a truck details collection. It is an object type. The main properties of the schema are: 'truckDetails' is an array of objects containing detailed information about trucks. Each object must include the following required fields: 'objectId', 'address', 'applicant', 'block', 'blocklot', 'cnn', 'facilityType', 'fooditems', 'latitude', 'location', 'locationDescription', 'longitude', 'lot', 'permit', 'priorpermit', 'received', 'schedule', and 'status'. The 'location' property is an object that requires the fields 'type' and 'coordinates', where 'coordinates' is an array of integers. No additional properties are allowed at the root level.
