The schema describes PawBlock's browser storage state. It is an object type. The main properties of the schema are: 'on' is a boolean indicating whether or not the rules should be applied. 'rules' is an array of unique items containing objects, where each object has properties 'domain' (a string) and 'path' (a string), with no additional properties allowed within each object. 'storageVersion' is a string used to make schema changes easier in the future. 'allowedTabId' is an integer representing the tab that should not enforce a block. 'blockType' is a string that indicates the strictness of the block page, with allowed values being 'hard' and 'soft'. 'softBlockDelay' is an integer specifying the number of seconds that the user must wait before continuing, with a minimum of 0. 'reenableMinutes' is either an integer or null, indicating the number of minutes to wait before automatically turning PawBlock back on after turning it off, with a minimum of 1. No additional properties are allowed at the root level.
