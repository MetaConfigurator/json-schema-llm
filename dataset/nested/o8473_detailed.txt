The schema describes the process of activating RMA (Return Merchandise Authorization) for devices. It is an object type. The main properties of the schema are: 'fabric_uuid' is a string that represents the Fabric UUID, which is optional. 'rma_devices' is an array that contains items that are objects; each object describes a list of devices and corresponding serial numbers to RMA. Each object in 'rma_devices' contains the following properties: 'device_uuid' is a string formatted as a UUID and is required, and 'serial_number' is a string that is also required. No additional properties are allowed at the root level.
