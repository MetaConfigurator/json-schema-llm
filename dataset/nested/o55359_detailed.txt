The schema describes a TSIG Key. It is an object type. The main properties of the schema are: 'tsigkey' is an object representing the TSIG Key, which is required. It contains the following required properties: 'name' is a string with a maximum length of 255, 'algorithm' is a string with an enum of allowed values: 'hmac-md5', 'hmac-sha1', 'hmac-sha224', 'hmac-sha256', 'hmac-sha384', 'hmac-sha512', 'secret' is a string with a maximum length of 255, 'scope' is a string with an enum of allowed values: 'POOL', 'ZONE', and 'resource_id' is a string formatted as a uuid. The 'created_at', 'id', and 'updated_at' properties are read-only and formatted as date-time and uuid respectively. The 'links' property is an object with a read-only 'self' property formatted as a url. No additional properties are allowed at the root level.
