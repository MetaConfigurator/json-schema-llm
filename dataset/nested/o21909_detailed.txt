The schema describes a configuration for Ember CLI. It is an object type. The main properties of the schema are: 'port' is a number representing the server port (4200 by default), optional with a default value of 4200. 'host' is a string that listens on all interfaces by default, optional. 'proxy' is a string representing the server proxy, optional. 'insecure-proxy' is a boolean indicating whether to set false to proxy self-signed SSL certificates (default is false), optional with a default value of false. 'watcher' is a string representing the watcher type (default is 'events'), optional with a default value of 'events'. 'live-reload' is a boolean enabling live reload (default is true), optional with a default value of true. 'live-reload-host' is a string that defaults to host, optional. 'live-reload-port' is a number, optional with a default value of 49152, within the range [49152...65535]. 'environment' is an enum, required for one of the values: 'development', 'production' (default is 'development'). 'output-path' is a string representing the output path (default is 'dist/'), optional with a default value of 'dist/'. 'ssl' is a boolean enabling SSL (default is false), optional with a default value of false. 'ssl-key' is a string representing the SSL key (default is 'ssl/server.key'), optional with a default value of 'ssl/server.key'. 'ssl-cert' is a string representing the SSL certificate (default is 'ssl/server.crt'), optional with a default value of 'ssl/server.crt'. No additional properties are allowed at the root level.
