The schema describes an alignment of a transcript to an audio file. It is an object type. The main properties of the schema are: 'words' is an array of objects, representing individual aligned words of the transcript and audio file. Each object in the array is required to have 'word', 'alignedWord', 'start', and 'end' properties. 'word' is a string indicating the original word, 'alignedWord' is a string for the word processed by the alignment algorithm, 'start' is a number for the start time of the aligned word in seconds, and 'end' is a number for the end time of the aligned word in seconds. The properties 'speaker' and 'line_idx' are optional, with 'line_idx' being an integer indicating the index of the input line that the word came from. Additional properties are allowed at the root level.
