The schema describes a collection of events and their metadata. It is an object type. The main properties of the schema are: 'events' is an array that contains objects, each representing an event with required fields: 'id' (a string), 'name' (a string), 'distance' (a string), 'startTime' (a string), 'timeFromNow' (an integer), 'stats' (an object), and 'venue' (an object). 'stats' is an object that contains required fields: 'attending' (an integer), 'declined' (an integer), 'maybe' (an integer), and 'noreply' (an integer). 'venue' is an object with required fields: 'id' (a string), 'name' (a string), and 'location' (an object). 'location' is an object that requires 'latitude' (a number) and 'longitude' (a number). 'metadata' is an object with required fields: 'venues' (an integer), 'venuesWithEvents' (an integer), and 'events' (an integer). No additional properties are allowed at the root level.
