The schema describes a loaded asset. It is an object type. The main properties of the schema are: 'schema' is a string that represents the schema identifier for payload. It is required and must be one of the following values: 'avalon-core:container-2.0'. 'id' is a string that serves as an identifier for finding an object in the host. It is required and must be one of the following values: 'pyblish.avalon.container'. 'objectName' is a string that names the internal object, such as the objectSet in Maya. It is required. 'loader' is a string that indicates the name of the loader plug-in used to produce this container. It is required. 'name' is a string that provides the internal object name of the container in the application. It is required. 'namespace' is a string that defines the internal namespace of the container in the application. It is required. 'representation' is a string that specifies the unique id of representation in the database. It is required. No additional properties are allowed at the root level.
