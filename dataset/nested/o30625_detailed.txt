The schema describes an ARP configuration. It is an object type. The main properties of the schema are: 'all' is a boolean optional property. 'channel' is a readonly integer property. 'controlplane' is a readonly boolean property. 'flags' is a readonly integer property. 'ifnum' is a string optional property. 'ipaddress' is a string optional property. 'mac' is a string optional property. 'nodeid' is an integer optional property. 'ownernode' is an integer optional property. 'state' is a readonly integer property. 'td' is an integer optional property. 'timeout' is a readonly integer property. 'type' is a string readonly property that must be one of the following values: 'STATIC', 'PERMANENT', 'DYNAMIC'. 'vlan' is an integer optional property. 'vtep' is a string optional property. 'vxlan' is an integer optional property. No additional properties are allowed at the root level.
