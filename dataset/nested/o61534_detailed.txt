The schema describes metadata about a project in the Mopidy ecosystem. It is an object type. The main properties of the schema are: 'bugs' is a string URL to bug tracker. It is optional. 'distribution' is an object distributions of the project. It is required. 'homepage' is a string URL to project web site. It is required. 'id' is a string unique identifier for the project. It is required. 'is_extension' is a boolean if it can be installed as a Mopidy extension. It is optional with a default value of false. 'license' is a string license identifier from http://spdx.org/licenses/. It is required. 'maintainers' is an array reference to a person, as defined in people/<person>.json. It is required with a minimum of 1 item and unique items. 'name' is a string display name. It is optional. No additional properties are allowed at the root level.
