The schema describes an invocation manifest for FreeSurfer: MBIRN Defacer for structural MRI (mri-deface v1.22). It is an object type. The main properties of the schema are: 'inputs' is an object that is required and contains a required property 'anatomical' which is also an object that includes a property 'type' which is an enum with allowed values 'nifti', 'mgh', and 'dicom'. 'config' is an object that is required and contains required properties 'output_mgh' which is a boolean with a default value of false and 'output_nifti' which is a boolean with a default value of true. No additional properties are allowed at the root level.
