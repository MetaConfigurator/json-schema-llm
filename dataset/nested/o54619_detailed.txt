The schema describes an array of objects. It is an array type. The main properties of the schema are: Each object has a 'lastChanged' which is an integer representing the last changed timestamp and is required. 'isTemporary' is a boolean indicating if the item is temporary and is required. 'title' is a string representing the title of the object and is required. 'documents' is an array of objects with each document having required properties: 'filetype' (a string), 'path' (a string), and 'name' (a string). 'lastChangedBy' is an object representing the person who last changed the item, with required properties: 'email' (a string) and 'name' (a string). 'author' is an object representing the author, also requiring 'email' and 'name', both strings. 'id' is a string serving as a unique identifier for the object and is required. 'text' is a string containing additional text and is optional. 'date' is an integer representing the date and is required. No additional properties are allowed at the root level.
