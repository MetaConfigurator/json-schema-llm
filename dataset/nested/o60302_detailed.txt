The schema describes configuration properties for Kafka-manager. It is an object type. The main properties of the schema are: 'kafka-manager' is an object Kafka-manager configuration properties, required fields are 'zk' and 'application_secret'. 'marathon-lb' is an object Kafka-manager works best when deployed on a virtual host, it has no required fields. 'service' is an object DC/OS service configuration properties, required fields are 'cpus', 'mem', and 'instances'. No additional properties are allowed at the root level.
