The schema describes ParaDrop chute VLAN interface configuration. It is an object type. The main properties of the schema are: 'type' is a string that must be 'vlan' with a pattern match of 'wifi'. It is required. 'intfName' is a string that must match the pattern '[a-z][a-z0-9]*'. It is required. 'vlan_id' is an integer with a minimum value of 1 and a maximum value of 4094. It is required. 'dhcp' is an optional object that contains properties: 'leasetime' is a string describing the duration of client leases, 'limit' is an optional integer with a minimum value of 1, 'start' is an optional integer with a minimum value of 0, and 'relay' is an optional string for a relay agent address. 'ipv4_network' is an optional string requesting a specific IP network in slash notation. 'dns' is an optional array of strings to advertise DNS servers to connected clients. No additional properties are allowed at the root level.
