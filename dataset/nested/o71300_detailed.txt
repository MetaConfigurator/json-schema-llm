The schema describes a fabric object. It is an object type. The main properties of the schema are: 'fabric_uuid' is a string representing the UUID of the fabric object created, which is optional and has a default value of an empty string. 'fabric_fq_name' is an array of strings representing the FQNAME of the fabric object created. Each item in the array is optional and has a default value of an empty string. The array can include items such as 'default-global-system-config' and 'fabric01'. The schema enforces a minimum of 1 property and a maximum of 1 property at the root level. No additional properties are allowed at the root level.
