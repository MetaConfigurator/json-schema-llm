The schema describes a configuration object. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'extensions' is an array of extension points, each item having properties 'type' (a string) and 'order' (a number). 'dependencies' is an array of strings. 'scripts' is an array of strings. 'resources' is an array of strings. 'skip_compilation' is an array of strings. 'experiment' is a string. No additional properties are allowed at the root level.
