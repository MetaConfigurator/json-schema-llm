The schema describes a configuration object for identifiers. It is an object type. The main properties of the schema are: 'identifier' is an array that contains objects with the following properties: 'alias-name' is a string, 'ip' is an array of strings, 'prefix' is an array of strings, 'port-range' is an array of objects with properties 'lower-port' (an integer) and 'upper-port' (an integer), 'traffic-protocol' is an array of integers, 'FQDN' is an array of strings, and 'URI' is an array of strings. No additional properties are allowed at the root level.
