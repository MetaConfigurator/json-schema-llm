The schema describes application settings. It is an object type. It defines several reusable components under 'definitions': 'SettingsLanguage' is a string, with allowed values: 'cs-CZ', 'de-DE', 'en-US', 'es-ES', 'fr-FR', 'it-IT', 'ja-JP', 'ko-KR', 'pl-PL', 'pt-BR', 'ro-RO', 'ru-RU', 'sr-SP', 'uk-UA', 'vi-VN', 'zh-CN'. 'SettingsTheme' is a string, with allowed values: 'dark', 'dracula', 'light', 'system'. The main properties of the schema are: 'addQueryDepthLimit' is a number, optional, describing how deep the 'Add query' functionality would go. 'alert.disableWarnings' is a boolean, optional, specifying if warning alerts should be disabled. 'disablePushNotification' is a boolean, optional, specifying if push notifications should be disabled. 'enableExperimental' is a boolean, optional, enabling experimental features in Altair. Note: The features might be unstable. 'historyDepth' is a number, optional, specifying the number of items allowed in the history pane. 'language' is a string referencing 'SettingsLanguage', optional, specifying the language e.g. 'en-US', 'fr-FR', 'ru-RU', etc. 'plugin.list' is an array of strings, optional, specifying a list of enabled plugins (requires enableExperimental to be true). 'request.withCredentials' is a boolean, optional, specifying if requests should be sent with credentials (with cookies) or not. 'schema.reloadOnStart' is a boolean, optional, specifying if the schema should be reloaded when the app starts. 'tabSize' is a number, optional, specifying the tab size in the editor. 'theme' is a string referencing 'SettingsTheme', optional, specifying the theme. Options: 'light', 'dark', 'dracula'. 'theme.editorFontFamily' is a string, optional, specifying the font family for the editors. 'theme.editorFontSize' is a number, optional, specifying the font size for the editors. 'theme.fontsize' is a number, optional, specifying the base font size (Default size - 24). No additional properties are allowed at the root level.
