The schema describes configuration for forwarding metrics from Atlas LWC to AWS CloudWatch. It is an object type. The main properties of the schema are: 'email' is a string representing the email address of the service owner. It is required and must be in email format. 'expressions' is an array of objects containing metric expressions. It is required and must contain unique items. Each item in the array is an object with the required properties: 'metricName', 'atlasUri', and 'account'. 'metricName' is a string that describes the name of the metric to be created in CloudWatch, following a specific pattern. 'atlasUri' is a string for the Atlas query, also following a specific pattern. 'account' is a string representing the AWS account to send the metric to and must match a specific pattern. The properties 'dimensions' and 'region' are optional. 'dimensions' is an array of objects, each with required properties 'name' and 'value', both following specific patterns. 'region' is a string that describes the AWS region, following a specific pattern. 'checksToSkip' is an array of strings that lists validations to skip, with each item having a minimum length of 1. No additional properties are allowed at the root level.
