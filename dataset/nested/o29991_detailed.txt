The schema describes an application firewall policy. It is an object type. The main properties of the schema are: 'builtin' is an object that is readonly and has an enum with values 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'comment' is a string that is optional. 'feature' is a string that is readonly and has an enum with multiple values including 'WL', 'WebLogging', 'SP', 'SurgeProtection', and others. 'hits' is an integer that is readonly. 'logaction' is a string that is optional. 'name' is a string that is optional. 'newname' is a string that is optional. 'policytype' is a string that is readonly and has an enum with values 'Classic Policy' and 'Advanced Policy'. 'profilename' is a string that is optional. 'rule' is a string that is optional. 'undefhits' is an integer that is readonly. No additional properties are allowed at the root level.
