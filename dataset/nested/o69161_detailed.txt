The schema describes network device discovery. It is an object type. The main properties of the schema are: 'model' is a string which is optional. 'vendor' is a string which is optional. 'serial' is a string which is optional. 'tags' is an array of strings which is optional, each item in the array is a string. 'interfaces' is an object which is optional and cannot have additional properties, it contains pattern properties where any string can be a key. Each key maps to an object with properties: 'type' which is a string, and 'ip' which is an object that has pattern properties for IP addresses and cannot have additional properties. 'router_ips' is an array of strings which is optional, each item in the array is a string. No additional properties are allowed at the root level.
