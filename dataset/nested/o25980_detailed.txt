The schema describes geometric shapes. It is an object type. It defines several reusable components under 'definitions': 'Square' is an object, which has required fields 'kind' (a string with the enum value 'square') and 'size' (a number). 'Rectangle' is an object, which has required fields 'kind' (a string with the enum value 'rectangle'), 'width' (a number), and 'height' (a number). 'Circle' is an object, which has required fields 'kind' (a string with the enum value 'circle') and 'radius' (a number). The main properties of the schema are: The schema accepts any of the following shapes: one of 'Square', 'Rectangle', or 'Circle'. No additional properties are allowed at the root level.
