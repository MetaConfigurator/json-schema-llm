The schema describes a loaded asset. It is an object type. The main properties of the schema are: 'id' is a string identifier for finding object in host, required, with allowed value 'pyblish.mindbender.container'. 'objectName' is a string name of internal object, such as the objectSet in Maya, required. 'name' is a string full name of application object, required. 'author' is a string name of the author of the published version, required. 'loader' is a string name of loader plug-in used to produce this container, required. 'families' is a string families associated with the this subset, required. 'time' is a string file-system safe, formatted time, required. 'subset' is a string name of source subset, required. 'asset' is a string name of source asset, required. 'representation' is a string name of source representation, required. 'version' is a number version number, required. 'silo' is a string silo of parent asset, required. 'path' is a string absolute path on disk, required. 'source' is a string absolute path to file from which this version was published, required. No additional properties are allowed at the root level.
