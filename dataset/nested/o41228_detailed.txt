The schema describes an invocation manifest for DCMTK: DICOM Send - storescu. It is an object type. The main properties of the schema are: 'inputs' is an object that is required and defines the input parameters, including a required property 'api_key', which is an object, and an optional property 'file', which is an object with a property 'type' that must be the enum value 'dicom'. 'config' is an object that is required and defines configuration parameters, including required properties 'destination' and 'called_ae', both of which are strings; 'calling_ae', which is a string with a default value of 'flywheel'; and 'port', which is a number with a default value of '104'. No additional properties are allowed at the root level.
