The schema describes an audit log action configuration. It is an object type. The main properties of the schema are: 'acl' is a string with allowed values 'ENABLED' and 'DISABLED'. 'alg' is a string with allowed values 'ENABLED' and 'DISABLED'. 'appflowexport' is a string with allowed values 'ENABLED' and 'DISABLED'. 'builtin' is an object, readonly. 'contentinspectionlog' is a string with allowed values 'ENABLED' and 'DISABLED'. 'dateformat' is a string with allowed values 'MMDDYYYY', 'DDMMYYYY', and 'YYYYMMDD'. 'domainresolvenow' is a boolean. 'domainresolveretry' is an integer. 'feature' is a string, readonly, with many specified allowed values. 'ip' is a string, readonly. 'logfacility' is a string with allowed values 'LOCAL0', 'LOCAL1', 'LOCAL2', 'LOCAL3', 'LOCAL4', 'LOCAL5', 'LOCAL6', and 'LOCAL7'. 'loglevel' is an object. 'lsn' is a string with allowed values 'ENABLED' and 'DISABLED'. 'name' is a string. 'serverdomainname' is a string. 'serverip' is a string. 'serverport' is an integer. 'sslinterception' is a string with allowed values 'ENABLED' and 'DISABLED'. 'subscriberlog' is a string with allowed values 'ENABLED' and 'DISABLED'. 'tcp' is a string with allowed values 'NONE' and 'ALL'. 'timezone' is a string with allowed values 'GMT_TIME' and 'LOCAL_TIME'. 'urlfiltering' is a string with allowed values 'ENABLED' and 'DISABLED'. 'userdefinedauditlog' is a string with allowed values 'YES' and 'NO'. No additional properties are allowed at the root level.
