The schema describes a zoo export. It is an array type. The main properties of the schema are: 'title' is a string and is required. 'url' is a string and is required. 'type' is an enum with allowed values 'page' and 'image' and is required. 'added' is a string and is required. 'public' is an enum with allowed values 'y' and 'n' and is required. 'referer' is a string and is optional. 'description' is a string and is optional. 'packs' is an array of strings, allows unique items, and is optional. 'tags' is an array of strings, allows unique items, and is optional. No additional properties are allowed at the root level.
