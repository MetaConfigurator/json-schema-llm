The schema describes metadata about a person in the Mopidy community. It is an object type. The main properties of the schema are: 'email' is a string representing the email address, required. 'homepage' is a string representing the URL to personal web site, optional. 'id' is a string representing a unique identifier for the person, required. 'name' is a string representing the full name, required. 'profiles' is an object representing user profiles, required, which contains: 'discuss' as a string for the username on discuss.mopidy.com, optional; 'freenode' as a string for IRC nickname on freenode.net, optional; 'github' as a string for GitHub username, optional; 'twitter' as a string for Twitter username, optional. No additional properties are allowed at the root level.
