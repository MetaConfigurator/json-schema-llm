The schema describes a configuration object for widgets and packages. It is an object type. The main properties of the schema are: 'widget' is an object that displays information from a resource with the data of a widget.json. It is required and must include 'resource', which is a string formatted as a topic and serves as the topic of the resource with content of a widget.json. 'package' is an object that displays version and dependencies of a widget from a resource with the data of a package.json. It is optional and contains 'resource', which is a string formatted as a topic and serves as the topic of the resource with content of a package.json. No additional properties are allowed at the root level.
