The schema describes server information. It is an object type. The main properties of the schema are: 'fabric_fq_name' is an array of strings representing the fully qualified names of the fabric. It is optional. 'ipmi' is an object that contains required properties: 'ipmi_subnet_list' is an array of strings that holds subnet prefixes formatted as 'x.x.x.x/y' where x is a number between 0 and 255 and y is between 0 and 32. 'ipmi_credentials' is an array of strings representing IPMI user:password credentials. 'ipmi_port_ranges' is an array of objects; each object must have required properties: 'port_range_start' is an integer for the start of the IPMI port range, and 'port_range_end' is an integer for the end of the IPMI port range. 'ironic' is an object that contains required properties: 'auth_url' is a string representing the Ironic Auth URL, 'username' is a string for the Ironic Auth username, and 'password' is a string for the Ironic Auth password. It also has an optional property: 'introspection_flag' is a boolean with a default value of true indicating if introspection is needed. 'cc_username' is a string representing the CC username. 'cc_password' is a string representing the CC password. 'contrail_command_host' is a string representing the Contrail command host. No additional properties are allowed at the root level.
