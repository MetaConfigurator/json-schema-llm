The schema describes a course or path on which something is moving or pointing. It is an object type. The main properties of the schema are: 'location' is an object representing the location details, which is required. It has properties like 'time' (an integer), 'segment_distance' (a number that describes the current segment length), 'name' (a string for the direction name), 'start_location' (a string for the start location name), 'end_location' (a string for the end location name), 'directions_error' (a string for the error message), and 'error_code' (an integer for the error code). 'steps' is an array of objects, which is required, where each object has properties like 'directions' (a string with enum values: 'Head', 'Go Straight', 'Turn Left', 'Turn Right', 'Turn Slight Left', 'Turn Slight Right', 'Turn Sharp Left', 'Turn Sharp Right', 'Roundabout Left', 'Roundabout Right', 'Uturn Left', 'Uturn Right', 'Ramp Left', 'Ramp Right', 'Fork Left', 'Fork Right', 'Keep Left', 'Keep Right', 'Ferry', 'Ferry Train', 'Merge', 'Reached Your Destination'), 'direction' (a string for the current direction), 'distance' (a number for the step distance), 'distance_unit' (a string with enum values: 'mi', 'km' for distance unit), 'maneuverType' (a string with enum values: 'HEAD', 'GO_STRAIGHT', 'TURN_LEFT', 'TURN_RIGHT', 'TURN_SLIGHT_LEFT', 'TURN_SLIGHT_RIGHT', 'TURN_SHARP_LEFT', 'TURN_SHARP_RIGHT', 'ROUNDABOUT_LEFT', 'ROUNDABOUT_RIGHT', 'UTURN_LEFT', 'UTURN_RIGHT', 'RAMP_LEFT', 'RAMP_RIGHT', 'FORK_LEFT', 'FORK_RIGHT', 'KEEP_LEFT', 'KEEP_RIGHT', 'FERRY', 'FERRY_TRAIN', 'MERGE', 'REACHED_YOUR_DESTINATION'), 'compass_direction' (a string with enum values: 'N', 'S', 'W', 'E', 'NW', 'NE', 'SW', 'SE'), 'maneuverPoint' (an object containing latitude and longitude properties), and 'duration_sec' (an integer for the step duration in seconds). No additional properties are allowed at the root level.
