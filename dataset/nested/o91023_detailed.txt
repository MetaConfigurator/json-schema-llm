The schema describes files and their associated metadata in the Zenodo system. It is an object type. The main properties of the schema are: '_buckets' is an object containing information about various buckets, with properties 'deposit', 'record', and 'extra_formats', each being a string; '_files' is an array of objects, each describing a file, with required properties 'bucket', 'checksum', 'file_id', 'key', 'size', 'type', and 'version_id', all being strings except 'size', which is an integer. No additional properties are allowed at the root level.
