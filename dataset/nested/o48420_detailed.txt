The schema describes an egg group. It is an object type. It defines several reusable components under 'definitions': 'translations' is an object, containing several properties for different language translations. The properties are 'cz', 'dk', 'fr', 'de', 'gr', 'it', 'pl', 'tr', 'en', 'jp', and 'es', all of which are strings. The 'en' and 'de' fields are required, and no additional properties are allowed. The main properties of the schema are: 'names' is a translations reference which is required. No additional properties are allowed at the root level.
