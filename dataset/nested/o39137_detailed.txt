The schema describes universal target options for Build Facade. It is an object type. It defines several reusable components under 'definitions': 'fileReplacement' is a oneOf type, allowing for two different object structures. The first structure requires 'src' and 'replaceWith', both of which are strings, and disallows additional properties. The second structure requires 'replace' and 'with', also both strings, and similarly disallows additional properties. The main properties of the schema are: 'browserTarget' is a string required field, describing the target to browser. 'bundleDependenciesWhitelist' is an array of strings, marked as optional, and is deprecated. 'deleteOutputPath' is a boolean optional field with a default value of true, indicating deletion of the output path before building. 'fileLoaderEmitFile' is a boolean optional field with a default value of false, marked as deprecated since Angular v9. 'fileReplacements' is an array of 'fileReplacement' items, marked as optional, with a default value of an empty array. 'partialBrowserConfig' is a string optional field describing partial webpack config for browser. 'partialServerConfig' is a string optional field describing partial webpack config for server. 'serverTarget' is a string required field, describing the target to server. 'verbose' is a boolean optional field with a default value of false, adding more details to output logging. No additional properties are allowed at the root level.
