The schema describes access control lists. It is an object type. The main properties of the schema are: 'access-lists' is an object, representing a collection of access control lists. It is optional and contains an array called 'acl'. Each item in 'acl' has the following properties: 'acl-name' is a string, which represents the name of the access control list. 'acl-type' is a string, which denotes the type of the access control list. 'access-list-entries' is an array, which contains entries for the access control list. Each entry has: 'rule-name' as a string, representing the rule name; 'matches' as an object with 'source-ipv4-network' and 'destination-ipv4-network', both as strings; and 'actions' as an object containing 'deny', 'permit', and 'rate-limit', each as an array of strings. No additional properties are allowed at the root level.
