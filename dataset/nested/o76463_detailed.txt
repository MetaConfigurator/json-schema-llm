The schema describes a pubsub push subscription request. It is an object type. The main properties of the schema are: 'message' is an object that represents the details of the message, which is required. It contains the required properties: 'attributes', which is an object with string properties that must match the pattern '^[A-Za-z0-9_\-]+$' and has a minimum length of 1; 'data', which is a string with a minimum length of 1; 'publish_time', which is a string with a minimum length of 1; 'publishTime', which is a string with a minimum length of 1; 'message_id', which is a string with a minimum length of 1; and 'messageId', which is a string with a minimum length of 1. No additional properties are allowed at the root level.
