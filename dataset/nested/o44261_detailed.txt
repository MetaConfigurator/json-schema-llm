The schema describes an instance of a group. It is an object type. The main properties of the schema are: 'children' is an array that holds the UID of scenes included in this group. It is optional. Each item in the array must be a string. 'default' is a string that represents the default child of the group. It is optional. 'description' is a string that provides the description of the group or its i18n key. It is optional. 'name' is a string that specifies the name of the group or its i18n key. It is optional. 'slug' is a string that indicates the slug name of the group or its i18n key. It is optional. 'uid' is a string that represents the UID of the group. It is required. No additional properties are allowed at the root level.
