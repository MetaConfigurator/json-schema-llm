The schema describes an AWS network access control list (ACL) rule. It is an object type. The main properties of the schema are: 'protocol' is a string that represents the protocol for this ACL. It is required and must be one of the following values: 'TCP', 'UDP', '-1', 'ICMP'. 'rule_no' is a number that specifies the number of the rule for ordering. It is required. 'action' is a string that defines the action that should be implemented. It is required and must be one of the following values: 'allow', 'deny'. 'cidr_block' is a string that describes the CIDR to associate this rule to. It is required and must follow the CIDR format. 'from_port' is a number that indicates the from port number, with 0 representing all. It is required and must be between 0 and 65535. 'to_port' is a number that indicates the to port number, with 0 representing all. It is required. No additional properties are allowed at the root level.
