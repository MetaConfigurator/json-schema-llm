The schema describes the structure of a configuration object. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'headline' is an object that contains the HTML text to be shown as a headline. It is required. It has properties: 'i18nHtmlFormat' which is a string or an object describing the internationalized HTML headline content using placeholders for the selections. 'level' is an integer with a description of the hierarchical level of the headline (1 to 6), with a minimum of 1, a maximum of 6, and a default value of 1. 'selections' is an array, defaulting to an empty array, which contains items that are objects. Each item in the array is required to have an 'options' property and has properties: 'parameter' which is a string with a description of the URL parameter name, and has a default value of 's'; 'resource' which is a string describing the resource under which to publish the selection; and 'options' which is an object with various properties including 'i18nHtmlFormat', 'parameterFormat', 'fields', 'resource', and 'init', each elaborating on how to display and format selections. 'i18n' is an object with a property 'locale' which is a string that describes the locale setting with a default value of 'default'. No additional properties are allowed at the root level.
