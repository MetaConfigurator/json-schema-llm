The schema describes a content item structure. It is an object type. It defines several reusable components under 'definitions': 'guid' is a string, formatted to match a UUID pattern. 'guid_list' is an array of unique 'guid' items. The main properties of the schema are: 'bulk_publishing' is a boolean that indicates if bulk publishing is enabled. 'links' is an object that contains several properties: 'facet_groups', 'facet_values', 'finder', 'government', 'lead_organisations', 'mainstream_browse_pages', 'meets_user_needs', 'ordered_related_items', 'ordered_related_items_overrides', 'organisations', 'original_primary_publishing_organisation', 'parent', 'people', 'policy_areas', 'primary_publishing_organisation', 'related_policies', 'roles', 'suggested_ordered_related_items', 'taxons', 'topical_events', 'topics', 'world_locations', and 'worldwide_organisations', all of which are arrays of unique 'guid' items, except for 'government', 'primary_publishing_organisation', and 'parent', which have a maximum of one item. 'previous_version' is a string that may contain the identifier of a previous version. No additional properties are allowed at the root level.
