The schema describes a configuration file for making 'db' from 'spk' and 'vstim'. It is an object type. The main properties of the schema are: 'inputs' is an object that contains the path of input file which must include 'spk' and 'vstim'. It is a required property. 'outputs' is an object that contains the path of the output file. It is an optional property. 'params' is an object that contains several required properties: 'dt_sec' is a number describing the time resolution, in seconds; 'trial_length_sec' is a number representing the length of each trial, in seconds; 'window_size_sec' is a number indicating the length of the window of computing the psth, in seconds; 'kernel_size_sec' is a number determining the length of the Gaussian kernel for smoothing psth, in seconds; 'sample_size_sec' is a number describing the length of each input/output sample, in seconds; and 'sample_intersection_sec' is a number for the length of intersection between each two input/output samples, in seconds. No additional properties are allowed at the root level.
