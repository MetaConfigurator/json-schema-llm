The schema describes an IP tunnel configuration. It is an object type. The main properties of the schema are: 'channel' is an integer, readonly; 'encapip' is a string, readonly; 'grepayload' is a string, which must be one of 'ETHERNETwithDOT1Q', 'ETHERNET', 'IP'; 'ipsecprofilename' is a string, optional; 'ipsectunnelstatus' is a string, which must be one of 'DOWN', 'UP', 'PARTIAL-UP', 'UNKNOWN', readonly; 'local' is a string, optional; 'name' is a string, optional; 'ownergroup' is a string, optional; 'protocol' is a string, which must be one of 'IPIP', 'GRE', 'IPSEC', 'UDP', optional; 'refcnt' is an integer, readonly; 'remote' is a string, optional; 'remotesubnetmask' is a string, optional; 'sysname' is a string, readonly; 'tunneltype' is an object, readonly, with no additional properties defined; 'type' is an integer, readonly; 'vlan' is an integer, optional. No additional properties are allowed at the root level.
