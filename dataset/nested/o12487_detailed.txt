The schema describes a rating structure. It is an object type. The main properties of the schema are: 'grade' is a string that is required. 'score' is a number that is required. 'esrAccessibility' is a boolean that is optional. 'normalized' is a boolean that is optional. 'preview' is a boolean that is optional. 'description' is a boolean that is optional. 'metadata' is a boolean that is optional. 'content' is a string that is optional. 'recommendations' is a string that is optional. 'comments' is a string that is optional. No additional properties are allowed at the root level.
