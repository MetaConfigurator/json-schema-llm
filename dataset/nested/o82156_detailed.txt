The schema describes a GitHub webhook event. It is an object type. It defines several reusable components under 'definitions': 'repository' is an object, representing the repository associated with the event, requiring 'name' and 'owner' fields. 'owner' is an object that can be defined in two ways: it requires either 'name' or 'login', with both being strings and having a maximum length of 100. The main properties of the schema are: 'ref' is a string for the GitHub branch/tag with a max length of 255 and is optional; 'after' is a string for the GitHub push (after commit) with a max length of 250 and is optional; 'deleted' is a boolean indicating if the event relates to a deleted repository and is optional; 'repository' is an object referencing the 'repository' definition and is required. No additional properties are allowed at the root level.
