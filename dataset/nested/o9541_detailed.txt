The schema describes a configuration object. It is an object type. The main properties of the schema are: 'title' is a string and required. 'icon' is a string and optional. 'background' is a string and optional. 'background-color' is a string with a format of 'css-color' and optional. 'icon-size' is an integer and optional. 'window' is an object and optional, with required properties: 'position' (which is also an object with required properties 'x' and 'y') and 'size' (which is another object with required properties 'width' and 'height'). 'format' is a string with an enum of allowed values: 'UDRW', 'UDRO', 'UDCO', 'UDZO', 'ULFO', 'UDBZ', and optional. 'contents' is an array of objects with required properties: 'x' (integer), 'y' (integer), 'type' (string with allowed values: 'link', 'file', 'position'), and 'path' (string), and it is required. 'code-sign' is an object that is optional with a required property: 'signing-identity' (string) and an optional property: 'identifier' (string). No additional properties are allowed at the root level.
