The schema describes Piwik Core Collector Options. It is an object type. The main properties of the schema are: 'additionalFields' is an object with optional string properties 'event_detail', 'event_type', 'journey_stage', and 'service_identifier'. 'chunk-size' is a number that is optional. 'empty-data-set' is a boolean that is optional. 'filters' is an array of optional strings. 'idMapping' is an array of optional strings. 'mappings' is an object that is optional and contains properties: 'label', 'nb_conversions', 'nb_visits', and 'nb_visits_converted', all of which are optional strings. No additional properties are allowed at the root level.
