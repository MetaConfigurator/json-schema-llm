The schema describes a gene text representation. It is an object type. The main properties of the schema are: 'entrezGene' is an array of strings representing Entrez Gene identifiers, optional, with no additional items allowed. 'hgnc' is an array of strings representing HGNC identifiers, optional, with no additional items allowed. 'id' is a string representing the unique identifier for the gene text, optional. 'name' is a string representing the name of the gene, optional. 'omimGene' is an array of strings representing OMIM gene identifiers, optional, with no additional items allowed. 'symbol' is a string representing the gene symbol, optional. 'synonyms' is an array of strings representing synonyms for the gene, optional, with no additional items allowed. 'type' is a string representing the type of gene, optional. 'uniprotkbSwissprot' is an array of strings representing UniProtKB/Swiss-Prot identifiers, optional, with no additional items allowed. No additional properties are allowed at the root level.
