The schema describes a configuration profile for a macOS system. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string describing the payload, optional with a default value of 'Proxy Configuration Payload'. 'PayloadDisplayName' is a string naming the payload, optional with a default value of 'Proxy Configuration Payload'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, dot-delimited, optional with a default value of 'com.apple.SystemConfiguration'. 'PayloadOrganization' is a string describing the issuing organization of the profile, optional. 'PayloadType' is a string indicating the type of the payload, a reverse DNS string, optional with a default value of 'com.apple.SystemConfiguration'. 'PayloadUUID' is a string that uniquely identifies the payload, conforming to a specific UUID pattern, optional with a default value of an empty string. 'PayloadVersion' is a number representing the version of the configuration profile, optional with a default value of 1. 'Proxies' is an object containing macOS system proxy configuration details. It has the following properties: 'ExceptionsList' is an array of strings that bypass proxy settings for specified hosts and domains, optional. 'FTPEnable' is a number enabling or disabling FTP Proxy, optional with a default value of 0. 'GopherEnable' is a number enabling or disabling Gopher Proxy, optional with a default value of 0. 'HTTPEnable' is a number enabling or disabling HTTP Proxy, optional with a default value of 0. 'HTTPPort' is a number indicating the HTTP Proxy Port, optional with a default value of 80. 'HTTPProxy' is a string for the HTTP Proxy Hostname, optional. 'HTTPSEnable' is a number enabling or disabling HTTPS Proxy, optional with a default value of 0. 'ProxyAutoConfigEnable' is a number enabling the use of a Proxy AutoConfig file, optional with a default value of 0. 'RTSPEnable' is a number enabling or disabling RTSP Streaming Proxy, optional with a default value of 0. 'SOCKSEnable' is a number enabling or disabling SOCKS Proxy, optional with a default value of 0. No additional properties are allowed at the root level.
