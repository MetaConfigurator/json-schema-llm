The schema describes an array of objects representing game sessions. It is an array type. The main properties of the schema are: Each item in the array is an object containing: 'id' is a string and required. 'map_id' is a number and required. 'started' is a string and required. 'ended' is a string and required. 'result' is a string and required. 'team' is a string and required. 'profession' is a string and required. 'rating_type' is a string and required. 'scores' is an object and required, which has properties: 'red' is a number and optional, 'blue' is a number and optional. No additional properties are allowed at the root level.
