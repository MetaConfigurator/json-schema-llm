The schema describes get user response. It is an object type. The main properties of the schema are: 'credentials' is an array of objects that must each contain the required fields 'key', 'secret', 'createdOn', and 'expiresOn', and may also include 'revokedOn'. Each item in the array requires a minimum of 1 item. 'dockerImageName' is a string with a minimum length of 1 and is required. 'email' is a string with a minimum length of 1 and is required. 'links' is an object that must contain the required property 'self', which itself is an object requiring 'href' as a string matching the URI format. 'password' is a string with a minimum length of 1 and is required. 'userId' is a string with a minimum length of 1 and is required. No additional properties are allowed at the root level.
