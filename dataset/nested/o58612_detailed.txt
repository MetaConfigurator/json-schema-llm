The schema describes a test schema for CSV, for standard and formatted data types. It is an array type. It defines several reusable components under 'definitions': 'MyEnum' is an enumeration, it allows the values 'red', 'orange', 6, 9.0, and null. The main properties of the schema are: 'aString' is a string, optional; 'aInteger' is an integer, optional; 'aNumber' is a number, optional; 'aBoolean' is a boolean, optional; 'aEnumRef' is a MyEnum, optional; 'aLong' is a string in int64 format, optional; 'aDateTime' is a string in date-time format, optional; 'aDate' is a string in date format, optional; 'aTime' is a string in time format, optional; 'aInstant' is a string in utc-millisec format, optional; 'aBigInteger' is a string in big-integer format, optional; 'aBigDecimal' is a string in big-decimal format, optional; 'aOctetEncoded' is a string in binary format, optional; 'aBase64Encoded' is a string in byte format, optional. No additional properties are allowed at the root level.
