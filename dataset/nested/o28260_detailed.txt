The schema describes a date and time representation. It is an object type. The main properties of the schema are: 'day' is an integer representing the day of the month, required, with a minimum value of 1 and a maximum value of 31. 'hour' is an integer representing the hour of the day, required, with a minimum value of 0 and a maximum value of 23. 'minute' is an integer representing the minute of the hour, required, with a minimum value of 0 and a maximum value of 59. 'month' is an integer representing the month of the year, required, with a minimum value of 1 and a maximum value of 12. 'second' is an integer representing the second of the minute, required, with a minimum value of 0 and a maximum value of 59. 'timezone' is a string representing the time zone, optional, with a default value of 'UTC'. 'year' is an integer representing the year, required. No additional properties are allowed at the root level.
