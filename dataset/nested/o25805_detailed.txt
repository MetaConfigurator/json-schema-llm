The schema describes a file metadata object. It is an object type. The main properties of the schema are: 'created' is a string representing the creation time, read-only. 'duration' is either null or an integer representing the duration, read-only. 'file_name' is either null or a string representing the original file name. 'file_size' is an integer representing the file size in bytes, read-only. 'hash_md5' is a string representing the MD5 hash, with a maximum length of 255 characters, read-only. 'hash_sha1' is a string representing the SHA1 hash, with a maximum length of 255 characters and a default value of an empty string, read-only. 'height' is either null or an integer representing the height, read-only. 'mime_type' is a string representing the mime-type of the file, with a default value of 'application/octet-stream' and a maximum length of 255 characters. 'modified' is a string representing the last modification time, read-only. 'url' is a string, read-only. 'version' is an integer representing the stream version, with a default value of 1 and read-only. 'width' is either null or an integer representing the width, read-only. No additional properties are allowed at the root level.
