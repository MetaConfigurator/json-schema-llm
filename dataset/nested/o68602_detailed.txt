The schema describes a peer set. It is an array type. The main properties of the schema are: Each item in the array is an object. 'apiVersion' is a string with an enum allowing values 'v1beta1', 'v1.0', 'v1'. 'kind' is a string with an enum allowing values 'Peer', 'peer', and is read-only. 'metadata' is an object that is required and contains: 'name' which is a string, required, with a minimum length of 3 and a maximum length of 30; 'createdOn' which is a string and is read-only; 'modifiedOn' which is a string and is read-only; and 'ref' which is a string, read-only, with a minimum length of 3 and a maximum length of 24. 'spec' is an object that is required and contains: 'credentials' which is an object that is required and contains: 'username' which is a string with a minimum length of 3 and a maximum length of 30; and 'secret' which is a string and is write-only. 'device' is an optional string with a minimum length of 1. 'contactAddr' is an optional string with a minimum length of 1. No additional properties are allowed at the root level.
