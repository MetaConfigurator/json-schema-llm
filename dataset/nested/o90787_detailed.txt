The schema describes MySQL configuration properties. It is an object type. The main properties of the schema are: 'database' is an object MySQL database configuration properties. It includes optional properties: 'name', 'password', 'root_password', and 'username', each with default values of 'defaultdb', 'password', 'root', and 'admin' respectively. 'mysql' is an object MySQL service configuration properties. It is required and contains properties: 'cpus' and 'mem', both with minimum values of 0.3 and 512.0 respectively, and default values of 0.3 and 512.0. 'networking' is an object MySQL networking configuration properties. It includes optional properties: 'external_access' as an object, which has properties 'enable' with a default value of false and 'external_port', defaulting to 13306; and 'port', defaulting to 3306. 'service' is an object DC/OS service configuration properties and includes an optional property 'name' defaulting to 'mysql'. 'storage' is an object MySQL storage configuration properties. It contains optional properties: 'host_volume' defaulting to '/tmp', and 'persistence' as an object with properties 'enable', defaulting to false, and 'volume_size', defaulting to 256. No additional properties are allowed at the root level.
