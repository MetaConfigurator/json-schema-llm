The schema describes the security assessment of JSON Schemas. It is an object type. It defines several reusable components under 'definitions': 'schemaArray' is an array, containing at least one item that is a reference to the root schema. The main properties of the schema are: 'additionalItems' is a reference to the root schema, optional; 'additionalProperties' is a reference to the root schema, optional; 'dependencies' is an object with additionalProperties defined as either an array or a reference to the root schema; 'items' can be either a reference to the root schema or a reference to 'schemaArray'; 'definitions' allows additional properties as references to the root schema; 'patternProperties' allows additional properties as references to the root schema; 'properties' allows additional properties as references to the root schema; 'if' is a reference to the root schema, optional; 'then' is a reference to the root schema, optional; 'else' is a reference to the root schema, optional; 'allOf' is a reference to 'schemaArray'; 'anyOf' is a reference to 'schemaArray'; 'oneOf' is a reference to 'schemaArray'; 'not' is a reference to the root schema; 'contains' is a reference to the root schema; 'propertyNames' is a reference to the root schema. No additional properties are allowed at the root level.
