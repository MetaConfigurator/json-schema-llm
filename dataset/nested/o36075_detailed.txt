The schema describes JSON Schema for `ng-package.json` description file. It is an object type. The main properties of the schema are: '$schema' is a string optional description of the schema URL. 'assets' is an array optional A list of files which are simply copied into the package, with default value as an empty array and items being strings. 'deleteDestPath' is a boolean optional Delete output path before build, with a default value of true. 'dest' is a string optional Destination folder where distributable binaries of the Angular library are written, with a default value of 'dist'. 'keepLifecycleScripts' is a boolean optional Enable this to keep the 'scripts' section in package.json, with a default value of false. 'lib' is an object optional Description of the library's entry point, with additionalProperties set to false and default value as an empty object, containing properties: 'amdId' is a string optional ID for AMD module. 'cssUrl' is a string optional Embed assets in css file using data URIs, with a default value of 'inline' and allowed enum values 'none', 'inline'. 'entryFile' is a string optional Entry file to the public API, with a default value of 'src/public_api.ts'. 'flatModuleFile' is a string optional Filename of the auto-generated flat module file. 'styleIncludePaths' is an array optional Any additional paths that should be used to resolve style imports, with items being strings. 'umdId' is a string optional ID for the UMD bundle. 'umdModuleIds' is an object optional A map of external dependencies and their correspondent UMD module identifiers, with additionalProperties set to true. 'whitelistedNonPeerDependencies' is an array optional A list of dependencies that are allowed in the 'dependencies' and 'devDependencies' section of package.json, with a default value including 'tslib' and items being strings. No additional properties are allowed at the root level.
