The schema describes an event. It is an object type. It defines several reusable components under 'definitions': 'simple_date' is an object, defining a property 'value' which is a string that must match the pattern '^[0-9]{4}(-[0-9]{2}){0,2}$' and is required. 'simple_integer' is an object, defining a property 'value' which is an integer and is required. 'simple_string' is an object, defining a property 'value' which is a string and is required. 'simple_string_array' is an object, defining a property 'value' which is an array of strings and is required. The main properties of the schema are: 'type' is a string that is required and must be equal to 'event'. 'division_id' is a string that is required and must match the pattern '^ocd-division/country:[a-z]{2}$'. 'gid' is an integer that is required. 'row' is an integer that is required. 'id' is a string that is required and must match the pattern '^[a-f0-9]{8}-[a-f0-9]{4}-[1-5][a-f0-9]{3}-[89ab][a-f0-9]{3}-[a-f0-9]{12}$'. 'start_date' is a reference to '#/definitions/simple_date' and is required. 'end_date' is a reference to '#/definitions/simple_date' and is optional. 'location' is a reference to '#/definitions/simple_string' and is optional. 'geonames_name' is a reference to '#/definitions/simple_string' and is optional. 'geonames_id' is a reference to '#/definitions/simple_integer' and is optional. 'admin_level_1_geonames_name' is a reference to '#/definitions/simple_string' and is optional. 'admin_level_1_geonames_id' is a reference to '#/definitions/simple_integer' and is optional. 'classification' is an object that is required, containing a property 'value' which is an array of strings with the following allowed values: 'Arbitrary Detention', 'Assault', 'Attacking Hospital', 'Beating', 'Death in Custody', 'Death in Detention', 'Denial of Medical Attention', 'Disappearance', 'Excessive Force', 'Illegal Detention', 'Inhuman Detention', 'Killing', 'Rape', 'Restricting freedom of speech and press', 'Restricting of freedom of association', 'Torture', 'Use of Child Soldiers', 'Violations against personal integrity', 'Violations against the right to liberty', 'Violations of rights of detained or imprisoned persons', 'Violations of the right to adequate housing', 'Violations of the right to life'. 'description' is a reference to '#/definitions/simple_string' and is required. 'perpetrator_name' is a reference to '#/definitions/simple_string' and is optional. 'perpetrator_organization_id' is a reference to '#/definitions/simple_string' and is optional. 'perpetrator_organization_classification' is a reference to '#/definitions/simple_string_array' and is optional. 'sources' is a reference to '#/definitions/simple_string_array' and is optional. No additional properties are allowed at the root level.
