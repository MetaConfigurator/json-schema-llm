The schema describes a widget information management system. It is an object type. The main properties of the schema are: 'fileListing' is an object that extracts widget information from a file listing, optional. It has properties: 'list' is an array that contains a list with json files which contain information about used widgets, default is ['widget-list.json'], and 'items' are strings; 'applicationUrl' is a string that represents the url of the application with the widgets, default is an empty string. 'widgetListing' is an object that provides a list of available widgets and activities as a resource, required. It has properties: 'resource' is a string with the format 'topic' and an axRole of 'outlet', describing the list resource topic. No additional properties are allowed at the root level.
