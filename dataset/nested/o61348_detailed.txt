It defines several reusable components under 'definitions': 'param' is an object, it requires the fields 'name', 'type', 'nullable', and 'since'. The 'name' field is a string. The main properties of the schema are: 'customTypes' is an array of objects that contain the 'name' property, which is a string describing the custom type, the 'params' property that is an array of 'param' definitions, and the 'returnWithFactory' property, which is a boolean. The 'since' property references the 'since' definition. The 'customTypes' property is required. No additional properties are allowed at the root level.
