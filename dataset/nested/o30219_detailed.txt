The schema describes a server GSLB service group binding. It is an object type. The main properties of the schema are: 'appflowlog' is a string that is readonly and can have the values 'ENABLED' or 'DISABLED'. 'boundtd' is an integer that is readonly. 'cip' is a string that is readonly and can have the values 'ENABLED' or 'DISABLED'. 'cipheader' is a string that is readonly. 'clttimeout' is an integer that is readonly. 'customserverid' is a string that is readonly. 'downstateflush' is a string that is readonly and can have the values 'ENABLED' or 'DISABLED'. 'dup_port' is an integer that is readonly. 'dup_svctype' is a string that is readonly and can have the values 'HTTP', 'FTP', 'TCP', 'UDP', 'SSL', 'SSL_BRIDGE', 'SSL_TCP', 'DTLS', 'NNTP', 'RPCSVR', 'DNS', 'ADNS', 'SNMP', 'RTSP', 'DHCPRA', 'ANY', 'SIP_UDP', 'SIP_TCP', 'SIP_SSL', 'DNS_TCP', 'ADNS_TCP', 'MYSQL', 'MSSQL', 'ORACLE', 'RADIUS', 'RADIUSListener', 'RDP', 'DIAMETER', 'SSL_DIAMETER', 'TFTP', 'SMPP', 'PPTP', 'GRE', 'SYSLOGTCP', 'SYSLOGUDP', 'FIX', 'SSL_FIX', 'USER_TCP', 'USER_SSL_TCP', 'QUIC', 'IPFIX', 'LOGSTREAM'. 'maxbandwidth' is an integer that is readonly. 'maxclient' is an integer that is readonly. 'maxreq' is an integer that is readonly. 'monthreshold' is an integer that is readonly. 'name' is a string. 'port' is an integer that is readonly. 'servicegroupname' is a string. 'serviceipaddress' is a string that is readonly. 'serviceipstr' is a string that is readonly. 'svctype' is a string that is readonly and can have the same values as 'dup_svctype'. 'svrcfgflags' is an integer that is readonly. 'svrstate' is a string that is readonly and can have the values 'UP', 'DOWN', 'UNKNOWN', 'BUSY', 'OUT OF SERVICE', 'GOING OUT OF SERVICE', 'DOWN WHEN GOING OUT OF SERVICE', 'NS_EMPTY_STR', 'Unknown', or 'DISABLED'. 'svrtimeout' is an integer that is readonly. No additional properties are allowed at the root level.
