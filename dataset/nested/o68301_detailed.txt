The schema describes parameters accepted by the Conversant adapter. It is an object type. The main properties of the schema are: 'site_id' is a string A Conversant specific ID which identifies the site, required. 'secure' is an integer Override http/https context on ad markup, optional. 'bidfloor' is a number Minimum bid price that will be considered, optional. 'tag_id' is a string Identifies specific ad placement, optional. 'position' is an integer Ad position on screen, optional. 'mobile' is an integer Indicate if the site is mobile optimized, optional. 'mimes' is an array Array of content MIME types, For videos only, optional, with items being strings. 'maxduration' is an integer Maximum duration in seconds, For videos only, optional. 'api' is an array Array of supported API frameworks, For videos only, optional, with items being integers. 'protocols' is an array Array of supported video protocols, For videos only, optional, with items being integers. No additional properties are allowed at the root level.
