The schema describes UTM parameters for tracking marketing campaigns. It is an object type. The main properties of the schema are: 'utm_source' is a string that must match the pattern '^[\w\/\.:-]+$'. 'utm_medium' is a string that must match the pattern '^[\w\/\.:-]+$'. 'utm_term' is a string that must match the pattern '^[\w\s\/\.,:-]+$'. 'utm_content' is a string that must match the pattern '^[\w\/\.:-]+$'. 'utm_campaign' is a string that must match the pattern '^[\w\/\.:-]+$'. No additional properties are allowed at the root level.
