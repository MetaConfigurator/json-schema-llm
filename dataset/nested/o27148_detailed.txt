The schema describes an invocation schema for ndmg. It is an object type. The main properties of the schema are: 'aal' is a string which must be one of the allowed values: '/ndmg_atlases/labels/AAL.nii.gz'. 'atlas' is a string which must be one of the allowed values: '/ndmg_atlases/atlas/MNI152_T1_1mm.nii.gz'. 'bval_file' is a string and is required. 'bvec_file' is a string and is required. 'clean' is a boolean and is optional. 'cpac200' is a string which must be one of the allowed values: '/ndmg_atlases/labels/CPAC200.nii.gz'. 'desikan' is a string which must be one of the allowed values: '/ndmg_atlases/labels/desikan.nii.gz'. 'dti_file' is a string and is optional. 'dti_file_minc' is a string and is optional. 'extra1' is a string and is optional. 'extra2' is a string and is optional. 'extra3' is a string and is optional. 'harvardoxford' is a string which must be one of the allowed values: '/ndmg_atlases/labels/HarvardOxford.nii.gz'. 'jhu' is a string which must be one of the allowed values: '/ndmg_atlases/labels/JHU.nii.gz'. 'mask' is a string which must be one of the allowed values: '/ndmg_atlases/atlas/MNI152_T1_1mm_brain_mask.nii.gz'. 'mprage_file' is a string and is required. 'outdir' is a string and is required. 'talairach' is a string which must be one of the allowed values: '/ndmg_atlases/labels/Talairach.nii.gz'. The schema imposes constraints: at least one of 'dti_file' or 'dti_file_minc' must be present; also, one of 'desikan', 'jhu', 'talairach', 'aal', 'harvardoxford', or 'cpac200' must be included. The presence of 'dti_file' avoids 'dti_file_minc' and vice versa as per the dependencies defined. No additional properties are allowed at the root level.
