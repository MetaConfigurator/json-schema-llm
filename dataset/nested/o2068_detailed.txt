The schema describes generating metadata from users' metadata TSVs or XLSs. It is an object type. The main properties of the schema are: 'program' is a string optional description. 'project' is a string optional description. 'center_name' is a string optional description. 'submitter_donor_id' is a string optional description. 'donor_uuid' is a string optional description. 'submitter_specimen_id' is a string optional description. 'specimen_uuid' is a string optional description. 'submitter_specimen_type' is a string optional description. 'submitter_sample_id' is a string optional description. 'sample_uuid' is a string optional description. 'analysis_type' is a string optional description. 'workflow_name' is a string optional description. 'workflow_version' is a string optional description. 'file_type' is a string optional description. 'file_path' is a string optional description. No additional properties are allowed at the root level.
