The schema describes session and hit tracking data. It is an object type. The main properties of the schema are: 'activeSessions' is an array of objects that represent currently active sessions, which are required to have 'ua', 'ip', and 'started' fields. 'endedSessions' is an array of objects representing sessions that have ended, which are required to include 'ua', 'ip', 'started', and 'ended' fields. 'hitSamples' is an array of objects capturing sample hits, which must contain 'ua', 'ip', and 'ts' fields. 'startedSessions' is an array of objects indicating sessions that have started, which require 'ua', 'ip', and 'started' fields. No additional properties are allowed at the root level.
