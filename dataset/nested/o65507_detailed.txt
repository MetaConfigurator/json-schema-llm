The schema describes a representation of a Winnow Canonical object. It is an object type. The main properties of the schema are: 'source' is a string that is required. 'path' is a string that is required. 'aliases' is an array of strings that is optional. 'base' is a string that is optional. 'version' is an array of numbers that is required, with exactly 3 items. 'changes' is a string that is required. 'costs' is an object with properties defined by regular expressions, where the keys are paths in the format '/[/.\w-]{0,254}'. The values are arrays containing objects with properties: 'value' (number), 'currency' (string), 'is_taxable' (boolean), and 'type' (string). 'shortcode' is a string that is optional. 'is_default' is a boolean that is optional. 'is_public' is a boolean that is optional. 'licenses' is an array of strings that is optional. No additional properties are allowed at the root level.
