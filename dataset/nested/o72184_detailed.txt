The schema describes how squatch js interacts. It is an object type. The main properties of the schema are: 'widget' is an object that contains properties defining the widget settings. It has an optional property 'defaultWidgetType', which is a string with a default value of 'REFERRER_WIDGET'. 'conversionUrls' is an array of strings that is optional. 'widgetUrlMappings' is an array of objects that is optional, where each object can have properties: 'url' (string), 'showAsCTA' (boolean), 'displayOnLoad' (boolean), and 'widgetType' (string). 'fuelTankAutofillUrls' is an array of objects that is optional, where each object can have properties: 'url' (string) and 'formSelector' (string). No additional properties are allowed at the root level.
