The schema describes configurations for a Kafka-manager and a service. It is an object type. The main properties of the schema are: 'kafka-manager' is an object Kafka-manager configuration properties. It requires the fields 'cpus', 'mem', 'instances', 'zk', and 'application_secret'. 'cpus' is a number CPU shares to allocate, with a minimum value of 0.1 and a default of 1.0. 'instances' is an integer Number of desired instances, with a default of 1. 'mem' is a number Memory to allocate, with a minimum value of 1024.0 and a default of 1024.0. 'zk' is a string Zookeeper URL for Kafka-manager, with a default value of 'zk-1.zk:2181,zk-2.zk:2181,zk-3.zk:2181,zk-4.zk:2181,zk-5.zk:2181'. 'service' is an object DC/OS service configuration properties. It has an optional field 'name', which is a string The name of the kafka-manager service instance, with a default value of 'kafka-manager'. No additional properties are allowed at the root level.
