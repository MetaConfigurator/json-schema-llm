The schema describes the configuration for a connect worker. It is an object type. The main properties of the schema are: [connect] is an object that contains configuration details for the connect worker. It has the following required fields: [cpus], [mem], [instances], [name]. [cpus] is a number defining CPU shares to allocate to each connect worker instance, with a default value of 2 and a minimum value of 1. [mem] is a number that specifies the memory (MB) to allocate to each connect worker instance, with a default value of 1024 and a minimum value of 512. [instances] is an integer indicating the number of instances to run, with a default value of 1 and a minimum value of 1. [name] is a string for the service name of the Confluent Enterprise Replicator, with a default value of 'replicator'. No additional properties are allowed at the root level.
