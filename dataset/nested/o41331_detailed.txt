The schema describes an invocation manifest for MRIQC: No-reference image quality metrics for quality assessment of MRI (v0.9.4). It is an object type. The main properties of the schema are: 'inputs' is an object that is required. It has a required property 'nifti', which is also an object with a required property 'type' that must be one of the enum values: 'nifti'. 'config' is an object that is required. It has three required properties: 'save_derivatives', which is a boolean with a default value of false; 'save_outputs', which is a boolean with a default value of false; and 'measurement', which is a string with a default value of 'auto-detect' and must be one of the enum values: 'auto-detect', 'functional', 'anatomy_t1w', 'anatomy_t2w'. No additional properties are allowed at the root level.
