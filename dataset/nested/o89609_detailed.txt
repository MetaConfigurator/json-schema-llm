The schema describes a configuration for a developer tools interface. It is an object type. The main properties of the schema are: 'open' is an object that contains properties for 'onActions', which is an optional array of strings with a default of an empty array, and 'onGlobalMethod', which is a string with a default value of 'laxarShowDeveloperTools'. 'button' is an object that allows to render a button that opens the popup, with properties: 'enabled', which is a boolean with a default of true, and 'htmlLabel', which is a string with a default of "<i class='fa fa-cube'></i> DeveloperTools". 'grid' is an object with properties: 'anchor', which is a string with a default of '[data-ax-page], [ax-page]', and 'columns', which is an object containing 'count', which is an integer with a default of 12, 'width', which is an integer with a default of 78, 'padding', which is an integer with a default of 13, and 'gutter', which is an integer with a default of 13. The default for 'columns' includes width of 78, gutter of 26, padding of 13, and count of 12. 'css' is an object with a default configuration for margin, box-sizing, position, top, right, bottom, left, and z-index. 'develop' is an object that allows the use of the development version (debug.html) of the developer tools, with an 'enabled' boolean property defaulting to false. 'tracker' is an object that tracks and validates pattern events with an 'enabled' boolean property defaulting to false. No additional properties are allowed at the root level.
