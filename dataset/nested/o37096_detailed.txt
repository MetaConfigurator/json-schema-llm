The schema describes the MCDA student's t distribution for the performance table entry of absolute data. It is an object type. The main properties of the schema are: 'type' is a string, required, and must be equal to 'dt'. 'parameters' is an object, required, and can have no additional properties. It requires 'mu', 'stdErr', and 'dof', all of which are numbers. 'input' is an object, optional, and can have no additional properties. It requires 'mu', 'sigma', and 'sampleSize', all of which are numbers. No additional properties are allowed at the root level.
