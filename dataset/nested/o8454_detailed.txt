The schema describes a configuration for fabric management. It is an object type. The main properties of the schema are: 'fabric_uuid' is a string that represents the UUID of the fabric object created, it is required and has a default value of an empty string. 'contrail_command_host' is a string that represents the UUID of the fabric object created, it is required and has a default value of an empty string. 'cc_username' is a string that indicates the Contrail Command Login Username, it is optional. 'cc_password' is a string that indicates the Contrail Command Login Password, it is optional. 'fabric_fq_name' is an array of strings representing the FQNAME of the fabric object created, it is optional and must contain strings as its items. 'manage_underlay' is a boolean that determines if underlay networking needs to be managed, it is optional. The schema requires a minimum of 2 properties and a maximum of 5 properties. No additional properties are allowed at the root level.
