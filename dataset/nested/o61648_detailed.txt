The schema describes a web content filtering configuration. It is an object type. The main properties of the schema are: 'AutoFilterEnabled' is a boolean that indicates if automatic filtering is enabled. It is optional and has a default value of false. 'BlacklistedURLs' is an array of strings where each entry contains a URL that will not be accessible. 'FilterBrowsers' is a number that specifies browser filtering options. It is optional and has a default value of true. 'FilterSockets' is a number that specifies socket filtering options. It is optional and has a default value of true. 'FilterType' is a string indicating the type of filter, either built-in or plug-in. It is optional and has a default value of 'BuiltIn'. 'Organization' is a string representing an organization that will be passed to a third-party plugin. It is optional. 'Password' is a string representing a password for the service. It is optional. 'PayloadCertificateUUID' is a string representing a UUID of the certificate payload containing an identity used as the client credential, matching the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. It is optional. 'PayloadDescription' is a string that describes the payload. It is optional and has a default value of 'Configures content filtering settings'. 'PayloadDisplayName' is a string representing the name of the payload. It is optional and has a default value of 'Web Content Filter'. 'PayloadIdentifier' is a string that is a unique identifier for the payload, dot-delimited. It is optional and has a default value of 'com.apple.webcontent-filter'. 'PayloadOrganization' is a string that describes the issuing organization of the profile, as displayed to the user. It is optional. 'PayloadType' is a string that indicates the type of the payload, a reverse DNS string. It is optional and has a default value of 'com.apple.webcontent-filter'. 'PayloadUUID' is a string that serves as a unique identifier for the payload, matching the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. It is optional and has a default value of ''. 'PayloadVersion' is a number indicating the version of the whole configuration profile. It is optional and has a default value of 1. 'PermittedURLs' is an array of strings that contain URLs accessible if 'AutoFilterEnabled' is true; if not, this field is ignored. 'PluginBundleID' is a string representing the Bundle ID of the plugin providing the filtering service. It is optional. 'ServerAddress' is a string that provides the server address, which may be an IP address, hostname, or URL. It is optional. 'UserDefinedName' is a string that will be displayed for this filtering configuration. It is optional. 'UserName' is a string representing a username for the service. It is optional. 'WhitelistedBookmarks' is an array containing objects with properties: 'BookmarkPath' (optional string indicating folder placement), 'Title' (string for the bookmark title), and 'URL' (string for the bookmark's URL). No additional properties are allowed at the root level.
