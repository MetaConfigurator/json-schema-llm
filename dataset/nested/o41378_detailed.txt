The schema describes an invocation manifest for Flywheel HeuDiConv. It is an object type. The main properties of the schema are: 'config' is an object that contains several required properties: 'action' is a string that is required, with a default value of 'Curate' and allowed values 'Curate', 'Export', 'Tabulate'; 'dry_run' is a boolean that is required, with a default value of true; 'do_whole_project' is a boolean that is required, with a default value of false; 'extended_bids' is a boolean that is required, with a default value of true. 'inputs' is an object that contains a required property: 'api-key' is an object that is required; 'heuristic' is an optional object that can contain additional properties. No additional properties are allowed at the root level.
