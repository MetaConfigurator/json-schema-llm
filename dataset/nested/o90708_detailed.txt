The schema describes configuration settings for an InfluxDB service. It is an object type. The main properties of the schema are: 'configuration' is an object containing influxdb networking configuration properties. 'networking' is an object containing influxdb networking configuration properties. 'service' is an object containing DC/OS service configuration properties, with required fields 'cpus' and 'mem'. 'storage' is an object containing influxdb storage configuration properties. No additional properties are allowed at the root level.
