The schema describes the representation of a badge class. It is an object type. The main properties of the schema are: 'alignment' is an array listing objects describing which educational standards this badge aligns to, if any; it is optional and must have a minimum of 0 items. 'criteria' is a string providing the URL of the criteria for earning the achievement; it is required and formatted as a URI. 'description' is a string giving a short description of the achievement; it is required. 'image' is a string containing the URL of an image representing the achievement; it is required and formatted as a URI. 'issuer' is a string specifying the URL of the organization that issued the badge; it is required and formatted as a URI. 'name' is a string stating the name of the achievement; it is required. 'tags' is an array listing tags that describe the type of achievement; it is optional and must have a minimum of 0 items. No additional properties are allowed at the root level.
