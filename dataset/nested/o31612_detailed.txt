The schema describes an authorization policy label. It is an object type. The main properties of the schema are: 'description' is a string that is readonly. 'flowtype' is an integer that is readonly. 'gotopriorityexpression' is a string that is readonly. 'hits' is an integer that is readonly. 'invoke_labelname' is a string that is readonly. 'labelname' is a string that is optional. 'labeltype' is a string that is readonly and must be one of the following values: 'reqvserver', 'resvserver', 'policylabel'. 'newname' is a string that is optional. 'numpol' is an integer that is readonly. 'policyname' is a string that is readonly. 'priority' is an integer that is readonly. No additional properties are allowed at the root level.
