The schema describes an organisation. It is an object type. The main properties of the schema are: '$schema' is a string that represents the schema to validate organisation records against. It is required, has a minimum length of 9, and defaults to 'https://ils.rero.ch/schemas/organisations/organisation-v0.0.1.json'. 'pid' is a string that represents the Organisation ID. It is required. 'name' is a string that represents the required name of the organisation. It is required and has a minimum length of 1. 'address' is a string that represents the address of the organisation. It is optional and must have a minimum length of 1. 'code' is a string that represents the code of the organisation. It is required and has a minimum length of 2. 'current_budget_pid' is a string that represents the ID of the current budget of the organisation. It is optional. 'default_currency' is a string that represents the default currency of the organisation. It is required and must match the pattern '^[A-Z]{3}$'. 'online_harvested_source' is a string that represents the online harvested source as configured in ebooks server. It is optional. No additional properties are allowed at the root level.
