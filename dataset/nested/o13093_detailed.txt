The schema describes a faculty object. It is an object type. It defines several reusable components under 'definitions': 'object_01' is an object, describes a faculty list. It requires a 'faculty' array, where each item is an object that must have 'firstname', 'lastname', 'title', 'org', 'category', 'imgsrc', and 'desc' as required fields. No additional properties are allowed at the root level.
