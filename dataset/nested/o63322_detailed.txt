The schema describes NgRx Effect Options. It is an object type. The main properties of the schema are: 'api' is a boolean specifies if effect has api success and failure actions wired up. It is optional and defaults to false. 'creators' is a boolean specifies whether to use creator functions for handling actions, reducers, and effects. It is optional and defaults to true. 'feature' is a boolean flag to indicate if part of a feature schematic. It is optional and defaults to false. 'flat' is a boolean flag to indicate if a dir is created. It is optional and defaults to true. 'group' is a boolean groups effects file within 'effects' folder. It is optional and defaults to false. 'minimal' is a boolean setup root effects module without registering initial effects. It is optional and defaults to false. 'module' is a string allows specification of the declaring module. It is optional and defaults to an empty string. 'name' is a string the name of the effect. It is optional and has a default value sourced from argv at index 0. 'path' is a string the path to create the component. It is optional. 'project' is a string the name of the project. It is optional. 'root' is a boolean flag to indicate whether the effects are registered as root. It is optional and defaults to false. 'skipTests' is a boolean when true, does not create test files. It is optional and defaults to false. No additional properties are allowed at the root level.
