The schema describes configuration options for Webtrends Reports Collector. It is an object type. The main properties of the schema are: 'additionalFields' is an object containing optional properties: 'eventCategory', 'eventDetail', 'eventType', and 'timeSpan', each of which is a string; 'chunk-size' is a number, optional; 'empty-data-set' is a boolean, optional; 'idMapping' is an array of strings, optional; 'mappings' is an object with optional properties: 'Avg. Time on Site', 'Avg. Visitors per Day', 'Bounce Rate', 'New Visitors', 'Page Views', 'Page Views per Visit', 'Visitors', and 'Visits', each of which is a string; 'row_type_name' is a string, optional. No additional properties are allowed at the root level.
