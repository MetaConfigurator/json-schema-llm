The schema describes a list of chunk translations. It is an object type. The main properties of the schema are: 'data' is an array containing translation objects. Each translation object includes the following properties: 'id' is a number and is required. 'translator' is an object and is required; it has the required properties: 'id' (number), 'nickname' (string), 'created_at' (string), and 'updated_at' (string). 'body' is a string and is required. 'rating' is a number and is required. 'rates' is an object that is required; it contains 'data', which is an array of translation rate objects. Each translation rate object includes the required properties: 'id' (number), 'rater' (object with required properties 'id' (number), 'nickname' (string), 'created_at' (string), and 'updated_at' (string)), 'value' (number, required), 'created_at' (string, required), and 'updated_at' (string, required). No additional properties are allowed at the root level.
