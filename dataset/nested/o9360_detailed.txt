The schema describes the configuration for media content display. It is an object type. The main properties of the schema are: 'medium' is an object that identifies which media content to display. It is required and has the following properties: 'resource' is a string that specifies a resource topic under which a document resource can be obtained, and is required. 'showTitle' is a boolean that indicates if a title should be shown above the media content, with a default value of false. 'showCaption' is a boolean that indicates if a caption should be shown below the media content, with a default value of false. 'onActions' is an array that lists actions to trigger loading media content; each item in the array is a string formatted as a topic. 'layout' is an object that tells how the display size is calculated. It has the property 'sizeToContent', which is a boolean indicating if the media content determines rendering size, with a default value of false. 'fallback' is an object that allows opening content in a new browser window on specific platforms. It has a property 'i18nHtmlText', which can be a string or an object, specifying text to display on the fallback link, with a default value for English and German texts. 'integration' is an object that allows integration with embedded websites, with the property 'name', which can be a string or null, specifying the window.name for the embedded website, with a default of null. 'i18n' is an object that indicates the locale to use for displaying this widget. It contains the property 'locale', which is a string formatted as a topic, with a default value of 'default'. No additional properties are allowed at the root level.
