The schema describes a Juju Charm. It is an object type. The main properties of the schema are: '$schema-location' is a string that must be one of the allowed values: 'Charm-Schema.json'. 'id' is a string and is required. 'name' is a string with a default value of an empty string and is required; it describes the name of the Charm. 'summary' is a string with a default value of an empty string; it describes a summarized description of the Charm. 'description' is a string with a default value of an empty string; it describes a human-readable description of the charm. 'maintainer' is a string with a default value of an empty string; it identifies the maintainer of the Charm. 'category' is a string with a default value of an empty string and is required. 'repository' is a string with a default value of 'local' and is required. No additional properties are allowed at the root level.
