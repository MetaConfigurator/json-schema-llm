The schema describes an invocation manifest for CNI: Quality Assurance Report (fMRI). It is an object type. The main properties of the schema are: 'inputs' is an object that is required. It contains the required property 'nifti', which is also an object that includes a property 'type' with an enum value of 'nifti'. 'config' is an object that is required. It includes the required properties 'spike_thresh' and 'nskip'. 'spike_thresh' is a number with a default value of 6, described as 'z-score threshold for spike detector. DEFAULT=6.0'. 'nskip' is an integer with a default value of 6, described as 'number of initial time-points to skip. DEFAULT=6'. No additional properties are allowed at the root level.
