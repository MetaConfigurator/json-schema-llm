The schema describes a data structure for outfitting messages. It is an object type. The main properties of the schema are: '$schemaRef' is a string, required; 'header' is an object, required, with additional properties allowed. It contains 'gatewayTimestamp' (a string, not required), 'softwareName' (a string, required), 'softwareVersion' (a string, required), and 'uploaderID' (a string, required). 'message' is an object, required, with no additional properties allowed. It includes 'horizons' (a boolean, not required), 'marketId' (a number, required), 'modules' (an array of strings, required, with at least 1 item, unique items, and matching the pattern '(^Hpt_|^Int_|_Armour_)'), 'stationName' (a string, required, with minLength of 1), 'systemName' (a string, required, with minLength of 1), and 'timestamp' (a string, required). No additional properties are allowed at the root level.
