The schema describes engine output. It is an array type. It defines several reusable components under 'definitions': 'result' is an object, representing analysis result. It has required fields 'path' (a string describing the path relative to the analysis root) and 'messages' (an array of messages in the path, with items being 'message' definitions). 'message' is an object, representing analysis message. It has required fields 'message' (a string describing the short description of the message), 'severity' (a string indicating the severity of the message, which can be one of: 'verbose', 'hint', 'information', 'warning', 'error'), and 'line' (an integer indicating the line where the message is located, with a minimum of 0).
