The schema describes a JSON Schema for Common Vulnerability Scoring System version 2.0. It is an object type. It defines several reusable components under 'definitions': 'accessVectorType' is a string, with allowed values: 'NETWORK', 'ADJACENT_NETWORK', 'LOCAL'. 'accessComplexityType' is a string, with allowed values: 'HIGH', 'MEDIUM', 'LOW'. 'authenticationType' is a string, with allowed values: 'MULTIPLE', 'SINGLE', 'NONE'. 'ciaType' is a string, with allowed values: 'NONE', 'PARTIAL', 'COMPLETE'. 'exploitabilityType' is a string, with allowed values: 'UNPROVEN', 'PROOF_OF_CONCEPT', 'FUNCTIONAL', 'HIGH', 'NOT_DEFINED'. 'remediationLevelType' is a string, with allowed values: 'OFFICIAL_FIX', 'TEMPORARY_FIX', 'WORKAROUND', 'UNAVAILABLE', 'NOT_DEFINED'. 'reportConfidenceType' is a string, with allowed values: 'UNCONFIRMED', 'UNCORROBORATED', 'CONFIRMED', 'NOT_DEFINED'. 'collateralDamagePotentialType' is a string, with allowed values: 'NONE', 'LOW', 'LOW_MEDIUM', 'MEDIUM_HIGH', 'HIGH', 'NOT_DEFINED'. 'targetDistributionType' is a string, with allowed values: 'NONE', 'LOW', 'MEDIUM', 'HIGH', 'NOT_DEFINED'. 'ciaRequirementType' is a string, with allowed values: 'LOW', 'MEDIUM', 'HIGH', 'NOT_DEFINED'. 'scoreType' is a number, with a minimum of 0 and a maximum of 10. The main properties of the schema are: 'version' is a string indicating 'CVSS Version', required, with allowed value: '2.0'. 'vectorString' is a string that must match a specified regular expression pattern. 'accessVector' is a string referencing 'accessVectorType', required. 'accessComplexity' is a string referencing 'accessComplexityType', required. 'authentication' is a string referencing 'authenticationType', required. 'confidentialityImpact' is a string referencing 'ciaType', required. 'integrityImpact' is a string referencing 'ciaType', required. 'availabilityImpact' is a string referencing 'ciaType', required. 'baseScore' is a number referencing 'scoreType', required. 'exploitability' is a string referencing 'exploitabilityType', optional. 'remediationLevel' is a string referencing 'remediationLevelType', optional. 'reportConfidence' is a string referencing 'reportConfidenceType', optional. 'temporalScore' is a number referencing 'scoreType', optional. 'collateralDamagePotential' is a string referencing 'collateralDamagePotentialType', optional. 'targetDistribution' is a string referencing 'targetDistributionType', optional. 'confidentialityRequirement' is a string referencing 'ciaRequirementType', optional. 'integrityRequirement' is a string referencing 'ciaRequirementType', optional. 'availabilityRequirement' is a string referencing 'ciaRequirementType', optional. 'environmentalScore' is a number referencing 'scoreType', optional. No additional properties are allowed at the root level.
