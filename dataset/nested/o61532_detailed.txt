The schema describes configuration options for a .mopedrc file. It is an object type. It defines several reusable components under 'definitions': 'appConfigDefinition' is an object, containing properties for various application configurations such as database migrations, client entry point, and server entry points. All properties are optional, and the schema disallows additional properties. 'monoRepoDefinition' is an object, containing a 'monorepo' property that is a constant true value. It also allows for pattern properties to apply options for different apps, which must include at least two properties. The effective root schema resolves to either 'appConfigDefinition' or 'monoRepoDefinition'.
