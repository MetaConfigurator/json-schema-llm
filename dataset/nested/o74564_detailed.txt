The schema describes the contents and locations of Wangscape output tilesets. It is an array type. The main properties of the schema are: each item in the array is an object specifying the terrains, filename, resolution, and tile resolution of a Wangscape output tileset. The properties are as follows: 'resolution' is an array specifying the resolution of tiles in the tileset. It is required and must contain exactly 2 integers, each at least 1. 'filename' is a string specifying the filename of the image containing the tileset. It is required. 'x' is an integer specifying the horizontal resolution of the whole tileset. It is required and must be at least 1. 'y' is an integer specifying the vertical resolution of the whole tileset. It is required and must be at least 1. 'terrains' is an array specifying the IDs of the terrain types used in the tileset. It is required and must contain at least 2 items, each a string matching the pattern '^[0-9a-zA-Z_-]+$'. No additional properties are allowed at the root level.
