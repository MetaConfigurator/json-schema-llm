The schema describes a configuration for an identity pool. It is an object type. The main properties of the schema are: 'IdentityPoolId' is a string that must match the pattern '[\w-]+:[0-9a-f-]+' and is required. 'AllowUnauthenticatedIdentities' is a boolean and is required. 'IdentityPoolName' is a string that must match the pattern '[\w ]+' and is required. 'DeveloperProviderName' is a string that must match the pattern '[\w._-]+' and is optional. 'OpenIdConnectProviderARNs' is an array of strings with unique items and is optional. 'SupportedLoginProviders' is an object containing properties 'graph.facebook.com', 'accounts.google.com', 'www.amazon.com', 'api.twitter.com', and 'www.digits.com', all of which are strings and optional. No additional properties are allowed at the root level.
