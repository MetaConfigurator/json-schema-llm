The schema describes the format of the geometrical tetrahedron-volume description. It is an object type. The main properties of the schema are: 'Nodes' is an array that represents the coordinates in space where the vertices/nodes of the mesh are located. It is required. Each item must be an array with exactly 3 numbers. 'Elements' is an array that describes the four node indices which together make up a tetrahedron. It is optional. Each item must be an array with exactly 4 integers, each greater than or equal to 0. 'Values' is an array that represents the individual mesh objects. It is required. Each item must be an object containing the required properties 'ElementIndices', 'MediumName', and 'Id'. 'ElementIndices' is an array of integers, each greater than or equal to 0, with at least 1 item. 'MediumName' is a string with a minimum length of 1. 'Id' is an integer, greater than -1, intended to be unique among all volumes. No additional properties are allowed at the root level.
