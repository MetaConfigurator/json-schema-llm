The schema describes a heading block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$', and it can be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string formatted as date-time, representing an update that occurred on the node this block is attached to. It is optional. 'aside' is a boolean indicating if the block represents a portion of a document whose content is only indirectly related to the document's main content. It is optional. 'text' is a string with a minimum length of 0 and a maximum length of 2000 characters. It is optional. 'size' is an integer with a default value of 0, a minimum of 0, and a maximum of 6. It is optional. 'url' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$'. It is optional. No additional properties are allowed at the root level.
