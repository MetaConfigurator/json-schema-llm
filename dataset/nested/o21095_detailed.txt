The schema describes a framework configuration. It is an object type. The main properties of the schema are: '_id' is an integer, optional. 'allow_declaration_reuse' is a boolean, optional. 'applications_close_at_utc' is a string formatted as date-time, optional. 'clarification_questions_open' is a boolean, optional. 'clarifications_close_at_utc' is a string formatted as date-time, optional. 'clarifications_publish_at_utc' is a string formatted as date-time, optional. 'framework' is a string, required. 'framework_agreement_details' is an object, optional. 'framework_expires_at_utc' is a string formatted as date-time, optional. 'framework_live_at_utc' is a string formatted as date-time, optional. 'has_direct_award' is a boolean, required. 'has_further_competition' is a boolean, required. 'intention_to_award_at_utc' is a string formatted as date-time, optional. 'lots' is an array of objects, optional, with a description indicating a relationship link to multiple Lot instances. 'name' is a string with a maximum length of 255 characters, required. 'slug' is a string, required. 'status' is a string, optional. No additional properties are allowed at the root level.
