The schema describes a bibliography entry. It is an object type. The main properties of the schema are: 'id' is a string unique entry id used to reference entries, required; 'title' is a string paper title, required; 'authors' is an array of objects, containing 'name' which is a required string full name of the author and an optional 'affiliation' which is a string, with a minimum of 1 item, and unique items; 'year' is an integer the publication year, required; 'peer-reviewed' is a boolean was this publication reviewed for a Conference or Journal?; 'venue' is a string name of the Conference or Journal this publication was published in, required if peer-reviewed is set; 'abstract' is a string; 'url' is a string URL where the publication can be downloaded; 'type' is a string the Bibtex type, with allowed values: 'inproceedings', 'masterthesis', 'article', 'book', 'techreport', 'misc', 'phdthesis', 'online', 'incollection', 'inbook'; 'sortkey' is an integer the key by which publications are sorted. No additional properties are allowed at the root level.
