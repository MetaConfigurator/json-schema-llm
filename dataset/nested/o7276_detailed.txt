The schema describes a package configuration. It is an object type. The main properties of the schema are: 'formats' is an array that lists available parsing formats configuration. Each item in the array is an object with a required field 'name' of type string, and may include optional fields 'email' of type string (formatted as email), 'homepage' of type string (formatted as uri), and 'role' of type string. 'config' is an object representing composer options, which may include an optional field 'process-timeout' of type integer describing the timeout in seconds for process executions, defaulting to 300 (5 mins). No additional properties are allowed at the root level.
