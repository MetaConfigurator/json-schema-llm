The schema describes the configuration for publishing various streams and inspecting page information. It is an object type. The main properties of the schema are: 'events' is an object that publishes a stream of event bus activity through wrapper events. It has an optional property 'stream' which is a string describing the topic through which to publish event items, with a default value of null. 'log' is an object that publishes a stream of log items through wrapper events. It has an optional property 'stream' which is a string describing the topic through which to publish log items, with a default value of null. 'pageInfo' is an object that publishes the current page for inspection. It has an optional property 'resource' which is a string describing the topic through which to publish the current page, with a default value of null. 'grid' is an object that publishes application-specific grid configuration set by the host application. It has an optional property 'resource' which is a string describing a resource through which to publish grid settings, with a default value of null. 'development' is an object that contains an optional property 'liveReload' which is a boolean with a default value of true. No additional properties are allowed at the root level.
