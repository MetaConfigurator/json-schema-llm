The schema describes a shipment pickup configuration. It is an object type. The main properties of the schema are: 'carrier' is a string that specifies the acronym of the carrier you want to use. It is required and must be one of the following values: 'ups', 'dpd', 'hermes', 'gls'. 'pickup_address' is an object that details the address where the carrier should pick up shipments. It is optional but, if present, must include the required properties: 'last_name', 'street', 'street_no', 'city', 'zip_code', and 'country'. 'pickup_date' is a string representing a deprecated date format, which should follow the pattern 'YYYY/MM/DD'. 'pickup_time' is an object that defines a time window for the pickup. It is required and must have the properties 'earliest' and 'latest', both of which are of type string and conform to the date-time format. No additional properties are allowed at the root level.
