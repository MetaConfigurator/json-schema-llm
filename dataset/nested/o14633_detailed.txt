The schema describes a configuration for analyses. It is an object type. The main properties of the schema are: 'version' is a string, optional. 'resources' is an object, optional, containing 'reference' as a string, optional, and 'dbsnp' as a string, optional. 'globalQsub' is an object, optional. 'analyses' is an array, required, consisting of objects with the following properties: 'analysis' is a string, optional; 'description' is a string, optional; 'inputFormat' is a string, optional; 'program' is a string, optional; 'enable' is an integer, optional, with allowed values 0 or 1; 'outdirs' is an array of objects, optional; 'outfiles' is an array of objects, optional; 'dependencies' is an array of strings, optional; 'modules' is an array of strings, optional; 'jar' is a string, optional; 'javavm' is a string, optional; 'qsub' is an object, optional, containing 'name' as a string, optional; 'slots' as an integer, optional; 'mem' as a string, optional; 'time' as a string, optional; 'directory' as a string, optional; 'queue' as a string, optional; 'project' as a string, optional; 'host' as a string, optional; 'pe' as a string, optional; 'account' as a string, optional; 'params' as an object, optional; 'args' is an array of command-line arguments, optional. No additional properties are allowed at the root level.
