The schema describes Menus configuration. It is an object type. It defines several reusable components under 'definitions': 'menuItem' is an object, Single menu item (hierarchical) which requires 'label' and 'url'. It has optional fields 'icon', 'desc', 'target', and 'order'. It also contains a 'children' array that can have items that conform to the 'menuItem' definition. No additional properties are allowed at the root level. The main properties of the schema are: 'main_menu' is an array of items, each of which must conform to the 'menuItem' definition; 'admin_menu' is also an array of items, each conforming to the 'menuItem' definition.
