The schema describes a configuration for article navigation and filtering. It is an object type. The main properties of the schema are: 'navigation' is an object that describes the navigation parameters, required to have the 'parameterName' property which is a string in 'topic' format, allowing for search string usage. 'articles' is an object that details the resource for the unfiltered articles list, required to have the 'resource' property which is a string in 'topic' format, representing the ID of the resource. 'filteredArticles' is an object that specifies the resource for the filtered articles list, required to have the 'resource' property which is a string in 'topic' format, indicating the ID of the resource for the filtered articles. No additional properties are allowed at the root level.
