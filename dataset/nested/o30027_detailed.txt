The schema describes application firewall settings. It is an object type. The main properties of the schema are: 'builtin' is an object that is readonly and has an enum with values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'ceflogging' is a string with an enum of 'ON', 'OFF'. 'clientiploggingheader' is a string. 'cookiepostencryptprefix' is a string. 'defaultprofile' is a string. 'entitydecoding' is a string with an enum of 'ON', 'OFF'. 'feature' is a string that is readonly and has a long enum with multiple values including 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', 'LoadBalancing', 'CS', 'ContentSwitching', 'CR', 'CacheRedirection', and others. 'geolocationlogging' is a string with an enum of 'ON', 'OFF'. 'importsizelimit' is an integer. 'learnratelimit' is an integer. 'logmalformedreq' is a string with an enum of 'ON', 'OFF'. 'malformedreqaction' is an object with an enum of 'none', 'block', 'log', 'stats'. 'sessioncookiename' is a string. 'sessionlifetime' is an integer. 'sessionlimit' is an integer. 'sessiontimeout' is an integer. 'signatureautoupdate' is a string with an enum of 'ON', 'OFF'. 'signatureurl' is a string. 'undefaction' is a string. 'useconfigurablesecretkey' is a string with an enum of 'ON', 'OFF'. No additional properties are allowed at the root level.
