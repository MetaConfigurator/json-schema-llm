The schema describes a notification of triggering runnable jobs on a resultset. It is an object type. The main properties of the schema are: 'version' is a string, required, with an enum value of 1. 'project' is a string, required, with a description as an identifier for a treeherder project, like `try` or `mozilla-central`. 'resultset_id' is a string, required, with a description as the project unique identifier for a resultset. 'requester' is a string, required, with a description as the requester of the action (usually an email). 'requested_jobs' is an array, required, with a description as the builder names and TaskLabels that should be added to a push. 'decision_task_id' is a string, required, with a description as the Gecko Decision Task's Task ID for this resultset. 'timestamp' is a string, optional, with a description as the UTC timestamp. No additional properties are allowed at the root level.
