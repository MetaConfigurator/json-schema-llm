The schema describes all the Contact Information. It is an object type. The main properties of the schema are: 'contactType' is a string that specifies the type of contact information (e.g. "emergency", "operations") and is required. 'name' is a string that indicates the name of the contact point and is required. 'phoneNumber' is an array of strings containing phone numbers at which this entity can be reached and is required. 'email' is an array of strings formatted as email containing email contact information for this entity and is required. 'address' is an array of strings providing the address for this entity and is required. 'note' is an array of strings giving human-readable information for this entity and is required. No additional properties are allowed at the root level.
