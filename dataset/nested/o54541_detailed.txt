The schema describes BarChart settings. It is an object type. The main properties of the schema are: 'title' is a string optional. 'description' is a string optional. 'xLabel' is a string optional. 'yLabel' is a string optional. 'legendPosition' is an enumerated type with allowed values: 'top', 'right', 'bottom', 'left', 'in', 'none', optional. 'orientation' is an enumerated type with allowed values: 'vertical', 'horizontal', optional. 'style' is an object optional, containing properties: 'maxValue' is a number or null optional. 'minValue' is a number or null optional. 'barArea' is a string matching the pattern '^[0-9]{1,3}%$' optional. 'showGrid' is a boolean optional. 'animationDuration' is an integer with a minimum value of 0 optional. No additional properties are allowed at the root level.
