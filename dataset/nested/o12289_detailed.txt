The schema describes a configuration for a table. It is an object type. The main properties of the schema are: 'head' is a boolean with a default value of true. 'subTotal' is a boolean with a default value of false. 'lockableRows' is a boolean with a default value of false. 'autoSort' is a boolean with a default value of false. 'elementsInPage' is a number with a default value of 8. 'stickedColumns' is a number with a default value of 2. 'columns' is an array with a default value of an empty array, containing objects with properties: 'key' (a string), 'label' (a string), 'iconName' (a string), 'prefix' (a string), 'className' (a string), 'isSortable' (a boolean), 'isSticked' (a boolean), 'sortDesc' (a boolean), 'footerLabel' (a string), 'linkLabel' (a string), 'hasCustomClass' (a boolean), 'hasShadow' (a boolean), 'beDisabled' (a boolean), 'hasSubTotal' (a boolean), and 'actions' (an array containing objects with properties: 'key' (a string), 'name' (a string), 'tooltip' (a string), 'tooltipKey' (a string), 'className' (a string), 'label' (a string), 'hideable' (a boolean), and 'beDisabled' (a boolean)). No additional properties are allowed at the root level.
