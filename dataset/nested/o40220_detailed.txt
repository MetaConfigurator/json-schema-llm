The schema describes a JSON-RSS feed. It is an object type. The main properties of the schema are: 'version' is a string and is required. 'channel' is an object and is required. It contains: 'title' is a string and is required. 'link' is a string and is required. 'description' is a string and is required. 'language' is a string and is optional. 'lastBuildDate' is a string and is optional. 'items' is an array of objects and is optional, where each item must have: 'title' is a string and is required. 'description' is a string and is required. 'contentEncoded' is a string and is optional. 'link' is a string and is required. 'pubDate' is a string and is optional. 'guid' is a string and is optional. 'categories' is an array of strings and is optional. 'georssPoint' is a string and is optional. No additional properties are allowed at the root level.
