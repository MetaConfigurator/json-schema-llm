The schema describes a stacktrace frame. It is an object type. The main properties of the schema are: 'abs_path' is a string or null, describing the absolute path of the file involved in the stack frame. 'colno' is a number or null, describing the column number. 'context_line' is a string or null, describing the line of code part of the stack frame. 'filename' is a string, describing the relative filename of the code involved in the stack frame, used e.g. to do error checksumming. 'function' is a string or null, describing the function involved in the stack frame. 'in_app' is a boolean or null. 'lineno' is a number, describing the line number of code part of the stack frame, used e.g. to do error checksumming. 'module' is a string or null, describing the module to which the frame belongs. 'post_context' is an array or null, describing the lines of code after the stack frame with a minimum of 0 items. 'pre_context' is an array or null, describing the lines of code before the stack frame with a minimum of 0 items. 'vars' is an object or null, describing local variables for this stack frame with no defined properties. The schema requires 'filename' and 'lineno'. No additional properties are allowed at the root level.
