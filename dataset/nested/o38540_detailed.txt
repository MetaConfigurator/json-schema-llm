The schema describes the parameters for generating Secret Santa messages and emails. It is an object type. The main properties of the schema are: 'message' is a string that represents the message to be sent in the email and is required. 'subject' is a string that represents the subject of the email and is required. 'participants' is an array that represents a participant of the Secret Santa, which is required. Each item in the 'participants' array is an object with the following properties: 'name' is a string that represents the participant's name and is required; 'email' is a string that represents the participant's email and is required; 'exList' is an array that represents a list of participant IDs whom this participant cannot buy for, with unique items required, and is required; 'id' is an integer that represents a unique identifier for the participant and is required. No additional properties are allowed at the root level.
