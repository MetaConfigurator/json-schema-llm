The schema describes a configuration for invoking a process related to Retinotopy Template Generation. It is an object type. The main properties of the schema are: 'inputs' is an object containing required property 'anatomical' which is also an object having a property 'type' defined as an enum with allowed value 'nifti', and 'config' is an object containing required properties: 'convert_aseg_stats', 'convert_surfaces', 'license_email', 'subject_id', 'convert_volumes', 'license_number', and 'license_key'. The 'convert_aseg_stats', 'convert_surfaces', and 'convert_volumes' properties are integers with a default value of 1. The 'subject_id' property is a string with a default value of 'rt_sub000'. The 'license_email', 'license_number', and 'license_key' properties are strings. No additional properties are allowed at the root level.
