The schema describes a unit hierarchy and board members. It is an object type. The main properties of the schema are: 'units' is an array of objects representing organizational units, each with required fields 'unitId' (an integer, minimum 1, maximum 9999), 'unitOf' (an integer, minimum 0, maximum 9999), and 'name' (a string). 'board' is an array of objects representing board members, each with required fields 'name' (a string), 'independent' (a boolean), and 'chair' (a boolean). No additional properties are allowed at the root level.
