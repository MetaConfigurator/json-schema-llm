The schema describes configuration for a piece of video content, over a stream. It is an object type. The main properties of the schema are: 'height' is a number that describes the height of the video. 'width' is a number that describes the width of the video. 'filesize' is a number that describes the size of the video, in bytes. 'audio_codec' is a string that describes the audio codec. 'video_codec' is a string that describes the video codec. 'stream_type' is a string that describes the type of video (e.g. mp4). 'url' is a string. 'bitrate' is a number that describes the bitrate of the video. 'provider' is a string that describes the provider of the video. No additional properties are allowed at the root level.
