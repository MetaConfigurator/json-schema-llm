The schema describes a theme entry in the Genesis Child Theme Index. It is an object type. The main properties of the schema are: 'accessibilityReady' is a boolean indicating true if the theme meets all of the accessibility guidelines, false if not. 'author' is a string representing the name of the individual or company credited as the author of the theme. 'browsersSupported' is an object that describes key-value pairs of browser names and their minimum supported versions, including properties for 'Chrome', 'Firefox', 'IE', 'Opera', and 'Safari', all of which are numbers. 'customBackground' is a boolean indicating if the theme supports the WordPress custom background feature. 'customColors' is a boolean indicating if the theme allows users to choose custom colors via an interface. 'demoUrl' is a string in URI format specifying the URL where a demo can be found. 'editorStyle' is a boolean indicating if the theme includes styles for the post editor matching the front-end. 'featuredImageHeader' is a boolean indicating if the theme displays a featured image as a header. 'free' is a boolean indicating if the theme is free. 'grunt' is a boolean indicating if the theme includes support for *grunt* for automated development tasks. 'html5' is a boolean indicating if the theme supports HTML5. 'layouts' is an integer representing the number of layouts the theme supports, with a minimum value of 1. 'mobileMenu' is a boolean indicating if the theme displays a mobile-specific menu at small screen sizes. 'name' is a string representing the name of the theme. 'parallax' is a boolean indicating if the theme design includes a parallax feature. 'pluginsSupported' is either a boolean, with the value false, or an array of strings representing a list of plugin names, with at least one item. 'postFormats' is either a boolean with the value false or an array of strings, with allowed values including 'aside', 'audio', 'chat', 'gallery', 'image', 'link', 'quote', 'status', 'video', with at least one item and a maximum of 9. 'psds' is a boolean indicating if the theme includes Photoshop files. 'purchaseUrl' is a string in URI format specifying the URL of the purchase or download page. 'responsive' is a boolean indicating if the theme displays well at various screen sizes. 'rtlLanguageSupport' is a boolean indicating if the theme includes right-to-left style sheets. 'starter' is a boolean indicating if the theme is considered a starter theme. 'translationReady' is a boolean indicating if the theme has all strings internationalised, loads a child theme text domain, and includes a .pot file. No additional properties are allowed at the root level.
