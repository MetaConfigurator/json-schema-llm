The schema describes a configuration for a system. It is an object type. It defines several reusable components under 'definitions': 'IConfiguration' is an object, which has an optional property 'pull' that references 'IPullOptions'. 'IPullOptions' is an object, which requires the property 'connection' (referencing 'MysqlConnectionOptions') and has optional properties 'index' (a boolean), 'interactive' (a boolean), and 'output' (a string). 'MysqlConnectionOptions' is an object, which requires the properties 'adapter' (an enum with the value 'mysql'), 'database' (a string), 'host' (a string), 'password' (a string), and 'user' (a string). It has optional properties 'connectTimeout' (a number) and 'port' (a number). The root schema resolves to the referenced definition 'IConfiguration'. The main properties of the schema are: 'pull' is an optional property that is a reference to 'IPullOptions'. No additional properties are allowed at the root level.
