The schema describes an address for shipments to or from. It is an object type. The main properties of the schema are: 'company' is a string optional, 'first_name' is a string optional, 'last_name' is a string required, 'care_of' is a string optional, 'street' is a string required, 'street_no' is a string required, 'city' is a string required, 'zip_code' is a string required, 'state' is a string optional, 'country' is a string required, described as 'Country as uppercase ISO 3166-1 alpha-2 code', 'phone' is a string optional, described as 'telephone number (mandatory when using UPS and the following terms apply: service is one_day or one_day_early or ship to country is different than ship from country)'. No additional properties are allowed at the root level.
