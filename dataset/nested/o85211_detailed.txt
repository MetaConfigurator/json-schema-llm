The schema describes a MWF supplemental navigation component's schema definition. It is an object type. The main properties of the schema are: 'content' is a object that is required. It contains a required property 'links', which is an array that must have items of type object. Each object in the array requires 'config' and 'content' properties. The 'config' property is an object that requires 'state', which is a string that must be either 'collapsed' or 'expanded'. The 'content' property is also an object that requires 'id', 'text', and 'links'. The 'id' is a string, 'text' is a string, and 'links' is another array of objects where each object requires 'config' and 'content'. The 'config' inside this array must have a boolean 'active' property, while the 'content' must include a required string 'text'. No additional properties are allowed at the root level.
