The schema describes a Per-App VPN configuration payload. It is an object type. The main properties of the schema are: 'OnDemandMatchAppEnabled' is a boolean that indicates whether the Per-App VPN connection starts automatically when apps linked to this service initiate network communication. 'PayloadDescription' is a string that describes the payload and has a default value of 'Per-App VPN Payload'. 'PayloadDisplayName' is a string that names the payload and has a default value of 'Per-App VPN Payload'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, with a default value of 'com.apple.vpn.managed.applayer'. 'PayloadOrganization' is a string describing the issuing organization of the profile. 'PayloadType' is a string indicating the type of the payload, which has a default value of 'com.apple.vpn.managed.applayer'. 'PayloadUUID' is a string that provides a unique identifier for the payload following a specific pattern and has a default value of ''. 'PayloadVersion' is a number representing the version of the configuration profile, with a default value of 1. 'SafariDomains' is an array of strings, where each string is a domain or partial domain name that should trigger the VPN connection in Safari, and this property is optional. No additional properties are allowed at the root level.
