The schema describes a structure for student grades and metadata. It is an object type. The main properties of the schema are: 'grades' is an array containing objects, each of which must have the required properties: 'mark' (a number), 'skill_id' (a number), 'lesson_id' (a string), 'student_id' (a number), and 'deleted_at' (a date-time string or null). 'meta' is an object that must contain the required property: 'timestamp' (a date-time string). No additional properties are allowed at the root level.
