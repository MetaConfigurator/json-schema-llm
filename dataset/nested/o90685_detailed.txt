The schema describes a configuration for a schema registry. It is an object type. The main properties of the schema are: 'registry' is an object that contains the configuration settings for the schema registry, including 'cpus', 'instances', 'kafkastore-connection-url', 'mem', 'name', 'role', and 'vip-label'. It is required. Required fields within 'registry' are 'cpus', 'mem', 'instances', and 'name'. 'cpus' is a number with a minimum value of 1 and a default of 1, 'instances' is an integer with a minimum value of 1 and a default of 1, 'kafkastore-connection-url' is a string with a default of 'leader.mesos:2181/confluent', 'mem' is a number with a minimum value of 256.0 and a default of 1024.0, 'name' is a string with a default of 'schema-registry', 'role' is a string with a default of '*', and 'vip-label' is a string with a default of '8.0.8.1:8081'. No additional properties are allowed at the root level.
