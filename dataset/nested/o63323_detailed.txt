The schema describes NgRx Entity Options. It is an object type. The main properties of the schema are: 'creators' is a boolean that specifies whether to use creator functions for handling actions and reducers. It is optional and defaults to true. 'flat' is a boolean flag to indicate if a dir is created. It is optional and defaults to true. 'group' is a boolean that groups actions, reducers, and effects within relative subfolders. It is optional and defaults to false. 'module' is a string that specifies the declaring module. It is optional. 'name' is a string representing the name of the entity. It is optional and has a default sourced from the first index of argv. 'path' is a string for the path to create the component, formatted as a path. It is optional. 'project' is a string representing the name of the project. It is optional. 'reducers' is a string that specifies the reducers file. It is optional. 'skipTests' is a boolean that, when true, does not create test files. It is optional and defaults to false. No additional properties are allowed at the root level.
