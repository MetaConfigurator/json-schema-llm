The schema describes stages. It is an object type. It defines several reusable components under 'definitions': 'text' is a oneOf type, allowing either a string or an array of strings with a minimum item count of 1. 'break' is an enum type, allowing values 'none', 'paragraph', or 'page'. The main properties of the schema are: 'start' is a reference to the 'break' definition, optional. 'text' is a reference to the 'text' definition, optional. 'end' is a reference to the 'break' definition, optional. 'stage' is a string, optional. 'choices' is an array of objects, which is required to have at least 1 item. Each object in 'choices' has required properties: 'text' (string) and 'prompt' (reference to the 'text' definition), with additional properties not allowed. No additional properties are allowed at the root level.
