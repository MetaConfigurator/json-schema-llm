The schema describes the configuration for DHCP subnet settings. It is an object type. The main properties of the schema are: 'dhcp_subnet' is an array that contains objects representing individual DHCP subnet configurations. Each object requires the fields 'subnet_address', 'subnet_domain', 'subnet_gateway', and 'subnet_mask'. The optional fields include 'default_lease_time', which is an integer with a default value of 21600, 'dhcp_range', which is a null with a default value of null, 'dns_server_list', which is a string with a default value of '['127.0.0.1', '8.8.8.8']', 'max_lease_time', which is an integer with a default value of 43200, 'search_domains_list', which is a string with a default value of '['englab.juniper.net']', and 'subnet_mask', which is a string with a default value of '255.255.255.0'. No additional properties are allowed at the root level.
