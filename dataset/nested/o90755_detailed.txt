The schema describes the configuration for a Marathon Load Balancer (marathon-lb). It is an object type. The main properties of the schema are: 'marathon-lb' is an object that defines various settings for the load balancer instance. It is required and contains the following required properties: 'cpus' is a number that specifies the CPU shares to allocate to each marathon-lb instance. It has a default value of 2 and a minimum value of 1. 'mem' is a number that indicates the memory (MB) to allocate to each marathon-lb task. It has a default value of 1024.0 and a minimum value of 256.0. 'haproxy-group' is a string representing the HAProxy group parameter. It has a default value of 'external'. 'instances' is an integer that determines the number of instances to run. It has a default value of 1 and a minimum value of 1. 'name' is a string that provides a name for this LB instance, with a default value of 'marathon-lb'. No additional properties are allowed at the root level.
