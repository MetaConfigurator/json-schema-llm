The schema describes a weather data structure for Fibaro Home Center. It is an object type. The main properties of the schema are: 'ConditionCode' is a string required, 'Humidity' is a string required, 'PreviousConditionCode' is a string required, 'PreviousHumidity' is a string required, 'PreviousTemperature' is a string required, 'PreviousWeatherConditionConverted' is a string required, 'PreviousWind' is a string required, 'Temperature' is a string required, 'WeatherCondition' is a string required, 'WeatherConditionConverted' is a string required, 'Wind' is a string required, 'saveLogs' is a string required with allowed values of '0' and '1', 'TemperatureUnit' is a string required with allowed values of 'C' and 'F'. No additional properties are allowed at the root level.
