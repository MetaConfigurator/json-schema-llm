The schema describes a course registration system. It is an object type. The main properties of the schema are: 'course' is an object that contains properties 'id' (an integer), 'number' (a string), 'title' (a string), and 'department_id' (an integer). 'quarter' is an object that includes properties 'id' (an integer), 'year' (an integer), 'name' (a string), and 'current' (a boolean). 'professor' is an object that consists of properties 'id' (an integer), 'first_name' (a string), and 'last_name' (a string). 'user_posted' is a boolean property. No additional properties are allowed at the root level.
