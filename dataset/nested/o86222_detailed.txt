The schema describes scheduling information. It is an object type. The main properties of the schema are: 'planning' is an object containing planning information, which is optional and consists of further properties. The 'scheduling' property is an object containing scheduling details. It has the following properties: 'planned_publish_date' is a string representing when the content is planned to be published, using the date-time format; 'scheduled_publish_date' is a string representing when the content was first published, also using the date-time format; 'will_have_gallery' is a boolean indicating if this content will have galleries; 'will_have_graphic' is a boolean indicating if this content will have graphics; 'will_have_image' is a boolean indicating if this content will have images; 'will_have_video' is a boolean indicating if this content will have videos. The 'story_length' property is an object containing story length information. It includes the following properties: 'word_count_planned' is an integer for the anticipated number of words in the story; 'word_count_actual' is an integer for the current number of words; 'inch_count_planned' is an integer for the anticipated length of the story in inches; 'inch_count_actual' is an integer for the current length of the story in inches. No additional properties are allowed at the root level.
