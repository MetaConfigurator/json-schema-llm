The schema describes gadget definition. It is an object type. The main properties of the schema are: 'module' is an object that contains details about the gadget's components. It has the following properties: 'dependencies' is an optional array that lists ResourceLoader modules this gadget depends upon. 'messages' is an optional array that lists messages this gadget depends upon. 'position' is an optional string, defaults to 'bottom', with allowed values 'top' and 'bottom', indicating whether this module should be loaded asynchronously after the page loads (bottom) or synchronously before the page is rendered (top). 'scripts' is an optional array that lists JavaScript pages included in this gadget. 'styles' is an optional array that lists CSS pages included in this gadget. 'settings' is an object that contains configuration details for the gadget. It has the following properties: 'category' is an optional string, defaults to an empty string, representing the key of the category this gadget belongs to. 'default' is an optional boolean, defaults to false, indicating whether this gadget is enabled by default. 'hidden' is an optional boolean, defaults to false, indicating whether this gadget is hidden from preferences. 'rights' is an optional array of strings representing the rights required to be able to enable/load this gadget. 'skins' is an optional property that can be either an array of strings representing skins supported by this gadget or a boolean indicating that all skins are supported. No additional properties are allowed at the root level.
