The schema describes an IFrame block. It is an object type. The main properties of the schema are: 'etag' is a string matching the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string matching the pattern '^[\w\s-]+$'. It provides a class attribute for DOM elements and is optional. 'updated_date' is a string in date-time format, representing the last update date of the associated node. It is optional. 'aside' is a boolean indicating if the block is an aside from the main content, and is optional. 'src' is a string matching the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$'. It is required. 'width' is a string matching the pattern '^\d+(%|px)?$'. It is optional. 'height' is a string matching the pattern '^\d+(%|px)?$'. It is optional. 'align' is a string matching the pattern '^(center|right|left)$'. It is optional. 'data' is an object with properties that can be any string matching the pattern '^[a-zA-Z_]{1}[\w\.:-]+$' and may have values that are strings with a minimum length of 0 and a maximum length of 255. It is optional. No additional properties are allowed at the root level.
