The schema describes a sensor event. It is an object type. The main properties of the schema are: 'name' is a string that represents the device name and is required. 'cpu' is a number that specifies the CPU utilization percentage, required, and must be between 0.0 and 100.0, with a default value of 0.0. 'mem' is a number that indicates the memory utilization percentage, required, and must be between 0.0 and 100.0, with a default value of 0.0. 'network' is an object that includes 'up' and 'down'; both are required numbers representing network utilization in KB/s, must be greater than or equal to 0.0, and have a default value of 0.0. No additional properties are allowed at the root level.
