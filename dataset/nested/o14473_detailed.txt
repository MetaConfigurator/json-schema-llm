The schema describes an event source template. It is an object type. The main properties of the schema are: 'name' is a string A name to identify the event source template. It is required. 'organizationId' is an integer The organization to link the event source template. It is required. 'public' is a boolean Make the event source template available to everybody. It is optional. 'configuration' is an object Providing a schema aims to help the WUI for configuration. It is optional. It contains required properties: 'schema', which is an object A valid JSON Schema, and 'url', which is a string The configuration URL. 'configurationUi' is an object Provide the configuration for the dynamic UI. It is optional. It contains 'schemaForm', which is an array Valid form schema accordingly to: http://schemaform.io. No additional properties are allowed at the root level.
