The schema describes a channon plan. It is an object type. The main properties of the schema are: 'name' is a string required. 'description' is a string optional. 'steps' is an array of objects, each containing a 'name' (string required) and a 'payload' (string required with a pattern matching '^#!'). 'notifications' is an array of objects, each containing a 'target' (string required, enum values: 'every-run', 'every-failure', 'result-changed') and a 'payload' (string required with a pattern matching '^#!'). 'triggers' is an array of objects, each containing a 'type' (string required) and an optional 'schedule' (string). 'tags' is an array of strings optional. No additional properties are allowed at the root level.
