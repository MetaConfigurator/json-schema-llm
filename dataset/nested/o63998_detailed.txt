The schema describes an object representing graphical or data elements. It is an object type. The main properties of the schema are: 'cx' is a number and required, 'cy' is a number and required, 'id' is a number and required, 'locked' is a boolean and required, 'name' is a string and required, 'render' is a boolean and required, 'source_cx' is a number and required, 'source_cy' is a number and required, 'type' is a string and required, 'volume' is a number and required, 'x' is a number and required, 'y' is a number and required, 'parentGroupName' is a string and optional, 'groupChildren' is an array of items and optional. No additional properties are allowed at the root level.
