The schema describes configuration for an NgAdd setup. It is an object type. The main properties of the schema are: 'i18nFormat' is a string that indicates the translation file format, with default 'xlf' and allowed values 'xlf', 'xlf2', 'xmb'. 'i18nLocale' is a string describing the default language used in ng template files; if not provided, defaults to the first language in the list or 'en'. 'languages' is a string representing target languages to support as a comma-separated list. 'localePath' is a string that specifies the optional path for creating translation files, defaulting to 'i18n'. 'path' is a string describing the optional path for the xliffmerge configuration, not visible. 'project' is a string that provides the name of the project. 'skipInstall' is a boolean that, when true, indicates not to install dependency packages, with default false. 'useComandlineForLanguages' is a boolean that, when set, passes all languages as command line arguments to xliffmerge, with default false. 'useXliffmergeBuilder' is a boolean that configures the use of the architect builder for xliffmerge when true, with default true. No additional properties are allowed at the root level.
