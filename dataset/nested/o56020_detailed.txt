The schema describes Fibaro Home Center User JSON Schema. It is an object type. The main properties of the schema are: 'id' is an integer identifier for the user, optional, with a minimum value of 0. 'name' is a string name of the user, required, with a maximum length of 20. 'type' is a string type of the user, optional, with allowed values 'user' and 'superuser'. 'email' is a string email of the user, optional, formatted as an email. 'hasGPS' is a boolean flag indicating if the user has GPS, optional. 'deviceRights' is an array of numbers representing the list of devices visible for the user, optional, with items having a minimum value of 0. 'sceneRights' is an array of numbers representing the list of scenes visible for the user, optional, with items having a minimum value of 0. 'hotelRoom' is an integer hotel room number, optional. 'sendNotifications' is a boolean flag indicating if Fibaro shall send notifications, optional. 'tracking' is an integer representing how frequently GPS position from the user's iPhone should be collected in minutes, optional. 'usePin' is a boolean flag for using a PIN, optional. 'useOptionalArmPin' is a boolean flag for using an optional arm PIN, optional. 'pin' is a string representing a PIN instead of a password for confirmations, optional. 'password' is a string representing the password, optional. No additional properties are allowed at the root level.
