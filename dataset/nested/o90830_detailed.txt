The schema describes configuration settings for a service and Spark. It is an object type. The main properties of the schema are: 'service' is an object that contains settings for the Zeppelin service. It has optional properties: 'name' which is a string with a default value of 'zeppelin', and 'zeppelin_java_opts' which is a string with a default value of '-Dspark.mesos.coarse=true -Dspark.mesos.executor.home=/opt/spark/dist' and is used to set arbitrary Spark configuration properties. 'spark' is an object that contains configurations for Spark. It has optional properties: 'cores_max' which is a string with a default value of '', 'executor_docker_image' which is a string with a default value of 'mesosphere/spark:1.1.0-2.1.1-hadoop-2.7', 'executor_memory' which is a string, and 'uri' which is a string with a default value of 'https://downloads.mesosphere.com/spark/assets/spark-2.1.1-bin-2.7.tgz'. No additional properties are allowed at the root level.
