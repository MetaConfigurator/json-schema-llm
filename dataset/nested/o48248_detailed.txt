The schema describes a configuration for a system. It is an object type. The main properties of the schema are: 'Containers' is an integer, optional. 'Debug' is a boolean, optional. 'DockerRootDir' is a string, optional. 'Driver' is a string, optional. 'DriverStatus' is an array of arrays containing strings, optional. 'ExecutionDriver' is a string, optional. 'ID' is a string, optional. 'IPv4Forwarding' is a boolean, optional. 'Images' is an integer, optional. 'IndexServerAddress' is a string, optional. 'InitPath' is a string, optional. 'InitSha1' is a string, optional. 'KernelVersion' is a string, optional. 'Labels' is an array of strings, optional. 'MemTotal' is an integer, optional. 'MemoryLimit' is a boolean, optional. 'NCPU' is an integer, optional. 'NEventsListener' is an integer, optional. 'NFd' is an integer, optional. 'NGoroutines' is an integer, optional. 'Name' is a string, optional. 'OperatingSystem' is a string, optional. 'SwapLimit' is a boolean, optional. No additional properties are allowed at the root level.
