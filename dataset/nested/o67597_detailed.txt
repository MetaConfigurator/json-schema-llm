The schema describes a representation of an ssl certificate. It is an object type. The main properties of the schema are: 'name' is a string for user reference. 'matches' is an array of Url match pattern strings, where each item is a string. 'key' is an object containing the path to the file containing the private key, with a property 'src' that describes the path to the file. 'cert' is an object containing the path to the file certificate, with a property 'src' that describes the path to the file. 'passphrase' is a string for the passphrase. No additional properties are allowed at the root level.
