The schema describes a postal Address. It is an object type. The main properties of the schema are: 'type' is a string that must be set to 'Address' and is readOnly. 'streetAddress' is a string that describes the street address, not including apartment or unit number and is optional. 'apartmentNumber' is a string that describes the apartment or unit number, along with any abbreviations, e.g. 'Apt 20' and is optional. 'postalBoxNumber' is a string that describes the postal box number and is optional. 'locality' is a string that describes the locality or city for this Address and is optional. 'region' is a string that describes the region or state for this Address and is optional. 'country' is a string that describes the country this address is located in and is optional. 'postalCode' is a string that describes the postal code for this address (as a string rather than a number) and is optional. No additional properties are allowed at the root level.
