The schema describes final configuration. It is an object type. The main properties of the schema are: 'result' is a string indicating the result, which is required. 'features' is an array representing which features are selected if result = sat, and is optional. Each item in the array must be a string. 'attributes' is an array indicating the values of the attributes if result = sat, and is optional. Each item in the array must be an object with required properties: 'id' (a string) and 'value' (an integer). 'constraints' is an array representing the constraints that causes the FM to be void, and is optional. Each item in the array must be a string. No additional properties are allowed at the root level.
