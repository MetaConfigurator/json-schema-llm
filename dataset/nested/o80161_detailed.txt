The schema describes an event. It is an object type. The main properties of the schema are: 'name' is a string or an array of strings, required; 'startDate' is a string formatted as a date, required; 'endDate' is a string formatted as a date, required; 'description' is a string or an array of strings, required; 'location' is an object with a 'type' field (which can be 'http://schema.org/Place' or 'http://schema.org/PostalAddress') or a string, required; 'contact' is an object with a 'type' field (which can be 'http://schema.org/Organization' or 'http://schema.org/Person') and 'properties', or an array of such objects, required; 'hostInstitution' is an object with a 'type' field (which can be 'http://schema.org/Organization') or an array of such objects, required; 'eventType' is a string or an array of strings, each being either 'Workshops and courses', 'Meetings and conferences', 'Receptions and networking', or 'Awards and prizegivings', required; 'url' is a string formatted as a URI, optional; 'keywords' is a string or an array of strings, optional; 'topic' is a string or an array of strings formatted as URIs, optional; 'targetAudience' is a string or an array of strings formatted as URIs, optional; 'programme' is a string or a string formatted as a URI, optional; 'deadline' is a string or an array of strings, optional; 'prerequisite' is a string or an array of strings, optional; 'capacity' is an integer, optional; 'eligibility' is a string, or an array of strings, or a string with enum values ('First come first served', 'Registration of interest', 'By invitation'), or an array of strings with enum values, optional; 'alternateName' is a string, optional; 'accreditation' is a string or an object with a 'type' field (which can be 'http://schema.org/Organization'), optional. Compositional constraints apply such that all required properties must be present. No additional properties are allowed at the root level.
