The schema describes a cart operation. It is an object type. The main properties of the schema are: 'article' is an object that represents the article to be added to the cart, required fields are 'resource' and 'onActions'. 'resource' is a string formatted as a topic, with 'axRole' set to inlet, describing the ID of the article resource to add to the cart. 'onActions' is an array of strings formatted as topics, with 'axRole' set to inlet, describing the actions that add the article to the cart. 'order' is an object that represents the order information, with 'target' as the required field, which is a string describing where to be used in the navigateRequest for placing an order. No additional properties are allowed at the root level.
