The schema describes a Docker encapsulated environment. It is an object type. The main properties of the schema are: 'envscript' is a string, optional, with a default value of an empty string. 'par_mounts' is an array of objects, optional, with a default value of an empty array. Each object must have properties 'mountpath' (a string) and 'jqscript' (a string). 'image' is a string, required. 'imagetag' is a string, optional, with a default value of 'latest'. 'environment_type' is a string, required, with a default value of 'docker-encapsulated' and an enum value of 'docker-encapsulated'. 'env' is an object, optional, with a default value of an empty object and no properties defined. 'resources' is an array, optional, with a default value of an empty array. No additional properties are allowed at the root level.
