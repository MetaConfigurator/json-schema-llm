The schema describes a network packet configuration. It is an object type. The main properties of the schema are: 'source_ip' is a string that represents the Source (device loopback IP) and is required with a default value of an empty string. 'destination_ip' is a string that represents the Destination (device loopback IP) and is required with a default value of an empty string. 'vni' is an integer that represents the VNI and is required with a default value of 100. 'count' is an integer that represents the Packet count and is required with a default value of 5. No additional properties are allowed at the root level.
