The schema describes a configuration profile for an AssetCache Client. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string that describes the payload. It is optional and has a default value of 'AssetCache Client'. 'PayloadDisplayName' is a string that represents the name of the payload. It is optional and has a default value of 'AssetCache Client'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, formatted as a dot-delimited string. It is optional and has a default value of 'com.apple.assetcache.client.osx.profile.plugin'. 'PayloadOrganization' is a string providing the name of the issuing organization of the profile, displayed to the user. It is optional. 'PayloadType' is a string that indicates the type of the payload, maintained as a reverse DNS string. It is optional and has a default value of 'com.apple.assetcache.client.osx.profile.plugin'. 'PayloadUUID' is a string that provides a unique identifier for the payload, formatted as '01234567-89AB-CDEF-0123-456789ABCDEF' with a defined pattern. It is optional and has a default value of ''. 'PayloadVersion' is a number indicating the version of the overall configuration profile. It is optional and has a default value of 1. 'PublicIPAddressRanges' is an array of objects where each object contains properties: 'first' (a string), 'last' (a string), and 'type' (a string with a default value of 'IPv4'). No additional properties are allowed at the root level.
