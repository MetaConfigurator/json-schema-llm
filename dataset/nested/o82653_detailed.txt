The schema describes a Facebook video block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute. It is optional. 'updated_date' is a string formatted as date-time, representing an update that occurred on the node. It is optional. 'aside' is a boolean indicating if this block represents content only indirectly related to the main content. It is optional. 'href' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$'. It is required. 'autoplay' is a boolean. It is optional. 'show_text' is a boolean. It is optional. 'show_captions' is a boolean. It is optional. 'poster_image_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and refers to an image asset for the poster. It is optional. No additional properties are allowed at the root level.
