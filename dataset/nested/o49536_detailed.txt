The schema describes a package configuration. It is an object type. The main properties of the schema are: 'description' is a string that provides a description of the package. It is required. 'framework' is a boolean that indicates if this package installs a new Mesos framework. It has a default value of false and is optional. 'images' is an object that contains image properties, which is optional and disallows additional properties. 'maintainer' is a string that specifies the maintainer of the package. It is required. 'name' is a string that gives the name of the package. It is required. 'postInstallNotes' is a string that contains post installation notes for the user. It is optional. 'preInstallNotes' is a string for pre installation notes that would be useful to the user. It is optional. 'scm' is a string which may contain source control management information. It is optional. 'tags' is an array of strings, each following the pattern '^[^\s]+$', representing tags associated with the package. It is required. 'version' is a string that specifies the version of the package. It is required. 'website' is a string that may contain the website URL for the package. It is optional. No additional properties are allowed at the root level.
