The schema describes service configuration settings for a Grafana deployment. It is an object type. The main properties of the schema are: 'grafana' is an object containing configuration properties for the grafana service, which is required. It includes 'admin_password', an optional string with a default value of 'admin'; 'cpus', a required number with a default of 0.3 that must be at least 0.3; and 'mem', a required number with a default of 512.0 that must be at least 512.0. 'networking' is an object containing Grafana networking configuration properties, which is optional. It includes 'external_access', an optional object that enables access from outside the cluster through Marathon-LB. This object has 'enable', an optional boolean with a default value of true; 'external_access_port', an optional number with a default of 13000; and 'virtual_host', an optional string with a default of 'grafana.example.org'. 'service' is an object containing DC/OS service configuration properties, which is optional and includes 'name', an optional string with a default value of 'grafana'. No additional properties are allowed at the root level.
