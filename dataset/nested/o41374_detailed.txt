The schema describes a configuration for an invocation manifest for PrepareFieldmap. It is an object type. The main properties of the schema are: 'inputs' is an object that is required and contains nested properties: 'in_phase' is an object, 'in_magnitude' is an object, and 'out_fieldmap' is an object; all have no defined properties. 'config' is an object that is required and contains nested properties: 'delta_TE' is a number with a default value of 2.46, 'output_type' is a string with a default value of 'NIFTI_GZ' and allowed values 'NIFTI_PAIR', 'NIFTI_PAIR_GZ', 'NIFTI_GZ', 'NIFTI', 'args' is an optional string, 'nocheck' is an optional boolean, and 'scanner' is a string with a default value of 'SIEMENS'. No additional properties are allowed at the root level.
