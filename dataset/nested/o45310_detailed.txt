The schema describes a JSON schema for VSCode coding tracker token file. It is an object type. It defines several reusable components under 'definitions': 'SimpleToken' is an object, which requires a 'token' field and allows an optional 'remark' field. The main properties of the schema are: 'adminToken' is an array that must contain at least one item, where each item references the 'SimpleToken' definition. 'viewReportToken' is either an array or a string, where each item also references the 'SimpleToken' definition. 'uploadToken' is an array that must contain items that are objects, each requiring a 'token' field and allowing an optional 'remark' field and an optional 'computerId' field, which can be either a string or an array of strings. No additional properties are allowed at the root level.
