The schema describes a video record in pytube/data. It is an object type. The main properties of the schema are: 'alias' is a null or string [Deprecated] Legacy path to video on pyvideo.org. 'category' is a null or string [Deprecated] Category of video. This is duplicated by category files and is thus depriciated in video files. 'copyright_text' is a null or string Copyright license for video or link to copyright information. 'description' is a null or string reStructuredText or plain text description of video. 'duration' is a null or integer Duration of video (in seconds). 'id' is a null or integer ID for video. Must be unique amongst all videos if present. 'language' is a null or string ISO 639-3 language code of spoken language used in video. 'quality_notes' is a null or string Notes on quality of video. 'recorded' is a null or string ISO 8601 Date on which video was recorded (YYYY-MM-DD). 'related_urls' is an array of related resources' URLs. 'slug' is a null or string Web-safe slug for video. Must be unique amongst other videos in same category. 'speakers' is an ordered array of speakers in video. 'summary' is a null or string reStructuredText or plain text description of video. 'tags' is an array of tags describing video. 'thumbnail_url' is a null or string URL to thumbnail for video. 'title' is a string Title of video. 'videos' is an array of video locations and types, with at least 1 item. Each item has required properties: 'type' and 'url'. No additional properties are allowed at the root level.
