The schema describes a Star Wars film. It is an object type. The main properties of the schema are: 'species' is an array The species resources featured within this film. 'url' is a string The url of this resource. 'opening_crawl' is a string The opening crawl text at the beginning of this film. 'director' is a string The director of this film. 'episode_id' is an integer The episode number of this film. 'starships' is an array The starship resources featured within this film. 'planets' is an array The planet resources featured within this film. 'vehicles' is an array The vehicle resources featured within this film. 'characters' is an array The people resources featured within this film. 'title' is a string The title of this film. 'producer' is a string The producer(s) of this film. All properties are required. No additional properties are allowed at the root level.
