The schema describes a configuration profile for GameKit. It is an object type. The main properties of the schema are: 'GKFeatureAccountModificationAllowed' is a boolean Optional. Set to false to disable account modifications. 'GKFeatureAddingGameCenterFriendsAllowed' is a boolean Optional. Set to false to disable adding Game Center friends. 'GKFeatureGameCenterAllowed' is a boolean Optional. Set to false to disable Game Center. 'GKFeatureMultiplayerGamingAllowed' is a boolean Optional. Set to false to disable multiplayer gaming. 'PayloadDescription' is a string Optional. Description of the payload. It has a default value of 'GameKit Payload'. 'PayloadDisplayName' is a string Optional. Name of the payload. It has a default value of 'GameKit Payload'. 'PayloadIdentifier' is a string Optional. A unique identifier for the payload, dot-delimited. Usually root PayloadIdentifier+subidentifier. It has a default value of 'com.apple.gamed'. 'PayloadOrganization' is a string Optional. This value describes the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string Optional. The type of the payload, a reverse dns string. It has a default value of 'com.apple.gamed'. 'PayloadUUID' is a string Optional. Unique identifier for the payload (format 01234567-89AB-CDEF-0123-456789ABCDEF). It has a default value of ''. It must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is a number Optional. The version of the whole configuration profile. It has a default value of 1. No additional properties are allowed at the root level.
