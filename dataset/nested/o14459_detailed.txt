The schema describes action target templates. It is an object type. The main properties of the schema are: 'id' is an integer Unique identifier given by the server. It is required. 'name' is a string A name to identify the action target template. It is required. 'public' is a boolean Make the action target template available to everybody. It is required. 'organizationId' is an integer The organization that offer the action target template. It is required. 'configuration' is an object Providing a schema aims to help the WUI for configuration. It is optional and has required fields 'schema' and 'url'. 'configurationUi' is an object Provide the configuration for the dynamic UI. It is optional. 'target' is an object The target configuration. It is required and has a required field 'url'. No additional properties are allowed at the root level.
