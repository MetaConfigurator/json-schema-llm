The schema describes the content of an email. It is an object type. The main properties of the schema are: 'status' is a string that describes the status of the email (queued, failed, sent). 'hash' is a string or null that is used to identify the email. 'content' is an object that contains detailed information about the email, including properties: 'FROM' which is an object with 'name' as a string or null and 'email' as a string or null; 'TO' which is an array of objects with each object having 'name' as a string and 'email' as a string; 'CC' which is an array of objects similarly structured; 'BCC' which is also an array of objects similarly structured; 'BODY' which is a string or null for the body of the email; 'SUBJECT' which is a string or null for the subject of the email; and 'ATTACHMENTS' which is an array of objects with 'PATH' as a string, 'NAME' as a string, and 'WIDTH' as a number. 'logs' is an array of strings. No additional properties are allowed at the root level.
