The schema describes an email configuration. It is an object type. The main properties of the schema are: 'emailDetails' is an object that contains the details of the email, including required fields 'to', 'from', 'subject', and 'html'. It also contains optional fields: 'attachments', 'bcc', and 'cc', which are arrays of strings or objects. 'params' is an object that provides additional parameters, including required fields 'firstName' and 'companyName'. No additional properties are allowed at the root level.
