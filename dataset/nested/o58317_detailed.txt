The schema describes the configuration for an auto-scaling setup. It is an object type. The main properties of the schema are: 'region' is a string which is required. 'onDemandAutoScaleGroup' is a string which is required. 'spotAutoScaleGroup' is a string which is required. 'numberOfDaysBackToConsider' is an integer which is optional. 'awsAccessKeyId' is a string which is required. 'awsSecretAccessKey' is a string which is required. 'maxSpotPerHour' is an integer which is required. 'scalesOn' is an object which is required and contains two properties: 'requestCount', which is an object and is required to have a 'perServerThresholdPerMin' integer, and 'cpu', which is an object and is required to have an 'upperThresholdPercent' integer and a 'scalesWhenGreaterThanThresholdForSecs' integer. No additional properties are allowed at the root level.
