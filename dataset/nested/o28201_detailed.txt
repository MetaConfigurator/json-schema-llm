The schema describes details of a message for an Order. It is an object type. The main properties of the schema are: 'code' is a string that is optional and has a maximum length of 12. 'correctionRequired' is a boolean that is required. 'field' is a string that is optional and has a maximum length of 100. 'messageInformation' is a string that is required and has a maximum length of 150. 'objectType' is a string that is optional and has an enum value of 'orderError'. 'orderItemReference' is a string that is optional and has a maximum length of 4. 'severity' is a string that is required and has enum values of 'ERROR' and 'INFORMATION'. No additional properties are allowed at the root level.
