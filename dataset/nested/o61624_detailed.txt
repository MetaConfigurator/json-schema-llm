The schema describes a configuration profile for FileVault Recovery Key Redirection. It is an object type. The main properties of the schema are: 'EncryptCertPayloadUUID' is a string that represents the UUID of a payload containing a certificate for encrypting the recovery key, which must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadDescription' is a string that describes the payload, defaulting to 'FileVault Recovery Key Redirection Payload'. 'PayloadDisplayName' is a string that names the payload, also defaulting to 'FileVault Recovery Key Redirection Payload'. 'PayloadIdentifier' is a string that serves as a unique identifier for the payload, typically formatted as 'com.apple.security.FDERecoveryRedirect', with a default value of 'com.apple.security.FDERecoveryRedirect'. 'PayloadOrganization' is a string that describes the issuing organization of the profile. 'PayloadType' is a string indicating the type of the payload with a default value of 'com.apple.security.FDERecoveryRedirect'. 'PayloadUUID' is a string that provides a unique identifier for the payload, formatted as '01234567-89AB-CDEF-0123-456789ABCDEF', matching the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$' and defaulting to an empty string. 'PayloadVersion' is a number representing the version of the whole configuration profile, defaulting to 1. 'RedirectURL' is a string that represents the URL to which FDE recovery keys should be sent, starting with 'https://' and matching the pattern '^https://.*$'. No additional properties are allowed at the root level.
