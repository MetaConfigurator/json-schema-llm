The schema describes a configuration for device management. It is an object type. The main properties of the schema are: 'deviceIds' is an array containing strings that match the pattern '^[A-Fa-f\d]{24}$' and has a maximum of 1000 items. 'deviceTags' is an array of objects, each requiring a 'key' (a string matching the pattern '^[0-9a-zA-Z_-]{1,255}$') and a 'value' (a string with a minimum length of 1 and maximum length of 255), with additionalProperties set to false, and has a maximum of 100 items. 'description' is an optional string with a maximum length of 32767. 'filterType' is an optional string that must be one of the following values: 'all', 'whitelist', 'blacklist', or 'none'. 'pubTopics' is an optional array of strings, each with a minimum length of 1 and a maximum length of 1024. 'subTopics' is an optional array of strings, each with a minimum length of 1 and a maximum length of 1024. No additional properties are allowed at the root level.
