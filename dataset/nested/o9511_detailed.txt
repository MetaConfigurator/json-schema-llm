The schema describes retrieving details on all exports that a user has access to across all of their identities. It is an object type. The main properties of the schema are: 'exports' is an array of objects containing details of each export, with each object being required to have 'id', 'name', 'list_type', 'format', and 'mime_type'. 'href' is a string that provides a hypertext reference to this resource. 'exports' contains items which are objects with properties: 'format', which is a string representing the format of the file, with allowed values being 'Comma-Separated Values', 'Microsoft Excel', 'Fixed Width Report', 'Comma-Separated Values in Multiple Files'; 'id', which is an integer unique identifier of this export; 'list_type', which is a string or null representing the type of the list used for this export, with allowed values 'all', 'clipboard', 'search'; 'mime_type', which is a string representing the MIME type of the file, with allowed values 'text/csv;charset=iso-8859-1', 'application/vnd.ms-excel', 'text/html', 'text/plain', 'application/zip'; and 'name', which is a string representing the human-readable name for this export. No additional properties are allowed at the root level.
