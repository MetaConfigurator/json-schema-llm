The schema describes AirPrint configuration settings. It is an object type. The main properties of the schema are: 'AirPrint' is an array of objects, describing a list of printers. 'PayloadDescription' is a string describing the payload, with a default value of 'Configures AirPrint settings'. 'PayloadDisplayName' is a string for the name of the payload, with a default value of 'AirPrint'. 'PayloadIdentifier' is a string for a unique identifier for the payload, with a default value of 'com.apple.airprint'. 'PayloadOrganization' is a string that describes the issuing organization of the profile. 'PayloadType' is a string for the type of the payload, with a default value of 'com.apple.airprint'. 'PayloadUUID' is a string that serves as a unique identifier for the payload, following a specific pattern and having a default value of ''. 'PayloadVersion' is a number representing the version of the whole configuration profile, with a default value of 1. No additional properties are allowed at the root level.
