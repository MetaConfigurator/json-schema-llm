The schema describes an Active Directory certificate payload configuration. It is an object type. The main properties of the schema are: 'AllowAllAppsAccess' is a boolean that allows all apps to access the certificate in the keychain, with a default value of false. 'CertServer' is a string that specifies the fully qualified host name of the Active Directory issuing CA. 'CertTemplate' is a string that defines the template name as it appears in the Certificate Templates’ Microsoft Management Console, with a default value of 'User'. 'CertificateAcquisitionMechanism' is a string that specifies the mechanism to acquire the certificate, which can be either 'RPC' or 'HTTP'. 'CertificateAuthority' is a string that indicates the name of the CA, determined from the Common Name (CN) of the Active Directory entry. 'CertificateRenewalTimeInterval' is a number that defines the number of days before the certificate expires to start showing the expiration notification, with a default value of 14. 'Description' is a string that provides the description of the certificate request. 'KeyIsExtractable' is a boolean that allows the admin to export the private key from the keychain, with a default value of false. 'Keysize' is a number that specifies the RSA key size for the Certificate Signing Request (CSR), with a default value of 2048. 'Password' is a string that contains the password to authenticate to the certificate server. 'PayloadDescription' is a string that describes the payload, with a default value of 'Requests an Active Directory certificate'. 'PayloadDisplayName' is a string indicating the name of the payload, with a default value of 'AD Certificate'. 'PayloadIdentifier' is a string that provides a unique dot-delimited identifier for the payload, with a default value of 'com.apple.ADCertificate.managed'. 'PayloadOrganization' is a string that describes the issuing organization of the profile. 'PayloadType' is a string that defines the type of the payload, with a default value of 'com.apple.ADCertificate.managed'. 'PayloadUUID' is a string that provides a unique identifier for the payload, with a default value of '', a specific pattern for formatting, and the format specified as 01234567-89AB-CDEF-0123-456789ABCDEF. 'PayloadVersion' is a number that indicates the version of the whole configuration profile, with a default value of 1. 'PromptForCredentials' is a boolean that prompts the user for credentials, with a default value of false. 'UserName' is a string that contains the user name to authenticate to the certificate server. No additional properties are allowed at the root level.
