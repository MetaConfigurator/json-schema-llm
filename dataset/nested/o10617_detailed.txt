The schema describes a configuration file for JasmineNodeTestAdapter. It is an object type. The main properties of the schema are: '$schema' is a string formatted as a URI, optional. 'Name' is a string describing the name of the test container, optional. 'BasePath' is a string describing the base path to resolve file paths, optional, defaults to the directory containing JasmineNodeTestAdapter.json. 'Helpers' is an array of strings representing non-source, non-spec helper files, optional. 'Specs' is an array of strings containing files for Jasmine specs, optional. 'Watch' is an array of strings for additional files that trigger a test run when changed, optional. 'BatchInterval' is a number representing the delay before running tests during file watching, optional, defaults to 250. 'Traits' is an array containing either strings or objects with a required property 'Name' and an optional property 'Value', optional. 'Extensions' is a string representing the path to a node.js module for extensions, optional. 'Disabled' is a boolean indicating if the adapter should be disabled, optional. 'LogToFile' is a boolean indicating if logs should be written to a file, optional. 'LogDirectory' is a string specifying where to save the log file if 'LogToFile' is true, optional. No additional properties are allowed at the root level.
