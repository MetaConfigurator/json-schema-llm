The schema describes Angular module options. It is an object type. The main properties of the schema are: 'commonModule' is a boolean flag to control whether the CommonModule is imported, optional with a default value of true and not visible; 'flat' is a boolean flag to indicate if a dir is created, optional with a default value of false; 'module' is a string that allows specification of the declaring module, optional; 'name' is a string representing the name of the module, optional with a default sourced from argv at index 0; 'path' is a string indicating the path to create the module, optional and not visible; 'project' is a string for the name of the project, optional with a default sourced from projectName; 'routing' is a boolean that generates a routing module, optional with a default value of false; 'routingScope' is a string specifying the scope for the generated routing, optional with a default value of 'Child' and an enum of allowed values 'Child', 'Root'; 'spec' is a boolean that specifies if a spec file is generated, optional with a default value of true. No additional properties are allowed at the root level.
