The schema describes reference data for the calculation of child support. It is an object type. The main properties of the schema are: 'MARate' is an array containing objects, each representing an alimony rate based on access, accommodation, and the number of children. It is required. Each object in the array has the following properties: 'id' is an integer that serves as an internal identifier and is required. 'AARight' is an integer with a default value of 1, representing visitation and accommodation, and is required. It allows values of 1 (reduced), 2 (classic), or 3 (alternated). 'nChildren' is an integer with a default value of 1 that indicates the number of children and is required. 'rate' is a number that represents the alimony rate and is required. No additional properties are allowed at the root level.
