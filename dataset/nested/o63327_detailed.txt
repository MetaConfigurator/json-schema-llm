The schema describes NgRx Reducer options. It is an object type. The main properties of the schema are: 'api' is a boolean that specifies if api success and failure actions should be added to the reducer. It is optional and defaults to false. 'creators' is a boolean that specifies whether to use creator functions for handling actions and reducers. It is optional and defaults to true. 'feature' is a boolean flag to indicate if part of a feature schematic. It is optional and defaults to false. 'flat' is a boolean flag to indicate if a dir is created. It is optional and defaults to true. 'group' is a boolean that groups the reducer file within the 'reducers' folder. It is optional and defaults to false. 'module' is a string that specifies the declaring module. It is optional. 'name' is a string representing the name of the reducer. It is optional and has a default value sourced from argv at index 0. 'path' is a string representing the path to create the component. It is optional and has a format of 'path'. This property is not visible. 'project' is a string representing the name of the project. It is optional. 'reducers' is a string that specifies the reducers file. It is optional. 'skipTests' is a boolean that, when true, does not create test files. It is optional and defaults to false. No additional properties are allowed at the root level.
