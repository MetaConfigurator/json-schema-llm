The schema describes the structure of a widget configuration. It is an object type. The main properties of the schema are: 'content' is an object that describes the content specifics, which is required. It contains 'resource', which is a string formatted as a topic and identified as an inlet, describing the resource topic by which the HTML content is identified. 'attribute' is an optional string with a default value of an empty string, describing the attribute path within the resource that leads to the (i18n) HTML content. 'i18n' is an object, which describes which locale to use for displaying this widget. It contains 'locale', which is a string formatted as a topic and identified as an inlet, describing the topic under which to expect the locale for this widget, with a default value of 'default'. No additional properties are allowed at the root level.
