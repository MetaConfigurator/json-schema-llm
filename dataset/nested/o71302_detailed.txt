The schema describes server information. It is an object type. The main properties of the schema are: 'fabric_fq_name' is an array which contains strings, optional. 'ipmi' is an object which contains required fields 'ipmi_subnet_list' and 'ipmi_credentials'. 'ipmi_subnet_list' is an array of strings, required, where each string must match the pattern '^([0-9]{1,3}\.){3}[0-9]{1,3}(/([0-9]|[1-2][0-9]|3[0-2]))?$'. 'ipmi_credentials' is an array of strings, required. 'ipmi_port_ranges' is an array of objects, each with required fields 'port_range_start' and 'port_range_end', both of which are integers. 'ironic' is an object which contains required fields 'auth_url', 'username', and 'password'. 'auth_url' is a string, required. 'username' is a string, required. 'password' is a string, required. 'introspection_flag' is a boolean, optional, with a default value of true. 'contrail_command_host' is a string, optional. No additional properties are allowed at the root level.
