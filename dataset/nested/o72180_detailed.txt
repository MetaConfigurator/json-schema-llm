The schema describes the customization points available in an HTML body email template. It is an object type. The main properties of the schema are: 'meta' is an object representing the email metadata, required and cannot have additional properties. It contains the following properties: 'fromName' is a string with a default value of 'My Company', with a minimum length of 1, and is validated using handlebars; 'fromAddress' is a string with a default value of 'company@example.com', with a minimum length of 5, and is validated using handlebars; 'subject' is a string with a minimum length of 1 and is validated using handlebars, with a specified input height of '50px'. 'htmlContent' is a string representing the raw HTML content that can use {{mergeTags}} from handlebars, and is required. No additional properties are allowed at the root level.
