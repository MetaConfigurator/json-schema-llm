The schema describes a graph structure with nodes and edges. It is an object type. The main properties of the schema are: 'edges' is an array of objects representing connections between nodes, where each object must have a required 'id' field of type string. 'label' is a string that must have a minimum length of 3 and a maximum length of 2048. 'nodes' is an array of objects that must each have required fields: 'gridX' (which can be an integer or null), 'gridY' (which can be an integer or null), and 'id' (of type string). 'view' is an object that must contain required fields: 'gridX' (which can be an integer or null), 'gridY' (which can be an integer or null), and 'zoom' (of type number). No additional properties are allowed at the root level.
