The schema describes a location. It is an object type. The main properties of the schema are: 'id' is a string that describes the location id. 'meta' is an object that includes: 'type' which is an integer that describes the location type. 'phone' is a string that describes the location phone. 'street' is a string that describes the location street. 'boxNumber' is a string that describes the location box number. 'streetNumber' is a string that describes the location street number. 'zipCode' is a string that describes the location zip code. 'town' is a string that describes the location town. 'country' is a string that describes the location country. No additional properties are allowed at the root level.
