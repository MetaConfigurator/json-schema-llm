The schema describes an event request. It is an object type. The main properties of the schema are: 'createdAt' is an object that contains the properties 'since' and 'until', both of which are optional strings formatted as date-time. 'level' is an object with optional properties 'comparison' and 'value'; 'comparison' is a string with a default value of 'gte' and can be one of the following: 'eq', 'gt', 'lt', 'gte', 'lte'; 'value' is a string with a default value of 'info' and can be one of: 'debug', 'info', 'warn', 'error', 'critical'. 'source' is an object that requires the property 'pattern', which is a string, and has an optional property 'options', also a string. 'tags' is an object with optional properties 'match' and 'value'; 'match' is a string and can be one of: 'all', 'exact', 'any'; 'value' is an array containing strings, where no item can be 'logs'. 'time' is an object that contains the properties 'since' and 'until', both of which are optional strings formatted as date-time. No additional properties are allowed at the root level.
