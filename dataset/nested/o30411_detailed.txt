The schema describes a DNS policy. It is an object type. The main properties of the schema are: 'actionname' is a string optional; 'builtin' is an object readonly, with enum values 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'; 'cachebypass' is a string optional, with enum values 'YES', 'NO'; 'description' is a string readonly; 'drop' is a string optional, with enum values 'YES', 'NO'; 'feature' is a string readonly, with various enum values such as 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', and many others; 'hits' is an integer readonly; 'logaction' is a string optional; 'name' is a string optional; 'preferredlocation' is a string optional; 'preferredloclist' is an object optional; 'rule' is a string optional; 'undefhits' is an integer readonly; 'viewname' is a string optional. No additional properties are allowed at the root level.
