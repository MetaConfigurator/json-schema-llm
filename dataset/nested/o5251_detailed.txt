The schema describes the result of the getQuoteLevel2 procedure. It is an object type. It defines several reusable components under 'definitions': 'QuoteEntry' is an object, a quotation with the properties 'price' and 'quantity'. The 'price' property is a string. The 'quantity' property is a string. The main properties of the schema are: 'bid_count' is a string required parameter bid_count. 'offer_count' is a string required parameter offer_count. 'bids' is an array required parameter bid, where each item is a reference to 'QuoteEntry'. 'offers' is an array required parameter offer, where each item is a reference to 'QuoteEntry'. No additional properties are allowed at the root level.
