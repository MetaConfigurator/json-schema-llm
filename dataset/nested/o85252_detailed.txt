The schema describes a WoWFinder Guild Parser. It is an object type. The main properties of the schema are: 'updated' is an integer defining the last update timestamp. It is optional. 'active' is a boolean indicating if the guild is active, with a default value of false. 'ranks' is an object containing properties for rank statuses. Each rank property ('rank_0' through 'rank_9') is a boolean with a default value of false and is optional. No additional properties are allowed at the root level.
