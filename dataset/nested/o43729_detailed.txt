The schema describes a Place. It is an object type. It defines several reusable components under 'definitions': 'PostalAddress' is an object, which has the properties 'addressCountry', 'addressLocality', 'addressRegion', 'postalCode', and 'streetAddress', all of which are strings. No additional properties are allowed for 'PostalAddress'. The main properties of the schema are: 'address' is a reference to the 'PostalAddress' definition, which is optional. 'description' is a string and is optional. 'geo' is an object that can be one of two configurations: it can either contain 'elevation', 'latitude', and 'longitude', all of which are numbers; or it can consist of a 'box', 'circle', 'line', and 'polygon', all of which are strings. 'image' is a string in URI format and is optional. 'name' is a string and is required. 'url' is a string in URI format and is optional. No additional properties are allowed at the root level.
