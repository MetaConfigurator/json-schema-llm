The schema describes the data contained in a MapChart. It is an object type. The main properties of the schema are: 'datasets' is an array containing objects. Each object requires a 'values' field, which is an array of objects, each of which has required fields 'x' (a number) and 'y' (a number), and an optional field 'id' (a string). Each dataset object also has an optional 'name' (a string) and an optional 'marker' (an enum with allowed values: 'standard', 'custom', 'flag', 'plane', 'bus') and an optional 'color' (a string matching the pattern '^#[0-9a-fA-F]{6}$'). No additional properties are allowed at the root level.
