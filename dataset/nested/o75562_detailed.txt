The schema describes a pickup request for shipments. It is an object type. The main properties of the schema are: 'carrier' is a string that describes the carrier used for this pickup request. It is required. 'carrier_pickup_number' is a string that describes the carrier identifier for this pickup request. It is optional. 'id' is a string that describes the shipcloud identifier for this pickup request. It is required. 'pickup_address' is an object that describes the address where the carrier should pick up shipments. It is required and does not allow additional properties. It requires 'last_name', 'street', 'street_no', 'city', 'zip_code', and 'country'. 'pickup_time' is an object that defines the time window when the carrier should pickup shipments. It is required and does not allow additional properties. It requires 'earliest' and 'latest'. 'shipments' is an array that contains items which are objects. Each object has an 'id' property that describes the id of the shipment included in this pickup request. It is required within each object. No additional properties are allowed at the root level.
