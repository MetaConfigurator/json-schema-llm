The schema describes configuration properties for Minio on DC/OS. It is an object type. The main properties of the schema are: 'credentials' is an object that includes required properties 'access-key' (a string with a default value of 'minio') and 'secret-key' (a string with a default value of 'minio123'). 'networking' is an object that includes a required property 'port' (a number with a default value of 9000) and an optional property 'public-access' (a boolean with a default value of true). 'resource' is an object that includes required properties 'cpus' (a number with a minimum value of 0.5 and a default value of 1) and 'mem' (a number with a minimum value of 1024 and a default value of 1024). 'service' is an object that includes required properties 'name' (a string with a default value of 'minio') and 'tls' (a boolean with a default value of false). 'storage' is an object that includes a required property 'volume-size' (a number with a default value of 1024). No additional properties are allowed at the root level.
