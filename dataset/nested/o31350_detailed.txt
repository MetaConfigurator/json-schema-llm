The schema describes parameters accepted by the Rubicon adapter. It is an object type. The main properties of the schema are: 'accountId' is an integer which identifies the publisher's account. It is required and must have a minimum value of 1. 'siteId' is an integer which identifies the site selling the impression. It is required and must have a minimum value of 1. 'zoneId' is an integer which identifies the sub-section of the site where the impression is located. It is required and must have a minimum value of 1. 'inventory' is an object defining arbitrary targeting key/value pairs related to the page. It has additional properties that are strings. 'visitor' is an object defining arbitrary targeting key/value pairs related to the visitor. It has additional properties that are strings. 'video' is an object defining additional Rubicon video parameters. It contains the following properties: 'Language' is a string representing the language of the ad, 'PlayerHeight' is an integer for the height in pixels of the video player, 'PlayerWidth' is an integer for the width in pixels of the video player, 'VideoSizeID' is an integer representing Rubicon size_id used to describe type of video ad, 'Skip' is an integer indicating whether the ad can be skipped (0 for no, 1 for yes), and 'SkipDelay' is an integer for the number of seconds until the ad can be skipped. No additional properties are allowed at the root level.
