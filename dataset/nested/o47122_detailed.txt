The schema describes a release of genomic data. It is an object type. The main properties of the schema are: '_release_id' is a string which is optional. 'name' is a string which is optional. 'number' is an integer which is optional. 'date' is a string which is optional. 'project_count' is an integer which is optional. 'live_project_count' is an integer which is optional. 'primary_site_count' is an integer which is optional. 'live_primary_site_count' is an integer which is optional. 'donor_count' is an integer which is optional. 'live_donor_count' is an integer which is optional. 'specimen_count' is an integer which is optional. 'sample_count' is an integer which is optional. 'ssm_count' is an integer which is optional. 'mutated_gene_count' is an integer which is optional. No additional properties are allowed at the root level.
