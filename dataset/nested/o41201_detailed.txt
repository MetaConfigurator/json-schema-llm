The schema describes the configuration for a BIDS-APP: C-PAC (Configurable Pipeline for the Analysis of Connectomes). It is an object type. The main properties of the schema are: 'inputs' is an object that contains required fields 'functional' and 'anatomical'. 'functional' is an object containing a property 'type', which is an enum that must be 'nifti'. 'anatomical' is also an object containing a property 'type', which is an enum that must be 'nifti'. 'config' is an object that does not define any properties. No additional properties are allowed at the root level.
