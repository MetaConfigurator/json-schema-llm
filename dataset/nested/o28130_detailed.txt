The schema describes a full representation of an address, as well as geocoding reference for the property. It is an object type. The main properties of the schema are: 'buildingName' is a string that allows for buildings with well-known names to be used in an address. It is optional with a maximum length of 60. 'city' is a string that represents the city that the address is in. It is required with a maximum length of 50. 'country' is a string that denotes the country that the address is in. It is required with a maximum length of 50. 'levelNr' is a string used where a level type may be repeated (e.g., BASEMENT 1, BASEMENT 2). It is optional with a maximum length of 25. 'levelType' is a string that describes level types within a building (e.g., Floor). It is optional with a maximum length of 25. 'locality' is a string that represents a bounded locality within a city, town, or shire principally of urban character. It is optional with a maximum length of 50. 'objectType' is a string that specifies the type of the object, with an enum of allowed value 'fieldedAddress'. It is required. 'postcode' is a string that serves as a descriptor for a postal delivery area, used to speed and simplify the delivery of mail. It is required with a maximum length of 6. 'postcodeExtension' is a string for the four-digit extension on an American postal code, what comes after the hyphen when specified. It is optional with a maximum length of 4. 'privateStreetName' is a string for private streets internal to a property with an internal name not recorded by the land title office. It is optional with a maximum length of 60. 'stateOrProvince' is a string that denotes the state or province that the address is in. It is required with a maximum length of 50. 'streetName' is a string that represents the name of the street or other street type. It is required with a maximum length of 60. 'streetNr' is a string that identifies a specific property on a public street, often combined with streetNrLast for ranged addresses. It is required with a maximum length of 10. 'streetNrLast' is a string for the last number in a range of street numbers allocated to an address. It is optional with a maximum length of 10. 'streetNrSuffix' is a string for the suffix of the first street number or the suffix for the street number if there is no range. It is optional with a maximum length of 10. 'streetSuffix' is a string that serves as a modifier denoting a relative direction. It is optional with a maximum length of 2. 'streetType' is a string that specifies the type of street (e.g., Street, St., Road, Rd., Alley). It is required with a maximum length of 25. 'subUnitNr' is a string representing the discriminator used for the subunit, often a simple number but may also be a range. It is optional with a maximum length of 10. 'subUnitType' is a string that denotes the type of subunit (e.g., Flat, Suite, Tower, Unit). It is optional with a maximum length of 25. No additional properties are allowed at the root level.
