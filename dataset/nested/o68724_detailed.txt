The schema describes information about the Resource API Transports in a Puppet code environment. It is an object type. The main properties of the schema are: 'name' is a string that represents the name of the environment queried. It is an optional field. 'transports' is an array that contains a list of available transports for this environment. The items in the array are objects that are not allowed to have additional properties and contain the following fields: 'name' is a string that represents the machine readable name of the transport. It is an optional field. 'desc' is a string that provides the human readable description of the transport this schema specifies. It is an optional field. 'connection_info' is an object that allows for additional properties, which must be objects requiring 'type' and 'desc' fields. These must be strings, and there is also an optional 'sensitive' field, which is a boolean. No additional properties are allowed at the root level.
