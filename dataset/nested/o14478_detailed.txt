The schema describes an event source collection. It is an array type. The main properties of the schema are: Each item in the array is an object describing an event source. 'id' is an integer unique identifier given by the server and is required. 'generatedIdentifier' is a string unique id generated by the server and is optional. 'name' is a string that serves as a name to identify the event source and is required. 'organizationId' is an integer representing the organization and is required. 'eventSourceTemplateId' is an integer indicating the event source template instantiated and is required. 'public' is a boolean that makes the event source available to everybody and is optional. 'deletable' is a boolean defining if the event source can be deleted and is required. 'configuration' is an object that must be a valid JSON object and is required. No additional properties are allowed at the root level.
