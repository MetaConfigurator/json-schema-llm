The schema describes an invocation manifest for CNI: GE P-File Classifier. It is an object type. The main properties of the schema are: 'config' is an object that contains the properties 'timezone', a required string with a default value of 'America/Los_Angeles', and 'sync', a required boolean with a default value of true. 'inputs' is an object that contains the property 'pfile', a required object with the property 'type' which must be equal to 'pfile'. No additional properties are allowed at the root level.
