The schema describes iAgree v2 states. It is an object type. The main properties of the schema are: 'agreementId' is a string and is required. 'stateType' is a string and is required. 'id' is a string and is required. 'period' is an object and is required. It contains 'from' and 'to', both of which are strings and required. 'scope' is an object and is required. It allows properties matching the pattern '^[a-zA-Z0-9]*$'. 'window' is an object and is required. It allows properties matching the pattern '^[a-zA-Z0-9]*$'. 'records' is an array and is required. Its items are objects that must have 'value' and 'time', both of which are required. 'value' can be a string, number, or boolean as defined by oneOf. No additional properties are allowed at the root level.
