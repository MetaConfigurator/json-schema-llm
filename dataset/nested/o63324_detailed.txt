The schema describes NgRx Feature State Options. It is an object type. The main properties of the schema are: 'api' is a boolean that specifies if API success and failure actions, reducer, and effects should be generated as part of this feature. It is optional and has a default value of false. 'creators' is a boolean that specifies if the actions, reducers, and effects should be created using creator functions. It is optional and has a default value of true. 'flat' is a boolean that indicates if a directory is created. It is optional and has a default value of true. 'group' is a boolean that groups actions, reducers, and effects within relative subfolders. It is optional and has a default value of false. 'module' is a string that specifies the declaring module. It is optional. 'name' is a string representing the name of the feature. It is optional and sourced by default from the first argument in argv. 'path' is a string that specifies the path to create the feature and is not visible. It is optional. 'project' is a string that represents the name of the project. It is optional. 'reducers' is a string that specifies the reducers file. It is optional. 'skipTests' is a boolean that, when true, does not create test files. It is optional and has a default value of false. No additional properties are allowed at the root level.
