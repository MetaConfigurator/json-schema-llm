The schema describes a social record. It is an object type. The main properties of the schema are: '@context' is a string and required, with allowed value 'http://sonic-project.net/'. '@type' is a string and required, with allowed value 'socialrecord'. 'globalID' is a string and required. 'type' is a string and required, with allowed values 'platform' and 'user'. 'platformGID' is a string and required. 'displayName' is a string and required. 'profileLocation' is a string and required. 'personalPublicKey' is a string and required. 'accountPublicKey' is a string and required. 'salt' is a string and required. 'datetime' is a string and required. 'active' is an integer and required. 'keyRevocationList' is an array and required, containing objects with properties: 'revokedPublicKey' (string), 'revocationDate' (string), 'revocationReason' (integer), and 'signature' (string). No additional properties are allowed at the root level.
