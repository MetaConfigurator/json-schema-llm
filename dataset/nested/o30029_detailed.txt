The schema describes application firewall transaction records. It is an object type. The main properties of the schema are: 'appfwsessionid' is a string and is readonly. 'clientip' is a string and is readonly. 'destip' is a string and is readonly. 'endtime' is a string and is readonly. 'httptransactionid' is an integer and is readonly. 'nodeid' is an integer and is optional. 'packetengineid' is an integer and is readonly. 'profilename' is a string and is readonly. 'requestcontentlength' is an integer and is readonly. 'requestmaxprocessingtime' is an integer and is readonly. 'requestyields' is an integer and is readonly. 'responsecontentlength' is an integer and is readonly. 'responsemaxprocessingtime' is an integer and is readonly. 'responseyields' is an integer and is readonly. 'starttime' is a string and is readonly. 'url' is a string and is readonly. No additional properties are allowed at the root level.
