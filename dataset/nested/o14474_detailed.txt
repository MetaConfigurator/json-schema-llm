The schema describes a collection of event source templates. It is an array type. The main properties of the schema are: 'id' is an integer unique identifier given by the server, required; 'key' is a string unique key generated by the server to be used in events, required; 'name' is a string a name to identify the event source template, required; 'public' is a boolean make the event source template available to everybody, required; 'organizationId' is an integer the organization that offers the event source template, required; 'configuration' is an object providing a schema aims to help the WUI for configuration, required properties are 'schema' (an object) and 'url' (a string); 'configurationUi' is an object providing the configuration for the dynamic UI, containing 'schemaForm' which is an array. No additional properties are allowed at the root level.
