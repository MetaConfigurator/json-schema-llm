The schema describes Fibaro Home Center Scene JSON Schema. It is an object type. The main properties of the schema are: 'id' is an integer Identifier for the scene, optional, with a minimum value of 0. 'name' is a string Name for the scene, required, with a maximum length of 20. 'roomID' is an integer Identifier for room belongs to, optional, with a minimum value of 0. 'iconID' is an integer Icon name defined, optional, with a minimum value of 0. 'enabled' is a boolean Scene enabled flag, optional. 'autostart' is a boolean Autostart flag, optional. 'protectedByPIN' is a boolean Scene protected by PIN flag, optional. 'killable' is a boolean Is scene killable flag, optional. 'runningInstances' is an integer Number of running instances of the scene, optional, with a minimum value of 0. 'visible' is a boolean Is scene visible flag, optional. 'isLua' is a boolean Is scene written in LUA, optional. 'lua' is a string LUA code, optional. 'html' is a string Html code, optional. 'liliStartCommand' is a string Lili start command, optional. 'liliStopCommand' is a string Lili stop command, optional. 'triggers' is an object Scene triggers, optional. The 'triggers' object contains 'properties', which is an array containing objects with 'id' as an integer Identifier for the device, optional, with a minimum value of 0, and 'name' as a string Identifier for the device's property, optional; and 'globals', which is an array containing strings as Variable names, optional. 'sortOrder' is an integer Sort order when displayed in GUI, optional, with a minimum value of 0. No additional properties are allowed at the root level.
