The schema describes a post object. It is an object type. The main properties of the schema are: 'id' is a string that represents the post id, required. 'title' is a string that represents the post title, required, with a minimum length of 1. 'content' is a string that represents the post content, with a minimum length of 1. 'creationDate' is a string that represents the post creation date. 'isPublished' is a boolean that represents the post publication status. 'pinned' is a boolean that represents the post pinned status. 'modificationDate' is a string that represents the post modification date. 'publicationDate' is a string that represents the post publication date. 'viewCounter' is an integer that represents the post view number. 'tags' is a string that represents post tags, comma separated. 'author' is an object describing the author with properties: 'id' is a string for the author id, 'firstName' is a string for the author's first name, 'lastName' is a string for the author's last name, and 'email' is a string for the author's email. No additional properties are allowed at the root level.
