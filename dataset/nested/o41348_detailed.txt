The schema describes an invocation manifest for a Philips to ISMRM-RD Converter. It is an object type. The main properties of the schema are: 'inputs' is an object that is required, containing: 'raw' is an object that is optional, with a property 'type' which is an enum that allows the value 'Philips Raw file'; 'user_stylesheet' is an object that is optional, containing: 'name' which has a pattern matching '^.*.xsl$'; 'sin' is an object that is optional, with a property 'type' which is an enum that allows the value 'Philips Sin file'; 'lab' is an object that is optional, with a property 'type' which is an enum that allows the value 'Philips Lab file'. 'config' is an object that is required, containing: 'debug' which is a boolean with a default value of false. No additional properties are allowed at the root level.
