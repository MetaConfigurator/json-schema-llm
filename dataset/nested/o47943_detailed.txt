The schema describes the PAAS Platform API. It is an object type. It defines several reusable components under 'definitions': 'app', 'array_data', 'deleted', 'maintenance', 'name', 'nested_coercer_object', 'nested_no_coercer_object', 'per_page', 'threshold', and 'update_time'.

'app' is an object, an app represents the program that you would like to deploy and run on a PAAS. It has properties: 'maintenance' which is a boolean with a default value of false, 'name' which is a string matching the pattern '^[a-z][a-z0-9-]{3,30}$' and describes the unique name of the app. 'array_data' is an object containing arrays such as 'datetime_array', 'integer_array', 'nested_coercer_array', and 'nested_no_coercer_array', each defined by their respective items referencing other definitions. 'deleted' is a boolean indicating whether deleted apps should be listed. 'maintenance', is a boolean indicating the maintenance status of the app. 'name' is a string for unique app identification. 'nested_coercer_object' is an object containing 'threshold' and 'update_time' properties, both referencing their respective definitions. 'nested_no_coercer_object' is an object with properties 'per_page' and 'threshold', each referencing their respective definitions. 'per_page' is an integer describing the number of items per page. 'threshold' can be either null or a number. 'update_time' is a string formatted as date-time representing the last update time.
