The schema describes configuration properties for an InfluxDB service. It is an object type. The main properties of the schema are: 'networking' is an object that contains configuration properties for InfluxDB networking, including properties for 'external_access', 'port', and 'port_api'; 'service' is an object that contains DC/OS service configuration properties, requiring 'cpus' and 'mem'; 'storage' is an object that contains InfluxDB storage configuration properties, including properties for 'host_volume_influxdb', 'persistence', 'pre_create_database', and 'pre_create_database_name'. No additional properties are allowed at the root level.
