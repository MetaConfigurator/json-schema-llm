The schema describes the O3D3xx ROI format. It is an object type. The main properties of the schema are: 'MultiROIs' is an array that contains objects, each with 'param' (an object) and 'rois' (an array). 'param' includes properties: 'angle' (a number), 'center_x' (a number), 'center_y' (a number), 'cols' (an integer), 'gridtype' (an enum with the value 'Regular'), 'height' (a number), 'rows' (an integer), 'shape' (an enum with values 'Rect', 'Ellipse', 'Square', 'Circle', 'Hexagon'), 'spacing' (a number), and 'width' (a number). 'rois' contains integers with values between 0 and 63. 'ROIs' is an array that can contain objects that adhere to one of two structures: one structure requires 'id', 'group', 'type', 'width', 'height', 'angle', 'center_x', 'center_y' (all integers or a number), where 'type' can be 'Rect' or 'Ellipse'. The other structure requires 'id', 'group', 'type', 'points' where 'type' can be 'Poly' and 'points' is an array of objects containing properties 'x' and 'y' (both numbers). No additional properties are allowed at the root level.
