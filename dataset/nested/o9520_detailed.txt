The schema describes user identities in WebAdMIT. It is an object type. The main properties of the schema are: 'href' is a string Hypertext reference to this resource. 'user_identities' is an array of objects, each representing a user identity. Each object contains: 'association' which is a string or null, describing the association that the user identity belongs to. 'cycle' which is a string or null, that represents the academic cycle this identity can access, matching the pattern '^\d{4} - \d{4}$'. 'id' which is an integer, the unique identifier for this user identity. 'institution' which is a string or null, the human-readable name of the institution. 'organization' which is a string or null, the name of the organization this user identity belongs to. 'type' which is a string defining the type of access granted by this user identity, with allowed values: 'Admissions User', 'Association User', 'Institution User'. No additional properties are allowed at the root level.
