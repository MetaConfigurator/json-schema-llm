The schema describes groups for Gemstate. It is an array type. The main properties of the schema are: The items of the array are objects that each require the properties: 'name', 'id', 'locality', and 'addedDate'. 'name' is a string representing the full name of the group. 'id' is a string serving as a unique identifier for the group, which can be empty upon submission. 'locality' is a string indicating the incorporated city/town or meeting location of the group, or could be 'Online' or 'Idaho'. 'addedDate' is a string formatted as a date, indicating when the group was added to the list. Each group object may optionally include 'description', 'tla', 'url', 'meetupComGroupName', 'eventbriteOrganizerId', 'eventbriteKeywords', 'iCalendarUrl', 'abandonedDate', and 'schedCom'. 'eventbriteKeywords' is an array of strings, allowing multiple keywords. 'schedCom' is an array of objects that require the 'name' property, which is a string. No additional properties are allowed at the root level.
