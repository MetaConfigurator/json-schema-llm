The schema describes configuration settings for a Minecraft launcher. It is an object type. The main properties of the schema are: 'roots' is an array containing the root paths of all the Minecraft installations. It is required and defaults to an empty array. The items in this array are strings. 'primaryRoot' is a string representing the primary installation root. It is required and defaults to an empty string. 'locale' is a string indicating the display language of the launcher. It is required and defaults to 'en'. 'autoDownload' is a boolean indicating whether the launcher should automatically download new updates. It is required and defaults to false. 'autoInstallOnAppQuit' is a boolean indicating whether the launcher should automatically install new updates after the application quits. It is required and defaults to false. 'allowPrerelease' is a boolean indicating whether the launcher should show pre-releases. It is required and defaults to false. 'useBmclAPI' is a boolean indicating whether to use the bmcl API in China Mainland. It is required and defaults to true. No additional properties are allowed at the root level.
