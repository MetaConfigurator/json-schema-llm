The schema describes Microsoft.ServerManagement Resource Types. It is an object type. It defines several reusable components under 'definitions': 'gateways' is an object, a Server management tools gateway. It has required fields: 'location', 'name', and 'properties'. The 'apiVersion' property must be one of the enum values: '2015-07-01-preview'. The 'type' property must be one of the enum values: 'Microsoft.ServerManagement/gateways'. The 'properties' property is an object with additionalProperties set to false. 'nodes' is an object, a Server management tools node. It has required fields: 'name', 'location', and 'properties'. The 'apiVersion' property must be one of the enum values: '2015-07-01-preview'. The 'type' property must be one of the enum values: 'Microsoft.ServerManagement/nodes'. The 'properties' property is an object with required fields: 'gatewayId' and 'connectionName'. The 'gatewayId' property is a string that describes the gateway to which this node belongs. The 'connectionName' property is a string that can be a NetBios name, IP (v4/v6), or FQDN. The 'properties' in 'nodes' has additionalProperties set to false. 'sessions' is an object, a server management session. It has required fields: 'location', 'name', and 'properties'. The 'apiVersion' property must be one of the enum values: '2015-07-01-preview'. The 'type' property must be one of the enum values: 'Microsoft.ServerManagement/gateways/sessions' or 'Microsoft.ServerManagement/nodes/sessions'. The 'properties' property is an object with required fields: 'username' and 'password'. Both are strings.
