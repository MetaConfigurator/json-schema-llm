The schema describes links this user can share to make referrals. It is an object type. It defines several reusable components under 'definitions': 'subShareLinks' is an object, containing a required field 'DIRECT' which is a string describing the share link associated with the DIRECT share medium. The main properties of the schema are: Each property defined by the pattern '^[a-zA-Z0-9-]{1,64}$' is an object describing the share links available for this program. The properties 'MOBILE', 'EMAIL', and 'UNKNOWN' are all required and each contains a reference to 'subShareLinks'. No additional properties are allowed at the root level.
