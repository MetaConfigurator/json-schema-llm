The schema describes a job listing in the High Energy Physics (HEP) field. It is an object type. The main properties of the schema are: 'closed_date' is a string representing the date when the job was closed, optional. 'contact_email' is an array of strings formatted as email addresses, required, and must have unique items. 'contact_person' is an array of strings denoting the names of contact persons, required, and must have unique items. 'continent' is an array of strings, required, that must contain unique items and each item must be one of the following values: 'North America', 'Europe', 'Asia', 'South America', 'Africa', 'Oceania'. 'deadline_date' is a string representing the application deadline date, optional. 'description' is a string providing the job description, required. 'experiments' is an array of strings, optional, where each string can store the ID of the associated experiment and must have unique items. 'institution' is an array of objects, required, where each object contains a boolean 'curated_relation', a string 'name' for the institution name, and an integer 'recid' for the institution Record ID; must have unique items. 'position' is a string representing the job position, required. 'rank' is an array of strings, required, where each item must be one of the following values: 'Postdc', 'Junior', 'Student', 'Senior', 'Staff', 'Visitor', 'Visiting Scientist' and must have unique items. 'reference_email' is an array of strings formatted as email addresses, optional, and must have unique items. 'research_area' is an array of strings, required, where each item must be one of the following values: 'usual', 'arXiv', 'list' and must have unique items. 'urls' is an array of strings formatted as URLs, optional, and must have unique items. No additional properties are allowed at the root level.
