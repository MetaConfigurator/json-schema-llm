The schema describes a configuration profile for system policy control. It is an object type. The main properties of the schema are: 'AllowIdentifiedDevelopers' is a boolean that determines if Gatekeeper’s 'Mac App Store and identified developers' option is chosen if the value is true. 'EnableAssessment' is a boolean that determines if Gatekeeper is enabled if the value is true. 'PayloadDescription' is a string that describes the payload, with a default value of 'System Policy Control Payload'. 'PayloadDisplayName' is a string that names the payload, with a default value of 'System Policy Control Payload'. 'PayloadIdentifier' is a string that provides a unique identifier for the payload, dot-delimited, with a default value of 'com.apple.systempolicy.control'. 'PayloadOrganization' is a string describing the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string that defines the type of the payload, with a default value of 'com.apple.systempolicy.control'. 'PayloadUUID' is a string that requires a specific pattern format (UUID) with a default value of an empty string. 'PayloadVersion' is a number representing the version of the configuration profile, with a default value of 1. No additional properties are allowed at the root level.
