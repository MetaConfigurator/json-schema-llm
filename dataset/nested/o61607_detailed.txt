The schema describes a configuration profile for domain management. It is an object type. The main properties of the schema are: 'EmailDomains' is an array containing strings, describing any email addresses that do not have a suffix matching the unmarked email domains specified, which will be considered out-of-domain. 'PayloadDescription' is a string providing a description of the payload, with a default value of 'Domains Payload'. 'PayloadDisplayName' is a string indicating the name of the payload, with a default value of 'Domains Payload'. 'PayloadIdentifier' is a string serving as a unique identifier for the payload, with a default value of 'com.apple.domains'. 'PayloadOrganization' is a string that describes the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string representing the type of the payload, with a default value of 'com.apple.domains'. 'PayloadUUID' is a string used for the unique identifier of the payload, adhering to a specific pattern, with a default value of an empty string. 'PayloadVersion' is a number indicating the version of the configuration profile, with a default value of 1. 'SafariPasswordAutoFillDomains' is an array containing strings, where users can save passwords in Safari from URLs matching the listed patterns. 'WebDomains' is an array containing strings, indicating URLs matching the listed patterns that will be considered managed, not supported in macOS. No additional properties are allowed at the root level.
