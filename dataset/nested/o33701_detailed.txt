The schema describes a named entity. It is an object type. The main properties of the schema are: 'schemaType' is a string that represents the schema type, required; '_documentID' is a string that serves as the internal document GUID, required; '_corpusID' is a string that acts as the internal Corpus GUID, required; 'offsets' is an array that contains at least one item, where each item is an object with 'begin' as an integer (minimum 0) and 'end' as an integer (minimum 0); 'TypeNE' is a string that specifies the named entity's type with allowed values 'personne', 'lieu', 'organisation', 'autre', searchable and required; 'TypeEn_Wikipedia' is an array of strings for the type of entity on Wikipedia, with unique items, optional, and allows a minimum of 0 items; 'description' is a string for the named entity's description, optional; 'wikipedia' is a string that serves as a reference URL for the entity on Wikipedia, optional. No additional properties are allowed at the root level.
