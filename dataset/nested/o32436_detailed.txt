The schema describes a single Ranked Choice Voting (RCV) race. It is an object type. The main properties of the schema are: 'isTest' is a boolean that specifies a value that is always true if it exists, with the allowed value being true. 'counts' is an array that specifies how many of each of the ballots in votes will appear in a test, containing items of type integer. 'votes' is an array that contains all of the votes in the election; each item is an object with the properties: 'id', which is an integer and is required, and 'ranks', which is an array of rankings on this ballot, where each ranking is an array of integers and is also required. 'tiebreak' is an array that specifies the candidate chosen in each successive tiebreak, containing items of type integer. 'results' is an object that records what happened as the election was tabulated, with a required property 'winner' which is an integer indicating which candidate won the election, and a property 'record', which is an array of objects that summarizes the results, with each object containing a required property 'tally', which is an array of objects with required properties 'candidate' and 'firstChoices' both of type integer, and an optional property 'eliminated', which is an array of integers. The schema has a dependency where if 'isTest' is present, 'results', 'tiebreak', and 'counts' must also be present. No additional properties are allowed at the root level.
