The schema describes the structure for a Generation object. It is an object type. It defines several reusable components under 'definitions': 'translations' is an object, representing a collection of translations for various languages. It requires the fields 'en' and 'de' to be present, and does not allow additional properties. The main properties of the schema are: 'number' is an integer that must be between 1 and 7 and is required; 'names' is a translations object that is required; 'dex_name' is a string that is required; 'games' is an array of translations objects that must have between 1 and 7 items and is required. No additional properties are allowed at the root level.
