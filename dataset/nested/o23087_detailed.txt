The schema describes a person. It is an object type. The main properties of the schema are: 'type' is a string that must be equal to 'person'. 'firstName' is a string that represents the person's first name. It is required. 'lastName' is a string that represents the person's last name. It is required. 'age' is an integer that represents the age in years, which must be equal to or greater than zero. 'citizen' is a boolean that indicates if this person is a citizen of this country. 'description' is a string for a longer text description with a maximum length of 2000 characters. 'homepage' is a string with a maximum length of 2000 characters, displayed as a single line. No additional properties are allowed at the root level.
