The schema describes the structure for an object containing lists of return data and messages. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'listaRetorno' is an array containing objects, each with optional properties 'numeroMRUC' (a string), 'documentosCarga' (an array of objects with optional properties 'nivel' (a string) and 'documentos' (an array of objects containing optional properties 'numeroDUE' (a string) and 'numeroRUC' (a string))) and 'permiteMovimentacao' (a boolean). 'listaMensagem' is an array containing objects, each with optional properties 'mensagem' (a string) and 'codigoErro' (a string). No additional properties are allowed at the root level.
