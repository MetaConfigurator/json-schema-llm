The schema describes a configuration file for PR Preview. It is an object type. The main properties of the schema are: 'type' is a string that specifies the spec processor used to render this spec. It is required and can take one of the following values: 'bikeshed', 'Bikeshed', 'BikeShed', 'respec', 'Respec', 'ReSpec', 'wattsi', 'Wattsi', 'html', 'HTML'. 'src_file' is a string that specifies the relative path to the source file from the root of the repository. It is required and must have a minimum length of 1. 'params' is an object that contains parameters for the spec processor. It allows any additional properties to be of type string, number, boolean, or null. 'post_processing' is an object that sets a post processor. It is optional and requires the property 'name', which is a string that can take one of the following values: 'emu-algify', 'webidl-grammar'. The 'options' property of 'post_processing' is an object that allows additional properties with no restrictions. 'multipage' is a boolean that indicates whether the rendered spec is multipage. No additional properties are allowed at the root level.
