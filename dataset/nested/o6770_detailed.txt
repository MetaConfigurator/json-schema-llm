The schema describes a data submission structure. It is an object type. The main properties of the schema are: 'type' is a string optional. 'some_variables' is an array required, containing objects each with a 'header' and 'values'. The 'header' is an object required, with a 'name' field required (string) and an optional 'units' field (string), disallowing additional properties. The 'values' is an array required, containing objects each with a 'value' field (string or number), an optional 'low' field (number), and an optional 'high' field (number), disallowing additional properties. 'name' is a string optional, used to reference a data record when everything is in the same file. No additional properties are allowed at the root level.
