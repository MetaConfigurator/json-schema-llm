The schema describes a beverage configuration. It is an object type. The main properties of the schema are: 'info' is a string that has a default value of an empty string. 'tag' is a string that has a default value of an empty string and an enum with allowed value 'core'. 'style' is a string that has a default value of an empty string and an enum with allowed values 'All Day Cocktail', 'Before Dinner Cocktail', 'Longdrink', 'After Dinner Cocktail', 'Sparkling Cocktail', 'Hot Drink'. 'ingredients' is an object. 'optional' is an object. 'variants' is an array. 'unit' is a string that has a default value of 'oz' and an enum with allowed values 'oz', 'mL', 'cL'. 'prep' is a string that has a default value of 'shake' and an enum with allowed values 'shake', 'stir', 'build', 'throw'. 'ice' is a string that has a default value of 'cubed' and an enum with allowed values 'cubed', 'crushed', 'neat'. 'profile' is an array of strings with items that have an enum with allowed values 'strong', 'tangy', 'citrusy', 'sweet', 'fruity', 'herbal'. 'glass' is a string that has a default value of 'cocktail' and an enum with allowed values 'martini', 'cocktail', 'rocks', 'collins', 'highball', 'flute', 'copper mug', 'pint', 'coupe'. 'origin' is a string with a description of 'Origin of the recipe'. 'garnish' is a string that has a default value of an empty string. No additional properties are allowed at the root level.
