The schema describes the configuration for an invocation manifest for SciTran: DICOM MR Classifier. It is an object type. The main properties of the schema are: 'config' is an object that contains the configuration settings. It is required and has two required properties: 'timezone', which is a string with a default value of 'UTC', and 'force', which is a boolean with a default value of false. 'inputs' is an object that contains input specifications. It is required and has two required properties: 'dicom', which is an object containing a 'type' property with an enum value of 'dicom', and 'classifications', which is an object that requires 'base' (a string), 'found' (a boolean), and 'value' (with no defined type). No additional properties are allowed at the root level.
