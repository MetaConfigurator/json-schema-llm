The schema describes the invocation manifest for SciTran: DICOM MR Classifier. It is an object type. The main properties of the schema are: 'config' is an object that contains required field 'timezone', which is a string with a default value of 'UTC'. 'inputs' is an object that contains required fields 'dicom' and 'classifications'. 'dicom' is an object containing a property 'type' which is a string constrained to the enum value 'dicom'. 'classifications' is an object with required fields 'base' (a string), 'found' (a boolean), and 'value' (with no specified type). No additional properties are allowed at the root level.
