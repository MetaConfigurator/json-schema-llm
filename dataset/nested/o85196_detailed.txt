The schema describes a MWF link navigation component's schema definition. It is an object type. The main properties of the schema are: 'config' is an object that contains the configuration settings. It is required and has a required field 'context', which is a string that can be one of the following values: 'links-with-title-image', 'links-with-image', 'links-with-title', 'links-only'. It also has properties 'header-tag', which is a string that can be one of the following values: 'h1', 'h2', 'h3', 'h4', 'h5', 'h6', and 'header-title', which is a string. The 'links' property is an array of objects, each of which requires the fields 'href', a string, and 'text', a string. Each link object may also have optional fields 'alternative-text', a string, and 'image', a string. This property includes dependencies where 'alternative-text' requires 'image' to also be present and vice versa. No additional properties are allowed at the root level.
