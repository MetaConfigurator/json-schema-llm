The schema describes an invocation manifest for the SciTran PAR/REC MR Classifier. It is an object type. The main properties of the schema are: 'config' is an object containing the configuration settings. It is required and has a required property 'timezone', which is a string with a default value of 'UTC'. 'inputs' is an object specifying the inputs for the classifier. It is required and has a required property 'parrec', which is an object containing a property 'type' that must be the enum value 'parrec'. The 'rec' property is an optional object that also contains a property 'type' requiring the enum value 'parrec'. No additional properties are allowed at the root level.
