The schema describes a race environment map file. It is an object type. It defines several reusable components under 'definitions': 'point' is an array, representing a coordinate with exactly two numbers. The main properties of the schema are: 'size' is an array of integers, required, containing exactly two items. 'max_time' is a number, required. 'cars' is an array, required, where each item is an array of exactly three numbers. 'walls' is an array, required, where each item is an array of 'point' definitions. 'headline' is an array, required, where each item is a 'point' definition. 'finish' is an array, required, containing exactly two 'point' definitions. No additional properties are allowed at the root level.
