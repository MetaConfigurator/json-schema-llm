The schema describes a service pricing structure. It is an object type. The main properties of the schema are: 'educationPricing' is a boolean indicating whether education pricing is applicable, required. 'incidentEscalation' is a boolean indicating whether incident escalation is in place, required. 'minimumContractPeriod' is an enum with allowed values 'Hour', 'Day', 'Month', 'Year', 'Other', required. 'priceInterval' is an enum with allowed values '', 'Second', 'Minute', 'Hour', 'Day', 'Week', 'Month', 'Quarter', '6 months', 'Year', optional. 'priceMax' is a string that may be an empty string or must match the pattern '^$|^\d{1,15}(?:\.\d{1,5})?$', optional. 'priceMin' is a string that must match the pattern '^\d{1,15}(?:\.\d{1,5})?$', required. 'priceUnit' is an enum with allowed values 'Unit', 'Person', 'Licence', 'User', 'Device', 'Instance', 'Server', 'Virtual machine', 'Transaction', 'Megabyte', 'Gigabyte', 'Terabyte', required. 'pricingDocumentURL' is a string formatted as a URI, required. 'serviceBenefits' is an array of strings where each item has a max length of 100, must match the pattern '^(?:\S+\s+){0,9}\S+$', with a minimum of 1 and a maximum of 10 items, required. 'serviceDefinitionDocumentURL' is a string formatted as a URI, required. 'serviceFeatures' is an array of strings with similar constraints to 'serviceBenefits', required. 'serviceName' is a string with a min length of 1 and a max length of 100, required. 'serviceSummary' is a string with a min length of 1, max length of 500, and must match the pattern '^(?:\S+\s+){0,49}\S+$', required. 'serviceTypes' is an array of enums with allowed values 'Implementation', 'Ongoing support', 'Planning', 'Testing', 'Training', with a minimum of 1 and a maximum of 5 unique items, required. 'sfiaRateDocumentURL' is a string formatted as a URI, optional. 'supportAvailability' is a string with a min length of 1 and max length of 200, must match the pattern '^(?:\S+\s+){0,19}\S+$', required. 'supportForThirdParties' is a boolean indicating if support for third parties is available, required. 'supportResponseTime' is a string with a min length of 1 and max length of 200, must match the pattern '^(?:\S+\s+){0,19}\S+$', required. 'supportTypes' is an array of enums with allowed values 'Service desk', 'Email', 'Phone', 'Live chat', 'Onsite', with a minimum of 1 and a maximum of 5 unique items, required. 'terminationCost' is a boolean indicating if a termination cost applies, required. 'termsAndConditionsDocumentURL' is a string formatted as a URI, required. 'vatIncluded' is a boolean indicating if VAT is included, required. 'vendorCertifications' is an array of strings with a max length of 100, matching the pattern '^(?:\S+\s+){0,9}\S+$', with a minimum of 0 and a maximum of 10 items, optional. No additional properties are allowed at the root level.
