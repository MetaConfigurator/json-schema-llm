The schema describes the metadata for a BlackBox 3.0 task. It is an object type. The main properties of the schema are: 'author' is a string optional, 'desc_en' is a string optional, 'desc_ru' is a string optional, 'flag' is a string required, 'flag_comp' is a string required, 'images' is an array of strings with a minimum of 1 item, and unique items, 'schema_version' is a string required with a pattern '^0\.1$', 'title_en' is a string required, 'title_ru' is a string required, and 'upload_timestamp' is an integer optional. No additional properties are allowed at the root level.
