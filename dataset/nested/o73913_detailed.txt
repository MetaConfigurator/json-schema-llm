The schema describes configuration files for Bower. It is an object type. The main properties of the schema are: 'name' is a string which is required and has a maximum length of 50 characters. 'description' is a string which is optional. 'version' is a string which is optional. 'main' is an array or a string which is optional. 'license' is an array or a string which is optional and has a maximum length of 140 characters. 'ignore' is an array or a string which is optional. 'keywords' is an array of strings which is optional and each item has a maximum length of 50 characters. 'authors' is an array which is optional and items can be either strings or objects. 'homepage' is a string which is optional. 'repository' is an object which is optional and contains properties: 'type' which is a string and must be 'git', and 'url' which is a string. 'dependencies' is an object which is optional with additional properties as strings. 'devDependencies' is an object which is optional with additional properties as strings. 'resolutions' is an object which is optional. 'private' is a boolean which is optional. 'exportsOverride' is an object which is optional and can have additional properties as objects containing additional properties as strings. 'moduleType' is an array which is optional and items must be one of 'amd', 'es6', 'globals', 'node', or 'yui'. No additional properties are allowed at the root level.
