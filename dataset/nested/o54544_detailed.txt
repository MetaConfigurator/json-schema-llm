The schema describes the data contained in a DonutChart settings object. It is an object type. The main properties of the schema are: 'title' is a string optional status, 'description' is a string optional status, 'innerRadius' is a string with a pattern that must match '^[0-9]{1,3}%$' optional status, 'labels' is an object optional status with properties: 'labelsEnabled' is a boolean optional status, 'labelRadius' is a number with a minimum of 0 optional status, 'color' is a string with a pattern that must match '^#[0-9a-fA-F]{6}$' optional status; 'legend' is an object optional status with properties: 'enabled' is a boolean optional status, 'position' is a string with an enum of allowed values: 'top', 'right', 'bottom', 'left' optional status, 'maxColumns' is an integer with a minimum of 0 optional status, 'markerType' is a string with an enum of allowed values: 'circle', 'square', 'line', 'triangleUp', 'triangleDown', 'triangleLeft', 'triangleRight', 'diamond', 'bubble' optional status, 'switchable' is a boolean optional status, 'color' is a string with a pattern that must match '^#[0-9a-fA-F]{6}$' optional status; 'style' is an object optional status with properties: 'backgroundColor' is a string with a pattern that must match '^#[0-9a-fA-F]{6}$' optional status, 'borderColor' is a string with a pattern that must match '^#[0-9a-fA-F]{6}$' optional status, 'borderAlpha' is a number with a minimum of 0 optional status, 'animationDuration' is an integer with a minimum of 0 optional status.
