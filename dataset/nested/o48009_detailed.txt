The schema describes a data structure for project information. It is an object type. The main properties of the schema are: 'acronym' is a string optional, 'code' is a string optional, 'enddate' is a string optional, 'funder' is an object optional, 'identifiers' is an object optional, 'internal_id' is a string optional, 'program' is a string optional, 'remote_modified' is a string or null optional, 'startdate' is a string optional, 'title' is a string optional, and 'url' is a string optional. No additional properties are allowed at the root level.
