The schema describes Sane Scanner options. It is an object type. The main properties of the schema are: 'host' is a string The host of the scanner, required. 'model' is a string The model of the scanner, required. 'name' is a string The name of the scanner, required. 'optionGroups' is an array The unique identifier for a product, required. Each item in the array is an object with properties: 'options' is an array containing objects with properties: 'name' is a string, 'range' is a string, 'title' is a string, 'type' is a string, 'units' is a string, 'value' is a number. 'title' is a string. 'type' is a string The type of the device, SANE can connect with some otrer peripherals besides scanners, optional. 'vendor' is a string The vendor of the scanner, required. No additional properties are allowed at the root level.
