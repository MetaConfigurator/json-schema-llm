The schema describes a platform configuration. It is an object type. The main properties of the schema are: 'BiosFiles' is an object containing pattern properties for file names mapped to arrays of strings, where each string must be a 32-character hexadecimal string. 'FileTypes' is an object that must have at least one item, requires the 'RSRC' key, and contains pattern properties for specific content types, including those matching 'BIOS' and a general matcher for any file type. 'FriendlyName' is a string representing a user-friendly name for the platform. 'MaximumInputs' is an integer defining the maximum number of inputs allowed. 'Metadata' is an object with properties starting with 'platform_', without allowing additional properties. 'PlatformID' is a string that uniquely identifies the platform. All defined properties except 'FriendlyName' and 'MaximumInputs' are required. No additional properties are allowed at the root level.
