The schema describes a diagram representation. It is an object type. The main properties of the schema are: 'diagram_id' is a string which is required. 'highlights' is an array of strings which is optional and does not allow additional items. 'protein_map' is an array of objects which is optional and does not allow additional items. Each object in 'protein_map' contains 'db_id', a string which is optional, and 'uniprot_ids', an array of strings which is optional and does not allow additional items. 'xml' is a string which is optional. No additional properties are allowed at the root level.
