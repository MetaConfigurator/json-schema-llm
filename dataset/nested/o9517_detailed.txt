The schema describes a PDF Manager batch representing the asynchronous execution and eventual population of a PDF Manager template, resulting in PDF Manager zip files. It is an object type. The main properties of the schema are: 'pdf_manager_batch' is an object containing details about the batch. It includes the required fields 'href', 'id', 'updated_at', 'state', 'download_hrefs', and 'pdf_manager_template'. 'download_hrefs' is an array that contains links for downloading the generated PDF files, applicable when the state is either 'available' or 'success_with_errors'. 'href' is a string that represents a hypertext reference to this resource. 'id' is an integer serving as a unique identifier of this batch. 'pdf_manager_template' is an object that contains required properties: 'href', 'id', and 'name', where 'href' is a string reference to the template, 'id' is an integer identifying the template, and 'name' is a string representing the human-readable name of the PDF Manager template. 'state' is a string that can have the following values: 'initializing', 'queued', 'in_progress', 'available', 'success_with_errors', 'empty_list', 'failed'. 'updated_at' is a string formatted as a date-time indicating when the batch was last updated. No additional properties are allowed at the root level.
