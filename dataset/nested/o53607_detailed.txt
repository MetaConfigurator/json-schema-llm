The schema describes a household member. It is an object type. The main properties of the schema are: 'memberId' is a string Member ID of the Household Member with a minimum length of 1 and is required. 'age' is a number Age of the Household Member with a minimum value of 1 and is required. 'gender' is a string Gender of the Household Member with allowed values of 'Male', 'Female', 'Other' and is required. 'race' is a string Race of the Household Member with allowed values of 'American Indian or Alaska Native', 'Asian', 'Black or African American', 'Native Hawaiian or Other Pacific Islander', 'White' and is required. 'isDisabled' is a boolean Flag to indicate if a household member is Disabled and is required. 'isHispanic' is a boolean Flag to indicate if a household member is Hispanic and is required. 'isSpecialNeeds' is a boolean Flag to indicate if a household member is special needs and is required. No additional properties are allowed at the root level.
