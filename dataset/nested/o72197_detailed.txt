The schema describes customization points available in a Simple Email Template. It is an object type. The main properties of the schema are: 'meta' is an object that describes the metadata of the email, which is required. It contains the required properties 'fromName', 'fromAddress', and 'subject'. 'fromName' is a string with a default value of 'My Company', a minimum length of 1, and is validated using handlebars. 'fromAddress' is a string with a default value of 'company@example.com', a minimum length of 5, and is also validated using handlebars. 'subject' is a string with a minimum length of 1 and is validated using handlebars. 'content' is an object that describes the email content, which is required. It contains the required property 'bodyContent'. 'bodyContent' is a string that supports markdown and handlebars expressions, and is validated using handlebars. 'headerImage' is an optional string for the header image. 'headerContent' is an optional string validated using handlebars. 'cta' is an optional object that can contain 'enabled', a boolean with a default value of false, 'text', a string, and 'link', a string. 'footerContent' is an optional string validated using handlebars. No additional properties are allowed at the root level.
