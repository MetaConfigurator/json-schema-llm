The schema describes a Web Map Tile Service Info. It is an object type. The main properties of the schema are: 'customLayerParameters' is an object that specifies a sequence of parameters used to append different custom parameters to a WMTS tile request, applied to 'GetTile', and takes precedence over 'customParameters'. It allows pattern properties with string values. 'customParameters' is an object that defines a sequence of parameters to append custom parameters to all WMTS requests, applied to 'GetCapabilities' and 'GetTile', and does not take precedence over 'customLayerParameters'. It also allows pattern properties with string values. 'layerIdentifier' is a string that serves as an identifier for the specific layer used in the WMTS service. It is required input by the user. 'tileMatrixSet' is a string that describes the tiling schema, set by the WMTS service. 'url' is a string that provides the URL to the WMTS web service and is required input by the user, adhering to the URI format. No additional properties are allowed at the root level.
