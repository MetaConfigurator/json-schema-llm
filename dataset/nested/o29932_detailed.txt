The schema describes LDAP configuration parameters. It is an object type. The main properties of the schema are: 'authtimeout' is an integer representing the authentication timeout; 'builtin' is an object and is readonly, representing built-in parameter properties; 'defaultauthenticationgroup' is a string that specifies the default authentication group; 'feature' is a string that is readonly, listing allowed values: 'WL', 'WebLogging', 'SP', 'SurgeProtection', 'LB', 'LoadBalancing', 'CS', 'ContentSwitching', 'CR', 'CacheRedirection', 'SC', 'SureConnect', 'CMP', 'CMPcntl', 'CompressionControl', 'PQ', 'PriorityQueuing', 'HDOSP', 'HttpDoSProtection', 'SSLVPN', 'AAA', 'GSLB', 'GlobalServerLoadBalancing', 'SSL', 'SSLOffload', 'SSLOffloading', 'CF', 'ContentFiltering', 'IC', 'IntegratedCaching', 'OSPF', 'OSPFRouting', 'RIP', 'RIPRouting', 'BGP', 'BGPRouting', 'REWRITE', 'IPv6PT', 'IPv6protocoltranslation', 'AppFw', 'ApplicationFirewall', 'RESPONDER', 'HTMLInjection', 'push', 'NSPush', 'NetScalerPush', 'AppFlow', 'CloudBridge', 'ISIS', 'ISISRouting', 'CH', 'CallHome', 'AppQoE', 'ContentAccelerator', 'SYSTEM', 'RISE', 'FEO', 'LSN', 'LargeScaleNAT', 'RDPProxy', 'Rep', 'Reputation', 'URLFiltering', 'VideoOptimization', 'ForwardProxy', 'SSLInterception', 'AdaptiveTCP', 'CQA', 'CI', 'ContentInspection', 'Bot'; 'groupattrname' is a string for group attribute name; 'groupauthname' is a string that is readonly for group authentication name; 'groupnameidentifier' is a string for group name identifier; 'groupsearchattribute' is a string for group search attribute; 'groupsearchfilter' is a string for group search filter; 'groupsearchsubattribute' is a string for group search subattribute; 'ldapbase' is a string for LDAP base; 'ldapbinddn' is a string for LDAP bind DN; 'ldapbinddnpassword' is a string for LDAP bind DN password; 'ldaploginname' is a string for LDAP login name; 'maxnestinglevel' is an integer for the maximum nesting level; 'nestedgroupextraction' is a string listing allowed values: 'ON', 'OFF'; 'passwdchange' is a string listing allowed values: 'ENABLED', 'DISABLED'; 'searchfilter' is a string for the search filter; 'sectype' is a string listing allowed values: 'PLAINTEXT', 'TLS', 'SSL'; 'serverip' is a string for server IP; 'serverport' is an integer for server port; 'ssonameattribute' is a string for SSO name attribute; 'subattributename' is a string for sub attribute name; 'svrtype' is a string listing allowed values: 'AD', 'NDS'. No additional properties are allowed at the root level.
