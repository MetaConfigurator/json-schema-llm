The schema describes Pssimistical config. It is an object type. The main properties of the schema are: 'inputs' is an array of objects that must each contain required fields 'path', 'table', and 'reader'. 'query' is an object that must contain required fields 'sql' and 'outputFormat'. 'readers' is an object where keys are strings matching the pattern '\w+', and each value must be an object with required fields 'type' and 'columns'. 'tables' is an array of objects that must each contain required fields 'name' and 'columns'. Each object in 'tables' must have 'columns' field as an array of objects, each with required fields 'name' and 'type', where 'type' must be one of the allowed values: 'number', 'text', 'date'. No additional properties are allowed at the root level.
