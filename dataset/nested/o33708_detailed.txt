The schema describes a document surface. It is an object type. The main properties of the schema are: 'schemaType' is a string that describes the schema type, required and locked; '_documentID' is a string that represents the internal document GUID, required and locked; '_corpusID' is a string that represents the internal Corpus GUID, required and locked; 'offsets' is an array of objects that must contain at least one item, each item is an object with properties 'begin' and 'end', both are integers with a minimum value of 0. No additional properties are allowed at the root level.
