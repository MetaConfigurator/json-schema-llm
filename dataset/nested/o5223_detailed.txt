The schema describes the result object of the getFuturesHolding procedure. It is an object type. The main properties of the schema are: 'futures_holding' is an object which represents the result. It is required and defines the following required fields: 'firmid', 'trdaccid', 'sec_code', 'type', 'openbuys', 'opensells', and 'session_status'. 'firmid' is a string describing the firm identifier. 'trdaccid' is a string describing the trade account identifier. 'sec_code' is a string describing the security code. 'type' is an integer describing the type. 'openbuys' is a number describing the total open buys. 'opensells' is a number describing the total open sells. 'session_status' is a number describing the session status. No additional properties are allowed at the root level.
