The schema describes a structure for managing authentication and user data within a GitHub API context. It is an object type. The main properties of the schema are: 'auth' is an object with a description of 'Authorization with the GitHub API'. It is required and must contain 'resource'. It can optionally include 'action', 'flag', and 'i18nHtmlLabel', which defaults to { 'en-US': 'Sign in' }. 'user' is an object with a description of 'User information'. It is required and must contain 'resource'. It can optionally include 'url', which defaults to 'https://api.github.com/user', and 'i18nHtmlLabel', which defaults to { 'en-US': 'Signed in as' }. 'menu' is an array with a description of 'Menu entries'. Each item in the array is an object and requires 'i18nHtmlLabel'. It can optionally include 'action' and 'place'. No additional properties are allowed at the root level.
