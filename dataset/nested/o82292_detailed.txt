The schema describes a configuration for a tracking system. It is an object type. The main properties of the schema are: '_schema' is a string that must match the pattern '^pbj:([a-z0-9-]+):([a-z0-9\.-]+):([a-z0-9-]+)?:([a-z0-9-]+):([0-9]+-[0-9]+-[0-9]+)$' and has a default value of 'pbj:gdbots:analytics:tracker:keen:1-0-0', and it is required. 'is_enabled' is a boolean that specifies whether the tracking is enabled. 'project_id' is a string that must match the pattern '^\w+$' and is optional. 'read_key' is a string that must match the pattern '^\w+$' and is optional. 'write_key' is a string that must match the pattern '^\w+$' and is optional. No additional properties are allowed at the root level.
