The schema describes a structure for managing first-timer contributions in a project. It is an object type. The main properties of the schema are: 'first_timers' is an object that contains pattern properties allowing additional object keys following the regex '^[^/#]+$', each mapping to an object that has a required string property 'pull_request' matching the pattern '^[^/#]+/[^/#]+#[0123456789]+$'. 'number_of_first_timers' is a positiveInteger which is required. 'number_of_first_timer_pull_requests' is a positiveInteger which is required. 'first_timer_pull_requests' is an object that contains pattern properties allowing additional object keys following the regex '^[^/#]+/[^/#]+#[0123456789]+$', each mapping to an object that has a required string property 'first_timer' matching the pattern '^[^/#]+$'. 'owner' is an object of type minimal_user which is required. All properties in 'first_timers' and 'first_timer_pull_requests' must not allow additional properties. No additional properties are allowed at the root level.
