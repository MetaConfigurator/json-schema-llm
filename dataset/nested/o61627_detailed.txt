The schema describes a configuration profile for identity preference. It is an object type. The main properties of the schema are: 'Name' is a string required. An email address (RFC822), DNS hostname, or other name that uniquely identifies a service requiring this identity. 'PayloadCertificateUUID' is a string optional. The UUID of another payload within the same profile that installed the identity; for example, a 'com.apple.security.pkcs12' or 'com.apple.security.scep' payload. It must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadDescription' is a string optional. Description of the payload, with a default value of 'Identity Preference Payload'. 'PayloadDisplayName' is a string optional. Name of the payload, with a default value of 'Identity Preference Payload'. 'PayloadIdentifier' is a string optional. A unique identifier for the payload, dot-delimited, with a default value of 'com.apple.security.identitypreference'. 'PayloadOrganization' is a string optional. This value describes the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string optional. The type of the payload, a reverse dns string, with a default value of 'com.apple.security.identitypreference'. 'PayloadUUID' is a string optional. Unique identifier for the payload (format 01234567-89AB-CDEF-0123-456789ABCDEF). It must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$', with a default value of ''. 'PayloadVersion' is a number optional. The version of the whole configuration profile, with a default value of 1. No additional properties are allowed at the root level.
