The schema describes a structure for a shipyard communication message. It is an object type. The main properties of the schema are: '$schemaRef' is a string, required; 'header' is an object, required, with additional properties allowed. It includes required fields: 'uploaderID', 'softwareName', and 'softwareVersion', which are all strings. 'message' is an object, required, with no additional properties allowed. It includes the required fields: 'systemName', 'stationName', 'marketId' (a number), 'timestamp' (with date-time format), and 'ships' (an array of unique ship names, string with a minimum length of 1). No additional properties are allowed at the root level.
