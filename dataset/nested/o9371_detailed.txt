The schema describes the structure for navigation elements. It is an object type. The main properties of the schema are: 'links' is an array that generates links for these navigation targets, each item is an object with required fields 'target' (a string) and 'htmlLabel' (a string). 'buttons' is an array that generates buttons for these navigation targets, each item is an object with required fields 'target' (a string) and 'htmlLabel' (a string). No additional properties are allowed at the root level.
