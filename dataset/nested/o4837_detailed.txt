The schema describes the validation for a term. It is an object type. The main properties of the schema are: 'attributes' is an object that may contain any number of properties matching a regex pattern. Each property must be an array containing at least one item. Each item in the array is an object requiring the 'value' property, which is a string with a minimum length of 1. Optionally, it may contain a 'units' property, which is also a string with a minimum length of 1. Additionally, each item can have a 'terms' property, which is an array containing objects that must have a 'url' property of type string formatted as a URI. No additional properties are allowed at the root level.
