The schema describes an invocation manifest for fMRIPREP: A Robust Preprocessing Pipeline for fMRI Data (1.0.0-rc5). It is an object type. The main properties of the schema are: 'inputs' is an object that is required and contains a required property 'api_key', which is also an object with no defined properties. 'config' is an object that is required and contains a required property 'freesurfer', which is a boolean and has a default value of true. No additional properties are allowed at the root level.
