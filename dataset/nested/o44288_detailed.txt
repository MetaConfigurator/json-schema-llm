The schema describes a plugin instance. It is an object type. The main properties of the schema are: 'actions' is an array A list actions associated to this plugin. 'data' is an object The eventual data associated to the plugin, specific to each plugin. 'enabled' is a boolean Is the instance enabled ? 'engine' is a string The UID of the engine the instance is of. 'events' is an object Events to react to, specific to each plugin. 'index' is a number The index of the plugin on the screen. 'keep' is a boolean Do we need to keep the plugin ? 'options' is an object The options of this instance, specific and different for each plugin. 'scenes' is an array The list of UID of scenes associated to the instance. 'tags' is an array A list of tags. 'uid' is a string The UID of the instance. The required properties are: 'uid', 'tags', 'options', 'actions', 'events'. No additional properties are allowed at the root level.
