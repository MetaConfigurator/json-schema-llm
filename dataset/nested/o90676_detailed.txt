The schema describes configuration settings for a rest-proxy service. It is an object type. The main properties of the schema are: 'proxy' is an object that contains settings for the rest-proxy application. It is required and has the following properties: 'cpus' is a number that specifies CPU shares to allocate to each rest-proxy instance. It is required, with a default value of 1 and a minimum of 1. 'instances' is an integer that specifies the number of instances to run. It is required, with a default value of 1 and a minimum of 1. 'mem' is a number that specifies memory (MB) to allocate to each rest-proxy task. It is required, with a default value of 1024.0 and a minimum of 256.0. 'name' is a string that specifies the name for this rest-proxy application. It is required, with a default value of 'rest-proxy'. 'schema-registry-url' is a string that specifies the connection URL for schema registry. It is required, with a default value of 'http://8.0.8.1:8081'. No additional properties are allowed at the root level.
