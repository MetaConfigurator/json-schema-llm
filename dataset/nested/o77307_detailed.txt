The schema describes a package descriptor. It is an object type. The main properties of the schema are: 'descriptor_version' is a string and required. 'group' is a string and required. 'name' is a string and required. 'version' is a string and required. 'package_maintainer' is a string and optional. 'package_description' is a string and optional. 'package_md5' is a string and optional. 'ackage_signature' is a string and optional. 'entry_service_template' is a string and optional. 'sealed' is a boolean and optional. 'package_content' is an array of objects, where each object must have 'name' and 'content-type' as required strings. 'artifact_dependencies' is an array of objects, each of which can include 'name', 'url', 'md5', and an optional 'credentials' object, containing optional 'username' and 'password' strings. No additional properties are allowed at the root level.
