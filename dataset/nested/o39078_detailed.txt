The schema describes generating a phone number list. It is an object type. The main properties of the schema are: '$schema' is a string, optional. '$tool' is a string, required, must be set to 'city-mobile-sections'. 'column' is a string, required, with allowed values: 'province', 'city'. 'dist' is a string, required, representing the output path (relative to the pwd). 'driver' is an object, required, defining the Typeorm driver with additional properties disallowed. It contains the following required fields: 'type', 'host', 'port', 'username', 'password', 'database'. 'type' is a string with a default value of 'postgres' and allowed values: 'mysql', 'postgres', 'mariadb', 'sqlite', 'oracle', 'mssql', 'websql'. 'host' is a string with a default value of 'localhost'. 'password' is a string with a default value of 'postgres'. 'port' is an integer with a default value of 5432. 'database' is a string with a default value of 'phone'. 'username' is a string with a default value of 'postgres'. No additional properties are allowed at the root level.
