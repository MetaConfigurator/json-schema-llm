The schema describes an agent profile. It is an object type. The main properties of the schema are: 'address' is a string Agent physical address. 'agentId' is an integer Id referencing agent. 'avatar' is a string Agent avatar image link, with a format of uri. 'description' is a string Agent biography. 'email' is a string Agent email address, with a format of email. 'id' is an integer Id referencing profile, required. 'name' is a string An agent name. 'phone' is a string Agent phone number. 'website' is a string Agent website link, with a format of uri. The properties 'id' and 'agentId' are required. No additional properties are allowed at the root level.
