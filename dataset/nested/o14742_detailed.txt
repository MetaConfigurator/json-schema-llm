The schema describes the configuration for notification services. It is an object type. The main properties of the schema are: 'pushover' is an object which describes Pushover notification settings. It is required and has properties like 'enabled' (a boolean), 'appToken' (a string), 'deliveryToken' (a string), 'priority' (an integer with a default of 2, minimum -2, maximum 2), 'retry' (an integer with a default of 30 and minimum 30), and 'expire' (an integer with a default of 3600, minimum 60, maximum 86400). 'slack' is an object which describes Slack notification settings. It is required and has properties like 'enabled' (a boolean), 'user' (a string), 'destination' (a string), and 'url' (a string). The schema requires 'pushover' and 'slack'. No additional properties are allowed at the root level.
