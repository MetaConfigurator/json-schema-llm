The schema describes an administrator SQL query. It is an object type. The main properties of the schema are: 'description' is a string that provides a brief one-line description of the query, required. 'resultFormats' is an object that allows custom formats for result columns, with additional properties that must be a string from the enum value 'pre'. 'params' is an array of objects representing parameters for the query, where each object is required to have 'name' and 'description' as strings. Additionally, 'default' is an optional string for specifying a default value. No additional properties are allowed at the root level.
