The schema describes a generic type of item. It is an object type. The main properties of the schema are: 'additionalType' is a string providing an additional type for the item, typically used for adding more specific types from external vocabularies in microdata syntax. It is optional. 'alternateName' is a string serving as an alias for the item. It is optional. 'description' is a string providing a short description of the item. It is required. 'id' is a string that uniquely identifies the item. It is required. 'image' is a string representing the URL of an image of the item, formatted as a URI. It is optional. 'name' is a string indicating the name of the item. It is required. 'sameAs' is a string for the URL of a reference Web page indicating the item's identity. It is optional. 'url' is a string for the URL of the item, formatted as a URI. It is optional. 'version' is an integer representing the version of the item. It is optional. No additional properties are allowed at the root level.
