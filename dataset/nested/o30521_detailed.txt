The schema describes a load balancer group configuration. It is an object type. The main properties of the schema are: 'backuppersistencetimeout' is an integer optional timeout for backup persistence. 'cookiedomain' is a string optional domain for the cookie. 'cookiename' is a string optional name for the cookie. 'mastervserver' is a string optional master virtual server. 'name' is a string optional name of the load balancer group. 'newname' is a string optional new name for the load balancer group. 'persistencebackup' is a string optional backup persistence method with allowed values: SOURCEIP, NONE. 'persistencetype' is a string optional persistence method with allowed values: SOURCEIP, COOKIEINSERT, RULE, NONE. 'persistmask' is a string optional mask for persistence. 'rule' is a string optional rule for the load balancer. 'td' is an integer readonly timeout duration. 'timeout' is an integer optional timeout setting. 'usevserverpersistency' is a string optional setting for virtual server persistency with allowed values: ENABLED, DISABLED. 'v6persistmasklen' is an integer optional length of the persistence mask for IPv6. No additional properties are allowed at the root level.
