The schema describes a configuration for yarn and loader timestamps along with arrays of fabric artifact versions. It is an object type. It defines several reusable components under 'definitions': 'FabricArtifactVersion' is an object, representing the version of a fabric artifact. It requires the fields 'maven', 'stable', and 'version'. The 'maven' and 'version' fields are strings, 'stable' is a boolean, and 'build' is a number. The main properties of the schema are: 'yarnTimestamp' is a string with a default value of an empty string, required; 'loaderTimestamp' is a string with a default value of an empty string, required; 'yarns' is an array of 'FabricArtifactVersion' items with a default value of an empty array, required; 'loaders' is an array of 'FabricArtifactVersion' items with a default value of an empty array, required. No additional properties are allowed at the root level.
