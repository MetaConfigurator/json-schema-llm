The schema describes an invocation manifest for Siemens to ISMRM-RD Converter. It is an object type. The main properties of the schema are: 'inputs' is an object which is required and contains: 'dat' which is an object and contains: 'type' which is an enum, allowing the value 'Siemens MR Raw'; 'user_stylesheet' which is an object and contains: 'name' which is a string that must match the pattern '^.*.xsl$'; 'user_map' which is an object and contains: 'name' which is a string that must match the pattern '^.*.xml$'. 'config' is an object which is required and contains: 'debug' which is a boolean with a default value of false; 'measNum' which is an integer with a default value of 1. No additional properties are allowed at the root level.
