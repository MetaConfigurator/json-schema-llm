The schema describes the format of the properties file for the endpoints. It is an object type. The main properties of the schema are: 'method' is a string that restricts requests to the given method, allowing values 'GET', 'POST', 'PUT', 'DELETE', 'PATCH'. It is required. 'delay' is an integer that delays the response of a request in milliseconds. It is optional. 'name' is a string that names a request, recommended in combination with other documentation. It is optional. 'description' is a string that adds an additional description to a request, improving documentation. It is optional. 'category' is a string used to group similar requests into categories for maintainability. It is optional. 'getParameters' is an object that contains a list of parameters added to the URL. It is optional. 'postParameters' is an object similar to 'getParameters', used for requests with a body. It is optional. 'postJson' is an object used for requests requiring JSON structure in the body data. It is optional. 'responseCode' is an integer representing the HTTP result code, defaulting to 200 in case of success if not specified. It is optional. 'alternatives' is an array providing alternative responses based on conditions, containing references to the entire schema. It is optional. 'generates' is a string that sets up things like an index page or file server, allowing values 'indexPage', 'fileList', 'commandConsole'. It is optional. 'redirect' is a string specifying a path to redirect to. It is optional. 'generatesJson' is a boolean that generates the file list as a JSON response when visiting the index page, applicable only for file servers. It is optional. 'includeSHA256' is a boolean that generates a sha256 hash of each file being listed in the JSON response of the file server. It is optional. 'hidden' is a boolean that hides alternative responses from the index page. It is optional. 'responsePath' is a string that specifies the prefix of the alternative response file. It is optional. 'checkHeaders' is an object that checks these headers when finding an alternative response. It is optional. No additional properties are allowed at the root level.
