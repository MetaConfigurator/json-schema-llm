The schema describes a planet. It is an object type. The main properties of the schema are: 'climate' is a string that describes the climate of this planet. Comma-separated if diverse. It is required. 'created' is a string that represents the ISO 8601 date format of the time that this resource was created. It is required. 'diameter' is a string that describes the diameter of this planet in kilometers. It is required. 'edited' is a string that represents the ISO 8601 date format of the time that this resource was edited. It is required. 'films' is an array that contains Film URL Resources that this planet has appeared in. It is required. 'gravity' is a string that denotes the gravity of this planet, where 1 is normal. It is required. 'name' is a string that describes the name of this planet. It is required. 'orbital_period' is a string that describes the number of standard days it takes for this planet to complete a single orbit of its local star. It is required. 'population' is a string that describes the average population of sentient beings inhabiting this planet. It is required. 'residents' is an array that contains People URL Resources that live on this planet. It is required. 'rotation_period' is a string that describes the number of standard hours it takes for this planet to complete a single rotation on its axis. It is required. 'surface_water' is a string that describes the percentage of the planet surface that is naturally occurring water or bodies of water. It is required. 'terrain' is a string that describes the terrain of this planet. Comma-separated if diverse. It is required. 'url' is a string that represents the hypermedia URL of this resource. It is required. No additional properties are allowed at the root level.
