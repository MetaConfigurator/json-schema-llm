The schema describes range information. It is an object type. The main properties of the schema are: 'currentRangeExtent' is an array that contains the min and max values within which the features are visible, with a minimum and maximum of 2 items of type number. 'field' is a string that describes the field name to be used for the range. 'fullRangeExtent' is an array that contains the min and max values of all the features for this rangeInfo, also with a minimum and maximum of 2 items of type number. 'name' is a string representing a unique name that can be referenced by webMap.activeRanges. 'type' is a string indicating the type of range object, with an enum value of 'rangeInfo'. 'field', 'name', and 'type' are required. No additional properties are allowed at the root level.
