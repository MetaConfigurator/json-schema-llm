The schema describes an Imgur post block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string in date-time format, indicating an update that occurred on the node this block is attached to, and does not indicate an update to the block itself. It is optional. 'aside' is a boolean indicating if the content represents a portion indirectly related to the document's main content, where true means it frequently appears as sidebars or call-out boxes. It is optional. 'id' is a string that must match the pattern '^[\w\/-]+$'. It is required. 'show_context' is a boolean that defaults to true and indicates whether to show the post details. It is optional. No additional properties are allowed at the root level.
