The schema describes the configuration of the visibility sender. It is an object type. It defines several reusable components under 'definitions': 'heap' is an object, it requires the fields 'num_stations' and 'num_pols'. 'stream_config' is an object, it requires the fields 'max_packet_size', 'rate', 'burst_size', and 'max_heaps'. The main properties of the schema are: 'destination_host' is a string, it is required and describes the hostname or IP of the send target. 'destination_port_start' is an integer, it is required and describes the port of the send target. 'heap' is a reference to the 'heap' definition. 'num_streams' is an integer, it is required and describes an unspecified aspect. 'num_workers' is an integer, it is required and describes an unspecified aspect. 'start_channel' is an integer, it is required and describes an unspecified aspect. 'stream_config' is a reference to the 'stream_config' definition. No additional properties are allowed at the root level.
