The schema describes the BBMRI Metadata. It is an object type. The main properties of the schema are: 'study_id' is a string the unique ID or acronym for the study. 'study_name' is a string the name of the study in English. 'study_description' is a string a description of the study aim. 'principal_investigator' is a string the name of the person responsible for the study or the principal investigator. 'study_design' is an array the type of study, which can include several of the following values: 'Case-control', 'Cohort', 'Cross-sectional', 'Longitudinal', 'Twin-study', 'Quality control', 'Population-based', 'Other'. 'disease' is a string the disease of main interest in the sample collection, if any. 'categories_of_data_collected' is an array the type of data collected in the study, which can include several of the following values: 'Biological samples', 'Register data', 'Survey data', 'Physiological measurements', 'Imaging data', 'Medical records', 'Other'. 'planned_sampled_individuals' is an integer number of individuals with biological samples planned for the study. 'planned_total_individuals' is an integer total number of individuals planned for the study with or without biological samples. 'sex' is an array the sex of the study participants, which can include several of the following values: 'Female', 'Male', 'Other'. 'age_interval' is a string age interval of youngest to oldest study participant, for example 40-80. 'material_type' is an array the nature of the biological samples that are included in the study, which can include several of the following values: 'Whole blood', 'Plasma', 'Serum', 'Urine', 'Saliva', 'CSF', 'DNA', 'RNA', 'Tissue', 'Faeces', 'Other'. No additional properties are allowed at the root level.
