The schema describes the definition of WDS web services. It is an object type. It defines several reusable components under 'definitions': 'datasource' is a string, FAOSTAT datasource. It has a default value of 'faostatdb' and allows the following values: 'faostat', 'faostatdb', 'faostatglbl', 'faostatprod', 'faostatproddiss'. 'lang_faostat' is a string, language of the output. It has a default value of 'E' and allows the following values: 'E', 'F', 'S', 'A', 'C', 'R'. The main properties of the schema are: 'groupsanddomains' is an object, list of FAOSTAT groups and domains. It is optional. It contains a 'schema' property. 'schema' is an object, description of the input parameters. It contains properties: 'method', which has a default value of 'GET'; 'endpoint', which has a default value of '/groupsanddomains'; 'datasource', which references the 'datasource' definition; 'lang_faostat', which references the 'lang_faostat' definition. 'bulkdownloads' is an object, list of FAOSTAT bulk downloads. It is optional. It contains a 'schema' property. 'schema' is an object, description of the input parameters. It contains properties: 'method', which has a default value of 'GET'; 'datasource', which references the 'datasource' definition; 'domain', which is a string, FAOSTAT Domain Code; 'lang_faostat', which references the 'lang_faostat' definition. 'procedures/usp_GetListBox' is an object, list of FAOSTAT bulk downloads. It is optional. It contains properties: 'datasource', which references the 'datasource' definition; 'domain', which is a string, FAOSTAT Domain Code; 'tab_group', which is an integer, tab group as displayed in FAOSTAT Download; 'tab_index', which is an integer, tab index within the group; 'lang_faostat', which references the 'lang_faostat' definition. No additional properties are allowed at the root level.
