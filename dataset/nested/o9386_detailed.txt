The schema describes a configuration for place redirection. It is an object type. The main properties of the schema are: 'redirectOn' is an object that provides globally defined redirects for certain edge cases. It is required and contains an optional property 'unknownPlace', which is a string that loads whenever the requested place doesn't exist. 'places' is an object that contains the places for this flow. It is required and the keys must be valid event topics. Each place can have several properties: 'patterns' is an array of strings, requiring at least one item that represents non-empty URL patterns for routing; 'page' is a string representing the page to render for this place; 'redirectTo' is a string representing the place to redirect to when accessing this place; 'redirectToPath' is a string representing the path to redirect to when accessing this place, including values for relevant place parameters; 'defaultParameters' is an object that can include additional properties of types string, boolean, or null, providing default values for optional parameters. Each property in 'places' cannot have additional properties. No additional properties are allowed at the root level.
