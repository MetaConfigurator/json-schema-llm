The schema describes the configuration for handling events and filters. It is an object type. The main properties of the schema are: 'events' is an object that displays event bus activity received through wrapper events. It is required and defines the property 'stream', which is a string that describes the event topic through which to gather events, and 'bufferSize', which is an integer with a default value of 5000 that defines the maximum number of event items to keep in memory. 'filter' is an object that controls filter settings. It defines the property 'hidePatterns', which is an array with a default value of ['visibility', 'i18n'] that controls pattern-related default filter settings, 'hideSources', which is an array with a default value of [] that controls source-related default filter settings, 'hideInteractions', which is an array with a default value of ['subscribe', 'unsubscribe'] that controls interaction-related default filter settings, and 'resource', which is a string with the format 'topic' and an axRole of 'outlet' that describes the event topic through which to receive filter information. No additional properties are allowed at the root level.
