The schema describes a configuration profile for disc recording. It is an object type. The main properties of the schema are: 'BurnSupport' is a string that sets disc burning options, with allowed values 'on', 'off', and 'authenticate'. 'PayloadDescription' is a string that describes the payload, with a default value of 'Disc Recording'. 'PayloadDisplayName' is a string that names the payload, also with a default value of 'Disc Recording'. 'PayloadIdentifier' is a string containing a unique identifier for the payload, with a default value of 'com.apple.DiscRecording'. 'PayloadOrganization' is a string describing the issuing organization of the profile. 'PayloadType' is a string indicating the type of the payload, with a default value of 'com.apple.DiscRecording'. 'PayloadUUID' is a string representing a unique identifier for the payload, formatted according to a specific pattern, with a default value of an empty string. 'PayloadVersion' is a number that indicates the version of the entire configuration profile, with a default value of 1. No additional properties are allowed at the root level.
