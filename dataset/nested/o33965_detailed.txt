The schema describes options for ng-swagger-gen. It is an object type. The main properties of the schema are: '$schema' is a string that specifies the schema version. 'apiModule' is a boolean that indicates whether or not to generate the Angular module which provides all services. It is optional and defaults to true. 'camelCase' is a boolean that generates service methods in camelCase instead of PascalCase. It is optional and defaults to false. 'customFileSuffix' is an object that defines custom suffixes for autogenerated files. It is optional and has default properties: example is '.example', model is '', and service is '.service'. 'defaultTag' is a string that specifies the assumed tag for operations that don't define any, with a default value of 'Api'. 'enumModule' is a boolean that indicates whether to export an additional TypeScript module for each enum model. It is optional and defaults to true. 'excludeTags' is an array that filters generated services, excluding any service corresponding to specified tags. It is optional. 'generateExamples' is a boolean that indicates whether to generate example files from model sections. It is optional and defaults to false. 'ignoreUnusedModels' is a boolean that indicates whether to ignore model files not referenced by any operation. It is optional and defaults to true. 'includeTags' is an array that filters generated services, including only those corresponding to specified tags. It is optional. 'minParamsForContainer' is an integer that indicates the minimum number of parameters to wrap in a container class, optional with a default of 2. 'modelIndex' is a boolean indicating whether to generate the file that exports all models, optional and defaults to true. 'output' is a string indicating where to write generated files, optional with a default of 'src/app/api'. 'prefix' is a string that provides a prefix to generated global classes, optional with a default of 'Api'. 'removeStaleFiles' is a boolean that indicates whether to remove files in the output folder not generated by ng-swagger-gen. It is optional and defaults to true. 'serviceIndex' is a boolean indicating whether to generate the file that exports all services, optional with a default of true. 'sortParams' is a string that determines how to sort operation parameters with allowed values: 'none', 'asc', and 'desc'. It is optional and defaults to 'desc'. 'swagger' is a string that specifies the location of the swagger descriptor in JSON format. It is required. 'templates' is a string that provides a path to override Mustache templates used for file generation. It is optional. 'timeout' is a number indicating the amount of time to wait for a server response when downloading files. It is optional and defaults to 20000. No additional properties are allowed at the root level.
