The schema describes retrieving the status of a running export file. It is an object type. The main properties of the schema are: 'export_files' is an object that includes details about the export file status, including a unique identifier of this export file ('id'), a hypertext reference to the resource ('href'), a unique identifier of the associated export ('export_id'), the current status of this export file ('status'), and a secure URL for downloading the generated file ('download_url'). All fields are required. The 'status' field can have enum values: 'Initializing', 'Queued', 'In Progress', 'Available', 'Success With Errors', 'Empty List', 'Failed'. Additional properties are not allowed at the root level.
