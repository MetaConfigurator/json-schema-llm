The schema describes ND6 RA variables. It is an object type. The main properties of the schema are: 'ceaserouteradv' is a string indicating whether Cease Router Advertisement is enabled, with allowed values 'YES' or 'NO'. 'currhoplimit' is an integer indicating the current hop limit. 'defaultlifetime' is an integer representing the default lifetime. 'lastrtadvtime' is an integer indicating the last Router Advertisement time and is read-only. 'linkmtu' is an integer indicating the link MTU. 'managedaddrconfig' is a string that specifies whether managed address configuration is enabled, with allowed values 'YES' or 'NO'. 'maxrtadvinterval' is an integer specifying the maximum Router Advertisement interval. 'minrtadvinterval' is an integer specifying the minimum Router Advertisement interval. 'nextrtadvdelay' is an integer indicating the next Router Advertisement delay and is read-only. 'onlyunicastrtadvresponse' is a string that determines whether to only respond to unicast Router Advertisement requests, with allowed values 'YES' or 'NO'. 'otheraddrconfig' is a string that indicates whether other address configuration is enabled, with allowed values 'YES' or 'NO'. 'reachabletime' is an integer representing the reachable time. 'retranstime' is an integer indicating the retransmission time. 'sendrouteradv' is a string indicating whether to send Router Advertisements, with allowed values 'YES' or 'NO'. 'srclinklayeraddroption' is a string that specifies whether to use the source link-layer address option, with allowed values 'YES' or 'NO'. 'vlan' is an integer representing the VLAN identifier. No additional properties are allowed at the root level.
