The schema describes a filter configuration. It is an object type. The main properties of the schema are: 'genericLabel' is a string optional label for the filter. 'length' is a number optional field representing the length of the filter. 'label' is a string optional label for display purposes. 'select' is an array of objects representing selectable options. Each object in 'select' contains: 'name' as a string, 'label' as a string, 'class' as a string, 'selected' as an object with 'value' as a string and 'label' as a string, and 'options' as an array of objects with 'value' as a string and 'label' as a string. 'multipleSelect' is an array of objects representing multiple selectable options, similar to 'select', but includes additional fields: 'labelSelected' as a string, 'labelSelectAll' as a string, 'labelSubmit' as a string, 'submitOnSelection' as a boolean with a default value of false, and 'hideOnSelection' as a boolean with a default value of false. 'search' is an object containing properties: 'id' as a string, 'placeholder' as a string, 'minLength' as a number, and 'label' as a string. No additional properties are allowed at the root level.
