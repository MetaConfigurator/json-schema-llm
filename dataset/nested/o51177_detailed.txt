The schema describes a dictionary that captures a set of key/value pairs. It is an object type. The main properties of the schema are: Custom hash keys that match the pattern '^[a-zA-Z0-9_-]{1,}$' are optional and map to a string. Specific keys for hash algorithms include: 'MD5' which is a string specifying the MD5 message digest algorithm, must match the pattern '^[a-fA-F0-9]{32}$'; 'MD6' which is a string for the MD6 message digest algorithm, must match the pattern '^[a-fA-F0-9]{32}|[a-fA-F0-9]{40}|[a-fA-F0-9]{56}|[a-fA-F0-9]{64}|[a-fA-F0-9]{96}|[a-fA-F0-9]{128}$'; 'RIPEMD-160' which is a string for the RIPEMD-160 algorithm, must match the pattern '^[a-fA-F0-9]{40}$'; 'SHA-1' which is a string for the SHA-1 algorithm, must match the pattern '^[a-fA-F0-9]{40}$'; 'SHA-224' which must match '^[a-fA-F0-9]{56}$'; 'SHA-256' which must match '^[a-fA-F0-9]{64}$'; 'SHA-384' which must match '^[a-fA-F0-9]{96}$'; 'SHA-512' which must match '^[a-fA-F0-9]{128}$'; 'SHA3-224' which must match '^[a-fA-F0-9]{56}$'; 'SHA3-256' which must match '^[a-fA-F0-9]{64}$'; 'SHA3-384' which must match '^[a-fA-F0-9]{96}$'; 'SHA3-512' which must match '^[a-fA-F0-9]{128}$'; 'ssdeep' which must match '^[a-zA-Z0-9/+:.]{1,128}$'; and 'WHIRLPOOL' which must match '^[a-fA-F0-9]{128}$'. No additional properties are allowed at the root level.
