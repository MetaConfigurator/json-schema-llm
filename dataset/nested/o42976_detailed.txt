The schema describes configuration settings for a system. It is an object type. The main properties of the schema are: 'dcos_url' is a string that specifies the URL to the location of the DCOS. It is optional. 'email' is a string that represents your email address. It is optional. 'mesos_master_url' is a string that specifies the Mesos Master URL, and must be of the format: "http://host:port". It is optional. 'refresh_token' is a string that represents your OAuth refresh token. It is optional. 'reporting' is a boolean that indicates whether to report usage events to Mesosphere, with a default value of true. 'timeout' is an integer that specifies the request timeout in seconds, with a default value of 5 and a minimum value of 1. It is optional. 'token' is a string that represents your OAuth access token. It is optional. No additional properties are allowed at the root level.
