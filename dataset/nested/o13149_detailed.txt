The schema describes job configuration settings. It is an object type. The main properties of the schema are: 'file1' is a string used to select 'file1' from the chosen inputs. It is optional and has an enum value of '__PIRUS_INPUT_FILES__'. 'file2' is a string used to select 'file2' from the chosen inputs. It is optional and has an enum value of '__PIRUS_INPUT_FILES__'. 'ref' is a string used to choose the reference genome from available databases. It is optional and has an enum value of '__PIRUS_DB_REFS__'. 'confirm' is a boolean indicating agreement, with a default value of false. 'witherror' is a boolean indicating if the job should crash, with a default value of false. 'notification_enable' is a boolean for real-time notification, with a default value of true. 'duration' is an integer specifying the job's duration in seconds, with a default value of 100. 'outfilename' is a string for naming the result file generated by the job. It is required and has a default value of 'result.txt'. No additional properties are allowed at the root level.
