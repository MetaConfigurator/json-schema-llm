The schema describes Fibaro Home Center Event JSON Schema. It is an object type. The main properties of the schema are: 'id' is an integer Event ID, optional. 'type' is a string Event Type, required with an enum value of 'DEVICE_EVENT'. 'timestamp' is an integer Event timestamp epoch, optional. 'deviceID' is an integer Device ID, optional. 'deviceType' is a string Device Type, optional with a maximum length of 20 characters. 'propertyName' is a string Updated property name, optional with a maximum length of 20 characters. 'oldValue' is a number Old value, optional. 'newValue' is a number New value, optional. No additional properties are allowed at the root level.
