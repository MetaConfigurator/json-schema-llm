The schema describes IBM App Connect Template Gallery metadata. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'templates' is an array of objects representing templates from the template catalog. It is required and must contain at least one item, where each item has required properties: 'name' is a string with a maximum length of 98 characters, 'summary' is a string, 'sourceApp' is a string, 'targetApps' is an array of strings with a minimum of one item, 'tags' is an array of strings, and 'offerings' is an array of strings. No additional properties are allowed at the root level.
