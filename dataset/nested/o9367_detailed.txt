The schema describes a configuration for search and localization settings. It is an object type. The main properties of the schema are: 'search' is an object required for the configuration, containing a required property 'resource' which is a string formatted as a topic with an axRole of 'inlet'. 'locations' is an optional object that may include a property 'resource' which is a string formatted as a topic with an axRole of 'inlet', and another optional property 'searchingOn', which is an array of strings formatted as topics with an axRole of 'inlet' and axPattern of 'flag', describing one or more flags indicating that geocoding is still in progress. 'i18n' is an optional object containing a property 'locale', which is a string formatted as a topic with an axRole of 'inlet' and a default value of 'default'. No additional properties are allowed at the root level.
