The schema describes an object format for educational resources. It is an object type. The main properties of the schema are: 'subject' is a string which is required and has a maximum length of 255. 'description' is a string which is required. 'category' is a string which is required and has a maximum length of 255. 'school_id' is a string which is required, with a minimum and maximum length of 36. 'attachments' is an array of objects, which are optional, and each object within this array has the following properties: 'id' is a string with a minimum length of 1; 'name' is a string with a minimum length of 1; 'size' is an integer with a minimum value of 1. No additional properties are allowed at the root level.
