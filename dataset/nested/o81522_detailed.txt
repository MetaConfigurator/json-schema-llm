The schema describes a conference session structure. It is an object type. The main properties of the schema are: 'speaker' is an object that contains required fields 'firstName', 'lastName', 'email', and 'bio', all of which are strings. 'session' is an object that contains required fields 'title', 'abstract', 'type', and 'length', all of which are strings. 'conference' is an object that contains required fields 'name', 'beginDate', and 'endDate', all of which are strings. 'topic' is an object that contains required fields 'primary', which is a string, and 'secondary', which is an array of strings. 'audience' is an object that contains required fields 'takeaway', which is a string, 'jobTitles', which is an array of strings, and 'level', which is a string. 'installation' is an array of strings. No additional properties are allowed at the root level.
