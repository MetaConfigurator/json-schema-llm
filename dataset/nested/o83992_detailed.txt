The schema describes an S3 event notification for Lambda. It is an object type. It defines several reusable components under 'definitions': 's3' is an object, containing the bucket & key data. It has required fields 'bucket' and 'object'. The 'bucket' has a required property 'name' and an optional property 'arn'. The 'object' has a required property 'key' and optional properties 'eTag', 'sequencer', and 'size'. The main properties of the schema are: 'Records' is an array of s3event, which is a required field. The s3event contains required fields 'awsRegion', 'eventSource', 'eventName', 'eventTime', and 's3', which references the s3 definition. No additional properties are allowed at the root level.
