The schema describes frame data for a Tekken 7 character. It is an object type. The main properties of the schema are: 'metadata' is an object containing data about the frame data and is required. It has required fields: 'ver', 'game', 'character', and 'type'. 'moves' is an array of objects representing a collection of character moves and is also required. Each move object contains required fields: 'notation', 'hit_level', 'damage', 'speed', 'on_block', 'on_hit', and 'on_ch'. No additional properties are allowed at the root level.
