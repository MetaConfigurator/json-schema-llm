The schema describes information about stations. It is an object type. The main properties of the schema are: 'id' is a string unique identifier for this station, required. 'city' is a string viewable city name where the station is located, required. 'cityid' is a string unique identifier of the city where this station is located, required. 'lat' is a number latitude of station location, required. 'lon' is a number longitude of station location, required. 'tanks' is an object tanks located at this station, required. It has required properties: 'regular' which is a string tankid value of the regular fuel tank located at this station, and 'premium' which is a string tankid value of the premium fuel tank located at this station. 'diesel' is an optional string tankid value of the diesel fuel tank located at this station. 'pumps' is an array list of pumps located at this station, containing string pumpId values for pump located at this station, required. No additional properties are allowed at the root level.
