The schema describes creating a new generic web worker definition in the given or default project. It is an object type. The main properties of the schema are: 'name' is a string that represents the name of the worker. It is required. 'path' is a string that specifies the path at which to create the worker file, relative to the current workspace. It is optional. 'project' is a string that denotes the name of the project. It is required. 'snippet' is a boolean that indicates whether to add a worker creation snippet in a sibling file of the same name. It has a default value of true and is optional. 'target' is a string that defines the target to apply web worker to. It has a default value of 'build' and is optional. No additional properties are allowed at the root level.
