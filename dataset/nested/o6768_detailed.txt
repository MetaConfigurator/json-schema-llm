The schema describes a data table for HEPData comprising independent_variables and dependent_variables. It is an object type. The main properties of the schema are: 'independent_variables' is an array of objects containing the independent variable data. Each object is required to have 'header' and 'values'. The 'header' is an object that has a required string 'name' and an optional string 'units'. The 'values' is an array of objects, where each object is required to contain a 'value' (which can be a string or number) and may optionally include 'low' and 'high' as numbers. 'dependent_variables' is an array of objects containing the dependent variable data. Each object in this array is required to have 'header' and 'values', and may optionally include 'qualifiers'. The 'header' follows the same structure as in independent_variables. The 'qualifiers' is an array of objects, each requiring 'name' and 'value' (both can be strings or numbers), and may include an optional 'units'. The 'values' in dependent_variables is also an array of objects, each required to have a 'value' (which can be a string or number) and may include an optional array of 'errors'. Each error must conform to at least one of two structures: one with a required 'symerror', or one with a required 'asymerror', which itself is an object requiring 'minus' and 'plus' (both can be numbers or strings). No additional properties are allowed at the root level.
