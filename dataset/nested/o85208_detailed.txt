The schema describes a MWF sequence indicator component's schema definition. It is an object type. It defines several reusable components under 'definitions': 'activeIndicator' is an object, which has a required field 'config' that is itself an object containing a required boolean field 'checked'. 'indicator' is an object, which has a required field 'content' that is also an object containing required fields 'label', 'title', and 'id', all of which are strings. The main properties of the schema are: 'content' is an object that has a required field 'items', which is an array that can contain items which must match either the 'indicator' or 'activeIndicator' definitions. No additional properties are allowed at the root level.
