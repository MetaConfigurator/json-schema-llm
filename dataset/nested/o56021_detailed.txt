The schema describes a Fibaro Home Center Variable. It is an object type. The main properties of the schema are: 'name' is a string identifier for the variable, required, with a maximum length of 18, a minimum length of 1, and must match the pattern '^[A-Za-z][A-Za-z0-9_]*'. 'value' is a string representing the value of the variable, required, with a maximum length of 8177. 'readOnly' is a boolean that indicates the read-only flag and is optional. 'isEnum' is a boolean that indicates if it is a predefined variable flag and is optional. 'enumValues' is an array of strings that contains predefined values and is optional. 'created' is an integer representing the variable created timestamp, optional, with a minimum value of 0. 'modified' is an integer representing the variable modified timestamp, optional, with a minimum value of 0. No additional properties are allowed at the root level.
