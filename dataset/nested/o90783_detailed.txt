The schema describes configurations for a MySQL service and its related properties. It is an object type. The main properties of the schema are: 'mysql' is an object that contains MySQL service configuration properties. It is required and does not allow additional properties. It has the following required fields: 'cpus' is a number with a default of 0.1 and a minimum value of 0.1; 'mem' is a number with a default of 128.0 and a minimum value of 128.0; 'instances' is a number with a default of 1.0 and a minimum value of 1.0; 'host_volume' is a string with a default of '/tmp'; 'root_password' is a string that specifies the password for the MySQL root superuser account. 'service' is an object that contains DC/OS service configuration properties. It has an optional field: 'name' is a string with a default of 'mysql'. No additional properties are allowed at the root level.
