The schema describes a RenderTheFuck line chart. It is an object type. The main properties of the schema are: 'labels' is an array of strings, optional. 'lines' is an array of objects, required, each object must have a 'name' of type string (required) and 'data' of type array (required), which contains numbers. 'settings' is an object, required, containing a 'title' of type string (required) with a minimum length of 1. 'wtf' is a string, required, with an enum value of 'line'. No additional properties are allowed at the root level.
