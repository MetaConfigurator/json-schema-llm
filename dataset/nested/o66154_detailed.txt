The schema describes a collection of processing steps. It is an array type. The main properties of the schema are: [0] is an object that represents an individual processing step. It is required and contains the following properties: 'configuration_file' is a string that describes the configuration file, e.g., my_job_config.py; 'output_data' is a string that describes the output data file, e.g., myfile-data.root; 'run_instructions' is a string that describes the run instructions, e.g., README.md. No additional properties are allowed at the root level.
