The schema describes a data structure for documenting a resource. It is an object type. The main properties of the schema are: 'authors' is an array containing at least one string, required. 'classifiers' is an array containing at least one string, optional. 'description' is a string, required. 'homepage' is a string, optional. 'pages' is an array containing at least one string, required. 'reference_urls' is an array containing at least one string, optional. 'title' is a string, required. 'version' is a string, required. No additional properties are allowed at the root level.
