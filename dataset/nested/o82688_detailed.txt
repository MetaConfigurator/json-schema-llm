The schema describes a quote block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string in date-time format representing an update that occurred on the node attached to this block. It is optional. 'aside' is a boolean indicating whether this block represents content indirectly related to the main content, commonly as sidebars or call-out boxes. It is optional. 'text' is a string with a minimum length of 0 and a maximum length of 5000 characters. It is optional. 'source' is a string with a minimum length of 0 and a maximum length of 255 characters. It is optional. 'source_url' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$'. It is optional. 'is_pull_quote' is a boolean. It is optional. No additional properties are allowed at the root level.
