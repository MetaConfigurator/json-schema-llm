The schema describes an object structure used for content publishing. It is an object type. It defines several reusable components under 'definitions': 'guid' is a string, matching a specific UUID pattern. 'guid_list' is an array of GUIDs, where each item is a reference to the 'guid' definition and must be unique. The main properties of the schema are: 'bulk_publishing' is a boolean, optional. 'links' is an object, optional, with the following properties: 'available_translations' is an array of GUIDs, required; 'facet_groups' is an array of GUIDs, required; 'facet_values' is an array of GUIDs, required; 'finder' is an array of GUIDs, required; 'lead_organisations' is an array of GUIDs, required; 'mainstream_browse_pages' is an array of GUIDs, required; 'meets_user_needs' is an array of GUIDs, required; 'ordered_related_items' is an array of GUIDs, required; 'ordered_related_items_overrides' is an array of GUIDs, required; 'organisations' is an array of GUIDs, required; 'original_primary_publishing_organisation' is an array of GUIDs, maxItems 1, required; 'parent' is an array of GUIDs, maxItems 1, required; 'policy_areas' is an array of GUIDs, required; 'primary_publishing_organisation' is an array of GUIDs, maxItems 1, required; 'sections' is an array of GUIDs, required; 'suggested_ordered_related_items' is an array of GUIDs, required; 'taxons' is an array of GUIDs, required; 'topics' is an array of GUIDs, required. 'previous_version' is a string, optional. No additional properties are allowed at the root level.
