The schema describes the structure for MAC details. It is an object type. The main properties of the schema are: 'mac_details' is an object that contains MAC address information. It is required and has the following properties: 'mac_address' is a string that must match the pattern '^\s*([A-Fa-f0-9]{2}(:{1}[A-Fa-f0-9]{2}){5})\s*$' and is required. No additional properties are allowed at the root level.
