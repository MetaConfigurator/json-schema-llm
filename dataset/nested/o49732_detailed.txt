The schema describes Twitter account settings. It is an object type. The main properties of the schema are: 'allow_contributor_request' is a string optional; 'always_use_https' is a boolean optional; 'discoverable_by_email' is a boolean optional; 'geo_enabled' is a boolean optional; 'language' is a string optional; 'protected' is a boolean optional; 'screen_name' is a string optional; 'show_all_inline_media' is a boolean optional; 'sleep_time' is an object optional, with properties: 'enabled' (boolean), 'end_time' (date-time string), and 'start_time' (date-time string); 'time_zone' is an object optional, with properties: 'name' (string), 'tzinfo_name' (string), and 'utc_offset' (integer); 'trend_location' is an array optional, with items of type object containing properties: 'country' (string), 'countryCode' (string), 'name' (string), 'parentid' (integer), 'placeType' (object with properties: 'code' (integer) and 'name' (string)), 'url' (string), and 'woeid' (integer); 'use_cookie_personalization' is a boolean optional. No additional properties are allowed at the root level.
