The schema describes a domain profile document. It is an object type. The main properties of the schema are: 'schemaType' is a string Schema type, required. 'documentID' is a string Internal document GUID, required. '_corpusID' is a string Internal Corpus GUID, required. 'domaine1' is a string 1er domaine prédominant du texte, optional. 'confiance1' is a number Niveau de confiance du 1er domaine prédominant du texte, optional. 'domaine2' is a string 2e domaine prédominant du texte, optional. 'confiance2' is a number Niveau de confiance du 2e domaine prédominant du texte, optional. 'domaine3' is a string 3e domaine prédominant du texte, optional. 'confiance3' is a number Niveau de confiance du 3e domaine prédominant du texte, optional. No additional properties are allowed at the root level.
