The schema describes mapping from the Vulnerability Rating Taxonomy to CWE. It is an object type. It defines several reusable components under 'definitions': MappingMetadata is an object, it requires a default field of null type. VRTid is a string, it follows the pattern '^[a-z_]*$'. CWE is an array or null, it contains items of string type that must match the pattern '^CWE-[0-9]*$'. It requires a minimum of 1 item and does not allow duplicate items. Mapping is an object, it requires id and cwe fields, where id is a reference to VRTid and cwe is a reference to CWE. MappingParent is an object, it requires id and children fields, where id is a reference to VRTid, children is an array that can contain either MappingParent or Mapping, and cwe is a reference to CWE. The main properties of the schema are: metadata is a required property and references MappingMetadata; content is a required property and is an array that can include either MappingParent or Mapping. No additional properties are allowed at the root level.
