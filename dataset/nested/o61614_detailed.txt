The schema describes a configuration profile for a home screen layout. It is an object type. The main properties of the schema are: 'Dock' is an array optional. It contains dictionaries that each must conform to the item format and can include the following properties: 'BundleID' is a string required if the item type is 'Application'; 'DisplayName' is a string; 'Pages' is an array optional; 'Type' is a string and must be one of the following values: 'Application', 'Folder'. 'Pages' is an array optional. It contains dictionaries that each must conform to the item format and has the same properties as 'Dock'. 'PayloadDescription' is a string with a default value of 'Home Screen Layout'; 'PayloadDisplayName' is a string with a default value of 'Home Screen Layout'; 'PayloadIdentifier' is a string with a default value of 'com.apple.homescreenlayout'; 'PayloadOrganization' is a string; 'PayloadType' is a string with a default value of 'com.apple.homescreenlayout'; 'PayloadUUID' is a string with a default value of '' and must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'; 'PayloadVersion' is a number with a default value of 1. No additional properties are allowed at the root level.
