The schema describes Deployment of the Hotels Micro Services. It is an object type. The main properties of the schema are: 'selected-region' is a string that represents the Bluemix region, required. 'selected-organization' is a string that represents the Bluemix org, required. 'selected-space' is a string that represents the Bluemix space, required. 'ui-app-name' is a string that represents the name of the UI, required. 'dealsQ-app-name' is a string that represents the name of the deals Query app, required. 'hotelQ-app-name' is a string that represents the name of the hotels Query App, required. 'locationQ-app-name' is a string that represents the name of the hotels Query App, required. 'controller-app-name' is a string that represents the name of the hotels Query App, required. 'openwhisk_auth' is a string for OpenWhisk authorization key, optional. 'apic-username' is a string that represents APIC username, optional. 'apic-password' is a string that represents APIC password, optional. 'api-app-name' is a string that represents API cf app name, optional. No additional properties are allowed at the root level.
