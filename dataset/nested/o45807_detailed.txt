The schema describes a set of departure functions for CoolProp. It is an array type. The main properties of the schema are: [items] is an object that describes a departure function. It has required fields: 'n', 'd', 't', 'type', 'Name', 'BibTeX', and 'aliases'. 'BibTeX' is a string that holds the BibTeX key for the reference. 'Name' is a string that denotes the name of the binary pair, which must be unique. 'type' is a string that specifies the type of the binary pair reducing function and can take the values 'GERG-2008' or 'Exponential'. 'n', 'd', 't', 'l', 'eta', 'epsilon', 'beta', and 'gamma' are all arrays of numbers with a minimum of 0 items. 'aliases' is an array of strings with a minimum of 0 items and items must be unique. No additional properties are allowed at the root level.
