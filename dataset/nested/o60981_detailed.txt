The schema describes the result of a Github worker. It is an object type. It defines several reusable components under 'definitions': 'github_extracted_details' is an object, which contains properties for Github inspection details including 'forks_count', 'last_year_commits', 'open_issues_count', 'stargazers_count', 'subscribers_count', and 'contributors_count', with 'last_year_commits' being a reference to another definition. It also has a 'topics' array of strings. 'github_last_year_commits_details' is an object, which contains properties 'sum' and 'weekly', with 'sum' being an integer and 'weekly' being an array of integers, both required. 'summary' is a required array of strings.
