The schema describes a stream of examples sent to the VW web service, which will submit them to VW. It is an array type. The main properties of the schema are: [items] is an object representing an individual example in the stream. It contains optional fields: 'label', which is a string that can be null/empty and can contain anything with a minimum length of 0; 'tag', which is a string that can also be null/empty with a minimum length of 0; and 'namespaces', which is an array of objects representing the list of namespaces in the example. Each item in 'namespaces' is an object with optional properties: 'name', which is a string that can be null/empty but must not contain whitespace, | or :, with a minimum length of 0; 'scale', which is a number representing the scaling factor, optionally defaults to 1 if not present (the 'name' must be present for this to be accepted); and 'features', which is an array of objects representing the list of features in this namespace. Each item in 'features' is an object with a required property: 'name', which is a string that must be present with a minimum length of 1, and an optional property: 'value', which is a number that defaults to 1 if not present. No additional properties are allowed at the root level.
