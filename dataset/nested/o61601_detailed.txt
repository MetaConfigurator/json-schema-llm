The schema describes a configuration profile for ShareKit. It is an object type. The main properties of the schema are: 'PayloadDescription' is a string that describes the payload. It is optional and has a default value of 'ShareKit Payload'. 'PayloadDisplayName' is a string that represents the name of the payload. It is optional and has a default value of 'ShareKit Payload'. 'PayloadIdentifier' is a string that serves as a unique dot-delimited identifier for the payload. It is optional and has a default value of 'com.apple.com.apple.ShareKitHelper'. 'PayloadOrganization' is a string that describes the issuing organization of the profile as displayed to the user. It is optional. 'PayloadType' is a string indicating the type of the payload as a reverse DNS string. It is optional and has a default value of 'com.apple.com.apple.ShareKitHelper'. 'PayloadUUID' is a string that provides a unique identifier for the payload, formatted as a UUID. It is optional and has a default value of an empty string. It has a pattern constraint for validation. 'PayloadVersion' is a number indicating the version of the whole configuration profile. It is optional and has a default value of 1. 'SHKAllowedShareServices' is an array of strings that represents a list of plugin IDs permitted to show up in the user’s Share menu. This array is optional. 'SHKDeniedShareServices' is an array of strings that represents a list of plugin IDs that will not show up in the user’s Share menu. It is optional and applies only if 'SHKAllowedShareServices' is absent. No additional properties are allowed at the root level.
