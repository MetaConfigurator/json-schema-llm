The schema describes an invocation manifest for ROI to NIfTI. It is an object type. The main properties of the schema are: 'config' is an object representing configuration settings that is required. It has three required boolean properties: 'save_binary_masks' with a default value of true, 'save_combined_output' with a default value of false, and 'save_slicer_color_table' with a default value of false. 'inputs' is an object representing input specifications that is required. It has two required properties: 'api-key' which is an object, and 'Input_File' which is an object containing a required 'type' property that has an enum with allowed values: 'nifti'. No additional properties are allowed at the root level.
