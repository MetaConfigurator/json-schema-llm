The schema describes o2r metadata for ui widgets. It is an object type. It defines several reusable components under 'definitions': 'radiobuttons' is an array, containing objects with properties 'label' (a string), 'value' (a number), 'is_checked' (a boolean), 'parameter_type' (a string), and 'reference_point' (a string). All properties are required. 'checkboxes' is an array, containing objects with properties 'label' (a string), 'value' (a number), 'is_checked' (a boolean), 'parameter_type' (a string), and 'reference_point' (a string). All properties are required. 'sliders' is an array, containing objects with properties 'label' (a string), 'value' (a number), 'value_min' (a number), 'value_max' (a number), 'parameter_type' (a string), and 'reference_point' (a string). All properties are required. The root schema resolves to the referenced definition for 'radiobuttons', 'checkboxes', and 'sliders'. The main properties of the schema are: 'radiobuttons' is an array referencing the 'radiobuttons' definition. 'checkboxes' is an array referencing the 'checkboxes' definition. 'sliders' is an array referencing the 'sliders' definition. No additional properties are allowed at the root level.
