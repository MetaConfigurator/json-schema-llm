The schema describes a pet. It is an object type. The main properties of the schema are: 'category' is an object that contains properties 'id' as an integer with format int64 and 'name' as a string. 'id' is an integer with format int64. 'name' is a string with an example value of 'doggie'. 'photoUrls' is an array of strings. 'status' is a string describing pet status in the store with allowed values: 'available', 'pending', 'sold'. 'tags' is an array of objects, each containing properties 'id' as an integer with format int64 and 'name' as a string. The properties 'name' and 'photoUrls' are required. No additional properties are allowed at the root level.
