The schema describes a context provided to a custom user metric JSONata filter expression. It is an object type. It defines several reusable components under 'definitions': 'eventContext' is an object, describing the variables for the event that triggered the metric. It requires properties 'key' which is a string, describing the key of the evaluated event (e.g. purchase), and 'dateTriggered' which is an integer, describing the date this event was created. It also has a property 'fields' which references 'eventFieldsContext'. 'eventFieldsContext' is an object, describing the event's fields, allowing additional properties. 'customFieldsContext' is an object, describing the user's custom fields, also allowing additional properties.
