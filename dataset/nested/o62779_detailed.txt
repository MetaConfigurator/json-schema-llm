The schema describes an Example molecule. It is an object type. The main properties of the schema are: 'modifier' is a string that can be one of the values: '', 'blue'. 'title' is a string with a minimum length of 3 and a maximum length of 100, and is required. 'links' is an array of objects with a minimum of 1 item, where each item has required properties 'uri', which is a string with a pattern of '^[a-z]*$', and 'text', which is a string. 'buttons' is a boolean that can optionally be true to show bottom buttons for client rendering functionality. No additional properties are allowed at the root level.
