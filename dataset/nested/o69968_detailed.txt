The schema describes the structure for item types. It is an object type. The main properties of the schema are: '$schema' is a string that specifies the schema to validate item_type records against, with a minimum length of 9 and a default value of 'https://ils.rero.ch/schemas/item_types/item_type-v0.0.1.json'. 'pid' is a string representing the ItemType ID, with a minimum length of 1. 'name' is a string for the name of the ItemType, with a minimum length of 1, and it includes a validation rule to check that the value does not already exist; an error message is provided if it does. 'description' is a string providing a description of the ItemType, with a minimum length of 1. 'type' is a string defining the type of the circulation category with a default value of 'standard' and allowed values being 'online' and 'standard'; it includes validation to ensure that the value does not already exist, with a specific error message for online types. 'organisation' is an object representing the organisation, which includes its URI as a string. No additional properties are allowed at the root level.
