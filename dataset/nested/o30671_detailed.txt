The schema describes a configuration for network device properties. It is an object type. The main properties of the schema are: 'channel' is an integer and is readonly. 'controlplane' is a boolean and is readonly. 'flags' is an integer and is readonly. 'ifnum' is a string and is optional. 'mac' is a string and is optional. 'neighbor' is a string and is optional. 'nodeid' is an integer and is optional. 'state' is a string with an enum of allowed values: 'INCOMPLETE', 'REACHABLE', 'STALE', 'DELAY', 'PROBE', and is readonly. 'td' is an integer and is optional. 'timeout' is an integer and is readonly. 'vlan' is an integer and is optional. 'vtep' is a string and is optional. 'vxlan' is an integer and is optional. No additional properties are allowed at the root level.
