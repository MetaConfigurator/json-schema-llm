The schema describes fabric information. It is an object type. The main properties of the schema are: 'fabric_fq_name' is an array of strings, required. 'role_assignments' is an array of objects, optional, with a minimum of 0 items, and describes fabric device role assignments. The objects within 'role_assignments' contain: 'device_fq_name' which is an array of strings, optional; 'physical_role' which is a string with allowed values 'leaf', 'spine', 'pnf', required; and 'routing_switching_roles' which is an array of strings with allowed values 'CRB', optional. No additional properties are allowed at the root level.
