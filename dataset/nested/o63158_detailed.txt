The schema describes a cmap-js configuration. It is an object type. It defines several reusable components under 'definitions': 'filter' is an object, it requires the fields 'column', 'operator', and 'value'. The property 'column' is a string, 'operator' is an enum with allowed values: 'equals', 'regex', and 'value' is a string. The 'not' property is a boolean, which is optional. 'linkout' is an object, it requires the field 'url' and conditionally requires either 'featuretype' or 'featuretypePattern'. The 'isLinkingService' property is a boolean with a default value of false, and 'text' and 'url' are strings. The main properties of the schema are: '#' is a string, optional. 'attribution' is a string, optional. 'header' is a string, required. 'initialView' is an array, required, containing objects that require 'source' and 'map'. 'sources' is an array, required, containing objects that require 'url', 'method', and 'id'. No additional properties are allowed at the root level.
