The schema describes a Node Profile Import. It is an object type. The main properties of the schema are: 'fabric_fq_name' is an array containing strings that is optional. 'encoded_file' is a string providing the Encoded File to Import, and is optional. 'file_format' is a string describing the format of the encoded input file, which is optional and must be one of the following values: 'yaml', 'json', 'YAML', 'JSON'. 'contrail_command_host' is a string providing the Contrail Command IP, and is optional. No additional properties are allowed at the root level.
