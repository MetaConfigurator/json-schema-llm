The schema describes an HTTP callout policy. It is an object type. The main properties of the schema are: 'bodyexpr' is a string optional field. 'cacheforsecs' is an integer optional field. 'fullreqexpr' is a string optional field. 'headers' is an object optional field. 'hits' is an integer readonly field. 'hostexpr' is a string optional field. 'httpmethod' is a string optional field with an enum of allowed values: 'GET', 'POST'. 'ipaddress' is a string optional field. 'name' is a string optional field. 'parameters' is an object optional field. 'port' is an integer optional field. 'recursivecallout' is an integer readonly field. 'resultexpr' is a string optional field. 'returntype' is a string optional field with an enum of allowed values: 'BOOL', 'NUM', 'TEXT'. 'scheme' is a string optional field with an enum of allowed values: 'http', 'https'. 'svrstate' is a string readonly field with an enum of allowed values: 'UP', 'DOWN', 'UNKNOWN', 'BUSY', 'OUT OF SERVICE', 'GOING OUT OF SERVICE', 'DOWN WHEN GOING OUT OF SERVICE', 'NS_EMPTY_STR', 'Unknown', 'DISABLED'. 'undefhits' is an integer readonly field. 'undefreason' is a string readonly field with an enum of allowed values: 'Failed to add service', 'Vserver not found', 'Not a HTTP or SSL vserver', 'Generated callout request is invalid', 'Content-Length header not found in callout request', 'Not enough space to put Content-Length value', 'Config incomplete', 'Server is DOWN', 'Creating callout connection failed', 'No memory to generate callout request packets', 'No memory to create callout task', 'No memory to create callout async', 'Callout request expression undef', 'No callout response expression', 'Skipped callout response eval', 'Callout response pixl init undef', 'Callout response expression undef'. 'urlstemexpr' is a string optional field. 'vserver' is a string optional field. No additional properties are allowed at the root level.
