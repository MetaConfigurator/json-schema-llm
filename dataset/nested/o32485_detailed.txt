The schema describes a role object. It is an object type. The main properties of the schema are: 'translationKey' is a string that describes the role translationKey. 'name' is a string that describes the role name. 'permission' is an object that includes several boolean properties: 'open' (optional), 'copy' (optional), 'delete' (optional), 'export' (optional), 'edit' (optional), and 'administrate' (optional), each describing associated role permissions. No additional properties are allowed at the root level.
