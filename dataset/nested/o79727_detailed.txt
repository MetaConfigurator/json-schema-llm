The schema describes info files for v3 elements. It is an object type. The main properties of the schema are: 'controller' is a string that is required and describes the name of the element's controller file. 'dependencies' is an object that is optional and describes the element's client-side dependencies. It defines several properties: 'coreStyles' is an array that contains items that are strings describing .css files located in /public/stylesheets. 'coreScripts' is an array that contains items that are strings describing .js files located in /public/javascripts. 'nodeModulesStyles' is an array that contains items that are strings describing .css files located in /node_modules. 'nodeModulesScripts' is an array that contains items that are strings describing .js files located in /node_modules. 'clientFilesCourseStyles' is an array that contains items that are strings describing .css files located in clientFilesCourse. 'clientFilesCourseScripts' is an array that contains items that are strings describing .js files located in clientFilesCourse. 'elementStyles' is an array that contains items that are strings describing .css files located in the element's directory. 'elementScripts' is an array that contains items that are strings describing .js files located in the element's directory. No additional properties are allowed at the root level.
