The schema describes a user profile. It is an object type. The main properties of the schema are: 'email' is a string formatted as an email with a maximum length of 1024 characters. 'firstName' is a string with a minimum length of 1 and a maximum length of 1024 characters. 'lastName' is a string with a minimum length of 1 and a maximum length of 1024 characters. 'companyName' is a string with a maximum length of 1024 characters. 'phoneNumber' is a string with a maximum length of 1024 characters. 'location' is a string with a maximum length of 1024 characters. 'url' is a string with a maximum length of 1024 characters. 'password' is a string with a minimum length of 8 and a maximum length of 2048 characters. No additional properties are allowed at the root level.
