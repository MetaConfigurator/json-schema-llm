The schema describes a container of instances. It is an object type. The main properties of the schema are: 'schema' is a string required field that defines the schema associated with this document and must be 'avalon-core:subset-3.0'. 'type' is a string required field that indicates the type of document and must be 'subset'. 'parent' is a string optional field that serves as a unique identifier to the parent document. 'name' is a string required field that denotes the name of the directory and must match the pattern '^[a-zA-Z0-9_.]*$'. 'data' is an object required field that contains document metadata. It includes a required array property 'families', which specifies one or more families associated with this subset. No additional properties are allowed at the root level.
