The schema describes a list of menu items. It is an array type. The main properties of the schema are: Each item is an object that contains the following properties: 'text' is a string and is required. 'id' is a string and is required. 'group_by' is a string and is optional. 'query_group' is a string and is optional. 'category' is a string and is optional. 'realm' is a string and is optional. 'defaultChartSettings' is a string and is optional. 'chartSettings' is a string and is optional. 'node_type' is a string and is required. 'iconCls' is a string and is required. 'description' is a string and is optional. 'leaf' is a boolean and is required. 'disabled' is a boolean and is optional. No additional properties are allowed at the root level.
