The schema describes a structured object for a stacked requisition pack. It is an object type. It defines several reusable components under 'definitions': 'StackedRequisitionPack' is a recursive reference type that allows for nested requisition packs. The main properties of the schema are: 'name' is a string that is required. 'description' is a string that is required. 'largeImageUrl' is a string that is required. 'isStack' is a boolean that is required. 'isFeatured' is a boolean that is required. 'isGiftOnly' is a boolean that is required. 'isNew' is a boolean that is required. 'creditPrice' is an integer that is required. 'isPurchasableWithCredits' is a boolean that is required. 'isPurchasableFromMarketplace' is a boolean that is required. 'xboxMarketplaceProductId' is a string or null that is required. 'xboxMarketplaceProductUrl' is a string or null that is required. 'merchandisingOrder' is an integer that is required. 'flair' is a string or null that is required. 'giftableAcquisitionMethods' is an integer that is required. 'stackedRequisitionPacks' is an array of 'StackedRequisitionPack' items that is required. 'id' is a string that is required. 'contentId' is a string that is required. No additional properties are allowed at the root level.
