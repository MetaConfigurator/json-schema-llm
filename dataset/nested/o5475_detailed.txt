The schema describes the structure of an object representing a game item. It is an object type. The main properties of the schema are: 'supportedGameModes' is an array of strings representing the game modes supported by the item. It is required and does not allow additional items. 'name' is a string representing the name of the item. It is required. 'description' is a string providing a description of the item. It is required. 'rarityType' is a string indicating the type of rarity associated with the item. It is required. 'rarity' is a string representing the rarity of the item. It is required. 'isMythic' is a boolean indicating whether the item is mythic. It is required. 'isCertification' is a boolean indicating if the item requires certification. It is required. 'isWearable' is a boolean specifying whether the item is wearable. It is required. 'hideIfNotAcquired' is a boolean that determines if the item should be hidden if not acquired. It is required. 'useType' is a string describing the usage type of the item. It is required. 'largeImageUrl' is a string providing a URL for a large image of the item. It is required. 'categoryName' is a string representing the category name of the item. It is required. 'internalCategoryName' is a string indicating the internal category name of the item. It is required. 'subcategoryName' is a string for the name of the subcategory of the item. It is required. 'internalSubcategoryName' is a string representing the internal subcategory name. It is required. 'subcategoryOrder' is an integer indicating the order of the subcategory. It is required. 'sellPrice' is an integer representing the selling price of the item. It is required. 'levelRequirement' is an integer indicating the level requirement for the item. It is required. 'certificationRequisitionId' is a string or null indicating the certification requisition ID. It is required. 'id' is a string providing a unique identifier for the item. It is required. 'contentId' is a string indicating the content ID of the item. It is required.
