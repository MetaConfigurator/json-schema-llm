The schema describes an invocation manifest for DCMTK: DICOM Send - storescu. It is an object type. The main properties of the schema are: 'inputs' is an object required with the property 'api_key' which is also an object. 'file' is an optional property of 'inputs', and is an object that contains a property 'type' with an enum value of 'dicom'. 'config' is an object required with the properties: 'destination' which is a string, 'called_ae' which is a string, 'calling_ae' which is a string with a default value of 'flywheel', and 'port' which is a number with a default value of '104'. No additional properties are allowed at the root level.
