The schema describes an edge deployment configuration. It is an object type. The main properties of the schema are: 'items' is an array containing the deployments, where each item is an object with properties such as 'id', 'edgeDeploymentId', 'applicationId', 'deviceId', 'deviceName', 'flowId', 'flowName', 'creationDate', 'lastUpdated', 'currentVersion', 'desiredVersion', and 'logs'. 'count' is an integer representing the number of items. 'totalCount' is an integer representing the total number of deployments. 'applicationId' is a string matching the pattern '^[A-Fa-f\d]{24}$'. 'deviceId' is a string matching the pattern '^[A-Fa-f\d]{24}$'. 'flowId' is a string matching the pattern '^[A-Fa-f\d]{24}$'. 'version' is a string with a length between 1 and 255 characters. No additional properties are allowed at the root level.
