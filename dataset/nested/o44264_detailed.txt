The schema describes an instance of a hotspot animation. It is an object type. The main properties of the schema are: 'autostart' is a boolean that indicates whether the animation autostarts, optional with an example value of false. 'enabled' is a boolean that indicates whether the animation is enabled, optional with an example value of true. 'loop' is a boolean that indicates whether the animation loops, optional with an example value of false. 'order' is an array of strings that represents the list of track UIDs to execute, ordered as written, optional with an example value of ['track-0', 'track-2']. 'random' is a boolean that indicates whether the tracks are played randomly, optional with an example value of false. No additional properties are allowed at the root level.
