The schema describes the game state for the PlanetWars game. It is an object type. The main properties of the schema are: 'expeditions' is an array that lists all expeditions in this game. These are the ships on the way to another planet, and it is required. Each expedition item is an object with properties: 'destination' which is a string, 'id' which is an integer, 'origin' which is a string, 'owner' which is a string, 'ship_count' which is an integer with a minimum of 0, and 'turns_remaining' which is an integer with a minimum of 0. 'planets' is an array that lists all planets in the game, and it is required. Each planet item is an object with properties: 'name' which is a string, 'owner' which is a string, 'ship_count' which is an integer with a minimum of 0, 'x' which is an integer, and 'y' which is an integer. 'players' is an array that lists all players in the game, it is required and must have at least 2 unique string items. No additional properties are allowed at the root level.
