The schema describes the configuration for DICOM Send using DCMTK. It is an object type. The main properties of the schema are: 'inputs' is an object containing required property 'file', which is also an object with a property 'type' that must be strictly 'dicom'. 'config' is an object with required properties 'destination', 'called_ae', 'calling_ae', and 'port'. 'destination' is a string, 'called_ae' is a string, 'calling_ae' is a string with a default value of 'flywheel', and 'port' is a number with a default value of 104. No additional properties are allowed at the root level.
