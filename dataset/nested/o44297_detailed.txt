The schema describes the configuration for a media source object level. It is an object type. The main properties of the schema are: 'device' is an object the device requirements. 'height' is a number the height of the source. It will be used with the width and tile to compute the value of the {x} and {y} patterns. 'name' is a string if set, specifies the {level} pattern for this level. If not, the value will be the index of the level in the levels array. 'pattern' is a string the pattern to respect in case of a multi tiles media. Four parameters are authorized: {face}, {level}, {x}, and {y}. 'tile' is a number the size of a tile, which corresponds to the width and the height of a tile. It will be used with the width and height to compute the value of the {x} and {y} patterns. 'url' is a string if set, overrides any computed URL by this one. If not, the URL will be computed from other values. 'width' is a number the width of the source. It will be used with the height and tile to compute the value of the {x} and {y} patterns. No additional properties are allowed at the root level.
