The schema describes configuration settings. It is an object type. The main properties of the schema are: '_id' is a string with a minimum length of 1 and a maximum length of 36, required. 'authentication' is an object, optional, that contains properties defining authentication methods without allowing additional properties. 'charts' is an object, optional, containing a 'repo_url' property without allowing additional properties. 'deleted' is a string, optional. 'hostname' is a string, optional. 'mail' is an object, optional, with required properties 'server' and 'no_reply_address', and it does not allow additional properties. The 'mail' object includes an 'authentication' object with required properties 'username' and 'password', and it does not allow additional properties. 'port' is an integer with a default value of 587. 'ssl' is a boolean with a default value of false. 'metadata' is an object, optional, containing properties that may include 'resourceVersion'. 'schema' is a string with an enum value of 'http://elasticbox.net/schemas/settings', required. 'schema_version' is an integer, optional. No additional properties are allowed at the root level.
