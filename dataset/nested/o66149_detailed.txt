The schema describes information from the CADI database. It is an object type. The main properties of the schema are: 'status' is a string that represents the status. It is optional. 'publication_status' is a string that represents the publication status. It is optional. 'pas' is a string that represents the PAS. It is optional. 'description' is a string that provides a description. It is optional. 'created' is a string that shows when the entry was created. It is optional. 'contact' is a string that indicates the contact person. It is optional. 'paper' is a string that refers to the paper. It is optional. 'twiki' is a string that links to Twiki. It is optional. 'name' is a string that gives the name. It is optional. No additional properties are allowed at the root level.
