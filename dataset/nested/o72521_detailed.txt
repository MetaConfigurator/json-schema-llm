The schema describes Microsoft Automation Resource Types. It is an object type. It defines several reusable components under 'definitions': 'automationAccounts' is an object, describing an automation account. It has required fields 'properties' and 'name'. The 'properties' field is also required and contains an object with 'sku', which is also required and contains a 'name' (with enum values 'Free', 'Basic' as the allowed choices') and optional fields 'family' (a string) and 'capacity' (an integer). 'runbooks' is an object, describing a runbook. It has required fields 'name' and 'properties'. The 'properties' field contains required fields 'runbookType' (with enum values 'PowerShellWorkflow', 'PowerShell', 'Graph') and 'publishContentLink' (which includes 'uri' as a required string). The main properties of the schema are: 'automationAccounts' is a reference to the 'automationAccounts' definition, required. 'runbooks' is a reference to the 'runbooks' definition, required. No additional properties are allowed at the root level.
