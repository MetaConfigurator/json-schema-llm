The schema describes a data structure for returning information in a customs context. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'listaRetorno' is an array of objects that contain properties including 'numeroRUC' (a string), 'numeroDUE' (a string), 'ocorreuDesembaracoOuAutorizacaoEmbarqueAntecipado' (a boolean), 'existeImpedimentoEmbarque' (a boolean), 'indicadorSeCargaRUCMaster' (a boolean), 'numeroRUCMasterDaCarga' (a string), 'conteineres' (an array of strings), 'documentosDeTransporte' (an array of objects with properties 'numeroDocumento' (a string), 'tipoDocumento' (a string), 'identificacaoVeiculo' (a string)), 'listaCargasSoltasVeiculos' (an array of objects with properties 'tipoEmbalagem' (an integer) and 'quantidade' (an integer)), and 'listaGraneis' (an array of objects with properties 'tipoGranel' (an integer), 'sgUnidadeMedida' (a string with enum values 'kg', 'm3'), and 'quantidade' (a number)). 'listaMensagem' is an array of objects that contain properties including 'codigo' (a string) and 'mensagem' (a string). No additional properties are allowed at the root level.
