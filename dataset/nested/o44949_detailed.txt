The schema describes a structure for character resources. It is an object type. The main properties of the schema are: 'cr' is an array of objects, required, with unique items. Each object must contain the following required properties: '_cr' (a string), 'pb' (an integer), 'ac' (an integer), 'hpmin' (an integer), 'hpmax' (an integer), 'attackbonus' (an integer), 'dprmin' (an integer), 'dprmax' (an integer), and 'savedc' (an integer). No additional properties are allowed at the root level.
