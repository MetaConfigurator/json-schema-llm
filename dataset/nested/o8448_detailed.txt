The schema describes a structure for gathering multicast statistics. It is an object type. The main properties of the schema are: 'warning_banner' is an object that provides details about the multicast statistics process and is optional. It includes nested properties: 'warning_banner_2' is an object providing assumptions about traffic; 'warning_banner_3' is an object assuming traffic originates from outside the fabric; 'warning_banner_4' is an object indicating traffic goes to devices attached to server leaves; 'warning_banner_5' is an object stating PIM is enabled in the fabric. 'source' is a string that represents the source, required, with a default value of an empty string. 'group' is a string that denotes the group, required, with a default value of an empty string. No additional properties are allowed at the root level.
