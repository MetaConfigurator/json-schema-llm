The schema describes an image atom. It is an object type. The main properties of the schema are: 'modifier' is a string which is optional and can be one of the following values: '', 'fit-parent', or '1x1'. 'alt' is a string which is required. 'srcset' is an object which is required and contains a property 'renditions', which is an array of objects. Each object in the 'renditions' array must contain the following required properties: 'media' which is a string and can be one of the allowed values: '100w', '180w', '230w', '290w', '320w', '360w', '460w', '480w', '580w', '640w', '760w', '960w', '1200w', '1280w', '1496w', '1920w', '1960w', '2880w', '3920w', and 'src' which is a string. 'customRenditions' is an object which is optional and contains a required property 'fallbackImage' which is a string representing the src to the fallback image to be rendered initially. No additional properties are allowed at the root level.
