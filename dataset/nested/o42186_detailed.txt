The schema describes Funjack's haptic script format. It is an object type. The main properties of the schema are: 'version' is a string specifying the Funscript version, optional; 'range' is an integer representing the range of moment to use in percent, optional, with a minimum of 0 and a maximum of 100; 'inverted' is a boolean indicating whether to invert position values (0=100, 100=0), optional; 'actions' is an array of actions, which is required. Each action is an object that contains: 'pos' which is an integer representing the position in percent, required, with a minimum of 0 and a maximum of 100; 'at' which is an integer denoting the time to be at position in milliseconds, required, with a minimum of 0. No additional properties are allowed at the root level.
