The schema describes a configuration for a chart widget. It is an object type. The main properties of the schema are: 'title' is an object optional, containing 'htmlLabel' which is a string and describes the title for the chart. 'timeSeries' is an object required, providing the data to display in another widget with a required property 'resource' which is a string formatted as 'topic', describing the name of the data resource. 'chart' is an object optional, setting options for the chart representation with a property 'type' which is a string, containing an enum of allowed values: 'pieChart', 'stackedAreaChart', 'multiBarChart', with a default value of 'pieChart' and describes the type of the chart, excepted types are defined in external library NVD3. No additional properties are allowed at the root level.
