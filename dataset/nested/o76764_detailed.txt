The schema describes a social media post. It is an object type. The main properties of the schema are: 'id' is a string matching the pattern '^[0-9]+$' and is required. 'username' is a string and is required. 'time' is an integer representing the UNIX time at which the post was made with a minimum value of 1136091600 and is required. 'type' is a string with an enum of allowed values 'image' and 'video' and is required. 'isRetweet' is a boolean and is required. 'retweet' is an integer with a minimum value of 0 and is required. 'favorite' is an integer with a minimum value of 0 and is required. 'reply' is an integer with a minimum value of 0 and is required. 'text' is a string and is required. 'images' is an array of strings where each item must be a URI format and is required. No additional properties are allowed at the root level.
