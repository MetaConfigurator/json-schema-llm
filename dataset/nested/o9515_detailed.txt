The schema describes the status of export files in the WebAdMIT application. It is an object type. The main properties of the schema are: 'export_files' is an array of objects, each representing an export file's details. It is required to have the fields 'id', 'href', 'export_id', and 'status'. Each 'export_id' is an integer describing the unique identifier of the associated export. The 'href' is a string that provides a hypertext reference to this resource, following the pattern '/api/v1/exports/\d+/export_files/\d+'. The 'id' is an integer describing the unique identifier of this export file. The 'status' is a string with an enum of allowed values: 'Initializing', 'Queued', 'In Progress', 'Available', 'Success With Errors', 'Empty List', 'Failed'. The 'href' property is a string with a hypertext reference to this resource, following the pattern '/api/v1/user_identities/\d+/export_files'. No additional properties are allowed at the root level.
