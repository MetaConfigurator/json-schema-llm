The schema describes an invocation manifest for CNI: GE P-File Classifier. It is an object type. The main properties of the schema are: 'config' is an object that contains the required property 'timezone', which is a string with a default value of 'America/Los_Angeles'. 'inputs' is an object that contains the required property 'pfile', which is an object that has a property 'type' with an enum that allows the value 'pfile'. No additional properties are allowed at the root level.
