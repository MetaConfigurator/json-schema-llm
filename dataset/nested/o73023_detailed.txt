The schema describes a container consultation response. It is an object type. It defines several reusable components under 'definitions': The main properties of the schema are: 'listaRetorno' is an array of unique objects representing container details, including optional properties such as 'numeroConteiner' (a string), 'pesoBruto' (a number), 'tara' (a number), 'lacres' (an array of unique strings), 'documentosCarga' (an array of unique objects with a 'documentos' property, which is an array of unique objects containing 'numeroDUE' and 'numeroRUC', both of type string), and 'permiteMovimentacao' (a boolean). 'listaMensagem' is an array of objects representing messages, with properties 'codigo' (a string) and 'mensagem' (a string). No additional properties are allowed at the root level.
