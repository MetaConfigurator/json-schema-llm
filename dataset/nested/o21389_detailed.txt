The schema describes a content item structure. It is an object type. It defines several reusable components under 'definitions': 'guid' is a string, representing a UUID that must match the pattern '^[a-f0-9]{8}-[a-f0-9]{4}-[1-5][a-f0-9]{3}-[89ab][a-f0-9]{3}-[a-f0-9]{12}$'. 'guid_list' is an array, containing unique items that are references to 'guid'. The main properties of the schema are: 'bulk_publishing' is a boolean, optional. 'links' is an object, containing other properties that are themselves defined by references to 'guid_list'. 'previous_version' is a string, optional. No additional properties are allowed at the root level.
