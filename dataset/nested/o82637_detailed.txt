The schema describes an article block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$'. It is optional and can be appended to the DOM elements' class attribute in rendering environments that support HTML. 'updated_date' is a string formatted as 'date-time' representing an update to the node this block is attached to. It is optional. 'aside' is a boolean that, when true, indicates the block represents a portion of a document only indirectly related to the main content. It is optional. 'node_ref' is a string that must match the pattern '^[\w\/\.:-]+$'. It is required. 'title' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'cta_text' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'link_text' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'show_image' is a boolean that defaults to true. It is optional. 'image_ref' is a string that must match the pattern '^[\w\/\.:-]+$'. It is optional. No additional properties are allowed at the root level.
