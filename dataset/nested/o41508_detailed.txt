The schema describes an invocation manifest for a SciTran PAR/REC MR Classifier. It is an object type. The main properties of the schema are: 'inputs' is an object that is required and must contain 'parrec'. It has a required property 'parrec', which is an object containing a 'type' that must be 'parrec'. It also has a property 'rec', which is an object that does not have required fields but includes a 'type' that must also be 'parrec'. 'config' is an object that is required and must contain 'timezone' and 'sync'. It has a 'timezone' property, which is a string with a default value of 'UTC', and a 'sync' property, which is a boolean with a default value of true. No additional properties are allowed at the root level.
