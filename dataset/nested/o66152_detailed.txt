The schema describes a physics object configuration. It is an object type. The main properties of the schema are: 'jet_type' is a string representing the jet type with allowed values 'AK4PF', 'AK5PF', 'AK6PF', 'AK7PF', 'AK8PF', 'AK4Calo', 'AK5Calo', 'AK6Calo', 'AK7Calo', 'AK8Calo', 'KT4PF', 'KT5PF', 'KT6PF', 'KT7PF', 'KT8PF', 'KT4Calo', 'KT5Calo', 'KT6Calo', 'KT7Calo', 'KT8Calo'; 'muon_type' is a string representing the muon type with allowed values 'PFMuon', 'GlobalMuon', 'TrackerMuon'; 'photon_type' is a string representing the photon type with allowed values 'Photon', 'Conversion'; 'pt_cut' is a number representing the pt cut; 'sel_criteria' is a string representing the selection criteria with allowed values 'tight', 'medium', 'loose', 'other'; 'met_type' is a string representing the MET type with allowed values 'PFMET', 'CaloMET'; 'track_type' is a string representing the track type with allowed values 'Heavy-Ion', 'pp'; 'object' is a string representing the object with allowed values '', 'electron', 'muon', 'tau', 'jet', 'bjet', 'photon', 'track', 'vertex', 'MET', 'HT'; 'number' is an object containing 'number' as a number and 'sign' as a string with a default value of '=' and allowed values '<', '>', '=', '<=', '>='; 'eta_cut' is a number representing the eta cut; 'pt_cut_max' is a number representing the pt cut max; 'electron_type' is a string representing the electron type with the allowed value 'GsfElectron'; 'eta_cut_max' is a number representing the eta cut max; 'tau_type' is a string representing the tau type with the allowed value 'PFTau'; 'eta_cut_min' is a number representing the eta cut min; 'pt_cut_min' is a number representing the pt cut min. No additional properties are allowed at the root level.
