The schema describes an array of objects. It is an array type. The main properties of the schema are: [id] is a number, required; [icon] is a string, optional; [name] is a string, required; [description] is a string, required; [requirement] is a string, required; [locked_text] is a string, required; [type] is a string, required; [flags] is an array of strings, required; [bits] is an array of objects, optional; [tiers] is an array of objects, required; [rewards] is an array of objects, optional. Each object in [bits] must have a required [type] property, while properties in [tiers] and [rewards] are optional. No additional properties are allowed at the root level.
