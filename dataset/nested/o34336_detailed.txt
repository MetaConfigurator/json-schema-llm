The schema describes a Multi-Tenant Service Definition. It is an object type. The main properties of the schema are: 'autoscalingPolicyName' is a string that specifies the Autoscaling Policy Name, required, with a default value of 'AS1'. 'cartridgeType' is a string that specifies the Cartridge Type, required, with a default value of 'appserver'. 'clusterDomain' is a string that specifies the Cluster Domain, required, with a default value of 'appserverdev.dev.domain'. 'clusterSubDomain' is a string that specifies the Cluster Sub Domain, required, with a default value of 'subdomain2'. 'deploymentPolicyName' is a string that specifies the Deployment Policy Name, required, with a default value of 'D1'. 'isPublic' is a string that specifies whether the service is public, optional, with a default value of 'true'. 'serviceName' is a string that specifies the Service Name, required, with a default value of 'appserver'. 'tenantRange' is a string that specifies the Tenant Range, required, with a default value of '*'. No additional properties are allowed at the root level.
