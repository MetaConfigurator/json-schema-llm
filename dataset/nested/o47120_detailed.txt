The schema describes a project text entry. It is an object type. The main properties of the schema are: '_summary' is an object that contains the property '_state', which is a string. The '_summary' property is optional and does not allow additional properties. 'id' is a string that is optional. 'name' is a string that is optional. 'primarySite' is a string that is optional. 'tumourSubtype' is a string that is optional. 'tumourType' is a string that is optional. 'type' is a string that is optional. No additional properties are allowed at the root level.
