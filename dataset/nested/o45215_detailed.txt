The schema describes an array of objects, each representing an item with various properties. It is an array type. The main properties of the schema are: [name] is a string, required; [type] is a string, required; [flags] is an array of strings, required; [restrictions] is an array, required; [rarity] is a string, required; [id] is a number, required; [icon] is a string, required. No additional properties are allowed at the root level.
