The schema describes a VCMI map object template format. It is an object type. The main properties of the schema are: 'animation' is a string that is required and specifies the path to the def file with animation of this object, 'editorAnimation' is a string that is optional and specifies the path to def file with animation of this object to use in the map editor, 'visitableFrom' is an array that is required and describes directions from which this object is visible. It must contain exactly 3 items, each a string with a length of exactly 3. 'allowedTerrains' is an array that is optional and lists terrain types on which this object can be placed. 'mask' is an array that is required and describes the object mask that indicates on which tiles the object is visible, blocked, or activatable. No additional properties are allowed at the root level.
