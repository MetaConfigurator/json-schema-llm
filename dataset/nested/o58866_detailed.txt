The schema describes a configuration file for ngx-i18nsupport xliffmerge. It is an object type. The main properties of the schema are: 'xliffmergeOptions' is an object required root of configuration options. It does not require any fields and it prohibits additional properties. The 'xliffmergeOptions' object has the following properties: 'allowIdChange' is a boolean flag, whether xliffmerge should merge transunits with changed IDs. (default "false"). 'apikey' is a string API key for usage of Google Translate. 'apikeyfile' is a string file that contains API key for usage of Google Translate. 'autotranslate' is either a boolean flag indicating if new units should be automatically translated by Google Translate or an array of strings representing a list of languages. 'beautifyOutput' is a boolean when set to true, the generated xml output will be sent through a beautifier to get consistent output. (default "false"). 'defaultLanguage' is a string the native language used in your templates (default "en"). 'encoding' is a string expected encoding of xlf, xlf2 or xmb files (default "UTF-8"). 'genDir' is a string directory, where files are written to (normally identical with srcDir) (default "."). 'i18nBaseFile' is a string base name of master file (default "messages"). 'i18nFile' is a string master file (relative to srcDir) (default "messages.xlf"). 'i18nFormat' is a string defining "xlf" for XLIFF 1.2 or "xlf2" for XLIFF 2.0 or "xmb" for XML Message Bundles (default "xlf"). 'languages' is an array of strings representing a list of languages (if not specified at command line). 'ngxTranslateExtractionPattern' is a string that defines what messages are exported to json translation files for ngx-translate. 'preserveOrder' is a boolean when set to true, new trans units will be at the same position as in the master file. When set to false, they will be added at the end (default "true"). 'quiet' is a boolean that controls output (default "false"). 'removeUnusedIds' is a boolean flag indicating if unused IDs should be removed during the merge (default "true"). 'srcDir' is a string directory where the master file is expected (default "."). 'supportNgxTranslate' is a boolean flag to activate json translation files for ngx-translate (default "false"). 'targetPraefix' is a string when 'useSourceAsTarget' is set and a source is copied to target, then the target string will be prefixed by this value. 'targetSuffix' is a string when 'useSourceAsTarget' is set and a source is copied to target, then the target string will be suffixed by this value. 'useSourceAsTarget' is a boolean flag indicating if source should be copied to target for new trans-units (default "true"). 'verbose' is a boolean that controls output (default "false"). No additional properties are allowed at the root level.
