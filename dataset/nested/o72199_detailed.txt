The schema describes settings that describe the behaviour and formatting of rewards corresponding to the configured unit or unit pattern. It is an object type. The main properties of the schema are: 'id' is a string A unique identifier generated by combining reward type and unit or unit pattern (e.g. CREDIT/USD), required. 'config' is an object Behaviour and formatting supported by this unit setting, required. It defines several reusable components under 'definitions': 'rewardModifications' is an object The modifications supported by rewards applicable for this unit setting. It requires the fields 'isCancellable', 'isRedeemable', and 'isExpirable', and no additional properties are allowed. 'supportsCurrencies' is a boolean Whether this unit supports multi-currencies. false by default, optional. 'translatableConfig' is an object Locale-specific configuration for rewards of this unit, optional. It defines the property 'name' is a string The name of this reward unit, optional, and 'prettyValueFormat' is an object Format for prettyValue for rewards of this unit. It requires the field 'template' and no additional properties are allowed. 'templateType' is a string The type of this template, "ICU" by default, with an enum of ['ICU'], optional. No additional properties are allowed at the root level.
