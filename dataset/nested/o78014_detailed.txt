The schema describes configuration options for an OpenStack SDK. It is an object type. The main properties of the schema are: 'auth_type' is a string that specifies the name of the authentication plugin to be used, required, with a default value of 'password'. 'disable_vendor_agent' is an object that holds image properties required to disable vendor agent, required, with no additional properties. 'floating_ip_source' is a string that indicates which service provides Floating IPs, required, with allowed values 'neutron', 'nova', and 'None', and a default value of 'neutron'. 'image_api_use_tasks' is a boolean that indicates whether the cloud requires the Image Task API, required, with a default value of false. 'image_format' is a string defining the format for uploaded images, required, with a default value of 'qcow2'. 'interface' is a string that specifies which API interface should connections hit, required, with allowed values 'public', 'internal', and 'admin', and a default value of 'public'. 'secgroup_source' is a string that indicates which service provides security groups, required, with allowed values 'neutron', 'nova', and 'None', and a default value of 'neutron'. No additional properties are allowed at the root level.
