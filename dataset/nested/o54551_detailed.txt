The schema describes the settings for a MapChart. It is an object type. The main properties of the schema are: 'title' is a string optional field. 'description' is a string optional field. 'zoomPosition' is a string that allows the following enum values: 'topright', 'topleft', 'bottomleft', 'bottomright', 'none'. 'legendPosition' is a string that allows the following enum values: 'topright', 'topleft', 'bottomleft', 'bottomright', 'none'. 'scalePosition' is a string that allows the following enum values: 'topright', 'topleft', 'bottomleft', 'bottomright', 'none'. 'allowFilter' is a boolean optional field. 'maxItems' is an integer optional field with a minimum value of 1. 'area' is an object optional field that contains properties: 'x' is a number optional field, 'y' is a number optional field, 'zoom' is a number optional field with a minimum value of 1. No additional properties are allowed at the root level.
