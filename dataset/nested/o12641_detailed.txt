The schema describes a Call to Action (CTA). It is an object type. The main properties of the schema are: 'name' is a string, required, with an enum of allowed values: 'cta'. 'template' is a string, optional. 'type' is a string, optional, with an enum of allowed values: 'primary', 'secondary', 'disabled'. 'href' is a string, required, with a format of URL. 'title' is a string, optional. 'text' is a string, required. No additional properties are allowed at the root level.
