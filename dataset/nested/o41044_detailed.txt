The schema describes configuration of curies and default values for the fict command. It is an object type. The main properties of the schema are: 'curies' is an array containing curie definitions, each having properties such as 'contentType', 'prefix', 'startsWith', and 'suffix', which are all strings with a minimum length of 1. 'outputFormat' is a string representing the output format, which is optional, with allowed values of 'json' and 'csv'. No additional properties are allowed at the root level.
