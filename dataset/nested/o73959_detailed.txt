The schema describes an organization of menu and navigation items. It is an object type. It defines several reusable components under 'definitions': 'AppMenuItem' is an object, which requires the fields 'name', 'label', and 'links'. 'links' must be an array containing items that reference 'LinkItem'. 'IconInfo' is an object with no required fields. 'LinkItem' is an object with a required field 'rel', which must be one of the following: 'apps', 'menu', 'navtree', 'navtags', 'search', 'notification'. 'MenuItem' is an object, which requires the fields 'label' and 'icon'. 'items' must be an array containing items that reference 'MenuItem'. 'NavTag' is an object, requiring the fields 'tag', 'label', and 'labelPlural'. 'links' must be an array containing items that reference 'LinkItem'. 'NavTreeNode' is an object, which requires the fields 'id' and 'label'. 'children' must be an array containing items that reference 'NavTreeNode'. The main properties of the schema are: 'AppMenuItem' resolves to the referenced definition 'AppMenuItem', 'IconInfo' resolves to the referenced definition 'IconInfo', 'LinkItem' resolves to the referenced definition 'LinkItem', 'MenuItem' resolves to the referenced definition 'MenuItem', 'NavTag' resolves to the referenced definition 'NavTag', 'NavTreeNode' resolves to the referenced definition 'NavTreeNode'. No additional properties are allowed at the root level.
