The schema describes a structure for child term indications. It is an object type. The main properties of the schema are: 'attributes' is an object containing dynamic properties matching any string pattern. These properties have array values, each containing objects with 'value', 'units', and 'terms'. 'value' is a string with a minimum length of 1. 'units' is an optional string. 'terms' is an array of objects, each requiring a 'url', which is a string formatted as a URI. No additional properties are allowed at the root level.
