The schema describes a record structure for a site. It is an object type. The main properties of the schema are: 'title' is a string that describes the record title; it is required. 'id' is a number that describes the Invenio record identifier; it is required. 'keywords' is an array of strings that represents free text keywords; it is optional. 'publication_date' is a string formatted as date-time that indicates when the record is published; it is optional. 'contributors' is an array of objects that contains contributors in order of importance; it is required and must have at least one item. Each contributor object has the following properties: 'ids' is an array of unique items, each being an object with required 'source' and 'value' fields; 'name' is a string representing the full name of a person or organisation, and it is required; 'affiliations' is an array of unique strings that represents the affiliations for the specific record; it is optional; 'email' is a string formatted as email for contact purposes; it is optional; and 'role' is a string that can be one of the following: 'ContactPerson', 'Researcher', or 'Other'; it is optional. No additional properties are allowed at the root level.
