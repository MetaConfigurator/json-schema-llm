The schema describes a PDF Manager template. It is an object type. The main properties of the schema are: 'pdf_manager_template' is an object that contains the configuration of the PDF Manager template. It has required fields: 'href', 'id', 'name', 'list_name', and 'document_source_title'. 'href' is a string that serves as a hypertext reference to this resource and must match the pattern '/api/v1/user_identities/\d+/pdf_manager_templates/\d+'. 'id' is an integer that uniquely identifies this PDF Manager template. 'list_name' is a string that provides a human-readable name of the list used when populating this template. 'name' is a string that serves as a human-readable name of this PDF Manager template. 'document_source_title' is a string that describes the human-readable name of the document source used when populating this template. No additional properties are allowed at the root level.
