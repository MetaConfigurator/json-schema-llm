The schema describes a knitting pattern specification. It is an object type. The main properties of the schema are: 'file_url' is a string that describes the name of the file that holds the pattern and is required. 'pattern_name' is a string that describes the name of the pattern and is optional. 'dimensions' is an object that contains several properties: 'image_length' is an integer, 'image_width' is an integer, 'physical_length' is a number, 'physical_width' is a number, 'units' is a string, and 'yarn_strength' is an object that includes 'units' as a string and 'value' as a number. 'colors' is an integer that is required. No additional properties are allowed at the root level.
