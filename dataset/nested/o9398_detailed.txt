The schema describes a configuration object for resource publishing. It is an object type. The main properties of the schema are: 'title' is an object that may include a 'htmlLabel' string describing the selection title. 'data' is an object that is required and must include 'resource' and 'items'. 'resource' is a string formatted as 'topic' that specifies the ID of the resource to publish. 'items' is an array that is required, must have at least one item, and contains objects with required properties: 'title' (a string describing the data set title), an optional 'description' (a string for a short description of the data set), and 'location' (a string providing the URL for the resource). 'messages' is an object that may include 'i18nFailedLoadingResource', which is either an object or string serving as a message for error events when loading a resource, with a default value of 'Could not load resource "[resource]"'. No additional properties are allowed at the root level.
