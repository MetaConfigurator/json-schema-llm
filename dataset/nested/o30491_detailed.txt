The schema describes a binding between a GSLB virtual server and GSLB service. It is an object type. The main properties of the schema are: 'cnameentry' is a string that is readonly. 'cumulativeweight' is an integer that is readonly. 'curstate' is a string that is readonly and can have one of the following values: 'UP', 'DOWN', 'UNKNOWN', 'BUSY', 'OUT OF SERVICE', 'GOING OUT OF SERVICE', 'DOWN WHEN GOING OUT OF SERVICE', 'NS_EMPTY_STR', 'Unknown', 'DISABLED'. 'domainname' is a string. 'dynamicconfwt' is an integer that is readonly. 'gslbboundsvctype' is a string that is readonly and can have one of the following values: 'HTTP', 'FTP', 'TCP', 'UDP', 'SSL', 'SSL_BRIDGE', 'SSL_TCP', 'NNTP', 'ANY', 'SIP_UDP', 'SIP_TCP', 'SIP_SSL', 'RADIUS', 'RDP', 'RTSP', 'MYSQL', 'MSSQL', 'ORACLE'. 'gslbthreshold' is an integer that is readonly. 'ipaddress' is a string that is readonly. 'iscname' is a string that is readonly and can have one of the following values: 'ENABLED', 'DISABLED'. 'name' is a string. 'port' is an integer that is readonly. 'preferredlocation' is a string that is readonly. 'servicename' is a string. 'sitepersistcookie' is a string that is readonly. 'svcsitepersistence' is a string that is readonly and can have one of the following values: 'ConnectionProxy', 'HTTPRedirect', 'NONE'. 'svreffgslbstate' is a string that is readonly and can have one of the following values: 'UP', 'DOWN', 'UNKNOWN', 'BUSY', 'OUT OF SERVICE', 'GOING OUT OF SERVICE', 'DOWN WHEN GOING OUT OF SERVICE', 'NS_EMPTY_STR', 'Unknown', 'DISABLED'. 'thresholdvalue' is an integer that is readonly. 'weight' is an integer. No additional properties are allowed at the root level.
