The schema describes a request to create a repository. It is an object type. It defines several reusable components under 'definitions': 'CreateHistory' is an object, describing the structure of history elements. It has required fields 'version' and 'commitId', as well as an optional field 'createdAt' with a date-time format. The main properties of the schema are: 'bumper' is a string, optional, describing the bumper to be used when creating a repo; 'scmUrl' is a string, optional, describing the URL to get the repo; 'description' is a string, optional, describing the repo; 'history' is an array, optional, seeding the repo with history, where each item must conform to the 'CreateHistory' definition. No additional properties are allowed at the root level.
