The schema describes a Pinterest pin block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and can be appended to the DOM elements' class attribute. It is optional. 'updated_date' is a string formatted as 'date-time' representing an update that occurred on the node this block is attached to. It is optional. 'aside' is a boolean that, when true, means this block represents content only indirectly related to the document's main content. It is optional. 'href' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$' and is required. 'terse' is a boolean that, when true, hides the description. It is optional. 'size' is a string with a default value of 'small' that must match the pattern '^[\w-]+$'. It is optional. No additional properties are allowed at the root level.
