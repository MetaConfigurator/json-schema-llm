The schema describes configuration options for the Covalent core ng-add schematic. It is an object type. The main properties of the schema are: 'codeEditor' is a boolean indicating whether the covalent code-editor should be set up, defaulting to true. 'dynamicForms' is a boolean indicating whether covalent dynamic-forms should be set up, defaulting to true. 'echarts' is a boolean indicating whether covalent echarts should be set up, defaulting to true. 'flavoredMarkdown' is a boolean indicating whether covalent flavored markdown should be set up, defaulting to true. 'highlight' is a boolean indicating whether covalent highlight should be set up, defaulting to true. 'http' is a boolean indicating whether covalent http should be set up, defaulting to true. 'markdown' is a boolean indicating whether covalent markdown parser should be set up, defaulting to true. 'markdownNavigator' is a boolean indicating whether covalent markdown-navigator should be set up, defaulting to true. 'styleSheetColors' is a boolean indicating whether covalent colors should be added, defaulting to true. 'styleSheetFlexLayout' is a boolean indicating whether covalent flex layout should be added, defaulting to true. 'styleSheetUtilities' is a boolean indicating whether covalent utilities mixin should be added, defaulting to true. 'textEditor' is a boolean indicating whether covalent text-editor should be set up, defaulting to true. No additional properties are allowed at the root level.
