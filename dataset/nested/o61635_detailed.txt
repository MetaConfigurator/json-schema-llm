The schema describes a configuration profile for Single Sign-On accounts. It is an object type. The main properties of the schema are: 'Kerberos' is an object containing properties related to Kerberos authentication, 'Name' is a string human-readable name for the account, 'PayloadDescription' is a string describing the payload with a default value of 'Single Sign-On Account Payload', 'PayloadDisplayName' is a string with a default value of 'Single Sign-On Account Payload', 'PayloadIdentifier' is a string with a default value of 'com.apple.sso', 'PayloadOrganization' is a string describing the issuing organization, 'PayloadType' is a string with a default value of 'com.apple.sso', 'PayloadUUID' is a string representing a unique identifier for the payload with a default value of '', 'PayloadVersion' is a number representing the version of the whole configuration profile with a default value of 1. No additional properties are allowed at the root level.
