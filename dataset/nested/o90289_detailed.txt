The schema describes a raster data elevation layer. It is an object type. The main properties of the schema are: 'id' is a string that describes a unique identifying string for the layer and is required. 'layerType' is a string that describes the layer type, allowing only the value 'RasterDataElevationLayer', and is required. 'listMode' is a string that indicates whether to show or hide the elevation layer in the layer list, allowing the values 'hide' or 'show', and is optional. 'path' is a string that provides a path to an ArcGIS Runtime supported raster data file with a pattern that must match '^file:.+$', and is required. 'title' is a string for a user-friendly title for the layer, which is also required. 'visibility' is a boolean determining if the layer is initially visible in the web scene, with a default value of true and is optional. No additional properties are allowed at the root level.
