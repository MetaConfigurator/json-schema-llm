The schema describes drug information. It is an object type. The main properties of the schema are: 'atc_codes_code' is an array of strings representing ATC codes, optional, with no additional items allowed. 'atc_codes_description' is an array of strings describing ATC codes, optional, with no additional items allowed. 'atc_level5_codes' is an array of strings for level 5 ATC codes, optional, with no additional items allowed. 'drug_class' is a string indicating the drug class, optional. 'external_references_chembl' is an array of strings for external references from ChEMBL, optional, with no additional items allowed. 'external_references_drugbank' is an array of strings for external references from DrugBank, optional, with no additional items allowed. 'id' is a string representing the drug ID, optional. 'inchikey' is a string for the InChIKey, optional. 'name' is a string for the drug name, optional. 'synonyms' is an array of strings for synonyms of the drug, optional, with no additional items allowed. 'trials_conditions_name' is an array of strings for trial conditions names, optional, with no additional items allowed. 'trials_description' is an array of strings for trial descriptions, optional, with no additional items allowed. 'type' is a string for the drug type, optional. No additional properties are allowed at the root level.
