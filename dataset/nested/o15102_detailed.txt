The schema describes a data publisher. It is an object type. The main properties of the schema are: '_id' is an integer optional ID of this publisher in a network. 'federation' is a string optional name of the federation. 'host' is a string required URL of the server. 'name' is a string optional name of the instance. 'port' is an integer optional port of the server. 'type' is an string required publisher type with allowed value 'IMB'. No additional properties are allowed at the root level.
