The schema describes notification settings configuration. It is an object type. The main properties of the schema are: 'NotificationSettings' is an array of objects describing notification settings for each app. Each object includes the following properties: 'AlertType' is a number representing the type of alert for notifications for this app, with a default value of 1. 'BadgesEnabled' is a boolean that indicates whether badges are allowed for this app, with a default value of true. 'BundleIdentifier' is a string representing the bundle identifier of the target app. 'NotificationsEnabled' is a boolean that indicates whether notifications are allowed for this app, with a default value of true. 'ShowInLockScreen' is a boolean that indicates whether notifications can be shown in the lock screen, with a default value of true. 'ShowInNotificationCenter' is a boolean that indicates whether notifications can be shown in notification center, with a default value of true. 'SoundsEnabled' is a boolean that indicates whether sounds are allowed for this app, with a default value of true. 'PayloadDescription' is a string that describes the payload, with a default value of 'Configures notifications settings for apps'. 'PayloadDisplayName' is a string that indicates the name of the payload, with a default value of 'Notifications'. 'PayloadIdentifier' is a string functioning as a unique identifier for the payload, with a default value of 'com.apple.notificationsettings'. 'PayloadOrganization' is a string that describes the issuing organization of the profile, as displayed to the user. 'PayloadType' is a string indicating the type of the payload, with a default value of 'com.apple.notificationsettings'. 'PayloadUUID' is a string serving as a unique identifier for the payload with a default value of an empty string, conforming to the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is a number indicating the version of the whole configuration profile, with a default value of 1. No additional properties are allowed at the root level.
