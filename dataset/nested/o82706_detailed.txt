The schema describes a Twitter tweet block. It is an object type. The main properties of the schema are: 'aside' is a boolean that indicates if the block represents a portion of a document whose content is only indirectly related to the document's main content; it is optional. 'css_class' is a string that can be appended to the DOM elements' class attribute in rendering environments that support HTML; it is optional and must match the pattern '^[\w\s-]+$'. 'etag' is a string that must match the pattern '^[\w\.:-]+$'; it is optional. 'hide_media' is a boolean that determines if media should be hidden; it is optional. 'hide_thread' is a boolean that determines if the thread should be hidden; it is optional. 'screen_name' is a string that represents the user's handle and must match the pattern '^\w{1,15}$'; it is required. 'tweet_id' is a string that must match the pattern '^\d+$'; it is required. 'tweet_text' is a string with a minimum length of 0 and a maximum length of 5000 characters; it is optional. 'updated_date' is a string formatted as a date-time that indicates when the block's associated node was updated; it is optional. No additional properties are allowed at the root level.
