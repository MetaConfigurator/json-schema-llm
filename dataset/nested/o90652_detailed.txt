The schema describes configuration for a connect worker application. It is an object type. The main properties of the schema are: 'connect' is an object that contains settings for the connect worker. It has required fields: 'cpus', 'mem', 'instances', and 'name', each with default values and specific constraints. 'cpus' is a number representing CPU shares to allocate to each connect worker instance, with a default of 2 and a minimum of 1. 'mem' is a number indicating memory (MB) to allocate to each connect worker instance, with a default of 1024 and a minimum of 512. 'instances' is an integer for the number of instances to run, with a default of 1 and a minimum of 1. 'name' is a string for the name of this connect worker application, with a default value of 'connect'. No additional properties are allowed at the root level.
