The schema describes the structure of a sample and its associated attributes. It is an object type. It defines several reusable components under 'definitions': 'Attribute' is an object, which contains an 'attribute' field of type string describing the name of the attribute, and a 'values' field of type number describing a list of values of the attribute. The main properties of the schema are: 'Attribute' is an object referencing the 'Attribute' definition, and 'Sample' is an object referencing the 'Sample' definition. No additional properties are allowed at the root level.
