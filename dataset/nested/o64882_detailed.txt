The schema describes the creation of a new generic interface definition in the given or default project. It is an object type. The main properties of the schema are: 'lintFix' is a boolean when true, applies lint fixes after generating the interface. It is optional and has a default value of false. 'name' is a string representing the name of the interface. It is required. 'path' is a string defining the path to create the interface, relative to the workspace root. It is optional. 'prefix' is a string specifying a prefix to apply to generated selectors. It is optional and has a default value of an empty string. 'project' is a string representing the name of the project. It is optional. 'type' is a string that adds a developer-defined type to the filename, in the format "name.type.ts". It is optional. No additional properties are allowed at the root level.
