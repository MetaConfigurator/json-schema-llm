The schema describes the structure of an object containing metadata and probe information. It is an object type. The main properties of the schema are: 'metadata' is an object representing the metadata details, required fields are 'host', 'protocol', 'httpVerb', and 'host header'. 'probes' is an array of objects, each containing required fields 'seq', 'httpStatus', 'bytes', and 'rtt'. Each item in the 'probes' array follows the structure defined for probe objects. No additional properties are allowed at the root level.
