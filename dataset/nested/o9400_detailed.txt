The schema describes a structure for managing articles and selections. It is an object type. The main properties of the schema are: 'articles' is an object that displays a list of articles. It is required and contains a required property 'resource' which is a string, formatted as a topic, with an axRole of inlet, and describes the name of the data resource with articles. 'selection' is an object that allows the selection of an article. It is required and contains a required property 'resource' which is a string, formatted as a topic, with an axRole of outlet, and describes the name of the resource for the user selection. No additional properties are allowed at the root level.
