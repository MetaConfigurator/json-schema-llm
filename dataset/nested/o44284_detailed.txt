The schema describes an instance of a track. It is an object type. The main properties of the schema are: 'author' is a string that describes the author of the track, optional. 'loop' is a boolean that sets whether or not the track repeats after a script tells it to play, optional, with a default value of false. 'name' is a string that describes the name of the track, optional. 'uid' is a string that describes the UID of the track, required. 'url' is a string that describes the file URL of the track, required. No additional properties are allowed at the root level.
