The schema describes the configuration for an OpenVPN service. It is an object type. The main properties of the schema are: 'openvpn' is an object that defines settings for the OpenVPN instances. It has no required fields and allows the following properties: 'cpus' is a number with a default value of 0.01, describing CPU shares to allocate to each openvpn instance, with a minimum value of 0.01; 'framework-name' is a string with a default value of 'openvpn', describing the OpenVPN framework name; 'instances' is an integer with a default value of 1, describing the number of OpenVPN instances to run, with a minimum value of 1; 'mem' is a number with a default value of 128.0, describing memory (MB) to allocate to each OpenVPN task, with a minimum value of 64.0; 'port' is an integer with a default value of 1194, describing the UDP port to bind to, with a minimum value of 1. No additional properties are allowed at the root level.
