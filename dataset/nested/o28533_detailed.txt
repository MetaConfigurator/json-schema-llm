The schema describes a list of systems. It is an array type. The main properties of the schema are: [items] is an object that represents a system. It has required fields 'name' and 'coords'. 'name' is a string that represents the system name. 'coords' is an object that contains required fields 'x', 'y', and 'z', all of which are integers representing system coordinates. 'type' is an optional integer representing the stellar type. 'infos' is an optional string providing a custom description for the system. 'cat' is an optional array of integers representing a list of categories for the system. No additional properties are allowed at the root level.
