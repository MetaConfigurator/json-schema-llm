The schema describes OPDS Metadata. It is an object type. The main properties of the schema are: 'identifier' is a string formatted as a URI and is optional. '@type' is a string formatted as a URI and is optional. 'title' is a string, array, or object and is required. 'subtitle' is a string, array, or object and is optional. 'modified' is a string formatted as date-time and is optional. 'description' is a string and is optional. 'itemsPerPage' is an integer, exclusive minimum of 0, and is optional. 'currentPage' is an integer, exclusive minimum of 0, and is optional. 'numberOfItems' is an integer with a minimum of 0 and is optional. No additional properties are allowed at the root level.
