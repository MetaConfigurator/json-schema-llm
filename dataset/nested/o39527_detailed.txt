The schema describes a survey structure. It is an object type. The main properties of the schema are: 'title' is a string, required. 'pages' is an array, required, with items that follow a specific object structure. Each page object has properties: 'actions' is an array of strings, required, with a default value of ['continue']. 'timelimit' is either an object or null, with a default value of null and specific requirements when it is an object; it must contain 'group' (which is a string or null with a default of null), 'seconds' (a number with a default of 5), and 'timeoutnotice' (a boolean with a default of true). The page also has an 'items' array, required, containing one of several types of page items, such as 'Text Message', 'Text Input', 'Image', 'Multiple Choice', or 'Page Stopwatch', each with their own required properties and constraints. No additional properties are allowed at the root level.
