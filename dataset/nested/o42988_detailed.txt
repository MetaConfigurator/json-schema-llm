The schema describes REST web services. It is an object type. It defines several reusable components under 'definitions': 'method' is a string, HTTP method. It has an enum consisting of the values 'GET', 'POST', 'PUT', and 'DELETE', with a default value of 'GET'. 'request' is an object, definition of the parameters for the request. 'response' is an object, definition of the parameters for the response. 'service' is an object, definition of the service. It requires the fields 'method', 'request', and 'response', where 'method' references the 'method' definition, 'request' references the 'request' definition, and 'response' references the 'response' definition.
