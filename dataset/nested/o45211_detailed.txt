The schema describes an array of crafting recipes. It is an array type. The main properties of the schema are: Each item in the array is an object containing the following properties: 'type' is a string and is required. 'output_item_id' is a number and is required. 'output_item_count' is a number and is required. 'min_rating' is a number and is required. 'time_to_craft_ms' is a number and is required. 'disciplines' is an array of strings and is required. 'flags' is an array of strings and is required. 'ingredients' is an array of objects, each containing 'item_id' (a required number) and 'count' (a required number), and is required. 'id' is a number and is required. 'chat_link' is a string and is required. No additional properties are allowed at the root level.
