The schema describes a configuration for Accompli. It is an object type. The main properties of the schema are: '$extend' is a string that is optional. 'hosts' is an array that is required, containing objects that have: 'stage' as a string with a pattern of '^test|acceptance|production$', required; 'connectionType' as a string, required; 'hostname' as a string, required; 'path' as a string, required; and a 'connectionOptions' as an object with additional properties allowed. 'events' is an object that is required, containing: 'subscribers' as an array with items that can either be a string or an object, where each object must contain 'class' as a string, required; and 'listeners' as an object with additional properties allowed. 'deployment' is an object that is required, containing: 'strategy' as a string, required; and 'connection' as an object with additional properties allowed. No additional properties are allowed at the root level.
