The schema describes a collection of fluids for cubic EOS in CoolProp. It is an array type. The main properties of the schema are: [name] is a string that represents the name of the fluid and is required. [CAS] is a string that holds the CAS registry number of the fluid and is required. [Tc] is a number representing the critical temperature (K), which must be between 0.1 and 20000, and is required. [pc] is a number for the critical pressure (Pa), which must be between 0 and 500000000, and is required. [rhomolarc] is a number for the critical density (mol/m^3), which must be between 0.1 and 2000000, but is optional. [rhomolarc_units] is an enum with allowed values: 'mol/m^3', 'kg/m^3', but is optional. [acentric] is a number for the acentric factor (-), which must be between -10 and 10, and is required. [molemass] is a number for the molar mass (kg/mol), which must be between 0 and 1, and is required. [molemass_units] is an enum with the allowed value: 'kg/mol', and is required. [pc_units] is an enum with the allowed value: 'Pa', and is required. [Tc_units] is an enum with the allowed value: 'K', and is required. [aliases] is an array of strings that can have a minimum of 0 items but is optional. [alpha] is an object with properties [type], which is an enum with allowed values: 'Twu', 'Mathias-Copeman', 'default', and [c], which is an array of numbers with a minimum of 3 and a maximum of 3 items, but this is optional. No additional properties are allowed at the root level.
