The schema describes a collection of logs and operators. It is an object type. The main properties of the schema are: 'operators' is an array of objects representing log operators, required fields include 'id' (a number) and 'name' (a string). 'logs' is an array of objects representing log entries, required fields include 'description' (a string), 'key' (a string), 'maximum_merge_delay' (a number with a minimum of 0), 'operated_by' (an array of numbers), and 'url' (a string). The 'logs' array must contain at least 1 item. No additional properties are allowed at the root level.
