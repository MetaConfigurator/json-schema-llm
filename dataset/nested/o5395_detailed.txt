The schema describes an event structure. It is an object type. The main properties of the schema are: 'meta' is an object that contains required properties 'id' (a string that must match the UUID format), 'type' (a string that can only be 'EiffelTestCaseCanceledEvent'), 'version' (a string with a default value of '3.0.0' and can only be '3.0.0'), and 'time' (an integer). 'data' is an object that may contain optional properties, 'reason' (a string) and 'customData' (an array of objects containing required properties 'key' (a string) and 'value' with no additional properties allowed). 'links' is an array of objects, each requiring 'type' (a string) and 'target' (a string matching a UUID pattern), with no additional properties allowed in those objects. No additional properties are allowed at the root level.
