The schema describes the JSON schema for CodeSync settings file. It is an object type. The main properties of the schema are: 'externalPath' is a string the path to where CodeSync should export. It is required. 'autoImport' is a boolean indicating if CodeSync should auto import things when VSCode launches. It is required and has a default value of true. 'autoExport' is a boolean indicating if CodeSync should auto export things when VSCode exists. It is required and has a default value of true. 'importSettings' is a boolean indicating if CodeSync should auto import/export settings. It is required and has a default value of true. 'importKeybindings' is a boolean indicating if CodeSync should auto import/export keybindings. It is required and has a default value of true. 'importSnippets' is a boolean indicating if CodeSync should auto import/export snippets. It is required and has a default value of true. 'importExtensions' is a boolean indicating if CodeSync should auto import/export extensions. It is required and has a default value of true. 'excluded' is an object defining excluded extensions, publisher.name. It is required and has the properties: 'installed' is an array listing excluded installed extensions. 'external' is an array listing excluded external extensions. Both 'installed' and 'external' are required inside the 'excluded' object. No additional properties are allowed at the root level.
