The schema describes a PayPal amount JSON schema. It is an object type. The main properties of the schema are: 'currency' is a string that represents a 3-letter currency code. PayPal does not support all currencies. It is required and has a minimum and maximum length of 3 characters. 'total' is a string that denotes the total amount charged from the payer to the payee, or the refunded amount in case of a refund. It is required and has a maximum length of 10 characters. The 'details' property is an object that can contain 'shipping', 'subtotal', 'tax', 'handling_fee', 'insurance', and 'shipping_discount', each of which is a string. The 'details' object does not allow additional properties. No additional properties are allowed at the root level.
