The schema describes a video block configuration. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$' and describes how the css_class can be appended to the DOM elements' class attribute. It is optional. 'updated_date' is a string in date-time format that represents an update that occurred on the node this block is attached to. It is optional. 'aside' is a boolean indicating whether this block represents a portion of the document only indirectly related to the main content. It is optional. 'node_ref' is a string that must match the pattern '^[\w\/\.:-]+$'. It is required. 'title' is a string with a minimum length of 0 and a maximum length of 255, serving as an optional override for the title of the node. 'autoplay' is a boolean. It is optional. 'launch_text' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'muted' is a boolean. It is optional. 'start_at' is an integer with a default value of 0, a minimum of 0, and a maximum of 65535. It is optional. 'show_more_videos' is a boolean. It is optional. 'poster_image_ref' is a string that must match the pattern '^[\w\/\.:-]+$' and describes a reference to an image asset to use as the poster. It is optional. No additional properties are allowed at the root level.
