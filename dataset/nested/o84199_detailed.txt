The schema describes a configuration object. It is an object type. The main properties of the schema are: 'Warning!' is a string and is required. 'config' is an object and is required. The 'config' object allows additional properties which must not have any properties if they are objects themselves and must match the property name pattern '^[a-zA-Z0-9_.\-]+$'. The 'config' object must also contain a 'type' field that is required. The 'type' can either be 'select' with required 'options', or any of the following: 'integer', 'text', 'boolean', 'array', 'password', 'email', 'color', 'float', 'graph', 'dashboard-select', 'snmp3auth', 'ldap-groups', 'ad-groups' without 'options'. The 'config' object contains dependencies between properties: 'group' requires 'order' and 'section'; 'order' requires 'group' and 'section'; 'section' requires 'group' and 'order'. No additional properties are allowed at the root level.
