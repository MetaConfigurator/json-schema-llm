The schema describes a weight field configuration for a form. It is an object type. The main properties of the schema are: '_schema' is a string that defaults to 'pbj:gdbots:forms:field:weight-field:1-0-0' and must match the pattern '^pbj:([a-z0-9-]+):([a-z0-9\.-]+):([a-z0-9-]+)?:([a-z0-9-]+):([0-9]+-[0-9]+-[0-9]+)$'. 'name' is a string that is required and must match the pattern '^[a-zA-Z_]{1}[\w-]*$' and serves as a unique identifier for the field. 'maps_to' is a string that defaults to 'cf' and must match the pattern '^[a-zA-Z_]{1}\w*$'. 'label' is a string that can have a length between 0 and 255 and represents the main text for the question/field. 'placeholder' is a string that can have a length between 0 and 255. 'description' is a string that can have a length between 0 and 65535 and provides a short description for the field. 'is_required' is a boolean. 'link_text' is a string that can have a length between 0 and 255. 'link_url' is a string that must match the pattern '^(https?:\/\/)?([\da-z\.-]+)\.([a-z\.]{2,6})([\/\w \.-]*)*\/?$'. 'pii_impact' is a string with an enum that can be one of the following values: 'unknown', 'high', 'moderate', 'low'. 'weight_units' is a string that defaults to 'pounds' and must match the pattern '^(kilograms|pounds)$'. 'min_weight' is an integer that defaults to 0, must be between 0 and 65535, and represents the minimum physical weight. 'max_weight' is an integer that defaults to 0, must be between 0 and 65535, and represents the maximum physical weight. No additional properties are allowed at the root level.
