The schema describes a structured event object. It is an object type. The main properties of the schema are: 'meta' is an object that contains required fields: 'id' (a string matching the UUID pattern), 'type' (a string that must be 'EiffelAlertAcknowledgedEvent'), 'version' (a string that defaults to '1.0.0' and must be '1.0.0'), 'time' (an integer), and 'source' (an object with required fields: 'domainId', 'host', 'name', and 'uri', and a required 'serializer' object with required fields 'groupId', 'artifactId', and 'version', disallowing additional properties). 'data' is an object containing a required 'acknowledgement' (a string), an optional 'entity' (a string), and an optional 'customData' (an array of objects, each requiring 'key' and 'value' strings, disallowing additional properties). 'links' is an array containing objects that require 'type' (a string) and 'target' (a string matching the UUID pattern), disallowing additional properties. The root schema disallows additional properties. No additional properties are allowed at the root level.
