The schema describes an applicant applies to a program at an organization. It is an object type. The main properties of the schema are: 'href' is a string providing a hypertext reference to this resource, and is optional; it must match the pattern '/api/v1/user_identities/\d+/organizations'. 'organizations' is an array containing objects, where each object includes: 'association_name' which is a string or null representing the human-readable name of the association that this organization belongs to, and is required; 'cas3' which is a boolean indicating whether this organization is using CAS3, and is required; 'config_portal_identifier' which is a string representing the organization identifier specific to Config Portal, and is required; 'cycle_name' which is a string or null representing the academic cycle of this organization, and is required; 'id' which is an integer representing the unique identifier of this organization, and is required; 'name' which is a string representing the human-readable name of this organization, and is required. No additional properties are allowed at the root level.
