The schema describes logging details associated with a capture. It is an object type. The main properties of the schema are: 'storedJpgSize' is a number representing the size in bytes of the JPG file(s) and is optional. 'storedRawSize' is a number representing the size in bytes of the RAW file(s) and is optional. 'totalCaptureSize' is a number representing the size in bytes of all pictures captured (SUM storedJpgSize and storedRawSize) and is optional. 'scriptStartDate' is a string or null, recording when the capture script started, and is optional. 'scriptEndDate' is a string or null, recording when the capture script ended, and is optional. 'scriptRuntime' is a number or null, recording script runtime in milliseconds, and is optional. 'processedPicturesCount' is a number representing the number of pictures captured; in some situations, multiple files might be processed in batch, and is optional. 'captureSuccess' is a boolean or null, recording if the capture was successful, and is optional. 'captureDate' is a string or null, indicating the date of the capture, and is optional. No additional properties are allowed at the root level.
