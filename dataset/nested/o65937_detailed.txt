The schema describes a collection of repositories. It is an array type. The main properties of the schema are: 'alias' is a string with a minimum length of 1 and is required. 'name' is a string with a minimum length of 1 and is required. 'url' is a string with a URI format and a minimum length of 1, and is required. 'type' is an enum, which can be one of the following values: 'yast2', 'rpm-md', 'plaindir', or null, and is required. 'enabled' is a boolean and is required. 'gpgcheck' is a boolean and is required. 'autorefresh' is a boolean and optional. 'priority' is an integer with a minimum value of 1 and optional. 'package_manager' is a string with a minimum length of 1 and is required. No additional properties are allowed at the root level.
