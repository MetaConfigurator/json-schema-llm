The schema describes a configuration profile for Per-App VPN mappings. It is an object type. The main properties of the schema are: 'AppLayerVPNMapping' is an array of mapping dictionaries. Each item is an object with properties 'Identifier', which is a string describing the app bundle ID, and 'VPNUUID', which is a string describing the VPNUUID of the Per-App VPN. 'PayloadDescription' is an optional string with a default value of 'Per-App VPN Payload'. 'PayloadDisplayName' is an optional string with a default value of 'Per-App VPN Payload'. 'PayloadIdentifier' is an optional string with a default value of 'com.apple.vpn.managed.appmapping'. 'PayloadOrganization' is an optional string describing the issuing organization of the profile. 'PayloadType' is an optional string with a default value of 'com.apple.vpn.managed.appmapping'. 'PayloadUUID' is an optional string with a default value of an empty string and must match the pattern '^[0-9A-Za-z]{8}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{4}-[0-9A-Za-z]{12}$'. 'PayloadVersion' is an optional number with a default value of 1. No additional properties are allowed at the root level.
