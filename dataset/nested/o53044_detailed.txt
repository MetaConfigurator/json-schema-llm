The schema describes a GeoBlacklight implementation for Solr 4.10+. It is an object type. The main properties of the schema are: 'dc_creator_sm' is an array of strings representing authors, which is optional. 'dc_description_s' is a string describing the layer, which is required. 'dc_format_s' is a string representing the file format for the layer, which is required and must be one of the following values: 'Shapefile', 'GeoTIFF', 'ArcGRID', 'GeoJSON', 'AAIGrid', 'USGSDEM', 'JPEG', 'TIFF'. 'dc_identifier_s' is a string providing a unique identifier for the layer, which is required. 'dc_language_s' is a string representing the language for the layer, which is optional. 'dc_publisher_s' is a string indicating the publisher, which is optional. 'dc_rights_s' is a string indicating access rights for the layer, which is required and must be one of the following values: 'Public', 'Restricted'. 'dc_subject_sm' is an array of strings indicating subjects, which is optional. 'dc_title_s' is a string representing the title for the layer, which is required. 'dc_type_s' is a string indicating the resource type, which is required and must be one of the following values: 'Dataset', 'Image', 'PhysicalObject'. 'dct_isPartOf_sm' is an array of strings indicating the holding dataset, which is optional. 'dct_issued_dt' is a string representing the issued date in XML Schema dateTime format, which is optional. 'dct_provenance_s' is a string indicating the institution holding the layer, which is required. 'dct_references_s' is a string for external resources, which is optional. 'dct_spatial_sm' is an array of strings indicating spatial coverage, which is optional. 'dct_temporal_sm' is an array of strings indicating temporal coverage, which is optional. 'georss_box_s' is a string representing the bounding box, which is required. 'georss_point_s' is a string for point representation, which is optional. 'layer_geom_type_s' is a string indicating the geometry type for layer data, which is required and must be one of the following values: 'Point', 'Line', 'Polygon', 'Raster', 'Scanned Map', 'Mixed'. 'layer_id_s' is a string providing the complete identifier for the layer, which is required. 'layer_modified_dt' is a string representing the last modification date, which is required. 'layer_slug_s' is a string for a unique visible identifier, which is required. 'solr_geom' is a string derived from georss_polygon_s or georss_box_s, which is required and must match the pattern 'ENVELOPE(.*,.*,.*,.*)'. 'solr_year_i' is an integer representing the year for which the layer is valid, which is required. 'uuid' is a string providing a globally unique identifier for the layer, which is required. No additional properties are allowed at the root level.
