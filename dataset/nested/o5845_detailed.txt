The schema describes a Farmware manifest. It is an object type. The main properties of the schema are: 'package' is a string that describes the package name, required. 'language' is a string that describes the package programming language, required. 'author' is a string that describes the package author, required. 'description' is a string that describes the package, required. 'version' is a string that describes the package version, required. 'min_os_version_major' is an integer that describes the minimum Farmbot OS version, required. 'url' is a string that describes the URL to locate this package, required. 'zip' is a string that describes the URL to locate the release of the package, required. 'executable' is a string that describes the name of the executable, required. 'args' is an array that describes arguments to the executable, required. No additional properties are allowed at the root level.
