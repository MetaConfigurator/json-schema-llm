The schema describes an applicant's designation process for programs. It is an object type. The main properties of the schema are: 'designation' is an object representing the designation details for the applicant. It includes properties such as 'decision' which is an object or null representing the decision details, and 'local_status' which is an object or null representing the local status related to the decision. The 'decisions_href' and 'href' properties are strings with specific patterns and descriptions regarding valid decisions and the resource reference respectively. No additional properties are allowed at the root level.
