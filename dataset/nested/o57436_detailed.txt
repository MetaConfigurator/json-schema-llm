The schema describes service sharing information related to data sets (e.g. Inspire). It is an object type. The main properties of the schema are: 'name' is a string representing the short name of a service (e.g. CERN), required, with a minimum length of 1; 'url' is a string representing the HTTP URL addressing the service home page, required, with a minimum length of 1; 'persistent_identifiers' is an array of objects, representing a list of identifiers known by the service (e.g. DOID), required, with a minimum of 1 item, and all items must be unique. Each object in the array has the following properties: 'type' is a string representing the type of persistent identifier (e.g. INSPIRE_RECORD_ID), required, with a minimum length of 2; 'description' is a string providing a short description of the identifier, required, with a minimum length of 1; 'url' is a string showing the identifier placeholder (e.g. http://inspirehep.net/record/&lt;INSPIRE_RECORD_ID&gt;), required, with a minimum length of 1; 'example_value' is a string providing an example value of a possible identifier (e.g. 2001192), required, with a minimum length of 1; 'example_url' is a string providing an example URL with an identifier (e.g. http://inspirehep.net/record/2001192), required, with a minimum length of 1. No additional properties are allowed at the root level.
