The schema describes a cross-site scripting binding for an application firewall profile. It is an object type. The main properties of the schema are: 'alertonly' is a string that can be either 'ON' or 'OFF'. 'as_scan_location_xss' is a string that can be 'FORMFIELD', 'HEADER', or 'COOKIE'. 'as_value_expr_xss' is a string. 'as_value_type_xss' is a string that can be 'Tag', 'Attribute', or 'Pattern'. 'comment' is a string. 'crosssitescripting' is a string. 'formactionurl_xss' is a string. 'isautodeployed' is a string that can be 'AUTODEPLOYED' or 'NOTAUTODEPLOYED'. 'isregex_xss' is a string that can be 'REGEX' or 'NOTREGEX'. 'isvalueregex_xss' is a string that can be 'REGEX' or 'NOTREGEX'. 'name' is a string. 'state' is a string that can be 'ENABLED' or 'DISABLED'. No additional properties are allowed at the root level.
