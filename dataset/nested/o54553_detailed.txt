The schema describes the data contained in a MapChart Movie update object. It is an object type. The main properties of the schema are: 'inplace' is an array of objects, each containing 'position' and 'data' properties which are required. 'position' is an object that either requires 'series' and 'index', or 'id', where 'series' and 'index' must be non-negative integers, and 'id' is a string. 'data' is an object that requires 'x' and 'y', both of which are numbers. 'stream' is an array of objects, where each object has a 'series' property that is an optional non-negative integer and a 'data' property which is required, containing 'x' and 'y' (numbers), along with an optional 'id' (string). 'delete' is an array of objects similar to 'inplace', requiring either 'series' and 'index' or 'id', with 'series' and 'index' needing to be non-negative integers, and 'id' being a string. No additional properties are allowed at the root level.
