The schema describes a graph structure. It is an object type. It defines several reusable components under 'definitions': 'node' is an object, which has required fields: 'fixed', 'id', 'index', and 'name'. It may contain optional fields: 'dashed', 'px', 'py', 'r', 'weight', 'x', and 'y'. The 'r' field can be an integer or a string. The main properties of the schema are: 'lastId' is an integer and required; 'links' is an array of objects containing properties: 'centerText' (optional string), 'dashed' (optional boolean), 'source' (required reference to 'node'), 'sourceText' (optional string), 'strength' (required integer or string), 'target' (required reference to 'node'), and 'targetText' (optional string); 'nodes' is an array of references to 'node'; 'scale' is a number; 'translate' is an array of numbers. No additional properties are allowed at the root level.
