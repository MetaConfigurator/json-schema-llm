The schema describes the configuration for displaying events, repositories, and gauges in a grid. It is an object type. The main properties of the schema are: 'scope' is an object that may contain a 'parameter' which is a string. 'gauge' is an object that describes display properties of the gauges and contains an optional 'interval' which is a number that defaults to 4500 milliseconds. 'events' is an object that specifies which events to display in the grid; it is required to have a 'resource' which is a string formatted as a topic, and may also include an optional array 'types' of strings that defaults to an empty array. 'repos' is an object to define projects to display in the grid; it requires a 'resource' which is a string formatted as a topic. 'search' is an object that requires a 'resource' which is a string formatted as a topic and may also contain an optional array 'fields' of strings or arrays. It also may define a 'match' object with additional properties that map regular expressions to arrays of JSON pointers. No additional properties are allowed at the root level.
