The schema describes a KVM manifest. It is an object type. The main properties of the schema are: 'bootstrapper' is an object that contains a property 'virtio', which is an array of strings and must have at least one item. 'system' is an object that contains a property 'bootloader', which is a string and can be either 'grub' or 'extlinux'. 'volume' is an object that contains a required property 'backing', which is a string and must be 'raw', and an optional property 'partitions', which is an object that has a property 'type', which can be 'none', 'msdos', or 'gpt'. No additional properties are allowed at the root level.
