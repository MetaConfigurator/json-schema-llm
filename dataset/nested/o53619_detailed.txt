The schema describes service configuration for a pantry service. It is an object type. The main properties of the schema are: 'pantryId' is a string [Output only] Unique ID of food pantry, optional, with a minimum length of 1. 'serviceConfigId' is a string [Output only] Unique ID of service, optional, with a minimum length of 1. 'serviceName' is a string Name of service, required, with a minimum length of 1. 'serviceType' is a string Type of the service, required, with allowed values: 'Full', 'EFAP', 'Supplemental', 'Non-food', 'Baby', 'Other'. 'maxOccurrencesInWeekPerHH' is a number Max occurrences of particular service per household per week, required. 'maxOccurrencesInMonthPerHH' is a number Max occurrences of particular service per household per month, required. 'minServicesInMonthPerHH' is a number Minimum number of services per household per month required for eligibility for service, required. 'isActive' is a boolean Flag to indicate if a service is active, optional, with a default value of true. 'isCommoditiesEligibleRequired' is a boolean Flag to indicate if a household must be commodities eligible to receive service, optional, with a default value of false. 'isHomelessRequired' is a boolean Flag to indicate if a household must be homeless to receive service, optional, with a default value of false. 'isMeasuredInActualPounds' is a boolean Flag to indicate if a service is measured in actual pounds, optional, with a default value of false. 'numberPoundsFamilySize1' is a number Number of pounds allowed for service for family with size of 1, required. 'numberPoundsFamilySize2' is a number Number of pounds allowed for service for family with size of 2, required. 'numberPoundsFamilySize3' is a number Number of pounds allowed for service for family with size of 3, required. 'numberPoundsFamilySize4' is a number Number of pounds allowed for service for family with size of 4, required. 'numberPoundsFamilySize5' is a number Number of pounds allowed for service for family with size of 5, required. 'numberPoundsFamilySize6' is a number Number of pounds allowed for service for family with size of 6, required. 'numberPoundsFamilySize7' is a number Number of pounds allowed for service for family with size of 7, required. 'numberPoundsFamilySize8' is a number Number of pounds allowed for service for family with size of 8, required. 'numberPoundsFamilySize9' is a number Number of pounds allowed for service for family with size of 9, required. 'numberPoundsFamilySize10' is a number Number of pounds allowed for service for family with size of 10, required. 'numberPoundsFamilySizeDelta' is a number Number of incremental pounds allowed for service for each family member beyond 10, required. 'createdAt' is a string [Output only] Creation time of the user role, optional, in date-time format, with a minimum length of 1. 'createdBy' is a string [Output only] Creator of the user role, optional, with a minimum length of 1. 'updatedAt' is a string [Output only] Last modification time of the user role, optional, in date-time format, with a minimum length of 1. 'updatedBy' is a string [Output only] User to last modify the user role, optional, with a minimum length of 1. No additional properties are allowed at the root level.
