The schema describes configuration properties for a Portworx deployment. It is an object type. The main properties of the schema are: 'portworx' is an object PX configuration properties. It has required fields: 'cmdargs', 'headers_dir', 'api_port', and 'instances'. The 'cmdargs' property has a default value of '-a' and is a string. The 'headers_dir' property has a default value of '/lib/modules' and is a string. The 'api_port' property has a default value of 9001 and is an integer. The 'instances' property has a default value of 3, a minimum value of 3, and is an integer. 'service' is an object DC/OS service configuration properties. The 'name' property has a default value of 'portworx' and is a string. No additional properties are allowed at the root level.
