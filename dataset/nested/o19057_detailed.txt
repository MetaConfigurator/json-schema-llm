The schema describes information about the vector spatial objects in the resource. It is an object type. It defines several reusable components under 'definitions': 'vectorObject' is an object, information about the geometric objects used in the resource. It requires the field 'objectType' and allows additional properties. The main properties of the schema are: 'topologyLevel' is a string that identifies the degree of complexity of the spatial relationships. It is optional. 'vectorObject' is an array that contains information about the geometric objects used in the resource, requiring at least one item. The items must adhere to the 'vectorObject' definition. The schema allows additional properties at the root level. Additionally, at least one of the properties 'vectorObject' or 'topologyLevel' must be present due to the anyOf constraint. No additional properties are allowed at the root level.
