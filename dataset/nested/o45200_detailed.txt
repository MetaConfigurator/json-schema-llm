The schema describes an array of objects. It is an array type. The main properties of the schema are: Each object in the array has an 'id' which is a number and is required; a 'finisher_id' which is a number and is required; a 'name' which is a string and is required; an 'icon' which is a string and is required; a 'min_rank' which is a number and is required; a 'max_rank' which is a number and is required; and a 'levels' which is an array of objects that is required. Each object in the 'levels' array has a 'min_rank' which is a number and is required; a 'max_rank' which is a number and is required; and a 'points' which is a number and is required. No additional properties are allowed at the root level.
