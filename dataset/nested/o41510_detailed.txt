The schema describes a configuration for an invocation manifest related to DWI Quality Assurance reporting. It is an object type. The main properties of the schema are: 'inputs' is an object that is required and contains the required properties 'bvec', 'bval', and 'nifti'. Each of these properties is an object with a required property 'type' that defines an enum with the allowed value 'bvec', 'bval', and 'nifti' respectively. 'config' is an object that is required but does not define any properties. No additional properties are allowed at the root level.
