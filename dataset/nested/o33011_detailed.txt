'edge' is an object, which has required fields: 'id' (a string), 'center' (a reference to 'point'), and 'ends' (a reference to 'vector'). 'hexInfo' is an object, which includes required fields: 'apothem' and 'circumradius', both numbers. 'point' is an object, which has required fields: 'x' and 'y', both numbers. 'tile' is an object, which has required fields: 'id' (a string), 'center' (a reference to 'point'), and 'type' (a string). 'vector' is an array, which must have a minimum of 2 and a maximum of 2 items, both of which are references to 'point' and must be unique.
