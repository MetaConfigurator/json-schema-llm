The schema describes a color scheme configuration. It is an object type. The main properties of the schema are: 'Vanilla' is a required object that contains color properties. This object requires the fields: 'accent', 'lightAccent', 'text', 'background', 'internal', 'incorrect', 'correct', 'inputColor', 'linkColor', and 'highlightColor', all of which are of type string and formatted as colors. No additional properties are allowed at the root level.
