The schema describes a value representation. It is an object type. The main properties of the schema are: 'name' is a string optional, 'description' is a string optional, 'long_description' is a string optional, 'type' is a string required, 'value' is a string optional, 'schema' is a string optional, 'definitions' is an object optional, 'images' is an array optional, containing items that are objects with a required property 'asset' of type string, and a property 'type' of type string which is optional. No additional properties are allowed at the root level.
