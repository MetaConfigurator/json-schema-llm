The schema describes a BioSamples sample. It is an object type. The main properties of the schema are: 'alias' is a string A sample unique identifier in a submission. It is required. 'attributes' is an object It is an object matching a pattern where each key is arbitrary and the corresponding value is an array of items each containing a 'value' (required, string with minimum length of 1), an optional 'units' (string), and an optional 'terms' (an array of objects each requiring a 'url' formatted as a URI). At least one item is required in the array. 'releaseDate' is a string Date from which this sample is released publicly. It is required and formatted as a date. 'sampleRelationships' is an array of objects List of relationships between samples. Each object requires 'target' (string) and 'nature' (string with a value of either 'derived from', 'child of', 'same as', or 'recurated from'). 'taxon' is a string The taxonomy name for the sample species. It is required. 'taxonId' is an integer The taxonomy id for the sample species. It is required. No additional properties are allowed at the root level.
