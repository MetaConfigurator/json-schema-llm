The schema describes configuration for a piece of video content, over a stream. It is an object type. The main properties of the schema are: 'height' is a number representing the height of the video. 'width' is a number representing the width of the video. 'filesize' is a number representing the size of the video, in bytes. 'audio_codec' is a string representing the audio codec. 'video_codec' is a string representing the video codec. 'stream_type' is a string representing the type of video (e.g. mp4). 'url' is a string representing where to get the stream from. 'bitrate' is a number representing the bitrate of the video. 'provider' is a string representing the provider of the video. No additional properties are allowed at the root level.
