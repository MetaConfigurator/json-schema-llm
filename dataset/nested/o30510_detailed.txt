The schema describes an IC latency profile. It is an object type. The main properties of the schema are: 'builtin' is an object and readonly; it has an enum of allowed values: 'MODIFIABLE', 'DELETABLE', 'IMMUTABLE', 'PARTITION_ALL'. 'isdefault' is a boolean and readonly. 'l7latencymaxnotifycount' is an integer. 'l7latencymonitoring' is a string; it has an enum of allowed values: 'ENABLED', 'DISABLED'. 'l7latencynotifyinterval' is an integer. 'l7latencythresholdfactor' is an integer. 'l7latencywaittime' is an integer. 'name' is a string. 'refcnt' is an integer and readonly.
