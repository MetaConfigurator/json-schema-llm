The schema describes options for the Mocha runner. It is an object type. The main properties of the schema are: 'mochaOptions' is an object, configuration for @stryker-mutator/mocha-runner. It is required and does not allow additional properties. The properties of 'mochaOptions' include: 'require' is an array that contains strings. 'ignore' is an array that contains strings. 'file' is an array that contains strings. 'opts' is a string. 'config' is a string. 'package' is a string. 'no-package' is a boolean. 'no-opts' is a boolean. 'no-config' is a boolean. 'timeout' is a number. 'async-only' is a boolean. 'ui' is a string. 'files' is either a string or an array containing strings, but is deprecated in favor of 'spec'. 'spec' is an array that contains strings. 'grep' is a string. 'extension' is an array that contains strings. No additional properties are allowed at the root level.
