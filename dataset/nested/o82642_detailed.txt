The schema describes a block mixin for canvas. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$'. It is optional and can be appended to the DOM elements' class attribute in rendering environments that support HTML. 'updated_date' is a string in date-time format that represents the update date of the node to which this block is attached. It is optional. 'aside' is a boolean indicating if the block content is indirectly related to the main content of the document. It is optional. No additional properties are allowed at the root level.
