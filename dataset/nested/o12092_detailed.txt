The schema describes an event structure. It is an object type. The main properties of the schema are: 'data' is an object that contains a required property 'schema' of string type with format 'url'. It allows additional properties. 'eventId' is a string that is required. 'followsFrom' is an optional string. 'origin' is a required string. 'schema' is a required string. 'timeIngest' is a required string with format 'date-time'. 'timeOrigin' is a required string with format 'date-time'. 'timeProcess' is a required string with format 'date-time'. No additional properties are allowed at the root level.
