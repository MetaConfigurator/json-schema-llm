The schema describes the structure of a catalog file for SchemaStore.org. It is an object type. The main properties of the schema are: 'schemas' is an array that represents a list of JSON schema references. It is required and enforces a minimum length of 1. Each item in the array is an object that has required fields: 'name', which is a string describing the schema's name; 'url', which is a string for an absolute URL to the schema location; and 'description', which is a string providing a description of the schema. No additional properties are allowed in the items of the array. The 'version' is a number that indicates the schema version of the catalog and is required. No additional properties are allowed at the root level.
