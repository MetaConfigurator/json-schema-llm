The schema describes an application event. It is an object type. The main properties of the schema are: 'agent' is an object representing the agent with required fields 'name' (a string with maxLength 1024) and 'version' (a string with maxLength 1024). 'argv' is an array or null that allows a minimum of 0 items. 'framework' is an object or null representing the framework with required fields 'name' (a string with maxLength 1024) and 'version' (a string with maxLength 1024). 'language' is an object or null with a required field 'name' (a string with maxLength 1024) and an optional field 'version' (a string or null with maxLength 1024). 'name' is a string that is the immutable name of the app emitting this event, which must match the pattern '^[a-zA-Z0-9 _-]+$' and has a maxLength of 1024. 'pid' is a number or null. 'process_title' is a string or null with a maxLength of 1024. 'runtime' is an object or null representing the runtime with required fields 'name' (a string with maxLength 1024) and 'version' (a string with maxLength 1024). 'version' is a string or null with a maxLength of 1024. No additional properties are allowed at the root level.
