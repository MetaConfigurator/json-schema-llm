The schema describes a Twitter user. It is an object type. The main properties of the schema are: 'created_at' is a string formatted as date-time, optional. 'description' is a string, optional. 'favourites_count' is an integer, optional. 'followers_count' is an integer, optional. 'friends_count' is an integer, optional. 'geo_enabled' is a boolean, optional. 'id' is an integer, optional. 'id_str' is a string, optional. 'lang' is a string, optional. 'listed_count' is an integer, optional. 'location' is a string, optional. 'name' is a string, optional. 'profile_image_url' is a string, optional. 'profile_image_url_https' is a string, optional. 'screen_name' is a string, optional. 'statuses_count' is an integer, optional. 'time_zone' is a string, optional. 'url' is a string, optional. 'utc_offset' is an integer, optional. 'verified' is a boolean, optional. No additional properties are allowed at the root level.
