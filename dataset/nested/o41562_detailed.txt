The schema describes an invocation manifest for a pipeline. It is an object type. The main properties of the schema are: 'inputs' is an object that is required. It contains a required property 'api_key', which is also an object that does not specify additional details. 'config' is an object that is required. It contains two required properties: 'freesurfer_license', which is a string, and 'clip', which is an integer. No additional properties are allowed at the root level.
