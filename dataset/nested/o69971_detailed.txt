The schema describes a loan. It is an object type. The main properties of the schema are: '$schema' is a string representing the schema. It is required. 'item' is an object representing the item associated with the loan. It contains a property '$ref' which is a string that must match the pattern '^https://ils.rero.ch/api/items/.+?$' with a default value of 'https://ils.rero.ch/api/items/'. 'extension_count' is an integer representing the loan extension count. It is optional. 'pid' is a string representing the loan PID. It is required. 'transaction_date' is a string formatted as date-time representing the loan date. It is optional. 'patron_pid' is a string representing the patron PID. It is optional. 'document_pid' is a string representing the document PID. It is optional. 'item_pid' is a string representing the item PID. It is optional. 'transaction_user_pid' is a string representing the transaction user PID. It is optional. 'transaction_location_pid' is a string representing the transaction location PID. It is optional. 'pickup_location_pid' is a string representing the request pickup location PID. It is optional. 'request_expiry_date' is a string formatted as date-time representing the request expiry date. It is optional. 'start_date' is a string formatted as date-time representing the transaction start date. It is optional. 'end_date' is a string formatted as date-time representing the transaction end date. It is optional. 'organisation' is an object representing the organisation involved in the loan. It contains a property '$ref' which is a string matching the pattern '^https://ils.rero.ch/api/organisations/.*?$'. 'state' is a string representing the state of the loan, with allowed values: 'CREATED', 'PENDING', 'ITEM_ON_LOAN', 'ITEM_RETURNED', 'ITEM_IN_TRANSIT_FOR_PICKUP', 'ITEM_IN_TRANSIT_TO_HOUSE', 'ITEM_AT_DESK', 'CANCELLED'. It is required. 'trigger' is a string representing the name of the explicit action that triggered the transition to the current state. It is optional. No additional properties are allowed at the root level.
