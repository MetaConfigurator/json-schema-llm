The schema describes a user registration form. It is an object type. The main properties of the schema are: 'email' is a string that must adhere to the email format and has a maximum length of 1024 characters. 'firstName' is a string with a maximum length of 1024 characters. 'lastName' is a string with a maximum length of 1024 characters. 'password' is a string that must be at least 8 characters long and can be up to 2048 characters long. 'tokenCutoff' is a string that must adhere to the date-time format. 'userTags' is an object that allows properties matching the regex pattern '^[0-9a-zA-Z_-]{1,255}$' with string values that must be between 1 and 255 characters in length. No additional properties are allowed at the root level.
