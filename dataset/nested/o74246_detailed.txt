The schema describes a data structure for storing file metadata. It is an object type. The main properties of the schema are: 'name' is a string and is required. 'created' has no specified type but is required. 'modified' has no specified type but is required. 'type' is a string and is optional. 'mimetype' is a string and is optional. 'size' is an integer and is required. 'hash' is a string and is required. 'modality' is a string and is optional. 'measurements' is an array of strings and is optional, with unique items. 'tags' is an array of strings and is optional, with unique items. 'info' is an object and is optional. No additional properties are allowed at the root level.
