The schema describes the DCOS Config API Endpoint. It is an object type. The main properties of the schema are: 'uiConfiguration' is an object that contains configuration settings for the user interface. It is required and includes the property 'plugins', which is also an object and is required. The 'plugins' object contains two properties: 'banner' and 'tracking'. 'banner' is an object that adds a custom header/footer banner to the UI. It is required and must include 'headerTitle', 'headerContent', and 'footerContent'. The 'tracking' property is an object that adds tracking to the UI and is required, containing the 'enabled' boolean property. No additional properties are allowed at the root level.
