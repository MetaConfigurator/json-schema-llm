The schema describes a Node Profile Import. It is an object type. The main properties of the schema are: 'fabric_fq_name' is an array of strings, optional. 'encoded_file' is a string, required, with the description 'The Encoded File to Import'. 'file_format' is a string, required, with the description 'The format of the encoded input file' and allowed values are 'yaml', 'json', 'YAML', 'JSON'. 'cc_username' is a string, optional, with the description 'The Contrail Command Login Username'. 'cc_password' is a string, optional, with the description 'The Contrail Command Login Password'. 'contrail_command_host' is a string, optional, with the description 'The Contrail Command Hostname/IP'. No additional properties are allowed at the root level.
