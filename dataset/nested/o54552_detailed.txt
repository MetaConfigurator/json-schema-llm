The schema describes the data contained in a MapChart InPlace update object. It is an object type. The main properties of the schema are: 'inplace' is an array containing objects that require 'position' and 'data'. Each 'position' is an object that includes required integer properties 'series' (minimum 0) and 'index' (minimum 0). Each 'data' is an object that includes required number properties 'x' and 'y', and an optional string property 'id'. No additional properties are allowed at the root level.
