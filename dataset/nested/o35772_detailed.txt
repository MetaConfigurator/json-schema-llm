The schema describes representing the AWS-EC2 as the deployment Environment. It is an object type. The main properties of the schema are: '$schema-location' is a string that must be one of the enum value 'AWS-Provider-Schema.json'. 'id' is a string and is required. 'type' is a string, is required, and has a default value of an empty string. 'region' is a string with a default value of 'us-east-1' and is optional. 'access-key' is a string and is required with a default value of an empty string. 'secret-key' is a string and is required with a default value of an empty string. No additional properties are allowed at the root level.
