The schema describes the structure for a Codex Coverage Schema. It is an object type. The main properties of the schema are: 'id' is a string and is required. 'range' is an array containing objects, each with 'begin' and 'end' as strings. Additional items are allowed, and items must be unique. 'statement' is a string and is optional. 'embargo' is an array containing objects, each with 'begin' and 'end' as strings. Additional items are allowed, and items must be unique. No additional properties are allowed at the root level.
