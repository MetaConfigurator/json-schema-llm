The schema describes a configuration for deploying Humio agents on a cluster. It is an object type. The main properties of the schema are: 'humio' is an object representing configuration details for Humio, required fields are 'host', 'dataspace', 'ingestToken', and 'agents'; 'node' is an object for node-specific settings, required fields are 'cpus', 'mem', and 'datadir'; 'service' is an object for service details, required field is 'name'. No additional properties are allowed at the root level.
