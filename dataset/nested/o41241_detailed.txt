The schema describes an invocation manifest for fMRIPREP: A Robust Preprocessing Pipeline for fMRI Data (1.0.0-rc1). It is an object type. The main properties of the schema are: 'inputs' is an object and is required. It contains a required property 'bids_dataset', which is also an object. 'bids_dataset' has a property 'type' that is a string with an enum value of 'archive'. 'config' is an object and is required, but it does not define any properties. No additional properties are allowed at the root level.
