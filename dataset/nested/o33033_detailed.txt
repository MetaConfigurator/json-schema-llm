The schema describes the configuration for the Geniventure tutorial authoring document. It is an object type. The main properties of the schema are: 'levels' is an array containing configuration for the levels that comprise the application, which must contain objects representing each level. Each level object must have a 'missions' property, which is an array of mission objects. Each mission object must have a 'challenges' property that is an array of challenge objects. Each challenge object must have a 'steps' property that is an array of step objects. Each step object is required to have the 'element' and 'text' properties. The 'element' and 'text' properties are strings, while 'longText', 'more', and 'pulseElement' are optional string properties. The 'location' property is a string conforming to the pattern '(top-left|top-center|top-right|middle-left|middle-center|middle-right|bottom-left|bottom-center|bottom-right)'. No additional properties are allowed at the root level.
