The schema describes the metadata format for all federal open data. It is an object type. The main properties of the schema are: 'id' is a string that represents the UUID. 'revision_id' is a string that represents the Revision ID. 'url' is a string that represents the URL. 'description' is a string that represents the Description. 'format' is a string that represents the Format. 'hash' is a string that represents the hash. 'state' is a string that represents the state. 'Position' is a string that represents the position. 'revision_timestamp' is a string that represents the Revision Timestamp. 'name' is a string that represents the Name. 'resource_type' is a string that represents the Resource Type. 'url_type' is a string that represents the URL Type. 'mimetype' is a string that represents the MIME Type. 'webstore_url' is a string that represents the Web Store URL. 'size' is a string that represents the Size. 'created' is a string that represents the Created date. 'resource_group_id' is a string that represents the Resource Group UUID. 'cache_last_updated' is a string that represents when the Cache was Last Updated. 'webstore_last_updated' is a string that represents when the Web Store was Last Updated. 'datastore_active' is a string that indicates if the Datastore is active. 'last_modified' is a string that represents the Last Modified date. No additional properties are allowed at the root level.
