The schema describes a collection of processing inputs. It is an array type. The main properties of the schema are: 'input_data' is an object with an optional property 'dataset' that is a string. 'user_code' is an object with an optional property 'config_files' that is a string. 'run_instructions' is an object with an optional property 'type' that is a string and must be one of the following values: 'readme', 'makefile', 'upload'. 'output_data' is an object with an optional property 'output_url' that is a string. 'datacard' is an optional property that is a string. The items in the array are validated against the defined object structure. No additional properties are allowed at the root level.
