The schema describes OpenVPN configuration settings. It is an object type. The main properties of the schema are: 'openvpn' is an object that contains settings for OpenVPN. Required fields include 'admin_port', 'cpus', 'framework-name', 'instances', 'mem', 'ovpn_password', 'ovpn_username', and 'server_port'. Each property has default values: 'admin_port' defaults to 5000; 'cpus' defaults to 0.01; 'framework-name' defaults to 'openvpn'; 'instances' defaults to 1; 'mem' defaults to 128.0; 'ovpn_password' defaults to ''; 'ovpn_username' defaults to 'admin'; and 'server_port' defaults to 1194. The 'admin_port' must be an integer with a minimum value of 1. The 'cpus' must be a number with a minimum value of 0.01. The 'framework-name' must be a string. The 'instances' must be an integer with a minimum value of 1. The 'mem' must be a number with a minimum value of 64.0. The 'ovpn_password' must be a string with a minimum length of 1. The 'ovpn_username' must be a string with a minimum length of 1. The 'server_port' must be an integer with a minimum value of 1. No additional properties are allowed at the root level.
