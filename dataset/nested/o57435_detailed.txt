The schema describes a claim provided by a service. It is an object type. The main properties of the schema are: 'claimant' is a string representing the short name of the service used for subscription, required. 'subject' is an object describing the subject of the predicate (claim), required, with properties: 'type' (string, required) for the type of persistent identifier, and 'value' (string, required) for the actual value of the identifier; no additional properties are allowed. 'claim' is an object representing the type of claim, required, with properties: 'predicate' (string, required) which must be one of ['is_same_as', 'is_different_than', 'is_erratum_of', 'is_superseded_by', 'is_cited_by', 'is_software_for', 'is_dataset_for'], 'datetime' (string, optional) for the date time, 'certainty' (number, required) which must be between 0 and 100, and 'arguments' (object, optional) with properties: 'human' (string, enum, can be 'True' or 'False'), 'actor' (string, optional), and 'role' (string, optional). 'object' is an object describing the object of the predicate (claim), required, with properties: 'type' (string, required) for the type of persistent identifier, and 'value' (string, required) for the actual value of the identifier; no additional properties are allowed. No additional properties are allowed at the root level.
