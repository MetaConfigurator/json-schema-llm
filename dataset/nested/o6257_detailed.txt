The schema describes a configuration for a time series query. It is an object type. The main properties of the schema are: 'start' is a number optional, 'end' is a number optional, 'duration' is a number optional, 'resolution' is a number optional, 'aggregation' is a string optional, with allowed values: 'FIRST', 'LAST', 'COUNT', 'MAX', 'MIN', 'MEDIAN', 'MEAN', 'SUM'. 'attributes' is an array optional, containing strings that match the pattern '^[0-9a-zA-Z_-]{1,255}$'. 'deviceTags' is an array optional, containing objects with properties: 'key' is a string with a pattern '^[0-9a-zA-Z_-]{1,255}$' and 'value' is a string with a minimum length of 1 and maximum length of 255. 'deviceIds' is an array optional, containing strings that match the pattern '^[A-Fa-f\d]{24}$'. No additional properties are allowed at the root level.
