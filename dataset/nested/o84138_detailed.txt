The schema describes a RenderTheFuck stack diagram. It is an object type. The main properties of the schema are: 'layers' is an array that contains objects, each of which has a required property 'items' that is also an array. Each item in 'items' is an object with a required property 'name' (a string with a minimum length of 1) and an optional property 'parent' (a number). 'settings' is an object that has a required property 'title' (a string with a minimum length of 1). 'wtf' is a string with an enum value of 'stack'. All three properties, 'wtf', 'settings', and 'layers', are required. No additional properties are allowed at the root level.
