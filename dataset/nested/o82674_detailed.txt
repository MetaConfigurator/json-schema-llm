The schema describes a node reference block. It is an object type. The main properties of the schema are: 'etag' is a string that must match the pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the pattern '^[\w\s-]+$'. It is optional and in rendering environments that support HTML, it can be appended to the DOM elements' class attribute. 'updated_date' is a string in date-time format that represents an update that occurred on the node this block is attached to. It is optional. 'aside' is a boolean that, when true, indicates this block represents a portion of a document whose content is only indirectly related to the document's main content. It is optional. 'node_ref' is a string that must match the pattern '^[\w\/\.:-]+$'. It is required. No additional properties are allowed at the root level.
