The schema describes event source templates. It is an object type. The main properties of the schema are: 'id' is an integer that represents a unique identifier given by the server, required. 'key' is a string that serves as a unique key generated by the server to be used in events, required. 'name' is a string that identifies the event source template, required. 'public' is a boolean that indicates whether the event source template is available to everybody, required. 'organizationId' is an integer that refers to the organization offering the event source template, required. 'configuration' is an object that provides a schema to assist the WUI for configuration, which includes 'schema' (a required object representing a valid JSON Schema) and 'url' (a required string representing the configuration URL). 'configurationUi' is an optional object providing the configuration for the dynamic UI, which includes 'schemaForm' (an optional array that represents a valid form schema). No additional properties are allowed at the root level.
