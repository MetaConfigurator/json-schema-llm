The schema describes a gallery block mixin. It is an object type. The main properties of the schema are: 'etag' is a string that must match the regex pattern '^[\w\.:-]+$'. It is optional. 'css_class' is a string that must match the regex pattern '^[\w\s-]+$' and may be appended to the DOM elements' class attribute in rendering environments that support HTML. It is optional. 'updated_date' is a string formatted as date-time. It represents an update to the node this block is attached to, and it is optional. 'aside' is a boolean indicating if the block represents a portion of a document related indirectly to the main content; when true, it represents an aside. It is optional. 'node_ref' is a string that must match the regex pattern '^[\w\/\.:-]+$' and is required. 'title' is a string with a minimum length of 0 and a maximum length of 255, serving as an optional override for the node's title. 'launch_text' is a string with a minimum length of 0 and a maximum length of 255. It is optional. 'poster_image_ref' is a string that must match the regex pattern '^[\w\/\.:-]+$' and serves as a reference to an image asset used as the poster. It is optional. 'start_at_poster' is a boolean that defaults to true; when true, the gallery opens at the selected poster image, which should be included in the selected gallery. It is optional. No additional properties are allowed at the root level.
