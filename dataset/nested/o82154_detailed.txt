The schema describes a schema for Travis Hook payload. It is an object type. It defines several reusable components under 'definitions': 'repository' is an object, containing properties 'name' and 'owner_name', both of which are required strings with a maximum length of 100 characters. The main properties of the schema are: 'branch' is a string describing the name of the branch, required, with a maximum length of 100 characters; 'commit' is a string representing the Git SHA Commit ID, required, with a maximum length of 100 characters; 'repository' is a reference to the 'repository' definition; 'status' is a number indicating the status of the build, required, with an enum of allowed values: 0, 1. No additional properties are allowed at the root level.
