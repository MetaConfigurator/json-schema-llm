The schema describes O3D3xx Distance model configuration. It is an object type. The main properties of the schema are: 'Name' is a string required property. 'PrivateType' is a string optional property with a default value of 'Distance'. 'Type' is a string required property with an enum that only allows the value 'Distance'. 'ROIs' is a string required property. 'SwitchPoint1' is a number required property. 'SwitchPoint2' is a number required property. 'ValueOfBenefit' is an integer required property with a minimum value of 0, a maximum value of 2, and a default value of 0. 'UseAnchorTracking' is a boolean optional property with a default value of false. 'HasAnchorTeachData' is a boolean optional property with the 'readonly' attribute. 'AnchorData' is a string optional property with the 'private' and 'noexport' attributes. No additional properties are allowed at the root level.
