The schema describes a data structure for various URIs and associated counts. It is an object type. The main properties of the schema are: 'users' is a string formatted as a URI, optional. 'tracks' is a string formatted as a URI, optional. 'groups' is a string formatted as a URI, optional. 'sensors' is a string formatted as a URI, optional. 'phenomenons' is a string formatted as a URI, optional. 'measurements' is a string formatted as a URI, optional. 'schema' is a string formatted as a URI, optional. 'termsOfUse' is a string formatted as a URI, optional. 'privacyStatements:' is a string formatted as a URI, optional. 'statistics' is a string formatted as a URI, optional. 'counts' is an object with properties defined by the pattern '.+' that map to integer values and are optional. No additional properties are allowed at the root level.
