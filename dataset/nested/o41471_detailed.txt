The schema describes an invocation manifest for SCITRAN: DWI Split Shells. It is an object type. The main properties of the schema are: 'inputs' is an object that is required, containing required properties 'bvec', 'bval', and 'dwi', each of which is an object with a property 'type' that has an enum of allowed values: 'bvec' for 'bvec', 'bval' for 'bval', and 'nifti' for 'dwi'. 'config' is an object that is required, containing a required property 'doNorm' which is a boolean with a default value of true. No additional properties are allowed at the root level.
