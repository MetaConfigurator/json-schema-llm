The schema describes the configuration for a marathon-lb instance. It is an object type. The main properties of the schema are: 'marathon-lb' is an object that contains several properties. 'cpus' is a number that specifies CPU shares to allocate to each marathon-lb instance, with a minimum value of 0.5 and a default of 0.5. 'framework-name' is a string that specifies the framework name, with a default of 'marathon-lb'. 'instances' is an integer that indicates the number of instances to run, with a minimum value of 1 and a default of 1. 'mem' is a number that specifies the memory (MB) to allocate to each marathon-lb task, with a minimum value of 128.0 and a default of 256.0. 'role' is a string that indicates the deployment of marathon-lb on nodes with a specific role, with a default of null. 'ssl-cert' is a string that represents the TLS Cert and private key for HTTPS. No additional properties are allowed at the root level.
