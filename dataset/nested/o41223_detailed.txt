The schema describes an invocation manifest for DICOM Send. It is an object type. The main properties of the schema are: 'inputs' is an object, required. It contains a required property 'file', which is also an object with no defined properties. 'config' is an object, required. It has three required properties: 'destination', which is a string with a default value of '0.0.0.0'; 'called_ae', which is a string with a default value of 'scanner'; and 'port', which is a number with a default value of '104'. No additional properties are allowed at the root level.
