The schema describes feedback submission. It is an object type. The main properties of the schema are: 'email' is a string that contains the end user's email address. It is required. 'feedback' is a string field containing feedback from the user. It is required. 'id' is an integer representing UTC milliseconds to create a unique timestamp based ID. It is required. 'name' is a string for the username in 'GivenName FamilyName' format. It is required. 'rating' is an integer denoting a star rating, with valid values being 1, 2, 3, 4, or 5. It is required. 'source' is a string indicating where the feedback came from; it has a default value of 'webapp' and is required. No additional properties are allowed at the root level.
