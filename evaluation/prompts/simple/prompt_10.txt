The schema describes a bot that helps onboarding new open-source contributors. It is an object type. The main properties of the schema are: 'labels' is an array of strings that sets the labels if 'first-timers-only' is not what you are looking for. 'template' is a string that specifies the path to your template, relative from the repository root. 'repository' is a string that allows you to specify a different repository than where the problem is. The bot must be installed on the configured repository. No additional properties are allowed at the root level.
