The schema describes configuration settings for JSInspect. It is an object type. The main properties of the schema are: 'identifiers' is a boolean A flag indicating whether to limit the search to nodes with matching identifiers. It is optional and has a default value of false. 'ignore' is a string A regular expression used for matching paths to ignore. It is optional. 'jsx' is a boolean A flag indicating whether to process JSX files. It is optional and has a default value of false. 'reporter' is a string The name of the reporter to be used. It is optional, has a default value of 'default', and accepts the following values: 'default', 'json', 'pmd'. 'suppress' is an integer The number of lines at which diffs should be suppressed. A value of 0 is off. It is optional, has a default value of 100, and must be greater than or equal to 0. 'threshold' is an integer A threshold determining the smallest subset of nodes to analyze. It is optional, has a default value of 15. No additional properties are allowed at the root level.
