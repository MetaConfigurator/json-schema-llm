The schema describes configuration settings for JSInspect. It is an object type. The main properties of the schema are: 'identifiers' is a boolean indicating whether to limit the search to nodes with matching identifiers, optional with a default value of false. 'ignore' is a string that represents a regular expression used for matching paths to ignore, optional. 'jsx' is a boolean indicating whether to process JSX files, optional with a default value of false. 'reporter' is a string that specifies the name of the reporter to be used, optional with a default value of 'default'. Allowed values are: 'default', 'json', 'pmd'. 'suppress' is an integer that specifies the number of lines at which diffs should be suppressed, optional with a default value of 100 and a minimum value of 0. 'threshold' is an integer determining the smallest subset of nodes to analyze, optional with a default value of 15. No additional properties are allowed at the root level.
