The schema describes configuration settings for Visual Studio Live Share files. It is an object type. The main properties of the schema are: 'excludeFiles' is an array of strings indicating the files that should be completely unavailable to guests when shared (e.g. secrets), 'gitignore' is a string that indicates how .gitignore files should be treated with respect to excluding/hiding files from guests. It allows the enum values 'none', 'hide', 'exclude' and has a default value of 'hide', 'hideFiles' is an array of strings indicating the files that should be hidden from guest's file trees, but still accessible.  No additional properties are allowed at the root level.
