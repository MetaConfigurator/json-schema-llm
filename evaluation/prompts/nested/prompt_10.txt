The schema describes configuration settings for an ES6 Import Sorter. It is an object type. The main properties of the schema are: 'sourcePrefixes' is an array of strings representing the prefixes of your source imports. 'sectionPrefix' is a string indicating the prefix of your section title. 'sectionNames' is an array of strings containing the names following 'sectionPrefix' of your section title. 'startImportBlockSign' is a string that indicates the sign to mark the start of the import block, with a default value of an empty string. 'endImportBlockSign' is a string that indicates the sign to mark the end of the import block, with a default value of an empty string. 'statementTerminator' is a string that indicates the sign to mark the end of an import statement, with a default value of ';'. 'preCommands' is an array that contains commands run before sorting, which can either be a string as a terminal command or an object that defines a command with a vscode command option. If it is an object, it requires a 'command' field, which is a non-empty string, and can have an optional 'system' field defining where the command will execute, which defaults to 'terminal' and can be either 'vscode' or 'terminal'. No additional properties are allowed at the root level.
