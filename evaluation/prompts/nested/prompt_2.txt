The schema describes configuration for Crowdin, a crowd-translation platform. It is an object type. The main properties of the schema are: 'project_id' is a string identifying the project ID, optional. 'project_id_env' is a string for the project ID environment variable, optional. 'api_token' is a string for the API token, optional. 'api_token_env' is a string for the API token environment variable, optional. 'base_path' is a string for the base path, optional. 'base_path_env' is a string for the base path environment variable, optional. 'base_url' is a string formatted as a URI for the base URL, optional. 'base_url_env' is a string for the base URL environment variable, optional. 'preserve_hierarchy' is a boolean to preserve hierarchy, optional. 'commit_message' is a string for the commit message, optional. 'append_commit_message' is a boolean indicating whether to replace the default commit message with the one specified in 'commit_message', optional. 'export_languages' is an array of strings specifying a list of specific languages to export, optional. 'files' is an array of objects where each object represents a file configuration with various properties : 'source'is a string, 'translation'is a string, 'ignore' is an array of strings, 'translation_replace' allows additional properties of type string, 'excluded_target_languages' is an array of strings, 'scheme' is a string, 'dest' is a string, 'type' is a string, 'update_option' can have values update_as_unapproved, update_without_changes, 'translate_content' is a number type that should have a minimum value of 0 and maximum value of 1 and default value of 1, 'translate_attributes' is a number type that should have a minimum value of 0 and maximum value of 1 and default value of 1, 'content_segmentation' is a number type that should have a minimum value of 0 and maximum value of 1 and default value of 1, 'translatable_elements' is an array of strings, 'skip_untranslated_strings' is a boolean, 'skip_untranslated_files' is a boolean, 'export_only_approved' is a boolean, 'escape_quotes' is a number type that should have a minimum value of 0 and maximum value of 3 and default value of 3, 'escape_special_characters' is a number type that should have a minimum value of 0 and maximum value of 1 and default value of 1, 'first_line_contains_header' is a boolean, 'languages_mapping' is an object with 2 properties "two_letters_code", "android_code" both allows additional properties of type string , and 'labels' is an array of strings, all optional. No additional properties are allowed at the root level.
