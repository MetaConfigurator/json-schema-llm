The schema describes a JSON schema for a .mocharc.[json|yml|yaml] file. It is an object type. It defines several reusable components under 'definitions': 'bool' is a boolean, 'int' is an integer with a minimum value of 0, 'string' is a string, and 'string-array' can be either a string or an array of strings. The main properties of the schema are: 'allow-uncaught' refers to bool in the definitions, optional. 'async-only' refers to bool in the definitions, optional. 'bail' refers to bool in the definitions, optional. 'check-leaks' refers to bool in the definitions, optional. 'delay' refers to bool in the definitions, optional. 'exit' refers to bool in the definitions, optional. 'forbid-only' refers to bool in the definitions, optional. 'forbid-pending' refers to bool in the definitions, optional. 'global' refers to string-array in the definitions, optional. 'jobs' refers to int in the definitions, optional. 'parallel' refers to bool in the definitions, optional. 'retries' refers to int in the definitions, optional. 'slow' refers to int in the definitions, optional. 'timeout' refers to int in the definitions, optional. 'ui' refers to string in the definitions, optional. 'color' refers to bool in the definitions, optional. 'diff' refers to bool in the definitions, optional. 'full-trace' refers to bool in the definitions, optional. 'growl' refers to bool in the definitions, optional. 'inline-diffs' refers to bool in the definitions, optional. 'reporter' refers to string in the definitions, optional. 'reporter-option' refers to string-array in the definitions, optional. 'config' refers to string in the definitions, optional. 'package' refers to string in the definitions, optional. 'extension' refers to string-array in the definitions, optional. 'file' refers to string-array in the definitions, optional. 'ignore' refers to string-array in the definitions, optional. 'recursive' refers to bool in the definitions, optional. 'require' refers to string-array in the definitions, optional. 'sort' refers to bool in the definitions, optional. 'watch' refers to bool in the definitions, optional. 'watch-files' refers to string-array in the definitions, optional. 'watch-ignore' refers to string-array in the definitions, optional. 'fgrep' refers to string in the definitions, optional. 'grep' refers to string in the definitions, optional. 'invert' refers to bool in the definitions, optional. 'spec' refers to string-array in the definitions, optional. 'enable-source-maps' refers to bool in the definitions, optional. No additional properties are allowed at the root level.
