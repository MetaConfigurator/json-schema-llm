The schema describes the structure of a .talismanrc configuration file. It is an object type. The main properties of the schema are: 'fileignoreconfig' is an array of objects with each object having properties 'filename' is a required string, 'checksum' is an optional string, 'ignore_detectors' is an optional array of strings with allowed values "filecontent", "filename", "filesize", and 'allowed_patterns' is an optional array of strings. 'scopeconfig' is an array of objects with a required 'scope' field. 'allowed_patterns' is an array of strings for keywords to reduce false positives. 'custom_patterns' is an array of strings for custom regex patterns. 'custom_severities' is an array of objects requiring 'detector' and 'severity' fields with 'severity' values of 'low', 'medium', or 'high'. 'threshold' is a string with allowed values 'low', 'medium', or 'high'. 'version' is a string for the .talismanrc version. No additional properties are allowed at the root level.
